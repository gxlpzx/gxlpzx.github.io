把握三个特点　解决三个问题　发挥三个作用

山西团省委副书记　刘志杰


　　江泽民同志关于中国共产党始终代表中国先进社会生产力发展要求、始终代表中国先进文化的前进方向；始终代表中国最广大人民的根本利益的重要论述，高屋建瓴，总揽全局，是运用辩证唯物主义和历史唯物主义的观点和方法，总结历史，立足现实，前瞻未来作出的精辟论断。“三个代表”的论述作为一个科学的体系、完整的概念，内涵是相当丰富的。联系共青团工作实际，学习三个代表的论述，我最深的体会是：“三个代表”是党的建设的根本指针，团工作的行动纲领。这不仅是“三个代表”丰富的思想内涵决定的，而且是我们共青团工作的性质决定的，更是我们共青团建设的现实情况决定的。贯彻落实“三个代表”的重要精神，应该把握三个特点，解决三个问题，发挥三个作用。

　　一、把握三个特点，深刻理解“三个代表”重要思想的本质特征

　　把握三个特点主要指“三个代表”的论述是在新的历史条件下，我们党对党的性质、党的根本宗旨这一系列立党之本做出的新概括；是对马克思主义、列宁主义、毛泽东思想、邓小平建党理论的新发展；是在当前形势下对各级党的领导和党员干部提出的新要求。这是最本质的东西。每书记讲“三个代表”是我们的立党之本、执政之基、力量之源，就在于这三个特点。联系现实，“三个代表”也具有鲜明的时代性，具有鲜明的开拓性，具有鲜明的现实性。鲜明的时代性就是因为现在我们确实处在一个世界范围内的大转折，整个国内经济的大发展，我们党的建设出现一系列新的大变化这样一个新的大背景下，“三个代表”的思想充分反映和体现了当前时代的特点。它具有鲜明的开拓性是因为“三个代表”思想从本质上讲是对我们党的性质、党的宗旨的一个新的概括，它赋予了我们党的性质、党的宗旨以新的内涵，新的内容，新的要求，开拓了一个在新的形势下党怎么样能更好地代表全国各族人民的根本利益，怎么样更好地坚持工人阶级先锋队的性质这样一个深刻的时代内容。同时，“三个代表”从生产力发展的要求，从先进文化发展的方向，从对广大人民群众根本利益的代表这样三个方面去重新界定我们党的性质，具有非常深刻的现实意义，所以说它也有非常鲜明的现实性。我们党是工人阶级的先锋队，党的阶级基础是工人阶级，但并不意味着除了工人阶级之外，其它成份人员就不可以进入我们党的组织，这应该是在新的形势下，我们党所面临的最现实的问题。从马克思主义的建党学说来讲，党的性质不单单取决于赏队伍的构成，而最关键的是决定于党的最高纲领。正是有这样的建党思想，所以我们党从建党初期吸收赏时，不仅有工人阶级，而且有工人阶级的同盟军，有农民，更有一大批的知识分子，特别是建党初期一大批的党的领导人，都是知识分子出身，结合不同阶段任务也吸收了相当一部分小资产阶级的人士，改造过来，加入我们党的组织。那么在现在新的经济组织蓬勃发展，原来我们过去确认的党的阶级基础，大批工人重新转岗再就业这样的情况下，怎么去认识党的先进性，怎么去保持党的活力，“三个代表”的思想给了我们新的启示，给了我们新的召唤，所以说这三个特点是学习总书记“三个代表”重要思想首先必须把握的。

　　二、解决三个问题，真正把“三个代表”的重要思想作为共青团的行动纲领

　　联系我们共青团工作的实际，团的建设应该突出地解决好三个问题，即巩固团的组织体系，明晰团的法律地位，拓宽团的管理领域。

　　所谓巩固团的组织体系，本身就是我们党的性质的要求，是《党章》的要求。因为共青团是党的助手，后备军，共青团是先进青年组织，党提出“三个代表”，首先必须有一个巩固的、健全的、充满活力的、有战斗力的组织体系，这也正是我们前一段在运城召开的党建带团建工作会议想着力解决的一个问题。在这个会上省委领导做了非常重要的讲话，特别是刘泽民副书记代表省委的讲话，全面阐述了党团之间的关系，夜印象最深的是八个字，即“党盛团兴，团兴党强”，这充分反映了党和团的关系，这也是在新的形势下省委对我们团的工作的一个新的具体要求。巩固团的组织体系，当前最重要的是要解决好我们团的组织覆盖面小和辐射力弱的问题。就是在一些新的经济组织、新的社区、新的地方，我们团的组织还没有覆盖到，在一些过去传统的行业、传统的领域建立的团的组织还有相当一部分处于松散瘫痪的状态。所以只有一方面确实增强我们现有团组织的整顿建设的力度，增强它的辐射力，影响力，另一方面加大在新的领域、新的社区建立我们团的组织、发挥我们覆盖面大的优势，才能真正从组织上为实践“三个代表”的要求提供保证。

　　第二个需要解决的问题是必须进一步呼吁明晰团的法律地位。在我们现在经济结构大调整、大转换的过程中，我们国家正在经历一种由人制到法制的转换，在这个过程中群团组织如果没有法律的保障，要想有序地开展工作，确实比较困难了。过去我们基本是人治的结构，哪个领导重视，哪个方面的工作就可以推进。在今后法制逐步健全的情况下，团的组织如果没有法律的保障，没有明晰的法律地位，我们共青团的工作必然会遇到许多新的矛盾和问题。比如说现在企业改制的过程中，把群团组织作为企业内部的行政机构砍掉了；比如说教育体制改革的过程中，学校的团组织就和学校的行政部(处)合并了；往往是说机构改革首要裁减的就是这些务虚的部门，这样的思想对我们团的组织最大的冲击就是因为共青团的组织还没有一个明晰的法律地位。特别是在企业改制的过程中，在企业构建法人治理结构的过程中，共青团组织还没有明确地在企业的三个权力机构中占有它应有的地位，这也直接影响到我们共青团组织今后的发展。令我们欣喜的是在这次政府机构改革的过程中，有意识地给共青团组织留了一席之地，比如，政府的部门机构，就我们全省来说裁减48%，这是力度最大的，党委部门机构改革的方案已经出台，裁减20%。群团、人大、政协作为又一个系列专题研究，这本身已经给了我们共青团一个应有的法律地位。但是在基层操作中，它仍然是过去的党政工团这一个概念，这就带来一定的问题，所以，明晰共青团的法律地位是十分重要的。这已经引起了团中央及各级团组织高度的重视。

　　第三个需要解决的突出问题应该是拓宽我们共青团工作的管理领域。具体地讲就是总结我们这几年共青团工作的成功经验，非常重要的一条就是政府把过去不该管、管不了的事务特别是涉及青年工作的事务，依靠政府的规定、依靠政府的授权明确地转换到我们的青年团组织上来，从而进一步增强了我们其青团组织对青年工作的针对性，真正发挥像胡锦涛同志所讲的共青团在管理国家事务中的作用。实践证明这是扩大共青团的影响力、凝聚力最有效的最便捷的一种办法，也是新的形势下共青团雯想真正发挥“三个代表”的作用必须解决的一个地位问题。

　　三、发挥三个作用，争做实践“三个代表”重要思想的排头兵

　　l.代表中国社会先进生产力均发展要求，就要发挥团组织在经齐建设和科教兴国战略中的生力军作用。生产力是使社会文明进步的重要的推进力，也是一个重要的标志。人作为生产力中最活跃的因素，青年作为最有思想、最有创新意识主样一个特定的群体，一旦和科学交术这一第一生产力结合起来，它对社会生产力的推动作用将是巨大的。人的发明创造最佳的年龄都是主青年时期。从古至今，大的科学家、思想家包括军事家，许多是在青年时期就有所作为的，所以我们共青团组织，代表中国社会生产力发展的要求，最根本的就是在推进经济建设，推进科教兴国战略中把对青年人才的培养，营造青年人才发挥作用这样一个氛围作为我们推进工作的一个重要的着力点，确实发挥好我们党的助于、后备军作用，发挥好青年这样一个特定的群体在推进生产力，在推进科教兴国战略中的生力军作用。

　　2.代表中国社会先进文化的发展方向，就要发挥我们团组织在社会主义精神文明创建中的排头兵作用。青年具有开风气之先的“闯”劲，作为我们团组织就应该保护青年开风气之先这样的精神，按照我们有中国特色的社会主义文化的本质要求，即十五大报告中指出的“四有”公民和“三个面向”(面向现代化、世界、未来)，这样的一个民主的、科学的、大众的文化的发展方向来定位我们共青团在文化方向发展中的排头兵作用。团组织树立的青年的楷模首先应该成为我们青年的偶像，团组织弘扬和倡导的文化，应该成为青年追求的热点。我们不能说我们要求的是青年不愿意要的，青年愿意要的又是我们不拥护、不赞成的，如此看来，在方向问题上，共青团将应有更大的作为。

　　3.代表中国最广大人民的根本利益就要重视发挥我们团组织在广泛地团结联系青年中的桥梁和纽带作用。如果我们按国家的人口来讲，青年的覆盖面在1/3左右，那么这样一个大的青年群体，我们团的组织发挥它的作用对我们作为党的助手来说将是一个非常重要的一个纽带作用，非常重要的一个桥梁作用。把这一批青年团结好，联系好，我们国家的政治稳定、社会稳定、经济发展就有了一支很强的生力军，把这一批青年的积极性调动起来，中国社会文化发展的方向、中国先进生产力的代表，我们的青年也就在其中。

　　总之，“三个代表”是我们新时期党的建设的重要指针，是我们共青团建设的行动纲领心只要我们深刻地理解、准确地把握，我相信我们党的建设，我们团的建设，必然会得到长足的进步和发展。

　

（《共青团山西通讯》2000年第8期）
