News and text scroll show 滚动新闻文字发布

如下是其中一个例子的代码，全部范例和必须的ProScroll可以下载得到。

<applet code="ProScroll"width="400"height="16">
<param name="TEXT"value="这是一个演示，可用于页面的新闻滚动发表，#green#占地面积小，#yellow#适用于小量信息的展示。">
<param name="STYLE"value="plain">
<param name="SIZE"value="12">
<param name="SPEED"value="fast">
</applet>

TEXT

颜色可选，#red# #green# #blue# #yellow# #orange# #white# #lightGray# #gray# #darkGray# #black# #cyan# #magenta# #pink# ，想要字体显示颜色，在文字前加上#颜色#即可。

插入一个图片，只要加入一个$字符即可，还需要为IMAGES参数设置图片文件。
<PARAM NAME=TEXT VALUE="$$">
<PARAM NAME=IMAGES VALUE="image1.gif image2.gif">

example：
<PARAM NAME=TEXT VALUE="$ This is a demo $">
<PARAM NAME=IMAGES VALUE="image1.gif image2.gif">
先显示image1.gif，再显示文字内容This is a demo，再显示image2.gif。

URL <PARAM NAME=TEXTURL VALUE="http://www.anysite.com/page.htm"> 点击的连接
FONT 字体，默认TimesRoman。
SIZE 字体号，默认12 points
STYLE 字体风格，可选plain, bold, italic 和 bolditalic。默认plain(普通)。
TARGET 页面打开的方式，可选_top,_blank,_self或已经定义的frame的名字，默认_self
SPEED 滚动速度，可选slow, medium, 和fast。
IMAGES 见 TEXT参数。
BGCOLOR 滚动条背景色，不需要#，或者直接用red,white等直接色。
