耳目一新的Windows ME

    微软已经在10月14日悄悄发布了Windows Me，这是一个非常适合家庭的操作系统，下面我就一一介绍Windows Me。Windows Me的安装非常简单，几乎就是Win98安装程序的翻版，这也许就是有些人把它称作Win98第三版的原因吧！

一、支持的硬件

    我在Win98中只安装显卡、声卡、Modem卡这三种硬件驱动。刚安装上Windows Me的时候，WinME和Win98一样，一副灰面孔，没想到WinME的显卡支持性这么差，连我的Savage3D都没有认出来，不过装了Savage3D For Win98的驱动后就一切没事了。Windows Me的声卡支持性还可以，因为我第一次启动听到了音箱里发出了启动声音，然后我查了一下设 备管理器，发现它默认装的是WDM版的驱动（通用各种Windows的驱动）（图1），好啊！ 我播放了几首MIDI，效果比Win98好多了！原来装Modem驱动还要装两次，到了Windows Me， 装了一次就装好了！

二、崭新的界面

    Windows Me有和Office2000中一样的智能菜单，将不常用的程序或收藏夹项目隐藏起来（如图2）。它的桌面图标制作得也很有立体感。打开帮助，崭新的“帮助与支持” 浮现在你眼前（如图3）。如果你的系统非正常关机，他不像Windows98那样，到英文DOS版的ScanDisk里检测磁盘，而是自动在进入图形界面后打开磁盘扫描程序，不仅界面好， 而且可以自动修复错误。

三、更好的电脑保护

  1.系统还原

    大家可能都有这样的经历：把重要的文件修改了或修改了注册表，但是为刚才的操作特别后悔，甚至导致系统无法使用。以前那就只有改回去或重装系统了，而到了Windows Me， 新增加了一个系统还原的功能，也就是说有了后悔的机会了！进入系统工具中的还原，选择第一项，你就可以看到如图4的画面：选择还原点。选择粗体的还原日期和还原的时间， 就可以按下一步进行系统还原了！

  2.Windows Me的DOS。

    大家可能会听到这样的传言：Windows Me不再提供DOS，所有DOS程序全部无法使用。其实这样的传言是极不正确的。Windows Me不但支持DOS命令，而且能运行大部分DOS软件。只是WinME把DOS更隐蔽了一些，运行SYS A:会得到“You can only SYS drive C: to try  and repair the boot hard disk. Use the Startup Disk option in Add/Remove programs to create an emergency boot disk.”的提示，而且运行命令Format A:/S也会得到 “Microsoft Windows no longer supports the format /s command. To create a  Startup Disk, click the Add/Remove Programs icon in Control Panel.”的提示，但 Windows Me不是不提供制作启动盘的功能，其实在Windows ME的安装过程中，你会得到一 张WinME启动盘，可以用纯DOS方式启动电脑。

四、新的媒体播放器

    在Windows Me中，提供了一个与以前完全不同的媒体播放器，这就是Windows Media  Player 7.0（图5）。它不仅支持Wav/Midi/Avi/CD/VCD/Mpg等常用的格式，它还支持MPEG4格式（新的影片压缩格式，能在一张普通的CD-ROM盘片中，存放与一张DVD相同的影片长短）。 它还提供“换肤”的功能，可以从Internet上下载几十种不同的播放界面，好漂亮啊！

五、网络和Internet的改进

    如果你在家里有多台电脑，你可以使用WinME中的“家庭网络向导”方便的建立家庭网络。新的IE 5.5除了首次提供打印预览外似乎并没有给用户带来惊喜，而且我也没有感觉到它提高了网络速度和安全性能。不过它自带的MSN Messenger的确是一个不错的“网络寻呼机”（图6），可以在因特网中“寻呼”其他人，但是显然没有ICQ的人数多。

六、我的使用体会

    Windows Me仍然是一个Win9X核心的操作系统，稳定性和某些方面无法和Windows 2000比较，但是它非常适合于普通家庭，它的兼容性也是非常的高，我安装的所有For Win98的软件均正常运行。但它也有不好的地方，如它稳定性不好，我在使用过程中虽然没有崩溃，但是在半个月的使用过程中，有过4次死机经历，而且它的微软拼音输入法2.0中，没有了手写输入，也不能用一般的方法重新安装。希望以后的Windows版本能够有Windows2000的稳定性、 新的手写功能和WindowsME的易用性。
