V012 - 结构函数（Framework）

FindText

string := FindText("searchString", scopeIconID, textOrKeywords, matchPattern, resultsInContext,

convertResultsToPageIDs, searchInBackground)

表示搜寻某个字串　在 searchString

中填入所要搜寻的字串　而搜寻的地方则在後面的引数中表示　

GetTextContaining

string := GetTextContaining(n [, m, maxlen])

查询某个特定的字串　这个字串为搜寻动作後指定的那一个字　如　

string := GetTextContaining(5)

或者是二个指定位置之间的字串　这二个位置分别是所搜寻符合的字　如　

string := GetTextContaining(5, 10)

Keywords

string := Keywords(IconID@"IconTitle")

查询某个特定图示的关键字　如果关键字多於一个以上　则关键字之间会以

Enter 键区隔它　所以一行之中的字串表示一个关键字　

PageContaining

ID:=PageContaining(IconID@"IconTitle"[, @"framework"])

查询含有此图示的结构图示编号　也就是查询含有此页的结构图示编号　

PageFoundID

ID := PageFoundID(n)

当使用到找寻 Match

的功能时　本函数可以查询特定的图示编号　而这个图示是符合找寻的结果　如　PageFoundID(1)

表示第一个符合找寻结果的图示编号　同理 PageFoundID(2)

表示第二个符合找寻结果的图示编号　

PageFoundTitle

title := PageFoundTitle(n [,m])

当使用到找寻 Match

的功能时　本函数可以查询特定的图示名称　而这个图示是符合找寻的结果　如果同时使用了二个引数

n 与 m　则结果是这二个图示间所有的图示名称　

PageHistoryID

ID := PageHistoryID(n [, m])

查询最近所看过的页图示编号　例如　PageHistoryID(1)

表示最近这一次看过的页图示编号　PageHistoryID(2)

表示上上一次看过的页图示编号　

PageHistoryTitle

title := PageHistoryTitle(n [, m])

查询最近所看过的页图示名称　此函数的用法与上个指令相同　

PurgePageHistory

PurgePageHistory()

将看过的图示的记忆清除　如此程式就不记得使用者看过那些页了
