V017 - 数学函数（Math）

ABS

number := ABS(x).

传回数字 X 的绝对值　

ACOS

number := ACOS(x).

传回 x 余弦径度值　数值范围从 0 到 π　

ArrayGet

result := ArrayGet(n)

将阵列中的第 n 个元素的值指定给 result　元素值可以为数字也可以为字串　

ArraySet

ArraySet(n, value)

将 value 设定为阵列第 n 个元素的值　value 可为数字　也可为字串　

ASIN

number := ASIN(x)

传回 x 正弦径度值　数值范围从 -π/2 到 π/2　

ATAN

number := ATAN(x)

传回 x 正切径度值　数值范围从 -π/2 到 π/2　

COS

number := COS(angle)

传回某个”角度”的正弦函数值　

EXP　EXP10

number := EXP(x)

number := EXP10(x)

Exp 函数传回指数为 x 而底数为 e 的值　Exp10 函数传回指数为 x 而底数为 10 的值　

INT

number := INT(x)

将 x 去掉小数後传回其整数值　

LN

number := LN(x)

传回 x 的自然对数值　

LOG10

number := LOG10(x)

传回 x 对数值　其底数为 10　

MOD

number := MOD(x, y)

传回 x/y 後的余数　

Random

number := Random(min, max, units)

在 min 与 max 两个数值间　产生乱数　其单位为 units 所指定的位数　

Round

number := Round(x, decimals)

以 decimals 来指定小数位数　然後将值传回　

Sign

number := Sign(x)

判断数值 x 为正值　负值　或是 0　如果 x 为负值　则传回 -1　如果 x 为 0　则传回 0　如果 x 为正值　则传回 1　

SIN

number := SIN(角度)

传回”角度”正弦函数值　

SQRT

number := SQRT(x)

传回 x 平方根值　

TAN

number := TAN(angle)

传回”角度”正切函数值
