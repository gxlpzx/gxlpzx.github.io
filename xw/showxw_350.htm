V013 - 一般函数（General）

Beep

Beep()

产生哔哔的声音　

FlushKeys

FlushKeys()

清除在系统尚未处理之前所有使用者所按的键　

Initialize

Initialize()

Initialize(variable1,..., variable10)

将指定的所有变数回覆到其原来的初值　最多只可以使用 10 个变数　

MediaPause

MediaPause(IconID@"IconTitle", pause)

此函数应用在影片或是视讯的播映上　若 pause 的值为真时　表示影片会暂停播映　若为假时　则影片会继续播映的动作　

MediaPlay

MediaPlay(IconID@"IconTitle")

此函数应用在影片或是视讯的播映上　若表示开始播映之意　请在 IconTitle 中填入此影片的图示名称　

MediaSeek

MediaSeek(IconID@"IconTitle", position)

搜寻影片或是视讯图示中特定镜头的位置　请在 position 中填入所要寻找的镜头位置　

MoveWindow

MoveWindow(top, left)

将展示视窗移到”top　 left”的座标上　

PressKey

PressKey("keyname")

本函数执行後就好像使用者按了 keyname 键一样　

PrintScreen

PrintScreen()

列印画面　

Quit

Quit(options)

依照 options 来跳离执行的动作　选项有下列三种　

数字 结果

0 跳到程式管理员之下

如果原来执行分歧到其它的档案

则会回到原来的档案之中

1 跳到程式管理员之下

2 跳到 DOS 之下

QuitRestart

QuitRestart(options)

QuitRestart 与 Quit 函数唯一不同的地方就是　它会忽略 File Setup 对话方块中 Resume 的设定从程式的一开始执行　使用者无法回到上一次所离开的状态　引数的部份与 Quit 函数完全相同　

ResizeWindow

ResizeWindow(Width, Height)

将展示视窗以 width 及 height 两个引数来改变其大小　

Restart

Restart()

立即重新执行本档案　应用在输入错误的密码时　非常有用　

SaveRecords

SaveRecords()

将使用者的资料记录到指定的目录中　这些资料为使用者目前的状态　而记录的目录系统内定为 WINDOWS\A3W_DATA　

SetCursor

SetCursor(type)

设定滑鼠游标的类别　类别有下几种　

数字  结果

0 箭头

1 |

2 交叉号

3 十字号

4 空白

5 时间漏斗

6 手形符号 

ShowCursor

ShowCursor(display)

显示或是隐藏游标　以 display 引数作判断　设定引数为 off 表示隐藏游标　设定引数为 on 表示显示游标　

ShowMenuBar

ShowMenuBar(display)

显示或是隐藏功能表列　以 display 引数作判断　设定引数为 off 表示隐藏功能表列　设定引数为 on 表示显示标题列　

ShowTitleBar

ShowTitleBar(display)

显示或是隐藏标题列　以 display 引数作判断　设定引数为 off 表示隐藏标题列　设定引数为 on 表示显示标题列　

SyncPoint

SyncPoint(option)

将计时器重新设回初始的值　以 option 作判断　这个计时器的功用与 SyncWait 互相配合　选项有下列三种　

数字 结果

0 在展示现在图示之前

1 在展示现在图示之後

2 符合一个交谈回应之时

SyncWait

SyncWait(seconds)

本函数功能会在上一次函数 SyncPoint 执行完成後　暂停展示档案　直到指定的时间 seconds 到了之後　才会继续展示档案　在暂停期间　所有的动画　影片　以及其它动作会继续展示不受影响　

Test

Test(condition, true expression, false expression)

这个指令与 if...then 有异曲同工之妙　在 condition 中为一个判断式　若此判断式为真　则会执行 true expression 之中的表示式或指令　否则执行 true expression 之中的表示式或指令　

TextCopy

TextCopy()

将选定的文字拷贝到剪贴簿上　

TextCut

TextCut()

将选定的文字剪下来　贴到剪贴簿上　

TextPaste

TextPaste()

将剪贴簿上的文字　贴到指定的区域上　

Trace

Trace("string")

这个函数可与 Trace 视窗配合使用　主要在追踪程式用　每当程式执行到此时　此函数就会将 string 这个字串丢到 Trace 视窗中　表示程式已经执行到此地了

WaitMouseUp

WaitMouseUp()

本函数会暂停展示的动作　直到使用者放开滑鼠的按键　如果碰到本函数而使用者未按任何滑鼠的按键时　本函数的无法作用　

ZoomRect

ZoomRect(x, y)

将萤幕座标 (x, y) 的方块区域　放大显示
