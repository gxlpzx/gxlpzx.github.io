V014 - 图形函数（Graphics）

Box

Box(pensize, x1, y1, x2, y2)

在萤幕上　从座标 (x1,y1) 到 (x2,y2) 的方块区域　画上一个矩形　矩形外框区域的线段格式以引数 line width 作判断　

当引数 pensize 小於 0 时　矩形会以全黑的方式绘制　引数等於 0 时　矩形的外框的粗细为一个点图素　引数大於 0 时　外框的粗细依照引数的大小绘制　数字代表点图素的大小　这个 pensize 引数可以用应到 Graphics 上的任何一个函数上　

Circle

Circle(pensize, x1, y1, x2, y2)

在萤幕上　从座标 (x1,y1) 到 (x2,y2) 的方块区域中　画上一个圆形　圆形外框的线段格式以引数 pensize 作判断　

DrawBox

DrawBox(pensize)

DrawBox(pensize, x1, y1, x2, y2)

允许使用者在萤幕上绘制一个矩形　矩形外框的线段格式以引数 pensize 作判断

DrawCircle

DrawCircle(pensize [,x1,y1,x2,y2])

允许使用者在萤幕上绘制一个圆形　圆形外框的线段格式以引数 pensize 作判断

DrawLine

DrawLine(pensize [,x1,y1,x2,y2])

允许使用者在萤幕上绘制一条线段　线段格式以引数 pensize 作判断　

Line

Line(pensize, x1, y1, x2, y2)

在萤幕上　从座标 (x1,y1) 到 (x2,y2) 的绘制一条线段　线段格式以引数 pensize 作判断　

Overlapping

condition := Overlapping(IconID@"Icon Title1", IconID@"Icon Title2")

当 Icon Title1 中的物件与 Icon Title2 中的物件互相覆盖时　传回逻辑值真　否则传回逻辑值假　

RGB

RGB(R, G, B)

这是颜色的指令　R 代表红色　G 代表绿色　B 代表蓝色　由三个颜色组合而成的结果正是此函数的结果　而各个颜色有 256 种　由数字替代　从 0 到 255 皆可　

SetFill

SetFill(flag [, color])

设定图形填满的颜色　若 flag 的值为真　则表示此图形将要填满　而 color 代表颜色　颜色的指定可以由 RGB 函数来设定　

SetFrame

SetFrame(flag [, color])

设定图形的边框　若 flag 的值为真　则表示此图形将有边框　而 color 代表此边框的颜色　颜色的指定可以由 RGB 函数来设定　

SetLine

SetLine(type)

设定图形的线条样式　在 type 中可以填入四种数字　0 表示一边的线段　1 表示箭头在结束边　2 表示箭头在开始边　 3 表示二边都有箭头　

SetMode

SetMode(mode)

设定图形的样式　mode 代表此样式　0 表示周边透明(Matted)　1 表示全部透明(Transparent)　2 表示反白显示(Inverse)　3 表示将它清除的样式(Erase)　4 表示不透明(Opaque)
