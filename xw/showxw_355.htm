V018 - 时间函数（Time）

Date

string := Date(number)

传回数字日期　以系统所设定的日期格式为主　

DateToNum

number := DateToNum(day, month, year)

将 day, month, year 输入资料改变为数字格式的日期　然後传回　

Day

number := Day(number)

传回数字在每当月中的日期　

DayName

string := DayName(number)

传回数字当日的日期名称　这是一组字串　

FullDate

string := FullDate(number)

传回数字之完整日期名称　以系统所设定的格式为主　

Month

number := Month(number)

传回月份数值　

MonthName

string := MonthNmae(number)

传回月份的名称　这是一个字串　

Year

number := Year(number)

传回年份的数值
