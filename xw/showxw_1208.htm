美英军事打击阿富汗行动大事记（截至10月16日）

10月17日 17:59 

    美英从10月7日开始联合对阿富汗实施大规模军事打击。下面是截至10月16日的军事打击阿富汗行动大事记，均为阿富汗当地时间。

    10月7日

    1. 在阿富汗首都喀布尔和东南部城市坎大哈听到了几声爆炸。

    2. 美国总统布什宣布美英联合军事打击阿富汗行动开始。

    3. 9·11恐怖袭击的头号嫌疑犯奥萨马·本·拉登公然称赞发生在纽约、华盛顿和宾西法尼亚州的袭击，并称美国“从此将再也不知安全为何物”。

    4. 伊拉克总统萨达姆·侯塞因和伊朗方面发表声明谴责美英空袭阿富汗。

    10月8日

    1. 几个穆斯林国家爆发反美暴力示威。

    2. 美国告诉联合国安理会说，它将对其它国家和组织采取“进一步行动”。安理会一致投票通过决议，支持采取反恐怖主义军事行动。

    3. 美国对阿富汗第二天空袭开始，英国在空袭中向美军提供后勤支持。

    10月9日

    1. 美国宣布首次在白天对阿富汗进行轰炸。

    2. 4名为联合国工作的阿富汗人在喀布尔遭受空袭时被炸死。

    3. 一些穆斯林国家爆发反对美英军事打击阿富汗的示威活动。

    4. 根据伊朗官方报道，10月7日美英开始军事打击阿富汗以来，已有超过2万名阿富汗难民涌入伊朗境内。

    5. 《巴黎竞赛画报》的法国记者米歇尔·佩哈尔在塔利班被捕。

    6. 美国宣布开始进行新一轮的夜间轰炸。

    7. 美国声称已经取得阿富汗的制空权，并开始对阿富汗进行全天候的轰炸。

    10月10日

    1. 新一轮空袭开始。

    2. 塔利班号召居住在美国的穆斯林举行示威活动，反对美国的“暴行”。

    3. 《巴黎竞赛画报》的法国记者米歇尔 ·佩哈尔和两名陪同他进入阿富汗进行采访的巴基斯坦记者被塔利班指控犯有间谍罪。

    4. 伊拉克首都巴格达、伊朗首都德黑兰和也门首都萨纳都爆发了新一轮反对美英空袭阿富汗的抗议示威活动。

    5. 卡塔尔半岛电视台不顾美国指责，拒绝改变自己对阿富汗的新闻报道。

    10月11日

    1. 印尼和伊朗爆发反美示威游行。

    2. 塔利班指控美国袭击平民，不过美国政府否认了塔利班的指控。塔利班声称阿富汗东部一个村庄中已有超过200人在凌晨的空袭中被炸死。塔利班还说，空袭行动开始后，阿富汗总共有超过300人丧生。

    3. 巴基斯坦同意美军使用其境内的两个机场执行救援任务。

    4. 美英发动新一轮空袭。

    5. 美国总统布什称，如果塔利班交出奥萨马·本·拉登，他愿意“重新考虑”军事打击阿富汗一事。

    10月12日

    1. 联合国难民事务高级专员玛莉·罗宾逊呼吁美英暂停轰炸，以便空投救援物资，使阿富汗平民能在冬季到来之前得到生活必需品。

    2. 一些亚洲国家的数千名穆斯林举行了暴力示威活动。在这一天举行示威活动是因为12日是空袭开始后的第一个星期五，也是穆斯林的礼拜日。

    3. 阿富汗的穆斯林教士们发布宗教命令，要求他们的跟随者发动圣战，反击美国。

    4. 美国国防部长拉姆斯菲尔德表示，目前已经到了反塔力量向前推进，反攻塔利班的时候了。

    5. 美国称它没有在穆斯林的礼拜日这一天实施“预先计划好的作战行动”。

    10月13日

    1. 美英对阿富汗实施新一轮打击。

    2. 塔利班在喀布尔的高等法院对扣押的“国际即时保护组织”8名成员进行最后阶段的审判。

    3. 欧洲爆发大规模抗议美国军事打击阿富汗的示威活动，共有数千人参加。

    10月14日

    1. 阿富汗遭受新一轮轰炸。

    2. 美国总统布什拒绝就引渡奥萨马·本·拉登一事，同塔利班当局进行任何谈判。

    3. 美机首次向阿富汗民众散发宣传材料。

    10月15日

    1. 美英对阿富汗部分城市发动新一轮袭击。

    2. 巴基斯坦总统穆沙拉夫在与美国国务卿鲍威尔举行会谈时表示，巴基斯坦十分关注阿富汗平民的伤亡情况。

    3. 美国加大对阿富汗的打击力度，与此同时反塔力量声称他们在地面进攻中取得了进展。

    4. 联合国儿童基金会宣布，如果在今后6周内无法得到充足食物的话，今年冬天将有10万名阿富汗儿童面临死亡的威胁。

    5. 美国表示它将把空投给阿富汗平民的人道主义救援食品数量增加一倍。

    6. 美国国防部长拉姆斯菲尔德表示，塔利班所说的阿富汗一个村庄中有超过200名无辜平民被炸死一事，十分“荒谬可笑”。拉姆斯菲尔德形容塔利班领导人是“一些技术熟练的撒谎者”。

    7. 联合国秘书长安南敦促阿富汗军事冲突中的有关各方采取一切可能办法，控制平民的伤亡。

    10月16日

    1. 塔利班说美国在对东南部城市坎大哈实施夜间攻击时，首次使用了武装直升机。

    （许勤）
