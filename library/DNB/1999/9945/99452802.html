《 网上图片搜索器iFind Graphics 》

　　对于上网的朋友来说，从网上下载图片是常有的事，比如下载影星图片，下载一些制作个人主页的图像素材等。但由于一个站点中的图片往往分布在不同页面上，需要费时间浏览寻找，而下载图片又需要在线进行保存等工作，十分不便。本文向大家推荐的网上图片搜索器iFind Graphics是一个十分优秀的网上图片搜索器，它不仅能搜索下载保存网上图片，还能作为浏览、管理图片的工具，可谓是“一条龙”服务。
　　iFind Graphics 的30天试用版可以在 http：//www.skibbysoft.com下载。该软件稍微大了一些(5M)，但考虑到它提供的功能，还是值得的。安装运行iFind Graphics后，在操作界面的左侧为类似资源管理器的树状结构图，它的上方有两个标签。如果你选择了Local标签的话，那么此时iFind Graphics 的功能变为了图像浏览器，选定目录下所包含的图像会出现在右侧的显示框中。而如果你选择Internet这个标签，iFind Graphics 便是一个名副其实的网上图片搜索器了。此时你会看到三个文件夹，其内容与使用方法如下：
　　1．Image Search：文件夹下包括了四个著名的搜索引擎，你需要先选择搜索引擎，然后输入要查询的关键词，比如某个歌星的名字，iFind Graphics 便会为你搜索该歌星的图片了。
　　2．Pre－Set Web Page Scan：在这里，软件的制作者专门为那些个人主页制作者搜集了相关图像站点，如背景，按钮等，你可以很方便地获得需要的素材。
　　3．Input Web Page Scan：这里可以搜索特定站点中的图像。注意输入的不再是关键词，而是站点的网址。
　　无论通过上面所述的哪种搜索方式，iFind Graphics 都会将搜索到的图像以拇指图的形式显示在右面的图像显示区中。如果选择上方的Images标签，iFind Graphics 会显示出这些图片的地址。如果这些图片还包含其它连接的话，通过选择Link标签，你可以看到这些连接。如果你想看到某个图像的真实情况，选择该拇指图后，选择上方的Viewer图像工具钮即可。选择Slideshow图像工具钮，则会以幻灯的形式来显示所有图片。搜索到的图片，你可以单独选择后进行保存，如果你想保存所有搜索到的图片，可以选择File菜单下的Save all Galley Images命令来进行。
　　通过笔者的使用，感觉iFind Graphics 是非常实用的网上图像搜索软件，有兴趣的朋友不妨一试。
(天津　邵桓)
