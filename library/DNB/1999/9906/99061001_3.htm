一步一步学南极星全球通

文件内码的转换

　　南极星除可以在屏幕上显示不同内码的汉字之外，还具有对剪贴板中的文本或现有文件中的汉字内码进行转换，如将BIG5码的文本转换为GB码的文本，然后就可在简体中文系统中使用GB码文本。
　　1.启动内码转换工具
　　启动南极星，点击工具条界面的第一个“南”字图标按钮，选择菜单“Universal Code Convertor”，弹出内码转换工具界面窗口。也可以在南极星的安装目录运行“Njconv.exe”程序，启动该转换工具。
　　2.文件内码转换工具界面（图10）
　　程序弹出南极星内码转换界面窗口（“Njstar Universal Code Convertor”），如果剪贴板中有字符串，则将在窗口上面显示该字符串。
　　3.剪贴板文字内码转换
　　如欲将剪贴板中的字符串进行内码转换，则在左面“Input Code”列表框中选择转换前的文字串的内码，在“Output Code”列表框中选择转换后的目标文字串的内码。在“Options”列表框中选择适当的转换操作选项，单击“Convert Clipboard”按钮，剪贴板中的字符串内码即可按照要求进行转换，然后就可以用新的内码文字进行剪贴了。
　　4.文件内码转换
　　如果要进行文本文件的内码字符转换，可以选择界面上部的“Text in File(s)”单选框，程序即弹出文件选择窗口，选择好文件后，再回到转换界面单击“Convert File(s)”按钮进行转换。
　　5.其它转换设置
　　南极星的内码转换工具还提供了几个附加功能，在界面的“Options”列表栏中，可以设置文字增加空格（Add Spaces）、删除空格（Remove Spaces）、文字提行（Wrap Text at）、去除HTML标志（Strip HTML Code）、修复文件（Fix Corrupted Text）等。

文字输入

　　南极星2.0新增了汉字和日文输入功能，可以利用它输入各种不同内码的汉字和日文，在一个系统中实现多种内码文字的编辑处理。
　　1.启动汉字输入法（图11）
　　单击南极星快捷工具条上的汉字输入法选择按钮，并从弹出的快捷菜单中选择“Chinese Input”命令，系统即会在屏幕底部显示一个汉字输入法提示条,可以拖动成图12所示的浮动提示条。
　　2.输入法提示条界面（图12）
　　南极星的汉字输入法提示条有英文、简体中文、繁体中文等三种不同的显示界面，只需单击输入法提示条上的系统图标，然后再从弹出的菜单中选择所需的提示界面即可。
　　3.选择汉字输入法
　　南极星一共提供了拼音、内码、注音、简繁仓颉、粤语、日语、四角号码、偏旁＋拼音等近20种不同的汉字输入法，可以直接通过输入法切换图标按钮选择拼音、注音、内码等常用汉字输入法，用户若需使用四角号码等不常用的汉字输入法，则可单击输入法切换图标并从弹出的菜单选择“其它方法”项，然后再选择合适的输入法即可。
　　4.选择输入汉字的内码(图13)
　　南极星具有多内码转换输入功能，也就是说无论系统当前采用的是何种汉字内码及何种输入法，均可将输入的汉字转换为指定的内码字符储存。利用此功能，就可以实现通过拼音等汉字输入法直接输入BIG5、HZ、UTF7、UTF8等内码汉字。要选择输入字符内码，单击输入法提示条上的汉字内码选择图标，然后再从弹出的菜单中选择所需的内码即可。

网络信息服务

　　南极星除提供了强大的多内码支持功能之外，它还提供了优秀的网上信息服务功能，我们可直接利用它查找网上信息资料、人名、最佳应用软件、商业合作伙伴、各种传媒机构以及各种免费电子信箱、免费主页、广告交换网和分类广告等信息。只需单击南极星快捷工具栏上的网上信息服务按钮，然后从弹出的菜单中选择所需的菜单项，南极星将会自动启动缺省浏览器并打开相关网址，为用户提供完善的网上信息服务。
