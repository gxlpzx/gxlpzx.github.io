“克隆”目录

　　我们可能会因某些特殊应用而需要在磁盘上创建一系列诸如DISK1、DISK2、DISK3之类的序列目录（即目录名的前几个字符完全一样、而后几个字符分别为逐一递增的数字），若完全手工操作则过于麻烦，而利用Sequential Folder Creator就可轻易地解决这一问题。Sequential Folder Creator是一个专门用于创建多个连续数字序列目录的软件，它可帮助用户快速地“克隆”出诸如DISK1、DISK2、DISK3之类的序列目录，从而极大地方便了用户的使用。
　　Sequential Folder Creator的安装方法有点与众不同，首先将自解包SFC10.exe解压到“C:\Program Files\SFC”目录中，然后进入该目录，双击SFC.REG文件，将Sequential Folder Creator的注册信息输入注册表数据库中。安装Sequential Folder Creator之后，Win9X的鼠标右键菜单上就会多出一个“Sequential Folder Create”命令。用户若拟在某个目录下创建多个数字序列目录，则可右击该目录，执行快捷菜单上的“Sequential Folder Create”命令，启动Sequential Folder Creator程序，用户只需在程序界面上的“FROM”对话框中输入需要创建数字序列目录的前导字符（如“DISK”）及起始数字的序号（如“1”），然后在“TO”对话框中输入数字序列目录的结束序号（如“5”），最后单击“Create”按钮，Sequential Folder Creator就会在相应目录下创建出DISK1、DISK2……DISK5等5个数字序列目录。Sequential Folder Creator是一个免费软件，有兴趣的读者可向其作者发一个E－mail（sfc@mboxall.d.mon.co.uk）索取该软件。(湖北　胡锦承)
