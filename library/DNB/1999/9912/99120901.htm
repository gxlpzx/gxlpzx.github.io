IE5的问与答

　　Internet Explorer 5.0主要有什么改进?
　　答：IE5.0速度快、体积小，更加智能化、更加灵活、容易管理和定制。新版本中所带的Windows Media Player可以让用户方便地浏览网上的各种声、像资料。其新增组件Windows Media Player是一个多媒体播放器，可以在Windows平台上播放CD、VCD、MP3等格式的多媒体资料。
　　Internet Explorer 5.0主要采用了什么技术?
　　答：主要使用了IntelliSense技术。这种技术最早在Microsoft Office中引入，它通过简化复杂的任务，可以节省时间。
　　Internet Explorer 5.0包含哪些组件?
　　答：包括浏览器、电子邮件Outlook Express、Windows Media Player、聊天工具MS Chat、Web编辑器FrontPage Express、网上会议NetMeeting等。
　　运行Internet Explorer 5.0的最低系统需求是什么?
　　答：486/66以上，如果是在Windows95/98，需要16MB的内存，如果在Windows NT Workstation＆Server 4.0上，需要32MB内存以上和Service Pack 3。
　　Internet Explorer 5.0共有多少语言版本?
　　答：1999年3月18日，已同时发布10个语言版本；随后的2个月中，还有16个语言版本。
　　Internet Explorer 5.0是否可以阅读繁体中文、日文、俄文等语言的主页?
　　答：可以直接阅读。只须要在工具栏中点击“视图”，选择“编码”，然后选择不同的语言即可。几乎全世界的所有网页，都可以直接用Internet Explorer 5.0直接阅读，而不需要借助其他第三方的工具。
　　Internet Explorer 5.0可以支持什么平台?
　　答：可以支持Windows3.X、Windows95/98、Windows NT Workstation,Windows NT Server、HP－UX、Sun Solaris。
　　Internet Explorer 5.0支持什么标准?
　　答：Internet Explorer 5.0支持所有主流的Internet标准，如HTML40、CSS1.0和2.0、ECMA脚本、Document Object Model，以及XML、XLS等。
