康柏的精装服务——CarePaq

　　1999年，康柏电脑公司为广大用户推出“康柏关怀服务包(CarePaq)”。此举的目的旨在以客户为中心，以客户满意度第一为服务宗旨，为客户提供尽善尽美的服务。关怀服务包包括一系列精装服务和支持方案，协助企业发挥其信息科技投资的最大效能。它的推出，是为了“在您最需要的时候，为您排忧解难”。
　　关怀服务包首先在北京、上海、广州、成都、沈阳五大城市推出。用户可以随机得到或从最近的康柏授权代理商那里得到关怀服务包登记卡。
　　康柏关怀服务包内容
　　关怀服务包设有超过十五项的服务组合，内容涵盖Windows NT和Backoffice产品的安装和启动服务、Internet/Intranet的安装和启动服务、康柏硬件的扩展保修服务以及软件和应用系统的支持服务。关怀服务包的具体内容如下：
　　1.关怀服务包安装及启动服务
　　此服务设有七项组合，涵盖Microsoft Windows NT Server、Windows NT Cluster或Microsoft Exchange环境的安装和启动。这些服务组合与康柏的Microsoft授权支持服务相辅相成。关怀服务包同时设有保密Internet/Intranet环境的安装和启动服务。
　　一般的关怀服务包设有基本服务，提供基础的安装和启动项目。当基本服务不能满足客户复杂的配置需求时，便可另外选择合适的选件式服务，协助完成所需的安装程序。
　　2.关怀服务包扩展保修服务
　　康柏客户若要求比标准硬件保修范围更多的服务时，可选择关怀服务包的扩展保修服务，以便在有需要时享用专业的技术支持。扩展保修服务涵盖康柏服务器、工作站、台式机及笔记本电脑，以及在基本系统机箱中安装康柏选件。
　　关怀服务包设有六项扩展保修组合，除康柏取件服务外，其余均为三年保修期。其中，上门服务响应时间预计为下一个工作日内或四个小时内。每天或每星期的实际服务时间则视不同组合而定。送修服务是让客户将产品自行携带至最近的康柏授权服务中心进行检修。而取件服务则由康柏派工作人员收取客户的硬件，并在修好后返回。
　　3.关怀服务包软件支持服务
　　关怀服务包设有两款软件支持组合，通过电话为客户提供软件技术支持服务。其中，客户机应用系统支持服务为台式机系统软件应用和运行在Microsoft Windows 95及Windows NT操作系统的工具提供技术支持。服务器操作系统支持服务则针对Microsoft Windows NT和Novell Netware操作系统，提供技术支持。
　　4.关怀服务包优先服务：
　　关怀服务包优先服务旨在为用户提供超越基本保修支持以外的服务，实现高可用性的Microsft Windows NT环境。此项关怀服务包服务提供灵活的综合硬件及系统软件服务，以上门和远程诊断等预防性维护，到方便地访问技术资源等。
　　如何获得康柏的关怀服务
　　用户在购买机器时即可得到随机附上的关怀服务简介及登记卡，另外用户还可以在各康柏代理网点、维修中心和康柏客户服务部得到上述资料。用户只需选择所需的服务项目，填好服务登记卡，将所需的款项汇入康柏提供的账户，再把登记卡和银行付款凭证寄回康柏，即可得到所需服务。如有问题，可拨打康柏客户服务部的电话：010－62058866或免费支持热线800－8106888，也可发电子邮件：PRCbucsopr@compaq.com。
