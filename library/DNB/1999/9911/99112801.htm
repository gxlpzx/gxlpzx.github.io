读书郎的百宝箱

　　“小嘛小二郎，背着那书包上学堂……”
　　嗯嗯，各位网虫同学，上课了。今天我们讲课的内容是——网上读书，在网络上读书，除了要找到可读的书本外，配置称心的工具也很重要。所以我今天来向同学们推荐几款网络读书的必备工具。

阅 读 器

　　阅读器是读书者最常用的工具。一般我们可以用写字板等文本编辑器来阅读这些保存在硬盘上的文章，但如果用专门的阅读器来读，你会发现它比写字板等好用得多。下面请同学们和我一起来看看几种阅读器。
　　CView 2.23(下载地址：http://www.netease.com/~cloudwu/file/cv22.zip)是个DOS下的阅读软件，虽然是 DOS 程序,但全兼容 Windows，可以将多个文件拖入其内，且支持 Windows95的长文件名标准。它提供像素级的平滑滚屏，使得电子文章能自动滚动。CView 2.23提供了20级滚动速度,你可以根据自己的阅读速度进行适当的调整。提供的智能速度调整开关, 使滚屏速度随文字密度而改变。如果你不习惯或者觉得慢慢滚动不舒服，还可以打开“平滑滚动开关”, 使文字以半屏为单位上滚,然后暂停。总之，CView 2.23可提供最适合你的阅读方式，让你的双手得到解放。当然，如果你愿意，也可使用键盘和鼠标操作。
　　很多人在看一本书时，未看完，就会作个记号以方便下次再看。CView 2.23也有类似的功能，它提供阅读记录, 在每次阅读一个文件需中途退出时, CView2.23将把你的阅读进度记录下来，方便下次继续读书。
　　另外，CView 2.23还提供了源文件多内码智能识别, 除常见的 GB 和 Big5 码外, CView2.23 对因特网上的多种“乱码"也有很强的纠正功能。它的文件选择器能让你挑选阅读部分。它的空格压缩调整功能使在网上下载的许多汉字间夹空格的文件读起来更舒服。
　　ReadBook 1.0(下载地址：http://www.zhanjiang.gd.cn/nethome/magician/)是一个运行在Windows95/Windows98简体中文版下的32位中文程序，主要用于阅读中文文本文件。它提供历史纪录功能，对每一个阅读过的文件都会自动纪录文件的路径和文件名，让你下次阅读更加方便，最多可记住4096个文件。
　　ReadBook 1.0可智能消除HTML文件中的所有HTML标记，看HTML电子书非常方便(一般网上看到的文章多是HTML格式)，让同学们再也不用因看到那些讨厌的HTML尖括号标记把文章内容分割得支离破碎而烦恼。
　　当然它并不是只有上面提到的两个功能，它还提供全文查找功能、书签功能、滚屏阅读功能和智能分段功能等。
　　e－BOOK 1.04(下载地址：http://www.eb.com)。很多同学对CView、ReadBook这样的软件还是敬而远之，觉得看起来没有读书的感觉。那么我就推荐你们来用用e－BOOK，这个软件的界面被做成一本书的形状，设置好了以后，手都不用动。这本书可以一页一页地自动翻页，让你舒舒服服地阅读。
　　e－BOOK是用VB5设计的32位文本文件阅读器，尤其适合于阅读各种文本格式的电子小说文件，它的特点是采用了印刷书籍式的人性化界面，具有内码识别、智能分段、自动翻页、自动刷新、自适应环境、书签等功能，我给它的评语是“传统书籍与电子小说的完美结合”。
　　除了上面三种通用的阅读器之外，还有一些专用的阅读器，如专门针对PDF格式文件的Adobe Acrobat、WDL格式文件的DynaDoc Reader和中国国家图书馆的专用阅读器超星PDG远程浏览器等。同学们可根据自己的需要来选择一种适合自己的阅读器。

存 档 器

　　凡是经常读书的同学大概都有这样的烦恼：一篇篇的好文章看过以后，分门别类保存起来。但如果不勤加整理或用醒目的文件名进行标示的话，可能过一些时候，就会面对一大堆的HTML文件或者TXT文件发呆，不知道谁是谁了。谢天谢地，这个苦恼并不只是你我才有，程序员们也有，他们还有更痛苦的时候：寻找自己过去开发过的代码。终于，有程序员忍受不下去了，于是就有了电子书库的由来。
　　Code Manager(电子书库)(下载地址：http://www.netease.com/~kiven/soft/cm.zip)是一个功能较全的软件，它提供的主要功能有：可以添加任意的文本到该数据库，方便自己的查询与阅读；随时删除不想保留的文章，避免数据库过于庞大；用户可随时按需要分类；压缩数据库的功能有助于用户缩小数据库的尺寸，废弃数据库中没用的旧文章；引入和导出功能可将任意的文本引到数据库中或将数据库中的文章导出成文本；添加功能可以将成批的文件导入数据库。
　　怎么样，你是不是对它很感兴趣呀！

转 换 器

　　同学们，光是存档有时候是不够用的，当你急需使用文章中的资料时，特别是台湾同胞的一些文章，都是BIG5码(我们很多同学对这样的稿件是望眼欲穿啊)，不能直接粘贴使用是件很麻烦的事，怎么办？啊，当然是进行内码转换，然后再采用“拿来主义”即可。
　　MagicWIN(下载地址：http://ww.coconuthouse.com.my/magicwin/magicwin120.exe)可以让同学们顺利地浏览中日韩三国文字的网页，最大特点在于它可以在不同的窗口使用不同的编码来阅读不同的语言。支持 GB、GBK、BIG5、Shit JIS、EUC JIS、JIS、KSC等亚洲语言码。已注册者可以下载 24×24 字型，使显示效果更美观。使用方法是将字型解压至与 MagicWIN同一目录下即可。
　　Njstar Communicator(下载地址：ftp://ftp.download.com.cn/pub/chinese/njcom20.exe)支持各种 32 位版本的 Windows (95/98/NT) 和各种本地化版本的 Windows (简体中文、繁体中文、日文、韩文)。自动内码识别，可用各种输入方法(拼音、注音、双拼等)自动按所查看文本的内码输入。提供强大的内码转换工具，支持 4.x 版的 IE 和 Netscape。
　　Chinese Box 1.21(下载地址：ftp://ftp.download.com.cn/pub/chinese/cbox121.exe) 是拷贝猫出品的精致小工具，能对剪贴板中的中文进行处理。功能很简单，从剪贴板中获取文字资料，经过处理后自动放回剪贴板。Chinese Box1.21还有强大的段落格式整理能力；GB、BIG5、HZ汉字内码自由转换；在汉字间加删空格；体积轻巧，可与任何编辑器共存。运行于中文Windows95/98/NT。
　　TxtEasy！1.5.5(下载地址：ftp://ftp.download.com.cn/pub/chinese/TxtEasy!.exe)是小巧方便的文本转换工具。它可将HTML文件转换成TXT文本，也可以对TXT文件进行智能分析(去掉多余的空格、空行和不必要的硬回车)和优化处理(绝对保持原文段落，将半角标点符号转换成全角标点符号)。它能“批量”和“单个”处理任意目录下的TXT和HTML文件，而且转换速度快。
　　注意：缺省安装在C:\ProgramFile\TxtEasy!\目录。在Windows98下直接运行，在Windows95和NT下需安装VB5.0运行库。

另类软件

　　InsTalk(下载地址：ftp://ftp.download.com.cn/pub/chinese/instalk265.exe)是面向Windows95和WindowsNT的工具软件。该软件最大特点是当你在使用键盘输入数字和英文字符时，电脑附带的喇叭可以跟随录入的字符同步发出相应的语音提示，避免各种录入错误。不过同学们请注意，我们在这里利用的是该软件的校对功能，把选中的文字通过电脑的多媒体功能朗读出来。这样那些懒得阅读的朋友们或者读书累了的同学们，就可以闭目养神，竖起耳朵听书了。
