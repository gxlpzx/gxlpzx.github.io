书山有路网为径之网上购书

　　看够了，也说够了，考虑去买本书来看看吧。水手喜欢上网看书，但更喜欢一个人窝在床上，耳边回响起理查德的曲子或赵传的歌声，慢慢地翻着一本刚买的新书，翻着翻着，伴着书香入睡。毕竟，对着屏幕看书和一个人拿着本充满墨香的book是截然不同的感觉。当然，网络时代买书完全可以坐在家里，下面是水手常去的几个网上书店。
　　上海书城网上书店：(http://www.bookmall.com.cn)提供各类图书、音像制品和电子出版物，品种多达十二万余种，覆盖了实用生活、艺术等七大门类。该网上书店对所有VISA国际信用卡的海内外持卡人开放SET支付功能，对其它国内信用卡/借计卡的持卡人开放SSL支付功能。
　　中国现代书店(http://www.modernbooks.com/)是一个销售中国大陆出版社发行书籍报刊的网上书店，它最大的特点是一个字“全”。打开其杂志类的目录，从哲学、文学、文化等方方面面无所不有，其图书类的收集也相当全，几乎包括了所有在国内书店里所能看到的书籍。
　　博学堂网上书屋(http://www.chinesebooks.net)是个香港的站点，以网上销售书籍为目的，有10万种书目供读者选择。博学堂提供分类检索和书目检索两种方式，让你方便地找到感兴趣的想买的书籍。每周推荐300个书目，读者可按书名、作者、分类等进行检索。
　　广州购书中心(http://book.gznet.com/scripts/cgiip.exe/index.htm)广州购书中心将图书分为社科类、文学类、音乐类等共十大书目，同时开设有新书介绍、重点推荐和各类排行榜栏目。用户可按分类进行查询所需图书，该中心还为需大宗订货的用户提供本书店所有图书资料的下载。
　　四川人民出版社慧新源网上书店 ( http://www.booksss.com)界面尚可，栏目也很多，有每周新书、完整书目、故事连载、海外动态、免费读物等，免费读物有一些好看的文章，并可下载，能够网上购书！不过不是在线方式，只能用银行汇款或邮局汇款两种方式支付。
　　华储网上电脑书店 ( http://www.huachu.com.cn)界面简洁，新书快递和畅销书目里可看到书的完整目录，分类检索做得不错，并可进行模糊查询，速度很快。若要网上购书的话，付款方式还是银行和邮局这两种，一次购书超过200元可打八五折(谁要买书可别忘了约上水手！)，款到后三天内发书，并且邮政上调邮费后不涨价，可订阅新书快递的MAIL，是国内一个非常有潜力的图书站点。
　　南凌书城(http://www.szbookcity.com)是深圳南凌实业发展有限公司兴办的大型网上书店，主页面比较简单，有详细的分类，并提供上周销售龙虎榜，现在新书推荐上有一本厉以宁的《非均衡的中国经济》，不过每本书只能看到内容简介。付款方式除前面提到的两种外，现在支持招商银行一卡通付账。
　　智慧通网上书店 ( http://www.ideatone.com)这是本人见过的最漂亮且很快的国内书店站点。点击链接“电脑”还可订购联想电脑，但VCD＆CD和软件下载就只是一个简单的产品查询，另外在图书的链接下存在一些错误，需要改进。

本文出自：《电脑报》1999年03月22日第11期
