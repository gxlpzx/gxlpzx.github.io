我是这样管理自己的资料

TCL杯全国电脑应用方案大奖赛

方案背景

　　我是一名计算机人员，工作中经常与计算机打交道，许多文件、报告和表格均用计算机处理；生活中也离不开电脑来收发电子邮件、在Internet上冲浪等。两年多来，我用计算机完成的各种工作报告、表格，下载的各种工具软件、文章，收集的各种资料、购买的各种CD软件、CD唱片等不计其数。
　　对于这些日异增多、五花八门的电子化信息资料，我想若能用计算机来管理就好了，这样，在领导询问半年前完成的报告还在不在时，我就用不着翻遍所有抽屉、查过所有磁盘上的文档也不知道是否还留有备份；当手头急需使用某种工具软件，记得下载过却不知道在哪张磁盘或CD软件中时就用不着一张张地去读，而是只要输入几个关键字，程序就会自动反馈文件所在地、归档时间、内容提要等全部信息，大大方便工作和学习。
　　我曾经自己编过几个程序用于文档、图片和个人信息等的管理，但都不能满足以各种文件形式的电子化信息进行自动管理的需要。我还通过Internet搜索是否已有这种功能的软件，结果都只是单功能的。我想，各种单功能的软件集成在一起不就是全功能的，不就可以应用在实际之中吗？于是，我设计了下面这个方案用于“个人电子化信息资料管理”。

方案实现

　　根据自己的爱好，我将“个人电子化信息资料”定义为以下几类：
　　
　　1．文件型：包括工作报告和表格、下载的软件和文章、编程用控件和图标、图像文件、MP3文件等；
　　2．磁盘型：磁盘、CD光盘等；
　　3．其他型：篇幅不长的文章、Internet网址、电子邮件、日记、便签、个人通讯录。针对以上文件类型，我从网上搜索到一些软件可以很好地进行管理，分别介绍如下：文件型
　　 Mybase资料管理器 1.0：这是王俊蛟用C＋＋Builder写的一个很好的个人资料压缩管理工具，用于管理各类文档、磁盘文件、资料、名片、下载的精华、收集的资料等。该软件采用树形结构管理资料，任何资料都可以是一个节点，每一个节点都有“标题”、“内容”、“备份”等进行说明分类，将每一分类的各种文件作为节点附件，也有“文件名”、“原始路径”等说明，这样便于分类管理。同时，对篇幅不长的文章、Internet网址、个人通讯录等信息资料也可以作为节点分类管理(界面和功能说明如图一，下载网址：http://junjiao.yeah.net)。
　　
　　 ACDSee 32：这是一个著名的看图软件，不但可以浏览十多种格式的图片、对各种图片进行管理，还可以通过“描述”功能对硬盘中的其他文件进行管理说明(界面和功能说明如图二，下载网址：http://www.acdsystems.com)。
　　磁盘型
　　DiscoMania 2.03：这是一个免费的将硬盘、软盘、光盘、ZIP中的文件制成索引的软件，可以给每个文件加上详细的说明，进行关键字查询等，因而管理磁盘型文件非常方便(界面和功能说明如图三，下载网址：http://www.fortunecity.com/skyscraper/jobbs/79/index.html)。
　　
　　Advanced Disk Catalog 1.2：这是俄罗斯Elcom公司的软件，能把光盘、磁盘中的文件目录进行扫描并记录成数据库文件，可以对文件进行说明和查询，需要注册费$20，否则每个数据库不能超过5个盘的内容(界面和功能说明如图四，下载地址：http://www.elcomsoft.com/adc.html)。
　　其他型
　　Microsoft Outlook 97：这是微软公司著名的个人信息管理软件，可以管理约会、联系人、任务、电子邮件、日记、便签等，功能强大(界面和功能说明如图五)。
　　
　　根据信息的分类和以上各软件的功能，我主要选取了三种作为“个人电子化信息资料”管理工具(如图六)：
　　1． Mybase资料管理器 1.0：主要管理我个人的工作报告和表格、下载的文件和文章、图像文件、MP3文件、个人喜好的Internet网址、个人通讯录等文件。2． DiscoMania 2.03：主要管理我的软盘、CD软件和CD唱片等磁盘文件。3． Microsoft Outlook 97：主要管理我的电子邮件、日记、便签、联系人等资料。其它一些软件则作为辅助管理。通过使用这些软件对我个人的信息资料进行分类整理，使之条理化，大大方便了自己对各种资料的查询和使用，从而提高了工作和学习的效率。
(湖南　华　文)
