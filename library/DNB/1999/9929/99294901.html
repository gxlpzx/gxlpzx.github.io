《 TNT2大比武——小影霸UNIKA TNT2全系列测试 》

　　随着G400、Permedia3的上市,第四代显卡的大战已经全面开始了，各个厂家都鼓吹自己的产品，可以说现阶段的形势还是一片胶着状态。虽然在这场争斗中孰优孰劣暂时还不能给出一个定论，但是随着时间的流逝，广大用户还是能够分辨出真正的好坏。
　　TNT2作为这代3D加速显卡中佼佼者，目前在显卡市场上可以说是炎手可热。TNT2系列是一个标准的多面手，有着比较优秀的游戏和专业设计功能，在一般的游戏和设计工作中也能应付自如，所以许多人对TNT2的性能感到很满意——不输于Voodoo3的游戏性能，比Voodoo3的图形处理功能更强悍。虽然在专业图形设计领域和Permedia3可能有一点差距，但是以价格的优势,把这些差距弥补了回来。TNT2的优势在于各方面都比较均衡。
　　在我们以前的3D显卡测试中已经提过，现在的显卡在命名的方式上已经向CPU的命名方法靠近了，都是以频率的不同来区别同一系列的不同产品的。以TNT2家族为例，以芯片频率的高低将TNT2分成了2类4种型号：
　　核心频率/显存频率　外部数据接口
　　TNT2 Ultra　　　150/183　　　128位
　　TNT2　　　　　　125/150　　　128位
　　TNT2 Model 64　125/150　　　 64位
　　Vanta　　　　　100/125　　　 64位
　　这四种产品就是目前TNT2家族的全线产品。在四种产品和TNT均采用同一核心架构，也使用同一驱动程序驱动，除了速度上的差别以外，在3D画质表现上基本相同。TNT2/Ultra功耗很大，部分供电不足的主板会出现不稳定现象。对于非Intel芯片组主板，TNT2系列兼容性非常好。nVIDIA的标准驱动程序异常优秀，兼容性、稳定性都非常好。TNT2号称D3D性能之王，在OpenGL ICD方面表现也非常突出。
　　既然TNT2推出了这么多种型号，那么在性能上究竟有什么区别呢？它们在外形上的区别又在哪里？现在市场上已经有人将低端的TNT2 Vanta和TNT2 Model 64以TNT2的价位卖给消费者，还将普通TNT2加上较高频率的显存再超频到150/183当Ultra卖给消费者，损害了消费者的利益。通过这次测试希望给大家建立正确的观念。
　　这次测试以小影霸UNIKA TNT2系列的全线产品来说明这四款TNT2的不同之处。说起小影霸，大家都不会陌生,相信许多人都用它的产品。小影霸是香港双敏电子UNIKA拥有的品牌，产品涉及3D加速卡、多媒体产品、CD－ROM、PC 133/100内存条。小影霸的显卡在市场上走的是性能价格比路线，其产品品质较高，稳定可靠，性能也在水准之上，价格基本在同类产品的中低价位。小影霸也正是以中低价位和较高的品质赢得了市场和用户。
　　测试时考虑到读者实际使用情况，设定为100Hz刷新率(刷新同步打开),使用最新TNT2家族Detonator 2.08雷管驱动程序，对3DNow!和KNI(SSE)作了优化。
　　测试产品介绍：

小影霸TNT2 2500/3500高品质豪华系列

　　从今年开始，小影霸nVIDIA 产品线在原来的基础上增加了一条高品质豪华系列，是委托欧美大厂Guillemot 制造，这就是TNT2 2500/3500,分别对应Guillemot Game Maxi Game Xentor 16/32,从命名上就知道是为游戏准备的3D加速卡。TNT2 2500/3500均采用双面SMT工艺，做工精细,用料非常好，在卡的背面有大量的电阻以辅助散热。小影霸推出该系列产品意在和其他大厂最顶尖的产品抢夺高端市场。
　　●小影霸TNT2 3500 TNT2 Ultra 32MB/TV
　　采用TNT2 Ultra芯片,32MB(16颗 1M×16)世界先进 5.5ns 183MHz SDRAM,Rockwell BT869视频输出芯片，带有主动散热风扇,频率设定为 175/183,高于标准TNT2 Ultra 150/183的设定，因此性能表现非常突出，大约比标准TNT2 Ultra性能高3～5％，也是测试实验室目前测试到最高性能的TNT2 Ultra。它的性能价格比也非常好，是目前国内最便宜的TNT2 Ultra。
　　(参考价格：1880元)
　　●小影霸TNT2 2500 TNT2 16MB
　　采用TNT2芯片,16MB(8颗 1M×16)世界先进6ns 166MHz SDRAM,只采用了一个大的散热片,频率设定为135/166,高于标准TNT2 125/150的设定，性能接近标准TNT2 Ultra，大约比标准TNT2性能高5％～10％。加上散热风扇后，该卡的超频能力极强，极限在155～160/183左右。TNT2 2500的价格处在TNT2 产品的中下游，非常有竞争力。(参考价格：1130元) 小影霸 3D－TNT2/Model64/Vanta平价系列
　　小影霸AGP 3D－TNT2系列是委托台湾YUAN(聪泰)制造，该厂是nVIDIA的重要合作伙伴之一，该系列显卡附送了台湾讯连Cyber Link最新Power DVD 16版本。
　　小影霸 3D－TNT2共有4个版本，均采用nVIDIA TNT2芯片，分别为16MB/32MB,带或不带TV输出，带有主动散热风扇。测试时只挑选了32MB带TV和16MB不带TV两种。该TNT2系列频率均设定为 140/160,性能高于标准TNT2 大约5～10％,散热风扇噪音非常小。
　　●小影霸AGP 3D－TNT2/TV＋/32MB
　　和其他32MB版本的TNT2不同，采用8颗2MB×16SDRAM芯片,看上去和16MB版本的卡非常像，也采用Rockwell BT869视频输出芯片。
　　(参考价格：1330元，不带TV版本为1250元)
　　●小影霸AGP 3D－TNT2/16MB
　　采用16MB(1M×16)台湾南亚7ns SDRAM,该卡的超频性能非常突出，极限可以达到150/183,所以设定在140/160频率上很稳定。这款卡的价格非常突出，仅在1000元左右，和价格相近的其它品牌TNT2 Model64 32MB相比，3D－TNT2的性能可以说大大超出，在1024×768×32位色性能已经超过TNT2 Model64 50％，接近Model64同分辨率16色性能。
　　(参考价格：1030元，带TV版本为1100元)
　　请注意小影霸以下两个TNT2系列，－M表示Model 64,－V表示Vanta,在包装上已经清楚写明。从外观上看两张卡“长”得一模一样，简洁、小巧而精致。卡上的BIOS虽是长条状的，仍然是可更新的FLASH EEPROM。
　　●小影霸AGP 3D－TNT2－M/32MB版本
　　频率设定在125/150，整体性能略高于16MB TNT。800元的价格，TNT的性能，32MB显示内存, TNT2 Model64在800元以下的显示卡中竞争能力非常强。
　　(参考价格：800元)
　　●小影霸AGP 3D－TNT2－V/8MB版本
　　Vanta芯片的频率设定在100/125，加上只有8MB SDRAM,使得1024×768×32位的测试无法进行。在Vanta超到125/150的频率时，得分和Model64完全一样，得益于TNT2 Detonator雷管驱动程序的优良表现。Vanta非常稳定，在低分辨率下的表现相当不错，对比nVIDIA 的前一代产品Riva128/ZX, Vanta要好很多。在性能上,Vanta没有太多的可圈可点的地方，但就其低廉的价格，在600元以下的3D加速卡中，性能价格比依然不错。(参考价格：8MB 590元，16MB 700元） 　　测试结果：
　　2D性能：
　　在2D性能方面，由于都是使用的TNT2核心，在16bit下没有什么大的差别。但在32bit时，TNT2 Vanta和Model 64在得分上与TNT2的差距就比较大了，而TNT2和TNT2 Ultra的差距还是不大。借助250/300MHz RAMDAC在未来窗显示器上图像锐利清晰。
　　3D性能：
　　从测试的结果看得出在低分辨率下这四种TNT2的差距不是很明显，但是当分辨率提高以后它们之间的差距就明显地拉大了，特别是在用到了大材质和32位色渲染的时候这样的差距就更加的明显了。在3DWinbench99中，在高分辨率下使用大材质时，TNT2 Model 64的成绩几乎只有TNT2 Ultra的一半，和TNT2也相差很远。
　　TNT2 Ultra：借助最高频率的内核和显存，TNT2 Ultra是TNT2家族中性能的王者。在高分辨率和32位色下的表现尤为突出。
　　TNT2：TNT2的测试结果令人满意。TNT2的性能已经比Model 64性能高出了30％左右，TNT2 Ultra则超出更高。此外，TNT2和TNT2 Ultra的性能差距在10％～20％左右。不同TNT2和TNT2 Ultra的性能差距主要来源于它们的内核和显存频率，此外使用16M和32M的显存的TNT2在性能上的区别也只是在使用大材质和使用真彩色渲染的时候才表现得较为明显。
　　TNT2 Model 64：由于核心和内存的频率都大大超过了TNT的90/110, 虽然只是64位数据外部接口，整体性能比仍TNT略快一点。事实上TNT2 Model64 32MB的竞争对手是S3 Savage4 Pro 32MB。在测试中我们对比了Savage4 Pro 32MB(110/125)的性能，发现小影霸TNT2 Model64全面超越了S3 Savage4 Pro,性能超过10％～20％以上，特别是在利用OpenGL的游戏和程序中差距更明显。然而和TNT2相比，TNT2 Model64的性能大大落后，在高分辨率和32位色彩下特别明显，几乎落后一倍。
　　Vanta：
　　整体来说，TNT2 Vanta的性能是最差的，与TNT2 Model 64相比较都有差距。无论在专业测试还是在游戏测试中，它的分数都列在最后。不过和Savgae4 Pro仍有可比性，性能相当接近，在专业OpenGL部分则大比分领先。
　　超频性能：
　　除了32M的TNT2，小影霸的这几款TNT2的超频性能都很不错的。其中以两款16M的TNT2 的超频性能最令人注目，已经达到了150/183 TNT2 Ultra的频率。而TNT2 Model 64能够超频到140/160这样的频率。而32M的TNT2只能到150/160，几乎没有什么超频性。相比较而言，由于小影霸的TNT2 Ultra本身的频率高于普通版本的TNT2 Ultra，已经很高了,尽管还有一定的超频空间，但个体之间的差异较大，可超频的幅度不一样。
　　测试中TNT2的核心频率可以超到极限155～160MHz，但TNT2 Ultra却可轻易达到175MHz以上，所以即使超到150/183MHz的TNT2也不是Ultra，虽然性能基本一致。此外，超频存在个体差异，每块卡的极限不一样。
　　TNT2系列超频一定要有良好的散热，频率升高会使功耗和发热量同步增加，不要太靠近显卡的极限稳定频率，因为长时间的运行，很容易使显卡过热而死机。
　　结论：通过以上的测试结果，可以清楚地了解到，决定同系列显卡性能高低的关键在于芯片频率的高低和使用显存的频率和大小。
　　TNT2 16MB是目前TNT2家族中间的最高性价比版本，其次是TNT2 32MB和TNT2 Model 64，而TNT2 Ultra则是狂热玩家追求极速的至爱。
　　TNT2 Model 64和Vanta是针对低端市场的产品，在性能上与TNT2/ Ultra有一定差距，但是价格却比较便宜。希望大家在选购TNT2系列时能够分清它们的区别，选购到一块称心如意的显卡。
