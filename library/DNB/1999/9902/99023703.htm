避免在Windows 95中死机

　　Windows 95虽然是一个十分娇贵的操作系统，但只要“调理得宜”，“她”不见得就会经常“生病”。笔者在此与你分享几个方法，可以有效地避免Windows 95经常性地死机。
　　1. 硬盘在写入时，尤其是像在使用Netscape Navigator，Microsoft Explorer等WWW的浏览器时，因为这些浏览器会读入大量的图形、文字、音乐、动画，甚至视频图像，而系统必须不时地把内存中的内容置换到硬盘中，使得新近读入的数据能够有足够的内存可以使用，因此当你看见硬盘的指示灯持续亮着时，最好不要再进行任何操作，包括键入、播放音乐、进入新的Web站……等到硬盘停止写入的操作之后，再继续浏览或打字。
　　2. 除非不得已或没有更新的版本，否则尽量不使用“不是专门为Windows 95设计的”软件。这些在DOS、Windows 3.1环境下本来很好用的软件，到了Windows 95之下就会产生不少问题。比如说，因不支持长文件名而使长文件名消失、降低整体性能，甚至因不完全兼容而导致无法启动等状况。虽然Microsoft宣称，过去的16位软件仍然可以在Windows 95环境下使用，但避免死机的“正本清源”之道，还是只使用专为Windows 95设计的软件。
　　3. 外部设备如Modem、打印机、扫描仪……等等外接式的装置，不可在Windows 95启动的过程中打开或关闭，而应在启动前尚未看到“Starting Windows 95……”之信息，或完全进入Windows 95图形环境之后，再打开这些外部设备。如果因为不适当地打开外部设备而导致Windows 95无法启动的话，请先用安全模式（Safe Mode）启动一次，关机后再用Normal模式启动即可恢复正常。
　　4. 定期（最少每周一次）用扫描病毒程序扫描硬盘一遍，并从各大BBS或FTP站取得最新的病毒扫描码。
　　5. 定期（最少每周一次）用Windows 95的“磁盘扫描程序”来检测硬盘的“健康状况”（可以设置由“系统代理程序”来代劳）。(黑龙江　梁伟)
