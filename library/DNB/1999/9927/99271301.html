《 冲击高端：Windows NT 》

　　Windows NT是Microsoft公司开发的网络操作系统，它采用与Windows一致的图形操作界面，具有易于学习、易于使用、可靠性高、安全性好等特点。自1993年面世以来，Windows NT以其友好的图形界面、简易的操作和丰富的应用程序，赢得了广大计算机用户，尤其是Windows用户的欢迎。
　　Windows NT特点
　　NT是New Technology的缩写，意思是“新技术”。作为网络操作系统，Windows NT包括Server(服务器)和Worktation(工作站)两个产品，其当前版本Windows NT 4.0的主要特点有：
　　1.高性能
　　Windows NT是一个32位高档网络操作系统，它的高性能来源于两个方面：32位地址空间和抢先式多任务处理。32位地址空间支持4GB内存空间寻址，能使用户实现性能更高、速度更快的应用程序。抢先式多任务处理方式则使得用户可以并行运行多个任务，减少了Windows由于一个任务阻塞而导致死机的问题。
　　2.易用性
　　Windows NT具有与Windows一致的图形化操作界面，系统的管理和维护非常方便。NT内部集成了专门的管理向导，诸如创建用户账号、设置文件的访问权限、设置网络打印机以及客户工作站的安装及维修等任务，都可以在管理向导的指导下一步步完成，大大地降低了系统管理的复杂性。
　　3.安全性和可靠性
　　在网络操作系统中，信息资源的安全性是非常重要的，Windows NT达到了C2安全等级，对安全性要求比较高的领域，如金融、军事、银行、商业等，都可以使用Windows NT构造其企业网络。
　　4.开放性
　　Windows NT采用独立于处理器的可移植设计，除了支持Intel系列处理器外，还可以很好地在PowerPC、Alpha及MIPS芯片上运行，具有很强的跨平台能力。
　　Windows NT也是一个完整的网络系统，支持广泛的网络协议，能够与目前流行的多种系统，如Novell、Macintosh、Unix等实现互连接。
　　5.Internet／Intranet支持
　　在Windows NT Server中集成了IIS(Internet Information Server)功能，企业能够快速高效地在Internet或内部局域网上建立Web站点。在Windows NT Server 4.0中还包括了索引服务器及FrontPage，使内容索引及Web站点的创建和管理更加容易。
　　6.RAS远程访问服务
　　Windows NT远程访问服务允许用户通过X.25、ISDN或标准的电话线从远程拨入Windows NT远程访问服务器，如同直接连接到网络上一样。有了RAS功能，用户可以在家中或者出差在外地时通过电话线与办公室的局域网连接，使用公司局域网资源，如同用户在办公室里一样。
　　Windows NT常用软件：BackOffice
　　下面介绍微软公司专门为Windows NT开发的软件组——BackOffice。BackOffice是一组后台服务软件，它由不同的服务器产品组成，主要包括：
　　1.Exchange Server(电子信息交换服务器)。一个功能强大的企业电子邮件服务器。
　　2. Proxy Server(Internet代理服务器)。运行在NT Server上的一个可扩展防火墙和Web缓存服务器。
　　3. Internet Information Server(因特网信息服务器)。集成了多种Internet服务(WWW服务、Gopher服务和FTP服务等)的服务器软件。
　　4. Small Business Server (小型商务服务器)。 微软公司为企业更好地提高商业操作和管理资源和客户信息构建的信息服务器软件。
　　5. SNA Server(系统网络结构服务器)。 提供大型机与小型机之间的通信服务，使企业不同的系统有机地结合起来，保护了用户的前期投资。
　　6. SQL Server(数据库服务器)。客户机／服务器结构的关系型数据库管理系统，是企业管理信息资源的重要工具。
　　7 .Systems Management Server(系统管理服务器)。提供对NT网络集中配置和统一管理功能。
　　迈向Windows2000
　　微软已将下一代Windows操作系统NT 5.0命名为Windows 2000。其中Windows NT Workstation 5.0被命名为Windows 2000 Professional；Windows NT Server 5.0则更名为Windows 2000 Server。与Windows NT 4.0相比，Windows 2000在易用性、安全性、可靠性等多方面都做了重大改进，它将Windows操作系统正式带入二十一世纪。
　　Windows NT知识卡片
　　1.客户机／服务器(Client/Server)
　　“客户机／服务器”是一种信息组织和流动的模式。它将信息体系结构分成客户(Client)和服务器(Server)两部分，服务器端集中管理信息资源，并处理客户端发来的请求，送出客户请求的结果；客户端向服务器端发出信息请求，显示服务器返回的结果。这种模式将资源集中管理，极大地降低了客户端的负担。
　　2.域(Domain)
　　在Windows NT中，用户在规划网络时可以根据需要将同类型的服务器或隶属于同一部门的多个服务器分在一个组中，称为域(Domain)。域是NT安全性和集成化管理的基本单元，在域中选定一个服务器为域控制器(Domain Controller)，它是用户的账户管理中心，用户只需在域控制器上建立一个账户，就可以访问域内所有的服务器和信息资源。
　　3.委托关系(Trust Relation)
　　在Windows NT中最基本的组织单位是域，域与域之间相互独立，但如果确实需要域之间的资源共享应如何处理呢？在Windows NT中就是通过委托关系实现的。委托关系是两个域之间的链接，它使其中一个域能委托另外一个域的用户。委托关系由委托域和被委托域两部分组成，所建立的域之间的委托允许用户拥有一个域中的账号即能访问整个网络的资源。
　　4.Windows NT文件系统(NTFS)
　　NTFS是专门为Windows NT而设计的一种文件系统，具有很好的安全性能，支持完整的Windows NT安全机制，如文件或目录的访问权限及等级，文件名或目录名的长度可多达255个字符等。另外该系统还能保留文件读取的活动日志，当发生断电等问题时，可以恢复磁盘中的文件。
(山东　尉鹏飞)
