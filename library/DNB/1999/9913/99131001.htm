在硬盘上虚拟光驱——Virtual Drive 3.1

　　Virtual Drive是一个把光盘映射到硬盘中运行的程序,它通过特殊的驱动程序“骗”过Win95/Win98操作系统，使操作系统认为有多个光驱，系统中多出的“光驱”就是Virtual Drive构造的虚拟光驱。Virtual Drive程序安装后,即在系统中创建了一个虚拟光驱，要使用它，还得先将真正光盘上的数据压缩储存到硬盘上，即制作虚拟光盘。将制作好的虚拟光盘“插入”到虚拟光驱后，即可像运行普通光驱一样运行虚拟光驱中的“光盘”了。Virtual Drive是全光盘文件虚拟，不但可以虚拟标准的光盘文件，还可以虚拟CD、游戏光盘上的CD音轨，对于需要使用光盘卷名和光盘属性的程序、游戏都可以在虚拟光驱上顺利运行。虚拟的光盘还有一个优点就是可以防止病毒入侵。

下载与安装

　　1.下载
　　Virtual Drive 3.1是Far Stone公司出品的共享软件，可以从网址http://www.farstone.com下载，由该网址下载的V3.10试用版有中文版、英文版和日文版三种语言版本，仅21天的试用期，并且限定只能在单机上虚拟一个光驱和两张光盘。要得到网络版、单机完全版必须购买，完全版最多支持23个虚拟光驱及无限张虚拟光盘。同样在该网址还可下载Virtual Drive的V2.0版，它与V3.10版的最大区别是不带光驱加速功能。
　　2.安装
　　网上下载的Virtual Drive是一个自解压文件，把它解压到硬盘上，然后双击setup.exe开始安装。安装拷贝结束后按要求重新启动电脑。然后双击“我的电脑”，如果一切正常，你就会看到在其中增加了一个光驱图标，这个光驱图标就是Virtual Drive在系统中虚拟的光驱，Win95/98会认为这是一个真正的光驱，下面将说明如何为该虚拟光驱制作虚拟光盘。
　　注意：安装过程中，如果出现某某程序正在被使用，无法打开之类的警告，那多半是系统中安装了防病毒软件所至，只需单击“确定”即可，并不影响程序的正常安装。
　　3启动
　　Virtual Drive通过一个名为“Virtual Drive Manager”的程序来完成虚拟光驱的各项操作，从开始菜单“Virtual Drive”程序组中单击“Virtual Drive Manager”项或者在任务条状态栏中双击该程序状态图标启动程序。

制作虚拟光盘

　　启动Virtual Drive Manager后，主界面。
　　1.点击工具栏中的“Build VCD”按钮(倒数第三个按钮)，进入“Build VCD”画面。注意，这里的“VCD”指的是“Virtual CD”即虚拟光盘，而非通常的影像VCD。
　　2.将要虚拟的光盘插入真正的源光驱中，在的“Destination”栏中设定虚拟光盘存放的位置和存放的文件名(文件名可以使用默认值)，在“Descripton”项可以填入光盘说明。在“Default VCD drive”栏中设定要虚拟光驱的盘符。
　　3.为了节省硬盘空间，Virtual Drive支持以压缩方式创建虚拟光盘，选中“Use data compression”复选框，用鼠标拖动滑竿调节压缩率，然后点“OK”，程序即开始将光盘上的数据复制压缩到硬盘创建虚拟光盘。Virtual Drive采用非常专业的压缩方式，根据文件在光盘中存放的形式不同，一般可达到15%～50%的压缩率。
　　4.虚拟光盘创建完成后，在程序主界面的右边列表框中将会出现虚拟光盘图标，现在就可以使用虚拟光驱了，Virtual Drive中可以无限制地创建虚拟光盘(都列在主界面中)，也就是将真正光盘中的数据以文件的形式存放在硬盘上，当然，每虚拟一个光盘都需要大量的硬盘空间。

使用虚拟光盘

　　1.“插入”光盘。在程序主界面的右边列表框中，单击选择虚拟的光盘，点工具栏上的“Insert”按钮(第一个按钮)，则该虚拟光盘就“插入”到Virtual Drive创建的默认虚拟光驱中。
　　2.回到“我的电脑”中，就可像平常使用物理光驱一样，使用虚拟的光驱了。注意，Virtual Drive在“我的电脑”文件夹中应有一个虚拟光驱的盘符。
　　3.“弹出”光盘。在程序主界面的左边列表框中，单击要“弹出”虚拟光盘的虚拟光驱，点工具栏上的“Eject”按钮(第二个按钮)，该虚拟光驱中的虚拟光盘即被“弹出”。
建立多个虚拟光驱
　　Virtual Drive 3.10允许最多建立23个虚拟光驱,每一个虚拟光驱中可以插入一张虚拟光盘并能同时运行。单击“File\Setup number of CD drives”，在弹出的对话框中可以设置想要的虚拟光驱数目，然后点“OK”重新启动电脑即可建立多个虚拟光驱。
　　在局域网中，可以用多个虚拟光驱来代替网络中昂贵的光盘阵列，使得只用一台物理光驱即可架构一个完整的光盘阵列，并且速度和稳定性有过之而无不及，当然，这需要Virtual Drive的网络版。

其它操作与设置

　　1．加入与删除虚拟光盘
　　●删除：单击“File\Delete VCD file”弹出删除选项对话框，选择第一项将彻底删除所选虚拟光盘；选择第二项则只删除光盘图标，不删除文件；若选择第三项，将不做任何删除动作。
　　●加入：这是针对已经有了虚拟光盘文件的操作，单击“File\Add VCD file”，在弹出的浏览框中选择要加入的文件即可。
　　2．禁用虚拟光驱
　　有时由于驱动程序间的冲突或某些设置方面的原因，造成加载虚拟光驱后，物理光驱使用不正常，这时只能暂时禁用虚拟光驱。打开“控制面板”，双击“系统”，单击“设备管理”，双击“CD－ROM控制器”下的“Virtual Drive”，选中“在此硬件配置文件中禁用”前的复选框，最后单击“确定”，无须重新启动电脑即屏蔽掉了虚拟光驱。以后当要重新起用虚拟光驱时再如法炮制即可。
　　3．查看虚拟光盘信息
　　在虚拟光盘名上单击右键，弹出“VCD Properties”对话框，显示有该光盘的存放位置、描述、默认对应的虚拟光驱、容量及创建时间等信息。通过此对话框还可修改光盘的描述和默认对应的虚拟驱动器(如果设置了多个虚拟驱动器)。
安装中应注意的问题
　　Virtual Drive“骗”操作系统，使操作系统将虚拟的光驱当作实实在在的物理光驱，因此安装中可能出现类似添加新硬件时的问题。既然是“硬件”问题，也可用解决硬件问题的方法解决。
　　1.如果“我的电脑”中只有一个光驱图标，多半该图标是虚拟光驱图标，而原有物理光驱丢失。用文本编辑器在“Config.sys”文件中加入“LASTDRIVE=Z”一行即可解决。
　　2.一般情况下，系统启动时，虚拟光驱的驱动程序会首先被加载，如果原物理光驱的盘符采用由系统自动分配的方法，则虚拟光驱会占用物理光驱的盘符，而物理光驱将分配新盘符。这将导致原来安装的一些程序运行时报告找不到光盘。这时需要为虚拟光驱手工设置盘符，打开“控制面板”，双击“系统”，点“设备管理器”，点“CDROM”，双击“Far Stone Virtual Drive”，在“设置”选项卡中更改“开始驱动器号”和“最后驱动器号”两项(如改成“Z”)，确定后重启计算机，则物理光驱的原盘符已恢复。用上述方法也可将物理光驱的盘符锁定。
　　3.如果物理光驱仍旧在用DOS下的16位驱动程序，安装虚拟光驱时会出问题，须把16位驱动程序换成Win95/98下的32位驱动程序。方法是用文本编辑器打开“config.sys”文件，将“DEVICE=×××SYS/D：MSCD00×”加载光驱驱动程序项目删除，再重新启动电脑即可。
