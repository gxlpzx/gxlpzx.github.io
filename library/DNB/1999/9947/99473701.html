《 编程短训班（第二期）、硬件短训班（第三期）招生简章 》

一、编程短训班(VB、VFP、VC、Delphi)

　　(一) VB编程班

　　1． 培训内容:Visual Basic 6.0的操作使用；Visual Basic 6.0的程序设计。
　　2． 教材及资料
　　《深入浅出Visual Basic 6.0中文版》 该教材简单扼要地介绍了Visual Basic 6.0程序设计。全书共24章，每章仅花1小时即可读完。
　　《Visual Basic 6.0实用编程技术》：该教材分为入门篇、提高篇、技巧篇三个主要部分，分别对Visual Basic的基本操作、数据库环境及实用编程技巧进行了详细介绍。
　　辅导教材：该教材介绍了Visual Basic 6.0的学习重点和编程技巧。
　　配套光盘：由北京育碟苑公司制作的《V Basic6.0编程经典》教学光盘（双CD）。

　　(二) VC编程班

　　1． 培训内容: Visual C＋＋ 6.0的操作使用；Visual C＋＋ 6.0的程序设计。
　　2． 教材及资料
　　《Visual C＋＋ 新起点6.0实用教程》：该教材系统介绍了Visual C＋＋ 6.0中文版的基本内容，由浅入深地讲述了数据库操作、创建以及开发数据库应用系统的知识。
　　《Visual C＋＋ 6.0编程实例与技巧》：该教材详尽介绍了Visual C＋＋ 6.0中文版的程序设计方法与技巧，内容详实丰富，适合数据库开发人员和广大编程爱好者使用。
　　辅导教材：该教材介绍了学习Visual C＋＋ 6.0的学习重点和编程技巧。
　　配套光盘：由北京育碟苑公司制作的《V C＋＋ 6.0编程经典》教学光盘（双CD）。

　　(三) VFP编程班

　　1． 培训内容: Visual FoxPro 6.0的操作使用；Visual FoxPro 6.0的程序设计
　　2． 教材及资料
　　《中文Visual FoxPro 新起点6.0实用教程》：该教材系统介绍了Visual FoxPro 6.0中文版的基本内容，由浅入深地讲述了数据库操作、创建以 及开发数据库应用系统的知识。
　　《Visual FoxPro 6.0开发宝典》：该教材详尽介绍了Visual FoxPro 6.0中文版的程序设计方法与技巧，内容详实丰富，适合数据库开发人员和广大编程爱好者使用。
　　辅导教材：该教材介绍了学习Visual FoxPro 6.0的学习重点和编程技巧。
　　 配套光盘：由北京育碟苑公司制作的《V FoxPro 6.0编程经典》教学光盘（双CD）。

　　(四) Delphi编程班

　　1． 培训内容: Delphi 4.0的操作使用；Delphi 4.0的程序设计。
　　2． 教材及资料
　　《Delphi 新起点4.0实用教程》：该教材系统介绍了Delphi 4.0中文版的基本内容，由浅入深地讲述了数据库操作、创建以及开发数据库应用系统的知识。
　　《Delphi4.0程序开发应用指南》：该教材详尽介绍了Delphi 4.0中文版的程序设计方法与技巧，内容详实丰富，适合数据库开发人员和广大编程爱好者使用。
　　辅导教材：该教材介绍了学习Delphi 4.0的学习重点和编程技巧。
　　配套光盘：由北京育碟苑公司制作的《Delphi4.0编程经典》教学光盘（双CD）。
　　每个编程班均特别赠送:由电脑报社策划出版的《'99软件精品屋》(含CD)，内容详实、新颖，是掌握最新软件信息的必备工具书。

二、硬件维修短训班

　　1． 培训内容
　　微机系统的主要配置原理和故障检修与维护技巧、选购和组装技术。
　　2． 教材及资料
　　《PC DIY用电脑》（含CD）：该教材是全国第一本硬件光盘杂志，由电脑报评测实验室制作,详细地介绍了电脑硬件基本知识和最新发展状况，同时在配套光盘中配以大量的硬件图片以及细致的讲解，使你对电脑的硬件有非常清楚的认识。 　　《电脑选购与组装》：该教材的编写目的在于一步一步引导读者进行电脑配件(主板、CPU、内存、显卡、调制解调器、打印机等)的选购与组装；同时该书还兼顾电脑硬件的手册功能，是电脑硬件发烧友的必备工具书。
　　《电脑硬道理之装机圣手》：本书详细介绍了当今电脑各种配件的知识和自己动手组装一台电脑的全过程。全书共分17章，内容详实、丰富，图片精美、制作装潢考究，代表了《电脑报》的最新出版水平。
　　特别赠送:由电脑报社策划出版的《'99硬件精品屋》(含CD)，内容详实、新颖，是掌握最新硬件信息的必备工具书。
