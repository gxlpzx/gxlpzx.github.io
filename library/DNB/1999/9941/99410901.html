《 歪歪线报——2000的故事 》

　　国庆晃晃悠悠的就过了，歪歪在家里看书看得颇有“家中方七日”的感觉，要不是突突催稿，歪歪是绝对不会看日历的。一看日历吓了一跳，以为很遥远的2000年马上就要到了，呵呵，歪歪终于要成为跨世纪的人才了（贝贝：我也是个跨世纪的婴儿了）。2000年意味着什么？一个文明的历程，一部人类奋斗的史诗，一个谁也不知道人类将会怎样的未来。离2000年还有2个多月，人类该做的事情还很多，回顾过去，展望未来。对付Y2K问题无疑是最重要的一件事情。这是一种悲哀和警钟，当人类在迎接一个新世纪的来临时做的第一件事便是解除威胁。所以，希望到3000年的时候，人类不是以原始人的姿态来迎接。无论对未来是乐观还是悲观，IT业必然成为下个世纪的支柱产业，而软件业也必然成为整个IT业的支撑点。

关注的焦点

　　在这个越来越商业化的世界里，2000年绝对是一个良好的商业宣传机会。大批软件以2000年为契机开始了新版本的发布和推广，其中很多软件当然也将在往后的几年中对我们产生深远的影响。Windows 2000是我们最为关注的一个操作系统，回顾Windows的发展历史，Winodws系列一直走的是两条主线：Windows 9x和Windows NT。众所周知，微软的操作系统最早是DOS，接下来是Windows 3.x。Windows 3.x并不是一个真正的操作系统，只是一个构建在DOS操作系统下的图形界面。Windows 9x是一个新的操作系统，可以调度进程，管理设备。但是由于要和数以万记的DOS程序兼容的原因，它的启动还是要依靠一个16位内核去引导，然后加载32位的模块去管理计算机。Windows 9x因为兼容了16位的模块，所以只是一个准32位的平台。
　　对企业级用户来讲，Windows 9x因为兼容了DOS,其稳定性和安全性得不到保证。微软为了占领UNIX的服务器操作系统市场，在开发Windows 9x的同时一直在从事纯32位平台的Windows NT开发，由于它的目的性非常明确，放弃了Windows 9x的种种易用性和兼容性，对一般用户来说是相当的难以使用。也正是这样的原因，Windows 98的FAT32和Windows NT的NTFS文件格式互不兼容。微软在1997年开始投入巨资进行开发，希望可以开发出一个具有Windows 9x的兼容性和Windows NT的稳定性的操作系统。 Windows 2000就是这样的一个产品， Windows 2000更多的是继承了NT的特性，Windows 2000 家族中包括专业版、服务器版、高级服务器版以及数据中心版。专业版直接取代了 Windows NT 4.0 Workstation，其设计对象主要是独立PC和网络客户PC，它拥有绝大部分的基本功能，但是在网络服务器方面有所局限，该版本的产品支持双CPU；服务器版中包括专业版的所有功能，也可以支持双CPU并且添加了一个Active Directory控件和一个完整版的Microsoft IIS 5.0；高级服务器版可以支持四个CPU，64GB的RAM，而且提供加载平衡和集群功能；数据中心版可以支持16个CPU以及高级集群功能。为了要解决“蓝屏”问题，Windows 2000版中添加了几个不同的功能，包括核心模式写保护、代码标记技术、库标记。核心模式写保护禁止任何设备驱动器或程序向核心使用的内存或是操作系统标记为写保护的内存进行写操作。代码标记技术处理系统文件保护，并且在将文件写入系统子目录之前使用一个数据标记来确认文件的来源。库标记可以允许设备驱动器供应商使用内存定位来定位在特殊的库中，而不是使用共享的系统内存库。现在市面上已经有了Windows 2000的测试版，歪歪已经看到了Windows 2000中文版模样。 在使用中，歪歪发现Windows 2000有着极好的稳定性（这是由于采用了Windows NT5.0的内核）,不过，令歪歪担心的是, 由于Windows 2000采用NT的核心技术，以前基于Windows9x的驱动程序和一些Windows程序再也不能像我们从Windows95到Windows98升级那样容易，歪歪在安装Windows 2000测试版的时候，系统告诉我需要到硬件厂商主页寻找新的驱动程序，当然在硬件厂商的主页上还没有，从某种程度上来说，Windows 2000的快速普及还需要第三方驱动程序和第三方软件的新版本的快速推出。歪歪最近在网上还听到一个不太确切的消息，微软在秘密开发基于Window9X内核的最后一个版本Windows Millennium（千年盛世），不过歪歪可不敢肯定。歪歪最后要说的是Windows 2000 Beta 3版的最低系统要求： 32MB RAM (推荐使用64MB), 650MB 硬盘空间。
　　Office2000是另一个令人不得不关注的东东，关于Office2000的具体细节，《电脑报》早有了大量的介绍，歪歪就不多说了。下面谈谈歪歪的感受吧。Office2000 的界面改变是一个软件界面变化的信号，菜单和工具栏完全动态化，第一次引入了微软的大型软件，根据以往的惯例，这种风格将会很快成为Windows软件的一种标准。据说，微软有一个易用性研究室，那里汇集了社会学家、心理学家、美术家等优秀的非计算机人才，我想这便是他们的杰作。（不过，歪歪好像是另类，不太喜欢他们的研究成果！如果全部的软件都变成了这样，那歪歪不是又被强迫接受了一种标准吗?）另外歪歪讨厌的是Accsess 2000的变化太大了，而且与旧的文件格式不兼容。当然歪歪还是比较喜欢FrontPage 2000的,歪歪的女朋友用它，只花了三个星期就从完全不懂做主页变成能独立完成http://perfume6.yeah.net主页，FrontPage 2000真是非常容易上手。当然，还有PhotoDraw 2000，它可以让歪歪这等美术弱智快速完成该网站的所有图形，微软第一次出图像处理软件就如此出色，应该表扬！
　　这期贝贝介绍的 Norton Utilities 2000也是借着2000年宣传自己的软件，不过作为系统软件Norton Utilities 2000可以说是最棒的了。Norton Utilities 2000是对电脑出现的各种软件问题的一个全面解决方案。如何解决Windows 9X的非法操作，大家一定要看看贝贝本期做的Norton Utilities 2000使用教程。Norton CleanSweep 2000 是另一个叫2000的新版本，这应该是反安装软件中最好的了。我们都知道，Windows9X的反安装并不能清除注册表和系统的所有垃圾文件，长时间的装软件、反安装软件会让系统变得越来越慢（突突的电脑就是这样由最快的变成最慢的）。Norton CleanSweep 2000可以让软件更干净和安全地删除，当然对注册表的清理也做得非常好。新版本加强了对Internet上带来的垃圾文件的清理。

国产软件的期待

　　说完了国外的软件，来说说国内值得期待的一个叫2000的软件吧：金山词霸2000。毋庸置疑，金山词霸Ⅲ是1998年一个非常成功的国产软件，现在马上要推出的后续版本也要借借2000年的势头，让我们来看看它有什么新的功能:1.据说采用了TTS(test to speak文本到语音)全程化语音技术，能准确发音所有英文单词及短语，可任意进行整句、整段乃至整篇文章的中英文流畅朗读。2.即指即译：新增鼠标右键选中翻译的模式，当鼠标指向需要翻译的词的时候，按鼠标右键进行在线翻译。3.常用词汇的真人语音全部重新录制，语音更清晰。4.增补并修正了6.5万条最新词汇、5万条实用例句、十多万单词的音标，绝大部分单词都有音标。囊括了金山词霸Ⅲ标准版、通译科技版和企业经贸版中的所有词汇，增补了两百万条科技类专业词库。金山词霸2000真正令歪歪心动的是词库大了很多，不过如果能把自己不知道的单词加入到一个文本文件，以供以后学习就好了。歪歪还没有见到这个软件，但是特地给大家拿了一幅界面图来，大家看看吧！歪歪和大家一样期待着这款国产软件的上市。

黑客的2000

　　最后的2000软件是一个黑客软件，一个黑客群体在美国的黑客年会DefCon VI上发布了一个黑客软件——Back Orifice 2000（以下简称BO2K），并把它放在网上供人下载。这是一个针对Windows95的黑客软件，可以攻击载网上的任意一台装有Windows95的计算机。另一个明显的变化是，CDC不将Back Orifice 2000定位于违法的“破解”程序。相反，该组织正把它作为一个成熟的合法网络管理工具进行推广。在这一点上，Back Orifice可能效仿了另一个远程控制程序NetBus的做法，该程序同样在去年从一个破解工具走上了“正道”。也许，正是因为有了这些黑客的存在，我们的2000年才会变得让人担心。
