专家坐堂

　　系统不能正常启动怎么办
　　广安读者程新礼来信问：
　　我有一台奔腾133组装机，操作系统DOS 6.22和Windows 95。前不久，因误删一大堆不知名的文件，导致计算机出现故障，蓝天白云的启动画面出现仅三四秒后，计算机自动进入DOS状态(以往计算机直接进入Windows 95界面)，而且这个DOS好像不是6.22，版本信息为“Windows 95，Version 4.00 950”)。我在C：\>后键入“WIN”与“EXIT”均无用，出现“Volume in driver C has no label”，又在C：\>Windows\后键入“WIN”，显示“Himem.sys is missing”，后又在启动时按F8键，进入多重引导设置，可无论哪种方式启动，均无法进入Windows界面，总是出现DOS状态。而且此时E盘(光驱盘符)无法成为当前目录，显示“Invalid drive Specification”导致无法重新安装PWIN95(可计算机启动时显示“Found CDROM”，而且光驱驱动程序并未丢失，我又重新安装了一次驱动程序，同样无用)。更糟的是我在DOS目录下发现一个Himem.sys文件，将其拷贝至Windows目录中，却丝毫不起作用。又将原有的Windows 95备份文件夹Lns 95内所有文件拷贝至Windows中，结果令人沮丧。请问：
　　1.为何无法进入Windows界面，应如何解决?
　　2.为何E盘无效，应如何解决?
　　3.C盘无label是什么意思，应如何解决?
　　答：从来信所说的情况看，你一定是误删了Windows 95的系统文件。Windows 95和Windows 3.1不同，它的系统文件不仅在Windows 95专门的子目录下，而且在根目录下面也有一些与启动有关的系统文件。如果不慎删除，必然导致不能正常启动。来信所说进入DOS的版本信息为“Windows 95，Version 4.00 950”，这是DOS7.0，也是Windows 95的支撑环境。来信所说的E盘无效，也是由于不能正常启动Windows 95所致。如果要在DOS7.0下面使用CD－ROM(你的E盘)，必须要在DOS7.0环境下的Config.sys和Autoexec.bat文件中写入使用CD－ROM的相应语句(请参阅DOS使用手册)。C盘无label是表示你的C盘在作格式化时没有给它取名字，你可以用label命令为它重新命名。
　　对于上述故障，最好的办法是重新安装Windows 95。如果要在DOS7.0下面安装，请先修改DOS7.0下的Config.sys和Autoexce.bat文件，使之能够识别CD－ROM。如果要在DOS6.22下面安装Windows 95，可在机器启动时按下F4强行进入DOS6.22，然后再进行安装。当然也要你的DOS6.22能够识别CD－ROM，如果不能识别，也请你在DOS6.22下面修改相应的Config.sys和Autoexec.bat文件。
　　怎样在Windows NT 4.0中安装AGP显卡
　　河南读者张志刚来信问：
　　我的机器配置为PⅡ233、Intel 440LX主板、S680AGP显卡，安装了Windows NT 4.0 Server，按显卡的安装说明安装驱动程序后，NT无论如何都不认这个显卡，总说显卡不对，是否因为NT比AGP早，不支持AGP?如何才能使NT与AGP相认?请专家予以解答。
　　答：AGP是直接连结主内存与显示卡的接口。采用AGP接口的系统不仅需要硬件(控制芯片组、主板)支持，也需要操作系统的支持。例如AGP允许3D纹理数据不存入拥挤的帧缓冲区(即图形控制器内存)，而将其存入系统内存，这种允许显示卡直接操作主存的技术称为DIME(Direct Memory Excute)。这种DIME技术就要求操作系统的支持。
　　早期的Windows 95和Windows NT 4.0的确不支持AGP功能，因此不能正确使用AGP，即使你安装相关的驱动程序也不起作用。新的Windows 95及其以后的版本、Windows NT 5.0及其以后的版本都支持AGP，只要直接安装相关的驱动程序就可以了。对于旧的Windows 95和Windows NT 4.0版本如果要使用AGP显示卡，应该按照以下三步进行软件安装。
　　1)升级操作系统，即安装操作系统的补丁程序；
　　2)安装显示卡驱动程序和Direct X；
　　3)安装主板芯片组的AGP Driver。
　　这些补丁和驱动程序在购买AGP卡时附带的安装盘中一般都有。详细的安装方法请你参照《电脑报》1998年第36期或《1998年电脑报合订本下册》第110页《AGP显卡安装手记》一文。
　　Windows 95为什么不能正常工作
　　四平读者姚永恒来信问：
　　我有一台586多媒体电脑，配置为联讯exp8661、Pentium 100 CPU、8M DRAM、1G硬盘、PCI显卡、配有T＆W声卡、解压卡、8速索尼光驱，在DOS下和Windows 3.X下运行软件都正常，但是安装Windows 95后速度非常慢，光标停滞非常明显(但在安全模式下工作光标停滞现象没有)，时而有死机现象，速度不如一台386，并且运行应用程序时有致命错误：如看VCD时有：“致命的异常错误0E发生在VXD ESDI－506(01)＋00000D3E的0028：C0FDD34A，当前的应用程序将被终止……”所以只好在Windows 3.2下工作，这种现象不知发生在CMOS的设置上，还是我的主板有问题。请专家予以解答。
　　答：出现来信所说的情况可能有以下几种原因：病毒、系统缺陷、系统设置不当、设备驱动程序安装错误、硬件故障。可以按照以下步骤检查：
　　1.检查病毒，建议用较新版本的杀毒软件，如KV300X＋＋检查有无病毒，旧版本不一定能够发现新病毒。
　　2.在启动菜单中选择第二项Logged(\BOOTLOG.TXT)方式启动系统。按照此方式启动系统时，将建立一个注册文件BOOTLOG.TXT，存放在C盘根目录下面。该文件是Windows 95的启动记录，其中装有启动过程的信息：它是文本文件，可以在DOS下用TYPE、EDIT等命令，或者在Windows 95下用记事本、写字板查看。如果系统本身有故障，该文件中将作出记录，因此可以发现问题之所在。请注意该文件是隐含文件，须修改文件属性才能在目录中列出。
　　3.系统设置不当主要可能是硬盘、内存、L2Cache和芯片级的CMOS参数设置不当，请你按照安全设置重新设置CMOS参数，看看能否解决问题。然后再逐个修改参数以达到最佳值。
　　4.安装Windows 95之后，应该安装显卡、声卡、解压卡等外设的Windows 95版本的驱动程序，如果没有安装，或者安装错误引起冲突也有可能出现不能正常工作和死机故障。可进入“控制面板系统设备管理”，逐个检查有无问题。
　　5.硬件故障可能是物理故障，也可能是接触不良，请你检查硬件有无问题。
　　重装Windows 95后为什么没有声音
　　读者刘行舟来信问：
　　1.我的机器发生故障，重装Windows 95后，无声音，打开“系统”—“设备管理”—“声音视频和游戏控制器”，其中有三项
　　“MPU－401 COMPATIBLE
　　YAMAHA OPL3－SAX GAME PORT
　　YAMAHA OPL3－SAX SOUND SYSTEM”
　　第三项前有一黄色惊叹号，删除刷新后，右下角出现喇叭图标，有了声音，但关机后重启动又无喇叭图标，无声音。之后，每次开机都要删除刷新后才有声音，不知有何解决方法?
　　2.重装Windows 95后，原先的五笔输入法丢失，在输入法“添加”中也没找到“五笔输入法”，如何找回?
　　答：第一个问题估计是重装Windows 95之后没有安装声卡的驱动程序，或者安装不正确。当驱动程序安装不当或者发生资源冲突时都会给出黄色惊叹号警告。安装驱动程序一般应该采用该声卡所配的驱动程序，并检查IRQ、DAM和IO地址有无冲突。
　　第二个问题是由于Windows 95本身没有配备五笔输入法，重新安装Windows 95后，原来安装的五笔输入法当然丢失。你只能重新安装五笔输入法。目前市面上出售的光盘中，可以买到五笔输入法软件，买来后在Windows 95下安装即可。
　　硬盘不能工作的原因和处理方法
　　重庆读者何林来信问：
　　我的电脑无法硬盘启动,检查CMOS参数没有问题。用软盘启动后可转入C：\提示符，但是不能对硬盘进行任何操作。是什么原因，如何处理?
　　答：用软盘启动后可转入C：\提示符，说明系统是可以识别硬盘驱动器的，而硬盘不仅不能引导系统，而且也不能进行其它操作，说明故障原因可能是硬盘的主引导区、分区表遭到破坏，使得硬盘不能正常工作。一般的处理办法是从软盘引导系统，用FDISK命令对硬盘重新分区，然后再作高级格式化，重装整个系统。由于可能是病毒破坏系统，建议使用FORMAT命令加U参数。
　　如何进行硬盘对硬盘的拷贝
　　江苏宜兴读者蒋永刚来信问：
　　1.硬盘对硬盘拷贝应该如何进行?
　　2.系统启动时，在出现Windows 95画面之后就出现“你可以安全关闭计算机了”。这是什么原因?如何处理?
　　答：来信的第一个问题没有说明源盘和目标硬盘是否在同一个微机中，因为不同的情况下硬盘和硬盘之间的拷贝方法不同。下面把几种情况都介绍一下：
　　1.如果源硬盘和目标硬盘在同一个微机中(即微机中配有双硬盘)，这种情况的拷贝十分简单，可以直接进行拷贝。
　　2.如果源硬盘和目标硬盘分别在两台微机中，可以用双机通信的方法连接两台微机，再进行拷贝。这需要用一条电缆把两台微机的并口(或者串口)连接起来，并通过专门的软件来进行拷贝。这种方法在《电脑报》曾经多次介绍过，并且介绍过通信电缆的制作方法。不过这种办法拷贝的速度较慢，对于大量数据的拷贝需要很多时间。
　　3.如果两台机器都有网卡，可以连接成一个对等网，这样拷贝既方便又快捷。
　　4.如果不具备上述条件，可按照以下办法进行：
　　1把目标硬盘从微机上取下，接入源硬盘微机中。
　　2注意：如果把目标盘与源盘连接在同一个IDE口时，必须保证两个盘的工作方式一个为主方式，另一个为从方式(一般说来源盘都是主方式，因此要把目标盘设置为从方式)。
　　3开机进入BIOS设置，对接入的目标硬盘作参数设置。
　　4引导系统后，检查目标盘已经存在，按照常规方法即可进行拷贝。
　　5拷贝完毕，关机后卸下目标硬盘；再开机修改BIOS设置。
　　6把目标硬盘设置为原来的工作方式，装入原来的机器。
　　5.上面谈了硬盘拷贝的硬件连接。现在再谈谈用什么软件进行拷贝。如果是数据文件，虽然可以用COPY、XCOPY或者在Windows下用鼠标进行拷贝，但是速度较慢。因此建议使用专门的硬盘复制软件，目前较好的是Ghost。在《电脑报配套光盘》98.3期上有详细介绍。而对于程序文件，特别是Windows 95／98以及许多Windows的应用软件，用上述方法拷贝后却不一定能正常工作。因为这些操作系统在安装时要识别和登录硬件系统，应用软件在安装时也要登录或者修改注册表等，因此拷贝的硬盘装入原来的微机中不能保证正常使用。一般说来Windows 95／98和一些大型的应用软件应该通过安装而不是简单的拷贝实现。
　　第二个问题一般是由于Windows 95的启动文件受到破坏造成的。用于Windows 95启动的文件有多个，要找到有缺陷的文件比较困难。最简单的方法是重新安装Windows 95。为了避免病毒破坏，应该先杀毒后再安装Windows 95系统。
　　硬盘为什么不能启动Windows 95
　　湖南张家界读者王晓来信问：
　　一块Seagate硬盘1.2G，1996年出品。该硬盘安装Windows 95的过程完全正常，但是安装之后不能正常启动Windows 95。该硬盘在DOS下可以完全正常工作。经检查，安装Windows 95的光盘是正常的(在其它硬盘上安装后可正常工作)，检查CMOS参数无误，检查硬盘没有病毒，用各种工具检查该硬盘也没有坏块。我已经在该硬盘上反复安装了三次Windows 95，故障依然。是什么原因，请专家指教。
　　答：如来信所说，已经进行了各种检查，没有发现硬盘的问题所在。笔者认为，可能问题不在硬盘，而在其它部分。笔者一位朋友的微机有类似的情况。这台微机在DOS下能够正常工作，而安装Windows 95后无论如何不能正常启动工作。反复杀毒、重新安装、检查硬盘均不能解决。后来想到该机CPU风扇曾经停转，可能伤及了CPU。于是把CPU降频使用，一切正常。究其原因，DOS对于微机的要求很低，而Windows 95则对整个系统要求很高。在CPU或者其它部件有缺陷时，Windows 95是不能正常工作的。
　　笔者建议你把硬盘换到其它机器上一试，看看能否正常工作。再用排除法逐个检查CPU、主板等部件或许可以找到原因。
