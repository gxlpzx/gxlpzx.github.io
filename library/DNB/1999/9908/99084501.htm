纯 32 位 模 拟 器 Mame32

　　近一段时间，模拟器在电脑玩家当中成了一大热点，想想看，PS＋SFC＋MD＋FC＋GB＋街机……这样一部在三年前还被某些报刊认为是愚人节恶作剧的“超级游戏机”，此刻就躺在大家面前；那些过去只能在商场柜台上饱饱眼福的成百上千种游戏，而今尽可一一尝试，心中不亦乐乎！
　　在众多模拟器中，最受格斗游戏迷欢迎的就是Neo－Geo机皇模拟器了，尽管这种游戏机在国内极少见，但它的游戏却随处可以看到，PS、SS甚至PC上，以及为数众多的大型街机里，《拳皇》系列、《侍魂》系列、《双截龙》系列等，这些我们熟悉得不能再熟悉的名字，给Neo－Geo带来格斗专家的美誉。现在，只要找一个Neo－Geo模拟器，便能把“机皇”请到自己的PC中，让咱们也过一回格斗专家的瘾。
　　Neo－Geo模拟器发展到今天，比较成熟的有两种：Neorage和Mame，其中，Neorage的画面模拟度差不多达到了尽善尽美的地步，且其它指标如流畅度、操作感等等也无可挑剔。唯一的遗憾是不支持音效，这对格斗迷来说，一个没有声音的格斗游戏简直就像一道没放盐的高档菜肴那样索然无味（注：据说Neorage的最新版本将支持声音，但未加证实），因此，我们又把目光投向了Mame。Mame在画面、流畅等方面丝毫不亚于Neorage，最重要的是它支持声音！虽然与游戏机相比音效稍微有些失真（主要是ROM制作的原因，现在已经有许多游戏推出了声音修补ROM，加上后声音就正常），但毕竟可以接受了。
　　无论是Neorage还是Mame，一直是16位DOS模式程序，相对于其它纯32位模拟器，多少显得有点落伍。而且目前Neo－Geo模拟器需要很大的内存，多数情况下依靠虚拟内存才能工作，在Windows的DOS窗口下运行，还得经历编辑MS－DOS快捷方式，用繁琐的命令行方式执行等麻烦，载入ROM的速度更是对你耐心的一种考验。还好现在不必再忍了，因为最近Mame推出了全新版本——Mame32，一个真正在Win95/98下运行的纯32位版本Neo－Geo模拟器。
　　Mame32无需安装，将ZIP包内的文件解压到一个文件夹中，然后双击Mame32.exe即可运行，当然可以从开始菜单栏的“运行”中以命令行方式运行Mame，但笔者不推荐这样作，除非你运行时遇到了问题，必须加某个参数。
　　Mame32的主界面乍一看很像Win95的资源管理器，左边是文件夹，右面是文件，其实就是将所有的Neo－Geo游戏与Mame游戏分门别类归纳了起来。
　　看到左侧文件夹中有一个“Neo－Geo”吗？它里面足足有101个游戏。别忙，看清楚上面都画着红红的禁止标志，说明Mame32在当前文件夹中并没有找到这些ROMS，那该从哪里得到ROM呢？这就得自己想办法了。而且还必须注意Mame运行任何一个ROM时都要求ROM文件夹或ZIP包中包括neo－geo.rom、ng－sfix.rom和ng－sm1.rom这三个游戏机版的ROM，这和Neorage有点不同。一般情况下，不管你的ROM来源怎样，那些文件夹或ZIP包中并不包括这三个ROM，我们只好一一手工添加进去，才能开始游戏。
　　从理论上来说，将ROM文件夹放在Mame32所在文件夹内，然后我们只需在列表中找到这个游戏的名字双击即可……且慢?/颐亲詈孟壬柚靡幌拢允筂ame在优化状态下运行。
　　在Mame32的Option选项下有四项，其中Large Screen Shots的作用是允许Alt＋R键抓图；Game List Font是游戏列表的字体；Directories是各种文件存放的路径，很容易看懂，根据自己的情况选就是了；关键的Default Options才是我们真正要找的。
　　打开Default Option的下级选项，分为两部分，一是Display（显示），一是Sound/Input（声音/输入设备）。
　　Display（显示）：
　　Mode：选择是在全屏或窗口状态下游戏，推荐全屏。
　　Size：指定游戏画面的大小，你可以从列表中选择，推荐选Auto Size，这样就不用咱们费心了。
　　Color：选择色彩，如果你的配置不错，可以选16位色，否则还是256色将就着吧。
　　Skip Lines和Cols：是行列扫描线的跳动值，建议默认0。
　　Gamma：游戏画面色彩的RGB值，默认为1.00，如果色彩有问题可以试着微调一下。
　　Brightness：亮度就不必调了吧，默认100％，如果太亮嫌刺眼可以调低些。
　　Use Directdraw：如果有DirectX的话还是允许吧。
　　Horizontal Scan Lines：模仿电视机的水平抽条显示模式。
　　Vertical Scan Lines：垂直抽条模式。
　　Anti－Alias反别名：建议开启，可以防止你将ROM文件夹搞错。
　　Translucency：半透明模式，有高配置的玩家建议开启，可使画面更漂亮。
　　Beam：调整光线，默认1.00。
　　Flicker：调整闪烁频率，默认0。
　　Frame Skipping：调整跳帧数。
　　Automatic Frame Skip：选项自动调整，一般不可用。
　　Rotation：旋转，可选向左/向右（顺/逆时针），也许是为竖版飞行游戏准备的，默认不旋转。
　　Flip horizontally：调整水平翻转。
　　Flip vertically：调整垂直翻转，以上两项没有特殊要求一般不选。
　　Sound/Input（声音/输入设备）
　　Sound System（声音系统）：如果安装了DirectX则默认DirectSound。
　　Audio Device（音频设备）：默认DirectSound。
　　Sample Rate（采样频率）：有11025Hz、22050Hz、44100Hz三种选择，频率越高音质越好，当然代价是……
　　Sample Bits：选择使用8位声音还是16位声音，以上两项一般选8Bit效果就不错了。
　　Stereo：是否采用立体声。
　　FM Synthesis For YM3812：雅马哈3812声卡的调频模拟。
　　Direct Input：选择输入设备，可以单独或同时选择键盘和游戏杆。
　　Use Joystick：使用游戏杆。
　　Trackball/Spin Inputs：在这项内可以选择是用鼠标、轨迹球或用游戏杆输入。
　　其它
　　Enable Game Cheats：是否允许作弊。
　　Auto－Pause：是否允许自动暂停。
　　Disable MMX：禁止MMX，如果你的CPU支持MMX的话，就没必要选这项和自己过不去。
　　Error Log：错误跟踪记录，这个功能主要供开发者使用，和普通玩家没太大关系。设置完了，这下可以开玩了吧。先别急，把一个完整的ROM文件夹放到Mame32所在目录中（注意文件夹的名字一定要与游戏列表上的一致），然后用鼠标右键单击列表中这个游戏的名称，选Properties（或在文件菜单中选），你可以得到这个ROM的详细资料，并且可以进一步的进行某些设置，这些工作做完后，双击游戏名，OK！
　　尽管Mame32对键盘支持得很好，但笔者强烈建议大家使用手柄玩Neo－Geo的游戏，因为Neo－Geo主要是格斗游戏，而格斗游戏最大的乐趣在于使用组合键发出各种华丽眩目的超必杀技，使用键盘你就不要指望能来个什么“天霸封神斩”，或者“大蛇锥”、“八稚女”之类的东东了，可以说乐趣尽失。反正现在PC用的手柄便宜者仅需数十元，而且还可以把其他手柄改造成PC手柄，“格斗家”们应该装备上这件武器。
