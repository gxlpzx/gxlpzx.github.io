《 网络“瑞士军刀”——NetScanTools 》

　　瑞士以出产钟表闻名于世，而它的瑞士军刀也由于小巧实用、功能强大成为很多人居家旅行的必备之物。而笔者这里要介绍的NetScanTools则是一把功能强大的网络“瑞士军刀”，它的容量不大，却几乎包括了网络所有的查询、分析功能。
　　NetScanTools提供的主要功能如下：

　　1Name Server Lookup（域名转换）

　　Name Server Lookup提供了域名转换的功能，NetScanTools不仅可以利用DNS在域名和IP地址之间进行转化，而且还可以返回映射的IP地址。
　　Name Server Lookup共提供了以下两种使用方式：1简单查询（Simple Query），在进行简单查询时，只需要输入相应的域名或IP地址，就可以获得它们相互之间转化的结果；2高级查询（Adv Query），高级查询方式除了可以进行域名和IP地址之间的转换外，更可以获得详细信息（如域名的别名等），同时可以自己指定其他的DNS服务器来进行域名转换。

　　2TraceRoute

　　TraceRoute功能可以查询从使用者到连接站点之间的所经过的网络路径（包括各网关的信息），并可通过返回的各网关之间的传输时间信息，获得各段网络的拥挤情况。这个功能对使用者分析网络的“健康”状况以及了解网络的连接路由是很有帮助的。

　　3Finger

　　Finger是网络高级用户经常使用的一条命令，它的主要作用是用来查找并显示欲Finger的用户在主机上的连接情况。它的用法很简单，只要输入想Finger的用户的E－mail地址（完整的地址表示只查询单个用户，如果想查询多个用户，请使用下列格式：@xxx.com），按下Finger键就可以了。

　　4Ping

　　Ping是网络中使用最多的一条网络命令。利用它不仅可以获得动态域名的相应IP地址，而且还能从回应信息中获得当前网络的“健康”情况。NetScanTools的Ping所提供的信息是目前所有此方面软件中最多的。

　　5Whois

　　Whois也是网络使用者查询网络信息的一个工具。它的具体操作步骤是：用户向Whois资料服务器提出关于某网站的询问请求，然后由Whois资料服务器返回它所拥有的关于此网站的详细信息（包括公司名、拥有者姓名、联系方法、域名、IP地址等）。NetScanTools总共提供了isgate.is等17个Whois资料服务器地址供使用者查询，同时它的Whois功能采用了智能查询技术，优先查询离使用者近的Whois资料服务器以保证查询的速度以及准确率。更值得一提的是，NetScanTools支持防火墙的Whois服务功能。

　　6WinSock Info

　　此项功能能够为使用者提供正在使用的WinSock的详细信息，如WinSock的版本号、当前操作系统以及关于Socket各方面的信息。

　　7Netscanner

　　这是NetScanTools所提供的一项非常强大的功能，它可以让使用者指定一段想搜索的IP地址，然后由它来对这些地址进行测试搜索，从而找出其中正在使用的地址（利用相应的选项可以同时显示它的域名）和未用或不可用的地址。

　　8Daytime

　　Daytime可以显示使用者所连接的远程服务器上的系统时间，这对于了解服务器所在区域以及相应的工作状态是很有帮助的。

　　9Quote

　　Quote是类似于Cookie的小玩意，一些站点经常会利用Cookie来确认访问者的信息，而利用Quote可以让使用者获得连接网站的相应信息。

　　10Time Sync

　　不知大家注意没有，我们的计算机上的系统时间总是和北京时间有一定的差异，即使经过手工调整，在运行一段时间后，又会出现同样的现象。这主要是由于计算机硬件中的晶体振荡芯片在设计制作时受技术局限所造成的。NetScanTools提供的Time Sync功能不仅让使用者可以校对本地计算机系统的时间，而且可以做到让本地主机和远程主机时间同步。Time Sync共提供了tick.usno.navy.mil、tock.usno.navy.mil、wwvb.erg.sri.com等39个网络时间服务器，供使用者检查校对系统时间之用。

　　11TCP Term

　　NetScanTools 提供的TCP Term可以看作是个简易的Telnet 工具，利用它可以登录到使用者想要连接的主机。尽管TCP Term的操作和功能似乎简单了些，但它能让使用者选择登录主机时连接的方式（也就是不同的端口），从而能对确认E－mail账号、测试Finger 主机等功能有很好的辅助作用。

　　12Character Generator

　　Character Generator能为使用者提供当前网络连接的大致速度，供使用者分析、比较当前网络的繁忙程度。

　　13Echo

　　Echo的中文意思是“回音”，而在网络操作中则代表客户机和服务器之间的相互联系。当使用者使用Echo连接远程服务器并发送信息时，远程服务器将把使用者发出的信息全部返回给使用者。使用者可以比较发送信息和返回信息的一致性，获得自己和远程主机之间网络连接的顺畅情况。

　　14IDENT Server

　　IDENT Server通常和上面提到的Finger等连接功能有关，部分服务器在用户提出Finger等连接请求后会要求确认用户的身份，于是它们会向请求用户的主机发送信息来获得请求人的身份信息，而此时服务器和本地之间的联络将通过IDENT端口进行。IDENT Server正是为本地主机提供了和远程服务器进行这种双向交流的能力。

　　15Database Tests

　　它主要包括Socket Services Test和Socket Protocols Test这两项测试内容。其中前者测试Winsock的主机数据传输的执行性能；后者测试Winsock协议的数据传输的执行性能。
　　尽管Windows也提供诸如查询IP、TraceRoute等简单功能，但其“朴素”的DOS操作环境以及稀少（几乎没有）的查询说明，使一般的使用者很难理解其查询结果的意义。NetScanTools为每项功能的返回结果都给出了详细的说明，并可存储为文件或直接打印出来供使用者分析比较；更值得一提的是NetScanTools提供了关于防火墙的设置，使得那些局域网的用户也可以使用它。可以说NetScanTools所提供的这些功能，是所有同类软件中最多的。想更好地了解和使用网络吗，那么就快些来试试NetScanTools吧!
