实现MICROTEK图像扫描

　　随着计算机的日益普及，扫描仪已越来越多地被人们选作为方便的图像扫描工具和快捷的文本输入工具。在目前众多的扫描仪中，Microtek(全友)平台式图像扫描仪是质量优良、使用方便灵活、扫描图像清晰、应用广泛的一种家用扫描仪。
　　Microtek图像扫描仪配合ScanWizard扫描软件和尚书汉字识别软件，可以方便、高效、清晰地实现图像、文本的输入。扫描的文本可以以纯文本文件方式存储，用任何编辑工具都可进行编辑；扫描的图像则以位图(BMP)文件方式存储，并可用Windows的画笔工具进行编辑，也可放到WORD中进行图文混排。
　　购买Microtek图像扫描仪时，随机附有扫描软件和各种驱动程序的光盘及软盘，且有详细的说明文档。仔细地阅读说明文档，安装扫描仪驱动程序后，便可操作扫描仪进行工作了。
　　1.扫描仪与计算机的连接
　　Microtek ScanMaker V310家用平台式图像扫描仪没有提供SCSI接口卡，而是采用软件方法构造了一个虚拟的SCSI接口，只需将扫描仪连接到计算机的并行打印端口即可。
　　①连接。将25－25引脚的随机电缆线一端接到扫描仪的Port－A端口，另一端接到计算机的并行打印端口；然后再将打印机电缆线的一端连接扫描仪，另一端接打印机，即计算机的打印信息通过扫描仪的端口再到打印机。
　　②设置。为了安装扫描软件，使扫描仪能与计算机通信，先要对计算机的CMOS进行设置，主要是对“Onboard Parallel Mode”(并行模式)进行设置。现以目前使用最为普遍的AWARD BIOS为例，介绍其具体操作：开机→按Del键→选CHIPSET FEATURES SETUP项→选Onboard Parallel Mode项→用PgUp键或PgDn键选择EPP模式即可。
　　③安装软件。购机时随机附带了一张光盘，驱动程序及扫描软件均在这张盘上。将光盘插入光驱，在Windows 95环境下，点击“我的电脑”，再选Microtek图标，即出现如下的软件安装菜单，然后选择相应软件安装即可。
　　·安装Windows 95扫描驱动程序(安装Windows 95下的扫描驱动程序ScanWizard)
　　·安装Microtek ScanSuit扫描软件(该软件包括扫描、拷贝、传真/E－mail、OCR工具)
　　·安装Ulead PhotoAssisant(安装扫描辅助软件)
　　·安装Ulead IPhoto Express 1.0(安装32位图像处理软件)
　　·安装Ulead PhotoImpact 4.0(安装32位图像编辑软件)
　　·安装PaperMaster SEC 20(安装档案管理软件)
　　在安装扫描驱动程序要求选择安装接口卡时，应选择“EPP打印机并口”。软件安装完毕后，桌面上将出现“扫描仪检测”图标，点击该图标可检测扫描仪与计算机的连接。
　　2.扫描仪与计算机的通信检测
　　扫描仪与其它的外部设备不同，使用时一般需先打开扫描仪电源，再打开计算机的电源，计算机才能检测得到扫描仪。若在打开了扫描仪电源和启动了系统后，点击“扫描仪检测”图标，屏幕显示：“没有连接扫描仪”，就表示扫描仪没与计算机连接或不能与计算机通信。通过“控制面板/系统”，调出“系统/属性”窗口，选择“设备管理”，发现SCSI控制器条目为：
　　“!Microtek Parallel Port Host Adapter”，即扫描仪与别的设备发生了地址冲突。
　　扫描仪检测出现的常见故障有：
　　·扫描仪不能与计算机通信
　　·计算机在并口上连接着扫描仪的情况下，启动时需要很长一段时间
　　·在扫描仪连接着打印机的情况下，打印机或扫描仪不能正常工作
　　·计算机在启动过程中死机
　　对于上述故障，可采用如下方法来解决：
　　①检查
　　·检查并确认通讯电缆线可靠连接，扫描仪面板上绿色指示灯保持常亮；
　　·扫描仪背后的“Port A”口与计算机的打印口通过随机提供的白色电缆相连；
　　·若连有一台打印机，先去掉打印机连线，试一试计算机只连接扫描仪的情况；
　　·保证打开计算机之前打开扫描仪电源；
　　·如果以上的措施都告失败，就需要修改Windows 95的一些设定，使计算机与扫描仪能够通信。
　　②修改Microtek扫描仪并口电缆设定
　　如果遇到任何与扫描仪通信有关的问题，可尝试对系统的设置作更改或加入下面的一些参数：
　　参数1：/N /V /SF=5
　　参数2：/N /E
　　参数3：/N /V /SF=5 /U
　　可任意选择上面三条参数中的一条，并通过关机重新启动后来尝试。在Windows 95下修改Microtek并口电缆 的设定，按以下步骤执行：
　　·选择“开始/设置/控制面板”；
　　·在控制面板窗口中双击“系统”图标；
　　·选择上面的“设备管理”栏；
　　·找到SCSI控制器的条目并双击它，会看到“Microtek Parallel Port Host Adapter”；
　　·双击“Microtek Parallel Port Host Adapter”；
　　·在“Microtek Parallel Port Host Adapter”的属性窗口中选择设置栏；
　　·在适配器设置框中键入上面所提到的参数中的一条，并按确定按钮，关闭所有的对话框；
　　·当Windows询问“是否重新启动计算机”时，选择“否”(N)，再选择“开始/关闭计算机系统”，再关闭计算机电源开关，等待5秒钟后再打开计算机电源。如果扫描仪仍旧存在通信问题，则尝试用另外的两个参数。
　　③.通信检测经验
　　笔者按照上述方法连接扫描仪后，点击“检测扫描仪”图标，屏幕显示“没有连接扫描仪”，或者扫描仪指示灯熄灭，程序进入死循环。偶尔一次扫描仪连接和通信正常，可以进行扫描操作，但若关机后再启动，又出现“没有连接扫描仪”提示的故障。上面的三组参数都使用过，问题还是解决不了，最后只好把打印机电缆线从扫描仪上拔下，以后每次开机，扫描仪工作都正常了。另外有一种情况，就是不将打印机电缆线从扫描仪上拔下来，但使打印机电源打开，扫描仪也能正常工作，此时点击“检测扫描仪”图标，屏幕显示“扫描仪连接正常”，显示扫描仪型号为ScanMaker V310 at ID=5，接口卡型号为Smplscsi。通过“控制面板/系统”，调出“系统/属性”窗口，选择“设备管理”，发现如下设备信息：
　　“Microtek Scanners
　　Scanners 300A4
　　SCSI控制器
　　Microtek Parallel Port Host Adapter”
　　即扫描仪与别的设备没有冲突，可与计算机进行正常通信。从这一现象可知，Microtek图像扫描仪和ScanWizard扫描软件尚没有很好地解决由计算机的并行打印端口连接扫描仪然后通过扫描仪连接打印机的配合问题。也就是说，计算机的并行打印端口要么只连接扫描仪，要么只连接打印机，不能两者都连接；在两者都连接时必须保持打印机的电源是常打开的(不管是否需要做打印操作)，否则就会发生地址冲突的问题，扫描仪不能正常工作。
　　3.图像编辑的一则经验
　　用Ulead iPhoto Express 1.0扫描黑白图形时，往往需选择256级灰度模式，这样扫描出的图像清晰度高，但其背景不是纯白色(在白纸黑色图形时)，有一定灰度，这对于一般图像没什么问题，如果要想对图形作些编辑、剪裁操作，就会出现问题。因为删除、移动、剪裁后，会留下纯白色的背景，这就与整幅图的背景不协调了。笔者摸索了一个解决这个问题的方法，那就是在Windows 95附件的图画中，先作图像的编辑、剪裁处理，然后选取一块没有图像且色调相近的区域，将其复制到剪贴板，用粘贴的方法将纯白处覆盖掉，这样就可做到天衣无缝，整个图像的背景色彩完全(或基本)一致。
