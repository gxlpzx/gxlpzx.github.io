Socket 370 CPU安装设置

Socket 370 CPU 安 装 设 置

　　一、认识Socket 370赛扬处理器
　　Intel公司1月5日正式推出新型的赛扬处理器——Socket 370和支持它的440ZX芯片组。从本质上说，它仍然使用了Slot 1架构的赛扬内核，只不过采用了Socket架构的全新PPGA封装方式(见图1)，使之具有安装容易价格更低的优越性，在性能上和Slot 1架构的赛扬处理器没有区别。
　　可以看到PGA封装的Socket 370在外形上很像黑金刚的Pentium MMX CPU，但它的针脚麻点分布比黑金刚要多一圈，有370根的引脚，而黑金刚只有321脚，从而决定了Socket 370 CPU无法使用Socket 7的插槽。在它的表面除了标志的缺角外，在同侧另一角也缺一根针，可正确的安装而不致插错针脚。另外Socket 370内置了128KB的二级缓存，CPU顶部核心部分比黑金刚要大许多。在CPU引脚一面，标注 “Intel Celeron”表明它是一颗PⅡ 级赛扬处理器，其中包括产地、序号、生产日期等。
　　二、认识Socket 370主板
　　Socket 370主板在外观布局上和Slot 1主板很相似，理论上Slot 1主板支持的特性它都支持。图2所示为华硕的MEL－M Socket 370主板，采用Intel 440LX芯片组，内置YAMAHA声音芯片。
　　芯片组针对Socket 370处理器，Intel推出了相应的440ZX支持芯片组，有440ZX－66和440ZX两种，后者以440BX为核心，支持100MHz外部频率，而前者只支持66MHz。由于性能和设计制造工艺相近，许多厂商只要对原有的440LX和440BX芯片组的主板稍加改动，把Slot 1插槽替换成Socket 370插槽即可完成产品换代，因而现在多数Socket 370主板都是基于LX或BX芯片组的。在兼容Slot1芯片组方面，VIA（威盛）电子的Apollo Pro及最新的Apollp Pro Plus芯片组都支持Socket 370处理器。
　　Socket 370插槽。PPGA赛扬处理器的安装紧固部件，功能和外形与Socket 7架构的插槽一样，只不过插孔多了一圈，而且靠起拔器手柄一边的两个角都少一根针孔，分别对应CPU的相应位置（见图3）。
　　主板结构Socket 370主板大都使用了MicroATX结构，集成显卡或声卡芯片而相应减少扩展槽数目，但它仍兼具ATX电源结构的各种特征。

　　三、认识Socket 370转接卡
　　对于Socket 370仅支持赛扬处理器的局限性，及兼顾当前PⅡ主板的升级，许多主板厂商都推出了Socket 370—Slot1的转接卡。PPGA封装的处理器插入卡上的Socket 370插槽，就能够在Slot 1主板上使用Socket 370的赛扬。图4为台湾承启公司推出的转换卡，支持66/100MHz，外频最高433MHz的赛扬处理器，1.3～2.05伏的电压调节。
