为ZIP加密文件解密

　　Winzip、Autozip等一些压缩软件都采用ZIP文件格式压缩文件。这些压缩软件都提供了对ZIP文件加密的功能，可以采用先压缩再加密的方法来保存一些重要的文件数据。但如果忘记了密码，连自己也无法解压缩了。别担心，其实我们可以用对付此类加密压缩文件的软件Ultra Zip Password Cracker(简称UZPC)来解密。
　　和其他的破解软件一样，UZPC也是采用穷举法逐一试验每一个密码。UZPC的使用方法十分简单，运行UZPC后，选择“Task－>New”,打开一个要解密的ZIP文件，在“Attack Type”中选择解密方法：“Brute－force”为强制使用穷举法逐一按A～Z、0～9的顺序进行搜索；“Dictionary－base”为按字典顺序破译。
　　如果采用“Brute－force”方式,下一步选择“Password charset”(密码类型设置)，有以下几种类型：
　　Didits　数字型0、1、2、...9等
　　Capital letters　大写字母A、B、C、...Z等
　　Small letters　小写字母a、b、c、...z等
　　Special chars　特殊字符～、＄、＃、@、＆、＊等
　　User defined　用户自定义
　　用户可以是采用单一类型，也可以采用几种类型混合使用，选择时只需在相应类型左边的方框里打上“√”。然后设定密码长度：Minimum为最小长度，Maxnimum为最大长度。最后单击“GO”就可以开始工作了。
　　如果采用“Dictionary－base”方式，还需选定一个字典文件(以TXT文本文件形式存放)。
　　UZPC将显示当前的工作状态、当前检验的密码、平均检验速度以及估计剩余时间。
　　UZPC在工作时不会占用很多的系统资源，因此你可以一边玩游戏，一边破译密码。等你游戏玩够了，密码也破译出来了。UZPC还支持“断点续传”功能(是不是很像某些下载工具)，你可以中途停止破译并保存当前状态，下次再接着破译。(浙江　王培坤)
