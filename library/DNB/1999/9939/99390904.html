《 世纪之窗 》

　　当初谁也没有想到微软公司会在NT5.0基本完成后将其改名为Windows 2000，因为Windows 2000原计划是对应于第6版Windows的。可以说Windows 2000是微软公司迄今为止投入最大的一个产品，它集中了Windows98和Windows NT 4.0的诸多优良性能、功能。IDC评论员Daniel Kusnetzkg说：“Widnows 2000的代码有80％是新的。它简直就是个新操作系统。”微软将Windows 2000吹捧为“未来的操作系统”，这个系统真的有这么大的威力吗？且让我们来看看其真面目吧！

　　一、全新的安装升级向导

　　Windows 2000目前分为四个版本，即Windows 2000 Professional、Windows 2000 Server、Windows 2000 Advanced Server和Windows 2000 Datacenter Server四种。其中的Windows 2000 Datacenter Server是一个新的品种，主要是通过OEM方式销售，是一个64位的产品，支持8个以上的CPU和64GB的内存，推出的时间要晚一点。
　　在DOS下安装Windows 2000和NT4没有多大区别，还是秉承NT4的蓝底白字的风格，只是加入了自动启动功能而已。但是，在重装Windows时，DOS下的操作要比NT4前进了一大步，包括重新覆盖、修复系统、修复启动盘均比NT4显得智能化。
　　在Windows下安装Windows 2000采用智能化的安装向导，每一步都给出简洁的提示，告诉安装者这一步要做些什么，即使是初次接触的初学者，也会轻易上手（IE5的安装界面相信大家都已熟悉）。安装过程中只要做少许的参与，就可以轻松完成安装。值得一提的是Windows 2000中的“升级报告提交功能”。在将Windows的早期版本升级到Windows 2000时，Windows 2000将对系统进行扫描，在升级安装过程中，会产生一个升级报告，详细列出了在升级过程中机器存在的软、硬件问题，哪些程序在Windows 2000里运行会有问题，哪些硬件的驱动程序需要更新，可以将此报告打印，一目了然。

Windows 2000语录：微软的5000名全职开发人员在每周7天、每天24小时不停地进行Windows 2000的开发工作，这个软件巨人希望其巨大的投资能够使这个系统成为有史以来最强大、最灵活的操作系统。(Cnet)

　　二、丰富的硬件支持,

　　Windows 2000采用了新的Windows驱动器模型（WDM）规范技术，并向下兼容，Wondows95/98中的许多外设都可以在Windows2000中进行工作，它包括了7000余种硬件设备驱动程序，可谓是一个驱动大全。各种硬件的安装非常方便，一些比较老的硬件外设如声卡、显卡等均可以一次性检测完成安装，并配置好，不需要再次人工安装了！即插即用特性在此得到了完美的体现。支持的硬件外设有：USB设备、数字相机、扫描仪、IEEE 1394、（防火墙）设备、IrDA等。另外，Windows 2000还内置了包括Direct 3D和DVD在内的最新DirectX视频体系结构，支持双显示器，让你更方便直接地享受多媒体效果。
　　由于NT采用了叫做“硬件抽象层”的技术，对一般人员而言，硬件管理在Windows NT4中可以说是很难配置好。而Windows2000把Windows98方便的硬件管理功能进行了深化。既可以方便地对硬件资源进行调整调用，还可以对可移动存储器进行管理。

Windows 2000语录：这个新版本具有满足企业级应用需求的潜力，将使NT比过去更为强大。但是这同时也增加了该系统的复杂程度。在正式版本推出之前，这个网络操作系统还有很多的查错和操作简化工作要做。(Zdnet)

　　三、方便的文件管理功能

　　Outlook的导航栏功能首次引入Windows 2000中，为使用者快捷地调用使用频率最高的文件提供了方便。这些功能存在于Windows 2000自带的程序中，在出现“Open"（打开文件）、“Save"（保存文件）、“Save As"（另存为）、 “Print"（打印文件）等四种通用的对话框时，可以见到导航栏。除了可以本地存取外，还可以调用全新的“My Network Places"文件夹，通过网络进行存取。
　　在控制面板中，Windows 2000添加了文件夹选项，可以不打开资源管理器自定义文件和文件夹。Windows 2000首次让用户能够自定义Windows的工作条。在RC1版本中，Windows Explorer提供了大约20种命令按钮。在以前的 Windows系列中无法实现同窗口的文件的复制与移动，在Windows 2000中就内置了这两种功能。
　　在茫茫的信息中查找文件可谓头痛的问题。Windows 2000提供了更丰富的服务器查找功能，集成了HTML风格的新查找功能，可以对储存在网络工作站硬盘中的文件或储存在网络服务器中的文件（企业数据库）、数据进行快捷的查找，完全不同于Windows“开始"选单中的查找命令。这种查找随处可以实现， Windows 2000能够让用户在任何一个打开文件对话框中，使用查找功能。

Windows 2000语录：这是一个值得升级的选择。我们发现NT 5.0的Beta 2版能够很好地完成预期的工作。Windows 2000无疑将是一个值得从NT 4.0和98升级的新版本。也许，这种升级的重要性可以与从NT 3.51到4.0的升级相提并论。(CMP TechNet)

　　四、多语种支持

　　Windows 2000采用了Unicode2.1作为基本字符集，支持60余种版本。使用Windows 2000中文版的用户可以在不启动日文版操作系统的情况下处理日文文档。多语言支持功能对于那些只是偶尔使用其他语言的用户来说是非常适合的。
　　简体中文和繁体中文在Windows 2000中是不同的语言。简体中文版的用户，不用安装任何其他的支持软件，就可以方便地查看和编辑繁体的文档或网页了。要想查看和编辑其他语言的文档或网页，只需在“区域设置”中选择相应的语言，Windows 2000就会安装该种语言的支持模块，用户即可使用了。系统虽然支持了中文显示，但还无法输入汉字。RC1版的Windows 2000自带了4种中文输入法,分别为：内码、全拼、双拼、郑码。可以添加其中的一种或几种，也可以添加微软的拼音输入法和五笔字型。

Windows 2000语录：Windows 2000这个系统灵巧、稳定，但仍有许多工作要做。如果微软能够把目前存在的一些粗糙的地方打磨得更光滑，Windows 2000将更具吸引力。如果你拥有合适的硬件并且希望获得一个比Windows 98更健壮的操作系统，那么升级到Windows 2000吧。(PC World)

　　五、智能镜像功能

　　智能镜像功能是Windows 2000的又一大特色之一。它能够将用户对应用软件、文件与系统的设置做出镜像。智能镜像能够自动在服务器上将应用软件、文件和系统设置进行备份（策略配置），让网络管理员能够轻易地执行应用软件的升级等工作。无论什么时候或因什么原因，如果网络管理员重新安装了Windows 2000，在用户下一次登入网络时，智能镜像会自动进行检测，所有的应用软件、文件和系统设置都会被自动恢复到这部新安装的工作站中，相当于引入注册表信息（注册表引入功能当然不能与之并论）。不论使用哪一部网络工作站登入，服务器都会根据请求将该用户的个人备份数据（策略配置）恢复到该工作站之中，这样就可以使环境配置保持一致，大大减少了网管的工作量。
　　活动目录是NT最新的目录服务功能，为网管实施网络管理工作带来了方便。Windows 2000 Professional工作站和Windows 2000 Server服务器互相连接，用户能够方便地进行网络打印机、服务器、磁盘驱动器和其他网络装置。使用活动目录能够将应用软件分配给用户，让应用软件能够显示在用户的开始选单之中。网络管理员还能够将选择性的应用软件公布在控制台的“添加/删除程序"清单中。在Windows 2000的活动目录中，层次结构的域将大大地加强安全性管理。

Windows 2000语录：这个操作系统——拥有适合桌面、服务器和大型公司网络的若干版本——将戏剧性地改变我们的工作方式……服务器和高级服务器版本将使工作组内的协作和跨部门的协作变得更容易。数据中心版本的稳定性和可靠性将使因特网服务的创建和维护能力获得提升。(Windows Magazine)

　　六、强大的“网络"功能和安全机制

　　在Windows 2000中随处可以见到网络的身影,不但集成了IE5和OE5，在控制面板中还增加了一个“Network Connections"文件夹，取代了Windows98/NT 4.0控制面板中的网络设置（Network Setting）功能，并将拨号网络配置和MODEM配置合在一处。设置拨号网络和设置电话号码等上网工作，要比NT4方便得多。
　　Microsoft的零管理（Zero Administration）工具是Windows 2000的一个标准组件，它可以锁定客户机。终端服务器是另外一个标准件，在活动目录层次中，用户策略限制可以被应用到所有任何层次中。可以锁定整个部门、部门中的一个工作组或者单个机器。
　　Windows NT 4．0中使用的公共密钥安全机制在Windows 2000中得到了延续。可以发布安全证书，并且限定那些拥有安全证书的用户才能享受网络服务和资源。Windows 2000中的另外一个安全特征就是加密文件系统（Encrypted File System，EFS），它通过用户证书进行磁盘数据加密。只有加密数据的用户才能够看到这些数据。Windows 2000中还对IPSec标准进行了集成，在所有的Windows TCP／IP传输中加入了IPSec的支持。

Windows 2000语录：在未来两年里基于Windows 2000的应用将增长13％。与此相比，预测显示Unix将持平，而Novell公司的NetWare则将下降10％。(Computer World)

　　总的来说， Windows 2000还是一个不错的操作系统，特别是Windows 2000 Professional工作站，更适合于个人电脑平台使用。随着各软件厂商对Windows 2000的支持，相信不管是运行游戏、办公软件还是学习软件，Windows 2000 都会给你带来更愉悦的享受！让我们一起来开启世纪之窗！
(徐州　吴付愿)
