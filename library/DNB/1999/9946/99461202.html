《 软件一点通 》

　　●在Windows 98下使用硬盘缓存

　　运行SYSEDIT.EXE编辑WINDOWS目录下的SYSTEM.INI文件，寻找[VCAHE]项,其下面一般是空的,加上两行:
　　MINFILECACHE=4096
　　MAXFILECACHE=4096
　　那个4096就是你设置的缓存的大小，建议两个数值设为一样，而且如果你的内存比较大，可以设大些。注意，这是用你的物理内存，具体设多少可以自己试试。

　　●优化软盘性能

　　双击打开控制面板中的“系统”图标，选择“性能”标签，按下高级设置中的“文件系统”，单击“软盘”标签，将“每次启动计算机时都搜索新的软盘驱动器”前面的钩去掉，再按“确定”。这项改动可以缩短Windows 98的启动时间。

　　●删除相似的字体文件

　　Windows 98 安装时附带了几十种字体文件，不仅浪费了硬盘空间，也使Windows 98的启动速度变慢，还浪费了宝贵的内存，使用如下方法可以删除一些相似的字体：打开控制面板中的“字体”，在“查看”菜单中选择“按相似性列出字体”，然后在字体下拉列表中选择一种字体，如“Times New Roman”，就可以看到文件夹中显示的字体了，你可以有选择地加以删除。

　　●Windows 98安装程序SETUP可以使用的参数

　　格式：SETUP [batch] [／ie] [／im] [／is] [／iv] [／iw] [／id] [／nr] [／T:tmpdir]
　　batch 指定包含安装选项的脚本文件名及其位置。
　　／ie 跳过制作启动盘。
　　／im 跳过常规内存检查。
　　／is 不运行 ScanDisk（使用新的或刚格式化硬盘时可用）。
　　／iv 安装时不显示公告板(即有关Windows 98新特性的介绍)。
　　／iw 跳过版权确认
　　／id 跳过可用磁盘空间检查
　　／nr 跳过注册表检查。
　　／T:tmpdir 指明要存放 Setup 所复制的临时文件的目录。该目录必须已存在，但任何在此目录中的现有文件将被删除。建议将其指向非安装盘，可以减少安装盘上的文件碎片。

　　●Format的隐含参数

　　／autotest　　自动测试磁盘（即模拟格式化，完毕后显示统计信息）
　　 ／backup 　　同上，但有提示输入卷标(Volume label)
　　／z:n 　格式化后每一簇的大小，n=1=512bytes,2=2×512bytes,3=3×512bytes，依此类推，格式化时硬盘要小心使用！
　　 ／U 执行无条件格式化，将每一字节均用十六进制值F6覆盖，无法进行恢复。

　　●Fdisk的隐含参数

　　 ／mbr 重写主引导扇区，可以清除一些引导型病毒
　　 ／fprmt 　　允许512兆以下的硬盘使用FAT32（通常512M以下的硬盘不允许能使用FAT32
　　Fdisk x ／pri:size ／ext:size ／log:size
　　x = 驱动器号 (1, 2, 3... )，1对应第一个硬盘，2对应第二个硬盘……
　　／pri:size = 创建一个大小为XX兆的主分区
　　／ext:size = 创建一个大小为XX兆的扩展分区
　　／log:size = 在扩展分区创建一个大小为XX兆的逻辑驱动器
　　这些参数在编制自动分区的批命令时很实用。

　　●Commander.com的隐含参数

　　／D: 用于Config.sys文件中Shell=命令行，可以防止启动时运行Autoexec.bat。例：Shell=C:\COMMAND.COM C:\ ／D
　　／Z: 可以显示″ERRORLEVEL″ 的返回值信息，用法同上。
　　（福建　林斌）
