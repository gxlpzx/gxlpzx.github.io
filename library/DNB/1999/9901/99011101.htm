贝贝推荐压缩工具

　　在文件压缩工具中，除了WinZip以外，还有不少很好的工具，它们大多是共享软件，并提供试用版本，下面贝贝推荐其中一些具有代表性的软件给大家，如果有兴趣，不妨一试。

ZipMagic 98 2.02

　　这是一个以独特的ZipFolder功能闻名的工具。该工具以VxD方式驱动系统，可以自动将ZIP压缩文件视作普通的文件夹，因此可以直接在系统资源管理器窗口或文件窗口中运用压缩包中的文件，该工具提供全功能试用版本，程序主页是http:∥www.mijenix.com/。
　　贝贝推荐：超级方便的压缩文件管理工具，网友必备。其功能比WinZip少，在压缩双字节文件时会产生错误。

PKZIP for Windows 2.06.03

　　这是最为经典的ZIP压缩包管理工具，由PKWARE公司出品。该工具的独特之处是支持同时在程序窗口中打开多个压缩包，并允许用户使用拖放功能在不同的压缩包之间交换压缩包文件，使用很是方便。该工具可能具有目前最快的ZIP算法压缩速度。该工具提供全功能试用版本。程序主页是http:∥www.pkware.com。
　　贝贝推荐：管理较多压缩文件时建议使用。

WinRAR 2.06

　　这是我们非常熟悉的RAR的Win 95/98版本，程序有高压缩比优点，提供类DOS环境下的操作方式，支持在硬盘中创建分卷自解压压缩包。该工具提供全功能试用版本，程序主页是http:∥www.creabel.com/softronic/rar_archiver.htm。
　　贝贝推荐：很有特色的压缩工具，特别是制作分卷自解压文件。贝贝常常使用，以对付坏软盘。

WinACE 0.95 Beta

　　这是一个界面极为华丽的高压缩比工具。该工具的卖点是提供类资源浏览器的界面方式，直接提供系统文件管理功能，提供在硬盘中创建自扩展分卷压缩包功能。该工具提供全功能试用版本，程序主页是http:∥www.emerge.de。
　　贝贝推荐：有赶超前者之势。

Sonic Cabinet Manager 1.0

　　专门支持微软发布的CAB压缩格式的管理工具，该工具提供7天试用版，程序的主页是http:∥www.mimarsinan.com。
