《 道听突说 》

　　突突来上班的时候，在公车上听到了熟悉的老歌《光阴的故事》，“流水它带走光阴的故事，改变了一个人……”，悠扬的旋律让人想起以前的时光。突突一下子变得怀旧起来。不过，在这个瞬息万变的信息社会里，旧的东西已经不太有人去怀想。而我们都爱的软件，也在不断地推陈出新，不断地吸引着我们的目光。
　　说到新东西，突突倒是发现了一个，一家新公司I2Go.com推出一种新的便携式数字音乐设备eGo，这个新东东优点可不少，除了能播放MP3外，还包含浏览器和其他一些软件，这样，你可以直接设定好定时下载MP3的任务，不用多费心思了。另外还有一项绝妙的功能，你可以直接通过eGo，听到别人给你的电子邮件了。这是通过专门的文字和语音的转换引擎来实现的。怎么样，拥有一个一定很酷吧。
　　有很多程序员都来信告诉突突，喜欢看软件版的文章，我想你们也一定希望多了解些关于程序开发方面的消息。当然如果你是 Java的拥护者，一定不会错过新的Java2 Development Kit程序开发套件吧。过去有不少人抱怨Java的执行效能远远不及 C＋＋等各种传统惯用的编程语言，不过这次Sun Microsystem可是有备而来的，不仅在运行速度上有大幅度的提高，而且在内容上也做了一些修改和增强。在本版的 JDK中新增了最新的Java API，除此之外还有相当多的新功能如支持拖放的功能、 Java interface definition language (IDL Java界面定义语言)、 Java Virtual Machine Debugger Interface (JVMDI Java虚拟机调试界面)等。这么多新功能，不试一下真是可惜哦。
　　贝贝这期讲了不少的播放器，还介绍了把录像带转换成VCD的工具软件。突突也来凑个趣，介绍一个将CD转换成MP3的软件CDex可以将音乐CD中的音轨转录成WAV或者 MP3格式的文件。它已经内建录制MP3的编码器：LAME3.30，所以可以直接将CD音轨转为MP3格式；也可以将WAV文件转成MP3。不过我想CDDB功能才是突突最喜欢这个软件的原因。CDDB功能可以让你不用输入歌名、歌手、专辑名称等资料，因为CDex会自动从网络上把资料抓回来。但是，抓不抓得到资料还得看CDDB资料库里面有没有！目前最容易找到的还是英文专辑的资料。如果你愿意的话，可以到http://www.cddb.com/ CDDB资料库网站上去建立你喜爱的专辑资料！突突决定先去建立一个罗大佑的资料库，你会喜欢吗？
　　有朋友告诉突突，在夜里遥望无边的星空，所有的烦恼就会烟消云散。在突突居住的城市，真的很难再看到满天繁星的夜晚了，不知道这是不是现代人烦恼特别多的原因之一，也不知道这是不是让虚拟技术变得越来越重要的原因呢，突突这样想，聪明的你说呢？
