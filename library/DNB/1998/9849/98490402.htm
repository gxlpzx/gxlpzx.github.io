Visual Basic使用技巧

  　 在一个与工具条关联的图片列表控件中添加和删除图片。
　　当一个工程仍处于设计状态时，我们需要保证在一个与工具条关联的图片列表控件中添加和删除图片时的灵活性。这是因为，当一个图片列表控件与一个工具条关联的时候，你不能动态地改动图片列表控件中的图片。我们可以使用以下这种办法来避免这一限制：
　　1往图片列表控件中加入位图图片并为加入的每张位图赋予一个唯一的关键字。
　　2往工具条控件中加入按钮并将位图图片的关键字赋给按钮关键字。每一个带位图的按钮的关键字应该与图片列表控件中相应位图的关键字一致。诸如tbrSeparator、tbrPlaceHolder不带位图的按钮没有关键字。
　　3在窗体的Load加载事件代码中将图片列表控件与工具条关联起来：
　　Set ToolBar1ImageList=ImageList1
　　4将位图图片赋给工具条上的按钮:
　　Dim myButton as Variant
　　For Each myButton in ToolBar1Buttons
　　If myButtonKey<>Empty Then
　　myButtonImage=myButtonKey
　　′如果关键字是可读的
　　′就将其赋给myButton的Description和ToolTipText属性
　　myButtonDescription=myButtonKey
　　myButtonToolTipText=myButtonKey
　　Endif
　　Next  (陕西　陈　剑）
