歪歪线报

    歪歪废话 　
    歪歪这星期为两个事情感到特别自豪，一是歪歪上期推荐的Ministars SafeClean（一个在Windows95／98／NT下运行的文件清理软件）被ZDnet评为五颗星，再次证明了英雄所见略同。二是歪歪的热线yy.y@263.net每天都收到数十封信，大家对这个栏目的认同便是给歪歪的最好鼓励（歪歪只有继续减少睡眠时间以回报大家）。当然歪歪这星期也有倒霉的事（被小生称为“中奖”），不幸身中Win95.CIH病毒，话说这个Win95.CIH病毒可是厉害得不得了，每月26日发作，一般表现是格式化硬盘，但如果你的主板BOIS支持Flash且没有屏蔽掉Flash功能的话，它会改写你的BIOS！！！造成主板BIOS程序破坏而不能启动!幸亏王大师第一时间给了我AV98行天反病毒软件（共享版本），杀遍我的7G硬盘，得以彻底清除病毒（注：共享版本功能同正式版本一样，只是杀一个病毒必须敲一下回车键，歪歪共计敲键1486次）。
    升级消息
　　这期升级消息当然首先是杀毒软件的升级了，有各类国产杀毒软件正式版本的朋友注意，歪歪也收集了以下的网站，它们提供的升级版本都可以杀掉Win95.CIH病毒，KV300W＋: http:／／www.jiangmin.com；瑞星杀毒软件：http:／／www.rising.co.cn；KILL98认证版：http:／／www.kill.com.cn；VRV病毒防火墙：http:／／www.vrv.com.cn。
　　本期最值得注意的是超级解霸5.01A升级补丁程序的发布，据说有13点改进。歪歪就拣几点大家感兴趣的说： 1.在高档计算机中允许无损画面质量的解码。 2.支持小画面非标准MPEG文件。 3.释放CPU占用时间，播放VCD时间占用率为20％左右。4.支持CPU降温，采用Windows 95内核支持降温法，不占用任何进程时间，专门使用系统空闲时间降温，所以在允许降温时任何程序都运行如飞。5.由助手工作室帮助制作的新的帮助。 6.支持以前不能播放的非标准结构MPG文件，能更准确地识别MEPG文件。7.CD解霸支持直接播放带音轨的数据光盘。歪歪试用了一下,安装后发现开始菜单的解霸五变成了豪杰解霸五，在测试中歪歪用豪杰解霸五自带的测试选项测试，结果播放VCD时间占用率（包括视频、音频）仍然达到了45％左右（歪歪的机器为k6－233,64M）,CPU的温度在播放的15分钟内只上升了1度。不管怎么说，用了超级解霸的朋友应该赶快去升级（注意只能在5.01版上升级）！那个升级狂人WinAMP又到了1.99.78（快到2.0版了）其他的升级消息太多了，歪歪限于篇幅就不多说了。
    新软件热眼看
　　由于《PC世界3》即将于9月底发布，歪歪本周除了上网给大家发现好东西之外，更是偷看了王大师12G的硬盘，给大家第一时间的情报。这次歪歪就给大家介绍一些大家伙，如果你觉得从网上拖下来不划算的话，别忘了我们的《PC世界3》。Characterz Builder是一个好玩又好用的三维造人工具（8M大），它可以让你自如地定制三维人物，并改造其身体的各个部分、动作、皮肤，从而建立起他们的外貌举止。在它的素材库里更可找到大名鼎鼎的劳拉。歪歪用这个软件造了一个自画像，看看吧。你还可以把图形外貌单帧或多帧的 bmp 文件以及 VRML 动画文件单独输出。更妙的是，如果你已经有了 Microsoft Liquid Motion 的话，只要在其中加上一个叫作 Liquid Characterz 的插件就可以编辑 Characterz Builder 生成的文件了。CDH Image Explorer Pro 2.9是一款图片浏览处理工具（6M），它的特点是支持超过五十种格式的图形文件。Ulead PhotoImpact V4.2是一个非常专业的 Web 和 非 Web 图像编辑软件(15M)，完全兼容 Microsoft Office97 和 OLE2 支 持 。图像编辑王中王 Photonyx 1.0(8M), 是一个全功能的图像绘画、修饰、合成软件。它具有快捷的缩放操作和工具面板、调色板，以及一套完整的编辑和绘声绘图工具。其撤销／恢复操作的次数不受限制，操作对象可完全反复修改，对象个数不限。它包括50个以上的图像处理滤镜，支持30种以上的文件格式。
　　编辑部在歪歪的推荐下（除了海天一色和头头外）都用上替代记事本的编辑器EditPlus,这个软件的最大好处便是具有自动监视剪贴板功能，也就是说当你浏览网页的时候，你对其中的文字感兴趣，你只需用右键复制一下，文字就会自动粘贴到EditPlus里去了，相当方便，据说网上有它的汉化版本！
    呐喊！国产软件
　　歪歪的民族立场得到了大家的一致赞同，歪歪这期也会给大家介绍一些和中文密切有关的软件——输入法（当然都是国产的）：首先出场的是小生的至爱——智能五笔 3.1版，歪歪是不用五笔的，但小生信誓旦旦地保证让歪歪相信智能五笔的确是一个好东西。拼音之星3.0是歪歪发现的一个很好用的输入法，它在 Win98、Office97和Outlook上表现良好，没有乱码，其智能整句输入相当出色。如果能把双拼中把击键的音节直接显示出来就更好了。自然码也有新的东西出来：自然码6.0B＋试用版。
　　本周歪歪推荐软件：EditPlus。(打★ 的软件已被收入《PC世界3》中，歪歪热线：yy.y@263.net)
