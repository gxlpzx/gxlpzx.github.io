电脑"洗衣机"

   　各位朋友,今天我向各位介绍的是一台全自动的“洗衣机”。“型号”是Windows Washer；产地：美国；适用范围：公共机房、网吧、个人用户……看到这个标题，不要怀疑自己的眼睛,你并没有错拿一份广告。因为如果把Windows95系统比做一件我们每天都要穿的外衣，那么随着时间的推移，上面也自然也会多了一些“污迹”。当然我是不会用水去“洗”它了。
　　Windows Washer将自动为你进行清洗!在你需要的任何时候，洗去任何你想要洗掉的东西。它可以在后台进行清洗；也可以在Windows系统开始或结束的时候清洗。你可能会问：能够定时清洗吗？当然啦！要不怎么叫全自动呢！
　　在安装Windows Washer时你可以选择是否在每次启动Windows95时自动载入,它运行后将在Windows95任务条上展示出它的商标：一个橡胶清洁滚刷。如果你点击一下橡胶滚刷，将出现三个开关，按下“Wash now”开关，Windows Washer将马上进行清洗工作；按下“Setting”开关，将调出Windows Washer的主窗口去配置你的清洗项目和其他清洗选项；按下“Quit”开关， 当然是关机了。
　　这台洗衣机清洗时的面板和设定时是一样的。它主要由五个部件组成：最上面是它的“商标”Windows Washer；下面一栏是上一次清洗的时间；中间两大部分，左边是标准清洗项目，右边是定制清洗项目；最下面一排“开关”，第一个是现在清洗，鉴于大家都有“洗衣”经验，就恕不赘述了。
　　在Windows Washer的左边的标准清洗项目选项中，可以进行如下设定：
　　通过选择相应的选择框，可以清洗Windows95开始菜单文档中保存的用户打开文档的历史记录，Windows95的查找功能中的历史记录，Windows95运行功能中的历史记录，临时文件目录，回收站，ICQ网络传呼机历史记录，NETSCAPE 4.X的历史记录，IE3.X/4.X的历史记录。
　　需要注意的有以下几点:1.清除临时文件目录包括\Ttemp和\Windows\temp 目录。2. Netscape 4.X和IE3.X/4.X的历史记录包括Cache、History、Cookies等。3.由于一般的“洗衣机”不能够清洗废纸篓（回收站），所以这个功能只有在注册后，花一些Money才能使用。4.浏览器中的Cookies 功能也只能在注册后才能使用。
　　右边是“自动”清洗功能。自动项目允许你根据自己的不同要求，增加在标准清洗项目中没有的文件夹和注册表中的键值，使其更为实用并且更富有个性化。值得注意的是注册表项目的自动清洗功能，如果对于注册表并不熟悉的话，最好不要贸然行事。
　　下面一部分选项，主要实现一些自动功能：
　　1．Load Washer at Windows start
　　选择它，将在Windows95系统启动时被自动载入。
　　2．Auto wash items when Windows is
　　选择Started在Windows95系统启动时自动清洗。选择Shutdown，在Windows95系统关闭时自动清洗。
　　3．Automatic wash every
　　选中这个项目后，在它右边组合框的下拉列表中可以设定按每15分钟，每30分钟直至每个月的全自动清洗程序。
　　4．每一个单词第一个字母大写
　　选中这个项目后，Windows Washer将进行“漂白”功能。怎么样，这台“洗衣机”功能够全的吧！标准的清洗是将文件进行删除，如果用一些相应的恢复软件如：Undelete、Unerase等，可以进行恢复工作。选择了漂白这个功能后，Windows Washer先将这个文件覆盖，然后将文件改名为X.X后，再进行删除。这样会延长清洗时间，但却可以大大增加系统的安全性，可以防止一些人将已经洗掉的“污迹”再还原回来。
