ICQ的使用技巧及安全问题

　　一、ICQ的使用技巧
　　1.重装ICQ时保存原有列表和谈话记录
　　在 C:\Program Files\Icq 目录中有个 db 目录和 Bookmark 目录。把它们拷下来，下次再装 ICQ 后再覆盖上去即可。
　　2.为朋友设置在线提示
　　在列表中你朋友的名字上点右键，从弹出菜单中选择 Alert/Accept modes， 在Online Alert 上点上个勾。这样，当朋友在线时，ICQ会用声音和弹出对话窗来提醒你。
　　3.自动通过服务器发送
　　ICQ不能直接发送信息时总会问你 Send Through Server 还是 Send letter，如果你不小心选了 Send letter， 那么你的信息可能要经过较长时间才能送到朋友那里，所以最好还是让 ICQ 自动通过服务器发送。在 Preferences 中的 Events 的 Auto Send Messages Through Server前打个勾就可以了。
　　4.为你的ICQ增加16个服务器
　　到 pualgao.yeah.net 上下载 ICQ中文聊天室和给ICQ添加16个聊天室的软件。
　　二、ICQ的安全问题
　　1.由于ICQ能完全地控制你的机器并自由地与网络交流，而且不会有任何警告信息。所以一定要注意从它的主页http://www.icq.com上下载。以免别有用心的人修改ICQ并利用它盗取你的机密和破坏你的机器。
　　2.给你的 ICQ 加密码
　　在 ICQ 菜单的Security ＆ Privacy项中Security level栏选High即可。
　　3.不让某人看到你是否在线
　　在列表中某人的名字上点右键，从弹出菜单中选择 Alert/Accept modes，在Invisible To user 上打个勾。
　　4.如何在不让别人知道的情况下把他添加到列表
　　先另注册一个临时ID,添加上自己原来的ID和你希望悄悄添加的那个人的ID。然后在列表中你的名字上点右键。选 Contacts 把那个人的列表传送给你。最后反注册刚才的ID即可。
　　5.防止你的IP泄露
　　这是为防止别人通过IP推断你的地理位置而用IP炸弹攻击你的措施。简单的 IP 隐藏只要在Security ＆ Privacy 中选 Do not allow others to see IP address 即可。但是目前有些工具可以查看这种隐藏的 IP 。所以，要更好地隐藏 IP，还需要组合下面的设置：
　　①首先在 Preferences 中的 Contact list 中选“Show Online Only Tab in the Contact”，以保证只有在你列表上的人才能看到你在线。
　　②然后在 Security ＆ Privacy 中选 My authorization is request，不让别人随意地把你添加到列表中。
　　③最后将列表中某些可疑的人设置为Invisible to user。

　　(老农)
