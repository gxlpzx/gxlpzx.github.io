Junior——孩子们用的浏览器

　　我认为适合孩子们用的浏览器应该是界面简洁有趣，使用方便，并且可以让家长很方便地锁定站点，防止孩子浏览一些不该看到的东西。HexaBit Junior（以下简称Junior）就是符合这些要求的浏览器。
　　Junior安装方便，安装完成后，在Win95的桌面上自动生成一个Junior图标，图标是电视机的样式，这正是Junior的一大特色，它的浏览器界面就是按照电视机设计的，启动Junior后，就会看到Junior简洁但富有特色的界面，在屏幕的下方整齐地排列着几个按钮，但它却不像其他浏览器有那么多的操作选项，它把能省的都省略了，在URL的输入框下，依次排列着“Home”、“<<”、“RELOAD”、“>>”和“Favorites”等5个按钮。点击Home能够到Junior的家去看看有没有新版的Junior发布，“<<”，“RELOAD"和”>>"对于一个浏览器来说当然是必不可少的功能键，点击“Favorites“可以到你预先设定的你喜欢去的站点看看，作用相当于其他浏览器的Bookmark。在URL的输入框的左边有一个ON按钮，它的作用与电视机上的ON按钮相同，点击它即可打开和退出该浏览器。在URL输入框的右边，还有一个CR按钮，这就是Junior的精华所在。
　　按下CR按钮后会弹出一个对话框，首次进入会让你输入一个密码，以后每次按下CR后，都会让你输入密码，密码正确后，才允许进入。CR的含义就是Parental Remote Control(家长远程控制)。在这里你可以进行十分详尽的设定，帮助你的孩子畅游因特网并从中得到有用的信息。下面就来介绍CR中的功能：
　　1.Generel选项。在这里有控制（Control）和颜色（Color）的两项选项。在Control中:
　　Stop Address Control：让你决定是否禁止你的孩子访问地址表中的站点;
　　Stop Time Control： 让你决定是否按照时间表来控制孩子的上网时间；
　　Surf Only in Favorites Sites：让你决定是否让你的孩子只在“Favorites”中设定的站点上浏览；
　　Disable the Desktop and Windows Taskbar： 可以设定Junior启动后，是否出现Win95的任务栏。
　　在Color中：
　　TV color：让你设定浏览器的颜色，漂亮的颜色会使你的孩子更感兴趣；
　　Legend Color :让你设定在浏览过程中状态栏中字体的颜色。
　　2.Addresses选项。在这里你可以输入一些关键字来禁止你的孩子访问某些不健康的站点。Junior会根据地址表中的关键字检查站点的URL，对于那些名称中出现了关键字的站点，Junior会禁止访问。
　　3.Favorites选项。在这里你可以为你的孩子编辑一些有趣的适合孩子们浏览的站点，并且可以为它们加上一个通俗易懂的名字，让他上网更加方便。
　　4.Times选项。在这里Junior为你提供了一张从星期一到星期天包含每个时段的十分详尽的时间表，为了防止你的孩子上网没有节制，你可以设定某些时间禁止使用Junior。
　　5.Surf Log选项。在这张冲浪日志表中，详细地记录着每次上网、下网的时间，曾经浏览过的网页，在这里你可以清清楚楚地看到你的孩子曾经浏览过的站点。
　　6.Password选项。在此选项你可以修改密码。
　　该浏览器可以从http://www.hexabit.com下载，Junior只有420k左右。演示版与正式版的差别在于，正式版的Favorites项中提供了大量的儿童站点，并且演示版中的Addresses表中只能输入5个关键字。Junior目前的版本是2.0版，它的编程语言是Visual Basic5.0，因此Junior运行时需要VB5.0的运行库文件，但为了减少程序大小，在下载的程序包中未带有VB5.0的运行库文件，如果你已安装了IE就不必再下载了，否则你可以到http://www.hexabit.com/junior/download/download.
htm去下载VB5.0的运行库文件,也可以在《电脑报配套光盘第三辑》中找到它。
