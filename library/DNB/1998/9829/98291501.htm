电脑工具软件大拼盘  

　   编者按：也许你已是玩转电脑的行家，也许你只是初识电脑的新手，走进可亲可爱的电脑世界，面对五花八门的软件，难免头晕目眩、不知所选。尤其是在群星闪烁的工具软件王国里，究竟该如何构建自己得心应手的工具箱呢?大浪淘沙，让我们挑选出久经考验的真正“珍宝”，供大家分享。

缉毒骑士——KV300

　　要选择杀毒软件，如果你常常关注连邦软件排行榜，就知道一直高踞工具软件类榜首的KV300一定错不了。
　　只要运行KV300，敲击几个功能键，可恶的病毒就纷纷灰飞烟灭了。从KV300的网页中还随时可以下载最新升级文件，并且各种主要的电脑媒体都刊载了最新病毒特征代码，只要把它们输入并存成文本文件，就可以利用扩展病毒库查找最新病毒了。KV300还有两个特别之处，一是可以将硬盘的主引导记录备份到软盘上，当由于病毒破坏导致硬盘无法启动时，随时可以进行修复；二是具有自我修复功能，保证主文件清洁和完整，不会因为自身染毒而死翘翘了。

注册表维护快刀手——RegClean 4.1a

　　Windows95的注册表并非铁板一块，它很容易在增删软件过程中埋藏祸根。及时纠正注册表中的错误，便成了一个必修工作。
　　不过别害怕，有微软的快刀手RegClean 4.1a给你作后盾。你只要双击RegClean图标，它就自动开始对注册表检查分析，若报告有错误，只要单击“Fix Errors”按钮，RegClean会先进行备份，然后再将错误更正。用不了几分钟，就一切搞定。
　　末了，你还会发现相应的Undo文件出现在RegClean文件夹内，用户名、年月日、时分秒标记得清清楚楚。一旦注册表修改之后出现异常，你只要双击较新的Undo文件，注册表立马就变回老样子了。

SYSTEM文件夹清洁工
——Clean System Directory 1.6

　　Win95的SYSTEM子文件夹内，存放着大量动态链接库文件，后缀统统为DLL。如果你发觉Win95启动变慢了，八成是那些没用的DLL文件在捣鬼。
　　想不想还SYSTEM文件夹一个清洁，让启动时那难熬的等待早点儿结束？去把Clean System Directory 1.6请回来吧。安装过后运行Clean System Directory，单击“Start”按钮，Clean System Directory开始工作。它先把SYSTEM文件夹里的DLL文件总数查点清楚，再扫描选定驱动器里的各个文件夹，逐个分析DLL文件的出身，然后，会转到另外一个窗口，列出所有认定为无用的DLL文件，这时点一下“OK”按钮，垃圾就被清除出SYSTEM文件夹了。
　　Clean System Directory的安全措施也蛮周到的，那些废DLL文件并没有像垃圾一样被“焚化”，而是被搬到了SYSTEM下的Bkupdlls文件夹里。万一出现缺少某个DLL文件的意外，完全可以补救。

压缩大王——WinZip 6.3 SR－1

　　WinZip是众多压缩软件中无可争议的霸主，Internet的迅速发展，更让它老树发新芽、风光无限。WinZip6.3 SR－1版本突出了与Internet的联姻，处处方便Internet用户的使用，例如可直接对网上下载的压缩文档查杀病毒，在浏览器中查看、解压缩文档。另外它还具有制作自解压文档、分卷压缩、直接运行压缩文档中的程序等等功能。
　　WinZip使用起来非常顺手，它能够智能识别文件格式。假如你在资源管理器选中某个Zip文档，单击鼠标右键，弹出的快捷菜单中，会有解压缩、制作自解压文档等内容，而不会出现压缩命令；当选中普通文件后，单击鼠标右键，则只会出现压缩命令，解压缩项用不着，所以就不露面了。

MP3播放至尊——Winamp 1.92

　　在流行的几款MP3播放软件中，Winamp因声色俱佳，而受到玩家的格外垂青。
　　对于音频播放软件，玩家最在意的首先是音质，在这方面Winamp称得上无可挑剔，那饱满的音色、强烈的立体声效果，让人不由地频频颔首，再加上内置十段均衡器，对音质控制就更加精细。Winamp播放面板也很讲究，提供播放、选曲、调音等等按钮，显示曲目的详尽信息，还有类似组合音响的实时输出电平指示，既饱耳福又饱眼福。Winamp支持按钮、菜单、快捷键、鼠标右键快捷菜单多种操作方式，播放面板灵活多变，更难得的是还内置了节目单编辑器，不须通过专门软件，就能轻易管理各种曲目。
　　目前最新的1.92版还支持MMX和AMD K6－2的3D Now!。

三栖抓图明星——PCS 2.0

　　抓图软件多如牛毛，竞争非常激烈，号称“专业抓图系统（Professional Capture System）”的PCS 2.0，之所以能够经久不衰，实在是它太招人爱的缘故。
　　PCS可以跨平台使用，无论是DOS、Windows3.x，还是Windows95 OSR2，PCS都应付自如；PCS没有属于自己的抓图界面，抓来的图形不是直接送到剪贴板，就是送到打印机，甘当幕后英雄；PCS总是静静地蹲在屏幕的角落里，而当你按下热键时，它就立刻冲向抓图前线，而大多数同类软件，抓图过后马上就会切换到自己的界面，向你邀功炫耀。

Win95桌面及资源管理专家——TrayMenu 1.3

　　TrayMenu 1.3运行后会在Win95任务栏最右边的系统条位置出现一个小小的台灯图标。可别小瞧了这个小图标哦，这里可称得上Windows95桌面及资源管理的司令部。
　　你只须用鼠标左键单击台灯图标就可以在出现的菜单中找到所有的桌面快捷方式，其中通过“我的电脑”菜单项可任意访问包括软驱、硬盘、光驱上的每一个文件夹和文件。而右击该图标，则可以在菜单中迅速打开没有设快捷方式的“资源管理器”和“显示器属性”。当然，菜单中的“Hide Desktop”则可使你的桌面立刻一尘不染，怎么样，很棒吧!

图像浏览高手——ACDSee 2.3

　　图像浏览软件之争也不逊于抓图软件，而ACDSee当属其中的头号弄潮儿，它浏览图片速度出奇地快，采用小图标方式，一次就能把文件夹里全部的图形文件都显示出来，预览、查找图像自是游刃有余；对选中的图形文件，进行拷贝、移动、删除、换名操作，单击工具栏上的相应按钮，就可轻松完成。ACDSee还可以按照幻灯片方式播放图像，可直接将喜欢的图片设置成屏幕壁纸，可进行图形文件的格式转换。总之，ACDSee绝对是个称职的图片管理专家，把你的图片交给它管理，就放心好了。

语言大师——金山词霸Ⅱ

　　许多电脑用户都会经常遇到使用西文软件、阅读英文文档的尴尬，这时你可以让金山词霸Ⅱ来帮你。
　　金山词霸Ⅱ可以将文档、软件菜单、工具按钮中的文字即时进行汉英、英汉互译，操作简单极了，鼠标指针指向哪儿就译到哪儿。生词不会读？没关系，稍稍往下移移鼠标，点一下浮动条上的喇叭图标，你的音箱中就会发出读得非常标准的美音，男声、女声随你挑。有了金山词霸Ⅱ，中英文互译也不过是举手之劳。金山词霸Ⅱ还是一本分类详尽的词典，既可以屏幕取词，也可以手工输入生词查看译义。启动金山词霸Ⅱ后，若又想暂停使用，很简单，只需在译文提示条区域双击鼠标左键就行了。

网络“吸血鬼”——Net Vampire

　　一提到断线续传软件，许多人总是不由自主地想到GetRight、GO!Zilla等大哥大级的软件，可是我的至爱依然是Net Vampire。
　　首先因为它下载软件的速度快得惊人。使用GetRight配合336Kbps的Modem，一般下载速度为每秒35KB左右，而Net Vampire则可以达到每秒55KB左右，有时甚至可达到每秒7KB，怎么样，够快吧。其次是它的功能强大，Net Vampire支持FTP和HTTP断线续传，具有很强的抗干扰能力，特别是在网络繁忙、连接不可靠时，它也能完成其它软件不能完成的工作。
