WebTwin将网站搬回家

　　已经好长时间没有动手写介绍新软件的东东了，这回，当我从http://www.webtwin.com/站点把这个“小家伙”拉回来的时候，安装使用之后，为之折服，于感叹之余，不由想到：“这么好的东西，怎能独自享用？”，一种强烈的感觉驱使我拿过键盘，噼里啪啦地写下此文。
　　我们熟悉的第一代离线浏览器，如经典的Teleport仅能下载主页所包含的文件，却不能打包，同时离线浏览WWW时离不开浏览器，如果要拿到其他的电脑上看，首先必须把文件压缩、拷贝软盘等，十分烦琐；第二代离线浏览器，如WebSnake、Webzip之类，在下载完站点文件之后可以将站点文件打包，又方便了不少，但是离线浏览时依然无法离开浏览器；WebTwin的出现，以其独具匠心的革命性功能，被我“断然”划归为第三代的离线浏览器。
　　WebTwin能把一个站点（也可以是本地文件）的包括图像在内的所有东东转换为Windows的帮助文件，这样浏览时将不再需要浏览器，彻底摆脱了浏览器的束缚，同时，由于帮助文件可以包含图像、文字等，所以一般只需要拷贝单个帮助文件就可以实现浏览地点的转移，比如从办公室带回家中浏览，不用浏览器也一样可畅游网络，很爽吧。正由于这些强大的独一无二的功能，该软件被www.softseek.com授予“编辑选择奖”，应该是当之无愧！
　　WebTwin安装非常简单，当你从http://www.webtwin.com/站点下载完webtwin.exe（2.29M，当前版本为v1.1，如果国外下载有困难，请去我个人主页下载：http://tiger.yeah.net）。双击webtwin.exe即可顺利安装。
　　WebTwin的主画面，界面非常简单，一般操作也只需要四步即可完成：
　　1. 在待处理地址位置（URL to Process）输入站点地址，如：http://www.gxb.com，或者是按“Browse”按钮选择一个硬盘文件；
　　2. 在帮助文件名位置（Help File Name）输入输出文件名，扩展名为.HLP，如d:\gxb.hlp；
　　3. 按“选项（Options）”按钮设置一些参数，这一步可以省略；
　　4. 按一下“Star”按钮。
　　好了，没你的事了，剩下的工作由WebTwin来完成吧。你可以去泡杯咖啡……哦，对了，可不要走远啊。
　　这时，WebTwin首先进行网页文件的下载工作：你可在屏幕上看到WebTwin的工作过程，比如发现的URL数目、已经处理了多少URL、跳过去多少、已经接受的字节数、已经耗费的时间等。如果你发现某个文件下载困难，你可以按“Skip”跳过这个文件，当你不想下载更多的文件时，按下“Enough”即可终止下载，当然，你也可以按“Exit”退出……接下来，WebTwin开始进行HTM到HLP格式的转换，这个速度很快，通常不需要太多的时间，如我制作一个硬盘目录下的HTM文件的HLP浏览方式，文件共有2.93M之多，只经过2分40秒的时间便将整个目录转换为单一HLP文件（机器配置为：Cyrix M2 166＋、32MRAM、336K MODEM），并且文件长度减少为2.74M。例中所叙述的站点（http://www.gxb.com）内容有181K，从下载到转换成HLP文件仅耗时55秒！
　　哇，好酷啊！你现在可以进行全文查找，按主题排序，寻找关键字了。只要帮助能使用的“招数”，这里全部好使。
　　这个版本的WebTwin还支持代理服务器（Proxy，如果你机器安装的浏览器已经设置好了，你可以按设置中的按钮直接导入浏览器的此项设置），磁盘Cache和更多的选项，包括设置检索深度、检索范围（当前服务器，当前域还是整个Internet）、URL最大数目、终止时间等，并包含有最新版本的微软帮助文件编译器（Winhelp v3）。
　　由上所述，WebTwin对于创建离线浏览的站点文件当然是不在话下；对于写作软件的朋友们来说，以前制作帮助文件的烦琐过程，现在可以简化为制作HTM，然后使用这个工具来进行转换，是否可以称之为“另辟蹊径”呢？当然，此软件并非完美无缺，由于受HLP文件的限制，所以不支持FRAME；表格处理也比较欠缺。注意，你下载的WebTwin是一个30天的试用版本，如果你满意的话，不妨去注册。

　　(江苏　顾晓斌)
