从DOS直接入网NT

　　作为NT网的用户，你可能遇到过这种情况：网络硬件已连好，服务器也已安装Windows NT Server，并进行了各项必要的配置，但由于准备入网的计算机没有光驱，所以无法安装Windows 95，自然也无法利用Windows 95实现入网。这时你该怎么办呢？能不能从DOS直接入网呢？答案是肯定的。回到服务器，你将发现Windows NT为DOS环境下的联网提供了解决办法，一经实践，OK!整个过程可分为以下三步：
　　一、制作网络安装盘
　　1．启动服务器的NT Server,单击“开始”，选择“程序”项，打开“管理工具”栏，单击“网络客户管理器”栏。
　　2．在弹出的“网络客户管理器”对话框中，选择第二项：制作安装磁盘组（I）。然后单击“继续”按钮。
　　3．在“共享网络客户安装文件”对话框中，选择第一项：使用已有的路径（E）。然后单击路径编辑框右边的按钮，弹出“网络客户分布目录”对话框。在下拉框中选择“msclient”项，单击“确定”按钮，返回“共享网络客户安装文件”对话框。这时在“路径”编辑框中可看到“\\服务器名\Clients\”，单击“确定”。也可以直接在“路径”编辑框中键入客户软件所在的路径。
　　4．在“制作安装磁盘组”对话框中选择“Network Client V30 for MS－DOS and Windows”。单击“确定”。
　　5．插入软盘，单击“确定”，这时计算机开始制作网络安装盘，总共需要两张35英寸144M软盘。但安装时，一般只需第一张安装盘。
　　二、安装网络客户软件
　　1．启动工作站的DOS操作系统。
　　2．在工作站的软驱中插入第一张安装盘，键入命令行：A:\Setupexe。
　　3．屏幕显示“Setup for Microsoft Network Client for MS－DOS”安装程序提示信息，按回车即可。然后键入一目录以便安装程序拷贝网络客户软件，例如：C:\Net。
　　4．安装程序开始检查系统文件，在随后列出的一系列网络适配器中选择正确的选项。回车后输入用户名。例如：WangPing。最多不超过20个字符。
　　5接下来屏幕上列出四个选项：
　　（1）Names；
　　(2)Setup Option；
　　(3)Network Configuration；
　　(4)The list options are correct。
　　选中前三项中的一项后按回车可以看到该项的详细信息，并可以对某项设置加以修改。例如要改变网络配置，选中第三项并回车，选择“Remove”删除网络适配器，再选择“Add Adapter”添加新的适配器（在随后出现的列表中选择恰当的适配器）。选择“Add Protocol”可以添加网络协议。选择“Network configuration is correct”返回。
　　6．如果对上面的三项设置均无异议，则选择“The list options are correct”项，回车后开始安装。
　　安装程序将修改系统设置，注意configsys文件应包含下列语句：
　　Files=20
　　Device=C:\NET\ifshlpsys
　　Last Drive=Z
　　在autoexecbat文件中应加入下列语句：
　　C:\Net\Net initialize
　　C:\Net\Nwlink
　　C:\Net\Net Start
　　当然你也可以在计算机启动后再执行这几个命令。
　　安装完毕后，重新启动计算机，如果网络初始化成功，你就可以与服务器连接了。
　　三、与NT服务器连接
　　1．C:>Net，屏幕左上方出现连接（Connection）对话窗口。
　　2．按Alt＋P，输入你将访问的网络上其他计算机的共享目录，假如你想访问服务器的光驱，则可输入：“\\My_Server\Cdrom ”。（这里假定服务器“My_Server”的光驱已经设为“共享”，且其共享名为“Cdrom”。）
　　3．按Alt＋V，给你要访问的路径赋予一驱动器号，例如“F”，注意不要与已经使用的驱动器号重叠。
　　4．按Alt＋C，开始连接，如果连接成功，则在当前连接（Current Connection）列表中看到新增的一项：“F：Is \\ My_Server\Cdrom”。
　　5．按Alt＋X退出。
　　6．C:>F:，这时F:成为当前盘符。
　　7．F:>dir，如果在服务器的光驱中已经放入光盘，那么你就可以浏览光盘内容了。
　　现在你就可以通过服务器上的光驱拷贝和安装文件了。当然你首先要做的是：安装Windows 95操作系统。然后在Windows 95中完成网络配置。

　　（四川　李海川）
