《Outwars》的历程

　　《Outwars》（外部战争）是由SingleTrac Studio制作，微软发行的一款3D射击类游戏，类似于有名的《MDK》。它有出色的3D场景及人物，在图形加速卡的渲染下，每个场景更是被表现得淋漓尽致，非常值得一玩。现在就让我们进入这次星球大战吧！
　　游戏背景：
　　许多个世纪以来，人类不停地在搜索其它比地球生存环境更美好的理想地方。越过漫长的时间和空间，我们发现了一个非常好的地方，但它看起来比地球的自然资源更加衰竭，人类被迫到其他行星去开发。然而最近这种状态发生了改变，有几种神秘的种族在其前哨地区开始了疯狂的破坏和掠夺。
　　一支精锐的“殖民防御力量（CDF）”部队已经开始调查这起骚乱，绰号：Dreadnauts。他们有喷气包这类能飞越许多崎岖不平的地形的高科技武器装备，他们保护着的飞船上有一个武器军械库，这使那些士兵们每一个都像陆战坦克一样强大，高质量和高强度的战术训练使他们能够熟练地完成每一项任务。这次CDF部队奉命破获骚乱者的头领并摧毁他们的势力。
　　实用的快捷键：
　　游戏中的快捷键非常之多，按F1可进行快速查阅，以下只列出几个使用频率最高的键：
M：显示地图 R：显示雷达 0：显示过关任务
1～6：选择武器　 ,:上一个武器 .：下一个武器
Alt＋H：高质图像显示　 Alt＋PgUP：增加分辨率
　　单人游戏（Solo）：
　　适应性任务：在3分钟内到达飞船。
　　此任务是要你熟悉雷达的使用，目的是返回飞船“Nav Brave ASAP”，时间很充裕，千万别停下来吸引敌人的火力，跟着雷达上的蓝色指示器走就行了。

Oasis星球

　　1. 跳跃训练
　　在规定时间内跳过所有的支撑点，时间很充裕，向蓝色指示器所指的方向走就行了，这儿唯一的难点就是不好掌握跳跃的力度，最好用鼠标右键连击进行平行飞行，在着陆时存在着时间差、视角差等因素，多练练就好了。
　　2. 射击训练
　　在有限时间内破环所有的规定目标，顺着指示，在三个支撑点上分别破坏3个钢架上的移动火焰目标，然后在岩洞里破坏4个固定目标，出洞后飞到前面的平台上破坏1个移动目标，在另一个平台上击落1个飞行目标，最后在山地上炸掉2个贮油塔（要用定时炸弹）。
　　3. 自由训练
　　消灭3个敌人，敌人的火力与你一样，开始时最好是避开他们，然后再找机会射击，这关不用费心去找他们，他们会自动来找你，四周有不少补给物品。
　　4. 乱枪之下
　　适应与同伴合作作战，破坏两个贮油塔及一个萨姆发射车。朝前飞行到对面的山坡，左拐不远处可看到停着的萨姆发射车，将其击毁。根据雷达显示，贮油塔就在前方，果然，飞往塔顶，用定时炸弹来搞定，前方还有一个，不会很困难吧。
　　5. 撤离
　　吸引敌人的火力，保护受损的飞船。这是跟跳跃训练那关同样的地形，虽然不是很复杂，但比较难。我们的飞船被敌人击落在小河尽头，你必须快速到那儿进行保护，从起点跟着指示器穿过两个瀑布看见小河后降落，继续朝前走到小河的拐弯处发现我军的飞船，可射击它来看看它还有多少生命值。飞船会自动恢复，当然这需要你来争取足够的时间，在雷达上可以看见背后有密密麻麻的太空蜘蛛，一定要阻止它们的进攻，我方还有队员与你合作，但他的射击对你同样有效哟！
　　6.焦土
　　破坏在主连器周围的三个冷却器，然后在它即将爆炸前返回。打开地图，按照地图所示朝前走，右拐后，朝上看可以看见半空中有一条长廊，飞上去顺着长廊继续前进。在每个长廊段都有个八边形的连接柱，可以下落到长廊内部行走，当然在路中会有许多太空怪兽。走到尽头后可以发现一个灯塔状的柱子（如果开始在长廊顶上行走，那么在最后一个连接柱时必须落至内部）。走到廊边朝上看发现上面有一层回廊，飞上去，找到通往另一个柱子的入口。在这个柱子底部也有一层回廊，但不是我们要走的。而在入口与底部的回廊之间有一块突出的隔台，这就是目的地连接器的入口处了，走进去你就可以看见三个被连接的锤形冷却器，飞上去用定时炸弹将其炸毁。此时雷声大作，一片混乱，赶快在2分钟之内离开这儿并回到原来的一个密道内。此任务的关键在于熟悉地形。
　　7. 调查设备厂
　　潜入CDF的设备厂并定位滑翔机翼的位置，将平台升至设备厂顶部。朝前走，跳下平台，通过传输通道后会发现两个滑翔机翼，继续前行，跳下隧道后继续朝前走，再次下跳后消灭几个太空怪物，发现了一张通行卡。拾起它后向上飞，升至原跳下的入口时再往上飞，这时会看见有一扇门的通道，走过去门自动打开，这就是设备厂了。在门左侧的地上会发现一块类似于下水道的盖子，走上前一踩，原来是一个升降机关，这时平台开始上升，会有许多怪物来攻击你，但千万不要走开，否则平台会停下来，当平台升至原入口处时任务完成。

Anubis星球

　　1. Hobb的殖民地
　　找到所有的殖民地。在空地上可以发现第一个殖民地（屏幕指示呈白色处），然后直走、再左拐后顺着道路至尽头会发现一个废弃的坦克履带，跳上去，根据雷达指示向上飞会找到第二个殖民点。从平台跳下后右拐，在有三个灯塔处向上飞至平台又会发现一个。然后跳到下面的起重机上，依指示飞向对面的平台上，发现第四个。回到起重机处，蓄满燃料，越过护栏，飞至对面的一块陆地上，前行会发现一个洞状通道，进入后左拐，找到第五个殖民点。再前行通过下一个通道，左转在地上有一个弹夹，这是第六个。弹夹后方又有一通道，在三岔路口处往右拐，会发现是一条死路；但抬头一看，看见半空中有一伸出的平台，飞上去，过第二个平台后左转，后又经过一次左转、着陆，发现了第七个殖民点。右方尽头处有一通道，在最深处有第八处殖民点。在此通道的途中还有一处通往上方陆地的通道，飞上去，右拐，按照雷达提示一直前行，找到第九处殖民点。
　　2. 油塔
　　破坏所有5个油塔，保护科技飞船的安全。此任务在三个分开的陆地上进行，每块大陆上有1～2个油塔，在最短的时间内炸掉油塔是保护飞船最可靠的保证。一开始就飞到对面第一块陆地，右边就有一个油塔，它的样子像一个大鸟巢。用定时炸弹炸掉它，然后跟着雷达指示继续向前，可以毁灭其它4个油塔。在行动期间，有很多的敌人，在天空中还有敌人，但你处于高速运动中，他们很难击中你，所以不必去管他们，总之速度要快，否则飞船就……
　　3. 飞行
　　消灭空中的敌人，在山谷中破坏所有的机枪塔，等待我军飞船打开通道。这是一次飞行任务，在空中的控制很难掌握，特别是要面对很灵敏的太空蝙蝠更是毫无办法，还是慢慢跟着它绕圈子吧！在空中，方向显得很重要，跟着雷达飞是最好的办法，注意不要撞着山峰，不然机翼会爆炸。当看到有座山峰上有三个耸立的灯塔时，就表明你到目的地了。它看起来像一座火山，中间全部是深渊，四周耸立着5座光圈炮台，我们的任务就是要击毁它。在深谷中有伸出的平台，如果你想着陆，可以去撞山峰，这样等机翼爆炸之后用燃料包将你送到陆地上。用一般的机枪即可以搞定炮台，击毁后，我军的飞船会在山上辟出一条通道来，这是一次无比艰巨的任务。
　　4. 隧道
　　消灭沿线的所有敌人，到达Nav point。这次任务很明确，到达终点Nav point，由于隧道只有一条路可走，所以不存在迷路的问题，可能洞里有些暗，那就打开地图照着地图走吧！沿途的敌人非常多，天上飞的，地上爬的都有，伙伴射击的火光也老是在你的身边乱闪，还是跑远点再攻击敌人吧。Nav point的特征就像一座被蛇盘旋的小山，中间有一大条口子，不会很难找的。
　　5. 洞穴
　　找到并摧毁洞里女王蜂的巢。从上次任务结束时的那个大口子掉下来后，打开地图，会看见有两条路可选，选择从右走。行至此层尽头会跌至下一层，继续左行到本层的尽头再次下跌。顺着路线游过一片水域后来到这层的尽头，抬头会发现有通道与上层相通，赶紧飞上去，行至此层的最深处又会发现一条大口子，先别忙跳下去，走到落口边缘，头朝下成俯视视角，再往下跳。因为下面不是平地而是深渊，只有两块石钟乳形状的圆形平台可以落脚。站稳后可以发现在洞穴的顶部有一突出的女王蜂巢，袭击它会用光圈来反击你，这就是最终目标了。

Ragnarok星球

　　1. 撤离
　　找到同伴，炸掉北面的巨石，撤离这个地方。我们的一个同伴在左边受到敌人的围困，赶快过去支援她。跟着雷达可快速找到她的所在。然后朝北面进军（在游戏前任务介绍时右面的地图上有北面的指向），在最尽头发现挡住出口的巨石，用2个定时炸弹炸毁它之后撤离。
　　2. 粒子弹
　　摧毁敌人的控电中心及粒子弹。右转，按着雷达指示，最好是打开地图一直前飞，直到看见控电中心，就是由五根电塔组合成的五边形状的建筑，地图上的显示是蓝色圆圈。暂时别管它，先把敌人消灭干净。然后继续向前飞行，这时跟着雷达指示就最方便了。最后的目的地是一个白色平台，平台中间是深渊，勇敢地跳下去！下面好像是一个殿堂，四周有不少的通道可以进出，但只有一条通道是最长的，粒子弹就在里面。一直在壁顶闪光的就是它了，摧毁它之后重新回到原控电中心，在飞过它的一刹那扔一颗定时炸弹就搞定了。
　　3. Ulysess部队
　　消灭敌人的所有攻击力量。这次任务比较简单，消灭完有限的敌人就可以过关，敌人有坦克和太空蜘蛛，行踪不定，你大可到处走走，看到有火光就过去吧。
　　4. 援救
　　找到Russell上校，并在他那儿得到情报室的通行卡，然后在情报室偷走文件。开始时向左，跳到最底层，会发现地上有一绿色的按钮状机关，踩上去下跌至下一层。如此反复，经过三次下落到底层，干掉一个怪物，这时有两条路可以选择：前方是有一障碍物，可飞过去继续前行，后方，则可看见墙上有一个白色的机关。向后走，用枪击毁白色机关继续前行。在前方壁上有红白两个机关，还有一个可通往下层的跳台。从跳台跳下去，踩住绿色机关经过两次下落，再击毁白色机关最后可到达“引擎系统”。在此系统周围有一条很长的通道，从通道进去，走到尽头再往上飞，经过两次上飞可到达动力室，右行经过连接通道到另一个动力室，从中间十字形空隙跳下去，一直落至旋转桨叶层。环顾四周可看见壁上有一扇网状的门，用枪击毁，可开启通往Russell上校囚室的通道。在屋里找到了Russell上校，屋里的左上方有一台绿色的计算机，上面有通往情报室的通行卡。获得卡后按原路返回到引擎系统，再一直下跳到最底层（一个锅状凹形层）。在雷达指示下找到情报室入口的白色机关，门开后进入情报室，飞到左边绿色计算机的最上面，右转，朝前飞可得到文件。出情报室会有一个骷髅怪兽来袭击你，搞定它就可完成任务。
　　5. 救援小组
　　护送装甲车辆到太空飞船，抵御反飞船舰队的袭击。前方的两架装甲车会自动地行驶到不远处的太空飞船内，你必须保护它们。向前飞可以发现有3个怪物在守候着，干掉它们便可暂时休息一下。等到装甲车上了飞船后，紧跟着飞船，因为此时空中有另一个敌军飞船，必须把它击毁，这次的任务很简单，可轻松过关。
　　6. 竞赛
　　找到并开启引擎中心的自爆装置，在5分钟之内撤回起点。这是第4关时的地图，相信你早对这张地图了如指掌，先到引擎中心发射激光束的地方开启自爆装置，再按原路返回到起点就是了，如果实在忘记了原路，就用“Go Home”吧。

Juggernaut星球

　　1. 统领
　　阻止任何敌人进入警报区域，消灭统领身边的所有卫士。一开始前方便有不少的敌人，它们的行动时刻关系着任务的完成。用最快速度消灭它们，千万别让它们进入警报区（中间有一个绿钮的圆形区域）。歼灭所有的敌人后从左边的一条通道继续向前行，会来到一个存放着许多箱子的大厅，再消灭其中的所有敌人，在最尽头的地上发现了一个鹅卵形的地下通道，跳下去，干掉桥上的两个怪物继续向前走，会来到由多个支梁构成的空旷的大厅，消灭完所有的敌人，在最前面的一个平台上有两个大怪物保护着一个婴儿大小的怪物，这就是统领了。不必攻击它，只需将它的二个卫士消灭掉就行了。
　　2. 关闭气体供给
　　协助工程师完成他们的工作。工程师将要给反应器中心进行关闭动力工作，你的任务就是消灭沿线的敌人，协助他们的工作，只需一直向前跑，遇到敌人开枪就是了。
　　3. 交叉火力
　　破坏敌人的女王巢。沿途飞行到最尽头（注意沿线的怪鸟，它会令你很头痛，最好是快快逃离它）。墙上有一圆形的通道，飞进去可以看见伸出的女王巢，由于旁边还有对你不利的火力，所以还得小心为妙。击毁它就完成了此次任务，真是太简单了！

Deadworld死亡星球

　　1. 守护
　　保护引擎中心，削弱敌人所有攻击力量。向对面绿色亮点飞去，原来这又是绿色机关。冲进去再经过两道绿门之后向左转，击毁白色机关，向左通过绿门，从向下的通道掉下去。再一直前行经过两扇绿门右转，最后经过两道绿门就可以到达引擎中心了。此时的中心已被敌人打掉了不少生命值，你必须最快地投入到战斗中去，底层的敌人先别管它，先把激光束周围的敌人消灭干净再说。攻击这种大怪物的最佳武器是导弹枪，两枪就可以解决一个，总之不能让敌人摧毁引擎中心，记住：这次的任务是击灭敌人的所有攻击力量。
　　2. 取回
　　进入设备研究室，取回Inertial Dampening。从右边斜下方的通道进入，然后一直前行，在尽头处跳下平台，继续往前走，又在尽头处下跳。落到底层后发现前方有一扇可自动开启的门，走过去，等门开启后看见下方是一个装有紫色液体的大池子，转过身来，背对着液体往下跳。在下落的途中会出现壁上还有一扇可自动开启的门，飞进去，在一个四方形平台上可找到Inertial Dampening。取走它后按原路返回到起点。这时候，你的行动已被敌人发现，唯一的办法只有干掉这些怪兽。
　　3. 渗透
　　破坏五个热能接收装置，破坏之后迅速赶到出口。这次又是一次飞行任务，非常之艰巨。当飞到五角建筑物中心时，打开地图，热能接收装置的图标在地图上呈五边形，在中心以外有三个很明显的建筑物，还有两个看似被其它建筑物裹着，其实这两个在五角建筑的最底层的山洞中。根据地图即可大致走到接收器的位置，反正不是上就是下了，雷达有很清楚的提示。当破坏了这五个接收器后，五角建筑的中心会爆炸，炸出的大洞就是你的出口了，飞过去吧！
　　4. 最后决战
　　找到敌人的能源吸取点，摧毁敌人的战机。勇敢的往下跳吧，这是最深的一个深渊了。中间有许多柱子的周围都有一突出的环形隔层，如果你不停地往下跳，等到最底层时，生命值就快没了。所以不时的休息一下肯定有好处。在最底层会发现一个方形的大洞，可别再往这个洞跳了哟！在旁边会找到一个能源采集点。这时敌人的飞船怎么会放走发现它们秘密的人，最后的决战终于开始了。敌人的火力非常强劲，用激光加光圈炮，唯一的战术就是打一枪换个地方，多与敌人周旋，10分钟的限时很充裕。
　　随着一声爆炸声，敌人最后的基地总部被炸毁了，我们的部队逃出了死亡星球，这次的调查进行得非常顺利，虽然有些不小的挫折，但我们的将士始终克服了所有困难，最后胜利总是属于我们的。
