9811请你编程

　    本栏目由苦丁香公司协办
    本期请你编程是一个古老的数学问题，看来读者们对这样的问题都很感兴趣，埃及分数是要求将一个正的真分数分解成几个不同的单位分数的和，由于出题时没有说清楚，有的读者将分数分解成为几个相同的单位分数的和，因此求出的解不符合原题的要求。我们也为自己在出题时考虑不周向大家道歉，下面来看看江苏读者谢文丽的解答。
    编程思路：若真分数的分子a能整除分母b,则真分数经过化简就可以得到埃及分数。若真分数的分子a不能整除分母，则可以从原来的分数中分解出一个分母为b/a＋1的埃及分数。用这种方法将余数部分反复分解，最终可得到结果。
    源程序：
    main()
    {
    longinta,b,c;
    while(1)
    {
    printf(″请输入一个正的真分数(a/b):″);
    scanf(″％ld/％ld″，＆a,＆b);
    if((a==0)||(b==0)||(a/b&gt;=1)/＊判断输入正确与否＊/
    continue;
    else
    break;
    }
    printf(″此分数可分解为：″);
    while(1)
    {
    if(b％a)/＊若分子不能整除分母＊/
    c=b/a＋1;/＊则分解出一个分母为b/a＋1的埃及分数＊/else
    {
    c=b/a;/＊否则输出化简后的真分数＊/
    a=1;
    }
    if(a==1){
    printf(″1/％ld\n″，c);
    break;/＊a为1标志结束＊/
    }
    else
    printf(″1/％ld＋″，c);
    a=a＊c－b;/＊求出余数的分子＊/
    b=b＊c;/＊求出余数的分母＊/
    if(a==3)/＊若余数为3,输出最后两个埃及分数＊/
    {
    printf(″1/％ld＋1/％ld\n″，b/2,b);
    break;
     }
    }
    }
