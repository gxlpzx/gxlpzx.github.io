中、日、韩文处理多面手Global IME 5.0

　　由于工作需要，本人除了大量使用英文函件外，还经常与日本及我国港台地区客户互发电子邮件，共享数据文件。在以前，由于语言字符集的不同和汉字多内码的障碍，为了处理这些任务，笔者不得不在一台电脑上安装多种语言平台的 Windows95(98) 操作系统，频频执行“重启动，换系统”的操作，既浪费时间又影响机器寿命，更难容忍的是难以在一种平台上达到多种资源的共享。虽然已有很多优秀软件如南极星、四通利方、中文之星、MagicWin98、Ucwin Gold 等等，可以解决同一平台上的中、日、韩多内码汉字显示问题，为需要浏览不同内码文件的用户带来了福音。但它们的效果并不能令人十分满意：有的不够稳定（有时不能完全正确地识别不同内码）、有的略显低能（例如把本应保持原貌的浏览器菜单条和工具栏上的文字也转换成乌七八糟的乱码）、有的转换后的字体相当难看（让人顿失上网的愉悦）……而它们最大的缺点则是“只准眼看，不能手干”——只是解决了显示和浏览的问题，基本上都没有解决多内码输入的问题。然而，在如今多元化的信息时代，我们不仅仅是被动的受众。如果不能进行多内码输入，当你需要向使用BIG5码平台的港台重要客户发送一封中文电子邮件时，如果对方碰巧没有内码转换软件，你能强求对方为了你而去购买一套吗？如果你的客户在日本，而他又不懂英文和中文，就算他拥有一套很不错的多内码转换软件，可又有什么用呢？其实，如果不能进行多内码输入，你甚至无法享用因特网上的一些最基本的功能，比如基于关键字的搜索与查询。
　　现在好了，利用微软新近推出的 Microsoft Global IME 5.0 多内码网上浏览和输入工具,上述问题你都可以迎刃而解。下面介绍Golbal IME 5.0的安装注意事项及其使用方法。
　　一、Global IME 5.0的安装注意事项
　　Global IME 5.0 目前是免费提供的，你可以到微软站点 http://www.microsoft.com/ie_intl/cn/ 处按提示下载。它是作为微软IE的一个增强组件发布的，因此，你的系统中应该安装它。另外，Global IME 5.0 对系统的要求是：386以上微机（建议486以上）、8MB以上内存、Windows95、Windows98 或 Windows NT4.0 操作系统。在上网下载某种语言的 Global IME 5.0 之前，你应该先查看一下你目前的 Windows 系统中是否已经装有该语言的套件，如果未安装该语言套件，则应在选项中同时选中该语言套件安装，只有在安装了各自的语言套件之后，不同语言版本的 Global IME 5.0才能工作。安装程序一般都能正确地侦测出你目前的 Windows 平台，如非必要，你不要更改“选择语言”项的默认值。在“选择选项”下拉列表中，列出了你所能选择的全部选项：简体中文、简体中文加语言套件、繁体中文、繁体中文加语言套件、日文、日文加语言套件、韩文、韩文加语言套件，笔者强烈建议你看清这些选项，默认值是你现行平台的内码组件，而这个组件你是完全不需安装的！你每次只能选择一个组件安装，每次需用1～2小时。
　　二、Global IME 5.0的使用方法
　　现在，你再到网上冲浪，感觉可大不一样了：你可以打开 IE ，连接上海在线先了解国内行情，同时开一个窗口到台湾看看报纸，另外再开一个窗口与远在东京的朋友用日文实时 Chat……只要在任务栏上轻轻一点，瞬间即可转换。如果你嫌上网聊天的费用太贵，你可以离线打开 Outlook Express,撰写几份新邮件，第1封用日文联络你的日本客户，第2封换成韩文与你的韩国朋友谈天说地，末了，可不要忘记用 BIG5 码问候一声自己的港台同胞，甚至，你可以把这些内容集成到一封邮件里，发给一个“国际家庭”。这一切，操作起来都是十分简单：Web 冲浪时，你只需单击（各自窗口的）工具栏上的“字体”按钮（或“查看”菜单中的“字体”命令），在弹出的语言选项中作出相应的选择即可完成不同内码页的浏览，如果要在浏览器窗口内输入某种语言的文字，只需单击任务栏上的输入法按钮，选择相应语言的输入法即可（安装完某种语言版本的 Global IME 5.0 后，系统会自动在任务栏的输入法选项中添加该种语言的输入法）。同样，你要用 Outlook Express 打开一份某种内码的电子邮件时，只需单击工具栏上的“语言”按钮（或“格式”菜单中的“语言”命令），从弹出的内码选项中作相应选择即可（如果你不能肯定，可以一一试验，直至满意为止）；如果你要用 Outlook Express 撰写一份某种内码的电子邮件时，只需单击“格式”菜单中的“语言”命令，从弹出的内码选项中作相应选择后，再单击任务栏上的输入法按钮，选择相应语言的输入法即可。
　　另外，如果你对排版效果不太在乎，你完全可以让 Global IME 5.0利用 Outlook Express 成为一个可以混排简体中文、繁体中文、日文、韩文并可打印输出的多功能文字编辑器，同样功能的字处理软件，目前在市场上的售价可是贵得惊人且极不容易找到。即使你很在乎排版效果也没关系，在字体支持的情况下，你可以先在 Outlook Express 中编辑，然后再拷贝到你的字处理软件（如 Word97、WPS97）中进行处理。
　　怎么样？Global IME 5.0的功能够强大的吧。遗憾的是，它目前只能在启动 Internet Explorer 或者 Outlook Express 后才能使用。也许将来，微软如果能够打败网景（天知道！），逼走南极星等等（可能吗？）之后，会让 Global IME 适用于所有应用程序吧。不过那时，微软可能要开出一个天价了，你要有所准备。还是趁早行动吧，毕竟现在提供的还是一份免费的午餐。
