Win98的启动盘的新特点

    随着Win98的推出，对启动盘作了重大改进，使它变得更加实用、好用。Win98启动盘具有以下新特点。
    一、增加了通用光驱驱动程序
    如果用Win98启动盘启动机器，将会出现含有三个项目的多重引导菜单：   
    Start computer with CD－ROM Support
    Start computer without CD－ROM Support
    View the help file
    选择第一项就可以加载通用光驱驱动程序，此驱动程序能支持绝大多数ATAPIIDE和SCSI光驱。这样你就可以直接从光驱上安装Win98或其它软件，这对于经常攒机的朋友尤其方便。
    二、创建虚拟磁盘(RAMDrive)
    在启动过程中，还将创建一个其大小为2MB的虚拟磁盘。由于在单张144MB磁盘上没有足够的空间存放诊断工具以及CD－ROM驱动程序，因此这些文件被压缩在EBDCAB文件中，启动时再释放到虚拟磁盘上。以后运行磁盘扫描、分区、格式化等工具软件的速度将大大加快。
    三、增加了EXT.EXE程序
    EXT.EXE实际上是EXTRACT.EXE的外壳程序，使用EXT.EXE程序可以很容易地将压缩在CAB包中的文件释放到硬盘上，运行EXT后按提示输入Win98安装文件的位置、所要释放文件的位置和存放文件的位置即可。而直接使用EXTRACT程序则比较灵活、复杂，适用于高级用户。
    即使你不是Win98用户，相信Win98启动盘也会给你使用电脑带来很大方便。
    附：Win98启动盘的制作方法：
    方法一：在安装Win98时根据屏幕提示插入软盘制作Win98启动盘。
    方法二：在“控制面板”中选择“添加/删除程序”，在弹出的“添加/删除程序属性”对话框中选择“启动盘”标签，单击“创建启动盘”按钮，根据提示即可制作Win98启动盘。(湖北 张杰)
