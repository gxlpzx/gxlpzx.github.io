最快的MP3制作工具--MPLifier 0.52  

　　自行制作MP3音乐，相信已成为一些电脑音乐爱好者的日常工作之一，但MP3音乐的制作却是很占用时间的，制作一首5分钟的MP3音乐，通常都要等上十多分钟才能完成。最近从1998年第1期《电脑报配套光盘之PC世界》中找到了一个十分快速的MP3制作工具MPLifier 0.52(以下简称MPLifier)，使用该工具，我在短短的3分钟内就完成了对一首5分钟的WAV文件的压缩转换，而把该首MP3音乐还原为WAV文件时，也只用了大约3分钟左右即告完成。如此好物不敢私藏，速速与大家分享。
　　首先介绍一下MPLifier的系统设置。在MPLifier窗口中的“Priority Class”一项，可以设置CPU的资源占用量，为了实现快速压缩，建议把滚动条拨到High状态，让CPU尽可能100％地进行压缩工作。
　　按“Configuration”按钮进入另一设置窗口。在该窗口，“Destination Directory”是设置压缩或还原后新文件摆放的位置，“Path to Tompg.exe”是设置MPLifier调用的Tompg.exe文件的存放位置，“Show message box on error”是设置出错时给出信息，“Delete source files after encoding”是设置在压缩或还原完成后，自动删除源文件以节省硬盘空间，“Convert to Source Directory”是设置把压缩后的新文件存放在与源文件所在的同一目录中，此时上面的“Destination Directory”一项失效；在“MPEG Layer”中提供了MPEG Ⅰ、MPEGⅡ、MPEGⅢ(即MP1、MP2、MP3格式)；在“Channel Mode”中提供了Stereo、Joint Stereo、Dual、Mono的模式；还有Bitrate项(其默认值为128KB/s)提供了大量的可选项。
　　以上是对MPLifier设置的一些介绍，下面就来说说如何使用它制作一首MP3音乐。
　　首先把喜爱的CD音乐用音轨截取工具进行截取，并保存为WAV格式，然后运行MPLifier，出现操作窗口，该窗口一共分为左中右三部分，最左边是“Source File Selection”(源文件选项)，从中可以选择源文件所在的路径和目录；当选定了源目录后，中间的“Sound Files”窗口不会显示出该目录下的MP3及WAV文件，之所以出现MP3格式的文件，是因为MPLifier除了可以把WAV压缩为MP3文件外，还可以把MP3还原为WAV格式文件。这时选择目标文件，双击欲转换的音乐文件，使其出现在最右边的“Converter queue”窗口中，选中该源文件，按一下“Start”按钮就可以开始进行压缩了，在窗口下方可以观察压缩的进度。
　　除此以外，MPLifier还可以将MP3还原为WAV，操作与压缩MP3文件基本相同。(广东　陶　健）
