CD-R光盘使用技巧大公开

　　过去，用软盘可以备份大量文件，今天，不要说多媒体作品，就是插入几张图的WORD文档，软盘也难于胜任。用CD－RW或CD－R光盘作存储介质是越来越有必要了。
　　CD－RW的“RW”是“ReWritable”的缩写，即可重写；而CD－R的“R”是“Recordable”的缩写，即可记录的意思。两种盘在使用上的共同点是都可写、可读，不同之处有两点：第一，CD－RW上的信息删除后，空间会释放出来供重写，这和我们熟悉的磁盘没有什么两样，而删除CD－R盘上的文件只有白搭，空间是不会释放的；第二，CD－R上的信息资料可以被组织成ISO 9660标准的格式，以便在普通的标准光驱上读出，而CD－RW盘不能这样处理，只能在CD－RW光驱上读出。
　　使用CD－R盘有几点很值得注意，这里以在惠普的CD－Writer Plus 7100i CD－RW光驱上的使用为例，谈谈光盘特别是CD－R盘的使用特点和注意事项，也许有助于初用者更上手地使用它们，至少可以避免某些损失。
　　硬件安装好后，安装随机软件DirectCD,在任务栏右边会出现DirectCD图标(以下简称为CD－RW图标)。DirectCD的作用是：格式化光盘；让我们可以用与磁盘读写相同的方法读写CD－RW或CD－R光盘；从CD－RW光驱弹出光盘；将CD－R盘上的文件存储格式转换成ISO 9660格式以便该盘可以在标准光驱上使用，或将ISO 9660格式的CD－R转换回来，以便在CD－RW光驱上继续写入新的信息。右击CD－RW图标调用DirectCD，就可以实现以下的操作任务。
　　一、格式化和查看光盘资源使用情况
　　在CD－RW驱动器插入的如果是未格式化的CD－RW或CD－R盘，当你右击CD－RW图标，弹出的快捷菜单有两个选择：
　　1.Format,就是格式化。不管什么光盘，使用之前都要格式化。选中这个命令，输入11个字符长度以内的“卷标”后，就开始格式化。CD－RW格式化需要1小时左右，CD－R只需20秒左右。
　　2.Properties，就是查看资源。选择此命令只是显示两个信息，第一是光盘的类型，即CD－RW或CD－R，第二是光盘的可用空间，即100％未用。写CD－R时，随时查看其剩余空间是很有必要的，可以避免损失。
　　二、取出光盘
　　在CD－RW光驱插入的如果是写有信息的光盘，右击CD－RW图标弹出的快捷菜单就多了一个Eject选择，就是弹出光盘。我们要取出光盘时，应该使用这个命令。这很重要，因为使用光盘读写时，需要在内存中保存其文件格式的特定信息，这些信息必须在我们选中Eject命令时写入光盘。这和我们使用WIN95时，在断开电源之前，要先执行“关闭系统”是一样重要的。如果你没有使用Eject命令取出光盘，欲写入光盘的信息就可能丢失，更严重的是，如果这是CD－R光盘，你本来想以后在标准光驱上读的，现在就没有办法达到这个目的了。
　　三、让CD－R光盘在标准光驱上可读
　　在CD－RW光驱插入的如果是已有信息的CD－R盘，Eject命令就更重要了。选中它后，有两个选择：
　　1.Leave the disc as it is so that it can be read and written,意即让该盘仍为可读写。如果这盘还有大量的空间并且你还准备以后用它继续刻录信息，就请作出这个选择。
　　但要重点说明的是第二个选项：
　　2.Organize the disc so that it can be read in a standard CD－ROM drive.Note:You will have to make the disc writable to add more files to it。这条信息的前一句说的是，CD－R盘将要被处理成其它标准光驱能读的盘，后一句是提醒：你很可能需要往该盘写入更多信息。这意思很清楚，被处理成标准光驱可读以后，就不能再写入信息了。
　　使用DirectCD格式化的光盘，不管是CD－RW还是CD－R，都只能在CD－RW光驱上读。而我们多数使用的CD－ROM光驱是标准化产品，不管它是什么品牌，都能阅读符合ISO 9660格式标准组织、储存在光盘上的信息。为了让其它机器上的标准光驱能够读出CD－R，就要额外地往CD－R盘上写入一些特定的、以便与标准光驱兼容的信息，就是ISO 9660格式。具体步骤就是在Eject选项中，选择这第二项。
　　为了顺利地写入这种特定的信息，必须注意以下几点：
　　1.Eject选中之后，还要在其弹出的选项中选择“Enable ISO 9660 capability”，这可确保CD－R光盘在弹出之前被写入ISO 9660的存储格式。
　　2.在CD－R盘上写入这种格式，第一次需要22M的空间，写入过程需要2分钟左右。
　　3.CD－RW盘并不支持这种格式(但一种称为“MultiRead”的新型的多功能光驱是可以读CD－RW的)。
　　4.如果写入过多信息，第一次就没有留足22M的剩余空间，或在写入时断电、死机，这光盘就不可能被处理成标准光驱可读的了，因为ISO 9660格式标准所需要的信息将无法得到或没有足够的空间写入。
　　5.有一些CD－ROM可能读不出这样处理过的CD－R。碰到这种情况，可以进入“控制面板”，在“资源管理器”的“系统属性”中选中“性能”的“高级设置”，单击其中的“文件系统”的“CD－ROM”，将“优化访问模式”设为“不预读”。如果还不行，则可能是CD－ROM本身的问题了。
　　CD－R盘写入ISO 9660标准存储格式的信息后，能不能继续用于写入呢?
　　被转换成ISO 9660格式的CD－R盘，如果还有剩余空间，将这格式转换回来，就可以继续写入。具体做法是：将它放进CD－RW光驱，右击任务栏的CD－RW图标，在弹出菜单中选“Make Writable”就可以往盘上写入新的信息了。这里也有几点要注意：
　　1.写入了新信息的CD－R，已有的信息在标准光驱上虽然照样可以读，但新写入的信息却读不出来。怎么办呢?你要再次将它放进CD－RW光驱，右击任务栏的CD－RW图标，再次选中Eject的第二个选项，经过大约2分钟的整理后，自动弹出。标准光驱就能读出新写入的信息。
　　2.每一次这样的处理需要额外13M的空间，因此往CD－R写入时，要注意查看其未用空间是否大于13M。比如说，剩余空间只有20M，那么你最多只能再写入7M的信息。

(福建　郑炯扬)
