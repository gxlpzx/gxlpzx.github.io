当i740碰上Voodoo2

　　3Dfx的Voodoo2推出已经有一段时间了，与此同时，Intel的i740也开始在国内市场“风风火火闯九州”。一种“情愁”，两种滋味，作为最终掏钱选择产品的用户来说，自然是希望将这两件“宝物”一网打尽。只是无奈银子有限，希望花最少的钱而取得最大的收益，你的这种心情我们理解！于是就有了这篇──i740 VS Voodoo2……

　　先睹i740

　　鲲鹏工作室这次共挑选了中凌、耕宇、华硕、丽台4种品牌的，以i740为主芯片的3D加速卡进行对比测试。经过反复比较，觉得这4块卡的主要规格大致相同：清一色地具备2D/3D处理能力；支持MPEG1/2软件解压缩技术和AGP2×规格；在应用程序接口（API）方面全部支持OpenGL（Windows NT）以及DirectX，内建24Bit 205MHz RAMDAC。在3D功能表现方面，除支持Alpha透明化、MIP贴图处理、Z－buffer、边缘平滑处理、硬件材质贴图等常用3D功能外，Intel还针对如何使PentiumⅡ性能得到最佳发挥而特别开发了Hyperpipelined 3D技术。在这项技术中，内建了三项特殊功能：平行资料处理（PDP－Parallel Data Processing）、精确图素插补（PPI－Precise Pixel Interpolation）、直接内存执行（DME－Direct Memory Execution）。i740允许使用SDRAM或SGRAM，送测的4块加速卡全部为8MB 100MHz SDRAM。据厂商“交待”，虽然SGRAM在效能上比SDRAM高出不少，但由于价格因素，IT厂商在生产i740加速卡时，大都只会选择以下3种规格：8MB SDRAM、4MB SGRAM以及8MB SGRAM。就在我们开始正式测试前，Intel公司通过非正式渠道对外宣称，i740只有与440BX芯片组组合才是最佳搭配。惨啊！因此，为了公正、充分地测试出i740的效能，笔者强忍悲痛，自费购置了一块芯片组为440BX的升技BX6主板。测试主机：CPU，Pentium Ⅱ 300MHz；内存，64MB SDRAM。
　　相信90％的朋友选择3D加速卡的前提都是以“游戏”为目的，故而，测试3D加速卡能否在游戏中发挥正常的作用就是测试的重点了。4家公司的i740加速卡在Windows95中都是通过Direct3D来使硬件加速的，因此只要测试游戏能够支持Direct3D，i740在游戏中就可以顺利执行。
　　我们测试的第一个游戏是《Fight Force》，这是一个完全移植于大型街机的3D动作格斗游戏，可以充分展示出3D加速卡在电脑游戏中的神奇威力。《Fight Force》是通过Direct3D来使硬件加速，用i740来测试时，执行Gameindex程序，游戏会自动进入DEMO画面，然后在左下角可以显示出测试数据。
　　我们测试的第二个游戏是目前比较普遍在测试显示速度时用的《恐龙猎人》。这也是个动作游戏，有点类似于《QUAKE》，但画面场景色较之《QUAKE》却有几个档次的进步，尤其是游戏中运用了相当多的3D特效，因此，许多Voodoo的使用者都把它作为测试显示速度的依据。
　　测试的第三个游戏是由法国UBI SOFT公司出品的《G－POLICE》（特警雄鹰）。这个游戏曾被美PC GAMER杂志评为“世界上最佳视觉游戏”，游戏画面的豪华程度不用笔者再详细介绍了吧。游戏中以空中飞行及作战为主，绝对是用来测试3D加速卡性能的最佳选择。测试方式是执行游戏中的DEMO功能，最后再记录平均值。
　　除了对游戏的测试以外，我们还使用了Final Reality来进行了D3D的测试。Final Reality有一个好处，就是可以测试AGP卡的传输效率。Final Reality仅能测出11项3D功能，不过这些对于游戏来说已经足够用了。同时，在3D Winbench的测试中，全部41项测试，i740通过了38项，仅Fog Table Liner、Fog Table Exponetial以及Anti－Aliasing没有过关。
　　从测试的结果来看，4块i740加速卡在应用效能上可以说是不分伯仲。不过，华硕和丽台虽然使用了自己的驱动程序，但在效能上却并没有什么较大的优势。在测试的整个过程中，笔者最突出的感觉就是，i740在所有雾化的场景中表现均不是很顺畅，究其原因，可能是由于不支持硬件的Fog Table功能，启动了软件模拟而白白消耗了大量的系统资源。但i740在纹理层次的处理上却相当成熟，这一点尤其是在注重表现层次感的游戏中，借助OpenGL更大地发挥了i740的威力。

再探Voodoo2

　　Voodoo2与其“胞弟”Voodoo一样，仍然是一张3D附加卡。目前它的显示内存最高可达到12MB，4MB作为Frame Buffer(帧缓存)，另外两个4MB分别分配给两个材质处理单元。在使用Z－buffer时，最高解析度可达到800×600，而在使用SLI模式串接时，则最高可达到1024×768。从国外的一些测试中我们发现，由于Voodoo2的处理速度实在是太快了，以致于在整个3D成像的过程中，如果没有与其速度匹配的足够快的CPU，所有效果可能出现的瓶颈反应倒成了CPU在“暗中作崇”。因此，如果你有一块Voodoo2，那自然要有一台PentiumⅡ 266以上的机器才能算作“好马配好鞍”。否则，在一般的机器配置上，如Pentium 166/200等，虽然在画质等方面的效能仍会比Voodoo有所提升，但提升的幅度就远远没有高速主机那么明显了。
　　我们进行测试的产品，就是新近才进入国内市场的“帝盟”（Diamond）出品的Monster 3D Ⅱ。目前市场上的Voodoo2产品主要有8MB和12MB两种版本，笔者测试的这块Monster 3D Ⅱ为8MB版本，据Diamond权威部门透露的消息，该卡具有比其他厂牌12MB卡更好的效能。为了印证这一点，我们特别购置了Orchid Righteous 3D Ⅱ 12MB和创通的Voodoo2 12MB来做一比较。
　　从外形观察，三块加速卡的布线方式甚至元件位置都基本相同，其配置的显示内存也都是EDO RAM。只是Diamond的“怪兽”包装相当“雄伟”，打开一看，原来内附了六片游戏光盘。包含三片Lucasart的星际大战试玩光碟——《帝国生死斗》、《影子帝国》、《绝地武士》以及《Heavy Gear——机甲奇兵》完整版、EA试玩光碟、EIDOS的试玩光碟。至于Righteous 3DⅡ和创通的Voodoo2，笔者在包装盒内并没有发现附赠游戏或什么其它惊喜。
　　接下来就是紧张的测试工作了。测试程序上，为了便于比较，我们采取了与i740相同的测试方法和步骤。在测试过程中，我们使用的全部是公司在网络上提供的可供下载的最新版本驱动程序。最后的测试结果实在是让我们心满意足，Monster 3D Ⅱ在各方面的表现比两个12MB的版本略胜一筹，在《Fight Force》这部动作游戏的表现方面，所有的线性过滤、雾化处理都表现得十分平滑顺畅，分析原因，我们认为这应该主要归功于该卡在驱动程序方面所做的最优化处理。另外，Monster 3D Ⅱ的驱动程序居然实现了自动将卡本身的工作频率调高至92MHz，与其它预设的90MHz相比；自然在功效上不可同日而语。特别是它的驱动程序甚至“宽松”到了允许直接“超频”，而不需要重新启动。
　　不同品牌之间是否可做双插卡SLI模式？相信许多朋友对这个问题都比较感兴趣，笔者便是其中之一。抱着这种心态，我们试着将Monster 3DⅡ、Righteous 3DⅡ及创通Voodoo2分别交叉安装在同一部机器上，并将SLI连接线接上。在分别安装了驱动程序后重新开机，在Windows95的显示器选项中可以看到Voodoo2驱动程序已发现两块卡的存在，并且启用了SLI模式。不过在真正执行游戏后，却发现游戏竟然“笨”得无法找到其中任何一块Voodoo2，自然也就无法开启支持Voodoo的模式了。为此，笔者共试了六次移除、重装程序，结果是“涛声依旧”。这种测试得出的结论便是SLI模式还是得需要两片同厂牌的卡才能进行。

鹿死谁手

　　在同样的环境下，Voodoo2仍然是电脑游戏用3D加速卡中的佼佼者。i740的功效大约只能和Voodoo打个平手，即使i740有着诱人的价格，但想来还不能满足那些追求高品味玩家的要求。好在i740除了用做游戏的3D加速卡外，本身还具有TV out的功能，可以让玩家转接电视享受大尺寸画面的愉悦，也算作堤内损失堤外补吧。而就Voodoo2来讲，Monster3DⅡ则应该是目前玩家们的首选。虽然目前在国内市场上露面的绝对支持Voodoo2的游戏还不是很多，但从今年美国E3大展来看，从下半年到以后，恐怕会有越来越多的游戏要支持Voodoo2了。说了这么多，究竟现在应该买一块什么样的3D卡合适呢？综合以上，我们建议：如果你是追求品质型的，Monster 3DⅡ/8M是目前Voodoo2中的上选；如果你是追求长远型的，先等一等，i740的下一代产品已经在研发之中，据传品质超过Voodoo2，不过时间嘛……(天津鲲鹏工作室　冯宝坤)
　　编者注：本报评测室已拿到了大量市面上流行的各种3D显示卡，目前正在紧张测试中，拟在7月底发布测试报告，请读者留意。
