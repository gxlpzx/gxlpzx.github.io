优秀的HTML文件转换软件HTMASC

  　　由于HTML文件具有图文并茂、阅读方便、不易受到破坏等优点，因而受到了不少用户的欢迎，其应用范围也是越来越广泛。除Internet上广泛采用外，许多软件的说明书、自述文件以及电子读物、电子报刊等也都采用HTML文件形式，这就方便了用户的日常使用。不过HTML文件虽然在阅读方面颇为方便，但它也有一定的不足之处，即它的打印功能较差，我们无论是采用IE浏览器还是使用其他什么软件处理HTML文件时，其打印效果都不能令人满意。另外，HTML文件在剪切、复制、插入、链接等方面也都还存在着一定的不足，因此广大用户经常需要将它们转换为文本文件使用。以往，我们在将HTML文件转换为文本文件时大多是先利用IE等软件将它们打开，然后再利用这些软件的“另存为”命令实现以文本文件的形式存盘的目的，操作比较繁琐，功能也比较单一，不能完全满足用户的需要。本人近日试用了一个专门用于转换HTML文件的共享软件——HTMASC，发现其具有转换速度快、操作方便、可一次性转换多个文件、并可对转换的文件进行相关处理，大大地方便了用户对HTML文件的转换工作。
　　启动HTMASC之后，我们首先就可以发现它与其它软件有些不同，即它没有将自己的设置功能“隐藏”在菜单深处，而是直接放在了主菜单下，我们可直接对其转换状态进行设置，方便了用户的使用。HTMASC一共有10个设置选项，每个选项后面都有一个状态开关，当开关灯亮时表示该选项被允许，否则表示被禁止，我们只需使用鼠标单击相关选项按钮即可调整其状态，使用十分方便。一般来说，我们应着重设置以下几个选项：“Hints”是否显示提示信息；“Links”是否在转换后的文本文件中保留原有的超级链接信息（采用HREF…方式）；“SD”是否将转换后的目标文件存放到源文件目录中；“BAK”文件在存在同名目标文件时是否自动生成BAK备份文件；“COMP”是否自动删除目标文本文件中的空行。
　　设置完HTMASC的转换状态之后，我们就可以利用它对HTML文件进行转换了。如某用户欲将一篇包含有Internet超级链接地址的HTML文件转换文本文件，并在文本文件中保留原有的Internet超级链接地址，则可先行选定“SD”和“Links”选项，然后单击HTMASC快捷工具栏上的“Open”按钮，打开所需的HTML文件后，单击“Convert File To Plain Text”按钮，HTMASC就会对其进行转换，并将转换后的文档保存到源文件目录中，同时在目标文件中保留原有的超级链接地址，这可是利用IE等软件将HTML文件转换文文本文件时所不能做到的，大大地方便了用户的使用。
　　HTMASC的另一个优点，即它可以一次性将多个HTML文件转换为文本文件，这就减轻了用户转换文件时的工作量，深受广大用户的欢迎。为此，我们可单击HTMASC快捷工具栏上的“Multiple Convert”按钮，并在随后弹出的“Multiple File Convert”对话框中选择欲转换的文件，最后单击“GO”按钮，HTMASC即可一次性进行转换，效果非常好。
　　此外，HTMASC还具有可直接查看源文件及目标文件，以及直接对它们进行修改、剪切、复制、粘贴、查找、保存等操作的功能，这些对于用户更好的转换HTML文件都很有帮助，广大用户可充分加以利用（HTMASC是一个共享软件，其网址为http://ourworld.compuserve.com/homepages/starmail，有兴趣的读者可自行下载）。   (湖北 荆中院)
