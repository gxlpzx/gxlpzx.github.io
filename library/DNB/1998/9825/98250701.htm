全面了解Win98

Win98对硬件提供的支持

    Win98支持USB、MMX、 DVD、 IEEE 1394、 ACPI、 AGP、IrDA以及PC Card32(PC卡总线)等技术和规范。它带有1200个新的硬件驱动程序，同时也修补了Win95中所有的安全漏洞。而新的WDM(即Win32 Driver Model)允许用户在Win98中使用针对Windows NT编写的驱动程序， 使得硬件厂商不必再为两种操作系统写两种版本的驱动程序。另外，如果计算机中安装有专用的电视调谐卡， 就可以利用Win98中新引入的Web TV支持功能，在显示器屏幕上接收电视图像。
　　虽然在Windows 95系统中也可以使用这些新硬件规范，但是Windows 98能够自动识别这些新硬件并安装相应的驱动程序。其中对USB的支持实际上是Windows 98的一个性能热点。和通常的设备需要占用插槽、串并口等资源不同，USB设置根本不需要IRQ资源，在Windows 98操作系统的协调下，可以毫无问题地安装多达128个周边设备，相应的接口甚至能够提供设备使用的电源。这些USB设备常见的有扬声器、扫描仪、鼠标、键盘、数码相机、电话等，而这仅仅是开始，更多更新的USB设备目前正如雨后春笋般地涌现。Windows 98支持的其它新型硬件还包括使用火球技术规范的高速磁盘驱动器等外围设备。Windows 98对新硬件的支持，将使爱好游戏的用户受益不小，Windows 98支持的诸如DVD、AGP视频卡等设备可以说是娱乐导向下的产品。
　　在Windows 98中，即插即用功能依旧不很完善，但是和Windows 95中的相比已经进步了很多。在Windows 95中，当安装新硬件进行例行检查时，安装程序认为目前已有的硬件设备配置良好而不改变其设置。而在Windows 98中，一旦进行类似的操作，将重新配置所有的设备，这样可以更有效地避免冲突。

Win98对显示器的支持

　　Windows 98对于多显示器的支持堪称一绝。以往为了获得更多的“视觉空间”，似乎除了破费钱财购买大屏幕显示器以外别无它法。现在有了Windows 98，在购买新显示器之前大可以三思而后行了。因为Windows 98中包含了一个很爽的实用程序，不用花那么多的钱即可获得屏幕空间的升级。保留原有的显示器，然后加装一块显示卡，再连接上另外一台显示器(哪怕是小点儿的)，然后启动系统让Windows 98玩一次即插即用的魔术，就可以同时使用两台显示器了。只要计算机中有空闲的PCI插槽，Windows 98支持同时使用多达8个显示器。耳听六路不知道是什么境界，但是在Windows 98中，只要愿意，大可以尝试一下眼观八方的滋味(当然通常只要同时使用两台显示器就足够了)。使用中可以针对每个显示器选择不同的分辨率和颜色深度，将应用程序从一个显示器上拖放到另一个显示器上，而当最大化某个应用程序窗口时，其窗口就充满当前显示器。
　　使用多显示器可能会遇到一些古老的应用程序的兼容性问题，不过Web浏览器、微软的办公套件Office系列以及图形图像实用程序都能够在多显示器环境下工作得很好。就显示性能的改进而言，多显示器支持是最爽的。

Win98与FAT32

　　Windows 98系统核心的一重要改进就是对硬盘管理采用FAT32模式。 FAT16和FAT32最大的区别就是使用不同尺寸的簇来存放文件。DOS和低版本Windows采用普通的FAT格式来管理文件，当分区小于127M时，簇的大小是2K;分区介于128M和255M之间时，簇的大小为4K;分区介于256M和511M之间时，簇的大小为8K; 分区大小介于512M和1023M之间时，簇的大小为16K;当分区大小在1024M(1G)和2G之间时，簇的大小竟达到32K。这时即使一个文件只有一个字节，占用的磁盘空间也是32K。这对于磁盘空间的有效利用而言显然是不合理的。FAT32就克服了这个问题。采用FAT32以后，分区小于260M时簇的大小为0.5K(512Bytes);分区介于260M和8G之间时一律以4K为一簇;分区大小介于8G和16G之间时使用的簇尺寸才是8K……不难理解，FAT32的采用将节省大量磁盘空间。
　　要在Windows 95系统中使用FAT32模式，不付出代价是不行的:必须重新格式化硬盘，然后重新安装Windows 95,而这意味着要重新安装所有的应用程序。当然，使用诸如PowerQuest公司的Partition Magic这样的第三方实用程序，也许是一个好的选择，但是总是不如操作系统本身自带的转换程序更令人放心。
　　Windows 98是Windows家族中第一个包含转换FAT16到FAT32的实用程序的操作系统。将磁盘文件管理模式从FAT16转换到FAT32以后，通常能够挤出20％到30％的磁盘空间。在Windows 98中，内存管理和缓冲模块也针对使用4K大小的簇进行了调整、优化，使得从FAT32结构的硬盘上存取数据的速度更快。使用1G以上的大容量硬盘的用户，将从Windows 98的FAT32性能中获得极大的好处。
　　从FAT16到FAT32的转换非常容易，尽管转换的时候程序提示可能需要几个小时，但是实际上只要5分钟就足够了。唯一的问题就是，Windows 98包含的转换程序是单向的，转换到FAT32容易，可是要返回原来的FAT16就难了。

Win98的网络功能

　　Windows 98是有史以来和Internet集成最紧密的操作系统，它在将操作系统界面和浏览器集成方面做了大量的工作，特别是采用了浏览器的方式来查看本机上的数据资源。
　　微软当然要充分利用和Internet的紧密集成来方便自己，例如Windows 98利用网络来实现自动升级和修补。使用Windows 95的用户可能对于需要不断地下载补丁程序感到厌烦。Windows 98中开始菜单里的自动升级功能使用户彻底摆脱这种烦恼。在联网的情况下，自动升级功能可以打开浏览器访问升级站点，其中包含一个升级巫师程序能够自动扫描用户机器中的驱动器，寻找需要升级的文件，而后自动下载并安装需要升级的内容。
　　Windows 98中的活动桌面允许用户在其上放置频道和Internet组件，例如有规律地升级的新闻标题和股票信息。Windows 98甚至允许用户将一个Web站点设置为桌面壁纸。相比之下，Windows 95则只允许用户在桌面上放置Web页面的快捷方式。
　　值得欣慰的是，当我们需要安装Windows 98时，新的操作系统中包含许多组件已经在网络上了，只要从网络上下载相应的组件即可使Windows 95变得和Wiindows 98尽量地接近。最好的一个例子就是在Windows 95系统中下载安装IE 4.0。相应站点的地址是http://www.microsoft.com/windows95/info/updates.htm,这是一个Windows 95升级网页，其中包含了Service Pack 1、OSR2组件、系统升级内容等。
　　Windows 98自动安装IE 4.0浏览器，用户是没有选择的。集成的IE 4.0浏览器允许用户像浏览网络一样在应用程序之间切换。当然，喜爱Netscape Navigator浏览器的用户依旧可以使用它，甚至可以继续保持其缺省浏览器的地位。不过有一点要注意:如果将IE 4.0的图标从桌面上删除，就休想再在Windows 98中启动IE 4.0！这显然是使用Netscape Navigator的用户难以接受的，特别是在硬盘空间比较紧张的情况下。而对于没有联网的用户来说，当看到IE 4.0缺省地安装在那里却毫无用途时，会是什么感觉呢?

Win98集成的实用程序

　　Windows 98集成的应用程序中最引人注目的就是完全重新编写的System Information Utiliy(系统信息实用程序)。通过从注册表、启动文件以及正在运行的应用程序中取得的数据，系统信息实用程序，能够以详尽的图形方式显示当前系统的配置情况。而这种信息能够帮助用户找出资源冲突、识别设备、查看正在运行的程序及任务的完整列表，即使某些是隐含的也照列不误，这显然增加了透明度。
　　在Windows 98还包含了一个System Configuration Utility(系统配置实用程序)，可用来方便地设置故障检查的选项而不必再借助于DOS命令行。另外还有一个确实不错的注册表备份/恢复实用程序，据说使用起来也真的起作用。而新增的诸如维护巫师等程序则能够帮助用户对系统进行合理的微调以达到最佳的性能。
　　另外，微软也会向用户提供Windows 98 Plus软件包。Plus 98将包含诸如McAfee VirusScan(带有6个月免费升级病毒特征码列表)、用于优化和组织系统的Start Menu and File Cleaners(开始菜单和文档清除程序),以及用于存档文件的Compressed Folder(压缩文件夹)等额外的实用程序和娱乐程序。Plus 98还将包含新的桌面组件如Comic Strips，PictureIt Express(照片编辑程序)以及新游戏等。
　　当然要完全享用Windows 98的种种性能改进，需要使用完全支持ACPI(Advanced Configuration and Power Interface,高级配置和能源管理界面)和OnNow的计算机系统，这两项功能能够显著减少对于启动和关机的需要，而大多数用户正是在等待启动和关机的时候花费不少时间。
　　系统中还包含了能够使用户避免遇到常见的性能瓶颈的实用程序，其中效果最显著的就是Maintenance Wizard(维护巫师程序),它能够自动安排磁盘碎片整理操作。
　　除增加或者强化了如下一些实用程序外，Windows 98还引入了DirectX 5和最新版本的ActiveMovie，几乎所有Microsoft Plus 95中的功能都集成到了Windows 98中，其中包括所有桌面组件及强化功能，例如Windows 98界面中集成了PowerToy、QuickRes等著名的系统强化功能。
