来吧 来吧 Windows 98

  　是否需要升级到Win98?当然，如果我们只买我们必要的东西，可能到现在我们还在用Word 2.0。听了下面传奇般的描述，相信你会从中找到一个理由，让自己马上走进Win98。
    一、哪些人想升级
    高级用户
　　为了追求操作系统的稳定性能，许多网络或应用软件的高级使用者把系统按照自己的意愿条理化。但有时造成了系统崩溃，Win98比Win95更可靠。这是因为：Win98的操作环境直接与系统控件相联系。就是说一旦出现异常，造成系统崩溃的那部分应用软件将马上被关闭，不会影响同时使用的其它软件的继续运行。
     游戏玩家
　　如果游戏是你生活中不可缺少的一部分，那么Win98值得一览。只要你的硬件是一流的，一切由于软件不支持而带来的麻烦将成为历史。你将是首批USB接口的受益者。为了不断发挥硬件的潜能，Win98的自动更新向导总保持驱动程序是最新的，这很重要，因为一种新硬件推出后，第一代驱动程序往往不能充分发挥其潜能，经过软件工程师不断地改进，一些硬件的“内秀”才被挖掘出来。比如，创通公司对其声卡、显卡驱动程序的几次修改，使我们不断体会到SoundBlaster 64 Gold的震撼感受。你可能已经安装DirectX 5.0，但它并没有充分发挥潜能，Win98内置的DirectX 5.0将带你走进更平滑的3D世界。
    商用或家庭办公者
　　一直认为局域网的最好操作系统是Windows NT，它对硬件的要求高而其本身价格更是高不可攀，其地位一度被Windows for Workgroups和Win95所取代。现在有了Win98，它对TCP/IP协议的优化使得无论是局域网传输或是远程登录，上Internet都更加快捷。特别是对USB的支持，使你不再为匮乏的端口或为得不到理想的传输速率而心情不快。
    网络发烧友
　　不用多说，相信你早有耳闻那起美国司法部对微软的起诉吧！其核心问题是围绕着微软能否在其下一代Windows产品中捆绑IE。说实在的，站在消费者的立场上，我还是希望Win98能紧密地和IE结合。从Win98 Beta 版的测试环境来看，没有一个测试者怀疑过从Win98登录Internet的速度。通过对TCP/IP的改进，使得无论使用IE或是Netscape Navigator都觉得很理想。值得一提的是，当你下载一个软件包时，系统工具会像包裹过海关一样例行检查，以保持你系统的干净。
    Notebook使用者
　　作为一位笔记本电脑的使用者，你将面临艰难的选择：一方面，Win98中改善的对PCMCIA的支持、增强的拨号网络（Dial－Up Networking）和全新的TCP/IP通讯协议会使你对指定位置的呼叫更加简单。PLUS，使你所浏览的网页显示得更平滑，FAT32和系统备份工具的结合帮你释放更多的硬盘空间存放有用的数据。另一方面，假如你的笔记本系统配置很低，和Win95或更早期的Windows相比，Win98可能会降低运行程序的速度。
    二、升级途径
　　Win98就在这儿，它正期待着你的参与。Win98的安装不会成为阻碍你前进的绊脚石。和Win95的安装程序相比，安装Win98更像是在了解Win98，而根本不会让你觉得有什么困难。
　　当你从CD－ROM中运行安装程序后，系统会作一次常规的硬盘检查并收集一些关于你的计算机的信息，然后需由你选择哪些程序要安装。由于你正在安装新操作系统，Win98会在恰当的时候提示你做一张原系统备份盘——即启动盘，以备它用。接下来，你就不再用亲自动手了。在Win98的文件复制过程中，进度栏下的时钟会准确提示你安装剩余时间。
   从Windows 3.X升级
　　尽管Win98许多新程序在Win95中从未涉及，但一些基本功能都是相似的。如果你是在过去的3年内添换的硬件，你可能因为Win95不支持你的硬件而放弃了升级计划，还在用Win3.x，那么Win98强大的硬件兼用性会给你一个意外的惊喜。笔者测试的一块Opti929非Plug and Play声卡在Win98的自动检测后，被设置为MS－Windows Sound System Compatible，不但音频回放正常就连MIDI也表现出色，这就是Win98既同于Win95又高于Win95的地方。所以，你无需再为硬件的资格老而犹豫不决了。
　　笔者从Windows for Workgroups 3.11 升级到Win98后，DOS下声卡的驱动程序不得不重装，所有的硬件均能胜任在Win98中的工作。
   从Windows95升级
　　假如你的系统已经是Win95且运行良好，安装Win98就真是小菜一碟了！安装程序在简单地检查系统注册器后，把基于新系统的必要文件置于你的系统中，然后同样要经过约10分钟的系统更新过程。原则上，一切硬件和软件应该可以照旧使用。笔者检测的一台联想微机（Pentium 166MHz，16MB EDORAM，Als007芯片声卡，S3 Tri64 V＋显卡）不是为Win95所设计的，在以前Win95的使用中不能放MIDI，升级后不但恢复了MIDI功能，并且自动更新了新版本的驱动程序。
    三、特别提示
　　在使用Win98时，除了Win95上积累的经验外，还要格外注意以下几点：
　　●盘旋(Hovering)　即当你用鼠标在项目上移来移去时，鼠标指向的某个项目就会自动变成高亮度显示，这时就相当于Win95的选取了。你若单击它，它就会被启动或激活。这时图标、文件、文件夹和快捷方式等都是有效的(前提是Win98的True Web Interaction已打开)。
　　●右击　在Win95中，大家已经体会到了“要想快，点右键”的好处，在Win98中，这种快捷和方便性更加强了，大家不妨经常在你需要进行某项事务时先试着点一下右键，看看效果如何。
　　●浏览　由于Ture Web Interaction(真正的Web集成)的缘故，Win98中所有事务的界面都是非常类似的，所以你必须习惯用Web页的工作方式来对待其它所有资源，比如“后退”按钮就无处不在。所以在Win98下工作，大家会有一种我怎么像在网上浏览一样的感觉，实际上这正是微软所期望的效果：计算机就是网络，网络就是计算机。除了“我的电脑”、“Windows资源管理器”、各类文件夹等等看起来像Web页(但也可以让他们按从前的Win95那样显示)，连桌面也变成一个Web页，你可以把某个HTML对象直接放在桌面上以便下次快速进入，也可以把一个HTML文档当作桌面背景。
　　如果有些朋友直接从Windows 3.x直接跳到Win98，也是可以的，那么从界面上和工作方式上就会感觉到较大的不同，初期会感觉到Win98不如Windows 3.1方便，习惯之后就会觉得还是Win 98好
　　就像Win95的推出一样，不试不知其妙，最后还是提醒你注意一下Windows98的系统要求：
　　486 DX/66 MHz 或更高
　　16MB RAM
　　150至295MB 磁盘空间
　　CD－ROM或DVD－ROM驱动器（3.5英寸软盘驱动器）
　　VGA或更高级的显示器
　　Windows 兼容的鼠标或相似设备
