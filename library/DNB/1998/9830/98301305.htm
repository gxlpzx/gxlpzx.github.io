用 Java  Applet 制 作 水 波 效 果 图 片

　  在一些主页上你会见到有水波效果(即倒影)的图片，如象鼻山的倒影在漓江中随波荡漾。这些极酷的图片实际上是用Java语言编写的，而且非常简单，只需三步即可：
　　第一步：将Lakeclass类与需加工图片所在的主页放在同一目录中，这样才能显示出水波效果。
　　第二步：选择要制作水波特效的图片(当然要有水了!不过用沙漠、戈壁等其他图片制作也会起到意想不到的效果，不妨一试)。
　　第三步：编写Java Applet源程序：
　　<applet code=″Lakeclass″ align=″baseline″ width=″图片宽度″ height=″图片高度″ id=″Lake″>
　　<param name=″image″ value=″图片文件名″>
　　</applet>
　　注意：
　　1图片可以是JPG或GIF格式，程序中“图片文件名”需要包括扩展名。
　　2Java Applet是区分大小写字符的，同时编写源程序时要使用文本编辑器而不要用字处理程序，这两点在写程序时需要注意。
　　3程序中“图片高度”是指原图片高度加上倒影后的总高度，注意倒影的高度不能超过图片的原高度，即“图片高度”取值不能超过原图片高度的一倍。
