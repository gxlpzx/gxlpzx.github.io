Windows NT 5.0新功能

　　微软最近在其专业开发人员会议上发布了Windows NT 5.0的测试版。它在以前版本的基础上增加了下面一些功能。
　　1.Active Directory　如果你不知道电话号码，那么光有电话网是没有用处的，同样，如果你不知道某个人、某台计算机或外设的连接“地址”，计算机网络也是没有用处的。我们在分布式系统中进行的所有的工作都需要一个功能强大和范围广阔的“地址簿目录”。微软在开发这个目录的竞争中曾经比较落后。但微软交付Active Directory后，使它在这方面的竞争力大增。
　　2.Integrated Security　只要登录一次，验证你的身份一次，你就可以访问你所需要的任何信息，不管它是在自己的PC机中，还是在遥远的PC机中。
　　3.Intelli Mirroring　在网络中，有一个“总管”计算机保存着所有的应用和用户档案，这样，用户就可以使用网络中的任意一台计算机进行登录，然后得到“用户自己”的桌面和应用程序。实际上，用户的硬盘变成了一个缓存，只用来保存最经常使用的软件和数据。每次用户在本地硬盘进行写操作时，写入的信息会在后台“流入”服务器并永久地保存起来。
　　4.一体化的存储　这是一种新的存储系统，既有文件系统的高速性能，又有数据库的强大功能。由于这个系统在寻找信息方面有杰出的表现，它可以免除保存多个拷贝的必要。比如当你要同时向几个人发送一份庞大的Director演示时，这个存储系统会智能地把这几个人指向同一个存储拷贝。
　　5.自动的软件　软件可以自己进行安装和进行自动修复，并且在新的版本出现时自动升级。这可以叫做“傻瓜”技术。
　　6.零管理　不需接触硬件与软件，用户作出的改动将被自动执行，在进行改动时，系统会参考规则与设置以确定修改的结果。系统管理员可以从一个中央控制点控制整个网络的规则。

　　(北京　钮加明)
