ATA的两种传输模式

　　ATA硬盘有PIO模式和DMA模式两种传输方式。PIO( Programmed Input Output)即可编程输入输出，它是将数据传输的过程加以程序化，这明显加重了CPU的负担。因此在这种模式下，CPU的耗用率很高，由于它必须全力地去处理数据存取，因此无法去干别的事。目前的PIO有1、2、3、4四种模式。
　　DMA (Direct Memory Access)即直接内存访问，而ATA的DMA模式就是通过DMA通道传输硬盘数据了。DMA模式又分作Single Word和Multi Word两种模式，它们和PIO模式所对应的速率见下表——ATA的传输模式和速率。
