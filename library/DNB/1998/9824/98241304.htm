在应用程序中调用Internet Explorer 4.0

  　 随着Internet的发展与普及，越来越多的应用程序提供了Internet功能，在许多应用程序中，我们都可以看到Http:∥www.Microsoft.com之类的连接。当点击这些连接时，就可以自动打开Internet Explorer去访问。
　　现在，让我们用Visual Basic 5.0来实现这样的功能。
　　1．确认已经安装了Internet Explorer 4.0，然后在菜单“工程\引用”中选中“Microsoft Internet Controls”。
　　2．新建一个Form和一个Command Button。在 Click()事件中加入如下代码：
　　Private Sub Command1_Click()
　　Dim IE As InternetExplorer
　　Set IE=New InternetExplorer
　　IEVisible=True
　　IENavigate ″http:∥www.soundblaster.com″
　　′IE还有AddressBar、Menu、FullScreen等几个属性可以对它进行弹性的控制。
　　End Sub
　　OK,运行程序，按一下按钮，看看发现了什么？　　(陕西　任沁明)
