如何查看163的账单

　　如今我国的上网人数早已突破百万大关，但上网的费用依然是网民们较为头痛的事。如果你已经上了Internet，并且你的ISP是电信局（163）的话，你肯定很想随时知道你的账单，以控制自己的上网时间。这里我就带你“直接”到电信局去亲自看看你的账单吧!如果你觉得有必要，也可以把它拷贝下来，以便核对。
　　1.先用你的拨号工具拨号，等到出现终端窗口后，在Login后面输入t用户名，接着输入你的密码；接下去它会要求你再登录一次，但这次不用加t。
　　2.登录完毕出现下面提示：
　　Last login: Thu Sep 10 14:47:21 from comm3.zjnbptt.ne
　　　please input correct TERM Type!
　　　　Terminal_type(default vt100):
　　3.直接按回车，出现如下菜单：
　　＊＊＊＊＊＊＊＊＊
　　＊ Main menu ＊
　　＊＊＊＊＊＊＊＊＊
　　Option Description
　　0 ............. Change password
　　1 ............. File transfer and Archie
　　2 ............. Read/Send mail
　　3 ............. Connect to other nodes (telnet)
　　4 ............. Netnews
　　5 ............. Lynx access
　　6 ............. Gopher
　　7 ............. Other Internet services
　　8 ............. Unix Common Commands
　　9 ............. User self_query information
　　Select choice [ h (Input help file.) c (Switch to [e,c8,c7] menu.)
　　v (Display version message.) u(Return to up menu.)
　　t (Return to top menu. ) quit(Exit AIpink2.1)]:
　　4.选择“9”回车，出现如下提示：
　　请输入所要查询的月份(格式为YYYY/MM,形如1996/09,直接回车为当前月):
　　5.若直接回车则出现如下当月的账单，当然你也可以依提示查询以前的账单。
　　起始日期(From):08/21/1998 终止日期(To):09/20/1998
　　－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－
　　username 用户名: 1234
　　logname 账号名: xyzz
　　basefee 基本费: 100 Yuan元
　　connect time 本地上网时间: 1763.0 minutes分钟
　　half time 本地优惠时间: 322.2 minutes分钟
　　roam time 漫游时长: 0.
　　0 minutes分钟
　　roam fee 漫游费: 0.
　　00 Yuan元
　　disk space 平均磁盘占用量: 5.333 KBytes千字节
　　diskfee 磁盘存储费: 0.00 Yuan元
　　total to pay 费用总额: 13
　　6.15 Yuan元
　　－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－
　　Do you want to examine itemized billing information?(y/n)
