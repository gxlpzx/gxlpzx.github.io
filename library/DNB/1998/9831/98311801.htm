ICQ使用问与答

　　编者按：目前网络上ICQ(它是英文“I Seek You”的连读音缩写，由于可利用它在网上呼叫找人，故也称网络寻呼机。有关ICQ的安装使用请见《电脑报》1997合订本下册P69“网络BP机”一文)的全免费政策,使其用户猛增。据统计，ICQ的用户数量已经超过1千万!为了帮助大家更好地使用ICQ，特约请谢勇和阿彪两位作者分别撰写了“ICQ使用问与答”和“玩转“ICQ”两文奉献给大家。
　　Q(问)：如何避免经常性的断线或者老是联接不上服务器？
　　A(答)：ICQ本身安装成功后,默认的服务器是两个:ICQ.MIRABILIS.COM:4000,ICQ1.MIRABILIS.COM:4000。据我所知,可以在ICQ的菜单中选取“PREFERENCES”,然后在其中的“SERVERS”选项里,点击“ADD”按钮,再添加几个服务器:ICQ2.MIRABILIS.COM:4000,ICQ3.MIRABILIS.COM:4000,ICQ4.MIRABILIS.COM:4000。这样服务器多了,ICQ就会比较容易联接成功,也不容易断线。
　　Q：重新安装ICQ应注意哪些地方?
　　A：很多朋友重新安装了Win 95之后,不可避免的需要重新安装ICQ,由于你已经是它的注册用户(标志就是已经拥有了自己独一无二的UIN－Universal Internet Number),所以你重新安装完毕,需要注册时,应当选择“EXIST USER……”(已经拥有UIN)而不要选“NEW USER……”(新用户)方式。这样你不必更换UIN,朋友们也不会因为你的UIN改变而和你失去联系。否则你每重新安装一次Win 95都要注册一个新的UIN,不但浪费UIN资源,而且每次还要费事的通知朋友。
　　当然,如果你一直无法使用“EXIST USER……”的方式登录成功那就可使用“NEW USER……”新用户方式登录，成功后选择ICQ的菜单“Add/Change Current User”,点击“ADD ANOTHER REGISTERED USER”,然后输入你的原来的UIN和对应口令,就可以继续使用你原来的UIN,继续和朋友们保持联络。注意，别忘记把这个仅仅用来作为过渡的新UIN删去:选择ICQ的菜单“Add/Change CurrentUser”,然后选择“Unregister Existing User”,选择新申请的UIN,然后输入口令,点击“NEXT”,稍候片刻即可删去。
　　采用这种方法还可使一台机器拥有几个UIN，从而可使几个人使用一台机器上的ICQ而不发生相互干扰，即每个人的ICQ信息不被别人看到。每个人使用时，只需要分别换用各人自己的UIN,然后设置Security＆Privacy选项中的“Security Level”为“High”即可,这样每次换用UIN时都必须输入口令方可使用。
　　Q：如何保护和备份自己的ICQ数据?
　　A：重装ICQ后最大的痛苦是要手动的添加原来那些老朋友的UIN,以继续保持联络。人数不多倒也罢了,像我这样联络名单中有上百个名字的弟兄岂不太苦了。
　　解决的方法是:重装ICQ后,立刻用原来的UIN进行注册。如果采用新用户方式登录，获得新UIN后,必须立刻再用原UIN注册成功。然后断开网络连接,重新启动Win 95。如果此时你的ICQ已经自动运行,记住一定要先关闭ICQ(点击菜单中的“SHUT DOWN”)，然后将原来的ICQ压缩包中的所有EXE文件和DLL文件删去,将剩余的文件直接释放到新ICQ目录中,并覆盖全部的现有同名目录和文件。完毕后重新启动Win95,再运行ICQ,你就会发现原来的ICQ又出现在你的面前,一切数据,完整无损。
　　Q：如何加快你的ICQ的速度?
　　A：关键是要及时清理你的联络数据,过一段时间后,要及时把你和朋友们联络的HISTORY(历史)进行查看和整理,需要的就保存下来,否则就全部删去。清理方法:在具体的名字上单击鼠标右键,在菜单中选“View Messages Histroy”,然后点击“Save As”按钮,并输入具体的存放目录和存放文件名即可。
　　另外,登录两个UIN是个好的方法,其中一个只保留最重要朋友的名单,当网络阻塞时,联络名单较多的ICQ往往很难登录成功,此时就换用那个重要的UIN进行登录。要换用新的UIN,可选择ICQ的菜单“Add/Change Current User”,点击“Change The Active User”,然后选择具体的NICK即可。
　　Q：我安装完了ICQ,可是进行新用户登录时,总是无法登录到服务器怎么办?
　　A：这是一个很恼人的问题,因为如果无法登录,你就根本无法取得UIN。如果你登录过程超过15分钟仍然无法登录服务器,可采取以下方法:在未注册成功的窗口中点击“Cancel”,然后点击“Previous”回到上一步(见图),点击窗口左下角“For Admin Use”,将ICQ.MIRABILIS.COM:4000改为ICQ1.MIRABLIS.COM:4000,再点击“Retry”,重新进入登录过程,一般都可以登录成功。如果还不行,恐怕就是你的网络联接速度太慢,或者是你的Internet连接无法出国,那就只有放弃使用ICQ了。
