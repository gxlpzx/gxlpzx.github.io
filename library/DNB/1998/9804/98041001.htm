你的硬盘里该装些什么软件？

    有了电脑以后，面对容量巨大的硬盘，该安装哪些软件呢?市场上五花八门的软件让许多人束手无策，无从下手。本文将软件分门别类，一一道来，相信细读之后，会有你自己的选择。
    学习类
　　●首选：《大嘴英语》 零售价：98元
　　●可选：《轻轻松松背单词》　零售价：78元
　　目前，市面上的学习软件很多，上至天文，下至地理，古今中外，几乎无所不包。其中主要分以下几类：一是专门针对中小学生的教学辅助软件；二是知识百科软件；三是英语学习软件。其中，尤其以英语学习软件最为畅销， 这与当前的英语热有直接关系，而且因为电脑操作对英语的要求较高，这又从另一个角度刺激了英语学习软件的开发与销售。现在，市面上比较畅销的英语学习软件有《大嘴英语》、《轻轻松松背单词》、《挑战TOEFL》、《听力实战》等。
　　 《大嘴英语》是国内第一张声控视频交互英语学习软件，主要用于提高英语口语水平和听力。它采用流行的情境教学法，有35段场境，其中的角色全部由真人扮演。使用者可通过麦克风同电脑中的角色实时对话，其先进的语音识别技术，能“听懂”使用者所说的英语，并纠正错误的发音。
　　《轻轻松松背单词》是老字号的英语学习软件，知名度一直很高。其最新版本在继承原有特点的同时，增加了许多新的功能。这套软件采用循环记忆法，听、说、写并举，协同记忆，并有针对难记单词的强化记忆法，词库涵盖小学、初中、高中、大学及研究生、TOEFL、GRE等，词汇总量达57000个之多。除了单词记忆外，这套软件还包括有声字典、拼字游戏、黑匣子、装词改词等实用功能。
    编程类
　　●首选：Visual Basic 5.0　零售价：6000元
　　●可选：Visual C＋＋4.0　零售价：4840元
　　　　　Visual FoxPro 5.0 零售价：5850元
　　随着操作系统平台的升级,越来越多的用户转向Windows平台。编程工具也向可视化编程工具过渡，现在这一领域非常活跃。
　　与其它的编程工具相比，Visual Basic更简洁、易于使用、开发效率高，无论对初学者还是程序员都非常适用。它保留了许多Basic语言的语法规则，但功能已经完全超越了Basic语言。Visual Basic提供了拖放技术，属性检查，生成向导等功能，而且它是最先支持ActiveX技术的开发工具。
　　在管理信息系统开发软件中，Visual FoxPro是一个相当优秀的软件，它提供的生成向导，对初学者很有帮助。而且对软硬件的配置要求低，可以与Visual Basic和Windows API结合使用，风格统一，使编程更加灵活方便。
　　对于程序员来说，C语言最能够体现出编程的水平。现在较为流行的是Visual C＋＋，它不但增加了许多标准类库而且还加强了多线程的处理。另外， Visual C＋＋在支持Active X方面性能优越。
    字处理类
　　●首选：WPS 97　　零售价：480元
　　●可选：Word 97　 零售价：2700元
　　目前流行的字处理软件中，Word一统天下的局面已被打破，国产软件WPS 97向微软最新的Word 97发起了强有力的挑战。这两款相互竞争的软件的优劣比较，已有许多评论，它们的性能几乎不相上下。
　　从使用上来看，这两款软件各有所长，WPS 97的长处在于汉字处理能力，其使用方式可能更适合中国人的习惯，Word的长处在于其支持Internet网络功能。但是，二者的价格却相去甚远，从性能价格比的角度看，WPS 97显然更占优势。
　　事实上，WPS 97的真正意义已远远超越了它本身。金山公司敢于与国外大型公司竞争的精神的确让人激动，因为竞争的最终受益者必竟是用户。
    联网通信类
　　●首选：Internet Explorer 4.0
　　●可选：Netscape Navigator 4.0
　　　　　 “飞扬”电子邮件软件　零售价：118元
　　目前，Internet所提供的网络服务有：超文本传输，E－mail，远程登录等。就Internet的浏览软件来说，目前占主要地位的有Netscape的Navigator 4.0与微软公司的Internet Explorer 4.0。撇开其它的因素不谈，就使用的方便性与系统的融合性而言，还是应推Internet Explorer 4.0。它不仅有中文版，而且还是免费赠送，随Internet Explorer 4.0一起，提供了Outlook Express电子邮件软件与Web页面编辑创作工具。因此，用户得到的是一个功能齐全的网络套件，一般具有Office使用经验的用户，都可以很快熟悉和使用。
　　用于上网的工具软件有许多，我认为，至少有两类软件是必备的：一类是FTP断点续传软件，像笔者现在使用的 Getright（3.02版）就是一个易用的断点续传软件。因为从Internet网络上下载文件的时候，经常会因各种原因导致传输中断，如果再从头传，会浪费时间与金钱。断点续传的好处就是可以在重新开始时，从断点处开始传送。另一个工具软件就是可用来监视网络使用时间与自动计费的软件。这类软件根据当地的电话局与ISP商的收费标准设置每分钟费用率，在启动后，自动计算用户网络使用时间和费用，这对于合理计划使用Internet极有帮助。笔者使用后认为值得推荐的是EasyMonitor，该软件为共享软件，启动后，以后台运行方式自动计算当次使用的时间与费用，并累计全部使用时间与费用。得到的最后结果，保存在指定的文件中，可作为核对ISP收费的依据。EasyMonitor还可以代替拨号软件，用来直接拨号或断开与ISP的服务器的连接。通过“http://home5.swipnet.se/～w－54736/bbsoft.html”的网络地址可获得此软件。
　　电子邮件软件，洋货的确不少。但最近通过《电脑报配套光盘》试用了国产的“飞扬”电子邮件软件，感到无论是界面的“人性化”设计，还是使用的简易性与功能的多样性，都是相当不错的。尤其是对初用电子邮件的用户，因对许多参数的设置迷惑不解，不能正确设置，结果是自己的电子邮件就只能送给自己（发不出去）。“飞扬”为此提供了安装向导，可以明确告诉你这些参数的设置方法，依样画葫芦，也可以满有把握地正确设置。使用以后，真让人有爱不释手的感觉。唯一的缺点是目前不支持Windows 95的长文件名功能，如果加以改进，“飞扬”可能会牢牢占据我们的硬盘。目前，在“飞扬”网址上，还提供了2.0Beta版，供用户免费下载使用。如果你通过《电脑报配套光盘》认识了“飞扬”，就赶快去下载。其功能又增强了许多，甚至在查找、发送邮件等工作完成以后，“飞扬”可以立即自动断开与ISP服务器的连接，免得因为忘了断开连接而花冤枉钱，为用户的考虑可谓周全。
　　如果与在同一城市的电脑联接，可以利用终端软件采用点对点的通信方式，这比使用电子邮件更经济，更快捷。终端软件中，我趋向于采用Windows 95系统自带的“超级终端”，使用相当简单，而且能支持到115200波特率的传输速度，传送文件极快，勿需再花钱购买软件。
    文件管理类
　　●首选：Windows Commander
　　　 　　Winzip 6.3
　　●可选：WinRAR 2.2
　　　　　Magic Folder
　　Windows 95的文件管理功能已经相当完善。但是，目前国内用户普遍使用的是14"或15"显示器，通过桌面打开若干窗口进行文件操作，有时确实不方便。尤其安装了IE 4.0后，整个桌面按Web风格重新设置，不能同时打开几个窗口进行文件管理操作。为克服这一缺陷，应该利用其它的文件管理工具。这方面，值得推荐的就是Windows Commander，《电脑报》去年专门对此软件作过介绍。用它拷贝文件、压缩文件与移动文件和管理目录，在一个屏幕窗口中进行，一目了然，使用熟练后，管理文件的效率很高。该软件对鼠标拖放操作与键盘操作都支持得很充分。
　　文件压缩工具也是必备的软件。从使用角度讲，我认为配备两个就足以应付需要了：一个是Winzip（现在有6.3 sr版），该软件的易用性最好，且使用面广，在网络上进行文件交流的时候，用它压缩，一般不用担心对方能否解开，但压缩率实在不敢恭维，因此我经常使用WinRAR（2.2版），它的压缩率较高，并对老牌的压缩工具(如ARJ、LHA等)处理的文件，都能支持。
　　文件加密软件也是非常实用的，因为大多数电脑都是共用，几个人的文件资料放在一个硬盘中，文件的安全性和保密性难以保障。为此，可以采用针对文件的加密软件，如F－Secure Desktop,操作简单，很容易学会。另外再向大家介绍Magic Folder，它的作用就是隐藏目录（文件夹），将重要的文件放在被隐藏的目录下，其他人根本看不到，就没有被删除或偷看之虑了。
    图像抓取及浏览类
　　●首选：PCS
　　　　　 ACDSee 2.21
　　●可选：HyperSanp 3.07
　　　　　 Hijaak 95
　　抓图软件，有利于用户从网络，多媒体光盘以及其他 一些素材库上快速攫取图像为己所用。我认为需要配备两个软件：一个是PCS（Professional capture systems 2.0版），该软件强大的抓图的功能，是其他抓图软件很难比拟的。操作上，它是我见过的唯一不用抓图界面（窗口）的。这种设计，使在任何程序窗口中抓图时，不会影响对当前的程序的操作。通过自定义热键，避开与使用的程序热键的冲突后，它就可以抓取你需要的图像。但是，它缺少滚动窗口抓图的功能，因此，要抓取一个完整的Web页面时，就无能为力了，这就需要使用HyperSanp（目前见到的有3.07版），但在操作上它难以匹敌PCS，故配备它，仅为弥补PCS没有滚动窗口抓图的功能。
　　图像浏览类软件，还是首推ACDSee（2.21版），它有预读功能，因此在大内存（≥32M）系统上，浏览图像的速度奇快，并且有索引浏览的功能，还具有文件管理的功能，操作也异常简单。它能支持的图像格式有13种，且有Web页浏览功能。因此，对于用户检索光盘图片及网络上下载的图像文件，其功能完全能满足需要。
　　图像格式转换类软件，向大家推荐SEA和Hijaak 95。SEA不仅是一个非常有名的图像浏览软件，其图像格式转换功能也很不错，操作简单，容易掌握。说到功能齐全方面，至今我没有发现能超过Hijaak 95功能的。它提供多达75种图像的相互转换功能，可能已囊括了现在PC上所有的图像文件格式。唯一不足的是，该软件功能太齐全，得容忍让它占据你30多兆的硬盘空间。
    打印类
　　●首选：《轻松打印》
　　●可选：Fine Print
　　从实用的角度来讲，打印类软件是需要加以注意的。一般的家用电脑，都没有配备打印机，在家里所写的东西，往往需要拿到办公室去打印。有些文件可能还要求在指定的地方打印，因此针对不同的打印机，必须考虑通用性。有两款打印软件可帮你解决此问题，一个是由郑州的轻松软件技术有限公司出品的《轻松打印》软件。这款软件的特点是：安装Windows （3.x或95）后，无论实际使用的是什么打印机，安装一个Epson 1600K的打印驱动程序，并定义其为打印到磁盘后，当需要到办公室打印时，用《轻松打印》软件（可装在一张软盘上随身携带），无论是在DOS或Winodws下，都可调用你的打印文件，得到正确的打印结果。《轻松打印》还支持断针免修，打印针检测等常用功能。同时，对于不同的编排格式，可以实现软件调整而不需要在打印机上来回装纸来调整打印版面的位置。
　　另外一个是虚拟打印软件Fine Print，它可以将几页打印文件按自己的定义，同时打印在一页上（最多可合打8页），这对于有中间输出供传阅和校对草稿的文章极有用，可大大节省打印费用。
    多媒体类
　　●首选：PhotoShop 4.0　零售价：9000元
　　　　　  Winamp 1.6
　　　　　  超级解霸
　　●可选：3DS　　　零售价：1000元
　　　　 　 XingMPEG 3.3
　　要让多媒体电脑发挥其强大的功能，就需要有各种相应的软件，通常将多媒体软件分为：电脑美术、电脑音乐、VCD播放等几类。
　　电脑美术类软件常用的有PhotoShop、CorelDRAW、3DS等。这些软件在制作图片、图像以及动画方面有很强的功能，但使用比较复杂，不容易掌握，而且要求使用者具有一定的美术知识。有兴趣的朋友可以从PhotoShop入手，逐渐掌握电脑美术的一些技巧。
　　电脑音乐是目前的大热门，有一块好的声卡，加上一个MIDI键盘以及音源器等，可组成一套电脑音乐系统，配合相应的软件，可以用电脑进行音乐创作。普通的用户可能用不着这些，因为这显得过于专业。但是，用电脑播放音乐确实可以让大家在操作电脑时得到实实在在的享受。在此，我强烈向大家推荐使用MP3音乐文件，好处就是用同样的价格，你可以买到10倍于CD片的音乐曲目，音质绝对不比CD片差。要使用MP3，最好采用Winamp（最新的有1.6版 ）作为播放器。其功能的强大，是目前所有的MP3播放器难出其右的，且为共享软件。在使用中建议最好将MP3文件直接拷贝到硬盘上使用，如果一次拷贝60兆的话，就可以供1小时播放了，在硬盘空间允许的范围内一次可多拷贝一些，听够后，再换上其它的曲目。这样做的好处就是可以大大减少对光驱的磨损。在操作电脑时，美妙的音乐伴随你进入物我两忘境地，是不是有超凡脱俗之感呢！
　　在VCD播放软件方面，过去是Xing的地位难以撼动，现在则有点群雄争霸的味道。较早的《金山影霸》已让大家领略了国产软件的风彩，新近出品的《超级解霸》以其强大的功能，超强的纠错能力稳稳地占领了市场，它的播放质量绝不在Xing之下，而且还有自动关闭屏幕保护、拷贝VCD片段、画质可调的特点，除VCD光碟外，它甚至可以播放MP3音乐。
    杀毒类
　　●首选：KV300＋　
　　　　　　　　　　　　　　 零售价：260元
　　●可选：VRV杀毒套装软件　零售价：199元
　　　　　 瑞星杀毒软件　　　零售价：230元
　　　　　 KILL 95
　　计算机病毒早已成为计算机界的一个热点问题，参与研制、开发杀毒软件的公司很多，软件的前景也很好，所以市面上的杀毒软件也越来越多。新上市的KV300＋，在“查、杀、修、扩"的基础上新增了内存驻留防毒程序(KVSUNDOG.EXE)。用户从 KV300＋中将不加密的KVSUNDOG.EXE安装在自己的系统批处理文件中(AUTOEXEC.BAT),启动系统时就可自动实施监控防毒。而且它还为WORD用户专门设计了一套方便实用高效的杀毒工具KV300 FOR WORD 1.0，专门查杀WORD结构的宏病毒,可安装在Windows3.X、Windows95、Windows NT上使用。
　　KILL95也是一个不错的杀毒软件，具有很强的交互性。它是根椐已知的各种微机病毒结构和工作机理而设计的专用程序，用于清除病毒， 对已感染病毒的系统和程序文件进行检测和可靠的恢复。KILL 95的知识库存有每一种病毒的具体结构和突出特征， 因此可以准确地指出所感染病毒的名称及其位置，并对系统和文件进行可靠的恢复。 当用户系统或文件被病毒感染后，由于没有备份而不能删除时，使用KILL95是最佳选择。 考虑到普通用户的技术水平，KILL 95在设计上尽可能减少对用户的使用要求，对各种复杂情况的判定和处理均自动完成，并在屏幕上给出清晰明了的说明，不需用户对病毒有任何了解。 任何人均可在几分钟内学会使用。 KILL95还具有良好的软硬件兼容性，可在各种PC兼容机上运行。
    翻译类
　　●首选：金山词霸Ⅱ　零售价：78元
　 　　　　 东方快车 　 零售价：160元
　　●可选：即时汉化专家6.0(增强版) 零售价：260元
　　　　　  汉神网际套餐
　　翻译软件可分为两类。一类是动态汉化软件,以《东方快车》、《即时汉化专家6.0》和《汉神网际套餐5.0》为代表。其中《东方快车》上市以来，一直受到用户的好评，它可以智能识别程序区、文件名、目录名等不须翻译的内容，保持文章的原有风格；可智能处理汉化后容易模糊的字；可同屏显示国标码和BIG5码；可自动形成专用小词库，运行速度快；可对全屏实现鼠标即指即译。另一类是词典翻译软件,以《金山词霸Ⅱ》等一批在线词典软件为代表。《金山词霸Ⅱ》不仅能够提供在线式男女即时发音，而且还加强了对网络的支持，对于经常上网的用户，是一个不可缺少的助手。
