Windows95下的抓图软件Hyper Snap

　　 随着Windows95的流行，越来越多的游戏都把它作为运行的平台，这些游戏大多需要DirectX技术支持，由于DirectX的特殊处理，使一般的抓图软件对它无能为力。现在有了最新版本的HyperSnap3.05，可以很轻松地抓取使用DirectX技术的游戏画面了(该软件在《电脑报配套光盘第二辑》中可以找到)。
    HyperSnap3.05是一个共享软件，它具有抓图、看图等功能。下面简单地介绍一下使用方法。
    在文件(File)项中，包括打开、保存、设置打印机、打印等项。其中文件保存功能中的选项(Option)可以设置存储的格式，HyperSnap有BMP、GIF和JPG三种格式存储。存为BMP格式时可以设置颜色深度，也可以选择最下面的(Auto select format best for current display mode)来自动使用当前显示模式；以GIF格式存储时可以把文件存成交错模式(Interlace)并可以选择透明背景(Transparent background)的颜色，还可以选择颜色转换质量(Color Reduction)，分别是最快的标准模式(Match ins tandard, balanced color map)、中等质量的优化模式(Optimize colormap)、高质量抖动模式(Optimize colormap and dither)；以JPG格式存储时可以控制JPG的输出质量(Output quality)，范围从5％到95％之间，也可以存成渐进(Progressive)模式。
   编辑(Edit)项中可以进行拷贝、剪切、粘贴等操作，也可设置拷贝格式(Copyformat)，剪取部分图像(crop)或取消剪取(UndoCrop)。
    观看(View)项中可选择(Fullscreen)满屏方式来看图像，可以选择前一幅(Previousfile)或下一幅(Nextfile)。
    抓图(Capture)项中，可以进行各种形式的抓图。1.满屏(Fullscreen)方式，抓取屏幕上的所有图像；2.窗口(Windows)方式，当打开多个窗口时，可有选择地抓取；3.当前活动窗口(ActiveWindows)方式，只抓取处于打开状态的窗口；4.当前窗口不带边框(ActiveWindowswithoutframe)方式，只抓取窗口内的内容，不包括边框；5.指定区域(Region)方式，抓取时间用鼠标拖动选择。还可以重复上一次抓取(RepeatlastCapture)。另外，每一种抓图方式都有自己的热键。
  在这些功能中，最重要的就是抓取使用DirectX技术的图像。当按下“Scrolllock”键时，就意味着抓图已完成。笔者用它抓《神奇传说—时空道标》中的画面，感觉特棒。抓图菜单中可以用快速拷贝(Quickcopy)来把抓取的图片送到剪贴板，也可选择快速打印(Quickprint)输出到打印机，还有快速存盘的功能(Quicksave)，可以把抓取的图像保存起来。
    选择(Option)项中，可以重新定义一套自己的抓图热键(ConfigureHotkeys)；取消指定区域抓图时屏幕上的帮助信息(Hide help Region Capture)；还有最小化时只是一个图标(Display tray Icon only)；以及菜单切换(ToggleMenu)。
    帮助(Help)项中，包括HyperSnap的教程、注册信息等。有助于我们更好地使用HyperSnap。
    总的来说，HyperSnap是Windows95下非常优秀的抓图工具，操作简单，功能较强，对那些喜欢Windows95游戏的玩家来说，是必备的工具之一。 (黑龙江  李鹏)
