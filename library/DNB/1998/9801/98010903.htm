Intel Media Benchmark初试印象

      Intel Media Benchmark是Intel公司推出的一款全新的测试软件，主要用于测试计算机的系统信息及各种性能指标(在《电脑报配套光盘》第三辑中收录了Intel Media Benchmark 1.0 for Windows 95 MMX)。，我们经过使用，感觉该软件虽然小巧，但仍然不失为一种比较实用的测试软件。
   一、安装印象
　　该软件的安装过程简单，不需要用户任何多余的设置。Intel Media Benchmark深蓝色的安装底面是标准的微软风格，颇有些FoxPro 2.5b的味道。Intel Media Benchmark是Intel公司逐渐向软件领域靠拢的尝试，加之微软是Intel的盟友，其软件有微软之风就不足为奇了。（听说，微软也正在逐步向硬件方向靠，微软的硬件产品会不会有Intel的痕迹呢？）
　　该软件的使用也非常简单，内容不多，但实在，“麻雀虽小，五脏俱全”，能满足基本的测试要求。
    二、使用印象
　　安装完毕后，就可以启动它。首先出现在面前的是Intel Media Benchmark的版权信息窗口，系统信息（System Information）窗口重叠放置在版权信息窗口下面。在软件对系统的测试未结束前，版权信息窗口是不会自动消失的。当系统信息测试完毕，版权信息窗口自动消失，呈现出当前计算机的系统信息。系统信息的测试包括：Processor（处理器）、 Processor Speed（处理器速度）、Operating System（操作系统）、Memory（内存）、 Video Card（显示卡）、 Video Resolution（分辨率）、 Hard Drives（硬盘）、Multimedia（多媒体设备，主要用于测试光驱、声卡等多媒体设备）。我们选择的实验机器的CPU从386一直到Intel Pentium 166MMX，均能得到正确的测试结果。
　　将系统信息窗口最小化后，就会看见Result（结果）窗口，其测试项有：Overall（整体性能）、Video（视频）、Image Processing（图像处理）、3D Graphics（三维图形） 、Audio（声响）。选中某一测试项后，按右下方的Run按钮，就会执行一段相同的测试内容，然后将测试评分显示在Score中，可以藉此大致了解所使用计算机的性能。
    三、总体印象
　　Intel Media Benchmark的测试项目安排合理，尤其是系统信息的项目设置，基本上包含了一台个人微机的基本配置。并且，其系统信息测试结果比较准确，速度也比较快。
　　但同时，我们也认为，该软件存在着一些有待改进之处。
　　1.在Result（结果）窗口中，测试结果所得到的评分是一个很单一的数字，要是不通过多台机器的综合比较，仅从一个数字上根本看不出系统配置的优劣。
　　若实行百分制，并且说明哪一种类型机器的得分是多少，这样不就可以很清楚地了解所测试的机器的档次了吗？
　　2.我们认为在测试进行过程中，如因人为原因故意中断测试，那么在测试恢复后，所得的结果与正常测试的结果应相差不大，而在Intel Media Benchmark的使用中却发觉并非如此。用测试Video举例，正常得分在127上下浮动，于是便在另一次测试中，通过Ctrl－Alt－Del使系统暂停，恢复后，测试分数一下就跌到54，后来，又试了几次，发觉暂停时间越长，得分就越低。这种现象在其它选项的测试中同样存在。
　　3.测试项中有Image Processing（图像处理）测试项，而对图像效果却无法测试。
　　不过，这毕竟是Intel Media Benchmark1.0版本，要求不应太高，让我们期待着更新版本的Intel Media Benchmark。
