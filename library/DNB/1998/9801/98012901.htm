轩辕圣战录

　　传说在五千年前，曾被轩辕帝大败的蚩尤靠魔力重生，联合了夸父族、苗族及妖鬼邪神等部族打败了轩辕帝。他在杀死轩辕帝后大肆屠杀华夏大地上的汉族人民。你，一个生活在五千年后的无知少年，无意中进入远古的战场。怎样才能带领祖先们击溃蚩尤的魔军，挽救苍生于水火之中呢？只能扬起手中的剑，统率着传说中的部队，带着心中的希望去浴血沙场，收复失去的大好河山。
#1 祈神台
　　此关先建设出魔法城，这样你就有足够的魔法值使用回复术为你的部队加血。要研制出草衣给予英雄装备，并多练等级，以备下一关的恶战。
　　隐藏宝物位置：1.地图上方的铜矿右边，2.地图右上方丛林中，3.地图左边的山岳中（注意有2个），4.地图上方靠湖泊边（木桥对面小湾边），5. 地图左上方（地图上已有宝箱的左上方），6.敌方首都左上方，7.左方湖边村庄的上方（靠湖边）。
#1 青要之山
本关的资源很多，但铜矿和原木林都离你很远。先派工兵采集我方都市右下方河边的原木林，并占据村庄将其建设成魔法城与都市，派一辆指南车到左方河边的山岳旁搜索，可发现一座矿产值为5000的赤铜矿，占据它后，就不用担心铜矿资源了。
　　本关中，与我方首都右方村庄中的村民对话，三回合后一名叫雷的战士就会加入我方的队伍，有了他后，我方都市就可训练枪矛兵了（如不与村民交谈，雷会在十回合后加入）。
　　与我方首都左方村庄的村民交谈后，再至村庄前方用指南车搜索就能找到隐藏的桃花村，与村中的老人对话后，他会教给你训练灵猿的方法。
本关地图右下方河的右岸有一格平地，搜索后可得到九大神兵之鱼骨剑（魁装备）。
　　隐藏宝物位置：1.地图最左边居中位置，3.地图最上方居中位置，4.左方河岸对面铜矿上方，5.右方山岳之中（两个村庄及两个铜矿所在的山岳）6.阴风所在下方的山岳左边。
#1　　恶夜森林
此关将树林中的巨熊王收服后，可获得巨熊部队的帮助，并且我军将具备训练巨熊部队的能力。
　　地图中央的沼泽中有一条隐藏的小龙，打败它后，可得到轩辕黄帝遗留下来的轩辕战甲之一的轩辕战盔，并且战斗结束后，小龙会加入我方。
　　本关右下方的山岳之中，搜索后可得到八牛弓（给大山装备）。
　　隐藏宝物位置：1.地图右方居中村庄下方的大树下，2.森林中两颗枯树中间，上下各有一村庄与原木林的地方，3.离首都最近右下方村子的旁边，4.三个青铜矿所在左方顺数第二棵树下，5.右下方三个成斜线的村庄右边。
#1　 槐江之山——死守       
　　此关注意：战场下方的人头马兵为启动式战术，不走进其行动范围就不会主动攻击我军，因此不要去主动招惹他们，以免腹背受敌。
#1　　槐江之山——赌注
不用担心失败，如被英招打败后，游戏会自动让你重打直至胜利。
#1　　槐江之山——反击
本关进入地图中央的村庄中，村民会告诉你赵村中有一位铁匠会造战车，派指南车到一处树林环绕的平地搜索，可找到赵村，与赵师匠交谈后，就可训练战车了。          隐藏宝物位置：1.地图最左边的原木林旁边，2.火山与山岳中，3.火山左方（地图上已有宝箱与火山之间），4.左方桥的右河边，5.河边的森林中， 6.左边两个铜矿下方的原木林边。
#1 雷泽
打败蚩尤军的夸隆后，大家决定到不周山去找凤后，以壮大反抗军的力量。在去不周山的路上，经过一处叫雷泽的沼泽，这里的沼泽中往着一种雷兽，英招建议捕杀雷兽，用兽皮来作鼓。
　　此处若接爱英招的建议，杀死雷兽后，我方所有部队的士气能永久提升10％， 但雷兽不会帮助我军。如不接受建议，则打败一只雷兽后，所有的雷兽都将加入我军，我军可以训练雷兽部队，利用雷兽来对付此关蚩尤军的水妖部队。
　　本关可以从沼泽中搜索到轩辕战腕，不过只有雷兽和战船能拿到。在左方一处村庄中与村民对话，得到吴师匠的讯息后至河流源头处以指南车搜索即可发现河源村，如果与吴师匠对话的角色符合条件则可得到吴师匠传授造船技术（条件：天资7以上）。
　　隐藏宝物位置：1.一处海湖中，2.在两个雷兽之中的小海湖中，3.离我方大部队不远下方的沼泽中，4.右上方四个铜矿之间的大树下，5.右方屏幕居中的山岳旁，6.在一个原木林左上方的河泊边，7.左方的森林中三个成品字形的铜矿边， 8.部队下方的海湖中，9.第八处宝箱的左下方，10.左下方的山岳中（两个）， 11.中央的河泊边，12.敌方首都左边的小湖边，13.敌方铜矿群（五个在一起的铜矿）下方，14.右边靠草原的河泊中。
#1 不周山——突围
　　本关右下方雷一行人全部上船后走左边水路即可快速突破敌人封锁。当雷一行人中任一人走到左上角接近起义军之后，就会有事情发生，打败守将甘后即可过关。
　　此关中左下方一长形岛的东西端各有一村庄，接受东村的要求可得三样宝物的讯息，在岛的南方以指南车搜索可得灵狐尾等三样宝物。而接受西村的要求可得九大神兵之透甲枪的讯息，然后在岛的南山搜索可得透甲枪（雷装备）。
#1　　不周山——夹击
此关一定要注意在不周山右边的森林中有很多隐藏的敌人，并且各处有不少隐藏的陷阱。占领森林中靠近河湾的村庄后，村民会告诉你轩辕黄帝的战靴在森林里，在村庄的左边搜索就可发现轩辕战靴。
　　本关中有8个隐藏的宝物。
#1　　地狱谷
三个魔法连接点一个在右方湖泊下方的树林边，一个在右下方湖泊下方的绿洲上，第三个在左下角河流尽头，派指南车搜索后就可发现他们，全部占领后即可破坏锁龙柱。
　　本关下方一处村庄的前面有一小水窑，内有九大神兵之掩月刀（英招装备）
　　本关有11处隐藏的宝物，在右上角的山岳中有隐藏的铜矿。
#1　　英雄壕
敌军行动的第一回合，风伯就会召唤骷髅兵出来参战，这些骷髅兵被打死后下一回合仍会再生。在我方打倒三个骷髅兵后，少典会出现加入我军，由他打死的骷髅兵不会再生。少典打倒红色骷髅兵后则骷髅兵会全部消失。
　　在地图左上方一处山岳枯木下隐藏着轩辕战甲。
派指南车搜索我军首都上方沙漠可找到九大神兵之化天戟（应龙装备）。
　　此关中敌我阵营相当接近，攻下敌首都后也可过关。不过敌人首都附近有大量伏军和陷阱，特别是在双方之间的树林中，所以最好不要贸然前去进攻。
本关共有13个隐藏的宝物。
#1　　流波山——埋伏
本关密道在地图右上方山岳之中，即陷阱标志的右下方，用指南车搜索可找到一个隐藏的魑魅，打败他后就可找到密道。
#1　 流波山——牺牲
　　此关不论输赢，少典都会战死，但却为大伙撤退赢得了时间。
#1　　流波山——归剑
　　此关六个回合之后，我军就可训练战船了，用船将魁载到右边海中的七星岛上（东边海中一处七个小岛围绕的中央）。记往一定要魁亲往，否则张天师不会表明身份。
当魁找到了张天师后，第一次谈话时张天师会把少典的七星剑送给魁。在下一回合中，魁出村后要再次进入村庄与张天师交谈，如果这时玩家收集了四套轩辕战甲，则魁与他交谈两次后，他会加入我军并得到他手中的轩辕神剑。（注：加入我军是在击败风伯之后）
本关有十个隐藏的宝物。
#1　　魔星——妖塔
注意：此关必须有五件轩辕战甲才可进入。
　　小助一行人到达魔星，要将外围的八个魔法城全部占领后才能进入中心的妖塔与雨师决战。每座城内都有很强的蚩尤军守卫，要小心应付才能过关。战役开始后可强攻下首都左方的敌人堡垒，攻下后，敌人就不能再训练部队了。
　　地图东北方有一隐藏的硝石矿，找到后就可训练雷火兵了。
　　本关有五个隐藏的宝箱。
#1　　魔星——重逢
　　由于小助率领的反抗军主力迟迟没有回来，魁与张天师率领的起义军已经败退到了当初起义的祈神台，眼看战局已十分危险，但大伙都决心与蚩尤军战斗到底。
本关开始只有少量兵力，三回合后小助会率援助会率援军到达。
　　本关无隐藏宝物。
#1　　首山
经历了无数的征战，少典与雷的牺牲后，与蚩尤决战的时刻到来了。为了替死去的战士报仇，重建自己的家园，让我们为自由与光明的到来而战吧！
　　本关有九个隐藏宝物
#1　　逐鹿平原
　　眼看蚩尤就要被小助击败，风伯却挺身而出替蚩尤接下了小助的招式，蚩尤趁机逃到逐鹿平原招集了剩余的部队与反抗军开始最后的生死决战。
　　本关敌人就在左上方，数量众多，小心对付吧。
　　此关中仅有二个隐藏宝箱，在地图左下方（我方铜矿左边的平原上）
#1　　结局
小助与蚩尤的决战降临了，这时雨师突然出现在战场上，原以为他要帮助蚩尤作战，谁知他突然出手打死了蚩尤，并变化成了雷。他说雷已死在魔星之中，现在的他叫炎居，是炎帝之子，因被困在魔星之中，才寻回了自己失去的记忆，他的父亲当年被蚩尤战败后忧郁而死，现在他要向你发动挑战，得到天下，小心应战吧。
#1　　 注意事项
　　1. 游戏中要多利用指南车搜索隐藏的宝箱、陷阱及埋伏的敌人。
　　2. 每关中的木材与铜矿资源有限，要小心加以利用
　　3. 游戏中建议使用手动战斗，这样战胜敌人的机会会大大增加。
　　4. 当技术力和法术力的数值棍加至最满时，等级就可提升一级，这时你训练士兵所用的资源就会送减少。（如原先训练一个工兵需要150的钱币，技术力升级后只须90钱币）。
　　5. 每个人身上只能装备一种武器。
　　6. 在后期的战斗中，由于敌方一开始魔法力就很高，所以自己的首都一定要留守一名厉害的战士，否则敌人会用幻象术招集部队攻下你的首都。首都被占领与小助被打败，都会导致任务失败。
　　7. 士兵的天资如果太差，就不能转职。
