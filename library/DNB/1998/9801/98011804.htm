如何使主页更新后可自动通知网友

　　现在制作发布个人主页的人越来越多，各位主页主人为了使自己的个人主页门庭若市，想尽了不少方法。这里介绍一种方法，可以使自己的主页在每次更新后自动通知自己的网友。
　　只要在自己的主页中加入如下一段：
　　<form action =http://www.netmind.com/cgi－bin/uncgi/url－mind method =“GET">
　　<input type =“hidden" name =“url" value =“http://www.nease.net/～zgx">
　　<p align =“center">
　　<font color=“＃800040" size="2">如果你希望本页更新时能收到通知，<br>
　　请留下你的Email地址:</font><br>
　　<input type =“text" size=“30" name =“required－email">
　　<input type =“submit" value =“确认 "> </p> </form>
　　其中在<input type =“hidden" name =“url" value =“http://www.nease.net/～zgx">中应将地址http://www.nease.net/～zgx相应的换成你自己的主页地址，其它的中文字可以换成任何你认为合适的其它文字。
　　它在浏览器中的显示效果如图一所示：
　 只要到你主页的网友在文本框中填入他的E－mail地址并按“确认”键，浏览器就会带他到www.netmind.com服务器去确认一下，同时也可以在这里设定一些参数，如检查的频度等等。以后当www.netmind.com发现你的主页一有更新它就会自动发E－mail通知你的客户。
