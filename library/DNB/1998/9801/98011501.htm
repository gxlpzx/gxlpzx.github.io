Norton Utitlities for Windows 95

      在工具软件领域中享有盛名的Symantec公司，于微软推出Windows 95 Preview Program之际，推出了Norton Utitlities for Windows 95。
　　虽然Windows 95本身附有一些工具软件，像可以检查磁盘的ScanDisk及整理磁盘的Defrag等，而且这两者也是经Symantec授权给Microsoft，但跟Symantec自己Norton Utilities里的SpeedDisk以及Norton Disk Doctor比起来，Windows 95内附的工具还是相形见绌。
　　另外，一些原DOS或者Windows 3.X的工具程序到了Windows 95都变成了“累赘”，这是因为Windows 95本身是32bit，旧程序都是16bit，自然在效率上不太好。而且最重要的是，一些旧版工具中有关硬盘整理或检查的部分，更是无法使用——若有人偏不信邪要用，那结果将是整个硬盘的资料就此说Byebye。
　　安装
　　NU/Win95本身提供了一个叫“Pre－Installation Tune－Up”的工具来协助你做安装Windows 95“热身版”的准备工作，此程序对于尚未或准备安装Windows 95的用户来说非常方便安全，因为NU/Win95提供了一些可以在旧DOS(3.3版以上)下使用的工具程序来先帮你整理并清除硬盘空间、制作备份开机磁盘，以及修理文件分配表(FAT)等等，逐步引导用户做好安装Win 95的准备，并确保要是安装Win 95失败，还能有办法开机并挽救资料。
　　由于是“Windows 95专用程序”，所以安装时你可以使用Windows 95 Control Panel里的Add/Remove Programs来安装NU/Win 95。Win 95会自动从A、B或CD片里找出使用Windows 95标准安装或反安装程序(Standard installation/deinstallation procedure)的安装程序。Windows 95的Add/Remove Programs功能可以帮你自动找出安装程序，你再也不需要知道安装时要键入什么命令。
　　NU/Win 95的安装跟Win 95一样，是使用安装精灵(Setup Wizard)方式，一步一步引导用户去安装整个程序，所以非常方便。在安装的过程中，有一个有关DOS程序的菜单可能是比较“奇怪”的。
　　“奇了?怎还会有DOS的程序呢?不是说Windows 95已不用DOS了吗?那为何NU/Win 95还要提供DOS呢?这岂不是多余吗?”
　　原来，虽然Windows 95是“似乎”没DOS这东西的存在，可是Windows 95为了兼容性，原DOS还是保存在Windows 95下，只是你没看到而已。NU/Win 95之所以要提供DOS版的工具，主要是当你进不了Windows 95的图形用户界面(GUI)，而被迫开机后使用其Command line mode(即MS－DOS 7.0)时，可以照常使用NU/Win 95来整理硬盘而不致破坏Windows 95的VFAT文件分配表——由于Windows 95支持的长文件名(Long Filename)，所以若你使用一些旧的DOS硬盘整理程序，将会造成使用长文件名的文件被破坏掉。
　　安装好，重新进入Windows 95，开始了我们的“探索”……
　　NU/Win 95的成员介绍
　　NU/Win 95在Windows 95主要有几大主角(程序)，这些程序全部都是32位，而且完全支持Windows 95的新功能，如长文件名。分别介绍如下：
　　●SysDoc(系统医生)
　　当我们安装好NU/Win 95后，再次进到Windows 95，首先看到上场的是一个SysDoc(系统医生)的“仪表板”。这个“仪表板”并不是做来好看的，而是用来监视并随时测试你系统的“Norton System Doctor”。
　　这位医生会从你开始使用Windows 95到关机为止，时时刻刻保护你的系统：当系统任何的资源(resource)快被“吃”光时，它就会出来警告你并给你建议采用什么适当的方法预防系统死机；它也会时时刻刻帮你注意着硬盘，当硬盘文件分配表的断裂(fragmentation)太严重的时候，它也会自动跑出来警告你，并询问你是不是要叫出“Speed Disk来整理你的硬盘；当有文件系统损坏时，它则会出来问你是不是要请它的兄弟“Disk Doctor”来修复。
　　由于这位“医生”扮演着如此重要的角色，所以将它称为“主治医生”不为过。
　　●SysInfo(系统资料)
　　系统资料家(SI)可以提供电脑的所有资料，包括：系统、打印机、磁盘机、网络、显示器、内存及多媒体等类别的资料。所列出来的资料可用丰富、详细来形容，甚至远比Windows 95本身的Device Manager要来得仔细。
　　另外，SI除了可以列出你的电脑的所有资料外，也具有测试(benchmark)的功能。使用其32bit的benchmark功能来测试你的电脑，并将结果与其它标准电脑的表现作比较，让你知道所使用的电脑是否已发挥了全力。
　　NU/Win 95跟中文版Win 95
　　很不幸，NU/Win 95跟中文版Win 95并不太相容，就如许多文章所说的那样，“Speed Disk”跟“Space Wizard”便会有问题，而且“Norton Disk Doctor”在检查时往往会报告说你的硬盘目录有问题。
　　●NDD(磁盘医生)
　　Windows 95本身附有一个可以检查硬盘的工具，叫ScanDisk。虽然说它能修复硬盘上有问题的文件夹(Folder)等，但跟“诺顿磁盘医生NDD”比起来，那可逊色多了。
　　NDD可以让你先对数据进行备份，以免在修复时不小心丢失了你的资料。除了检查文件与簇(clusters)的问题，NDD也会帮你检查并修复硬盘上的分区表(partition)和文件分配表(FAT)。
　　另外NDD还提供许多选择，像发现磁盘或文件有问题时，你可以选择让NDD自动修复，或者由你个别决定是否要修复某丢失掉的资料。我们这位“医生”跟上面介绍的“主治医生”诺顿系统医生(SysDoc)刚好是一对兄弟，当“主治医生”发现您的磁盘有问题时，它就会请NDD出来治疗检查一番。
　　当然，更让人开心的是，它也可以自动地在后台随时为您检查、保护并修正文件写入写出的问题。
　　●SpeedDisk(磁盘整理)
　　Speed Disk算是老成员了，用NU 8.0或更早版本的用户都认识它，因为它能帮用户清除或修复零散扇区(fragmentation)，让你的资料不会被破坏，甚至可以加快你硬盘的存取效率。
　　可是，原旧版本的Speed Disk并不能在Windows 95下使用，因为Windows 95使用了不同文件分配表(FAT)，若用了则会破坏掉所有长文件名(longname)的结构。所以，NU/Win 95里的Speed Disk自然除了是32bit外，对于长文件名有相应的支持。
　　新版本的Speed Disk除了在补救并整理零散扇区上有很大的改进外，它也提供了许多新方法，如您可以决定哪些文件或文件夹(目录)要放在最前面或者依不同的顺序排放文件，来降低以后再发生零散扇区的可能性，并且加快硬盘存、取的速度，让你使用Windows 95时能高枕无忧。
　　Speed Disk跟后面介绍的Space Wizard对于中文Win 95有点水土不服，原因就在中文版Win 95安装后会造出几个中文名字的目录，而Speed Disk根本不认识这些目录，所以当Speed Disk处理到这些目录时，会认为这些目录是错误的，而无法加以处理，并询问你是否要使用上面提到的诺顿磁盘医生(NDD)来检查及修复，还是要结束Speed Disk。
　　其实，就算你用NDD来检查也无济于事，因为NDD也不认识那些中文目录，在检查后它会报告您的文件目录有错误。这点算是最大的遗憾(其实，中文版Windows 95里的Defrag也处理不了中文目录)。
　　●SpaceWizard(空间精灵)
　　通常，我们的硬盘里堆放了很多文件，这些文件有的我们常用，有的可能一年也用不到一次，如果说要您每一个目录去一一清理出这些文件，我相信你也不愿意。NU/Win 95里的Space Wizard就是用来帮你做这件工作的，它会去帮你找出硬盘上所有的暂存档、暂存目录、旧的备份文件、或者一些比你指定日期(日数)还旧的文件等，然后列出一张清单给你，由你来决定哪些要砍掉，哪些要留着。当然，这些文件也不一定要被清掉，你也可以要Space Wizard将之压缩或者搬到其它的磁盘存起来。Space Wizard是一套空间清理及管理家，它除了可以做好上面的事外，还可以为你以后的其它32bit程序分配最大的空间来安装。
　　●UnEraseWizard及RescueDisk
　　NU/Win 95这两个成员(程序)算是一般用户较常用的。虽然Win 95本身有一个RecycleBin(资源回收站)，可以将你本来要砍掉的文件给回收。可是，如果你在Win 95里的DOS视窗里将文件不小心砍了，那资源回收站可就帮不了你的忙。你也许有习惯备份系统，可是要是某一天你的系统不能开机了，你怎么办呢?
　　而且，如果你发现在装某套程序时，Win 95的设定不见了或者Win 95的注册文件(Registry)被破坏，甚至CMOS资料被清掉，你又怎办呢?这些问题的解答就是NU/Win 95的“UNErase Wizard反删除精灵”及“Rescue Disk急救箱”。
　　UnErase会采用精灵(Wizard)式的界面来简化你挽救文件的过程，它会去找你不小心在DOS视窗或者被某些应用程序不小心砍掉的文件，然后让你选择要由UNErase自动帮你或者由你自己救回文件。UnErase会取代掉原Win 95的RecycleBin(资源回收站)，改用自己的SmartCan(智能回收站)，这是为了能百分之百地救回您的文件。
　　Rescue会帮你制造(或叫备份)一张急救磁盘，让你万一系统出错误时能抢救或者恢复您的系统资料。虽然Win 95本身也提供类似的功能，可是Win 95本身的急救功能并不能让你选择要备份哪一个系统设定、文件及资料，而且,Rescue可以跟其它的成员如UnErase及SysInfo密切配合，将系统的资料一五一十地记载备份入急救箱磁盘，让你万一不幸系统出问题时，可以很快地恢复原状。
　　反安装
　　诚如先前说的，NU/Win 95是一套专给Windows 95用的程序，所以除了在安装上可以自动外，连反安装都可以很轻松过地完成。同样只要用Windows 95控制面板Control Panel里的Add/Remove Programs来反安装NU/Win 95即可。你会看到Windows 95列出一些你安装过的程序(都是Windows 95专用的)，只要选用NU/Win 95项，按下Remove删除)，即可很轻松地将NU/Win 95给反安装掉。这是for Windows 95程序的一项便利之一。
　　结语
　　NU/Win 95本身是一个相当不错的工具软件，撇开跟中文版Win95的不相容，它除了的确是每一位Windows 95用户必备的工具外，更是原NU 8.0或更早期版本的用户不可错过的升级软件。（广州   阿伟）
