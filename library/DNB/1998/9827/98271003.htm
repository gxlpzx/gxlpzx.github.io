电脑大管家---AutoMate

　　AutoMate能够自动完成用户手工对计算机的几乎所有操作，从打开/关闭应用程序到键盘组合操作、各种按钮的点击甚至菜单的选取。
　　设置：请点按“View”，接着按“General Preferences”，出现的选项应全选。当第二项被选中后，一个用户自定义任务（New Task）在启动时，下一次加载的时间AutoMate就为你自动地安排好了。AutoMate按一定的时间间隔来扫描所有的Task，然后把结果保存在数据库中。这样，当AutoMate启动时就会到这个数据库中自动寻找相关信息。
AutoMate实用功能详解
　　1. 用户自定义任务制作向导
　　在AutoMate的窗口选“New”下拉式菜单，有两个选项：Procedure(应用程序启动任务制作)和Reminder（信息提示器制作）。
　　选Procedure出现“Add Task Wizard”向导模式，为任务取名，如“Office 97”，点“Next”，下面是自定义任务启动的频度选项，计有“Once”（仅此一次），从每分钟、每一小时乃至每一年时间间隔不等（当任务以一定时间间隔启动后，AutoMate将会自动地设置好下一次启动该任务的时间），或选第三项意即不需AutoMate来启动该任务而是用户自己想在何时启动就何时加载。
　　接着，是选第一次加载该任务的时间，在“Set Time”中按鼠标左键是选分针，右键是选时针，然后是“Late Tasks”选项，共有三项选择，用户可参看其中的详细说明来各取所需。接着点“Next”，这一步很重要，选项较多，因此您自定义的任务在此会有较多的个性化设置，点选“Create Macro”，这就是创建“宏”操作，它为您的工作带来极大的方便。
　　在AutoMate弹出的“Macro Builder”对话框中，先浏览应用程序“Office 97”所在路径，点击最上面的下拉式菜单选“Start An Application”，在“Command Line”的“Browse”中寻找“Office 97”的路径(可以是Windows桌面的快捷方式)，然后不要忘记了点击“Add”、“Apply”以及“Save Macro”。
　　在最上面的下拉式菜单中有很多选项可同时选择，如“Miscellaneous”（杂项），其包含有定制任务启动时发出的声音，自动与Internet断线，关闭Windows等，您可各取所需。最后点“Save Macro”与“Finished”。 “Macro Builder”对话框下拉式菜单选项还有Record Events（录制一个新宏）、Focus a Windows、Wait for a Windows等，当选择了其中的选项后，点击“Add”，这些选项即出现在下面的Action列表中，此时你点选右边的向上、向下的箭头，就能将Action列表的选项的次序重新排定，这样的好处是：AutoMate会为您按Action列表的安排顺序来执行任务（Task）。
　　2. 自定义任务（New Task）的属性调整
　　一个新任务创建以后用户还能根据实际情况进行调整，在图的AutoMate主界面把要修改的Task点亮后选“Properties”，在其后的对话框里选择并修改，方法与创建一个New Task是一样的。
　　3. 启动Task并查看Task运行记录
　　在AutoMate主界面里选一项Task，然后按下“Launch”即可立即启动该任务了。点按File\View Task Log即可查看电脑在各个时间里运行的情况，所有的操作均逃不过它的眼睛。
　　AutoMate的更新：该软件最新的V3.8版本为初学者增加了动态帮助(Active Help)，提供了增强型的任务信息和管理界面。安装后，在“启动组”里生成一程序项。
　　运行环境:有16位和32位版本，32位版本分为标准版与专业版，均可运行于Win95及NT4.0，需不小于10M的硬盘自由空间。其当前版本为V3.8,你可到http://www.unisyn.com去下载评估版本（V3.8e）。下载文件的名称为Automate.exe，大小为6.3M。
