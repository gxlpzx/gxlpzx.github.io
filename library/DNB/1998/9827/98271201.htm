精兵简政 从硬盘开始

　　我们都有这样的体验：Windows95时间用长了，磁盘中不同文件夹下常常会有一些重复的文件，产生的原因不一而足，但对系统性能的不良影响却是显而易见的，有没有什么办法将它们找出来，清理掉呢？答案是肯定的，下面列举两种方法。
　　一、用NU3中的Space Wizard
　　Norton Utilities是大家都很熟悉的工具软件，其中的Space Wizard在磁盘空间管理方面的确有独到之处，用它来清理临时、备份和重复文件，使用简单，效果也相当不错。
　　Space Wizard有两种运行方式：快速清理（Express）和全面清理（Comprehensive）。我们这里要用到的是后一种方法全面清理（Comprehensive），它提供了清理临时文件、一段时间未访问的文件、大小超过某一值的文件、重复文件等多种选择。
　　运行Space Wizard，选择Comprehensive方式，指定要清理的磁盘后，缺省情况下是依次对该盘中的临时文件、6个月以上未访问的文件、大小超过1M的文件进行搜索，每一项搜索后列出符合条件的文件供你选择删除，接下来就是搜索重复文件，注意这里的缺省设置是不搜索重复文件，选择下面“No,perform duplicate file processing”，单击下一步，就开始对指定磁盘中的重复文件进行搜索，稍后列出全部重复文件的文件名、地址、大小等，在列表中单击欲删除的文件（选中的文件名前方框中出现小对号），单击下一步进入删除确认对话框，选择删除、移动和压缩（使用WINZIP）中一种即可。
　　二、使用“图像清洁工”
　　“图像清洁工”（Image Sweeper）是一个专门用来清理磁盘中重复的图片、多媒体文件等的小工具，尽管它主要针对的是图像，但对其它文件的搜索能力也毫不逊色。
　　运行后出现主窗口，输入要搜索的文件夹或磁盘名，单击“Find Duplicates”按钮，就开始对其中的重复文件进行搜索；单击“List All”则列出文件夹中的全部文件。中间是搜索到的文件列表，包括文件名、路径、大小、创建时间等几项。如果是图形文件，则下面一排幻灯式小预览窗口中会出现图像内容。在文件名或小预览窗口上右击就出现一个快捷菜单，可以对文件进行查看、删除等工作。
　　“图像清洁工”可以查找的文件不限于图像文件，通过适当的配置可查出特定的各类重复文件来。在其主窗口左上部的“Option”中，第一行有“Name”、“Size”、“CC”、“Time”四个选项，分别代表按相同的文件名、相同的大小、相同的内容和相同的创建时间进行比较；四种搜索方式还可以组合使用，但要注意其中选中按大小比较之后才能使用按内容进行比较，即条件“CC”依附于条件“Size”。
　　搜索的文件夹也可以按要求指定，单击搜索路径栏旁边的“＋”按钮，就弹出添加文件夹对话框，找到欲比较的文件夹即可。单击“－”则清除选中的文件夹或盘符，单击“C”是全部清除。
　　通过这些设置，我们可以很方便地把磁盘中重复的文件“揪”出来。(湖北　张洁　简之武)
