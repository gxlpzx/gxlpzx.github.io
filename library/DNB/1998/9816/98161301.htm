9804 请你编程

  　9804请你编程的正确答案是：在不考虑只有一个儿子的情况下，同时要求财产份数为整数份且最终分完，则国王至少有9个儿子，财产为81份，每个儿子分得的财产数相等，均为9份。这次读者的编程思路分为两种：循环试验法和剩余财产凑整法。下面介绍两个读者程序供大家借鉴：
　　1唐山读者李文的程序：
　　思路：设变量S为国王最初将家产分成的份数，T为每个儿子的编号亦即第一次分得的份额，则每个儿子应得的全部份额F=T＋（S－T）/10。由题意可知，能满足第一次分配条件的S的最小值应是11，所以可从11试起。如F为整数，说明可以继续分下去，这时将F存入数组D（T），S重新赋值为S－F，T值增1，继续试下去。如遇F不是整数时，说明已不符合要求条件，这时将最初份额增10，重复上述循环，直至S=0为止。最后从数组中依次输出每个儿子应得的份额并输出最后一个儿子的编号（即国王儿子的总数）。
　　10 N=11
　　20 S=N
　　30 T=1
　　40 WHILE S<>0
　　50 F=T＋(S－T)/10
　　60 IF F=INT(F) THEN D(T)=F:S=S－F:T= T＋1 ELSE N=N＋10:S=N:T=1
　　70 WEND
　　80 PRINT ″国王至少有″;T－1;″个儿子″
　　90 FOR I=1 TO T－1
　　100 PRINT ″第″;I;″个儿子得″;D(I);″份″
　　110 NEXT
　　2广西读者黄朝礼的程序：
　　思路：设第i个儿子分得财产前、后剩余财产分别为k(i),k(i＋1),则第i个财产为:a(i)=i＋(k(i)－i)/10,剩余:k(i＋1)=k(i)－a(i)，则可推出:k(i)=i＋k(i＋1)＊10/9；假定国王已将财产全部分到每个儿子手中,为保证财产刚好全部分完,最后一个儿子m所得财产份额为m份,剩余财产k(m＋1)=0，由此推出k(m)，k(m－1)……直到k(1)，若k(1)为整数，则k(1)为财产总数，若k(1)不为整数，则调整m的值再反推。
　　＃include <stdioh>
　　＃define MAX 100　/＊最多100个仔＊/
　　main()
　　{
　　int i,m;
　　float a[MAX],k;
　　for (m=2;m<=MAX;m＋＋)
　　{k=a[m]=m;　/＊最后一个所得＊/
　　for(i=m－1;i>=1;i－－)
　　{k=i＋k＊10/9;　/＊剩余财产＊/
　　a[i]=i＋(k－i)/10;　/＊第i个所得＊/
　　}
　　if((int)k==k)　/＊财产总份数为整数＊/
　　{printf(″国王共有％d个儿子,将财产分成％d份,每人得数如下:\n\n″,m,(int)k);
　　for(i=1;i<=m;i＋＋){printf(″\t\t第％d个儿子得:％f份\n″,i,a[i]);
　　}
　　printf(″\n程序设计:黄朝礼　199834\n″);
　　return 0;
　　}
　　}
　　printf(″没有满足条件的解!!\n″);
　　return 0;
　　}
