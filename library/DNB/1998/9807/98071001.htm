将网上世界搬回家
----Webzip2.0 PR4版介绍

    网上的世界很精彩，然而对于新手来说，总是习惯在线浏览网上的内容，这往往造成时间和金钱不必要的浪费。本人初上网不久，就感觉线上浏览既耗时间、金钱，又消耗精力。即使是在中文主页上，要想迅速找到对自己有用的信息，也不是一件容易的事，更何况找到后还得把它存起来，就更费时间了。因此便想是否有这么一种软件，能够快速地把要访问的一个主页、一个目录、甚至是整个站点都保存下来，断线后再慢慢地看。用心地在网上找了一下，果然找到了不少的离线浏览器(OFFLINE－BROWSER)，下载了几个最新的，使用后，感觉各有所长，其中Webzip的界面友好、使用方便、操作简单、功能强大，本人最为爱用，现介绍给大家。
　　Webzip是一款集下载、存储、在线浏览及离线浏览诸多功能为一体的共享软件。它的主要功能有:
　　●下载一个完整的站点并把它压缩为一个Zip文件
　　●有选择地下载并压缩一个Web站点中所需要的内容
　　●离线浏览已下载的Web站点
　　●方便地将一个已压缩的Web站点移到另一台计算机
　　●通过E－mail将一个已压缩的Web站点传给同事
　　●创建一个自己的“个人内部网”
　　Webzip的安装十分方便，整个安装过程完全为傻瓜式，你只须点Yes或 Ok即可。安装完后自动在“开始/程序/”中添加Webzip项，激活后基本上无须作什么设置，可直接使用。
　　Webzip的界面非常友好，为典型的 Windows风格，分为菜单栏、工具栏、地址栏、功能按钮与信息窗口。需要理解与掌握的是Profiles(屏幕上方最中间)，Profiles功能允许定义所希望下载的内容，软件本身已定义了六项选择，分别为：
　　●Selected Page(All)——下载所选择的页，包括所有的文本、图像、声音、Java、ActiveX等信息，资源局限于当前站点。
　　●Selected Page(Text,Images)——仅下载当前站点中所选择页的文本与图像信息。
　　●Selected Directory(All)——下载当前站点中所选择目录的所有信息,包括子目录。
　　●Selected Directory(Text, Images)——仅下载当前站点中所选择目录的文本与图像信息，包括子目录。
　　●Selected Site(All)——下载当前站点中所有信息。
　　●Selected Site(Text，Images)——仅下载当前站点的所有文本和图像信息。
　　其中Selected Directory(Text，Images)最常用。然而对于目前的网线速度来说，下载图像资料耗时太多，因此可以自己定义一个Profiles文件，以达到只传文本的作用，步骤如下：
　　在Address的位置随便键入一些字符，在Profile位置选择None，点击右上角的 Go 按钮，一直点击Next按钮直到出现 Download Method(Profile) 页。选择Create new profile项，点击Next。出现File Types页，取消 All之前的勾，点击选中 HTML 及 Other(前面出现勾)，点击Next，在 Depth of retrieval中选择 All levels 项，在 Page Location 中选择 Current directory 项，然后在Resource Location 中选Current site项。点击Next，直到出现Save Download Method(Profile)?页，选 Yes 项，点击 Save As，在出现对话框中输入名称如Selected Directory(text only)。点击Next，出现Congratulations!页，选择Don't schedule this task项，点击Finish，即完成。
　　现以一实例说明该软件的具体操作方法。
　　如本人欲将中国公众多媒体信息网－南宁站点的文本与图像信息下载至硬盘，步骤如下：
　　1.在Address位置键入南宁信息网的网址 http://www.nn.gx.cn/。
　　2.用鼠标在Profile中选择Selected Directory(Text，Images)。
　　3.点击右上角的Go。
　　4.点击Next，至出现 Download Destination页，选择 Download and zip this website，点击Next，至出现Congratulation!页，点击Finish。
　　5.此时如尚未连入Internet，则自动弹出拨号窗口，提示你拨号；如已经连入Internet，将显示Schedule Tasks 窗口，然后 Webzip 软件会自动寻址，自动下载选定的站点内容，此时你可以点击左边的功能按钮 Connections，观看下载的情况，如连线时间、传输速率及下载的文件信息等，也可以点击Browse按钮在线浏览下载的内容，完成后自动提示断线。
　　Webzip是一个功能强大的离线浏览器，除了基本的下载与离线浏览外，还支持智能更新(允许通过仅下载新的或更动过的内容来快速更新信息,而不用将已有信息重复下载)、智能恢复、继续 (允许在下载过程中随时中断或继续)、日志等功能。
　　无论是对新手或是老手，离线浏览器都是必不可少的，它能节约大量宝贵的时间，使你不必手忙脚乱地下载网上的信息，Webzip确实是一个不错的选择。
