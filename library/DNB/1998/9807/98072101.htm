Windows NT Server——支持Windows 95的最佳网络平台

　　随着Windows 95的推出,许多企业开始考虑用Windows 95的新功能来增强他们前台桌面机的能力:如多任务,Internet连接,集成的FAX服务与电子邮件服务。当你在考虑Windows 95带来的新机会时,实际上也是一个理想的时机来考虑利用Windows NT Server来增强你现有的网络系统或者是建立一个新的网络。
　　Windows NT Server既是一个出色的文件与打印服务器,同时也是出色的应用服务器。它也被相当广泛地选为Internet的平台(作为Internet Server),使Windows 95的客户机可以方便地访问Internet,并且Windows NT Server还可以作为强大的通讯服务器,出色地发挥Windows95内置的远程通讯功能。
　　作为Windows家族的成员之一,Windows NT Server被设计成可以最好地发挥Windows 95 优势的网络操作系统。Windows NT Server是一个健壮的(robust)多用途的网络操作系统,它不仅提供了高性能的文件与打印服务功能,还提供了支持Client/Server应用的强大的基础结构。
　　MICROSOFT是唯一可以提供一整套被设计与调整成专门支持WINDOWS网络方案的客户端软件与服务器软件的一家公司。Windows网络方案的主干是Windows NT Server,它是Windows 95客户机的最佳服务器。
　　·Windows NT Server与Windows 95被设计成一起协同工作。
　　Windows NT Server与Windows 95的结合,意味着简便的安装、配置与操作,也意味着从最终用户到网络管理员的全面一致的集成。
　　·一个服务器满足你全部的网络需要,并提供资源与工具来帮你支持与维护网络。
　　Windows NT Server RAS给MS－DOS客户、Windows客户和任何PPP客户提供远程访问服务。它还支持基于TCP/IP、IPX/SPX和NetBEUI LAN等局域网的互操作性。
　　·Windows NT Server是一个可靠性很高的操作系统。
　　Windows NT Server是一个稳定的具有高可靠性的操作系统。它具有微内核体系和软件模块，彼此通过安全接口连接。Windows NT Server的错误条件由结构化例外处理来捕获，它是一种操作系统或处理器处理例外的方法，不寻常事件由例外处理代码进行处理。通过结构化的例外处理，表现失常的应用程序不会影响系统发挥功能。PC机所必需的硬盘引入了一种在别的网际设备中从未有过的构成因素。文件系统(NTFS)可以从各种类型磁盘错误(包括使磁盘严重受损的错误)中恢复过来。NTFS使用冗余存储和基于传递的机制存储数据，在鲜见的系统崩溃时确保数据可以恢复。
　　·Windows NT Server提供安全的远程访问服务和路由。
　　Windows NT Server RAS支持基于域的安全，与登录进Windows NT Server系统分离的远程访问验证，密码加密和验证过程加密，回呼安全性，支持中介安全宿主机，审计，受限网络访问和数据加密。
　　Eicon用于Windows NT Server的广域网服务包括先进的IP过滤，由管理员决定哪些包可以在广域网上传输。管理员可以严格配置过滤器，截下或传送与过滤条件相匹配的包。
　　·MICROSOFT提供一组被设计与优化成协同工作的集成的应用（OFFICE ),Windows NT Server实际上可以增强OFFICE 95的能力与功能。
　　总之,Windows NT Server 与Windows 95的结合,将为你节省用于业务与网络管理的时间与金钱。

　　（上海　张元凯）
