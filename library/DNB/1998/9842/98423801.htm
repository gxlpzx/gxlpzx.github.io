ULEAD家族新成员--particle

    相信大家对ULEAD决不会陌生，它在图像处理等方面的成就有目共睹。在此，为大家介绍ULEAD家族的滤镜新成员：Particle.Plugin。Particle.Plugin是一个图像效果处理的好工具，它通过模拟自然界的云、雨、烟等现象来渲染画面，营造烘托画面气氛。下面以几幅Tatanic的剧照作为范例，通过Bubble效果详细介绍Particle.Plugin的功能和用法。
    在图一的对话窗我们看到预览窗口中的Bubble效果， 你可以在窗口中用鼠标直接移动气泡，并可选择单个气泡进行效果调节。（Ctrl/Shift键加鼠标单击可多选）
    在预览窗下方是效果选择窗，可以选择不同的粒子效果，如气泡、光晕、雨、雪、云雾等。在Effect control域内，可调参数为：
    Density（密度）：调节气泡的数量。
    Size（大小）：调节气泡的尺寸。
    Variance（差异）：控制气泡大小的差异程度，数值较小时，气泡几乎一般大小，着数值的增加，气泡间出现差异，数值越大，差异越明显。
    Reflection（反射度）：调节气泡反射光线的强度。
    在Particle parameters域，
    Ambience（环境）：控制环境光线，单击前面的小方窗可调节光线色彩。
    Border（边界）：设置边界的色彩。
    Body（主体）：设置主体的色彩。
    Spot（光点）：设置气泡高亮部分的强度。
    Direction（方向）：控制气泡反射的角度。
    Elevation（仰角）；调节Z轴反射光线的高度。
    在窗口右下角，可进行以下调节：
    Wireframe：显示模式，使预览窗口中的气泡只显示为方框，便于观察气泡的位置。
    Select All：全选，为气泡整体调节设置。
    Wrap：环绕，当气泡位于图像边缘时使不可见的部分显示于图像相反方向。
    Order：顺序，调节气泡间的前后顺序，包括调到下（上）一层，调到最底（顶）层。
    除了以上的可调节参数外，ULEAD还为你提供了省时的随机调节Random按钮，可以重新设定位置、大小等参数综合调节后的效果。作为开放式的外挂插件，你还可以存储认为满意的效果，单击Add按钮即可，ULEAD会在其滤镜目录下将其存储为＊.pst文件，当然ULEAD还为你预先存储了许多已设计好的效果图供你选择，逐个欣赏一下吧。(见图二)
    下面重点介绍几个比较典型的效果，
一、Firefly效果（图三），其特有参数控制如下：
    Opacity：透明度，控制Firefly的中心透明度，色彩可调。
    Softness：柔和度，控制Firefly的边缘模糊程度，色彩可调。
二、Rain效果（图四），调节参数为：
    Density：密度，调节雨滴的数量。
    Length：长度，决定雨滴的长度。
    Width：宽度，设置雨滴的宽度。
    Variance：差异，控制雨滴间方向的差异。
    Body：主体，决定雨滴的颜色和在图像中的明暗度。
    Opacity：透明度，雨中图像的透明度。
    Bkgrd blur: 背景图的模糊度，为增加真实感可用此项控制图像在雨中的清晰程度。
三、Snow效果：(图五)对于Snow的调节参数较Rain相对简单些，只需调节Density、Variance、Size和Body、Opacity等，但效果仍然生动逼真，与Rain相同的是不能对单个粒子进行调节。
四、Cloud效果(图六)，其参数与 Bubble和Firefly类似，即可对单个Cloud进行调节，具有Wrap可选项。其特有参数为：
    X scale：控制水平轴向的平滑度，较低的数值会破坏图像的效果。
    Y scale：控制垂直轴向的平滑度。
    Frequency：频率，数值低时可产生大团云朵的效果。
    与云参数相同的还有雾、火效果。
五、 Star效果(图七)，其特有参数为：
    Star spikes ：调节星星射出光线的数量。
    Sun size ：调节星星中心部分的大小，颜色可调。
    Halo size ：调节光晕的大小及颜色。
    Star size ：调节放射光线的尺寸、颜色。
    Star ratio ：决定同一个星星的放射光线间相对大小。
    Direction ：调节光线的方向。
    Opacity ：控制星星的明亮度。
