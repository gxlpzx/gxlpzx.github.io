利用Windows for Workgroups3.11和Windows95
组建办公网

　　编者按：自从本报在第38期(1998年10月5日)的试刊“OA专刊”办公网络版中刊出“四世同堂办公网”一文后，许多读者纷纷来信、来电，对合理利用各种档次的计算机表示了浓厚的兴趣，并提出了许多新奇、独到的见解。本期特选登一篇利用Windows for Workgroups3.11和Windows95组建经济型多功能办公局域网的文章，该文与利用Netware 3.12和Windows95组建办公网的“四世同堂办公网”一文在功能上互为补充、各有特色，希望能对广大读者有所裨益。

　　目前许多单位微机的档次和性能参差不齐，多数由386、486和Pentium组成。怎样将这些机器全部利用起来，不使一些“过时”的机器被无情地“抛弃”掉？最好的方案是建立一个网络，实现资源的共享。本文介绍如何利用Windows for Workgroups3.11和Windows95组建一个经济型多功能的办公局域网。
　　一、网络的规划和硬件的安装
　　网络选用对等式（Peer－to－Peer）结构，网卡采用普通ISA总线的16位NE2000以太网兼容卡，此类网卡比较便宜（市面上仅售60元左右）。网络连线使用细同轴电缆线，每段同轴电缆线的两端装有BNC连接头，相邻两段同轴电缆以专用的T型头连接，T型头的一端接入网卡的BNC头，最后还需在整个网段的两端各装上一个50欧姆的终端电阻。
　　此网组建简单、成本低，并易于管理，不需要专用的网络服务器。不但每台机器（包括低档的386机型）上的资源可与其他机器共享，而且每个用户可以共享网络上的高质量打印机和CD－ROM。需要注意的是：386和较早的486机型不具备即插即用（PnP）功能，因而在这些机器上安装网卡前一定要进行中断IRQ和I/O地址的设置，以防止与其他设备发生冲突。设置工作可由网卡驱动程序盘中附带的Setup程序来完成。如果你的机器具有PnP功能，那么目前市面上大量出售的无跳线网卡是不必进行上述设置的。
　　二、软件的安装
　　安装和运行Windows95时，计算机至少应具有8M RAM、80M硬盘空间；而Windows for Workgroups3.11是基于Windows点对点间的网络操作系统，与Windows3.x相比，它增加了文件及打印共享、电子邮件、事件调度等网络功能。并且Windows for Workgroups3.11对硬件的要求也低，只要是具有4M RAM、20M硬盘空间的386以上的兼容机就可顺利地安装并运行。为此，可在较高配置的486机和Pentium机上安装Windows95，其他低性能的机器全部安装Windows for Workgroups3.11。
　　1.Windows95下网卡的安装和设置
　　当按照正常方法安装完成中文Windows95后，需进行以下三项工作：
　　（1）安装网卡驱动程序　进入“控制面板”，选择“网络”选项，单击“添加”后再选择“适配器”，并点击旁边的“添加”按钮，出现“选定网络适配器”的窗口，这时如果你十分清楚网卡的生产商及类型，便可在窗口的“厂商”下方列表中选择网卡的生产厂商，并在“网络适配器”下方列表中选择网卡对应的型号；如果你不能确定网卡的生产厂家和类型或列表中没有与你的网卡相对应的选项时，可选择“从磁盘安装”，系统要求你插入网卡的驱动程序盘，所需的网卡信息全部位于网卡附带驱动程序盘的Windows95目录下。之后按“确定”，返回“选定网络组件类型”窗口；
　　（2）选择网络协议　接着选取“协议”并单击“添加”，出现“选定网络协议”的窗口，选择左边“厂商”下的“Microsoft”后，再选择右边“网络协议”下方的“NETBEUI”选项（当你不确定时也可以全选，其他多选的协议并不会影响系统正常的工作），按“确定”返回“网络”窗口；
　　（3）标识计算机　在“网络”窗口中选择“标识”选项，在“计算机名称”后输入在网络中识别这台计算机的名字，在“工作组”后输入该机所属于的工作组名（可输入Workgroup，因为Windows for Workgroups3.11在安装中自动生成的工作组名为Workgroup），如需要也可在“计算机描述”后输入描述本台计算机的说明。
　　2.安装并设置Windows for Workgroups3.11
　　（1）安装Windows for Workgroups3.11　安装Windows for Workgroups3.11之前，必须先安装Windows3.x或在WW3.10上升级，且安装中文Windows for Workgroups3.11时原系统必须为中文版本，安装英文Windows for Workgroups3.11时原系统也必须为英文版本。本文是在先装有英文Windows3.1的基础上安装Windows for Workgroups 3.11。具体的安装步骤和过程与单独安装Windows3.x时基本相同，一切按系统默认方式进行，安装完成并启动进入后，你会发现Windows for Workgroups3.11的界面与Windows3.x的相同，只是多了一个名为网络“NetWork”的程序组。Windows for Workgroups3.11网络功能的实现就是靠这个程序组来完成的。为了方便操作和电子邮件等信息的发送及接收，在Windows for Workgroups3.11上安装了Richwin4.21汉字系统，每次启动Windows for Workgroups3.11后再运行Richwin4.21，同时它还可以将部分英文提示汉化。
　　（2）设置Windows for Workgroups3.11的网络功能　打开“NetWork”程序组，双击程序组下的“Network Setup”图标，出现如图1所示的“Network Setup”窗口。先选择“Network”选项，在新出现的窗口中选择安装微软网络“Install Microsoft Windows Network”一项，同时选择下方的“No additional network”一项，按“确定”返回到图1的窗口。再选择“Sharing”功能项，将“I want to be able to give others access to my files”和“I want to be able to allow other to print to my printer(s)”两项全部选取，这样这台机器上的文件和打印机就可以被网上的其他用户共享。按“确定”再返回窗口2。最后选择窗口下的“驱动程序”项，在弹出的窗口下选取“Add Adapter”安装网卡（最好选择从磁盘安装），并设置网卡的I/O和IRQ参数。当这两个参数有错时，系统会自动提示你进行改正，至无冲突后按“确定”返回上层菜单；再选择“Add Protocol”安装网络协议，在弹出窗口中选择“MS NETBEUI”协议，连续按三次“确定”后，安装结束。重新启动计算机后，系统要求输入登录名和口令，并对口令进行确认。当登录成功后就可以进行共享资源的设置了。
　　三、设置网络资源的共享
　　当上面工作结束后，重新启动计算机并进行登录，对机器上的有关资源进行共享功能的设置。
　　1.Windows95下资源共享的设置
　　在Windows95下设置资源共享是非常容易的。先打开“Windows资源管理器”，用鼠标点击需要共享的文件夹或设备名，按鼠标右键，选择“共享”。在弹出窗口的“共享”一栏下选取“共享为”一项，并输入共享名。接着点击“添加”功能项，添加可访问该资源的网上用户名，同时给每一用户提供了只读、完全访问和自定义三种访问方式，根据需要可选择一到三项。如果选了“自定义”访问方式，则可以设置密码，网上用户在访问这些资源时要输入密码才能进行。
　　2.设置Windows for Workgroups3.11资源的共享
　　选择“主群组”下的“文件管理器”，并选择需要被共享的资源，如目录、整个硬盘或CD－ROM等。假如选择了整个硬盘C之后，可点击“磁盘”菜单下的共享为“Share As”选项，在出现的“Share Directory”窗口下，输入有关的共享名，以后网络上的计算机就可以看到这个名字并共享其资源，当然也提供了三种不同的可供访问的方式（和Windows95相似）。按“确定”设置结束。同时也可将这台机器上的打印机与网络用户共享，方法为：选择“主群组”下的打印管理图标“Print Manager”，在弹出窗口中选择“打印机”菜单并点取“Share Printer As”，在又弹出窗口的“Share As”处输入打印机的共享名，如需要也可在“Password”后输入口令，让使用者根据口令确认能否使用打印机。按“确定”后，本地打印机就被设置成网络打印了。
　　四、访问网上其他计算机用户的资源
　　当装有Windows95的用户访问网上其他用户的资源时，可双击屏幕上的“网上邻居”图标，凡是与该机器相连的所有网上的用户名将全部呈现出来。选择你需访问的用户名后，将显示该用户开放给你的目录或设备名，这时你便可以像使用本地文件或设备一样来使用这些网上资源了。在Windows for Workgroups3.11的机器上访问网上资源时，先打开“文件管理器”，选择“磁盘”菜单下的“连接网络驱动器”，在弹出窗口的“驱动器”后任选一个逻辑驱动器名（系统默认为E），在“路径”后输入要访问的用户所在网上的路径（如：\\WQ），这时所选用户（WQ）上的共享目录和设备名将全部显示到“Shared Directories”列表下。同样可以像使用本地文件或设备一样来使用这些资源了。
