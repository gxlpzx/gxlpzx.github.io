“网上高手速成完全手册”答疑

　　拙作“网上高手速成完全手册”在《电脑报》(第29期20、21版)发表后，很多朋友来信询问各种问题，现就来信中的典型问题做一解答。
　　一、关于注册表的修改
　　1.Modem的所在COM口速率如何设为921600？
　　在桌面按住“我的电脑”→“属性”→“设备管理器”→“端口(Com ＆Lpt)”，选择你的Modem所在的端口，如COM2，选择端口属性，将端口设置中的波特率改为最大值921600，流量控制选择“硬件”。在高级属性中将接收缓冲区及传输缓冲区设到最大。
　　2.如何修改maxsockets键值？
　　点击“开始”→“运行”→“regedit.exe”(回车)，打开注册表编辑窗口寻找：
　　\HKEY_LOCALMACHINE\System\CurrentControlSet\Services\VxD\NWLink\Ndi\params\maxsockets\max，将内容改为1020。
　　以上的修改要在机器重新启动后才能生效。上述修改方式仅供各位参考，所以改过后，不一定能正确执行,速度也不一定能提升多少,完全因Modem而异。若改后出现问题，请改回来(注意记下修改前的值)。
　　其他键值的修改，可以使用MTUSpeed这个软件完成。MTUSpeed使用非常简单，不再多述。

　　四、代理服务器的设置问题
　　代理服务器(Proxy)由于很多是非公开的，一旦公开很可能遭到封闭！所以连接不上属于正常现象。并且有些代理服务器只对指定范围内的IP服务。
　　1.Netscape Navigator下的设置（以4.0以上版本为例）：
　　选择Edit下的Preferences,双击左边的Category栏中的Advanced项，在其下选择 Proxies ,右边即出现Proxy的设置栏，有以下三个选项：
　　Direct Connection to the Internet（直接连通Internet）
　　Manual Proxy Configuration （手动设置Proxy）
　　Automatic Proxy Configuration （自动设置 Proxy）
　　选择中间的 Manual Proxy Configuration ，同时按旁边的View 按钮，即出现一个 Mannual Proxy Configuration的 窗口，在HTTP一栏中填上 198.161.102.27 ，在旁边的Port 填上 8080 ，按 OK ，即完成设置。
　　此后，输入网址连接一个站点时，会首先连接Proxy中转站然后再连至目的站点，由于代理服务器通常缓存了大量站点的内容资料，因而速度有一定提高，当然，各个国家地区的速度会有所不同，可选择不同的Proxy进行连接。如果你想取消使用Proxy，请选择 Direct Eonnection to Internet即可。
　　2.IE4.0下的设置（以中文版为例）：
　　选择查看下的 Internet选项 ，在上边选连接 ，在下面出现的窗口中选择中间的通过代理服务器访问Internet ，再选取高级选项，在 HTTP 类型那行将代理服务器地址一栏填上198.161.102.27 ，端口 填上 8080 , 按 确定 即完成设置。
　　如果你想取消使用Proxy，请取消选择通过代理服务器访问Internet 即可
