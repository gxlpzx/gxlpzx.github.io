专家坐堂

　　河北网友阿朋问：我从拥有电脑至上网，不过数月，面对问题常常不知所措，这不，又遇到问题了：1.前几天我收到了一封莫名其妙的Email，主题是：delete! don't open me....warned ，内容是:please ignore this randomly penerated error message! delete it as soon as possible! 2.这几天我的电脑出毛病了，使用光驱时老是显示“设备没准备好”，但重启时光驱又可以用，不过用一些软件，如《弹指神功》、《东方快车》等时常会莫明其妙地死机。这些问题是不是因为那封Email带病毒引起的？
　　答：1.上网收到莫名其妙的Email是常事（一些无聊者的恶作剧），删掉就是了，只要不打开随信的附件（可执行文件或WORD文件），就不会“中毒”，看看信的正文无妨。
　　2.用光驱时老是显示“设备没准备好”，说明光驱读盘能力下降，可多换几张光盘试试，如问题依旧，可清洗一下激光头(具体请见本报1998年第6期第36版的相关文章)。运行某些软件死机，可采取如下步骤处理：
　　1排除系统软件方面的问题：
　　·用“干净的软盘”启动微机，然后用最新版的杀毒软件杀毒
　　·运行磁盘扫描程序修正硬盘软错误
　　·删除所有类似＊.TMP 等临时文件
　　·用 REGCLEAN4.1 修正注册表。
　　2排除应用软件本身的问题：
　　·完全卸载该应用软件后重新安装
　　·若不行，则完全卸载该应用软件后，安装更新版本的该应用软件。
　　3重新安装Windows 95。
　　广西钦州罗宇峰等朋友问：我是《电脑报》的一位读者,现有一个问题想请教：我的电脑最近发现有宏病毒,但我在电脑中从未装过盗版软件,也从未拷过有问题的程序，怎么办?我想问一问，上网浏览信息会不会染上病毒(未下载过程序)?Cache目录下的HTM文件和JPG文件会不会有毒?万望赐教!
　　答：宏病毒是WORD文件带来的，其实宏病毒并不可怕，只要装上WORD97（OFFICE97）就能有效遏制。打开文件时，如有宏的警告，一定要选择“取消宏”，再厉害的宏病毒不运行也无害。用WORD97还能杀宏病毒：先用WORD97建新文件，再用“取消宏”方式打开“有毒文件”，用“复制/粘贴”将“有毒文件”内容拷到新文件，关闭“有毒文件”然后删掉它，另存新文件就得到了“无毒文件”。
　　上网浏览信息不会染上病毒，HTM文件和JPG文件也不会有毒，不要谈毒色变嘛!
　　广东阿猫等多位网友问：如何使局域网中数台微机通过一条电话线上Internet?
　　答：可用如下方案解决：一台装有Windows 95的微机充当代理服务器，用MODEM上Internet，局域网上的其它微机作为工作站(装Windows 95)，通过代理服务器连通Internet。几点重要提示如下：
　　1代理服务器安装WinGate 2X for Windows 95，正确填上ISP相应的Server(服务器)名，Adminstrator(管理员)的口令最好不填。
　　2设定代理服务器的TCP/IP：在“控制面板/网络”中，找到TCP/IP协议项，点击“属性”，按如下进行设置：
　　[IP地址]
　　IP地址：选择“指定IP地址”，填上自定的IP地址值，如：1001001001(不能与局域网上的工作站重复)
　　子网掩码：2552552550
　　[WINS配置]
　　选择“禁用WINS解析”
　　[DNS配置]
　　选择“启用DNS”，然后在“主机”中填入电脑名称，如：“NETBOY”，“域”中填入你的ISP的域名，DNS服务器搜索顺序中填入你的ISP的域名服务器的IP地址，填入后按“添加”。“域后缀搜索顺序”项等可不填。设完之后重新开机即可!
　　3工作站的安装与设定
　　工作站上不需要安装WinGate，也不需安装拨号适配器，只需设定TCP/IP协议即可。在“控制面板/网络”中，找到TCP/IP项，点击“属性”按钮，进行如下设置：
　　[IP地址]
　　IP地址：请自定一个，如：1001001002(不能与局域网上的其它工作站重复)
　　子网掩码：2552552550
　　[WINS配置]
　　选择“禁用WINS解析”
　　[DNS配置]
　　选择“启用DNS”，然后在“主机”中填入代理服务器的名称，如：“NETBOY”，“域”可不填。“DNS服务器搜索顺序”中填入代理服务器的IP地址，如1001001001，填入后按“添加”。“域后缀搜索顺序”等可不填，设完后选“确定”。
　　重新开机后，在Windows 95的目录下将hosts.sam拷成hosts文件(无扩展名)，加入一行：1001001001 gateway，后面多按几个回车键。
　　4代理服务器上Internet的软件设定
　　Netscape：
　　选“Options/Preferences/Proxies”，填入：
　　HTTP PROXY:gateway PORT:80
　　Microsoft IE4:
　　选“查看/Internet选项/连接”，“通过代理服务器访问Internet”中的地址填gateway，端口填80。
　　完成后，就可实现多台微机通过一个账号一个MODEM上Internet了。
