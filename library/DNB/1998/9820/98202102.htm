NT Server的特点

　　Windows NT Server是Microsoft公司1994年推出的新一代网络操作系统，它具有以下特点：
　　1.简单易用：NT Server采用图形界面，直观明了。
　　2.应用面广：NT Server是为高性能机器设计的，但同时也适用于486等低性能的机器。
　　3.安全性高：尽管NT Server 1995年才获得C－2认证，但它从最开始就以高安全性为设计目标之一。
　　4.互操作性强：NT Server具有内置连接Netware的功能，并支持多种传送协议，因此能与装有其它网络操作系统的网络实现互访。
　　5.采用集中式控制：NT Server以域（domain）的形式对计算机组进行集中控制。
　　6.支持长文件名：NT Server支持大小写敏感的长文件名，这增加了服务器的安全性。
　　7.具有日志记录功能：NT Server有3种日志记录形式，它们记录了登录企图、系统事件等，对增强系统安全性和查找故障原因有很重要的作用。
　　8.内置远程访问服务（RAS）功能。
