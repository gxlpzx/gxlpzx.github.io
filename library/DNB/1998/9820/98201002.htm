让你的桌面更富有生气

　　由MOON VALLEY SOFTWARE公司推出的这个名为Aptive Desktop Customization（以下简称ADC）的软件是一个很有趣的桌面布景修改工具，能修改一些WIN95不允许用户修改的地方，和大名鼎鼎的WINHACKER一样，能使你的WIN95看起来更加悦目，用起来更加顺手。
　　好了，闲话少说，还是让我来介绍一下ADC的主要功能吧。ADC的主界面中一共有两个主要的菜单，Features和Start Menu Banner，即功能和开始菜单标签。Features菜单中的选项如图1所示，现在我来为大家讲讲各个选项的作用。
　　＊ Turn On Bubble Help　它能允许你为WIN95中的每一个图标编制一个说明文档，当你将鼠标箭头移至该图标上时，说明文档便会显示出来。右边的Choose按钮可以选择四种显示风格，分别为Rectangle（方形）、Ellipse（椭圆形）、Rounded（圆形）、Rick Rack（锯齿形）；而Edit Text按钮则可以让你选择图标并编辑文档。
　　＊ Highlight Icons On Flyover　它可以在鼠标箭头移至一图标上时显示出特殊效果。右边的Choose按钮有五个选项，分别为Raised（突出效果）、Sunken（凹进效果）、Etched（蚀刻方框）、Bump、（浮雕方框）、Inverse Frame（反白方框）。
　　＊ Turn On Visual Launch Effects　它可以使你在双击一个图标执行它的时候，该图标产生动态效果。右边的Choose按钮有四个选择，分别为Pivot（纵轴翻转）、Flip（横轴翻转）、Squash（纵向压缩效果）、Move up（向上移动）。
　　＊ Animate Icons　动画图标，右边的Choose中你可以选择桌面上的任一个图标，并采用一个动画图标来代替它，但只限于桌面上的图标。此外ADC还提供了许多种类的动画图标文件。
　　＊ Animate Windows on Minimize/Restore　它可以设置最小化和还原一个窗口时是否采用动态效果。
　　＊ Show Shortcut Overlays　它可以允许你将捷径图标上的那个小箭头去掉。
　　＊ Place Aptive Desktop System Tray　你可以通过它选择是否将ADC的图标放在WIN95的任务栏上。
　　另一个菜单Start Menu Banner的模样如图2，其主要功能是设置任务栏上开始菜单上的动画的。
　　首先将Activate Aptiva Start Menu Replacement选框打上勾，然后setting框中有Movement、Selected Banner、Speed三个项目。在Movement中可选None（静止）、Horizontal（水平移动）、Vertical（垂直移动）、Random（随机方向）四种方案。在Selected Banner中又可选择CANE、GRAFFITI、Ibmblue、Ibmyellow、MEMPHIS、PHOENIX六种旗帜样式。在Speed中可选择移动速度。
　　Aptive Desktop Customization简短易用，活泼有趣，可以使你沉闷的WIN95变得生机勃勃。有兴趣的朋友可以去这个网址下载： http://www.moonvalley.com/specials.htm 。 (广东  李雳)
