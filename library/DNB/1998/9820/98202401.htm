登录BBS一次成功

　　初学BBS的人，容易迷失在各种软件的复杂配置里。再加上初学者在BBS站台上的级别又较低，上站时间有限，给人的感觉常常是刚上站就提示你时间用完了。我以前刚上BBS站时，简直像一只没头苍蝇，到处瞎碰乱撞，自己冤枉花了不少的电话费和时间不说，还不时惹得站长给我一封警告性的电子邮件，比如：你的载波中断了，你的信投错信区了……经过长时间的摸索，我才弄明白点门道。现在，我每天上站5～10分钟(按我现在的级别，每天我可以上站30分钟)就可以获取大量的信息了，既省下了话费，又可以省下时间让更多人上站。
　　结合我自己的体会，在这里，我端出一个BBS“套餐”，让想上BBS站的人“享用”，算是对大家的帮助吧。你只需要根据你的机器配置，跟着我一步步改变软件的设置，你也可以一次成功。
　　1.安装调制解调器
　　首先你必须安装好调制解调器(Modem)，这在许多报刊上已经有介绍了，大家可以参看。要测试Modem是否正常，最简单的方法就是在Windows95里用“附件”里的“电话拨号程序”拨打一个电话，如果成功了，就可以证明Modem及连线基本正常。
　　2.建子目录
　　在C：根目录下建一个子目录BBSTC(当然也可以建在其他驱动器上)，然后在BBSTC下建立以下子目录。
　　C：\BBSTC\
　　　　　　TWAY
　　　　　　TELIX
　　　　　　BWAVE
　　　　　　PACK
　　　　　　DL
　　　　　　UL
　　3.具体步骤
　　(1)TWAY用于存放汉字系统，推荐用天汇V3.1袖珍版(电脑报专用版)，理由是它小巧且功能不差，同时它专门针对Windows95作了优化。TWAY既可以在一般DOS运行，也可以在Windows95的DOS窗口中运行，并且不花屏，不出现乱码。安装方法是，把《电脑报配套光盘第一辑》放入光驱，按提示运行安装程序(在光盘hzsoft\tway31m目录下)，把程序指定安装到C:\BBSTC\TWAY目录中，程序会自动把路径添加到AUTOEXEC.BAT文件里。刚开始，只有区位码和天汇ABC两种输入法，你可以借助天汇提供的程序挂接金山五笔输入法。执行TWAY.BAT，进入汉字系统。
　　(2)TELIX用于存放联机通讯软件，推荐使用TELIX V3.21汉化版，这是一个共享软件，很多地方都可以找到，也可从许多BBS站上下载。一般是压缩文件，需要用解压缩软件将它解压到TELIX下，接着，键入TELIX，执行联机通讯软件。TELIX的参数设置比较复杂，除了以下的参数必须设置外，其他的参数大可不必管它，用它的默认设置好了。记住，任何时候都可以按“Alt＋Z”获取帮助。
　　A.设置BBS电话号码。按Alt＋D进入“拨号目录选单”；按“a”，光标向下移动到最后一个空行；按“e”，输入你想上的BBS的站名和电话号码，其他则不改变。按“Esc”退出拨号目录选单。
　　B.设置通讯参数。按Alt＋P进入“通讯参数设置单”，窗口顶上显示“现在状态：X，N，8，1，Y”，其中X表示你的Modem速率，可设置为你的Modem速率的四倍，比如14.4K的Modem可选速度57600。Y表示Modem使用的串口号，根据你的实际选择。其他参数不更改。按回车退出。
　　C.参数设置。按Alt＋O进入“系统设定”窗口，里面有许多选项，但这里只需改动几个就行了。按“T”进行终端模式设定，选ANSI－BBS类型；按回车回到刚才的系统设定窗口；按“F”设定文件名称和路径，在下传文件存放的路径下填入C:\BBSTC\DL，在上传文件存放的路径下填入C:\BBSTC\UL，回车退出；按“W”存盘保存设置。
　　D.退出TELIX。按Alt＋X，再回车就可以退出了。
　　其实，现在你已经可以上BBS了。不过，如果这样就出发的话，你的钱包恐怕吃不消了。如果在联机状态读信和写信，那也太浪费了。幸好，有蓝波快信帮忙，有了它，我们可以把自己喜欢的资料和信件下载到自己的机器中，然后挂断电话，从容地阅读和写回信，然后在下次上站时再把回信上传到BBS站台。
　　(3)BWAVE存放蓝波快信阅读系统。把得到的蓝波软件解压到BWAVE下，先执行BW220KEY.COM，输入你的ID(即你在BBS上的“名字”，一般用你的姓名的汉语拼音组成，至少两个字，中间用空格隔开，如“Wen Jun”。用笔记下你的ID和DOS版的注册号。
　　用EDIT等文本编辑软件编辑一个如下表所示的文件，文件名为WJ.SIG，保存在C:\BBSTC\BWAVE下，不超过三行。一般包括你的姓名、ID地址、电话号码、电子邮箱号码(如果有的话)、一句你喜欢的话等，当然这些不一定都得有，随你所好吧，这时你可以充分发挥想象力，因为这就是你在BBS网络上的“名片”，它会自动附加到你给别人的每封信后边。
　　
　温　钧　ID:Wen Jun
　四川资中TEL:(0832)5529075
　让网络连接你、我、他!
　　
　　再用EDIT编辑名为TELIX.BAT的批处理文件存放在BWAVE下，目的是要从蓝波中方便地执行通讯程序TELIX，文件内容包括以下两行：
　　cd c:\bbstc\telix
　　telix
　　现在，我们键入BWAVE进入蓝波主菜单，光标下移到“Setup／Configure Mail Reader”按回车进入蓝波设置窗口。
　　A.在Default Preferences上回车，选File and String Preferences回车，再选Signature File Name，输入C:\BBSTC\BWAVE\WJ.SIG，按Esc退回设置窗口。
　　B.光标横移到Archivers回车，在相应的位置填入压缩软件程序所在的路径，比如，ZIP压缩软件处填入C:\BBSTC\PACK\PKZIP.EXE －ex @F @I然后回车，也可按F2设置默认路径，但默认路径也许不符合当前压缩软件的真实路径。其他各行类推。
　　C.光标横移到External回车，在Message Editor Cmd行输入C:\DOS\EDIT.COM @F;在Terminal／CommCmd行输入C:\BBSTC\BWAVE\TELIX.BAT；在Editor／Browser Cmd行输入C:\DOS\EDIT.COM @F。这三行主要是用来设定回信用的编辑程序、联机用的通讯软件等。这里编辑程序用的是DOS自带的EDIT，当然你也可以用你喜欢的其他编辑软件。
　　D.光标横移到Directories回车，分别在六行上输入：
　　C:\BBSTC\DL
　　C:\BBSTC\UL
　　C:\BBSTC\BWAVE\WORK\
　　C:\BBSTC\BWAVE\REPLY\
　　C:\BBSTC\BWAVE\DATA\
　　C:\BBSTC\BWAVE\
　　这六行分别用来设定下载文件存放路径、上传文件存放路径、BWAVE工作路径、回信文本文件存放路径、数据存放路径、临时交换文件存放路径。
　　E.光标横移到Registration回车，这是注册BWAVE，分别输入刚才记下的ID和注册号。
　　(4)PACK把常用压缩解压缩软件拷贝到这个目录中。
　　(5)DL以后进入BBS，下载信件等会自动存放在这个目录中。
　　(6)UL想上传到BBS的信件，蓝波系统也会自动存放在这里。
　　(7)为了方便进入BBS，使用蓝波软件，在C:\BBS\下建立一个批处理文件BBSTC.BAT，它的内容如下：
　　@ECHO OFF
　　CD\BBSTC\TWAY
　　　CALL TWAY
　　CD\BBSTC\BWAVE
　　　BWAVE －T －1
　　CD\BBSTC\TWAY
　　　QUIT
　　CD\BBSTC\
　　　CLS
　　　DIR
　　@ECHO ON
　　在AUTOEXEC.BAT文件加入：
　　PATH=％PATH％;C:\BBSTC;C:\BBSTC\TELIX;C:\BBSTC\BWAVE
　　4.重新启动电脑
　　进入BBSTC目录，键入BBSTC，回车，屏幕上出现“OK”后，按ALT＋D，把光标移到你前面输入的BBS站名那一行，按“D”，你的电脑将试图向BBS站拨号，如果拨号成功，你又是刚上站的新手，BBS站台将要求你输入你的姓名、ID、生日、地址等，如实填写吧。现在你可以下载、上传信件和软件了。完成后按Alt＋x退出联机状态，系统自动回到BWAVE，你再慢慢读信吧，花五至十分钟下载的文件也许就够你读几小时的了。
　　祝登录BBS一次成功!

　　(四川　温钧)
