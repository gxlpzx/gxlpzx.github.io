在VB中设计倒计时程序

　　在程序设计时，经常需要用倒计时，倒计时程序一般不能用时间函数来进行设计，常用的方法是在计时器的触发过程中设置计数变量来进行控制，程序如下：
　　Dim hh,mm,ss
　　Sub Form_ Load()
　　hh=0:mm=5:ss=0
　　timer1.Interval=1000
　　End Sub
　　Sub Timer1_Timer()
　　s=IIf(ss<10,″0″＆ss,″″＆ss)
　　m=IIf(mm<10,″0″＆mm,″″＆mm)
　　h=IIf(hh<10,″0″＆hh,″″＆hh)
　　t=h＆″:″＆m＆″:″＆s
　　Me.Caption=t
　　ss=ss－1
　　If hh=0 And mm=0 And ss=－1 Then
　　msg=″练习时间已到，按回车键结束!″
　　MsgBox msg,48,″注意″
    End If
    If ss=－1 And mm>0 Then mm=mm－1:ss=59
    If mm=0 And hh>0 Then hh=hh－1:mm=59
    End Sub
　　程序说明：程序中用到三个窗体级变量hh、mm、ss，它们分别代表初始的小时、分钟、秒，可根据情况在程序中指定，也可配合文本框在程序开始运行时输入，程序中用到一个计时器，它每秒钟被触发一次，程序运行时在窗体的标题栏动态地显示倒计时，当计时完毕则显示对话框提示，同时结束程序的运行。(广东 李勇)
