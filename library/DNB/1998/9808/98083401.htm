专家坐堂

　　浙江读者周红华问：我的微机配置为：Pentium 133 CPU、华硕TVP4主板、昆腾21GB大脚硬盘、SONY CDU311光驱、View Top S3 765显示卡。微机原先工作正常，但最近一次开机之后，微机不能从硬盘启动，用软盘启动后发现硬盘丢失。仔细观察启动过程，发现硬盘指示灯常亮不灭，启动速度明显变慢，在微机自检表格Primary disk一栏中，原来的“LBA MODE 4 2112MB”变为“CHS MODE 0 2112MB”。检查CMOS的其它参数均正常。当天下午开机，故障自动消失。但是一周之后，又出现同样的故障。开机检查硬盘的连线，没有发现异常。请问这是什么原因造成的?此外，我的硬盘用NU80测试，得分仅为16，与贵报1997年第20期《硬盘速度大测试》中昆腾大脚硬盘的得分相差甚远，请问我的硬盘是否是“假冒”产品?
　　答：来信所说故障的疑点有二：一是硬盘的工作模式(应在CMOS设置环境中设定)自动地由原来的“LBA MODE 4 2112MB”变为“CHS MODE 0 2112MB”，此时硬盘丢失，而后来硬盘自动地恢复，此时故障也自动消失；二是一周之后故障又重复出现。如果能够排除人为修改CMOS参数的因素，故障原因可能在以下几点：
　　1病毒定时发作，或者某个应用程序修改了CMOS参数中的硬盘工作模式，造成重新启动时硬盘丢失。再联想到你的硬盘测试得分低，不能排除染毒的可能。
　　2而从启动过程中硬盘指示灯常亮不灭且启动速度明显变慢的情况来看，有可能是硬盘驱动器本身质量差、性能不稳定所致，因此硬盘有时能够正常工作，有时又不能正常工作，在选中硬盘而又不能正常读写时，就会出现硬盘指示灯常亮不灭的情形。为确定是否是此种原因造成的，可以采用交换法，把这个硬盘换到其它微机上一试。
　　3硬盘信号电缆部分引脚接触不良、电源电缆接地引脚接触不良，都可能造成无法访问硬盘的故障。可以更换电缆，或者将电缆接头插入主板上另外一个IDE插槽试一试。
　　4如果硬盘在本台微机上不能正常工作，而换到其它微机上可以正常工作，表示问题还可能与主板有关，例如接触不良、硬盘读写通道工作不稳定等，都可能造成上述故障。
　　由于对你的具体情况不是十分清楚，以上分析仅供你检查时参考。
　　河南读者何必武问：一台微机，配置为：Cyrix486 DX2－80 CPU、540MB硬盘、8MB内存、光驱、声卡、电影卡、144软驱。关闭电源一段时间后，重新启动时，屏幕无显示、喇叭无声，但可听到主机读硬盘的声音，此后死机，按键盘无反应，只能按RESET按钮复位，而重新热启动时的现象与冷启动时的现象完全一样。可是只要多按几次RESET按钮，微机就能正常启动，按RESET按钮的次数少则二三次，多则要十几次。正常启动后，微机一切正常，无任何异常现象。
　　答：首先打开机箱，拔去RESET按钮开关与主板的连线后重新启动微机，如果故障排除，则RESET开关可能有短路或漏电现象。
　　如果RESET开关正常，建议开启电源，等待30秒后再按RESET按钮，如果能启动，则可能是电源送往主板的POWER GOOD(电源好)信号延时时间过长造成的。因微机电源的输出端电压(＋5V和＋12V)的建立需要一定的时间，为使微机工作稳定可靠，开启电源后，微机电源的POWER GOOD信号线(＋5V和＋12V)在准备好后会延时01ms～05ms向微机主板输送一个幅度为＋5V的微机自检启动信号，在微机断电时，POWER GOOD信号提前01ms～02ms消失，从而通知主机以防止硬盘盘片被划伤。若该信号电平不能达到＋5V或不能及时达到＋5V均会导致你所描述的现象，请用万用表检查POWER GOOD信号线(标准颜色是桔红色，注意另有几根红色线是＋5V线)在开机时送出的信号电压是否正常(正常时电压值为＋5V)或是否能及时达到＋5V，如果此信号电压不正常，请断开电源主板的连线，然后再加电检查POWER GOOD信号电压(注意电源不能空载)，由此可判断是电源信号供给电路出现故障还是主板接收电路出现故障，POWER GOOD信号电缆中的电容漏电通常是造成此类故障的原因。
　　有时POWER GOOD信号电压虽能达到＋5V，但由于天长日久，延时电路的电容失效致使信号无延时或延时时间不够，也会导致微机不能正常启动，这种故障是微机的常见故障，须加大延时电路的电容，可在POWER GOOD信号线与地线之间加一只33μF的电解电容(注意极性)，通常可以解决问题。
　　如果还是不能解决问题，估计是某一设备的电解电容或钽电容漏电，须充电一定时间才能慢慢恢复。个别半导体芯片也可能出现故障，只有逐一取下板卡试试，如果到只剩下显示卡还未解决问题时，则通电后用手触摸板上各电容、芯片，看有无异常发热现象，如果电容或者芯片有异常发热现象，应当更换之。
　　北京读者肖美雁问：一台500W后备式UPS电源，在使用中时而发出“啪!啪!”轻响，显示器屏幕随之跳动，但UPS指示灯并未变化，此时用万用表测试，发现UPS输出电压在210V～250V之间变化，测量输入电压，一直保持在220V左右，未有明显变化，用电笔测输入插座极性为“左零右火”，未见异常。在UPS不接计算机等负荷时，故障依旧。先后更换三台不同型号的后备式UPS(两台山顿UPS－500C，一国产宝合PH－500M)，均有此现象，不知是UPS故障还是其它原因，如何解决?
　　答：在后备式UPS电源中，通常采用变压器分级调压稳压技术，当输入电压在180V～250V之间变化时，利用继电器改变变压器的抽头，可以使UPS的输出电压稳定在200V左右。你所听到的“啪!啪!”轻响很可能是继电器切换时发出的响声，属正常现象。虽然会引起显示器屏幕跳动，但对微机的正常工作无影响。
　　用万用表测试UPS输出电压，如果输出电压在UPS轻响时在210V～250V之间变化，且影响了微机正常工作，则需要找找原因。因先后更换了三台不同型号的后备式UPS，且在UPS不接计算机等负荷时故障依旧，所以故障源很大可能是来自外部，请考虑以下几个故障源：
　　1输入电源引线是否接触不良；
　　2是否有感性用电器带来的干扰，如复印机、空调机、电冰箱、电焊机，这些设备起动时会使电源电压瞬间降低，关闭时会使电源电压瞬间升高。尤其是UPS的负载不能有感性用电器，如日光灯；
　　3负离子发生器产生的负离子浓度过高，会诱发UPS电源或微机印刷电路板产生瞬间静电击穿，使UPS出现故障；
　　4化纤、纯羊毛地毯也是静电之源，尤其是在干燥的冬季，所以机房应使用防静电地毯。
