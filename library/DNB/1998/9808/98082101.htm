两种流行的网络操作系统——NetWare和Windows NT Server

　　在客户机/服务器型的网络操作系统(NOS)当中，目前最为普及的是Novell公司的NetWare;与之相抗衡的引人瞩目的产品是Microsoft公司的Windows NT Server。下面就这两种网络操作系统的特点，分别作一介绍。
　　一、NetWare
　　1.NetWare的最大特点是高速文件处理。它通过文件高速缓存和目录高速缓存，将读取频度高的数据，预先从硬盘中读到存储器(RAM)里。因此，要充分发挥NetWare的作用，服务器中必须备有容量足够大的RAM。
　　2.NetWare的第二个特点是灵活地共享打印机。要实现打印机共享，就需要有管理打印数据与打印机的打印服务软件。在NetWare中，可以将这种打印服务软件装入各种硬件(包括文件服务器)中，而且，可以同打印机、文件服务器、打印服务器、客户机等连接。
　　3.安全性方面，对入网的用户，进行注册登记，这样就可以对各个用户使用网络的时间及可以使用的资源等进行限制;另外，可以修改NetWare的文件 /目录属性的种类以起到保护作用。
　　4.在抗故障能力方面，使用SFT(系统差错容限)和TTS(事务跟踪系统)等保护系统与数据，并且装备UPS(不间断电源)和设置监视功能以保护服务器计算机。
　　5.NetWare有许多种类和许多版本，大家比较熟悉的有NetWare 3.X和NetWare 4.1。NetWare 4.1是Novell公司于1995年4月份发表的网络操作系统。NetWare 4.1采用网络目录服务(NDS)，可以让用户在不必知道打印机或文件位于何处(位于哪一个服务器中)的情况下，使用该打印机或者文件；另外，在NetWare 4.1中有文件自动压缩/解压功能，使得硬盘可以容纳更多的东西；安全性和抗故障能力方面，也有所提高，采用名为RSA公开密码的存取方式，以防止非法存取;采用双服务器方式，当一台服务器停机时，另一台可以替代进行处理作业；用户接口也有所改善，图形功能更为有效;安装作业所需的时间也大幅度缩短;同WAN(广域网络)的连接性能(这曾是NetWare 3.X的缺陷)也获得改善，性能有所提高。
　　二、Windows NT Server
　　目前，非常引人瞩目的操作系统是Microsoft公司的Windows NT Server，NT是New Technology(新技术)的缩写。Windows或者NetWare所实现的多任务，只是一种模拟的多任务;Windows NT所实现的都是真正的多任务。它具有以下特点：
　　1.Windows NT Server是最容易安装、使用和管理的网络操作系统，部分原因是因为它可以让你充分利用你已经拥有的关于基于Windows桌面的工作经验。Windows NT Server有着与你的基于Windows的PC同样的界面，它的设计是从底层向上与那些客户机协同工作并运行着同样的应用程序。这意味着你不需要花费大量的时间来培训自己和你的用户如何使用Windows NT Server,你对这个操作系统已经了解了许多。Windows NT Server非常易于安装。例如，它将自动检测你的硬件配置，消除耗时和令人困惑的配置。
　　2.NetWare是一个非常好的文件和打印服务器，但它没有设计来支持客户机－服务器应用程序的运行，虽然Novell开发了可装载模块（NLM）使NetWare可以运行像数据库和消息传送这样的应用程序，但操作系统本身缺乏一个强健的基础来可靠、高效地运行应用程序。一个运行于Windows NT Server上的错误的应用程序将不会让系统崩溃，这是因为应用程序被一个安全内存墙（Secure Memory Wall）与操作系统分离开来。
　　3.采用OS/2作基础，它消除了MS－DOS的640KB的存储容量制约，因而利用OS/2的HPFS(高性能文件系统)实现了对于多任务、虚拟存储、大容量硬盘的支持。
　　4.Windows NT Server帮助你方便地连上、访问Internet。Windows NT Server有内置的工具，其设计目的是让你更容易地在Internet中创建一个Web实例。Windows NT Server是唯一的将一个Internet服务器内置于基本产品中的网络操作系统。它还内置了一个浏览器，该工具将使你简单并有效地操作Web。
　　4.另外，Windows NT Server为你提供了一个出色的备份的功能。它不仅允许你把服务器中关键的商业数据备份到一个磁带驱动器或是类似的设备中，还允许你很容易地备份个人PC中的数据。这意味着如果一个雇员的PC崩溃或是被盗窃，你仍然可以恢复你的关键数据。
　　5.Windows NT Server的界面非常友好,使用全窗口化的界面,方便用户操作。

　　(重庆大学　雷玲)
