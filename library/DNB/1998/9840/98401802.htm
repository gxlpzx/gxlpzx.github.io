孩子们的“帆船”——Kids Web Browser

　　http://www.indiana.edu/~slizzard/mouse/home.html
　　最近笔者上网看到了一个界面简单、设计新颖的儿童浏览器，试用了之后发觉很不错。现介绍如下：
　　第一眼看到Kids Web Browser这个浏览器就深深爱上了它。它不像网景或IE那样技术化，整个界面轻松写意。颜色方面完全是七彩，所有的图像都是各种卡通老鼠，令孩子们一看就会喜欢上它。
　　这个浏览器的功能不是很多，但却已足够，包括Back、Forward、Reload、Help，以及Search等多个重要的功能。如“Search”（搜寻）也是完全以适合儿童的形式出现：画面是一只老鼠在地下挖泥土，好像正在寻找东西，一看就知道是搜寻工具。另外，配合了Yahoo的儿童搜寻版YahooLigans，使用起来更显童趣。笔者尝试过搜寻一些所要的简单资料，如节日来源等，都有很好的结果。该浏览器绝无不适合儿童的色情资料，是一个可以令小朋友用得开心的浏览器。
　　我认为Kids Web Browser是目前网络上最为简单也最具儿童色彩的浏览器，虽然在功能上远不如那些主流浏览器，但确是孩子们自己的网络“帆船”，驾驶他们，孩子们可以轻松自如地在网络的海洋里遨游。

　　(黑龙江　李传明)
