八仙过海各显神通---常用卸载软件综合评述

　　用过WIN95的人都知道尽管多数软件都提供了一个卸载的程序，但往往不能将所安装的的东西全部清除，需要手工清除的应用程序就更不用说了，往往是用不了多长时间，硬盘上就会垃圾为患。尤其对于需要跟踪了解最新软件或对最新软件很好奇的用户来说，硬盘空间总是频频告急。这时，拥有一个功能强大的卸载软件就很有必要了。现在能见到的卸载软件很多，下面介绍几种最常见的卸载软件。
　　一、Symantec公司的Norton Uninstall Deluxe 1.0
　　秉承Norton Deluxe系列软件的一贯特色，该软件界面极其豪华，不负盛名。整个面板以冷色调为主，全面3D化设计。看起来好像在一个浅灰色的大理石窗框上嵌了两块蓝色的玻璃，左边一块略小，呈深蓝色，为主控菜单。右边一块约占2/3的大小，呈浅蓝色，有一种通透的效果，为左菜单项控制下的分菜单。无论是面板左上方金黄色的 “Norton Uninstall Deluxe"字样还是面板下方浮雕一样的按钮设计，都给人一种赏心悦目的感觉，再加上灵敏的鼠标感应功能，用起来心情十分舒畅。
　　1.程序的基本选项介绍
　　Remove　删除一个程序
　　Move　　把程序从一个目录移到另一个目录
　　Copy　　把程序从一个系统拷贝到另一个系统，生成一个自解压文件
　　Store　 把一个不太常用的程序备份起来再删除
　　Restore 把已删除的文件恢复回来
　　AutoClean　自动清除不需要的文件
　　File Cleanup　有选择地清除文件
　　Registory Cleanup　 清除注册表内的无效项
　　Restore 恢复以上操作所清除的内容
　　Install Guard 确认或撤销上一次安装
　　Update SafetyScan 更新磁盘文件数据库
　　2. 总体评价及使用注意事项
　　优点：不管是否由Norton Uninstall记录其安装过程，Norton Uninstall都可以对EXE和DLL文件作彻底的分析，其可靠性和智能化程度较高。另外，Install Guard可以对系统的关键文件做备份，因此在卸载时，甚至能对被修改的DLL文件进行恢复。整套软件安全性较高，适合多数人使用。尤其在临时安装(对某一软件感到好奇但又不想永久安装)时，更显出其优越性。
　　缺点：可以手工控制的东西少，像一个高级傻瓜相机。
　　使用注意事项：每次记录一个安装过程前，一定要重新进入或手工更新数据库；如果一个安装程序没有任何提示就开始安装，那么一定要先启动一个假的安装程序，等卸载程序将系统信息记录以后，才能启动真的安装程序；要及时接受或撤销以前的安装，否则数据库本身要占很大的空间。
　　3.下载
　　感兴趣的朋友可以从这里下载：ftp://ftp.symantec.com/misc/nud.exe。大小约10M。30天试用期。
　　二、Vertisoft Systems公司出品的Remove－It 95 V3.0
　　这是我要向各位朋友郑重推荐的卸载软件，尽管功能朴实无华，出乎意料地地好用。
　　1. 基本介绍
　　这套软件由三部分组成，Log－It主要用来监视安装过程；Smart Disk Agent记录文件被访问的情况，如果一个文件在一定时间内未被访问，可以认为该文件没用并将其从磁盘上删除；软件的主体部分是Remove－It ，下面简单地介绍一下它的主要功能:
　　Remove It ：删除Windows或DOS应用程序
　　Find It ：找出相同的文件并删除
　　Watch It ： 根据Smart Disk Agent所作的记录生成一个统计文件，以便删除不太常用的文件
　　Reinstall It：从备份中恢复已删除的内容
　　Clean Coach ：在软件的引导下，一步一步完成Remove－It，Clean－It，Find－It和Watch－It四个过程
　　Store It：把一个程序备份起来再删除
　　Move It：把一个程序从一个目录挪到另一个目录
　　Transfer It：把程序从一个系统拷贝到另一个系统，生成一个自安装文件或IRF文件
　　2. 关键功能Transfer It的简要说明
　　在作Transfer (系统传送)时，我们可以指定哪些文件和内容需要转移，哪些不需要转移，我们甚至可以加入自己认为需要加进去的东西。这样我们在安装一套软件(如PhotoShop)时，就能把软件的主体部分安装在一个固定的目录(如D:\PShop，最好不要放在C盘上)，用Log－It记录其安装过程。安装完毕后，将 D:\PShop 改名为 D:\tmp－PShop。作Transfer的时候，Remove It 95发现原来的目录不存在了，就会忽略过去。这样，在选择Transfer功能时，就只剩下安装过程对系统的改动了(包括对注册表、INI文件的改动以及加入的DLL等其它文件)，接下来你可以生成一个自安装的EXE(如PShop.exe)文件或一个IRF文件(由Remove It所带的RTINST.EXE执行安装，仅200K)。这样，我们就得到了该软件在Windows 95下的那部分安装程序。
　　现在重新安装WIN95(包括所有的应用程序)，只需要30分钟，把WIN95系统装好后，比如要安装PhotoShop，只需要执行一下PShop.exe，10秒钟之内就解决问题。 你还可以把声卡和显卡的驱动程序以及所有的应用程序做到一个自安装的EXE文件中去，安装WIN95就简单多了! 所以，我现在可以随心所欲地就格式化C盘了。
　　3. 总体评价及使用注意事项
　　Remove It 95安装后，约占4M左右的空间。手工可对其所有功能进行控制，这样使用起来非常灵活。值得一提的是，Log It可以不受系统重新启动的影响，在系统重新启动后可继续工作，这为记录一些驱动程序的安装过程提供了很大的方便。
　　使用时应注意到，Log It不能记录对已存在的DLL文件或其他已存在的文件的替换和修改，需要人工记录这一情况(可以和下面要提到的Easy Clean配合使用)，如果要作系统恢复时，还要将这些文件加进去。另外，在作删除操作时一定要小心。总之，初学者要完全学会使用这套软件需要一定的时间。
　　4.下载
　　这是Vertisoft System在1996年出的一个商业软件，48.93美元。到网上去找一找吧，说不定还能找到。
　　三、Quarterdeck公司出品的Clean Sweep Deluxe 3.06
　　虽然称为豪华版，但与Norton Uninstall Deluxe 1.0的用户界面比起来，Clean Sweep的界面只能说是很粗糙了，看起来就好像某个应用程序的选项对话框。
　　1. 功能介绍
　　Uninstall Wizard卸载软件
　　Archive Wizard 备份某一软件并删除
　　Backup Wizard 备份某一软件但不删除
　　Move Wizard　 把某个软件从一个目录移到另一个目录
　　Transport Wizard　把某个软件从一个系统拷贝到另一个系统
　　Duplicate File Finder　找出硬盘上相同的文件
　　Orphan Finder 清除被遗弃的文件
　　Internet Cache 清除Internet缓冲区
　Restore Wizard　重装程序
　　View Master Log　查看主记录
　　View Savings　查看已节省的空间
　　View Folder Usage　查看文件目录所占用的空间
　　2. 总体评价
　　综合了Norton Uninstall Deluxe和Remove It的长处。由于增加了对Internet的支持，可以说功能比Norton Uninstall Deluxe 1.0更强,但界面较为平淡，不符合豪华版的称谓。和Remove It比起来，手工控制的东西也很多。美中不足的是，Transport生成的是一个BUA格式的文件，安装时需要Clean Sweep软件的支持，也就是说要先安装Clean Sweep软件。
　　3.下载
　　http://www.quterdeck.com/trialware/csdtrial.exe。约7M，15天试用期。
　　四、轻量级的卸载软件
　　轻量级指的是展开后占用磁盘空间在1M以下的卸载软件，有Add/Remove Cleaner、InCtrl3、Easy Clean、WinDelete 97、RemoveR V1.67、Dustman V2.0等。这里向大家推荐Easy Clean。
　　这个软件安装后不到1M，在WIN95或WindowsNT下都可以运行，功能比较单一，只能删除对其安装过程有记录的软件。用法也比较简单，先保存系统状态，然后运行安装程序，然后将当前系统状态与先前的状态比较，结果存入一个ECL文件，以后就可以用这个ECL文件做卸载工作了。
　　优点：所占的空间小、 启动速度快。在程序的主窗口内，可以非常清楚直观地看到安装过程对系统INI文件、注册簿所做的修改以及增加和改动过的文件。
　　缺点：只能记录C盘上的文件变动情况，不能恢复autoexec.bat、config.sys以及被替换过的DLL或其他文件，尽管可以记录到。
　　下载：ftp://ftp.zdnet.com/pub/private/swlib/utilities/file_utilities/ezclen.zip。　　(禾 木)
