几种常见网络操作系统

　　流行的网络操作系统主要有NETWARE系列、WindowsNT Server、Windows95、Windows for Workgroups、VINES。下面分别予以介绍。
　　一、NETWARE系列
　　NETWARE是比较流行的网络操作系统，原因是NETWARE有良好的运行性能以及安全保证。但是在小型网络中，NETWARE是价格最贵的操作系统之一。另外，NETWARE在安装和管理时比较复杂。NETWARE的TSR程序在工作站中的备份将占用约60K内存。NETWARE文件服务器是一个用NETWARE网络操作系统来控制的个人计算机。文件服务器将协调各工作站并控制它们对网络资源的共享方式。所有文件都存在文件服务器的硬盘上，而不是存在软盘上或不同的工作站上。
　　二、WindowsNT Server
　　WindowsNT Server是由微软公司开发的Windows的客户机 /服务器操作系统。它在市场上占有一定的位置，是NOVELL公司的直接竞争对手。WindowsNT Server也提供与NETWARE相同的安全保护级别、特性及可靠的性能，并提供方便的Windows界面。它支持所有的网卡和电缆连接。它可以与其伙伴产品WindowsNT Workstation交互，也可以与其它软件开发平台交互：像MS－DOS、OS/2、Windows3x、Windows for Workgroups及Windows95。它提供一个完整的、集中化的管理软件包，用它可以简化与大型多服务器相关联的管理问题，以及必须支持不同信息协议的网络管理问题。
　　三、Windows95
　　微软公司把对等网络服务集成到它的Windows95操作系统中。它的优点是把基本的对等处理功能融合到一个独立的操作系统中，而无需购买额外软件。如果我们在对等网络环境中只想共享文件和资源，要求成本低，维护少而且对网络安全要求不高时，Windows95是比较好的选择。
　　四、Windows for Workgroups
　　在Windows for Workgroups中，微软把一组对等网络服务和网络应用集成在一起，可以说它是Windows31的更新版本。它提供了用户熟悉的、方便的窗口界面，并能与其它的Windows系统及软件连接起来，对于熟悉Windows的用户，在文件安全性要求不高时，选择Windows for Workgroups是明智的。
　　五、VINES
　　如果需要强有力的客户机 /服务器网络而且对UNIX特别是UNIX中的一些难点比较有经验的用户，VINES是个比较好的选择。当准备建立一个非常大的网络时，需要广泛的数据安全性，并希望所有用户看到的整个系统具有一致性，VINES非常有用。
