PC提速30招

    Windows95我们使用计算机的目的之一是提高工作效率，当然希望它的速度越快越好。毫无疑问，选择更快、更新的CPU、更大容量的硬盘，使用更多的内存、功能更强的主板，都会使你的PC速度有很大提高，但前提是你不免要多“破费”一些。因此，在考虑硬件升级之前，先想想在现有条件下，你的PC机是否已物尽其用，达到了最佳状态，这样则更现实一些。此外，良好正确的操作方法也会帮助你提高效率，本文就会告诉你这方面的30个秘诀。
　　一、将经常使用的软件放入Win 95的“开始”菜单
　　为了节省用鼠标频繁点击菜单的时间，可以将经常要使用的程序放入Win 95的“开始”菜单。具体作法是：左击“开始”、“设置”，选择“任务条”，在对话框中左击“开始菜单程序”、“添加”，输入文件所在路径或通过“浏览”选择所需文件，左击“下一步”，选择“开始菜单”文件夹、“完成”即可。
　　二、将频繁使用的文件的图标放在桌面上
　　如果你不介意桌面上的图标太多，你可以为这些频繁使用的软件建立快捷方式，并为它们在桌面上设置图标，这样只需用鼠标双击图标便可直接运行。具体做法是：右击桌面，选择“新建”、“快捷方式”，输入文件所在路径及文件名，单击“下一步”，输入图标的名称(可以和文件名不同)，单击“完成”。
　　三、把每次启动Win 95都要执行的文件放入Win 95的启动菜单中
　　如果有一些文件是你每次进入Win 95都要运行的，那么你可以直接将它们放入Win 95的启动菜单，每次启动Win 95，它们均会自动执行。在“\PWin95\start menu\program\启动”文件夹便存放着这些文件，因此，只需把文件移动到其中即可。
　　四、设置个人密码
　　如果你与别人共用一台机器，可能会有与其他人不同的桌面设置，这意味着你每次启动Win 95后都要重新设置一番。为了避免这个麻烦，你可以单独设置一个口令，用这个口令进入Win 95，就可以直接进入你特定设置的桌面。你只需进入“控制面板”，双击“口令”，选择“用户自定义桌面设置”和你所要的全部设置，按“确定”重新启动，再输入你的口令，试试怎么样?
　　五、检查Win 95系统的工作方式
　　双击“控制面板”、“系统”、选择“性能”，如果有“Direct X:using MS－DOS compatibility mode file system”(驱动器X：使用MS－DOS兼容方式文件系统)，则意味着你的硬盘或光驱还在老式的16位方式下工作，这种方式下驱动器的速度只有32位方式的一半。查看Win 95的帮助文件，如果它只针对某一个驱动器，可能是因为它还在使用老式的DOS方式的驱动程序，更新驱动程序即可，另一个原因是可能带有病毒。
　　此外，在“性能”菜单下单击“文件系统”，将Read－ahead Optimization设为“完全”，使硬盘能够预测你下一步需要的文件。如果你有超过20MB的RAM，可以试着把“Typical role of this Machine”设为“网络服务器”，这将分配更多的内存作为磁盘高速缓冲。
　　在“系统参数”下单击“图形”按钮，设置硬盘加速器为“完全”，返回“系统参数”，再按“虚拟内存”按钮，选择由Win 95自动设置虚拟内存的数量。
　　六、选择适当的显示器工作方式
　　通常显示器会工作在两种分辨率之下：600×480或800×600，16色或256色，但256色显然比16色的显示速度更慢一些。因此，为了提高速度，文本工作方式可尽量使用16色，而一般的图片也只要在256色下就可以满足，无须24位真彩色。
　　七、安装更合适的显示驱动程序
　　图像的显示速度很大程序上取决于显示卡，因此显示卡的驱动程序非常重要，你可以比较一下显卡生产商所提供的驱动程序与Win 95自带的显卡驱动程序在使用效果上的差异，实际上，最新、最快的驱动程序驱动下的显卡往往会有两倍于Win 95自带驱动程序下显卡的速度。
　　八、注意随时保存
　　自动存储功能很好，但也不能完全依赖它，学会用最快的方法来保存各种文件，能节省重新输入所花费的大量时间。
　　九、适当地并行工作
　　如果有足够的内存，并行工作能帮助提高效率，例如一边打印一个文档，一边输入另一个文档。
　　十、不要开太多的窗口
　　同时运行太多的程序会造成性能的下降，尤其是当你的内存不太“富裕”的时候，因此尽量关闭在当前一段时间内不需要运行的程序，而不要让它们只是以最小化状态存放在任务条上。
　　十一、尽量清除剪贴板中的内容
　　如果你在剪贴板中保存了一个很大的图像或文本文件，它也会占用很大的内存空间，降低速度，尽量清除这些内容，具体可采用再剪切一个单字节的内容到剪贴板中的办法。
　　十二、及时清除“回收站”
　　不要让你的“回收站”积压太多的“垃圾”，它会让整个系统的速度慢下来，这是因为Win 95“预定”了整个硬盘10％的容量作为“回收站”，如果你的硬盘负荷太重，速度自然会慢下来。
　　如果要直接将文件删除，只需同时按Shift键和Del键，也可以修改“回收站”的设置，右击“回收站”图标，选择“属性”，左击“全局”，选择“不将文件移入回收站”即可。
　　十三、阅读Readme，注意软件安装过程中的提示
　　通常软件都附带有Readme文件，其中包含有关于软件运行方式的说明，以及对可能会出现的一些故障的解决方法，在安装软件时阅读一下Readme和提示信息，可能会节约你很多重复操作或解决故障所花费的时间。
　　十四、定时整理硬盘
　　一个混乱的硬盘同样会使系统速度下降，频繁地安装、拷贝、删除、移动和升级软件都会使硬盘的文件杂乱无章，这样在读写文件时，系统会需要更多的时间。Win 95提供了几个程序帮助你整理硬盘：
　　在“开始”、“程序”、“附件”、“系统工具”下，用Scandisk可以帮你检查并自动修复硬盘的错误；而Disk Defragment能够消除硬盘中的“碎片”，重新有序地存入、组织文件。你还可以使用其它的工具软件，如Norton Utilities等。
　　十五、确定你的各个硬件设备已设置成最佳的工作状态
　　例如，你所使用的是增强型的IDE硬盘，检查一下你的CMOS中是否将它设置正确以便它发挥最大作用，如果你的CMOS中硬盘的设置有“LBA”、“Fast－ATA”、“PIO－mode”或“Multi－word DMA”几项，它就适合你的硬盘，否则，你可能需要将BIOS升级，不然无法发挥硬盘的效力。
　　十六、配置图形加速卡或更多的显存
　　如果你是在图形方式下工作，那么选择图形加速卡或增加显存，都会大大提高速度，因为它们能使图像在主机和显示器之间更快地进行交换。
　　十七、关闭网络浏览器的自动下载功能
　　因特网和WWW浏览器可以给你的工作带来很多帮助，但它们也会使系统的速度下降，一个办法是关闭你的Web浏览器自动下载图像、声音和图片文件的。具体可以在Microsoft Internet Explore中，打开“查看”菜单，单击“选项”，在“常规”图标下，将你不想自动下载的内容去掉；在Netscape下，可打开“查看”菜单，不选“自动下载”图片功能。
　　十八、用一只更快的“猫”
　　提高上网速率的另一个办法是使用一个快速的Modem(调制／解调器)。如果你还是用一个144Kbps的Modem“慢”游的话，你现在可以考虑换一个288Kbps、336Kbps或最新的56Kbps的Modem，如果你经常上网， 还可以设置一条专用的电话线。
　　十九、自己设定网络浏览器的缺省主页
　　Web浏览器通常都有缺省的主页设置，每次启动浏览器该主页都会出现，如果你将每次都要进入的主页设置为缺省主页也可以节约一些上网时间。在IE中，只需打开“查看”菜单，左击“浏览”、“使用当前页”按钮，就可以改变启动主页；在Netscape中，打开“查看”菜单，选择全面性能”，左击“显示”，选择你要设置的主页就可以了。
　　二十、及时整理电子信箱
　　E－mail越来越成为人们交流信息的一个重要手段，但你不能让它失去控制，这同样会影响机器的性能。举例来说，如果你每隔几天才检查信箱一次，它可能已经被上百封信“阻塞”了。因此及时检查信箱，删除不要的E－mail是非常重要的，最好给你的信箱装个“过滤器”，每隔几小时花上几分钟查看，其效率比几天不看、再用一个上午的时间阅读更高。
　　二十一、多记几个快捷键
　　现在，每个程序都会给用户提供简单明了的菜单，但同时很多操作也可以通过按快捷键实现，而后者往往更快。节约时间，当然要从一点一滴做起!
　　二十二、如果出现死机，别犹豫，迅速关机
　　无论在DOS、Windows 3.X，还是在Win 95中，按Ctrl＋Alt＋Del重新启动，都是解决死机的有效方法。在Win 95中，按这三个键，会出现一个关闭程序窗口，其中显示出了系统中当前运行的所有文件，以及仍在运行但没有反应的一些程序，这时你有三个选择：1选择要结束的程序，并左击“结束任务”按钮，关闭出现问题的程序；2左击“关机”按钮；3或者左击“取消”，等待系统自动修复自身。大多数情况下，关闭出现问题的程序会使系统恢复正常，你只需要用比重新启动整个系统还要短的时间就能重新工作了。
　　二十三、注意检查病毒
　　没有什么比病毒更具有“杀伤力”了，轻的可以占据硬盘、破坏程序，使系统性能下降；重的可能让整个系统崩溃。所以一定要注意检查病毒，安装几个功能强的防病毒软件是个很好的选择，但是防病毒软件所能消灭的病毒各有侧重，而且不断有新的病毒出现。因此，也要不断地更新、升级防病毒软件。
　　二十四、压缩与解压缩
　　不要压缩你的硬盘，虽然表面上看起来它能“扩大”你硬盘的容量。一个被压缩的硬件要花费更多的时间来查找文件，而且如果运行过程中出现了问题，压缩过的硬盘也不便于处理。
　　二十五、将操作手册放在容易取放的地方
　　操作手册应该固定放置在一个容易取放的地方，并经常查阅，以便帮助你解决操作过程中遇到的一些问题。
　　二十六、经常作备份
　　将文件存在硬盘或软盘上并不是万无一失的，一旦磁盘出现问题，文件就有可能被破坏。因此，定期备份是非常必要的。在Win 95中也提供了程序帮助用户备份。左击“开始”、“程序”、“附件”、“系统工具”就可以调出备份程序，它可以按文件产生的时间、大小进行备份。
　　二十七、选择合适的打印分辨率
　　打印速度也是影响Win 95系统性能的一个因素。高分辨率方式的打印速度比低分辨率要低，例如，从600dpi降到300dpi，文件打印速度能提高一倍以上。因此，在打印文本文件时应尽量选择较低的分辨率，而在打印图像时采用较高的分辨率。
　　二十八、为打印机选择更好的驱动程序
　　打印机的语言是影响打印机速度的另一个瓶颈。大多数打印机采用HP的打印机控制语言(PCL)，但Win 95采用的语言是图形设置接口(GDI)。打印过程中的一部分延迟是由于Win 95必须把它的GDI翻译成打印机语言。部分打印机随机提供PCL和GDI两种驱动程序，如果你的PC机有足够的RAM，采用GDI类的驱动程序能使打印速度更快。
　　二十九、阅读软件安装指导
　　安装、使用一个新软件，当然是一件很令人激动的事，但事先阅读一下安装指导是很有必要的，虽然大多人不具备读完长长的说明书的耐心，可是其中的重点部分不应忽略，它往往有助于确定安装过程中要做的一些选择，可以省去你重新安装软件所花费的不少时间。
　　三十、安排一个舒适的工作环境
　　很多人对这一点不以为然，实际上选择一个好的工作椅、一个看起来舒服的显示器、使用方便的键盘，加上一个整洁的工作台，都会在无形之中帮你提高工作效率。
