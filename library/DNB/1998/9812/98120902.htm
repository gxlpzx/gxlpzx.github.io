一个全能的图像转换器

　　我们面对的是一个五彩缤纷的世界，反映到计算机中就是通过各种手段输入(如用扫描仪、数码相机，当然包括手工制作)，根据不同的要求及用途，以不同格式的文件保存到计算机中，再利用各种图形图像软件进行处理，以取得更好的效果。
　　图像格式的繁杂，一方面是为适应不同的要求，但同时也给我们的交流带来了麻烦。现在图像格式转化工具不少，但也存在转换格式有限、功能单一等缺点。那么有没有更好的选择呢?
　　Graphics workshop使你如愿以偿，它短小精悍，可以运行于Windows 3.1、95或NT，而且它是一个共享软件，你可以通过网络Download，其网址为：http:www.mindworkshop.com。
　　说Graphics workshop是一个全能的图形图像转换器，是因为它不但转换当今流行的几乎所有格式的图像文件(任意两种格式之间进行转换，当然有一些格式有些限制)，而且可以对图像进行诸如打印、色彩增强、抖动、变形、旋转、比例缩放(带有平滑防变形功能)(anti－aliasing)、剪取、强化、柔化、反转、量子化、屏幕捕捉、创建幻灯片等等功能。
　　本软件包支持40多种不同的图像格式，其中包括BMP，Casio(卡西欧)公司的CAM，CDR，FIF，GIF，IFF/LBM，JPEG，Kodak(柯达)公司的KDC，Photo－Cd,Icon(图标)，IMG，PCX，PNG，PhotoShop公司的PSD，SGI/RGB，TIFF，Uuencoded,WMF,WPG以及XBM。所支持的多媒体格式有MPG，MOV，AVI和FLI。还同时支持TWAIN兼容的扫描仪。
　　有一点不能不提，它可以直接把图像文件转换为.exe可执行文件，够霸道的了吧。
　　Graphics workshop的界面窗口上有菜单条和工具条，下方是文件或驱动器(如C、D)图标，选中你想处理的文件，利用工具条上的工具按钮(或菜单中的菜单项)进行处理；如果双击此文件，进入另一个窗口，可利用其中的菜单进行处理。要注意，它把当前文件夹中的所有类型的文件都列出来，选择时要看图标下方的后缀名。要改变当前文件夹，到“File”菜中选“change directory”执行选择。
　　如你选中一个.bmp文件(其下方后缀名变黑)，按工具条格式转换按钮(左方第一个按钮，其英文提示为“Convert between file formats”)，弹出一个工具条，上面布满各种图像格式按钮，用鼠标点中想要转换成的格式，在眨眼之间转换完毕，速度极快。并且它给你另取一个文件名，原文件安然无恙。
　　唯一不便的是其为英文版，但在众多的电子词典面前这已不是问题了。只用几个钟头，你就可以发掘出其强大的威力，各种图像信手拈来，何不赶快试一试呢?    (四川　牟善铸)
