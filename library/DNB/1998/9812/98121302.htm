计算机素描：他是谁？

　　曾经在一本杂志上看到一幅美国总统的低精度画像，这里我用BASIC语言把这幅画像献给大家，看看你能不能猜出他是谁?刚开始你可能还看不出是谁，那就离显示器远一点，再远一点，是不是有点像……
　　CLS:SCREEN 0,nonzero:DIM m(5)
　　PRINT TAB(27):″Who is he?″
　　m(0)=32:m(1)=176:m(2)=177:m(3)=178:m(4)=219:m(5)=219
　　FOR i=1 TO 18:READ st,ch＄：LOCATE 5＋i,20＋st
　　　　　FOR j=1 TO LEN(ch＄):a=VAL(MID＄(ch＄,j,1))
　　　　　　　　　IF a=5 THEN COLOR 15
　　　　　　　　　PRINT CHR＄(m(a))；CHR＄(m(a));:COLOR 7
　　　　　NEXT j:PRINT
　　NEXT i:END
　　DATA 7,″425″,5,″255555″,4,″45325555″,3,″355000055″,3,″5530100435″
　　DATA 2，″5541001105″,2，″45551453355″，2，″50552351553″，3，″305111013″
　　DATA 3,″505511225″，3，″155522055″，4，″35512355″，2，″1145555355″
