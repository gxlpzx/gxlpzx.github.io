Windows95的保护神WinSafe98 4.5

　　 编者按： 对于许多新手来讲，电脑的软件维护确实是一件麻烦的事，虽然小心谨慎，但总免不了出错。一但Win95不能启动，就会被弄得焦头烂额。其实，软件维护并不难，关键是要有得力的工具，本期给大家介绍的两个新的工具，在维护软件系统，保证Win95安全方面很有帮助。害怕Win95出错的朋友一定要看哟!
　　许多PC用户,特别是初级用户在使用Windows95时都会碰到这样或那样的故障，严重时导致数据丢失而使Windows95启动不了甚至系统崩溃，这时也许只有重装Windows95或格式化硬盘了，这可是一件相当麻烦和费时的事。
　　如果有了WinSafe95，则一切都变得相当简单了，WinSafe95能为你的Windows95保驾护航。每次启动Windows95后WinSafe95会首先对系统进行检测，能发现问题并指出问题之所在，即使出现了上面所说的故障，也只须在c:\Windows95（如系统崩溃不能启动则可由A盘启动后再进入）下键入Winsafe回车就行了，WinSafe95会自动使Windows95恢复到原来的状态。可以说有了WinSafe95，Windows 95出现故障时你一般用不着将系统重装。
　　下面简单介绍一下WinSafe95的功能：
　　安装好WinSafe95后在桌面上会有一个快捷图标，双击该图标则会出现界面。
　　界面上有20个按键，5个菜单项。各项按键功能简述如下：
　　Edit System：编辑系统文件，这些文件包括Win.ini、System.ini、Autoexec.bat、Config.sys、Protocol.ini及MSMail.ini等。
　　Edit Registry：编辑注册表，即调用Windows95的注册表编辑器。
　　Hardware：诊断各硬件，并列出其结果，内容相当详细。
　　File Report：检查Windows95文件夹下的系统文件，包括所有的ini文件，并与最后一次启动Windows95成功后所收集的系统信息进行比较，告诉你改变、增加、删除了哪些文件。
　　Restore with Current：按当前状态进行恢复。
　　Restore from Safe：如果你不小心将错误的注册表进行了备份，则用Restore with Current进行恢复时不能成功，使用本键可以恢复你在安装WinSafe95时的备份。
　　Restore with Original：按成功地安装完Windows95时的状态恢复。
　　Check Registry Size：检查注册表大小，并创建一个详细的报告文件detail.log，其中包含system.dat、user.dat、system.ini、win.ini、himem.sys、
ifshlp.sys、dblbuff.sys、mouse.sys、moˉuse.ini、ramdrive.sys、protocol.ini、vmm32.vxd、autoexec.bat、config.sys、drvspace.bin、command.com、io.sys、msdos.sys等文件在安装Windows95、WinSafe95时和当前的大小。
　　MSD：调用MSD.exe。
　　Anti Virus：调用病毒检测程序msav.exe。
　　Defragment：整理文件碎块。
　　Scandisk：硬盘扫描。
　　System Info：显示系统信息，内容相当详细。
　　Extract Files：从Windows95光盘中的cab包中释放所需的文件。
　　System Report：创建一个系统报告。此报告可由菜单Files中的View System Report项查看。
　　Repair System Files：修复系统文件，包括dll文件、drv文件及Vmm32.vxd文件等。
　　Repair System Icons：修复系统图标。
　　Bootlog：查找是否有bootlog.txt文件，若有则显示其内容。
　　Backup Current Files：备份当前的重要文件。如各系统文件、Vmm32.vxd、注册表等。
　　Exit：退出。
　　如果我们选菜单项Setup/Boot Options/Set Bootup On，则还会在桌面上出现另一个图标， 其功能是重新启动Windows95或关闭计算机。在使用中你还会发现WinSafe95的菜单中还有许多其他功能，这里就不详细说明了。从其功能可以看出WinSafe95 4.5确实是保证Windows95安全运行的一个优秀软件，但遗憾的是下载得到的只是一个试用版。有兴趣的读者可去http://members.aol.com/easydesk处下载，那里还有很多有趣的软件。也可向easydesk@aol.com发一封E－mail索取。
