超越Windows95资源管理器---PowerDesk Utilities 98

　　 你是不是对Windows95中的资源管理器不太满意呢？由Mijenix公司出品的PowerDesk Utilities 98 3.0可以满足你更高的要求。PowerDesk Utilities 98包括文件管理器、工具条、磁盘空间管理器等几个主要工具。
　　PowerDesk文件管理器是这套工具的核心，操作十分方便，可以完全由自己定制工具条；驱动器条让你快速地在各个驱动器之间切换；一个程序条可以自由安排各种应用软件的图标，支持文件拖曳操作；包含一个DOS命令行窗口，有些命令不必回到DOS下就可以使用；系统将许多Windows 95系统的控制项目嵌入管理器的菜单或工具条中，如建立启动盘、添加/删除程序、注册表编辑器，这样大大方便用户的使用。
　　不仅操作较Windows 95的资源管理器方便，它具有的强大功能更是资源管理器所望尘莫及的。
　　随着各类软件的不断膨胀，许多用户往往使用压缩方式保存软件或数据，现在主要的压缩软件不外乎ARJ、ZIP、RAR等，还有微软的CAB格式，在压缩管理软件中首屈一指的是WinZip，不过它不支持RAR，WINRAR虽支持RAR等软件，界面与操作却又不太理想，而PowerDesk的文件管理器却完全支持这些压缩软件，它将这些压缩包视为一个个目录，对压缩包内的文件操作如同对一个目录中的文件进行操作。
　　文档多了，从中查找所需的文档变得十分困难，一个综合性的浏览器窗口，就显得十分有必要。PowerDesk本身支持多种文件的浏览，不仅如此，PowerDesk能充分利用Quick View Plus（http://www.inso.com）在文件浏览方面的强项，将Quick View Plus的功能嵌入文件管理器的浏览窗口中，从而大大扩展浏览的范围，Quick View Plus支持包括文本、图像、多媒体、Office文档、数据库、可执行文件在内的许多文件格式。
　　PowerDesk在图像转换方面也有强大的表现，可以自由转换24种不同格式的图形图像，转换时还可以设定色彩、压缩率等选项。
　　PowerDesk可以对电子函件进行解码，包括UUencode, MIME/Base64和BinHex方式，并且可以建立UUencode文件。
　　PowerDesk具有的安全系统，可以对文件、目录进行加密、解密，系统采用其独有的加密算法（系统也支持标准的DES加密算法，但只限于美国或加拿大）。
　　PowerDesk Utilities 98的工具条类似于Windows3.1的SideBar或拖动条，当你对Start（开始）键中弹出的层层菜单手忙脚乱时，工具条可帮上忙。工具条完全由你定制，可以把常用的程序放在上面，或加上系统监视器，打印管理器，DOS命令条，显示器管理等。而且工具条也支持虚拟屏幕的管理。
　　硬盘空间管理器帮助你了解硬盘空间的使用情况，利用管理器中排序/搜索/图表和分析工具能了解硬盘上每个子目录的大小，是充分利用硬盘空间的“利器”。PowerDesk特别在管理器的工具上布置了“添加/删除程序”按钮，方便你删除程序。
　　Windows 95的再启动、再登录、计算机的关闭与再启动，在PowerDesk Utilities 98中都变成了一个个单独的小程序，用起来更加方便。
　　此外，在该公司的网址上（http://www.mijenix.com/），还能下载PowerDesk Utilities 98的一些附加程序，如AVI播放程序、屏幕保护、游戏、记事本等，而且可以利用SDK编制附加程序。
　　应该说，这套工具中的许多功能都能在其他单独的软件中体现，如Drag and File Gold、Quick View plus、WinZip、ACDSee、GWS、SideBar等，但PowerDesk Utilities 98将这些功能完整地组合在一起。不必装上这么多的软件就能得到这些软件的功能，是PowerDesk Utilities 98的特有价值的体现。
　　程 序 名：Pd98eval.EXE(30天评价版)；自解压缩/安装程序 约2.8M。
　　系统需求：－ Windows 95、98或 Windows NT 4.0；16M内存；19M硬盘。　(上海 陈海阳)
