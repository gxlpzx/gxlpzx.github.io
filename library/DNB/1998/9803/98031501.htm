IE4.0特色功能

      比尔·盖茨曾说过“终有一天Internet将会成为人们日常生活所需”（Someday we′ll take the Internet for granted），但要到Internet上轻松自如地冲浪，一套高性能、方便实用的网络浏览器是必不可少的。目前，被喻为世纪之战的Netscape（网景）公司与Microsoft微软公司浏览器之争正进行得异常激烈（以至于闹到法庭上去了！），两家公司分别于去年年底推出了最新版本的浏览器——Netscape Communicator 4.02、Microsoft Explorer 4.0(简称IE4.0)，尽管网景公司是以开发网络浏览器起家，并仅用三四年时间便急速成长为全球软件界的一棵大树（在最近排出的全球最有影响100家计算机公司中仅次于微软、Intel、IBM而位居第四），其浏览器的市场占有率也一度遥遥领先于微软的Explorer，但微软凭借其强大的技术资金势力及操作系统上的先天优势，将IE4.0与Windows95操作系统完全整合，并充分体现个性化设计、易用性及安全可靠等特点，所以IE4.0一推出便受到不少用户的好评，更令人叫绝的是微软的网上免费下载及与Windows95的捆绑整机预装策略，这无疑会使广大的电脑用户更倾向于IE4.0（谁还愿意掏一大笔钱去购买一个可免费得到而且功能差不多的软件？）。下面便对IE4.0，尤其是其特色功能的使用作一个总体介绍，相信你看过之后定会迫不及待上机一试。
　　【系统要求】
　　要运行 Internet Explorer 4.0，您的系统必须满足以下最低要求：
　　○CPU：66 MHz 的 486 处理器（建议用奔腾处理器）
　　○内存：对于Windows 95，最少需要 8 MB 内存，使用 Active Desktop 则需要 16 MB 内存；对于Windows NT，最少需要 16 MB 内存，使用 Active Desktop 则需要 24 MB 内存；对于 Windows NT，必须运行 Service Pack 3（或更高）
　　○硬盘：对于完全安装需要 40 到 70 兆的硬盘空间(依安装类型而定）
　　○鼠标
　　○调制解调器
　　○光驱
　　●方便快捷的安装
　　IE4.0的安装与以前版本相比并没有因为其功能的增强而变复杂，仍然让人感觉到方便、快捷。运行Setup.exe安装程序之后，整个安装过程将在“IE4.0智能安装向导”的指导下顺利完成，我们所做的操作多半是连续按“下一步”按钮。但下面几个问题仍然值得注意：
　　·询问是否安装“桌面更新”时，最好选择“安装”，否则我们将看不到整合了IE4.0后，Web风格的Windows界面，包括极富魅力的Active Desktop。
　　·选择“标准安装”，IE4.0将为我们安装上WEB浏览器、Outlook Express(电子邮件与新闻阅读)以及多媒体增强组件；而选择“完全安装”，除上述各组件之外还会安装上Net Meeting 、Microsoft Chat2.0等；为了节约空间，我们完全可以抛掉那些平时很少使用的组件，选择“标准安装”。
　　·对“Active Channel(热线频道)”的选择，一定要选择“中国”，这样我们得到的才是符合我们所在的国家及所使用的语言，为我们度身定制的热线频道。
　　●个性化的桌面
　　安装结束后重新启动计算机，你会惊喜地发现，桌面变得更丰富了，如图1所示。

图1

　　黑色的Active Desktop背景，桌面上增加了一组别致的按钮，这便是微软引以自豪的热线频道，在热线频道栏有“人民日报”、“CHINANET(中国国际互连网)”、“CEInet(中国经济信息网)”、“上海热线”、“计算机世界”等热线频道，在英文IE4.0中我们将看到AOL等美国热线。在任务栏“开始”按钮右边还增加了“查看频道”、“启动Internet Explorer浏览器”、“启动Outlook Express”、“显示桌面”四个图标。
　　你可以在Active Desktop上放置自己喜欢的或与生活有关的主动播放频道，如新闻、天气预报、股票接收机等，而且可以根据需要来设定更新的周期，这样便可保证你在桌面上永远看到最新信息。你也可以在Active Desktop上放置其他网站上的网页或你自己设计的HTML文件；更有趣的是，你可以将网上一些生动活泼的GIF动画也安置到Active Desktop上。总之，只要你愿意，你完全可以将桌面设计成自己的天地，一个充满个性的世界。
　　具体操作方法为：
　　·在Active Desktop上放置频道。用鼠标右键单击桌面，然后单击“属性”。单击“屏幕保护程序”选项卡。在“屏幕保护程序”列表中，单击“频道屏幕保护程序”。单击“设置”，然后选择所需的设置。
　　·用鼠标右键单击 Web 页上的任何图片，然后单击“设置为墙纸”。
　　·用鼠标右键单击 Web 页上的任何链接，将其拖动到桌面上，然后单击“创建 Active Desktop 项”。 也可以用鼠标右键单击桌面，然后单击“属性”，单击“Web”选项卡，然后单击“新建”，如果想浏览“Active Desktop 画廊”以查找要添加的组件，请单击“是”。如果想选择其他 Web 站点，请单击 “否”，然后键入目标 Web 站点的地址，或单击“浏览”以定位该地址。
　　●焕然一新的资源管理窗口
　　打开“我的电脑”，哇！你可别以为是点错了，整合了IE4.0后，所有的资源管理窗口，包括“我的电脑”、“网上邻居”以及“控制面板”等，都变成了类似Internet浏览器的窗口，如图2所示。

图2

　　在这里你完全可以像过去一样自如地访问各种资源，而且不需要在一层一层的窗口之间来回切换。如果使用好地址栏，将会真正体会到“网络便是计算机”。在地址栏可以选择访问计算机上包括磁盘、文件夹、打印机在内的任何资源，甚至网上的邻居，而且要是在地址栏输入Web地址(URL)，更可以到网上遨游，也就是说，通过无所不在的地址栏，对本地电脑、局域网、Internet的访问将会浑然为一体。另外，“前进”和“后退”按钮更能很好地为你记住曾经访问过的路径，令你随意切换。
　　●将常用的网页放在手边
　　点取Windows任务栏上的Internet Explorer图标，我们便可以进入网络浏览器了，如图3所示。

图3

　　为了方便地访问常用的网页，只需用鼠标将常用网页的超链接从地址栏或 Web页拖到“链接”工具栏，便可创建该超链接的按钮，以后我们不需要在地址栏输入地址，而只需点击该按钮就可以迅速地访问它。当然，也可以将 Web 页添加到收藏夹列表中（在预订中详细说明），以便通过“收藏”菜单或“浏览器栏”便捷地访问。
　　●使用“浏览器栏”方便快捷地漫游 Web
　　我们可以使用新的“浏览器栏”方便快捷地搜索 Web 站点。单击工具栏上的“搜索”按钮，“浏览器栏”将显示在浏览器窗口的左侧，并且将搜索结果陈列其中，这样，您就可以在查看左侧搜索结果列表的同时，单击相应的超链接，在屏幕的右侧方便地浏览网页了。除此之外，您还可以相同的方式浏览“收藏夹”、“历史记录”、“频道”或文档，操作方法完全一样。
　　●让 Internet Explorer 帮助您键入 Web 地址
　　IE4.0所提供的地址栏融入了“自动完成”的功能，当您在地址栏键入常用的Web地址（ URL） 时，Internet Explorer 会自动帮助您完成该地址的输入。如果键入或单击了错误的地址，Internet Explorer 甚至可以搜索相似的 Web 地址以查找匹配的条目。
　　●预订经常访问的站点
　　预订是Microsoft Internet Explorer 4.0 引入的一种查看网页内容的全新方式。当我们打开一个Web页，如果是自己经常访问的站点，我们可以在“收藏”菜单中，单击“添加到收藏夹”，如图4所示。

图4

　　这时选择“是：仅在该页更新时通知我”或“是：更新时通知我并下载该页以便脱机阅读”，便可以使用预定功能让 Internet Explorer 按照您的安排，检查收藏夹中的站点是否有新的内容，并可在有可用的新内容时通知自己，或者在自己正忙于计算机上的其他工作甚至休息时（比如夜间），将更新的内容自动下载到本地硬盘上，这样，我们便可以在闲暇时脱机浏览了。这种“预订”Web站点的方式并不需要向该站点支付任何费用，而且可以根据需要安排按天、按周或按月更新已预订的所有 Web 站点，当然也可只安排更新个别站点。
　　●新颖的网上频道
　　“频道”是设计用于从 Internet 向您的计算机传递内容的 Web 站点，这和预订最喜爱的Web站点有些相似。通过频道，可以将全球顶尖的内容提供商的最新 Web 内容自动传送过来，在IE4.0中预先设置了“人民日报”等国内的网站频道。我们既可以在“浏览器栏”方便地访问这些频道，还可以像开篇所说的那样，在桌面上或作为屏幕保护程序浏览这些频道（如果安装了Windows桌面更新组件）。多数频道还提供了设置预订的选项。
　　通过频道，不仅能看到 Web 页，而且还可以获得一张 Web 站点的详细地图，相当于该频道站点的主要内容目录，如图5所示，通过它我们可以快速选择和查看所需的内容。

图5

　　如果想要将新的频道添加到频道栏，可使用桌面上或浏览器中的“Microsoft Channel Guide（频道指南）”，通过 Microsoft Web 站点查看可用频道的列表（该列表将随着新的或现有的内容提供商所提供的最新内容而不断更新），按照屏幕上的提示操作即可。除“频道指南”外，内容提供商还为用户提供从 Web 站点直接访问频道的服务。
　　●让脱机浏览为你节约上网费用
　　点取“文件”菜单并选择“脱机工作”项，我们便可在不连接到Internet的情况下查看Web页了。因为频道和预订功能早已在我们连接到 Internet并处于联机状态时，将最新的网页内容自动下载到硬盘上了。除此之外，还可在脱机状态下浏览储存在“历史记录”文件夹或“Temproary Internet File”文件夹中的任何Web页。脱机浏览功能对于家中仅有一根电话线又不想让计算机长时间占用它的情况非常有用，况且 可以节约一笔上网费呢!
　　如果你要结束脱机状态，请重新点取“文件”菜单，并再次单击“脱机工作”复选框，清除复选标记。
　　●不断完善的安全性
　　由于IE4.0的前版IE3.0在安全性上存在不少漏洞，所以受到不少用户的指责。在IE4.0中微软可谓颇下了一番功夫，这也是97年春季数次推迟IE4.0发表时间的一个重要原因。为了保护你的电脑，IE4.0为不同地区设置了不同的安全级，并利用分级审查筛除那些令人不愉快或冒犯性的内容，如果有条件你还可以用Microsoft Wallet在你的电脑上存储个人信用卡及购物地址信息，利用安全服务器连接到WEB站点。
　　●卸载 Internet Explorer 4.0
　　如果你想卸载 Internet Explorer 4.0，可以执行以下步骤：
　　·在“控制面板”中，单击“添加/删除程序”图标。
　　·单击“Microsoft Internet Explorer 4.0”，然后单击“添加/删除”。
　　·按照屏幕上的提示操作。（徐 群）
