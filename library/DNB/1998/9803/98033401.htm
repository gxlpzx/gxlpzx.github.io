专家坐堂

　　重庆刘洋问：为什么我的计算机在安装Windows 95后第一次引导就死机，进入安全模式后发现显示器类型为未知的显示器，改为标准VGA后重新引导，仍然不行。请问何故？微机配置为：海洋主板，Cyrix 486/50 CPU,16MB内存，420M硬盘，8速光驱。
　　答：Windows 95 启动时死机的原因比较复杂，因为Windows 95在系统引导时要经过一个彻底的软、硬件检测过程。在这个过程中，任何一个环节有问题都可能引起系统工作不正常。系统第一次启动就死机，应怀疑硬件安装或设置有问题，可试着从下列几个方面入手检查：
　　1.禁止32位磁盘存取
　　如果你的硬盘不支持32位存取方式，那么系统会在启动过程中挂起，这时可以采取如下方法排除：
　　首先，在系统引导过程中，当看见“Starting Windows 95”时按下F8键，用“Safe mode command prompt only”(安全模式)启动系统，当屏幕出现DOS提示符后，键入“WIN/D∶F”来启动Windows 95。如果成功，则可禁止系统采用32 位磁盘存取模式来运行Windows 95,当然，这时整个系统的性能会大大降低，这是不得不退而求其次的办法，最好还是赶快换掉这个老掉牙的硬盘，以便可以真正领略一下Windows 95的风采。
　　2.保证基于BIOS的磁盘操作
　　如果采用前述方法不能成功，那么你的Windows 95可能需要使用基于BIOS的磁盘操作系统。方法与前述基本相同，只是在出现DOS提示符时，键入“WIN/D:V”，如果启动成功，那么可在SYSTEM.INI 文件中加入下列一行：
　　VirtualHDIR=False
　　3.禁止Windows 95使用ROM断点
　　一个ROM断点是PC BIOS中的一个地址，它含有Windows 95从安全模式转换成实模式时所使用的指令。一般情况下，Windows 95在指定的地址寻找那些断点指令。但是，如果您使用了第三方内存管理程序，Windows 95在运行时就可能遇到麻烦，这时可以禁止Windows 95使用ROM断点，方法与前述大致相同，当出现DOS提示符后，键入“WIN/D∶S”，如果成功，可在SYSTEM.INI文件中加入下列一行：
　　SystemROMBreakPoint=False
　　4.禁止Windows 95使用视频卡内存
　　在前述方法无效时，您也可以试一试用这种方法， 具体为：进入“Safe mode command prompt only”(安全模式)，当出现DOS提示符时键入“WIN/D∶X”，如果成功，在SYSTEM.INI文件中加入下列一行：
　　EMMExclude=A000－FFFF
　　5.修改系统注册表
　　当然，Windows 95在启动时遇到麻烦，也可能是系统注册表有问题，这时可以通过系统启动菜单的“Command prompt only”选项进入DOS状态，然后键入REGEDIT来进行注册表的检查和修改。由于注册表包含了系统大量的软、硬件配置信息，因此使用该方法需要对Windows 95有较为深入的了解，而且修改不当，反而会造成系统崩溃，因而，还是小心行事为好。
　　读者高峰、苏勇问：一台新组装的586兼容机，使用一段时间后，开机出现下列故障：硬盘灯亮，显示器灯变绿，但大约30 秒后硬盘灯灭，显示器灯变黄，屏幕始终无显示，系统无法引导。而待数十秒后再关、开机，则上述现象不会再出现，使用一切正常，且使用过程中热启动微机也一切正常，请问何故?
　　答：这种现象笔者也曾遇到过，从现象看似乎是显示卡接触不良，重新拔插显示卡并且更换一个扩展槽插入，情况有所好转，但不久故障依旧。后经检查发现，故障出在主板上。主板发生此种故障，主要有两种原因：一种原因是主板兼容性差，笔者曾经在用小皇板、S765显示卡、金鹰声卡组装电脑时就遇到过这种问题，只要插上声卡，计算机启动就不稳定，时有黑屏现象发生，而换用华硕主板则绝无此现象，将显示卡换为GOLD 16也无此现象。但从读者来信所述情况看，计算机以前一直工作正常，似乎可以排除此种原因。另一种原因实际上是主板供电不正常产生的，主要是电源部分电路中的电容有问题，造成其需要充电的时间太长，因此在第一次冷启动微机时，由于充电时间较短，电容不能很快充足电，所以不能进入正常工作状态，造成主板供电不正常。而通过第一次冷启动后，电容实际上已进行了初充电，关机，紧接着再次开机后，电容充电时间基本上可以满足供电要求，因而，微机再次启动正常。使用过程中热启动时，由于电容一直处于正常的充放电状态，所以不会影响主板的供电，故障现象自然不会再现。至于电源部分更详细的电路分析，本报1997年第46期曾有介绍，可以参阅。
　　江西省刘显问：一台松下激光打印机，以前使用一直正常，最近因墨粉用尽，添加墨粉后发现：打印输出时，在每页纸的同一位置总有一竖着的空行打印不上（空行贯通整页纸），但打印机自检可以通过，且打印机的其它情况一切正常，请问何故?
　　答：从来信所述情况分析，故障极可能出在打印机的激光镜面上。我们知道，激光打印机主要是由激光扫描系统、电子成像系统和控制系统组成，激光扫描系统的主要作用是利用激光束扫描形成静电潜像，电子成像系统则把静电潜像转变成可见的输出，而控制系统则负责整个系统的协调工作。如果激光扫描系统部分有问题，激光打印机根本不可能形成静电潜像，更谈不上打印输出，而现在打印机的输出只有某部分不正常，说明激光扫描系统并无问题。同时，打印机自检可以通过，说明控制系统工作正常。因而可以肯定故障出在电子成像系统，而电子成像系统一般包括硒鼓、高压发生器和激光镜面等，其中对输出成像有主要影响的是硒鼓和激光镜面。而如果硒鼓某部位有损伤时，其相应输出部位一般呈黑色，而现在则是呈白色，说明这是由于激光镜面某部位被某种东西挡住了，使得激光束无法通过此部位，因而在此处不能形成图像。由于故障出现前更换了墨粉，由此可以推断，由于填装墨粉和安装硒鼓时不小心，使得激光镜面（在硒鼓安装位置的背面）某部位沾染了墨粉，造成输出不正常。处理此种故障的方法是：取下硒鼓，用照像镜头纸轻轻擦拭激光镜面，使其清洁即可。
