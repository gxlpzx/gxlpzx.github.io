用F－Secure Desktop锁住你的秘密文件

      目前，在单位或家庭中，常常几个人共用一台电脑，硬盘上的文件毫无保密可言。希望有一种方法能够防止别人偷窥你的私人文件。市面已有的加密保护软件虽然也能用，但总觉得不够简便。带着这种渴望，近日在网上觅得一小巧、实用的“傻瓜”式高级保护软件F－Secure Desktop，试用之后感觉甚佳。好东西总想与网友共享，告诉你地址，快去把它Download下来吧。
　　F－Secure Desktop分为家庭教育版和商业版两类。其所在网址为：
HTTP：//www.datafellows.com/
HTTP：//www.europe.datafellows.com/
　　为节省篇幅，本文只简介F－Secure Desktop For Home＆Educational Use 2.0WIN95版的安装和使用方法。
　　1.安装　用Winzip将下载的d9enu20h.zip文件解压，执行Setup.exe安装程序，安装过程中将要求设置用户名(假设为user)和密钥(密钥不同于密码，密钥不仅可以是一串字符或数字，而且可以是一个英文短句，如“This is something for me to remember”)，更加便于记忆。同时产生一个以用户名命名的密钥文件(如：user.key)。最后要求设置加密级别，共分三个级别，初级为一次加密，时间短，保密性差；中级为二次加密，时间和保密性适中；高级为七次加密，时间稍长，保密性强。默认为中级加密。安装完毕后，WIN95将建立F－Secure Desktop程序组，其中包含Auto Ecrypt Files、Auto Decrypt Files、Destop Help、F－Secure Destop Help、F－Secure Destop Wizard、Readme等六个程序项。
　　2.直接加解密(Encrypt/Decrypt)F－Secure的操作十分简便，不用调用任何程序，只要将鼠标指向目标文件，按鼠标右键，即会跳出加密功能栏，可以进行加解密操作。加密后文件扩展名变为.fff，小图标变成带钥匙的图标，其它属性均不变。解密时只需双击目标文件，便会自动提示输入密钥，解密后文件扩展名和图标恢复原样。
　　3.自动加解密(Auto Encrypt/Decrypt files)　这是一项十分实用的功能，当设置自动加解密文件后，在进入WIN95前，将提示用户输入密钥以对所有被加密的文件进行解密，这使得用户不必对每个加密文件进行分别解密；在退出WIN95时，会也提醒用户输入密钥以便对被设定为自动加密的文件进行加密。设置自动加解密文件夹，可以在安装程序时根据提示指定需要保护的文件夹或文件，也可以调用F－Secure Desktop Explorer(不是WIN95的Explorer)来设定，只要将鼠标指向需自动保护的文件或文件夹，按鼠标右键便会跳出提示窗供用户设定。
　　4.不常用保护文件夹(Top Secret)　如果设置自动加解密的文件太多，那么进入或退出WIN95的时间就会相对延长，因此对于有些需保护但又不常用的文件，可将它们列入不常用保护文件夹中，这些文件除非直接解密，否则不会自动解密。不常用保护文件夹可在F－Secure Desktop Explorer中设定，只要将鼠标指向需保护的文件或文件夹，按鼠标右键便会跳出提示窗供用户设定。
　　5.参数的改变　运行F－Secure Desktop Wizard可按提示对用户名、密钥文件、加密级别等参数进行更改和建立。
　　6.使用技巧　一般的保护软件只要知道密码，便可在任何安装相同保护软件的电脑上进行解密，而F－Secure Desktop则不同，对于一个经F－Secure Desktop加密的文件进行解密最重要的是密钥文件必须相同，而密钥可以不同。这主要是由于F－Secure Desktop的密钥文件是一串随机产生的由16个字符或数字组成的文件。当用户重新安装该程序时，所产生的密钥文件是不同的，因此原密钥也随之失效，会出现解密出错的提示。这充分表明该程序的保护性能之优越，因为即使在网络传输中有人知道密钥，但若没有密钥文件，也是无能为力的。由于F－Secure Desktop拥有上述功能，因此，在F－Secure Desktop Wizard运行中设置了选用或重新建立密钥文件的选择提示，用户可以选择使用原密钥文件，这样便可以顺利解密(即使密钥不同也可)。若将密钥文件复制给某工作组的所有人员，那么这组人员便可共享被保护的文件资料。因此，F－Secure Desktop不仅为个人，也为工作小组提供便利。
　　当然F－Secure Desktop还有其它一些功能，在此不一一赘述，请朋友们自己去发现。 (福建　陈青松)
