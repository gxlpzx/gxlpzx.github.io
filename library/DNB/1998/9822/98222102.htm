来！试一把NWLITE 1.0

　　NWLITE 1.0是什么东西?它的全称是NOVELL NETWARE LITE 1.0。属于一种局域网络软件。如果你曾经是一位NETWARE 3.X或4.X的用户，你一定已体会到了局域网络的优越性，通过服务器，可以实现文件数据传输啦、共享啦，真方便。
　　假如你单位已经安装了局域网络。有空想和同事玩玩“越野赛车”游戏轻松轻松。可是此时，公司的服务器没开，你的程序运行到NETX.EXE就止步不前了，你也只有在那儿傻呼呼干瞪眼。有3个同事要求“参战”，可是，他们的机器既没有光驱，硬盘也太小装不下WIN 95，这这这，怎么办?
　　别急，待我用NWLITE来搞定。首先，确保联网机器都安装了网卡，且网线已通。调出NWLITE－V1.0软件，运行“INSTALL”。有几项选择，你可以选择本机为服务器、站点或服务器兼站点(也就是MAKE THIS MACHINE A SERVER AND A CLIENT)，按要求输入服务器名(该名字随意)，然后几个 YES、NO、OK!服务器装好了。这么简单?!当然是啦。一般来说，只需要安装一台服务器，其他站点选择MAKE THIS MACHINE A CLIENT就可以了。装好之后，系统会自动生成批命令STARTNET.BAT。从“兼职”服务器上启动STARTNET.BAT，(不过别忘了在这个批命令中(SUPERVISOR)登录 (LOGIN)，如果你曾安装过NOVELL 3.X或4.X网，就一定会驾轻就熟。你要是不熟练也没关系，键入NET HELP XX(其中的XX指相关网络命令，如LOGIN、MAP等)，各种命令的使用和格式即一目了然，十分方便。创建完毕，选择MAP项，把服务器硬盘C：映射成驱动器STARTNET，登录，映射L：和M：，然后敲入L：，哇!别人的硬盘和自己的一样好用。再试一下M：好，有了一个共享光驱。在光驱中放入“越野赛车”碟，在每个站点上分别安装它。要是工作站硬盘不够，安装在服务器L：上也行。安装完毕，各自启动游戏，(既可在站点也可在服务器上启动)真有意思，几台机器共用一个光驱、一个硬盘玩同一场赛车。
　　如果你的电脑水平一般，那么用网络批命令将会非常方便，建立以下这个 STARTNET.BAT批命令：
　　C：
　　CD \NWLITE
　　LH LSL.COM
　　LH NE2000.COM
　　LH IPXODI.COM A
　　LH NETBIOS.EXE
　　LH SERVER.EXE
　　LH CLIENT.EXE
　　 NET.EXE LOGIN SUPERVISOR
　　NET MAP L:CDRIVE
　　NET MAP M:DDRIVE
　　该命令将完成从DOS登录到NWLITE网络的过程，其中的LH SERVER.EXE一句为服务器专用，一般站点不用。对于其他用户，只需修改用户名一项。启动NWLINE之后，就能实现DOS下的资源共享，也可以兼容其他网络程序，比如网络版UCDOS之类。NWLITE非常适合于没有服务器且使用DOS系统的小网，比如办公室网络或家庭网络。在有服务器的网络中，一旦服务器关机或有故障，它也同样显示得很有用。整个软件500多K，简真太袖珍了。怎样?一起来学 NWLITE 1.0吧!

　　(广西　班兴宇)
