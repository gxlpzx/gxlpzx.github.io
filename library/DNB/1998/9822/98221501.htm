浏览加速器大比拚(下)

　　各有千秋——欲说选择莫困惑

　　(4)Speed Surfer
　　目前版本:3.2(Demo)
　　上网时间:97－07－10
　　软件出品:Kiss Software Corp.
　　注册费用:30美元
　　运行环境:Windows 3.1或者Windows 95
　　资源需求:8M内存，3M硬盘空间
　　配用浏览器:3.0及以上版本的IE系列或2.0及以上版本的NN系列
　　下载对象:speedtr.exe
　　下载大小:1.9M
　　下载链接：在download.com站点以speed surfer为关键词搜索得到
　　[功能概览]
　　Speed Surfer(以下简称SS)是一个功能比较单一、不具有离线浏览和缓冲区管理功能的浏览加速程序。不过，在预取与当前网页有链接关系的网页内容上，它工作得颇为出色。由于将自己安装成了代理服务器，所以它使用起来要麻烦一些。
　　[工作描述]
　　正常情况下SS启动为任务栏上的图标，点击图标可打开一个状态显示窗口， 列出它已经预取来的或者准备预取的URL链接。因为不能像在使用GG时通过单击链接将相应的网页内容显示到浏览器窗口中，所以列出的链接并没有多少实用价值。预取来的文件先是暂时存放在SS自己的缓冲区里，而后再转换到浏览器的缓冲区中，因此所有预取但尚未查看的网页内容在浏览器打开新的站Web点之后就作废了。
　　[综合评价]
　　使用过程中发现，该浏览加速程序在装载链接到当前网页上的网页内容时，虽然能提高速度，但是提高的幅度十分有限。
　　(5)Surf Express
　　目前版本:1.1.2(Demo)
　　上网时间:98－04－16
　　软件出品:Connectix Corp.
　　注册费用:34.95美元
　　运行环境:Windows 95或者4.0及以上版本的Windows NT
　　资源需求:8M内存，12M硬盘空间用于安装和工作
　　配用浏览器:3.0及以上版本的IE系列或3.0及以上版本的NN系列
　　下载对象:Surf_Express_1.1.2_try.exe
　　下载大小:2.0152M
　　下载链接:在download.com站点以surf express为关键词搜索得到
　　[功能概览]
　　Surf Express(以下简称SE)以其出色的搜索功能而显得与众不同。对于已经访问过的网页，SE能够提供惊人的加速效果。它的缺点是不预取当前显示在浏览器窗口中的网页里包含的指向其它网页的链接内容， 而是仅仅预取用户的FVP，这样就使得它和GG及NA等同类程序比较起来显得逊色不少。由于它不但作为浏览器的代理服务器来工作，而且关闭以后还不能自动恢复代理设置，所以用户要么总用它，要么一次也别用。
　　[综合评价]
　　如果为了换换使用浏览加速程序的口味，欣赏一些与众不同的功能，Surf Express是极好的选择。如果只是为了加速浏览，不用它而改用其它程序更好。
　　(6)PeakJet
　　目前版本:1.56(Demo)
　　上网时间:97－05－28
　　软件出品:PeakSoft Corp.
　　注册费用:29.95美元
　　运行环境:Windows 95或3.51及以上版本的Windows NT
　　资源需求:16M内存，安装后占用4M硬盘空间
　　配用浏览器:3.0及以上版本的IE系列或2.0及以上版本的NN系列
　　下载对象:PJ15.exe
　　下载大小:3M
　　下载链接:在download.com站点以peakjet为关键词搜索得到
　　[功能概览]
　　这是个完全用Java语言写成的程序，它预取和当前网页有链接关系的网页内容，并自动定期刷新保存在缓冲区中的网页内容。它可以通过选项设置功能来实现将预取的不同站点的内容保存到不同的缓冲区里。这个功能允许用户随时刷新预取的不同站点的内容。使用表明，PJ对于许多站点的浏览都具有明显的加速作用。不过PJ限于其Java语言界面，配置起来比较复杂，而其装载网页的风格也使得其在实用性方面和GG、NA以及WE相比有很大差距。
　　[综合评价]
　　PJ是同类程序中最不实用的，除非无法得到其它的浏览加速程序，否则没有理由使用这个憋脚的家伙，它带来的麻烦远多于好处。
　　(7)NetAccelerator
　　目前版本:1.1(Demo)
　　上网时间:97－08－13
　　软件出品:IMSI公司
　　注册费用:30美元
　　运行环境:Windows 95
　　资源需求:8M内存，8M硬盘空间
　　配用的浏览器:3.01及以上版本的IE系列或NN系列
　　下载对象:Na10.exe
　　下载大小:3.5278M
　　下载链接: 在http: ／／www.imsisoft.com或在download.com站点以netaccelerator为关键词搜索得到
　　[功能概览]
　　NetAccelerator(以下简称NA)是目前最简单、问题最少的浏览加速器程序, 也是唯一跳出将自己安装为代理服务器以及需要用户通过选项菜单来设置这个俗套的程序。安装NA以后，用户的浏览操作和没安装时一样，唯一的变化就是速度加快了。它主要在后台预取和当前网页有链接关系的其它网页及用户经常访问网页(FVP) 的内容。和GG比较起来，它缺少某些诸如离线浏览和控制方面的功能。
　　[工作描述]
　　启动以后即打开一个包含有几个按钮的控制板，这些控制板是用来启动IE或者NN浏览器。
　　事实上，在还没有单击启动按钮之前，NA就开始“暗地里”运行浏览器程序(姑且称为后台浏览)，单击浏览器启动按钮选定浏览器后，就再运行一个“公开的”浏览器程序(可称为前台浏览)。在进行前台浏览的同时， 后台浏览就预取和当前网页有链接关系的网页以及FVQ网页的内容，并将其保存在浏览器的缓冲区中以供前台浏览，进而提高浏览速度。当退出NA同时关闭前台浏览器后，会看到调制解调器依旧在工作。这就是NA的奇妙之处。它的另外一个与众不同之处在于不以代理服务器姿态出现，所以不需改变浏览器的任何已有的设置。
　　在NA的控制板上有一个Pause按钮，它可以用来暂停网页的预取工作，以便利用整个连接的带宽资源来处理E－mail收发或者其它非浏览性网络事务。使用中体会到，NA唯一的问题就是显示在屏幕上的控制板在某种程度上影响浏览，万幸的是可以将其最小化到任务条上。
　　[综合评价]
　　如果追求实用和朴实无华，NetAccelerator是一个较好的选择。它功能专一，简单易用，是初级用户仅次于NetSonic的选择。
　　(8)Java Booster
　　目前版本:1.0
　　上网时间:97－07－08
　　软件出品:Innovative Software GMBh
　　注册费用:29美元
　　运行环境:Windows 95或者3.51及以上版本的Windows NT
　　资源需求:4M内存,5M硬盘空间用于安装和工作
　　配合的浏览器:3.0及以上版本的IE系列或者2.0及以上版本的NN系列
　　下载链接:在download.com站点以java booster为关键词搜索得到
　　下载对象:JB10e.zip
　　下载大小:1.6874M
　　[功能概览]
　　Java Booster 1.0a(以下简称JB)和其他浏览加速器不同，仅仅加速已经访问过的网页，它使用经过智能化设计的可搜索缓冲系统来取代浏览器的缓冲作用。用作离线浏览器的时候，JB肯定是最好的选择，因为它不仅能够无限制地在其缓冲系统中保存指定的网页，而且还能够保存所有连接到这些网页的Java Applets，这样用户就不用下载这些Java Applets。
　　JB的出众之处是其智能检索功能。利用内置的文本检索功能，它可以列出缓冲区中包含指定文本的网页，单击某个链接即可将相应的网页内容显示在浏览器窗口中。
　　[工作描述]
　　使用JB的前提是改变浏览器的代理设置，以便让浏览器从JB取得网页数据而不再直接访问Internet。此后就只有先运行JB，而后才能通过它来访问Internet。JB通过一个小型控制板窗口中的按钮在离线和在线状态之间切换，也可通过按钮来打开显示在浏览器窗口中的选项设置窗口。
　　[综合评价]
　　它仅仅是通过缓冲Java Applets来节省在线时间的加速程序，远远不能满足复杂的浏览加速需要。突出之处是离线浏览功能。

最后抉择——浏览英雄排座次

　　以上从使用角度出发评价了目前流行的8种浏览加速器程序。 不难发现，大多数浏览加速器程序在完成基本的加速功能的同时，都或多或少带有其它一些独特的功能, 用起来的感觉是各有所长，选择起来还真的难以取舍。下面再将这些程序的基本性能归纳如下。
　　从表中可以看出， 就基本的加速性能来衡量，Go ahead Got it属于“五星级”，而NetSonic则属于“四星级”。如果从上述这些流行的浏览加速器程序中选择一半来使用的话，推荐的选择顺序为：Go ahead Got it!>NetSonic>WebEarly>NetAccelerator。
