电脑运行状态观察器---SizeWise SE v1.02

　　 你是不是有过这样的经历：将编辑的文件存盘时，电脑告诉你“磁盘空间不够！”。玩游戏正在兴头儿上时，电脑告诉你“内存不足，程序无法运行！”，你会想：要是提前知道会多好。
　　如果你有了SizeWise SE v1.02，随时观察你电脑的运行状态，无论是在游戏中还是在程序设计中，你只要将鼠标向右下角挪一下，就会看到你想看到的东西，方便不？
　　SizeWise SE v1.02支持 Windows 95/98 及 Windows NT4.0 ，安装完后，在Windows的任务栏的右方会出现一个小的绿色圆球，当鼠标指向它时，会出现当前硬盘的使用状态。用鼠标右键击后，你可以添加其它功能，可以添加对其余硬盘、物理内存、虚拟内存、交换文件、系统资源、光盘（如果是只读光盘，那个对应的小圆球会变成蓝色的）的监控。
　　除此之外，它还有一项最实用的功能：当剩余资源小于你指定的最小限度时，它会提前发出报警，它自带两个WAV文件，当然，你可以选择你自己喜欢的声音文件替代它。
