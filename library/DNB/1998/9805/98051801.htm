IE 4.0使用技巧四则

　　一、如何使IE4.0启动时显示你喜爱的画面
　　常在网上跑的人都有这样的印象：每次启动 IE 4.0 的时候，总是显示一张老网页，上面的内容千天一面；也许经常查阅某家新闻单位的联机日报或是极其欣赏某个主页的精美设计，却不得不至少进行一次跳转才能显示出来。能否一启动IE 4.0就能按你的要求显示出你喜爱的画面，IE 4.0可以做到这一点，方法是：
　　1. 启动 IE 4.0并跳转到你所欣赏的页面；
　　2. 打开“查看”菜单的“Internet选项"命令；
　　3. 在“Internet 选项"对话框中选择“常规"选项卡；
　　4. 在“主页"框中单击“使用当前页"按钮。此时可以看到地址栏中的内容变成当前所在页的地址（URL）；
　　5. 单击“应用"按钮，再单击“确定"按钮。
　　这样，以后再启动 IE 4.0 时，第一眼见到的，就是你想看到的画面。
　　二、如何设置浏览内容的分级审查功能
　　有相当多的人在还未接触 Internet 时就已经听说过网上的花花世界里有许多不合适的内容，也许有人因此而不敢上网：家里的孩子辨别能力和自制力太弱，在那样一个开放的世界里，无法保证孩子的身心健康。对于这个问题，世界上各个国家都制定有相应的法律条文以禁止污七八糟的内容上网，各种媒体都在呼吁对其加以防范。许多浏览器软件也都具有内容分级审查功能，启用这项功能，即可按照适当的标准（如暴力、性、裸体、语言等的不同分级）对网上所显示的内容加以限制。IE 4.0 中该功能的启用分两步，一是设置，二是启用。具体方法是：
　　1. 打开“查看"菜单中的“Internet 选项"；
　　2. 在对话框中选择“内容"选项卡；
　　3. 单击上方“分级审查"框中的“设置"按钮；
　　4. 此时系统会提示继续操作前需提供监护人密码。输入密码并加以确认；
　　5. 系统显示“分级审查"对话框。默认显示的是“分级"选项卡；
　　6. 根据屏幕提示设置相应的分级。如对于“暴力"可设置为“打斗"，也可设置为“带血腥的杀戮场面"；
　　7. 选择“常规"选项卡，确定对未分级站点的处理方式、监护人的特权，并可更改口令；
　　8. 选择“高级"选项卡，用以更改或添加分级系统，一般用户无须变动；
　　9. 单击“应用"按钮，再单击“确定"按钮。
　　这样便设置了分级系统。接下来重复上述的步骤 1 和步骤 2，然后单击“分级审查"框中的“启用"按钮。系统提示你提供监护人密码。输入密码并单击“确定"按钮后即启用已经设置好的分级系统，此时系统会给出有关的建议信息。单击“确定"按钮。
　　此后，只有能提供密码的人才能访问那些满足分级条件的站点。注意记好密码，忘掉密码是件很麻烦的事。
　　另外禁用该项功能时的操作类似于“启用”该功能，如上所述，只是要单击“禁用"按钮。
　　三、网页的打印方法
　　打印网页的方式很多。可以单击工具栏上的“打印”按钮，也可以选择“文件"菜单上的“打印”命令，还可以按下 Ctrl－P 组合键。但可能你还不知道，不必访问也可以直接打印网页。这对于熟悉网页内容的人，需要将诸如联机教程材料之类的内容文档化的时候非常有用。方法是：
　　1. 转到所需的网页；
　　2. 用鼠标右键单击所需的 URL（通用资源定位器）；
　　3. 在快捷菜单上选择“打印目标"命令；
　　4. 在“打印"对话框中设置打印范围、打印份数、是否打印所有链接页等信息；
　　5. 单击“确定"按钮。
　　注意：这种方式的缺点是直到打印完毕才能知道该页的长度。
　　四、巧输难记的长地址
　　对于很长很难记的地址，如果还记得开头几个字母，那么启用“自动完成(AutoComplete)功能真的可为你减少输入的麻烦。
　　1. 选择“查看"菜单中的“Internet选项"命令；
　　2. 选择“高级"选项卡；
　　3. 选中“浏览"下的“使用自动完成"复选框；
　　4. 单击“应用"按钮，再单击“确定"按钮。
　　现在在地址栏中键入地址，比如：www.microsoft.com。浏览到其他站点后再回到“地址栏"并键入www.micr，自动完成功能将自动填充剩余的内容为：www.microsoft.com。
