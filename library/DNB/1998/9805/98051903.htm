网络吸血鬼Net Vampire

　　相信提到断线续传软件,大家一定会想到GetRight和Go!Zilla这两个大哥级软件,它们强大的功能确实赢得了广大网友的青睐,我也一直使用GetRight。但近日在网上闲逛听闻有一新的断线续传工具Net Vampire(网络吸血鬼)非常好用。Net Vampire(简称NV),好酷的名字－－网络吸血鬼,连夜找来一试,果然不错。
　　我们下载软件最关心的是它的速度如何,在这一点上,NV是我所见过下载软件中最快的。以前我使用GetRight配合33.6kbps的Modem,一般下载速度是每秒3.5kb左右,但现在用NV却可以达到每秒5.5kb左右,有时甚至能达到每秒7kb左右,实在是令我喜出望外,真不愧为网络吸血鬼,它的确像贪婪的吸血鬼一样,能从有限的网络带宽中吸取最大量的比特。
　　Net Vampire是一位俄罗斯人用Delphi2.0编写的,是一个免费软件。我写文章时最新版是2.4版,一共有1865Kb。大家可以从http://www.kulichki.com/～vampire/处下载。NV支持FTP和HTTP断线续传,它有很强的抗干扰下载能力,特别是在网络繁忙、连接不可靠时,它能完成其他软件所不能完成的工作。NV界面简洁，使用非常方便,启动NV之后,NV会生成一个正方形的小图标,始终浮在屏幕的上面,当你看到想下载的文件后,只要按住鼠标左键拖动文件放在图标上(这一点和GetRight有类似之处),这时图标会很形象地打开一道门,你把文件扔进去,下载就开始了,在下载过程中,它很清楚地显示出下载的状态信息。特别值得一提的是它独一无二的柱状显示图,从中可以可以很直观地看出下载过程中的速率变化情况。NV界面分为上下两部分,上部分是下载软件任务表,分别列出了各个软件下载任务的完成情况、软件的地址、开始时间、文件大小等信息。下部分是下载过程中的状态信息记录表(log),包括NV在连接过程中客户端的操作命令和服务器的应答信息。
　　下面简单介绍一下它的主要菜单项的操作:
　　File:
　　　exit　退出。
　　Job:
　　　start　现在立即开始下载,不管时间表设定如何。
　　　start on schedule　到设定的时间开始下载。
　　　stop　停止下载。
　　　add　打开工作设定表(job setting)添加新的任务。
　　　edit　打开工作设定表(job setting)编辑已有任务的参数设定。
　　　delete　从任务表删除中删除任务。
　　　clear log　清除下载状态信息。
　　　save log　保存下载状态信息。
　　option:
　　　job defaults　预先设定参数,包括下载文件存放地址;当连接不上时尝试的次数;FTP服务器的登录帐号和密码等。
　　　histogram option　柱状图的设定,包括设定多少字节为100％的基准值;波形每几秒显示一次;柱状图为什么颜色。
　　　system option　设定新的下载任务以什么方式启动;决定NV启动时是否检查任务表中所列任务的状态;还可以设定当下载任务完成或失败后是否给其他工作组发送信息,在windows95中要完成这项工作需先启动winpopup.exe。
　　　show drop basket　当选中此项后,每次启动NV后,在屏幕上就会出现一个小图标(drop basket),每当你想下载时,就把待下载文件拖放到图标上就行了。
　　　detailed log　当选中此项后,在状态信息记录表中就会记录所有的客户端的操作命令和服务器的应答信息。
　　　all jobs in parallel　当选中此项后, 多个下载任务可以同时进行。
　　help:
　　　提供了详尽的帮助信息。
　　在Internet上,几乎每时每刻都有非常不错的软件出现,只要你仔细去找就一定能找到好东西,为自己上网带来方便。对了,要是你找到什么,请告诉我一声,我的Email地址是x.gao@usa.net。
