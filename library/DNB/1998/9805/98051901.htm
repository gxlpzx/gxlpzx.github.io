来自北欧的小巧玲珑浏览器——Opera

　　Netscape和Microsoft 的浏览器之争，使人感到似乎上Internet,非Netscape即MS IE。而且，浏览器越做越大，霸道的MS IE4.0，一装就是六、七十兆，并且，在你上网时不断地吞噬你的硬盘容量，你不得不两、三天清理一下，否则，十天、半月下来，你就根本无法上网遨游了，因为，你的硬盘已经没有空闲之地了。
　　本人近日上网漫游时，觅得一小巧玲珑、多快好省的Web浏览器——Opera,此软件早于1994年诞生于北欧之国——挪威的奥斯陆，最新版是V.3.0 Beta版，有32位，也有16位的，照顾到所有的操作平台。免费试用期是90天，网上可直接注册。详情和下载网址：http://opera.nta.no和http://www.operasoftware.com,你还会看到Opera用户和专家的赞誉之词。
　　下面就把这个惹人喜爱的小东东介绍给广大网友。
　　（一）主要特性：
　　1. 小巧玲珑：Opera卸载后为自释放软件o300e32.exe(32位for Win95)，大小为1.08M，安装后，也仅有2M。决不浪费你的宝贵领地，比起MS IE或Netscape，可真谓"小家碧玉"。
　　 2. 界面清晰：Opera的界面友好、清晰，菜单10个，工具快捷按钮14个，操作便利。浏览界面窗口一分为三，左部上下是书签窗，右部是浏览窗，指点书签极其方便。分隔条可自由移动，以独霸窗口，但是，书签栏会占据2CM的空间，以便插、看。底部的浏览提示格分别提示数据传送的大小、进程和传送率。并有设定传送数据文字或文字兼图像，以及色彩的设定图示按钮，操作便利快捷，即点即现，一目了然。
　　3. 多快好省： Opera虽然只有2M，但功能众多，几乎拥有浏览器必备的各种功能。最大的优点还支持多窗口同步浏览，即窗口平铺、重叠功能。数据下载速度很快，特别是JPEG的图片下载速度可谓之最。Opera支持JPEG、GIF和BMP，也是一个很好的看图软件。
　　4. 和二为一： Opera的书签功能极强，能兼容MS IE和Netscape的书签，分为两个文件夹，你可自由加载或浏览它们的书签，或者只用Opera也可以。＋（二）主要设置和使用方法：＋因为是打包软件，向导式傻瓜安装法，安装十分简便。下面介绍Opera的一些主要功能设置和用法。
　　1.File:文件
　　New　新建浏览窗口
　　Open... 打开本机的文件
　　Search... 查找
　　Print... 打印
　　Print Setup... 打印设置
　　2.Navigation 浏览导航
　　Previous 前一页
　　Home　主页
　　Set Home...设置缺省起始页
　　Open Remote... 打开新的网址
　　And Active To Host List 加入列表
　　3.Lists 书签列表
　　Auto Inported Item> 加入书签到MSIE或NETSCAPE的书签列表
　　4. Preference 参数设置
　　Generic... 一般总体设置，有个人信息、地址、电话、EMAIL等。
　　Link Presentation... 连接状态设置，主要是访问过的页面颜色，取消时间等。
　　Mail...你的EMAIL情况设置
　　Cache...你的硬盘上的CACHE文件夹设置
　　Proxy Server...服务器地址设置
　　Sound...声音设置
　　JPEG Image....图片设置
　　Security.....安全设置，可设定你的Opera密码等安全设置
　　5.Windows 窗口
　　Special Windows > 可启动新窗口，查看CACHE、卸载的和看过的网页及路径
　　Cascade　窗口平铺
　　Tile　窗口层叠
　　（三）注意事项：
　　1.Short Cut快捷键：Opera拥有众多的快捷键，熟悉Office办公软件的朋友，一定会得心应手，加上Opera图示化的工具按钮，操作Opera更是易如反掌。帮助文件里有快捷键的详细说明。
　　2.关于Email：Opera尽管也提供了Email的功能，但是只能发送，而不能接收。要接收Email就要起用别的Email软件，这似乎是Opera的缺陷，但对于只有2M的Opera来说，能够发Email已经不错了。
　　（四）精彩之处：
　　1.Stop按钮：脱机时，你也许看不到所有浏览器上都有的Stop按钮，是Opera作者的失误？不！因为，没有人会在离线查看时使用Stop按钮！只有上网感到速度在"耗钱"时，才会猛按STOP。Opera的作者也深谙此道，所以，只有在线畅游之时，才会在底部的网址栏旁出现红灯的图标，便于用户停止Opera搜索。这也许正是Opera广受北欧网友喜爱的缘故。
　　2.Zoom缩放窗口：在浏览JPEG图象时，可以利用Opera右下部的Zoom缩放窗口。Opera提供了从20％一直到1000％的缩放比例，图像清晰度一点也不亚于专业图像软件。爱好网上浏览图像的网友，使用Opera一定会大叫过瘾！
