Photoshop5.0新增功能一览 

　　经过了几个月的测试后，Adobe公司发布了一条令全世界都为之振奋的消息：Photoshop5.0正式发布。初露锋芒的Photoshop5.0究竟能带给我们什么新的感受呢?让我一一道来。
　　Photoshop5.0的操作界面与它的小弟Photoshop 4.0相比没有什么大的差别，不过如果你就此判断说Photoshop5.0没有什么改进，那你就大错特错了，Photoshop5.0与以前的版本相比，总的来说增加了7个令人振奋的新功能：
　 1.无限的UNDO和REDO功能。使用过Photoshop的朋友应该都有这样的体会，在过去的版本中，Photoshop只提供了一次UNDO和REDO功能，常因某些操作无法恢复而感到不便。现在有了简单、易用而又功能强大的HISTORY PALETTE（历史操作面板），你的所有操作都一一陈列在面板上，如果对其中某步操作不满意，只要将代表该步操作的图标用鼠标拖拽到右下方的垃圾桶中即可，就这么简单、神奇。
　　2.灵活、丰富的文本编辑、处理功能。Photoshop5.0提供了多种文字格式，能让你更加轻松地创建出高品质的文字。你现在可以对同一层上的文字分配不同的字体风格，单击按钮，就可以直接将文字设为水平排列或垂直排列；更妙的是还提供了堆栈修改功能，当双击文字层，对文字进行修改后，Photoshop5.0会自动将该文字在修改之前的所有操作，作用于修改后的文字，这样你就可以避免大量的重复劳动。同时Photoshop 5.0也增强了对中文、日文这类双字节文字的支持。
　　3.更加悦目、精确的颜色管理和校正功能。Photoshop5.0提供了多样的色彩管理和颜色产生的解决方案，足以保证专业的输出品质。其中包括GAMMA CONTROL PANEL (Gamma控制面板)、NON－MONITOR RGB（RGB色彩的监视器无关性）、ICC PROFILE SUPPORT（ICC工业标准的支持）、DOT GAIN CURVES（点增益曲线）。并提供了对48位和64位真彩色的支持；新增了多种图像色彩校正命令：CHANNEL MIXER（通道混合器）、IMPROVED CURVES（增益曲线）、REFINED CONTROL（微调控制）。 这些专业的色彩管理和色彩校正功能，能在打印时或在网络上提供高品位的图像质量。
　　4.新增加了SPOT－COLOR CHANNEL（SPOT色彩通道）。在广大用户的要求下，Photoshop5.0终于提供了一个令人激动的全新的色彩特性，即能为任意图像创建一个SPOT色彩通道。利用这种特性，你可以在打印输出时加入混合涂料、金属油墨和其它特殊的色彩，甚至可以为CMYK图像加入凹凸镀金材质（ bump plates），以产生比四色油墨更丰富的色彩。也可以直接从Photoshop5.0的文件或以DCS2.0格式输出的文件中将SPOT色彩打印出来。
　　5.增加了大量的自动流程和批量处理工具，简化了以往复杂的工作。Photoshop5.0提供了大量的新功能来加速图像作品的制作进程，并提供了各种更有效的新方式来对其进行处理。其中又包括AUTOMATION PLUGINS（精灵向导插件）功能，能一步一步指导你完成某项工作；3D TRANSFORM PLUGINS（三维变形插件）能轻松在二维平面上对三维图形进行处理；LIVE LAYER EFFECT（动态层处理）工具，能自动对层进行处理；增强了对ADOBE ILLUSTRATOR、EPS和PDF文件的支持。
　　(1)AUTOMATION PLUGINS（精灵向导插件）。Photoshop内置了多种的AUTOMATIONS PLUGINS，如RESIZE IMAGE WIZARD（改变图像尺寸）、 EXPORT TRANSPARENT IMAGES WIZARD（输出透明图像）、BATCH COMMAND、CONDITIONAL MODE CHANGE COMMAND（色彩模式改变）、CONTACT SHEET COMMAND、FIT IMAGE COMMAND（图形匹配）、MULTI PDF TO PSD COMMAND（多页PDF文档与PSD文档的相互转换）。
　　(2)3D TRANSFORM PLUGINS（3D变形插件）。Photoshop5.0内置了一个功能强大的3D PLUGINS，它能让你在2D平面上对3D物体实时地进行三维变形，这样就能直接对3D物体的大小、位置和定向进行实时地调整。
　　(3)LIVE LAYER EFFECT（动态层处理）工具。大家都知道Photoshop4.0提供的层工具很少，因此这次Photoshop5.0特意增强了它的层的处理功能。只要用鼠标单击菜单上的命令就可以快速地在层上实现很多以前在第三方厂家生产的PLUGINS上才能找到的特效。这些特效都是制作一些文字特效时常用的命令，有了它们，制作网页时就不必到处找像KPT之类的PLUGINS！Photoshop5.0的动态层处理特效的功能还远不止这些，它们的强大功能在于“动态更新”。举个例子说，当在某一层使用了DROP SHADOWS和BEVEL特效后，继续在这层上进行绘制时，Photoshop 5.0会自动地将修改后的图像显示为DROP SHADOWS和BEVEL，而不必重新对其进行层处理。其实这项功能并不是Adobe公司的独创，用过3D MAX2.0的朋友一定都知道在3D MAX2.0也有同类的功能，被称为MODIFY STACKS（修改堆栈）。
　　(4)在Photoshop5.0中，加强了对Adobe Illustrator和Generic Encapsulated Postscript（EPS）的支持，并且，它能够打开多页的PSF文件。这使得Photoshop可以对各种程序的文档进行无缝地共享，而且它还能将PDF文件置入高端的打印流程中。
　　6.更加精确、有效、易于控制的选择、测量工具。Photoshop5.0对其工具栏也进行了改进，增加了许多新的工具。包括MAGNETIC LASSO（磁性套索）、MAGNETIC PEN（磁性笔）、FREEFORM PEN（自由笔）、MEASURE TOOL（测量工具）、COLOUR SAMPLERS（颜色采样）。这些工具使得Photoshop在精确性和易控制性上比其它图像处理软件更进了一步。
　　MAGNETIC LASSO（磁性套索）。在Photoshop4.0中套索工具是一个功能强大而又非常好用的一个工具，不过它还是常常无法让你轻易体会到美国西部牛仔的感觉──经常无法正确选中你所预想的区域。Photoshop5.0中的这个工具就能神奇地解决这个问题，它能根据锁定对比度和色调来精确定位选择区域，轻易地将前景物体从背景中分离出来。
　　MAGNETIC PEN（磁性笔）。磁性笔的作用很像磁性套索工具。它会根据图像的对比度发现并锁定图像的边缘，不过它并不对其进行选择，它只是沿着该选择图像的边缘产生一条BEZIER曲线。利用它能轻松地创建一条路径，并将它输出应用于Illustrator。
　　FREEFORM TOOL(自由笔)。它实际是从Adobe Illustrator借用过来的。用它能够轻松地描绘出某个图形的边缘，产生一条BEZIER曲线。
　　MEASURE TOOL（测量工具）。你可以用MEASURE TOOL来精确测量或比较两物体之间的距离或角度，这就填补了以往Photoshop中没有测量工具的空白。
　　COLOR SAMPLES（颜色采样）。在Photoshop5.0中，COLOR SMAPLES允许你选择四种不同的颜色采样，并且在INFO PALETTE（信息面板）中能实时地显示出给颜色的命令数值。
　　7.为第三方厂家提供了更加开放的PLUGINS开发空间。在Photoshop5.0的光盘中免费提供了软件开发套件(SDK)和最新的API，任何人都可以利用它们作出FOR Photoshop5.0的PLUGINS。
