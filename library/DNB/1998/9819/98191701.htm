玩的是主页 赚的是美元

　　对个人来说，上网赚钱是件很惬意的事。“赚不到钱？咱压根儿就没想过这个。上网不就是为了自己玩的痛快吗？”而机会真来了的时候，顺便赚点儿钱则更加快乐。
　　现在公认的利用因特网能赚到钱的方式不是很多，大概给人做广告收赞助费是最流行的了，此外还有收取介绍费用以及实实在在卖东西等。
　　想在网上赚钱，前提条件是得有个属于自己的主页。如果这个条件都没有，恐怕只能上论坛发广告了(但估计很快就会臭名昭著)。
　　先说说在网上给人做广告收赞助费吧。有了自己的主页以后，你就可以去寻找Sponsor(赞助人)了。现在国内的厂商可没有兴趣在个人主页上做宣传呢！不要紧，在国外，有很多这样的地方，不管你是谁，只要你有自己的主页，他们就会专门对你提供赞助，具体的做法是：你只要到该广告公司申请一下，这样你就会得到一段HTML代码，然后你将这些HTML代码放在你的网页上，这样你的网页上就会出现该公司的链接和广告图档(Banner)，而每当这些图档在你的网页上显示一次或每当有人点击这个Banner时，你就会有所收入了。依照不同的广告公司，这些广告每显示一次或点击一次，广告公司所支付的金额也都不一样，这些支付金额通常都在1至5美分之间，偶有高达10美分的。如果你是一个网页编制高手，编出的网页受欢迎的话，说不定还能把你的上网费赚回来，来个以网养网呢。当然，赚钱的形式是不同的，有的地方每个不重复的 IP 地址刷新一次(One Impression)就给你几美分(如 Commonwealth Network)，而有的地方则是必须有人点击(Click)了广告才给钱(如 ValueClick.EAds)。你在选择的时候一定要注意这些，以免给人白做了广告。等到了一定金额的时候，你就可以收到从国外发来的支票了！至于金额是多少，由你自己注册时决定，别忘记留下地址，不然可收不到支票呀！
　　要想通过网上给人做广告收赞助费来赚钱，前提条件是：你的主页必需要有固定的访问量，这对个人主页设计者来说，不是件很容易的事。所以多在网上做链接，不失为一个好主意，“酒香也怕巷子深”嘛！现在有很多做广告的网友在网上结成联盟，每次上网都先互相访问。这绝对是一个办法，只是太专业、太商业化些。
　　在网上收取介绍费用，倒是比较像那么一回事。钱是这么来的：如果有人通过你的主页到了一个协会或者是网上商场什么的，你就可以得到一部分佣金。说得详细一些，举个例子：如果你是 CD Now (网上 CD 商店)的一个会员，你就可以在自己的主页上放一个它的标志(当然，里面链接到可以识别你的 ID 的 CGI 程序上)。然后，凡是通过这个标志到 CD Now 购物的人，他买 CD 的钱中，有 5％是要归你的。很容易吧？可惜在国内还很少有人去国外买东西。还有一种是介绍加入协会的，你可以提取一部分新会员的会费，关于这种服务，基本上涵盖了各个方面，你可以选择适合自己读者群的加入，相信会有效果的，而且，这种方式赚得比较多。
　　此外，还有不少稀奇的方式，甚至有专门通过自己主页提供信息，让人阅读来赚钱的，真是八仙过海，各显神通。
