资源管理器Drag And File Gold简介

　　看到Windows 95的资源管理器不够好，就有许多人造出更好的资源管理器想要替代它，Drag And File Gold就是其中的一个。
　　Drag And File Gold属于共享软件，目前的版本是4.17。文件的下载压缩包为自解压EXE文件，大小为2.06M。双击该文件即开始安装。安装过程十分简单，只要选定目标目录即可。安装完毕后会生成两个程序组，点击Drag And File Gold组中的Drag And File项即可启动该软件。
　　Drag And File Gold的界面和Windows95的资源管理器有些类似，标题栏、工具栏一应俱全，但细看就会发现功能要丰富得多，而且把文件管理器的很多优点都保留了下来，比如大家在标题栏Window/New Window一项中就可建一个新窗口，然后选择Tile Horizontally（水平平铺）或者Tile Vertically（垂直平铺）就可实现同屏显示不同磁盘或目录的文件列表，从而实现在一个视窗内对多个不同磁盘中的文件或目录进行拷贝、移动、删除、查找、建立新目录、打印目录及文档列表等文档管理工作。
　　不仅如此，Drag And File Gold 还有很多极具特色的功能，现简要介绍如下：
　　1.强大的网络功能
　　Drag And File Gold最为出色的地方莫过于此。它不仅可通过网络映射直接管理局域网中的资源，从而把Windows95中的资源管理器和网上邻居完美地合二为一，更棒的是，它可通过内置的FTP工具，通过Internet直接联到FTP服务器上，下载或上传文档，对其进行直接地管理。方法为：在工具栏上点击FTP Session（右起第五个）一项，即可调出Drag And File FTP，在Site Profile卡片中的Hostname一项中填入FTP站台的地址，在Host Type一项中选择登录方式，并对User ID和Password作相应的设置，再点击Connect即可联上远方FTP服务器。根据我的体会，Drag And File FTP在功能上虽和一些专业的FTP工具有一定差距，但在最重要的指标——传输速度上，与专业FTP工具相比并不逊色，且使用起来相当容易上手。具备这些功能，Drag And File Gold对本地磁盘、局域网和广域网上的资源都能方便地加以管理，真正适应网络时代的要求，从这一点上来讲，Drag And File Gold才是真正的“资源管理器”，它代表了资源管理器未来的发展趋势。
　　2.与Winzip的完美结合
　　Winzip连续几年荣登世界共享软件排行榜榜首位置，可见ZIP事实上已成为网络时代的压缩标准。因此，Drag And File Gold给予ZIP文件以充分的支持，其功能不在一些专业的ZIP配套工具之下。在Drag And File Gold里，每个ZIP文件都被视为一个文件夹，你可直接观看ZIP压缩包里的文件列表，并可浏览文件内容，甚至允许你直接运行压缩包中的可执行文件。
　　3.出色的文件过滤能力
　　Drag And File Gold有独特的文件过滤能力。工具栏右起第8项为文件过滤器（Filter files）的图标，在设置卡片里可以设置目标文件的生成时间段和后缀名，这样，当你再观看各个文件目录时，会发现Drag And File Gold将只列出符合要求的文件，其余的文件则被隐去了，这可使你寻找同一类文件时更觉方便、轻松。
　　Drag And File Gold 的程序组里还有一些程序，例如DF Launch Bar、DF Task Bar等。大家可以根据自己的需要来选用。
