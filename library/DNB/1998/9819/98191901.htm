Netscape浏览器使用经验十五则

　　Netscape浏览器(以下简称Netscape)作为目前应用最广泛的浏览器之一，受到越来越多人的欢迎，成为人们网上冲浪的重要工具。许多人由于对Netscape的特性不够了解，未能充分发挥它的作用。下面将我使用Netscape的经验介绍给大家，以达相互交流之目的。
　　1.做好你的个人书签 (Bookmark)
　　将你喜爱的网址收集到你的个人书签上，以方便将来的浏览。Netscape增加书签的方法主要有：
　　·点按工具条的Bookmarks按钮，然后选取 Add Bookmark的选项，即可将喜爱的网址加入你的个人书签。
　　·点按鼠标右键，选取 Add Bookmark选项即可。
　　2.轻松开启 /关闭 3个工具条
　　只要点一下各工具条最左边的开关板，就可以随时开启 /关闭Navigation、Location、Personal三个工具条。如果你希望某个工具条完全不出现在窗口内，可在View的下拉菜单中选取 Hide xxx Toolbar（ xxx代表各工具条的名称）。想要让某工具条再次出现于窗口内，选取 View下拉菜单中的 Show xxx Toolbar。
　　3.使工具条按要求排列
　　你可以按自己的习惯排列3个工具条的上下顺序。只要将光标放在某个工具条上（没有按钮或无功能的地方），直接上下拖拉排列即可。
　　4.利用热键加速浏览速度
　　充分利用Netscape的热键功能，能使你的网上冲浪变得更加轻松如意。常用的热键主要有：
　　·Tab热键　不断按下Tab键，可从一个超链接跳到下一个超链接（超链接的周围会显现出细虚线将它框住），再继续不断往下跳。当跳到你感兴趣的超链接时，按下回车键，就会链接到该站点。
　　·Esc热键　让正在下载中的网页停止下载，或是让不断跳动的动画停止下来。
　　·PgUp热键　可让网页右边的卷轴向上卷一屏
　　·PgDn热键　可让网页右边的卷轴向下卷一屏
　　·↓方向键　一点一点地将网页往下卷
　　·↑方向键　一点一点地将网页往上卷
　　·→方向键　一点一点地将网页往左卷
　　·←方向键　一点一点地将网页往右卷
　　5.快速返回曾经浏览的网页
　　按住工具条上的“Back"键，会弹出一个下拉菜单，记录了你刚才已浏览过的网页。你可以选择回上页或直接跳回曾经浏览的任一页。同样地，按住工具条上的“Forward"键，也可以选择向前跳一页或直接向前跳到任一页。
　　6.充分利用Netscape的搜索功能
　　点按工具条上的“Search"键，你将会链接到网景为你精心制作的搜索引擎大集合网页，包括 Yahoo、 Infoseek、 Lycos、 Excite、 Hotbot等著名的搜索引擎，不必再辛苦地奔走于各大搜索引擎网站寻找资料了。
　　7.清除分栏显示(Frames)的条条框框
　　当链接到由分栏显示(Frames)组成的网页时，可以将鼠标置于某个栏目上，然后按下右键，在下拉菜单中选取“Open Frame in New Window"，使得这个栏目单独出现在另一个新窗口内，扩大浏览的面积，也方便准确地收藏此页为书签，或甚至于将此页做成一个桌面上的快捷键。
　　8.方便使用Netscape的其它网络工具
　　在Navigator浏览器视窗的右下角，有另外一组细长的工具条，排列着 4个小按钮，方便你随时激活 Communicator内的其他网络工具。这 4个按钮从左至右分别可激活一个新的 Navigator浏览窗口、 E－mail工具、讨论组工具、 HTML编辑器。你可用鼠标点按在此工具条的左缘，将此工具条拉到屏幕的任何部位。或者你也可以让这个工具条竖直放置，或保持在桌面上：用右键点在此工具条的上方，然后在下拉选单中选取“Vertical"，或 “Always On Top"即可。
　　9.设定个人专用的上网环境 (Profile)
　　如果你必须与其他人共用一套 Communicator软件，你可以在启动 Communicator之前，先设定好你个人专用的上网 Profile，让Communicator提供你个人专用的书签夹、E－mail信箱、订阅的 News / BBS等信息，而不会与他人的资料混杂在一起。设定你个人专用的 Profile的方法是：先不要启动 Communicator，在Win95的“开始”菜单中选取程序/Netscape Communicator / Utilities /User Profile Manager。然后在对话框里选 New，按提示完成即可。一旦完成了新使用者的注册手续， Communicator会自行启动；以后你使用 Communicator时，它也会先问你希望使用哪一个 Profile。
　　10.关闭Java与 JavaScript功能
　　目前大多数网页上所看到的 Java与JavaScript的应用，多属于锦上添花型的小玩具，没什么特别的功用，只是好看、好玩的小把戏。如果你上网不仅仅是为了凑热闹，而是真的想要获取有用的信息，加速资料下载的速度，建议将浏览器内执行Java与 Javascript的功能关闭掉，这样做不仅可以加速下载，还可预防一些恶意的网虫 (bug)钻进你的电脑。关闭执行 Java与 Javascript的方法是：从Edit的下拉菜单中选Preferences。然后在对话框的左栏点选 Advanced，在右栏去除 Enable Java与 Enable JavaScript的选项。
　　11.Cache的选择
　　如果你网上冲浪的习惯是常常固定去某几个你喜欢的网点，那么你应该将Cache(高速缓存)设定得大一点（建议：4～5 MB），当你再次光临那几个网点时，可以利用到浏览器已经保存下来的图形等资料，大大地加速了下载的速度。设定Cache的方法是：从Edit下拉菜单中选取Preferences。然后在对话框的左栏点选Advanced→Cache，在右栏的 Memory Cache, Disk Cache中设置适当的数值。
　　12.关于 Cookie 的应用
　　有些站点为了追踪来访客人的浏览行径，或记录访客的某些特定资料，会在你的浏览器里安置一个 Cookie。如果你希望保有高度的隐私权，你可以禁止任何人安置 Cookie在你的浏览器上，或是当对方想安置 Cookie时，让浏览器先通知你一声。Cookie的设置方法是：从Edit下拉菜单中选取Preferences。然后在对话框的左栏点选 Advanced，在右栏的 Cookies处选择接受： Accept all cookies ；不接受： Disable Cookies；或让浏览器先通知你一声： Warn me before accepting a cookie。
　　13.去除工具条的图像
　　如果你的电脑屏幕并不很大，或者希望工具条所占的面积少一点，使浏览的面积大一点，你可以将工具条的图像都去掉，节省出不少空间。做法是：从 Edit下拉菜单中选取Preferences。然后在对话框的左栏点选Appearance，在右栏的 Show toolbar as处选择Text Only。
　　14.指定浏览器的首页
　　想要指定浏览器开始启动时，就去链接你想要去的网页，设定的方法是：从Edit下拉菜单中选取Preferences。然后在对话框的左栏点选Navigator，在右栏选择Navigator starts with Home page。并在 Location处填入你指定的 Home page地址。每次启动Netscape时，即可进入你想要去的网页。
　　15.复制网页中的精华片段
　　有些网页长篇大论，由好几个屏幕页组成，但其实你需要的却只是其中两三段的精华而已。这时你可以用鼠标在你真正需要的片段开头点一下，按住 Shift键，再指到你需要的片段的末尾点一下，然后按下 Ctrl＋C键，将它拷贝下来，粘贴到记事本或其他文字处理软件里。
