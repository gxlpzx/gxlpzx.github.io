使用默认程序打开文件

　 使用Win95文件的用户一定知道文件关联的概念，双击文件Win95就会自动地调用默认程序打开它。其实在VB中通过调用API函数“ShellExecute”可以实现这种功能，如打开默认浏览器浏览主页，发送邮件等。
    程序如下：
    OptionExplicit
    PrivateDeclareFunctionShellExecuteLib″shell32dll″Alias″ShellExecuteA″(ByValhwndAsLong,ByVallpOperationAsString,ByVallpFileAsString,ByVallpParametersAsString,ByVallpDirectoryAsString,ByValnShowCmdAsLong)AsLong
    PrivateSubCommand1_Click()
    ′检查文件是否存在′Filenm为一文本框的名字，在此输入待打开的文件名
    If Dir(Filenm)=″″Then
    CallMsgBox(″此文件不存在!″，vbExclamation)
    ExitSub
    End If
    ′使用默认程序打开文件
    CallShellExecute(hwnd,″Open″，Filenm,″″，AppPath,1)
    End Sub
    PrivateSubCommand2_Click()′用默认浏览器浏览主页
    CallShellExecute(hwnd,″Open″，″http:∥coolknightyeahnet″，″″，AppPath,1)
    End Sub
    PrivateSubCommand3_Click()
