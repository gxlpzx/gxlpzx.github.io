Visual Studio 6.0先睹为快

   　对所有软件开发人员和编程爱好者来说，九月都值得期盼的。Microsoft公司将于本月推出Visual Studio开发工具套件的最新版本——Visual Studio 6.0。也许这是比Windows 98更令人激动的升级，现在就让我们去一睹为快。
　　一、Visual Studio 6.0开发工具的新特性
　　Visual Studio 6.0是完全基于Windows DNA体系结构的开发工具套件。它将是业界第一个将客户机／服务器模式，Internet和PC计算模式集成一体的应用体系结构。Visual Studio 6.0仍然为我们提供了内容丰富的开发工具，并将它们的功能作了进一步的扩展：
　　1.Visual BASIC 6.0：将用于快速创建基于Windows 环境的客户机／服务器模式应用系统，Web应用系统及中等规模的应用系统。
　　2.Visual C＋＋ 6.0：该工具将用于快速创建高性能的应用系统的COM部件。
　　3.Visual J＋＋ 6.0：是Microsoft最新版本的JAVA开发工具，它将用于创建Applet工具，基于Windows环境的应用程序及使用JAVA语言的部件。
　　4.Visual Foxpro 6.0:它基于XBASE语言，用于创建数据库程序和开发部件的关系数据库开发系统。
　　5.Visual InterDev 6.0：用于创建基于HTML语言Script脚本以及Web应用程序的集成开发环境以满足团队开发Web应用程序的需要。
　　二、Visual Studio 6.0的版本
    Microsoft公司的Visual Studio 6.0分为专业版和企业版本。与此同时，也公布了Visual Studio 6.0的售价，作为企业版本，其完整版本售价为＄1619，升级版本为＄1079。专业版的售价略为便宜，完整版本为＄1079，而升级版本售价为＄549。
编者：我们都期待着Visual Studio 6.0中文版的早日问世，也期待着微软公司为中国的计算机爱好者给出一个公道的价格。       
    (四川 张军)
