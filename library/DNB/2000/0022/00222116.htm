《 原创天地 》

　　软件名称：Fantasia Web Explorer
　　软件类型：网页浏览
　　软件获取：http://bluehawk.yeah.net
　　设计思路：
　　传统浏览器Internet Explorer为单窗口浏览界面，系统臃肿、功能繁杂，同时也是消耗资源的大户，不少低档次的PC浏览网页只能开少数几个进程用于网络浏览以免资源消耗过多而导致系统瘫痪，或者换用Opera等低系统消耗软件进行网络浏览操作，但同时这也带来了网络规范的不兼容问题，导致浏览品质欠佳。于是，我有了开发这个低资源占用率浏览器软件的想法，并将其付诸实现。
　　Fantasia Web Explorer采用单进程多线程设计，使得它的系统占用率很低，同时拥有完善的内核保护措施，大大增强了系统的稳定性；为了方便在浏览的同时进行信息检索，内置了8大类28个分类搜索引擎，使得数据查找更为方便直接；改良的收藏夹管理功能，支持收藏页面和其群组操作，并且兼容NetScape书签数据，并且可以在Internet Explorer的收藏夹文件和Netscape的书签文件之间自由转换。
　　另外，拥有功能强大的IP工具，包括Ping、Tracert、Finger、whois以及对方主机查询功能；本地解析数据管理系统可根据本地的收藏夹信息，对你的本地解析库进行管理分析，从而使得你的网站连接速度大大加快；即时系统资源显示，用户可以即时掌握系统资源状况；定时事件提醒；广告窗口过滤、地址栏自动填充等都让它的使用更加方便。
　　简评：
　　完全仿真Internet Explorer的界面设置，新用户可以轻松上手，无须学习适应。
　　收藏夹中的打开当前所有连接的功能很好呀，对于像我这样时常需要打开一大堆类似网站的人来说，简直再方便不过了！如果能将这个功能再增强一下，引申到可选择打开当前页面的那些链接的话，那我可真会把IE“请”出我的系统了！
　　当然，有的小地方也应该注意一下，比如“关闭当前窗口”和“关闭当前窗口以外的所有窗口”两个按钮相隔太近，而且执行的时候没有确定对话框，容易产生误操作；“源代码”误写成了“‘原’代码”，可得注意了！建议后续版本在软件和相关数据库的自动检测升级上下点功夫。
