《 {MPG格式转换成ASF格式} 》

{　　目的：将MPEG－1格式的影像文件转换成微软最新的ASF视频流格式。
　　工具：Sonic Foundry Stream Anywhere和Windows Media Toolkit
　　步骤：首先检查你有没有旧版本的ASF压缩编码驱动库，请在Windows/System目录下面查找mpg4c32.dll这个文件，找到这个文件查看这个文件的属性，然后按“版本”标签，检查版本号是不是4.00.0.3688或3.0.0.2700，如果不是，那么请你删除mpg4c32.dll和mpg4ds32.ax（如果存在的话）这两个文件。然后，重新安装Windows Media Toolkit或把相应版本号的ASF压缩编码驱动库拷贝到Windows/System目录。
　　接着运行Sonic Foundry Stream Anywhere这个软件。打开你的MPG文件。选择“Save as”（另存为），选择你要储存ASF文件的文件夹，选择“Encode Windows Media Format”项，选择你要生成的ASF文件的压缩比例。如果是选250 Video的话就相当于2M/分钟的压缩速度，这个时候图像的质量不太好。同理，如果是选512 Video的话就相当于4M/分钟的压缩速度。其实最好的ASF是在320×240和有30帧/秒的情况下，它的图像和声音仍然可以同步。最后，把Encode RealMedia format前面的选的钩去掉即可完成。}
