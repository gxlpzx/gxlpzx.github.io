《 NETWARE5 快速安装 》

　　NOVELL公司新近推出NETWARE5，与以前几个版本相比有了很大的改进，本人最近刚用NETWARE5组建了一个局域网，也来谈谈NETWARE5的使用心得。
　　一、NETWARE5的安装准备工作
　　制作一张DOS启动盘,DOS6.22、WIN95启动盘皆可。使用FDISK.EXE命令对硬盘进行分区,建立一个DOS主引导区,要求不小于50M,也可建立扩展分区,但要留下不少于500M的磁盘空间作为NETWARE分区，即不要分区。
　　编辑CONFIG.SYS
　　加入FILES=30
　　BUFFERS=30
　　二、NETWARE5的安装
　　NETWARE5安装的最大障碍就是硬件驱动程序的加载,如果不能加载驱动程序,安装将不能继续。NETWARE5的安装程序会自动搜索硬件,并为其选择一驱动程序,有时该驱动程序并不是你机器硬件的。你可以在安装对话框中选择MODIFY,在弹出的对话框中选择驱动程序。如果还是没有你要找的,请按INSERT键,输入驱动程序路径进行加载。然后机器会检测硬件。
　　服务器的网络软件安装是一个图形界面，只要按步骤输入:计算机名,IP地址,子网掩码，网关,超级用户口令,DNS树名(HOPE),CONTEXT(O=HOPE),选择你要安装的服务即可。
　　三、网络用户和组的建立
　　网络用户的建立有两种:
　　在服务器上建用户和在客户机上建用户。
　　在此就介绍在服务器上建用户的方法:
　　启动服务器,在SERVERNAME:提示服下输入STARTX,进入NOVELL CONSOLEONE,点击NOVELL,选择CONSOLEONE在此窗口中双击THE NETWORK,双击服务器名,输入USERNAME:ADMIN,以及密码,和CONTEXT以超级用户登录,然后鼠标右键单击服务器图标,选择NEWUSER或NEWGROUP。输入用户名,组名即可。
　　用户登录脚本的输入:
　　只要右键点击用户,选择PROPERTIES,在LOAD SCRIPTS中编写。
　　四、用户登录服务器
　　1.有盘DOS用户的登录
　　将网卡驱动盘上的NETWARE目录COPY到硬盘。
　　编写一个批处理STARTNET.BAT如下:
　　SET NWLANGUAGE=ENGLISH
　　LSL.COM
　　DE22X.COM(此为网卡驱动程序)
　　IPXODI.COM
　　VLM.EXE
　　编写NET.CFG如下(我使用的是D－LINK DE22X网卡):
　　LINK DRIVER DE22X
　　PORT 300
　　IRQ 3
　　FRAME ETHERNET_802.2
　　NETWARE DOS REQUESTER
　　FIRST NETWORK DRIVE=F
　　NETWARE PROTOCOL=NDS BIND
　　NAME CONTEXT=″O=HOPE″
　　PB BUFFERS=0
　　运行STARTNET,就可看到F:
　　2.无盘DOS用户的登录
　　制作一张能登录到服务器的引导盘,用软盘启动,并以超级用户登录,在F:>SYSTEM输入DOSGEN.EXE，此时系统将会读软盘,并生成一个Net＄dos.sys文件,将此文件拷贝到SYS:\LOGIN目录下,加共享及只读属性将软盘上的AUTOEXEC.BAT也拷贝到该目录，设置网卡BOOTROOM=ENABLED，OK!现在你的无盘工作站可以上网了。
　　3.WINDOWS用户的登录
　　打开控制面板→网络,选择添加协议:MICROSOFT IPX/SPX兼容协议，添加NETWARE网络客户,重新启动计算机,就可在网上邻居看到服务器。不过,WINDOWS登录用户名,一定要和在服务器上建的用户同名,同密码。
　　4.NT访问NETWARE服务器
　　以Administrator登录，打开“控制面板”→“网络”,选择“添加服务”:NETWARE网关服务，在到服务中,启动该服务。此刻你的NT控制面板上会多一图标GSNW，打开,在首选服务器上键入NETWARE服务器名,在单击网关按钮选择启动网关,如果NETWARE上有Administrator用户,并有相同密码,你就可在网上邻居看到NETWARE服务器。
　　五、NETWARE5服务器上光驱的共享
　　很简单,只要在服务器的命令输入行键入:LOAD CDROM。不过你还要放入光盘,NETWARE会把光盘的卷标当做卷名,你只要在客户端映射一下就可。MAP P:=卷名:，如此你的P:就是光驱!
　　而WINDOWS用户直接可在网上邻居看到该卷。
　　以上是我使用NETWARE5的一些经验，但希望大家在阅读本文之后，会有所启发。

（浙江　陈杨）
