Windows2000可禁用的服务

Windows2000以其稳定性而广受人们的喜爱，但它对机器的配置要求高，即使在128M内存下，运行一些较大的程序时，也会让人怀念Windows98 Se。但可以通过禁用一些不常用的服务来提高Windows2000的运行速度。首先用右键单击我的电脑图标，选择“管理→服务和应用程序→服务”，在右边任务列表中双击任意一项，在弹出属性窗口里的启动类型中可选择启动或禁用。一般人用Windows2000都是因为它上网速度快，故下面就个人用户上网的条件下，列出Windows2000可禁用的服务：
Computer Browser 维护网络上计算机的最新列表以及提供这个列表
Task scheduler 允许程序在指定时间运行
Routing and Remote Access 在局域网以及广域网环境中为企业提供路由服务
Removable storage 管理可移动媒体、驱动程序和库
Remote Registry Service 允许远程注册表操作
Print Spooler 将文件加载到内存中以便以后打印。要用打印机的朋友不能禁用这项服务
IPSEC Policy Agent 管理 IP 安全策略以及启动 ISAKMP/Oakley IKE 和 IP 安全驱动程序
Distributed Link Tracking Client 当文件在网络域的 NTFS 卷中移动时发送通知
Com+ Event System 提供事件的自动发布到订阅 COM 组件
重新启动后，你会发觉Windows2000明显快了许多。
