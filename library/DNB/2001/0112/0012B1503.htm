{打造自己的音乐CD}

{你是否想把卡式磁带歌曲、MP3和自己的“作品”都做成CD唱片，在家庭音响上与朋友们共享﹖没问题，下面我将介绍一下如何为自己的家庭制作CD音乐碟。
一、 音频输入及处理
首先我们必须把有关的硬件设备连接起来，用音频线把录音卡座的音频信号输出孔与电脑声卡的Lｉｎｅ－ｉｎ孔连接好，打开录音软件GｏｌｄWａｖｅ（图1）。这是一个非常好用的数码录音及编辑软件，除了附有许多效果处理功能外，它还能将编辑好的文件存成．WAV、．AU、．SND、．RAW、．AFC等格式，而且若你的CD－ROM是SCSI形式，它可以不经声卡直接抽取CD ROM中的音乐来录制编辑。
1． 新建一个声音文件，根据你的录音要求设置各项参数；
2． 从“工具”菜单中勾选设备控制器，桌面弹出如图2所示对话框；
3． 打开卡座播放磁带，同时单击设备播放器中的红色按钮，设备控制器下部代表左右声道的示波器有彩色频谱条出现，左右声道记录窗口同时出现声波图像，此时表示录音工作正常进行。如果没有上述反应，请先检查硬件连接是否正常，并检查一下音量控制对话框，看看录音控制对话框中是否勾选Lｉｎｅ ｉｎ 选项及把音量调到最大。如果正常，播放磁带时连接电脑的音箱应该发声。注意：在录音控制对话框中选“Wａｖｅ” 选项及把音量调到最大很重要，不然MP3文件转录的文件将没有声音。
4．录音完毕，保存声音文件为WAV文件格式，然后你就可以利用“效果”菜单中的各项功能进行噪音去除处理工作，具体做法本文限于篇幅就不祥述，留给朋友们自己去探索。卡拉OK“作品”的录制也是一样，只不过把卡座换成VCD影碟机罢了。
二、CD刻录制作
硬件要求刻录机一台，刻录软件，可以使用Aｄａｐｔｅｃ Eａｓｙ CD Cｒｅａｔｅ4．0 Dｅｌｕｘｅ（图3），它带有操作向导，使用简便。
1．从开始菜单打开Eａｓｙ CD Cｒｅａｔｏｒ。单击工具栏上“新建”按钮旁边的小箭头，并从下拉列表选择“音频CD”。
2．将一张空白CD插入CD刻录机目标驱动器。如果是从已有的音乐CD刻录轨道，请将音乐CD插入CD－ROM驱动器源驱动器。如果是刻录WAV或MP3文件，则继续执行下一步骤。
3．在资源管理器的左窗格中，选择含有音乐CD的CD－ROM驱动器。CD上轨道的列表出现在右侧。如果要刻录 WAV 或 MP3 文件，请选择文件所在的文件夹。如果在本机硬盘上找到了关于音乐CD的信息，则CD标题，艺术家名称和每个轨道的名称显示出来。如果该信息没有出现，而你可以访问因特网下载信息。
4．在资源管理器节的右窗格中选择你想刻录的轨道，然后单击“添加”。重复此步骤，直到音频CD版面包含所有的你想刻录的轨道和文件（当然你也可以直接把它们拖入CD版面）。然后在CD版面上填上“CD标题”和“艺术家名称”。
5． 单击“制作CD”。“CD制作设置”框出现如图4所示，此时“写入方式”应选择“Tｒａｃｋ－Aｔ－Oｎｃｅ”中的“关闭片段并保持CD打开”选项，否则你刻出的CD无法在普通影碟机上播放，单击“确定”，开始刻录。
希望朋友们能顺利制作出自己的CD唱片，MP3转CD后音质相当不错，最后嘱咐大家不要用于商业目的，否则肯定侵犯他人版权，欢迎朋友们来信与我交流，Eｍａｉｌｚｈｐ＠2911．ｎｅｔ。（北京 支海平）}
