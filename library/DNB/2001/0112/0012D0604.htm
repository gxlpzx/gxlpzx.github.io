{支持“断点续收”的邮件专家}

{ 邮件专家Mａｉｌ Eｘｐｅｒｔ是一款不可多得的智能化、功能强大的E－ｍａｉｌ收发软件。它吸取了FｏｘMａｉｌ和Oｕｔｌｏｏｋ Eｘｐｒｅｓｓ的优点，为用户提供了诸多实用功能，足以满足个人用户、单位用户的需要。其最大特点就是体积小、收发邮件速度快、操作简单。它还是一款免费软件，有中文和英文两个版本。
邮件专家V2．01提供了以下功能：
1． 智能化的“断点续收”功能。使用过网络蚂蚁的人对“断点续传”一定不陌生，邮件专家提供的“断点续收”就是因网络繁忙或线路不好“掉线”后，可以从“断点”开始收信，而无须重头开始从而大大缩短了邮件收取的时间。
2． 多账户管理（最多支持5个）。现在每个人都有多个信箱。邮件专家可以同时管理多个信箱。如果用户只有一个信箱，还可以关闭多账户支持，从而节省线程。
3． 支持ESMTP（要验证身份的SMTP）。国内越来越多的SMTP服务器已经安装了Sｕｎ公司或微软公司的ESMTP Sｅｒｖｉｃｅ，也就是说，发件人必须提交自己的身份代号才能开始发送邮件。邮件专家可以自动帮你完成这一切。
4． 多窗口书写信件。这是邮件专家与FｏｘMａｉｌ、Oｕｔｌｏｏｋ Eｘｐｒｅｓｓ界面上最大的不同。这意味着你可以同时在屏幕上编写多封E－ｍａｉｌ，或者边浏览已下载的邮件，边撰写新邮件。
5． 离线E－ｍａｉｌ编写、E－ｍａｉｌ收发、回复、转发、重定向，建立自己的信件夹（包括收件箱、发件箱、未发信件箱、信件回收站）。
6． 功能强大的地址簿编辑器。如果你觉得Oｕｔｌｏｏｋ Eｘｐｒｅｓｘ的地址簿使用过于繁琐，不妨试试邮件专家的地址簿。邮件专家还可以将收到邮件中的发件人的E－ｍａｉｌ地址自动加入地址簿。
7． 智能账户设置向导。由于邮件专家内带了国内外100多个免费信箱的POP3、SMTP服务器地址，你只要输入自己的E－ｍａｉｌ地址后，在1分钟之内便可完成POP3、SMTP服务器的设置工作。如果没有找到，邮件专家将自动猜解并搜索整个网段，找出这些服务器。
8． 附件关联编辑器。对于E－ｍａｉｌ中夹带的不同格式的附件，可以设置用何种应用程序打开。目前支持的格式有：RTF、DOC、PDF、JPG、GIF、TIF、BMP、PS、WPD、ZIP、TXT、HTM、HTML、MPG、MOV、AVI。不过，有些格式要设置一下，才可生效
9． 支持HTML格式的浏览。邮件专家支持浏览HTML格式的邮件。使用IE的用户可以在“设置”→”关联设置”中，选中HTM格式，点击“浏览”，选择“C＼Pｒｏｇｒａｍ Fｉｌｅｓ＼Iｎｔｅｒｎｅｔ Eｘｐｌｏｒｅｒ＼Iｅｘｐｌｏｒｅ．ｅｘｅ”即可。使用其它浏览器的用户选择相应的应用程序。
10． 信件加密打包备份。采用Bｌｏｗｆｉｓｈ 128位加密算法，将信件打包，节省你的硬盘空间。
11． 可制作一张“网吧便携盘”。对于在“网吧”上网的朋友们，现在可以插入一张1．44M软盘，点点鼠标，将整套邮件专家系统装入软盘。它只占用600余KB，这样就可以在家中编写好E－ｍａｉｌ，再去“网吧”安装它后收发E－ｍａｉｌ了。
12． 支持代理服务器收信。
13． 将邮件中的附件或邮件打包后
放入FTP服务器。整合了一个与CｕｔｅFTP类似的FTP客户端，支持拖动操作。
14． 可定时收信。
15． 其它功能。人性化的E－ｍａｉｌ书写字体、信件导入和导出到磁盘或硬盘、设定个人签名档、联机打印E－ｍａｉｌ、集成了ICMP Pｉｎｇ小工具、绿色软件“零”注册表改动。
