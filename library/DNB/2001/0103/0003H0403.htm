{回收站里的学问}

{ 编者按：“回收站”的功能是回收系统运行中不再需要的文件及程序等,在Windows 95出现以前的年代,用户要是删除某个文件,这个文件就消失了,如果发现删除了不应该删除的文件,就必须要用其它的工具软件进行恢复,非常不方便。而Windows 95/98/Me/2000/NT中的“回收站”则“宽容”了许多,它们最显著的特点就是扔进去的东西还可以随时“捡回来”。对于“回收站”,人们一般都认为没有多少东西可以学习,其实不然,在“回收站”中还是很有一些技巧值得钻研的。
一、改变“回收站”的容量
“回收站”的最大容量为磁盘空间乘以滑杆显示的百分数,如果想要改变它的容量,用鼠标右键单击“回收站→属性”,拖动滑杆增减磁盘空间的百分比即可。如果要对不同的磁盘进行不同的配置,则单击“各驱动器的配置相互独立”,然后单击需要更改设置的磁盘标签；如果要对所有的磁盘使用相同的容量设置,则单击“所有驱动器均使用同一设置”即可。
二、更改“回收站”的图标
“回收站”的图标不能像一般文件的图标那样通过“属性/修改图标”来改变,它要通过修改注册表(单击“开始→运行”,输入“regedit”,便可以打开注册表编辑器)的方法来改变。打开注册表编辑器,按顺序选择如下主键：“HKEY＿LOCAL＿MACHINE\Software\CLA-SSES\CLSID\645FF040-5081-101B-9F08-00AA002F954E\DefaultIcon”,并修改右边窗口的“FULL”(或“默认”、“EMPTY”)数值名称,修改其数值数据为图标名,包括全路径(如“C:\winnt\system32\ras.ico”),这样才可以修改桌面上“回收站”的图标。
三、为“回收站”更名
打开注册表编辑器,选择“HKEY＿CLASSES＿ROOT\CLSID”下的 645FF040-5081-101B-9F08-00AA002F954E主键,在右窗口双击“默认”,将其值由“回收站”改为所想要的名称(如“垃圾箱”)。退出后重新启动系统,桌面上的“回收站”就变成“垃圾箱”了。
四、删除桌面上的“回收站”图标
桌面上的“回收站”图标 不能用一般方法删除,如果想删除它必须通过修改注册表来实现。打开注册表编辑器,按顺序选择以下各项：找到“HKEY＿LOCAL＿MACHINE\Software\Microsoft\Windows\CurrentVersion\EXPLORER\Desktop\NameSpace”主键,选择NameSpace 键下的分支,如“645FF040-5081-101B-9F08-00AA002F954E”,通过右窗口中的串值看它是否是“回收站”,再删除该分支。退出注册表重新启动系统,看看桌面,此时“回收站”的图标已经消失了。注意：在删除前最好将其备份下来,以便日后恢复。
五、永久性的删除文件
要想永久性的删除文件,可以用鼠标右键单击“回收站”图标,然后在弹出的菜单中单击“属性”,将“不将文件移动到回收站”这一项选上,确认并退出即可。
六、不显示删除确认对话框
删除选定的文件时,通常会出现一个“确认文件删除”对话框。如果你感觉太麻烦,则可以让系统不显示确认删除对话框。用鼠标右键单击“回收站”图标,然后在弹出菜单中单击“属性”,清除“显示删除确认对话框”复选框中的标记即可。
编注：关于用“回收站”来隐藏文件的方法请参见《电脑报》2000年43期14版《利用“回收站”给文件夹“加密”》一文。
}
