{游戏广场在线论坛精华帖}

{ 对于一个游戏公司来说,最大的幸福莫过于玩家对自己的游戏产品有极高的认知度。在国内,也许只有大宇公司有这样的殊荣。但是,现在对于一个游戏公司来说,仅仅做好游戏就够了么？而在单纯的游戏制作之外,能不能再想一些办法给玩家更多？我们的游戏公司,路还很长……
风清月明
前几天玩了大宇的新作《轩辕剑叁：天之痕》,太爽了。我已经很久没有像这样被一个游戏打动了,如果说《轩辕剑叁》让我感到了震撼,那么《天之痕》则带给我哀伤和惆怅,尤其是看到最后那段话“六百年后时人常在……”,我的鼻子阵阵发酸,这种感觉即使是在当年玩《仙剑》的时候也未曾有过。
从《仙剑》到《轩叁》再到《天之痕》,大宇的游戏一个比一个精彩。如果把国产武侠游戏看成一部部小说,那么大宇绝对是写这些小说的金庸大侠。实际上,“轩辕剑”系列和金庸作品完全具有可比性：都在真实的历史背景下依托虚构的故事；都有曲折的情节和瑰丽的想象；都是那么大气磅礴；都带给我们许多超出作品本身的东西。比如《轩辕剑叁》,顺便还给我们普及了一下历史地理知识。看看当今国产游戏,没有一部像大宇的“轩辕剑”系列那样有水准,《轩辕剑叁：天之痕》和《轩辕剑叁》就像两座山峰一样矗立在它们面前,那么遥不可及。可以这样说,二十世纪的国产游戏玩来玩去只有大宇。
星空
这个问题应该引起大宇公司和大陆各代理商的关注了。我买了大宇曾在大陆发行的很多游戏(正版),可是每次都让我很失望——不是因为游戏质量不好,而是我享受不到升级服务！因为在大陆发行的是简体版的,而在台湾那里的是繁体的,升级补丁只是为了服务台湾的玩家！补丁就不能在简体版的游戏中用,一旦安装就会发现界面变成了乱码,甚至无法正常运行！如果这次育碧代理的《天之痕》还会出现这个老问题,那么我不会去再买正版了——尽管盗版的盘片有这样那样的问题,但它毕竟可以享受到升级服务！
}
