{缝缝补补又一年——游戏补丁现象解析}

{ 当你走进一家豪华的服装店,四周金碧辉煌,琳琅满目的时装令人眼花缭乱,店面整洁一尘不染,店员都笑容可掬,礼貌地迎接着顾客的到来。看起来一切都是那样舒服,那样贴心,温馨的气氛直到店员把你最中意最漂亮的那套衣服递过来的时候：
“先生,这是今年最新款式,包您满意。”店员用绝无掺水的微笑向你推荐着。
你穿上在试衣镜前面转了一圈,真的太合适了,款式、大小、价格正是自己想要的,于是掏出钱包,准备付款,可正在这时,突然发现衣服很显眼的地方破了一个洞,要知道,这是一件漂亮的新衣服啊！
“这是怎么回事？衣服上有个破洞！”
“哦,没关系先生,我们可以为你提供补丁。”店员仍然微笑着说。
“不,不,我不需要补丁,新衣服干嘛要补丁？”你有点茫然。
“没关系,我们的补丁是免费的。”继续微笑,他拿出一块很大的布来。
“我不管免费不免费,新衣服怎么能补补丁呢？裁剪的时候难道不能量好吗？”
“哦,现在大家都这样啊,没办法啦,谁让要赶圣诞假期呢,你就将就一些吧……”永不疲倦的微笑。
“……”
上面的故事好像一个荒诞肥皂剧中的情节,可细想起来,却并不荒诞。对于你我被称为“玩家”的群体来说,这“满纸荒唐言”,写的正是大家铁定都曾遇到过的“一把辛酸泪”。想想吧,谁敢说自己没有吃过补丁的苦头？谁又敢说所有游戏,都能在自己的爱机上顺顺当当地跑起来？出门逛市场,经常看到有一些小小瑕疵的商品只能被处理出售,唯有电脑游戏可以在打满补丁的情况下,仍然堂而皇之摆上精品架,如果这种情况发生在其他厂家产品上,恐怕早就被3.15揪出来曝光了。
笔者的朋友Firefly曾有一个令大家津津乐道的“壮举”,就是一边补着补丁,一边打穿了《创世纪Ⅸ：升天》。本来《创世纪Ⅸ》应该是“创世纪”系列一个辉煌的休止符,但为了商业利益,为了赶一个档期,使得不列颠尼亚大陆蒙受了永远洗刷不掉的耻辱：层出不穷的致命BUG。最后Origin公司被迫推出了容量为整整一张光盘的补丁！堪称游戏史上一大奇迹,也是最大的笑料。真不知道Firefly是如何在诸如系统随时崩溃、无法存盘、无法取档……这些能使人发疯的错误中“熬”过来的。回过头来,其他无数游戏中,较小一些的BUG四处横行更是司空见惯,较小一些的补丁也因此而成为各大网站提高浏览量最有效的手段,我们不仅要问：电脑游戏,究竟怎么了？
当然,并非“补丁”两字就一定不好,概括说电脑游戏的补丁,根据其特性大致分为以下几类：
1.硬件/系统补丁
之所以说电脑游戏软件是一种特殊商品,就特殊在它必须在一个硬件平台环境下才能运行,或者说,它的表现,很大一部分取决于硬件平台和操作系统的性能。电脑硬件几乎每半年就有一次更新换代。游戏作为软件产品,就会不可避免地遇到一个问题：下一代硬件产品为了性能考虑,自然会牺牲一些兼容性,如果说操作系统和重要的工具软件还可以顾及的话,对配置要求各异,数目如过江之鲫的电脑游戏根本不可能去一一对应,顶多给一些大厂商的“重头戏”一些面子,而其他的,只能由制作厂商自己想办法了。无疑,补丁就成了最直接最有效的解决方案,只要替换掉原来的几个文件,就能享受到游戏在新平台上“奔驰”的感受,这类补丁都有益无害且较有针对性,只适用于拥有相应硬件的用户。另外操作系统的升级也可能需要厂商为之制作相应的补丁程序,如Windows98到Windows2000时,有一些游戏就会无法正常运行。应当说,硬件/系统补丁属于厂商完善售后服务的一部分,对它应该持积极鼓励态度。
硬件补丁的例子非常多,笔者印象最深的是在Voodoo Banshee(即名噪一时的“女妖”)推出后,正是3Dfx如日中天的时候,几乎所有游戏厂商都针对Banshee推出了大量补丁,从另一个侧面也能看出这种硬件的号召力。可以想见,硬件补丁在一段时间内,将会一直存在下去,成为电脑业界的一道独特的风景线。
2.升级补丁
升级补丁大概是平时最常见到的补丁了。通常我们玩一部游戏时总会在某一处画面中发现诸如v1.00.00.04之类的字样,这就是所谓“版本号”。尽管理论上在游戏制作过程中,所有细微的方面都应该考虑到,诸如剔除程序瑕疵、完善网络功能、使内部设定合理化等等。可是,实际操作起来确有一定难度甚至不可能。拿网络功能来说,在测试期想发现问题简直是天方夜谭,就算如《暗黑破坏神Ⅱ》那种“万人在线大测试”,也没法把所有需要改进的地方都挖掘出来,更何况一般厂商根本没有这样的实力。这样随着时间的推移,一些“潜伏”着的问题慢慢凸显出来,唯一的解决之道只能是推出升级补丁。此类补丁属于周期性推出,越是受玩家欢迎的游戏,或者以网络对战为主的游戏,此类补丁就越频繁出现。升级补丁可以看作厂商售后服务的一种手段,说不上好坏,只是一种比较正常的现象,大多数玩家也已经习惯了。
几乎每一部叫得上名字的游戏多多少少都会有升级补丁,而像“暗黑破坏神”系列、“三角洲特种部队”系列、Counter-Strike系列这样的大热门游戏更是几乎每星期都在更新,估计在可以预见的将来,升级补丁仍然会一直存在下去。
3.功能补丁
记得在FIFA98刚刚推出时,国内玩家一拿到游戏就气不打一处来：连马来西亚联赛这样垃圾的东东都堂而皇之出现在其中,与意甲、德甲、英超并列,却没有我们中国的甲A联赛！谁料过了几天,在某配套光盘上竟然见到“FIFA98中国甲A联赛补丁”！对于一个铁杆球迷来说,那感觉顿时就不一样了。此类补丁我们称之为“功能补丁”,能够拓展游戏原来不提供的功能,此外还有如NBA系列没有乔丹时有人就制作过乔丹补丁,至于玩家自己制作的Quake3或CS地图、Model,《英雄无敌》中用三国武将头像替换原来的英雄……都可看作功能补丁的一种。一般来说,功能补丁多由第三方(主要是业余爱好者)开发,作用大多以趣味性为主,对游戏并无什么实质性帮助,却一直深受广大玩家的欢迎。
4.BUG补丁
这是玩家最最痛恨,厂商最最无奈的补丁了。理论上说,一部游戏在摆上货架前,应该先通过内部的Alpha测试、外部的Beta测试,如今许多与网络沾边的游戏还需要进行一到数次范围大小不等的在线测试,虽然最后还是避免不了一些小问题可以理解。但出现致命BUG却实在不应该,特别是某些游戏还会出现BUG无数,“满目疮痍”的可怕状况。玩家遇到这种事,除了愤怒外,似乎也没有什么别的方式能够表示当时的心情。
《血狮》是中国游戏业界永远的痛,许多中国玩家也正是通过它知道了什么叫做“BUG”。不过当时好像还没什么补丁概念,直到《江湖》出品后,玩家才真正享受了一把“补丁瘾”。平心而论,《江湖》单论质量来说还应当算国产游戏中不错的,可在几乎一天放出一个补丁的情况下,游戏很难正常玩下去实在让人无法接受。当时以为只有国产游戏才会这样,孰料游戏界的“圣殿”Origin公司更加变本加厉,《创世纪Ⅸ》险些将它的一世英名毁于一旦。BUG补丁的出现原因比较复杂,许多人归结为游戏公司责任心不强导致。其实,并非那样简单,谁不想把最好的作品拿出来呢？可一沾上商业运作,难免身不由己：必须赶档期、必须完成利润,再加上一个急功近利的老板就更糟了……不管怎样说,“BUG补丁”毕竟是对玩家权益的一种损害,作为消费者,我们有权利拒绝这种不公正待遇。
补丁的起源和存在,与网络也有着密不可分的关系,正是有了网络,厂商才能够给玩家提供更完善的售后服务,更贴心的沟通,但同时也给了某些不负责任的厂商以可乘之机。可不管怎么说,“Patch”这个词,在今后仍然会一如既往伴随着玩家度过游戏人生的日子。21世纪到来了,新的一年又是一个新的起点。虽然缝缝补补又一年,但市场的规律毕竟是优胜劣汰,不负责任的公司总会付出应该付出的代价,只是希望,这样的事越少越好,也算笔者的新年祝愿吧。
}
