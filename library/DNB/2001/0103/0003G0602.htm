{别了，小猪……}

{ 坐在霍村的大树下,陪伴在我身边的是我的小猪乌力乌力。这只从一出世就开始与我一起作战的小猪,用它亮亮的小眼睛观察着我,看得出它很好奇,为什么一向天不怕地不怕的我,竟然在独自悲伤。
轻轻拍着小猪的头,我想起了我们到处去探险的情景,在黑暗的洞穴里跟丑陋的舔舔怪搏斗,在海底隧道一起从鲨鱼的利齿前逃跑,沙姆岛上大战奇宝中队,即使在吉鲁矿坑里面对着发了疯的长毛象群,我们也是一直在一起,从来没有过片刻分开……多少次,当怪兽的爪子在我面前划过,是小猪用它的身躯为我挡下敌人的进攻,多少次,我们默契配合由它从背后对敌人发动偷袭……身边组队的队友换过多少,一直与我共同成长的,就只有我的小猪。
看看天色,黑压压的一大片乌云正飞速地压了过来。我知道,当天空落下第一滴冰雨的时候,我将永远永远和小猪分开,再无相会之日。
我诅咒着天,我诅咒着地。
小猪在凉风中小声地打了个喷嚏。听着那可爱的声音,眼泪再也无法控制,刷地一下不争气地奔流了出来。我一下子紧紧地把小猪抱在怀里。小猪抬起头,用它粉红的舌头轻轻舔去遮住我眼睛的泪水。那冰凉的感觉,让我的心一阵阵绞痛。
为什么这个世界总是这么残忍？美好的事物最后总要被摧毁？
我永远也不知道问题的答案。这一刻,我想做的,只有紧紧抱住我的小猪,再多一秒、再多一秒……
皮皮：《石器时代》的一位女玩家的文章,充满了伤感和泪水。也许在商人的眼中,游戏中的小猪只是一块块色斑；但是玩家的心中,纵然是游戏,也是自己的心爱。
为了钱,可以悍然砍去玩家的爱,但是为了钱,也会失去玩家的心。
}
