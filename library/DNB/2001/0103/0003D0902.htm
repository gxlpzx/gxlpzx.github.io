{上网七件事之网上聊天}

{ 初涉入网的网友们,也许感到最为亲切的就是网上聊天了,网络聊天使天南海北的朋友打破了时空限制,可以“面对面”交流。网上聊天大多数是通过键盘输入的方式来实现的,最近又出现了语音和实时录像的交流方式,总之各种聊天的方式都作为一种有趣的交流方式,得到了广大网友的普遍认可。
当然了,实现网上聊天需要一个交谈工具,从广义上讲,交流工具可以分为两类,一个是使用交谈服务器作为中央站点,另一个就是使用直接连接的方式了。比如说吧,在使用网页(Web)聊天的时候,浏览器就是交谈工具,另外,大多数非Web交谈工具一般都采用独特的设计方式,需要配备专门的交谈工具,比如ICQ类交谈工具等。
在网上,最简单的交流工具就是Web聊天室了,它是由Web站点提供的聊天方式,支持很多人一起交谈。如果要参加Web聊天的话,首先需要用浏览器来访问提供此类服务的站点,这时你就好像进入了带有许多单间的大厅一样,可以任意从一个房间进入另一个房间旁听或者实时交流,有的聊天室是有固定的话题或者是针对某类人群的,不过有的也没有。比如说吧,进入网易提供的聊天室以后,可以看到当前聊天室在线的人数,如果要参加某个话题的讨论,可以申请成为网易的注册用户,当然这一切都是免费的了。另外,在进入聊天室以后,需要注意聊天室的礼仪,首先要尊重对方的语言习惯,不要使用不礼貌的问候语和一些不干净的语言,否则触犯众怒,就会把你给踢出去了。
随着网络技术的不断发展和完善,Web聊天室已经不能满足众多网友结交更多朋友的需求了,于是就出现了ICQ类聊天工具。ICQ就是“I Seek You”的读音缩写,中文意思就是“我在找你”,被网友取了一个别致而有趣的外号“网络寻呼机”。ICQ类工具可以查看当前在线好友,作为一种崭新的沟通模式,ICQ还提供了免费的网上BP机服务,可以从http：//www.icq.com处下载最新的ICQ软件。由于ICQ是个英文软件,所以国人常用的此类工具还是OICQ。在使用OICQ之前需要首先申请一个免费OICQ号码,然后就可以和ICQ上的好友交流了。使用OICQ可以实现收发消息、进入二人世界、传送文件、传送语音、进入聊天室、网络寻呼和手机短消息发送等功能。在OICQ中,可以在网上任意寻找一个陌生人来彼此交流,使每个上网的朋友都能感到身处“地球村”中的惬意。OICQ作为一个免费软件可以到腾讯公司主页http：//www.tencent.com下载最新版本,启动运行以后仅占系统资源的5％,并不会给你的机器带来多大的负担。
很多网友都要不约而同地发问,是不是只靠输入几个文字就可以称为聊天呢,网友梦寐以求的还是实现语音聊天,但是由于受到目前网络传输速度的影响,此项技术还没有真正走入千家万户,但是很多网站已经开始提供此类的服务了,语音聊天必将取代传统的网络聊天方式,成为网上实时交流的主力大军。可想而知,自由的和网友交谈,侧耳倾听来自大洋彼岸的问候,那种心情和感受绝对不是简单的几个文字所能描述的。
网络的技术是不断发展和前进的,随着互联网的数据带宽的不断增加,宽带网接入普通家庭的步伐越走越近,实现影像交流方式也将成为可能并会普及,现在市场上摄像头已经够便宜的了吧,给自己的爱机配备一个摄像头再也不是什么天方夜谭了,有了影像输入设备,再配备上声卡和麦克风,驶上高速的宽带网,呵呵,通过互联网就可以看到远在天涯海角的朋友的庐山真面目了,岂不快哉！
互联网缩小了整个世界,给人们带来了结识更多朋友的机会,打破了原有“社会”的概念,而作为网上交流的重要手段之一的网上聊天,在其中起到的作用是不可磨灭的,想必这也是众多网友对此情有独钟的原因之一吧。
}
