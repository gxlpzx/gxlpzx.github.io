{康熙在线 清晰可见}

{ 对文学著作的喜爱,做着依旧是“昨夜星辰,今夜星辰”的梦；对网络的迷恋和诸多不满,可是又身陷其间而不能自拔；对足球的现状牢骚满腹,却仍然是“爱也是你,恨也是你”的难舍难分；游走于山水之间,却也为放松了的神思和身躯再添上了疲惫。一切都是在矛盾中享受着乐趣,在矛盾中清晰知道了自己的成长、苦恼和喜悦以及困惑。
为了给自己一个可以把自己的想法、看法、牢骚、不满、苦水、喜悦尽情倾吐的空间,收发任由心生的 “康熙可见”(www.ksee.net)就应运而生了,有游记、读书记、随笔、侃IT等栏目。康熙的公正、幽默、开朗、心胸似海一直是自己心中所神驰的,而Ksee 就是 can see ,就是为了可以自己看清自己、分析自己、解剖自己、反省自己,也为了给自己开一扇让友人督促的窗口。
“康康慨慨做人,主人好客；熙熙攘攘有朋,宾朋满座”就是康熙可见的原则和愿望,希望大家给康熙多一点支持和掌声。这里就是 www.ksee.net。}
