{2000年的声音——2000年国内多媒体音箱市场综述}

{ 我国是多媒体音箱的最大生产国,占有全世界市场的50％以上,2000年的年产量超过了2000万对。已经进入2001年,我们对过去的一年该行业的情况作一个总结,看看有哪些事、哪些产品是值得我们关注的,还有哪些问题影响着这个行业的健康发展。
一、 音响企业进军多媒体音箱市场
前些年,音响行业在国内市场上可以说出尽了风头,推动了国产音响器材的发展。近年来,随着国内市场的变化,产品出现了供过于求,不得不寻找新的发展空间。在这样的情况下,一些音响企业开始寻求一些新的突破和发展,其中有一些厂家开始向多媒体音箱领域发展。从现在来看,它们的进入还不会对现有的多媒体音箱市场格局产生大的影响。目前,在市场上已经见到他们的产品和广告的有惠威、浪腾、索威。他们的产品基本定位在中档。及中高档价位上。这些音响厂家以前做“精品”已经习惯了,它们在产品的工艺上、音质上有自己的一套方法。但是,由于多媒体音箱与音响产品有许多不同,因此它们在生产管理上、经营上目前还适应不了现在多媒体音箱市场的发展。目前还无法对现有的多媒体音箱市场构成大的冲击。不过,从长远来看,音响厂家进入多媒体音箱市场,一定会对多媒体音箱行业的发展产生重要影响和有一定的推动作用。
二、 产品结构不断完善,新产品层出不穷
2000年,多媒体音箱产品结构进一步完善,从大量出口的“2美元音箱”,到市场上出现价格在1000元左右的高级Hi-Fi音箱；从2.0、2.1音箱到4.1、5.1音箱,从采用普通喇叭的音箱到采用同轴喇叭的音箱；从大而重的木质音箱到轻而薄的平板音箱；从PC用音箱到笔记本电脑用音箱；从USB接口到红外遥控……在品种上、样式上,几乎每个厂家都相继推出了不少新产品。从去年初 “幻彩2030” 开始的“多媒体音箱大换装”,拉开了多媒体音箱的“彩色肤色”时代的序幕；漫步者平面化设计的301,给人面目一新的感受；麦蓝的M-200开创了国内2.1产品先河,颠覆了以2.0结构为主导的产品结构。它们之中5.1结构的M-1000又让多媒体音箱走入了DVD领域。漫步者带遥控的R501T以690元的价格更让人心动。去年不被看好的采用NXT技术平面音箱,被麦蓝的M-400重新推向高峰,并被市场接受。深圳索威公司专为多媒体音箱而设计的“同轴扬声器”音箱,使多媒体音箱在理论与实践上前进了一大步。惠威公司推出售价高达1000元的Hi-Fi级的M-200多媒体音箱,更是把炙热的多媒体音箱市场推向顶峰。
三、 新厂家、新品牌多
2000年以前,市场上常见的品牌有漫步者、JS、轻骑兵(润宝)、麦蓝、DIBO/DIBA、三诺、奋达、丽歌。丽歌在2000年已经换了“面孔”,奋达则在国内比较低调,它们去年主要做出口市场。DIBO一直是以出口为主。而去年在市场上也出现了的许多新品牌：惠威、浪腾、索威。还有广州的南欣、速捷时。上海的鸿喜、百利。北京的冲击波、金豪迈,还有太阳花、爱国者等等。看来多媒体音箱由于市场进入的门槛低、技术含量少(相对与其它PC产品)、市场容量大、增长速度快、相对风险小,使得不少人“看好”这一市场。今年也许会有更多的厂家进入这一市场。
四、 加强基本建设和科学管理
2000年,随着市场的快速增长,一些厂家的原有基础条件,已经满足不了发展的需要,它们开始扩大厂房、购置新设备,为今后的发展打下基础。漫步者除了继续发展北京的基地外,又在南方的深圳开设了生产基地,增加生产面积近二万平方米；麦蓝公司自成立后,就一直在“搬家”,最近它们又再一次搬进了近三万平方米的厂房内,目前日产量可以达到8000套。像漫步者、麦蓝这样搞“基本建设”的多媒体音箱生产厂家,不仅仅是两家,而是一个比较普遍的现象。
科学管理是企业的基础,由于历史的原因,多媒体音箱行业多是“发展超前,管理滞后”的企业。现在这样的企业,在完成了原始积累后,发现管理的落后制约了发展。因此它们纷纷在管理上进行“补课”。例如,请一些具有经验的MBA人士或一些咨询公司参与公司的管理。科学地规范公司的行为；开始注意公司的形象,规范企业形象识别系统。在管理上进行ISO9000、EPR、MRPⅡ等等系统的认证与实施。在产品上进行“长城”安全认证等等。
五、 国际化意识增强
以目前多媒体音箱厂家产量的增长势头来看,仅仅只是以国内市场销售而言,可能已经出现“供大于求”了,因为目前仅以上述提到的厂家生产的能力来看已经超过国内市场的需求量了,因此,上述的一些厂家就比较注意自己产品的国际化,同时注意自身意识的国际化。淇誉、新捷超分别是台资、新资公司。而漫步者、麦蓝、三诺都先后在美国、加拿大注册了分支公司。最近一年来,在德国举行的CeBIT和在美国举行的Comdex Fall、WCES上,都有其中上述某些厂家的展位、产品。事实上,你现在能见到的几乎所有的多媒体音箱(包括以上没有提到的国外世界级的名牌),你都可能在某处找到“MADE IN CHINA”的字样。
六、 学术交流和探讨
伴随2000年多媒体音箱市场发展,与此相关的一些深层次的活动在有关媒体和厂家的支持下,结束了空白的历史。去年8月在《电脑报》等报刊、网站和深圳三诺公司的支持下,在深圳举行了第一次全国的“多媒体音响与多媒体音箱技术研讨会”。11月在惠威公司的支持下,在广州又召开了有全国二十多家著名IT媒体、网站参加的“多媒体音频技术座谈会”。这两个多媒体方面的会促进了多媒体音响、音频、音箱方面在技术上向更深的层次发展。
在网上,去年有关多媒体音箱、音频的活动也较积极,较有影响是在PC Show网站上办的“多媒体音频技术研讨会”和由PC Home主办的“多媒体音箱专题”的有奖征文活动。这两个网上的活动都取得了较好的效果。对推动我国多媒体音箱产品的发展也起到了一定的积极作用。
七、 2000年较出色的多媒体音箱
1. 惠威M-200
M-200是惠威公司进入多媒体音箱市场的首款产品。这款产品应用了惠威公司在喇叭单元、发烧音箱上成功的经验与技术,适合于常用多媒体音箱听音乐的用户。
2. 新捷超的DIBA138
DIBA138是一款可升级的2.1结构产品,再另加50元人民币就可以“升级”为4.1结构。这款产品外观非常大器和优美,外形、色彩、选材都颇具匠心,一套2.1的音箱价格大约280元。
3.漫步者R501T
R501是国内首款带遥控的5.1结构的产品,超低音是一款八吋喇叭,超低音分量充足,主音箱是采用“银笛”3英寸全频带单元,五个主音箱和低音炮都是木质结构。这款产品的市场零售价是690元。
4.三诺的2030
三诺的2030以“换皮肤”的流行概念,率先领军,展示多媒体音箱的外形丰富多变的形象。尽管这款产品的外形原作不是出自这家公司,它们成功“移植”,带动了国内多媒体音箱行业在产品外观上追求丰富和多样化的趋势。
八、 发展趋势
产品的两级化趋势
在今年,多媒体音箱会继续向两端发展。在低端,产品在价格上会有所稳定,虽有一点降价空间,但是幅度不大,而素质会有所提高。在高端,高素质的产品今年会有不断推出。
产品的外观变化的趋势
产品的外观今年会继续“丰富多彩”。在外形造型上会更大器一些,也会更丰富一些,文化味也会更多一些,有些厂家会逐步形成自己的风格。真木皮质音箱、铝合金音箱、金属质感的PVC贴皮音箱、超小型音箱、全透明音箱都会出现,有的会开始流行。
0.1结构的趋势
去年开始的2.1、4.1、5.1产品,开始受到市场的欢迎,今年这种趋势会继续,而比例会远大于2.0结构的产品。这是由于0.1结构的产品符合多媒体音箱的特点,有利提高音质、音效,同时又可降低成本。
向家电领域发展
在去年,音响厂家开始向多媒体音箱领域发展,而今年将会有多媒体音箱厂家向家电产品领域发展。这是由于它们(多媒体音箱厂家)在管理上、成本控制上、营销体系上都有与传统音响行业竞争的资本。我想这种相互“渗透”一定会推动家电音响、多媒体音箱行业的发展。
数字化趋势
2000年,创新公司在多媒体音频产品领域推出了“数码”概念,受到了市场的欢迎。尽管创新的产品在市场上反映差异较大,但是,数码产品是多媒体音频发展方向是没有错的。今年市场上会有内置(或外置)dts、AC-3解码的5.1产品出现,也会有带有D/A解码(CD音频解码)的产品出现。
九、 存在的问题
多媒体音箱产品和行业里存在的问题,不是去年一年所存在或产生的。这里我提出来是希望多媒体音箱行业能够从善如流健康发展。
产品同化问题
尽管目前多媒体音箱产品市场非常丰富,但是,却很少有体现厂家自己文化的系统产品,让人一看就知是哪家的产品。现在如果你不看铭牌,只看外观和听声音的话,你很难判别出是哪家的产品。这就是说每家产品的特点、特征太少,同化情况较严重。如果这样的话,那么我们产品的品牌价值就不高,市场的稳定度就不够,竞争力也不强。
不重视技术的深入、不尊重知识产权问题
对低端多媒体音箱产品而言,可以不需要多少技术,因此主要就是外观上的区别,目前一些厂家不做广告,不投入研发,更不重视对产品的深入研究,更甚者就是抄袭外形、抄袭设计、抄袭电路,甚至仿冒品牌。这些行为影响了行业的健康发展。
由于以上的原因,造成了多媒体音箱产品的品牌认同感低,严重影响和制约了行业的发展。最终的结果是大家都不重视产品的内在质量、不对新产品做深入的研究,最后是好的厂家只是变化外观；差的厂家就是一切都抄袭。希望这个行业里的厂家应该自律和联合,杜绝和反对抄袭、仿冒行为。
}
