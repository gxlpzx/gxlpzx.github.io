{浏览器Opera 5.0的新特点}

{ 1.由共享软件变为免费软件
Opera 5.0与它的“前辈”们相比,该软件已经由共享性质向免费性质迈出了可喜的一步。Opera以前版本只有30天的试用期,而且在试用期间该软件的某些方面的功能会受到限制。而Opera 5.0则不同了,虽然它并没有取消注册的机制,但是在没有注册的情况下,程序的功能方面并没有什么限制,只是会在该程序的操作窗口中多了一个广告条,而且不像以前的版本那样有使用时间的限制。该功能的实现是广告商赞助的结果,所以大家使用该软件的时候,不要忘记去点击一下广告条哟！
2.提高了系统的稳定性
笔者在连续一个月的试用中,发现Opera 5.0同时打开几十个站点时都非常稳定,没有出现一次非法操作而导致浏览器关闭的情况。而使用以前旧版本时,如果同时打开10个以上的浏览窗口时,系统将会变得非常不稳定。所以,Opera 5.0在系统的稳定性方面大大提高。如果从单纯打开一个浏览窗口的速度来说的话,Opera 5.0由于其内部配置大大优化,所以它打开一个网页的速度要比以前版本的浏览器要快许多。所以,Opera 5.0不仅运行速度快、资源消耗比IE少得多,而且稳定性很好,特别适合机器配置不高的用户使用。
3.增加了图像显示切换功能
我们在上网浏览页面的时候,考虑到速度方面的问题或者其他方面的原因,下载时需要屏蔽页面中的图像,目前的浏览器都具有屏蔽图像的功能,但是在IE和Netscape中设置了不显示图像之后,如果需要再次显示图像功能就需要重新进行设置,所以使用起来比较麻烦一点。而Opera 5.0浏览器针对上述浏览器的弊端,作了一些功能上的改进,它在每个浏览页面显示窗口的上方,放置了一个图像显示切换按钮,用鼠标按下这个按钮后,就能够很方便地对页面中的图像进行显示或屏蔽,从而取消了上述的烦琐设置,大大方便了用户的操作。
4.集成了自动页面刷新功能
如果你是利用网页来获取股市行情或者是去过论坛、拍卖站点,就知道只有不停地单击浏览器工具栏上面的“刷新”按钮,浏览器才能把当前页面的最新更新信息传送到用户的面前。很显然这样做是比较麻烦的,而且信息传送的实时性也比较差。为了解决这样的问题,Opera 5.0浏览器在这方面进行了改进,它在程序内部集成了一个自动页面刷新的功能,利用该功能,我们可以任意地设置页面自动刷新的时间间隔,从而很好地保证了页面信息的实时性。
5.提供完全版和简装版
由于各个用户使用的机器配置不尽相同,为了适应这些不同用户的要求,Opera 5.0浏览器提供了完全版本和简装版本两种,其中完全版本需要在奔腾150以上的计算机中安装,而简装版本只要在486以上的计算机就可以了。这两个版本的整体功能基本上相同,主要的差别就在于是否附带了Java程序的支持功能。如果我们上网是为了快速获取信息资料,并不想去欣赏过多的特效的页面以及那些花哨的图像,就可以在自己的计算机中安装它的简装版本。
6.提供强大的电子邮件功能
Opera 5.0浏览器在电子邮件功能方面也是非常出色,虽然是国外制作的工具,但该软件对中文的支持能力也相当不错,它不会出现某些纯英文邮件软件在删除文字时候半个半个文字被删除的现象,而且还支持多用户、多账号和POP3收信、SMTP发信、定时查信等功能。这就使得我们在线浏览页面的时候可以直接使用它提供的邮件服务来收信和发送信件。运行它内置的邮件程序之后,程序弹出的邮件编辑窗口只相当于多打开了一个浏览窗口,不像OutLook客户端那样占用很多的系统资源,影响计算机的操作速度。在该内置的邮件程序中,我们也可以在邮件中发送附件文件,接收信件和发送信件的速度也都不错,所以该邮件的综合性能要比其他浏览器内置的邮件客户端软件好。
7.集成了新闻阅读器、搜索引擎等功能
Opera 5.0浏览器还在程序内部集成了新闻阅读器、搜索引擎和即时信息等一些我们在网上经常需要使用的功能。利用这些功能,我们可以很方便地享受因特网给我们带来的其他服务。另外,在一台配置相同的计算机中,Opera5.0完全版本的浏览器启动完成只需要2～3秒钟的时间,而IE浏览器或者Netscape浏览器启动完成可能需要10秒钟左右。
8.采用了最新的128位加密技术
Opera 5.0浏览器使用了最新的128位加密技术,该加密技术已经超过了IE5.0浏览器的最高加密标准,使用Opera5.0浏览器在网上发送邮件或者提交表单信息时,能够最大程度地得到安全上的保障。另外,该浏览器软件还支持目前比较流行的CSS、XML、DHTML和JavaScript等网页制作方面的最新技术,换句话说无论我们使用什么样的网页设计工具,制作出来的网页页面在Opera 5.0浏览器中都能准确地显示出来,而不会出现不兼容的现象。
9.占用资源很少
Opera 5.0与IE等浏览器相比,Opera 5.0浏览器占用的资源极小。笔者发现,启动Opera 5.0完全版本的浏览器之后系统可用资源由原先的86％下降到78％,而打开一个IE浏览器窗口时,整个计算机系统则只有68％的可用资源了。当在Opera 5.0浏览器中同时打开10个浏览窗口之后,系统资源还有74％可用,但是打开10个IE窗口机器就慢得令人难以忍受了,可用的资源已经在30％以下了,这样操作起来很容易出现系统崩溃的情况。
10.提供了页面缩放功能
在浏览网页的过程中,有的网页需要在指定的显示模式下才能被正常地显示,如果没有按照指定的显示模式来显示,网页中的文字就会变得过小或者过大,影响我们的浏览。利用Opera 5.0浏览器提供的页面缩放功能可调整网页中显示的内容,并且这种缩放显示对于页面的内容没有任何损伤。当我们采用10倍放大和0.2倍缩小的时候,页面中的文字和图片还是一样的清晰。同样对于内容较多、一屏不能显示的页面,为了操作上的方便,我们也可以利用它的缩放功能让这些内容显示在一屏之内,从而方便快速浏览。
11.提供断点续传功能
当我们在浏览某一个页面的时候,突然遇到了诸如计算机死机、停电等意外因素不能正常浏览时,只要重新启动计算机并运行Opera 5.0,程序随后会弹出一个选择窗口,该窗口中有三个选项让用户进行选择,它们分别是“继续上次中断的连接”、“开启最后一次保存的窗口”、“只运行Opera 5.0并不打开窗口”。此时,如果我们选择第一个选项就可以实现页面的断点续传了。
12.在状态栏提供了网络传输信息
当我们使用Opera 5.0浏览器连接到一个站点上的时候,在Opera 5.0浏览器窗口中的状态栏下会显示出当前网络的具体传输细节信息,例如本次连接的传输速度、已经下载的字节数和全部页面下载的进度条等信息。而IE浏览器在连接到一个站点上面时,其状态栏中则没有提供这些详细的细节信息。}
