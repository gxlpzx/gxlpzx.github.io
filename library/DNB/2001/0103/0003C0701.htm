{电脑性能对比检测专家——passmark Performance Test}

{ Performance Test是专门用来测试电脑硬件性能的一个软件。它总共包含CPU浮点运算测试、标准2D图形性能测试等六大类、22种独立的测试项目,测试完毕它还会给出结果图,同时给出与基准电脑的性能对比图(该软件中预设了九个基准电脑的比较数据)。此外,在http://www.passmark.com/网站上还提供了其它知名电脑的性能测试数据,你可以把被测电脑与这些基准数据相比较,看一下自己的电脑是否达到了应有的水准。Performance Test可运行在Win9X/NT/2000平台上,下载地址为：http://www.win2000.com.cn/down/petst.exe。
一、安装和设置
安装前应先卸载Performance Test的老版本,然后下载petst.exe安装程序。双击该程序就会出现安装向导,确定好安装目录后,只要一路点击“Next”按钮,最后选择“Finish”即可。Performance Test还可以安装到一张软盘上,这样买机器时你就可以带上软盘,现场测试新机器的性能情况。安装到一张软盘上的方法是：先把它安装到硬盘某个目录下,然后把该目录(整个目录只有1.02MB大小)的所有文件拷到软盘上,最后卸载硬盘上的Performance Test,以后使用时只要双击软盘上的pt.exe文件即可。
值得注意的是：运行Performance Test前,必须安装Microsoft(微软)的 Directx 3D软件,Win98/Win2000/WinMe用户不存在这个问题,因为操作系统已经集成了Directx 软件,但Win95用户就必须到微软网站下载Directx(该软件是免费的),否则启动Performance Test时会显示“ddraw.dll not found”的出错信息。
在测试机器前应先简单地设置一下：打开程序主窗口中的“Edit”菜单中的“Preferences”选项,在“Disk drive under test”中确定Disk测试时要测试哪个分区,并在“This computer”中给被测机器命名。
二、主要功能及使用
Performance Test主要有Test和Baseline两大菜单功能。
1.Test 菜单功能
(1)Test 菜单有“Run all tests”(进行所有项目测试)、“Maths”(CPU算术术能力测试)、2D Graphics(2D图形能力测试)、“3D Graphics”(3D图形能力测试)、“Memory”(内存测试)、“Disk”(磁盘测试)、“MMX”(CPU MMX能力测试)七个菜单项(共六类测试项目),每个菜单项又有若干个测试子项(共29个)。六类测试项目分别是：
Math：有浮点测试、整数测试和FLOPS 每秒标准浮点操作测试三个子类,分成整数加减乘除、浮点加减乘除、最大MegaFLOPS等8项测试。
2D Graphics：测试在当前色深下,显示卡处理2D图形的速度,包括在测试窗口下画线、画位图、画椭圆、画圆角正方形的速度。
3D Graphics：只有“Many worlds” 一项测试,它产生四个球体,每个球体包含298个垂直面和588个多边形,测试每秒帧数,产生3D Graphics Mark数据。
Memory：测试RAM内存子系统的性能,有“Allocate small block”、“read Cached”、“read UnCached”、“Write”四项测试,产生“Memory Mark”数据。
Disk：测试硬盘子系统的性能,有“Sequential Cached Read”、“Sequential Cached Write”、“Random cached Seek RW”四项测试,软件默认测试C盘。当然你可进入“Edit”中的 Preferences”选项,从中选择其他分区进行测试。
MMX：测试Intel最近发布的CPU多媒体扩展指令,包括加、减、乘的测试。注意：Pentium 100以前的CPU不支持MMX功能测试。
(2)如果要进行某一类项目的所有测试,只要选择该类中的“All”选项即可。
(3)若要进行某类项目中的子项测试,只要选择该菜单子项即可,例如要测试你的CPU每秒最多能执行的浮点指令数,就可以选择“Math”类的“Max MegaFLOPS”子项,然后选择菜单“Test/Maths/Max/MegaFLOPS”。稍等片刻就会出现该项目测试结果图：一共两行,第一行是测试项目名称,第二行是测试结果。
(4)如果要进行所有项目的测试,只要选择“Test”中的“Run all tests”,屏幕上就会出现测试状态窗口,程序会自动按顺序进行每项测试,等一会儿,就会出现所有项目的测试结果图,每个测试结果占两行,第二行是测试分值。最后两行是被测机器的“PassMark Rating”值,该值是所有测试项目结果的加权平均值,用来简单地表示被测机器的性能。该值越大,表示机器性能越好,速度越快。
2.Baseline菜单功能
(1)俗话说“不怕不识货,就怕货比货”,只要把被测PC和基准电脑性能比较一下,就可以知道这台PC是否达到了应有的水平,这时候就要用到Baseline功能了：选择菜单“Baseline”中的“Select Baseline”选项,进入选择基准电脑窗口,软件内置了九台基准电脑的性能数据,显示在窗口的下半部,从中选择一台与被测机器相近的基准电脑,例如“Gateway Pentium Ⅱ 400”,然后按“ADD”按钮,最后按“OK”退出基准电脑窗口,于是原先测试结果图中就增加了基准电脑的对比图。其中红色图条是基准电脑数据,绿色图条是被测机器数据,通过这个对比图,就可以很容易地看出被测机器哪个指标未达到应有的水平。
(2)Baseline菜单有“Select Baseline”、“Baseline Information”、“Install Baseline from file”、“Download more Baseline”四个子菜单。其中“Baseline Information”子菜单显示被测机器和基准电脑CPU类型、内存大小、使用的操作系统等配置情况；“Install Baseline from file”是从文件中读入基准电脑数据。当然,你也可以自定义基准电脑,例如把当前被测机器定义为基准电脑,选择“File”中的“Save as Baseline”命令就把当前被测数据保存为基准电脑数据文件；“Download more Baseline”子菜单可让你从http://www.passmark.com/网站下载更多的基准电脑数据。
3.其他功能
对于当前测试结果,你可以选择“File”中的“Save as Baseline”命令保存为基准电脑数据,或者选择“Save as image”命令以图形文件格式保存,或者选择“Export”命令,把测试结果以TXT格式、Web格式保存,也可以选择“Print”命令输出到打印机。
如果你选择“Edit”中的“clear”命令,就可以清除当前显示的测试结果。选择“View”中的“disk read Graph”和“disk write Graph”,则可以看见磁盘读写平均速度和瞬时速度,其结果以图表形式显示,一目了然。
