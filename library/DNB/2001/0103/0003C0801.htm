{在老主板上使用USB设备}

{ 最近,朋友买了一款USB接口的Acer 640U扫描仪。由于是USB接口,在朋友的彩霸MVP3主板(AT/ATX兼容主板,AMD K6-2 400 CPU 超频至500使用,128M PC-100内存,G200 8M AGP显卡)上没有标准的USB输出接口,无法使用,这可把朋友急坏了,约我前去“诊断”。其实,一些老的Pentium主板(如：VP3,MVP3)都没有将USB接口作为计算机标准配备,而是作为选配,因此没有标准的输出接口,而是在主板上预留了呈一列或两列针状排列USB接口。虽然现在配件市场上有USB的外接插座出售,但却不能直接插入主板使用。若因此而更换主板甚至CPU的话,就太不经济了。经过一番研究,发现该板预留了USB接口,于是决定自己进行改制,利用简单工具让朋友彩霸MPV3主板可以使用USB接口的扫描仪,让它继续发挥余热。
为了使用它,我们花15元专门买了一根连接主板的USB插座连线,但买回来后仔细观察发现却不能直接使用。因为主板的USB出口连接为两排8针状接口,此插头接口为双排十孔(每排五孔,其中一针为空 ),显然不能直接插入,必须改造使用。通常,彩霸MVP3主板和VP3以及早期的主板大多支持1.0版本的USB,所以大多预留有8针接口,为两个USB接口,可以为两个USB设备使用,具体排列为1、5为USB-VCC,2、6为DATA-3、7为DATA+ 4、8为GND。再看我们所购买的USB连线,改造时,我们只需将红色电源线移到白色旁边即可,插主板一端为相同的两排四孔,第一个孔空出依次插入主板即可。每一单排连线从右至左依次为：
1——VCC(电源线,红色)
2——DATA- (信号线-,白色)
3——DATD+ (信号线+,绿色)
4——GND (地线,黑色)

怎么样,改造USB并不难。实际上无论USB插槽如何变化,4针还是5针,甚至是如磐英主板的16针,但是真正与USB接口连接时只用到四根连线。当然,如果有说明书,要认真对照主板说明书,不同主板的输出针的排列顺序可能不同,但只要按上面的方法找准这四根线正确连接,找准红色电源线和黑色GND线,你的USB设备就一定能正常运行,USB接口允许热插拔,即使插错信号线问题也不大。
最后需要说明的是除了连线正确外,要打开主板上的有关USB选项(在BIOS中设置),还要正确安装USB设备的驱动程序,当系统报告“USB DEVICE NOT FOUND”时要注意刷新“设备管理器”中“通用串行总线控制器”下的“USB ROOT HUB”。总之,只要胆大心细,再加上正确的设置,老主板同样能轻松用上USB设备,祝各位使用老主板的朋友们马到成功(如有问题请E-mail e2export＠sohu.com交流)。}
