{软“升级”Word 7.0}

{ 时下,Word 97/2000是大家广为使用的Word版本,如果由于某种原因你仍在用Word 7.0(即Word95)或者更低版本的Windows 3.x下的Word 6.0,当你收到了同事或网友的Word 97/2000文档时,你可能会为Word无法打开这些熟悉的文档而着急吧？在此,笔者告诉你一个“软”方法,让你的Word 7.0或Word 6.0同样能够打开这些Word 97/2000文档。该方法步骤如下：
1.单击“开始→程序→附件→写字板”,打开“写字板”窗口。
2.单击文件菜单中的“打开”命令,系统弹出“打开”对话框,在“文件类型”下拉列表框中选择“全部文档(＊·＊)”,在文件名称列表框中,单击需要的Word 97/2000文档,然后单击“打开”按钮。
3.单击“文件”菜单中的“另存为”命令,系统弹出“另存为”对话,在“保存类型”下拉列表框中,选择“Word for Windows 6.0”,然后单击“保存”按钮。
4.系统会弹出一个对话框,询问是否替换原有文件？单击“是”按钮,则将该Word 97/2000文档保存为同名的Word6.0文档。
现在,启动你的Word 6.0或者Word 7.0,你就可以直接对该文档进行编辑了。
}
