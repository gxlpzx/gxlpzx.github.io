{“虚拟光驱”使用要则}

{ 编者按：随着大容量硬盘的普及,现在模拟光驱的软件越来越多。比较常用的有“虚拟光驱”、“音速光驱”等等。一方面使用这些模拟软件可以最大程度地延长光驱的使用寿命,更重要的是它减少了电脑从光盘读取数据的时间,大幅度地提高了程序的执行效率。
笔者在使用这些“虚拟光驱”的过程中遇到过一些问题,现在把这些问题和解决方法总结成三点,供使用者参考。
1.使用软件虚拟光驱时,常常碰到莫名其妙的死机现象。
这种现象主要是由于虚拟光驱的使用,使数据的读取、传送、存储过程比平时大大加快(数十倍),系统稍不稳定,就会造成程序的退出、死机等现象,超频的机器尤甚。解决的方法其实很简单,只要单击“开始→设置→控制面板”,找到“系统”选项,再打开其中的“性能”选项,选择“文件系统”属性,找到有关CD-ROM的内容,把“最佳的访问方式”设置为“不预读”即可。如此设置的机器使用虚拟光驱软件造成死机的几率将大大减小。
2.使用过整理硬盘的工具后,虚拟的光驱突然不能用了。
这其实和虚拟光驱的机理有关。虚拟光驱时,软件除了虚拟了光盘上数据,还在硬盘上完整地虚拟了光盘上的数据分布结构。如果使用硬盘整理工具,如磁盘的碎片整理等,就会使本来按照特定顺序排列的数据被移动了位置,也就是你的“虚拟光盘”数据结构发生了改变,造成了“虚拟光盘”的“物理性损坏”,当然就不能用了。这个问题可以这样解决：在硬盘上专门分出一个区,用来放置光盘虚拟文件,具体的分区大小根据你的实际需要而定。当进行磁盘整理时,只要跳过这个分区即可。
3.虚拟光驱的盘符占据了实际光驱的盘符,有些程序不能运行了。
当实际光驱的盘符发生改变时,有些在此光驱上安装的程序由于“呆板”地记住了原来的光驱盘符而找不到光驱。其实不管是虚拟的还是实际的,光驱的盘符都可以任意改变(前提是不占用硬盘盘符)。打开控制面板中的“系统”选项(步骤同1),再打开其中的“设备管理器”,单击“按类型查看设备”,找到“CD-ROM”选项并展开,这时你就会看到所有光驱(实际的和虚拟的)。选择你需要改变盘符的那个光驱,点击下面的“属性”,在弹出界面中点选“设置”,在“保留的驱动器号”选项中将“开始驱动器号”和“最后驱动器号”都填上你需要的盘符即可。
另外我们应该尽量选用高版本的软件,它可以进一步减少出现机器故障的频率。其实还有一些虚拟光驱软件的“伴侣软件”也应该引起注意,比如一些软件可以编辑光盘的“虚拟文件”,以根据需要增加或删减一些内容,达到某些特别的目的。
以上就是笔者使用虚拟光驱软件的一点心得,欢迎大家与我交流(atgx＠263.net)。
}
