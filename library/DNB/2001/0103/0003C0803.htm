{华硕CUSL2主板的隐藏菜单}

{ 去年11月份,我趁着内存降价,攒了一台机器。主板我选了CUSL2。买回来后,先看说明书,哇噻,纯英文！弄了好半天才看懂。这款主板还有一个必杀技,那就是可以在不进入CMOS的情况下直接选择由哪个设备引导！在系统重启自检快完的时候,一直按着ESC键,就会跳出一个菜单,上面写着“Select Boot Device？”上面列出了我所有的可启动设备,旁边还有说明,可以按上下键选取设备。我觉得这个功能很实用,特别是像我这样偶尔从A盘启动一把的人,不用再像以前那样启动一次要进两次CMOS了(启动前一次,启动完毕了再改回来),太让人高兴了！
以上是我使用CUSL2的一点体会,希望大家多多指教。我的E-MAIL：idn＠netease.com。}
