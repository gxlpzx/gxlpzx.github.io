{使用IE查找下载Napster上的MP3资源}

{ Napster无疑是去年最成功、最有创意、最受争议的软件,是继ICQ后的又一个神话。Napster凭着让人们无需通过服务器就能方便地在各自的电脑之间共享文件这个简单而有富有创意的功能,在2000年征服了无数音乐爱好者的心,风靡全球。
由于Napster所使用的P2P(peer to peer,个人对个人)技术并不复杂,简单来说其实就是把所有用户的机器透过 Internal 连接起来组成一个大的局域网,让用户在上面交换文件。在 Napster 推出不久,就出现了相当多的模仿者,比较著名的有：Gnutella、CuteMX、Gnumm 等。但遗憾的是没有一个是有突破性改进的,都只是在 Napster 的基础上添砖加瓦而已。
前几天,以开发 GetRight！下载软件而闻名的 Headlight 公司发表了一款让人耳目一新的软件：File Funnel Beta1 。和前面说的那些跟风软件不同的是,这个软件的功能是让用户可以利用 Web 浏览器直接搜索 Napster 网络里面的 MP3 资源！够 COOL 的吧。换而言之,既然能用 IE 来浏览的话,那么里面的文件也顺理成章地可以用网络蚂蚁等软件来下载了(Headlight就是希望大家都用 GetRight 来下载),这样一来 Napster 不支持断点续传的难题也轻松地解决了。当然,如果你的带宽足够,也可以边下载边试听。由于软件革命性地利用了一种新的协议转换技术,把HTTP的HTML协议转换成Napster 所采用的协议,从而实现了它们之间的互相访问。相信这种技术发展下去,支持 Gnutella、CuteMX 等软件的协议也是迟早的问题而已。更让人兴奋的是,如果将来这种技术能得到广泛的应用,那么到时候随意访问 Napster 等网络里面的庞大资源将不再需要各自的专门软件,因为一个浏览器足够矣。下面我们来看看如何使用 File Funnel 。
首先到 http://www.filefunnel.com 下载一个安装版本的 File Funnel 回来,安装过程没什么特别,但因为涉及到 MP3 的版权方面的原因,所以要注意把协议窗口中“ I have read...”前面那个钩选上才能继续。
当然,安装到最后一步还要你输入一个 Napster 的账户和密码,如果你已经是 Napster 的用户就可以填上你的资料,如果不是,就马上申请一个,按下“Create Napster user account”按钮后等一下就可以创建一个Napster账户了。当然,如果你的名字已经有人申请了你就得要换一个,直到申请成功为止。安装完后,File Funnel 会在 Windows 的系统托盘里面放一个图标。
现在你只需要在这个图标上双击鼠标右键,File Funnel 就会马上打开一个你系统默认的 Web 浏览器(一般都是 IE),同时显示一个查找界面。
输入关键字后和几个参数后,按一下 “Search！”浏览器就会开始搜索 Napster 里面的资源了。 搜索完成后你就看到返回结果。
现在你已经可以用你喜欢的下载软件下载你需要的 MP3 歌曲了。
File Funnel 的用法简单到和通常的搜索引擎没什幺区别,但如果你仔细观察的话还是会看到一些端倪的,最明显的莫过于浏览器的地址栏了。
明显可以看到,File Funnel 是利用了自己作为代理,透过本机上的 8787 的端口来访问 Napster 资源的。而如果这个端口和你的其它软件有冲突,你可以在 File Funnel 的 Options 上面修改,并不影响使用。下面是一个File Funnel 的简单工作流程图。
File Funnel 这个软件由于目前还只是测试版本,所以在不少方面还有很多不如人意的地方,但我们看重的是这个软件所使用的这种技术。可以说如果将来这种技术真的完善后流行,那么除了 Napster 的地盘能进一步扩张外,我们也能方便地从几个现在互不相通的“局域网”里面查找其巨大的资源了。}
