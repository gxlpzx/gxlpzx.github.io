{微软的新游戏}

{ 去年3月份,比尔·盖茨在游戏开发者会议(Game Developer’s Conference)上宣布进入游戏机领域以来,XBox这个要与索尼PS2争夺游戏机天下的产品,一直让业界为之企盼。去年9月份,NVIDIA公司宣布将为XBox提供核心GPU(Graphics Processing Unit,图形处理器),同时NVDIA也将提供全新的MPU(Media Communications Processor,多媒体通信处理器),为XBox提供强劲的声音与网络系统。随后在11月份,希捷和西部数据分别宣布将为XBox提供硬盘存储系统,加上英特尔Pentium Ⅲ处理器的配合,XBox在人们心里的迫切期待度是显而易见。
在新世纪的拉斯维加斯消费者电子展(Consumer Electronics Show,CES上,微软正式向全球展出了第一台XBox的样品,揭开了这个未来游戏机霸主的神秘面纱。
接口
XBox看起来比人们想像的要大得多,它有12.2英寸长,9.8英寸宽,3.5英寸高,与PS2相比外观要漂亮得多和大得多,甚至超过了世嘉的土星游戏机。它的通信接口通过主机后面的100M以太网卡接口实现,相对于其他游戏机平台的窄带网络接口,它似乎更像一个宽带的控制器平台。
XBox的视频接口采用独特设计,仅通过一个端口实现音频和视频的输出,这也就意味着在连接音视频电缆时不会接错。不过问题也就随之而来,微软将强制用户必须使用特殊的连接线才能使用其独有的接口。例如想要XBox全面支持HDTV,用户必须购买微软指定的连接线,这根连接线上整合了视频插头以及供音频使用的同轴数字插头。对用户来说,这些复杂的连接插件是很麻烦的一件事,但微软仍极力地吹捧这个音频与视频连接器,至于到未来HDTV应用时,XBox是否可能有所改变,也未可知。
游戏手柄
一台好的游戏机的关键部分还包括它的游戏手柄,XBox的游戏手柄是个经过精心设计增加了诸多控制功能的产品。它的外形很大,握起来非常舒服,在主控制手柄上,微软设计了六键结构,其中包括四个主键和两个副键,所有按键都采用压敏感应设计,具有非常好的灵敏度。在游戏手柄上还有两个设计突起的摇杆,一个位置在左上部,另一个在右边六键的下边,对于用户的使用可以说是非常方便。在手柄的中间还有两个附加按键,上面标注着“Start”和“Back”。游戏手柄有“Start”键并不奇怪,但那个“Back”却让人有些奇怪,看起来是要用在网页浏览上。在微软的几个硬件产品上都包括有“Back”按钮,目前都还不能说明具体用处,或许是为未来功能开发所预留的吧！
4端口
在XBox主机的前面板一共有4个控制端口,这显得更为实用,以前索尼的PS2因为只有两个控制端口而被不少用户责难。4端口可以让4个用户同时操作XBox,给网络联战带来了极大的方便。
其他
此次微软在CES上还演示了运行在XBox的两款游戏,现在许多游戏公司都在积极运行在XBox上的游戏,当然微软自己也可能为XBox开发专门的游戏。另外微软还表示未来的XBox还将推出带麦克风的产品,游戏玩家可以一边玩游戏,一边通过IP电话进行交流。
争论与期待
微软只展示了XBox主机样品,各方面的技术规格都远超过索尼的PS2和任天堂,借助微软帝国的强大技术支持,XBox在游戏机市场中异军突起更在意料之中。不过XBox还是引起不小的争论。
争论的焦点之一是,XBox不支持DVD影碟的播放,大家都很清楚,微软对于普通用户来说,它不是游戏公司,而对于那些狂热的游戏机爱好者来说,它又不是像任天堂那样的游戏机巨擘。微软在研发XBox首先想到的就是它只是一个游戏的机器,因而它尽力避免与索尼的PS2争夺相同的市场而掉入失败的陷阱。在日本,PS2除了游戏机功能外,更多的是被用户当做廉价的DVD播放机使用甚至在美国上市的PS2,它的主要功能还是DVD播放机。这样XBox作为游戏机,微软并未特别提供DVD播放功能。不过微软仍会研发一个不同的XBox版本,增加DVD播放功能,同时也会配上红外遥控器。
第二个争论的焦点在于微软提供的XBox产品规格并没有如人们所愿。例如核心的图形处理器速度从原来的300MHz降到250MHzDVD-ROM的速度没有明确标识,只是2×-5×,而多边形最大执行性能也降低到125M/s。不过,这些规格的降低,并不一定会引起游戏玩家的注意。从实际意义上来说,XBox不是一台PC,并不需要运行高级的操作系列与众多应用程序,现在的配置已足以应付游戏平台的需要。
微软公布的XBox的发布日期是在今年的秋季,玩家的期待恐怕也像当时期待PS2一样激动。PS2刚推出时,掀起席卷全球的抢购狂潮,XBox是否会重新掀起游戏机市场的风暴,都是值得每个业者、玩家关注的。总之,XBox将证明微软进入游戏机市场的实力,这也是微软公司开始的一个新游戏,在操作系统这个游戏中,微软总是抢尽先机,而在XBox这个新游戏上,微软还能继续成为赢家吗？
}
