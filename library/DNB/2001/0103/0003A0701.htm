{前途光明广电股}

{ 记得1998年和我一个台湾网友在网上交流,他说台湾大部分人已经通过有线电视网上网了,方便且速度极快,这让用28.8K“猫”上网的我好不羡慕。这才知道原来有线电视线还有更多更好的用途。
进入2000年,中国股市在大肆炒作网络股的同时,广电概念也一起“鸡犬升天”。广电股是对涉足广电网络领域的上市公司的泛称,包含目前大家俗称的“宽带网股”、“三网合一概念股”、“有线网络股”等。没有很严格的划分,主要指涉足有线网络建设、宽带网络建设及其有关设备制造的上市公司。上网的接入媒介多种多样,除了我国运用广泛的电话上网外,在国外使用更多的是通过有线电视接入,至于电缆上网等新技术尚未推广。在深沪股市中正因为看到了网络的美好前景,有众多的上市公司加入到广电行业投资中。从广义上说广电股应归属网络概念,但在我国它具有很强的独立性,主要是在政策上关于电信和有线电视系统业务平行发展的政策界限,使它具有单独分离出来进行分析的必要。虽然许多上市公司涉足广电业是关注网络发展的前景,业务上也有交叉,但局限于政策界限,目前更多是介入有线和宽带网络基础建设而并没有能和互联网接轨。
我国上市公司对有线电视网的投资时间并不长,大部分公司是在1999年“5.19”行情启动后,有线电视网的投资价值初步被市场认可的背景下进行的。正是由于它和网络的密切关系,在2000年年初,广电股的走势和网络股如出一辙,市场也将其归并到网络股一起给予了充分的炒作。其后也和网络股一起进行了长时间的盘整。但下半年情况出现了一些变化,变化来自于政策上的松动。2000年10月11日通过的《中共中央关于制定国民经济和社会发展第十个五年计划的建议》中明确提出,要“加强现代信息基础设施建设。抓紧发展和完善国家高速宽带传输网络,加快用户接入网建设,扩大利用互联网,促进电信、电视、计算机三网融合”。这就是俗称的“三网合一”。广播电视传输网络在进行网台分离和企业化改造后,可以申请开展有关的电信业务。当然,电信网也可开展广播电视节目的传输业务。这样竞争在原来的一个垄断行业内全面展开,将给用户带来实惠,给企业带来新的利润增长点。
在2000年的股票二级市场上,中信国安已成为带动整个广电股板块走强的龙头。理由如下：中信国安的母公司与中信泰富的重组；它是目前国内拥有最多的有线电视网络的上市公司,并还将继续扩大规模；利用中信国安和中信泰富在宽带网络方面的优势(前者的有线电视网络与后者的全国光纤干线网),中信集团可能会申请经营电信业务的许可证,因其是国务院直属的公司,来头很大,会给人留下想象空间。所以,中信国安依据以上题材走强,而它的走势将成为整个广电板块的风向标。从走势上看,2000年1月到3月中信国安和其他科技股一样强劲上行,后来虽和其他科技股一样步入了漫长的调整之中,但其调整幅度大大小于大部分科技股,并有多次反弹表现,表现出了极其强的上攻欲望。
深沪两地股市中,有数十家上市公司涉足广电领域,其中涉足有线电视网络运营的有：聚友网络、中信国安、赣能股份、电广传媒、特变电工、青鸟天桥、东方明珠等,涉足城域宽带网运营的有：广州控股、重庆路桥、华侨城、特变电工、长城电脑、深科技、福地科技等,涉足宽带网信息服务的有：维科精华、北亚集团、广西斯壮等,涉足有线电视网络产品制造的有：青鸟华光、数源科技、浙江创业、广电股份、大显股份、前锋股份、四川长虹等。
随着国家政策的开放和落实,将迎来新一轮的广电网络建设高潮。由于有线广电网络深入民心,广电概念股较容易被广大股民接受,可以预期2001年广电股的良好走势。我们认为投资广电股应注意如下方面：
●广电股有“硬件”概念和“软件”概念之分,涉足有线电视网络运营、城域宽带网运营和宽带网信息服务的是“软件”广电股,涉足有线电视网络产品制造的是“硬件”广电股。一般来说在广电建设初期首先得利的是硬件厂家,但随着投资厂家的利润增加,较易趋同于社会平均利润,而涉足“软件”投入的公司一般较“硬件”厂家迟获利,但因为有线电视网络运营、城域宽带网运营和宽带网信息服务目前一般还是特许营运,所以获利将是长期和稳定的。
●广电网的投资价值将会成为下一轮股市投资热点。广电网投资收益将从2001年起对上市公司的业绩逐步产生影响。随着我国宽带互联网的快速发展和有线电视网络发展的规范化、产业化,有线电视网的投资收益将日渐明显,参与有线电视网投资的上市公司业绩将因此得到较大提高。
●政策因素对广电股的走势有很强的影响力。政策的导向以及政策的开放和落实程度将成为影响广电股走势的决定性因素。
●广电股与其他科技概念股的相互交叉和影响。它和网络概念的相辅相成自不必说,走势良好的“光谷”概念,其核心企业主营业务是光器件和光传输系统,也属于宽带网络建设的基础设备。它们的走势也将极大地影响广电概念股。
}
