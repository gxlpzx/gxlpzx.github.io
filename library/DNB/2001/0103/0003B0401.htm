{ 解答区
Q 1.1 在“我的电脑”的“属性”里显示“兼容模式分页技术降低了整个系统的性能”,我怎么才能取消兼容模式？
A：发生此问题一般有以下三种情况
1. 硬盘引导区有病毒导致系统性能降低,请用杀毒软件清除病毒；
2.由于安装了与硬件不相符的驱动程序或者在CONFIG.SYS等配置文件中使用了如“PATH A:\”的不恰当配置会降低系统性能,其明显表现就是速度降低。解决方法就是删掉带红叉的设备,重新安装带感叹号的设备,并在配置文件中去掉不恰当的配置。
3. 32位保护模式磁盘驱动程序被禁用,请右击“我的电脑→属性→系统属性→性能→文件系统”,在对话框中选择“疑难解答”,确认“禁用保护模式的硬盘中断处理”和“禁用32位保护模式的磁盘驱动程序”两个选项的复选框没有被选中。(简洪涛 马晋)
Q 1.2 在使用诺顿系统工具检查系统时,提示htmdeng.exe程序缺少运行所必需的ipcclient.dll文件,该怎样修复这个错误？
A： 到其它电脑上查找ipcclient.dll文件,并将它拷贝到诺顿安装文件夹下和系统文件夹(System文件夹)下。
Q 1.3 我的机器刚刚装上Windows 2000系统,请问怎样添加“输入法生成器”？
A：中文版的Windows 2000在安装时已将“输入法生成器”安装到系统上了,只是没有在开始菜单上建立快捷方式而已,要运行它可直接单击“开始→运行→浏览”,在“X：\Program Files\Windows NT\Accessories\”(X为系统安装盘)下找到“Imegen.EXE”运行即可。
(郑良 彭宏俊)
Q 1.4 想在每次用网络蚂蚁下载完一个文件后自动运行病毒扫描程序,该如何设置？
A：打开“网络蚂蚁”,单击“选项→参数设置→自动操作”,在“下载的文件进行病毒检查”前打钩,最后填入“病毒扫描程序”和“参数”即可。(马晋 韶亚军 金斌)
Q 1.5 前几天由于不慎把“c\pdos.def”文件删除,导致无法运行PDOS95.BAT,可是我无论是从其它机子上把PDOS.DEF拷贝到我的机子上还是拷贝光盘上的PDOS.DEF,还是不能运行PDOS95.BAT,请问如何解决？
A PDOS.DEF其中包含了启动PDOS95的路径和字体名称,其实删除掉也没关系。重要的是PDOS95.BAT中启动中文DOS的几个文件：Pbios.exe、Font16.exe、HzviO95.exe、Instdict.exe、Ctrlpan.exe及输入法文件：Pinyin.tbl、Shanpin.tbl、Quwei.tbl、Guobiao.tbl等等,要确定这几个文件没有被删除或破坏才能运行中文DOS(请参见《电脑报》2001年第2期B4版《制作中文启动盘》一文)。如果要在任意目录下执行,还要先执行“PATH X:\WINDOWS\COMMAND”(X为系统安装盘)才行。
