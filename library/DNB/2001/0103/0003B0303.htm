{桌面装饰软件WindowFX}

{ WindowFX属于StarDock的“桌面装饰家族”,它功能强大,操作方便,绝对能令“个性一族”心动！该软件的安装简单,只需一路回车即可,主界面如图所示。
Desktop Icon(桌面图标)主要用于设置图标的属性,可以选择“Make desktop icon background transparent”(使桌面上图标的背景透明化),以免漂亮的壁纸被图标挡住。另外,点击“Set desktop icon text foreground”(设置桌面前台图标),再在选项后面点击“Set”(设置),然后就可以进行诸如图标颜色方面的设置。
System(系统)用于整体的设置,里面有两个选项和一个选择窗口。一个是“Run WindowFX at Startup”(WindowFX在系统启动时自动加载),另一个是“Show the WindowFX icon in the system tray”(WindowFX运行时缩小化)。最下面的选择窗口里,可以选择是否为应用程序添加WindowFX效果,先在对话框内选择要进行设置的程序,再点击“Add exclusion”(添加效果)和“Remove exclusion”(去除效果)就可以了。
该软件其他功能还有设置透明化(Transparency、窗口拖曳(Windows Dragging)、设置窗口最大尺寸(Maxinize to set Size等,不过非常遗憾某功能只适用于Win2000在Win98下则无法使用。
另外,如果同时使用WindowBlinds和WindowFX两款软件,要注意WindowBlinds需要1.3版或更新的版本。下载地址：http://www.stardock.com。}
