{HyperSnap-DX4使用技巧}

{ HyperSnap-DX4是一款功能强大的抓图软件,在这里我们介绍几个实用的技巧供大家参考。
一、捕捉鼠标
有时为了演示操作过程,需要将鼠标的位置一并捕捉,这时你可以选择“Capture(捕捉)”→“Capture Settings(捕捉设置)”,在“Capture”标签页中,将“Include cursor image(包括鼠标)”前的方框打上“√”。这样,HyperSnap-DX就会在抓取图像时同时将鼠标一并抓下来。
二、高级覆盖控制
某些DVD及覆盖设备会“谎报”它们的数据格式。如果你捕捉到奇怪的颜色,不妨试着选取“Capture”→“Enable Special Capture(DirectX Glide)(允许捕捉特殊画面)”,在弹出的窗口中点击“Advanced Overlay Control(高级覆盖控制)”,另选一种数据类型,然后重新捕捉。注意,有时需要捕捉好几次,才能最终确定正确的数据格式。
三、捕捉滚屏图像
相信很多朋友在Windows下抓图时都碰到过这种情况：想抓取的目标画面太长而在一屏上显示不了,例如网页的内容,须用“滚动条”上下拖动才能看到全部内容,这时用一般的抓图软件最多只能抓取整屏的画面,如果想抓取整幅画面,只好分屏抓成几个文件,再用绘图软件把它们拼起来。而HyperSnap-DX 4却突破了这个限制,可以自动滚屏抓取图像。鼠标点击“Capture”→“Capture Settings”,在“Capture”标签页中,选中“Auto-scroll window during Window Capture(在捕捉窗口时自动滚屏)”,HyperSnap-DX就会“自动滚屏”抓取图像；如果嫌自动滚屏刷新时间太慢,可以在“Auto-scroll(自动滚屏)”中设置刷新时间,默认是20毫秒。
有些朋友说已经设置了“自动滚屏”,仍然无法抓取滚屏信息,事实上,应该使用工具栏上的“Capture Window”图标或单击“Capture→Window,Button or Control”命令来抓取图像。
四、连续抓取多张图像
有些游戏软件运行时不能使用“Alt+Tab”键切换程序,并且锁定了一些抓图的快捷键,这样,一般的抓图方法就失效了。对此,我们可以让HyperSnap-Dx自动捕捉图像,单击“Capture”→“Capture Settings”,打开“Capture Settings”对话框,在“Quick Save(快速保存)”标签页中选中“Automatically save each capture to a file(每次捕捉完后自动保存)”,这样就可以在不退出应用程序或游戏的情况下依次将捕捉的图像保存到指定的地点。
为了确保图像捕捉的精确性,你可以通过“Quick Save”菜单来控制HyperSnap-DX连续抓取图像。“File”是自动保存的文件名,可点击“Change(更改)”来指定自动保存的地点、以及存储的图像类型。
如果选中“Increment file name(自动以增序文件名保存)”,HyperSnap-DX就会依次将抓取的图像按1、2、3……保存起来。请注意,HyperSnap-DX自动抓取的图像会保存为BMP图像,默认的是连续捕捉100幅图像,如果没有100G硬盘,用不了多久你的硬盘就会挤爆！因此我们应该在“Change”按钮中将自动保存的图像类型更改为JPEG图像,在“stop saving(停止保存)”栏中输入停止抓取图像的上限。如果因故要临时终止抓取图像,可按“Shift+F11”快捷键。
有时HyperSnap-DX会因为系统出错而停止捕捉图像,建议把“Continue on err(出错时继续抓图)”前打上钩,这样就不会终止捕捉图像。
为确保抓取的图像是连续的,你可以在“Repeat first capture(重复第一次捕捉)”栏中设置自动抓取间隔时间,设置单位为秒。如果有需要,可以使用小数输入,输入0则为禁用。
五、抓取编辑区域
我们抓取图像时,有时不需要窗口的边框,只需要编辑区域。这时我们可以选择“Capture”→“hctive Window without Frame(去掉当前窗口的边框)”或按快捷键“Ctrl+Shift+C”。这样,HyperSnap-DX就会将那些标题栏、菜单栏、工具栏去掉,仅抓取编辑区。}
