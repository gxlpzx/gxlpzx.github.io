{瑞雪纷飞兆丰年}

{ 又到了大家互相祝贺新年新气象的时候了,可是除了到各个贺卡中心去挑选中意的贺卡发送给好友之外,自己精心制作一张漫天飘雪的贺卡,岂不是更表心意？其实制作这样的贺卡一点也不难,使用一个名为Anfy小软件就可以很轻松地实现,前后绝不会超过5分钟,不信？请看：
1.安装好Anfy简体中文版。目前最新的版本为Anfy 1.4.5。如果手头上没有,可以到http://www.anfyteam.com/ln/chisimp/ajdownl.html去下载,大小约为2917KB。
2.选择好一张自己喜欢的图片(最好是有雪景的图,这样看起来就非常自然、逼真)并将其裁剪为合适的大小。
3.执行Anfy简体中文版,在种类中选择图像效果；然后在旁边的下拉式菜单中选择SNOW；执行下一步。
4.将背景图片改为自己选好的图,并设置好各项参数。执行下一步。
5.在“TEXT”一栏中输入自己想对朋友说的话,并调整好字体的大小和颜色。
6.按提示一直执行到最后,单击“复制全部文件到”按钮复制全部文件到你选的文件夹中,并指定要创建的文件名。
7.好了！点击你制作好的页面看一看最后的效果吧,没有问题的话,就将生成的所有文件打包发送给你的好友,请他(她)将这些文件全复制到任一目录下。哈哈！她一定会有一个意外的惊喜！
}
