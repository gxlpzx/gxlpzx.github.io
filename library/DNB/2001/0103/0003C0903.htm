{懒鼠整治记}

{ 近几个月来,一向灵活的爱鼠开始发起懒来,指上不上,指下不下,时而还采取“跃进”式移动——竟然消极抵抗！反正,就是不让你轻松地指到要去的地方。如此懒鼠,怎能不治。
既然行走不便,定是污秽作乱！打开后盖,拧下螺丝。定睛一瞧,果不其然,滚轮上的污物都快有滚动球大了。立即清除！不到10分钟,净亮的滚动轴已在眼前。装回清洗一新的滚动球,拧回螺丝,上机实验,先前鼠标移动的症状还是存在的。我想既然是刚刚清洗干净,那么这个就不是污物所致了,又因为鼠标还能使用,那么这也不是电路方面的原因。再次小心地拧开后盖,认真查看每一条连线,都很完好整齐。在查到滚动轴时,发现滚轴表面有些磨沙,并不十分光滑。哦,原来症结在此！鼠标整日牵拉移动,致使滚动轴与滚动球长期接触,表面严重磨损,已经有所减薄,不能与滚动球紧密接触,所以滚动起来有晦涩延误之感。已经查清病因,医治自然不在话下。取小刀一把,在滚动轴上斜向刻一小槽(划一下即可,不可太深),匀速转动滚轴,以第一个斜槽为基准,划等距平行线若干,直到转动一周为止。然后,在以此组平行线为基准,斜向45度再划一组平行线,最终使滚动轴表面形成渔网状条纹,另一滚轴也照此办理,而后装回滚动球。装好鼠标再次试验,觉得鼠标移动已经变灵敏。
以往文章都说滚动失灵为污物所致,其实对于大多数按键先于滚动轴损坏的鼠标来说是十分正确的,但一个容易让人忽视的问题是,对于有一个十分耐用的爱鼠的朋友来说,有可能你的按键还灵活时,你的滚轴已严重磨损失效了。所以,问题还是不能一概而论,具体问题具体分析。
}
