{网络时代的财富}

{ 网络咨询业在如今的互联网产业发展中已经开始扮演越来越重要的角色了。在世界范围内,IT咨询业和作为其子业的网络咨询业是方兴未艾的朝阳产业。国际IT咨询业1998年的总收入,占咨询市场总收入的2/3。1999年,24个主要从事网上购物和互联网资讯业的公司收入在一年内翻了一番。Forrester公司预测互联网咨询市场的规模将从1999年的106亿增加到2003年的648亿美元,互联网网络和电子商务时代的管理咨询与信息服务业将成为世界上最有发展前景的行业之一。
曾几何时,中国的互联网咨询业是一片空白,土鳖们习惯于作坊式的管理和发展模式,而海龟们则机械的复制国外的成功模式。随着越来越多的互联网企业诞生、竞争和淘汰使“网站的运作需要专业的知识和经验”被绝大多数人所认同。网络咨询业也因而浮出水面——麦肯锡、波士顿、安达信,HP、IBM陆续推出了网络咨询服务,而互联网实验室和时代财富的诞生更是意味着国内咨询企业正式介入网络咨询业。
如果仔细分析一下国内的网络咨询企业,那么我们便会发现,按照目标市场来划分的话,大致可以把它们划分成四种类型。
第一种是以HP Consultant、IBM Consultant为代表的技术咨询公司,他们可以告诉客户企业网络搭建中哪些服务器需要升级,使用这样的设备和技术业务能够达到什么样的规模,与竞争对手的差距能够拉开多大等等业务层面的问题。这类网络咨询企业的客户主要集中于对技术要求特别苛刻的互联网企业。
第二种是以互联网实验室、CII-互联网研究发展中心为代表的基础理论咨询公司,它们更关注的是中国互联网产业的宏观层面,对于现状进行调研,对发展趋势进行判断,咨询业务主要是通过研究报告的出售来完成的。
第三种是以传统的麦肯锡、安达信、波士顿为代表的传统企业咨询公司,它们侧重于宏观的战略决策及管理咨询,能够诊断出企业运营和管理上的问题。
第四种是以时代财富为代表的总体策划咨询公司,它们提供从网站策划、网站技术、网页设计到人才管理、推广策划、营运、融资咨询、提供咨询报告等全方位咨询,业务对象除了潜力巨大的“水泥+鼠标”的传统企业外,还有来自世界各地的风险投资公司。
前两类网络咨询企业的业务范围相对较小,虽然集中优势力量很容易成为那一方面的最优秀者,这应该是毫无悬念的。但是就整体的网络咨询业而言,他们不可能成为国内最大最好的咨询企业。就像在国际市场上麦肯锡的企业战略咨询是最优秀的,但是它的业务范围往往局限于此,所以最大的咨询公司是横跨战略咨询、财务咨询、调查等多个方面的安达信(见美国非公众企业500强排名,安达信咨询和安达信会计分列12和14,而麦肯锡排名为48)。所以在国内互联网咨询市场中,最有看头的还是后两类咨询公司,在这两类咨询公司中,更为业内人士看好的是后一种,也就是时代财富为代表的本土总体咨询企业。
看好时代财富,并不仅仅因为创始人张静君和杨震霆曾经经营过163.net、广州视窗等国内一流的网站,也不仅仅因为国内INTERNET的发展恰好到了呼唤专业顾问服务的节骨眼上。当然这些也很重要,但更重要的是因为时代财富在诸多方面的综合优势。
先从人才角度来看。咨询业提供的服务是咨询,咨询是完全靠Consultant们的脑子做出来的,所以人才是影响咨询企业好坏最关键的因素。而人才上的优势也一直是麦肯锡、安达信、波士顿等传统老牌咨询公司最引以为自豪的。但是,在如今的新经济时代,老牌咨询公司惯有的人才选拔制度,是否仍旧能够成为人才优势的保障值得怀疑。的确,在传统经济时代,老牌们做得到这一点,因为这方面他们有着足够丰富的成功案例和足够深厚的理论研究成果,可以用来做培训,可以把一个外行培养成咨询业的“熟练技术工人”。可是在新经济时代呢,互联网企业孰胜孰败还未见分晓,哪有多少参考的案例？至于相关理论,学术界尚且争论各不休,更何况咨询公司。缺少了这些,本来就没有足够专业背景以及相关经验的Consultant们又怎能够把璞玉变成美玉呢？
事实上,在目前的互联网界,从业的经验及成果以及良好的市场感觉才是最为关键的。然而大量这方面的人才却因为语言不过关,因为学历不过关而被拦在了老牌咨询企业的门外。与此正好相反,时代财富招聘人才不止看学历,更看重能力。“公司里水平最高的设计高手是学会计的”(语出张静君),真正把一批能够在中国互联网背景下经营的人才招纳于旗下。虽然员工的学历远不如老牌咨询公司动不动就多少个哈佛的MBA那么辉煌,但就实战能力,恐怕只有优势而没有劣势。
再从本土化的角度来看,麦肯锡们属于外来的和尚,他们对于中国互联网的研究更多的是来源于对于国外特别是美国互联网研究成果,而咨询的整个模式更多是照搬在美国成功的模式。但是问题在于,中国的经济制度有其特殊的地方,中国的制度文化内涵特别是企业文化有其特殊的地方,中国统计数据的披露、准确性有其特殊的地方,甚至中国的互联网市场因为种种原因也有其特殊的地方。如果老牌国外咨询公司们只是将国外的经验汉化,而没有足够的本土化,那么效果将大大折扣。从以往的案例来看,即使是在传统行业,国外的老牌咨询公司因为本土化工作不利,没有根据中国的某些特殊情况进行特别的考虑,导致整个咨询工作以失败告终的也不在少数,否则在传统行业,国内的咨询公司早就被清扫出场了。
还有对于互联网的产业研究方面,我们很惊讶地看到时代财富不仅致力于它们所擅长的实战操作上,而且还在中国互联网及企业电子商务的发展方向、以及风险投资与资本市场方面都有独特的理解。后来我们才知道,原来时代财富每月一期的《时代财富——互联网观察》竟然是含金量极高的内部专业研究月刊。一个实践能力很强的顾问公司,辅之以高水平的行业研究,其前景更加不可限量。
最后从社会资本的角度来看,老牌咨询公司在他们的本土有着足够强大的社会资本,与政府机关的关系、与媒体的关系、乃至与大量研究机构的关系。然而社会资本的最重要的一个属性就是不能转移性。老牌咨询公司来到中国,品牌可以带过来,资金可以带过来,员工可以带过,可是社会关系网却是无法带过来的。而且由于本土化进程不利,它们在中国的社会资本,特别是社会网络的重建进程,始终缓慢。
于此相反,时代财富这方面有着无可比拟的优势。张静君和杨震霆两位创始人作为中国互联网界的元老,本身就有着许多别人无法望其项背的社会资本,而且由于他们对于国内互联网产业的深入了解(无论是从政策、经济、文化角度)也使得时代财富在这方面的优势不断扩大。
}
