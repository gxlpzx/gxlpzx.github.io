{Windows 2000初级实用技巧（续）}

{ 编者按：Windows 2000的推出已有一段时间,想必朋友们的机器中不会少了它吧。在新年第1期《电脑报》H4版的资料特辑中,我们曾经介绍过一些关于Windows 2000的初级实用技巧,今天我们还将介绍一些实用的技巧,希望朋友们在使用Windows 2000时更加得心应手。
1.如何恢复双启动菜单
系统中安装了Windows 2000和Windows 9x/Me两套系统的朋友们都知道,在计算机启动时会默认出现一个双启动菜单,让你选择以何种操作系统启动。可能有些朋友使用了Windows 2000一段时间后,由于种种原因觉得还是Windows 9x操作起来更得心应手一些,索性就在“启动和故障恢复”(右键单击“我的电脑→属性→启动和故障恢复”)窗口中取消“显示操作系统列表”项,这样就可以在启动时不再显示操作系统列表,自然系统启动也大大加快。不过,假如当你的硬件升级之后,又重新选择Windows 2000时,还是恢复双启动菜单吧,但因为启动时不出现双启动菜单,那怎么进入Windows 2000呢？其实,只要在开机自检和启动前按住“F8”键(当然要速度快一些),就会出现原来的双启动菜单了。
2.在DOS下更改默认操作系统
一般情况下,我们都是在Windows 2000系统中选择默认的操作系统和等待时间设置。但是,在DOS下也可以完成,这对于系统不能进入Windows 2000时大有作用。启动MS-DOS,输入“EDIT”,启动DOS环境下的文本编辑器,再打开C盘根目录下的boot.ini文件,参考如下内容进行修改(先从资源管理器的“工具→文件夹选项”菜单下“查看”对话框中,设置为“显示所有文件和文件夹”才可以打开这个文件；不能进入Windows 2000时,在DOS下使用“attrib -a -s -h boot.ini”命令)：
boot loader
timeout＝30
default＝C:\
operating systems
C:\＝″Microsoft Windows″
signaturebc91bc91disk0rdisk0partition3\WINNT＝″Microsoft Windows 2000 Server″/fastdetect
3.打开Ultra ATA硬盘模式
Windows 2000对Ultra ATA 33/66硬盘接口模式提供了极好的支持,一般情况下对符合这一接口规范的硬盘都不成问题。朋友们可以按照如下步骤来打开Ultra ATA 33/66模式：
单击“系统→属性→硬件→设备管理器”,在设备管理器的“IDE ATA/ATAPI控制器”项下的“Primary IDE Channel”中的“属性”设置窗口进行检查,只要这里“高级设置”选项卡中将主设备的类型设置为“自动检测”,并将传输模式设置为“DMA”(如果你的硬盘的DMA可用的话),“Secondary IDE Channel”进行同样检查,结果一致的话,说明你的硬盘肯定是工作在Ultra ATA模式下了。
4.恢复最近一次的正确配置
与Windows 98不同的是,Windows 2000的运行极其稳定,但微软仍然为我们提供了“恢复到最后一次正确设置”的功能,万一你由于安装各种形式的新软件或升级驱动程序而导致系统出错的话,连安全模式也无效的话,那么可以在看到提示“请选择要启动的操作系统”后,按住“F8”键,选择“恢复到最后一次正确设置”项,以使用 Windows 上一次关闭时所保存的注册表信息来启动 Windows 2000。不过,这一选择一般只在无法启动系统或配置不正确时才可以使用,它不会解决损坏或缺少驱动程序或文件所导致的问题,但最后一次成功启动以来所作的任何更改也将丢失。
5.如何及时更新系统文件
单击“开始→Windows Update”能及时更新系统文件,通过它你可以访问微软站http://www.microsoft.com/上的 Windows Update,及时获取驱动程序、补丁程序、帮助文件及因特网产品等的更新信息。它可以扫描计算机中过时的系统文件并自动替换成最新版本。不过,你必须作为管理员(Administrator)或管理员组的成员登录系统后才能完成这个更新系统文件的过程。
6.使用自带的内码转换工具
过去,假如你需要阅读BIG5编码(一种繁体中文编码方式)的网页或其它信息,需要一些类似南极星之类的转码工具软件才能完成。现在可好了,只要在Windows 2000中打开“程序→附件”中的“代码转化工具”,在“来源”的窗口打开需要转换的BIG5编码的文件,然后在“选项”中设置好转换要求,就可以轻松地在简体和繁体之间快速转换。用“代码转化工具”还可以进行半角和全角的转换,并支持文件和剪贴板的输入。
7.将分区格式设置为NTFS
Windows 2000的分区格式设置为NTFS格式,有助于更好地发挥系统性能。假如原来的系统分区是FAT16或FAT32格式,只要单击“开始→运行”,输入命令“convert C/FSNTFS”就可以完成,不过NTFS格式的分区下可无法被Windows 9x访问,而且一旦由FAT16/32转换到NTFS格式,这个过程将是无法用“convert” 命令来逆转的(可以使用魔术分区大师来恢复),转换前一定考虑清楚。
8.最大限度地节约电能
从“设置→控制面板→电源选项”打开“电源属性选项”对话框,从“电源使用方案”下拉列表框中选择“最少电源管理”,从“关闭监视器”下拉列表框中选择“一分钟之后”,从“关闭硬盘”下拉列表框中选择“三分钟之后”,从“系统待机”下拉列表中选择“从不”。这样,只要用户超过一分钟不使用键盘和鼠标,系统就会自动关闭显示器、硬盘以节省电能,只要一按键盘或鼠标,系统又恢复原状。
9.如何与他人共享文件夹信息
我们可以与网络上的其他用户共享文件夹或驱动器的信息,只要选定该文件夹或驱动器后,从右键菜单中选择“共享”命令,打开“共享”对话框,从“共享”选项卡上,单击“权限”。假如要设置共享文件夹权限,只要单击“添加”按钮,然后键入要设置权限的组或用户的名称,单击“确定”关闭对话框。假如要删除某位用户的共享权限,请在“名称”中选择组或用户,然后单击“删除”。
但要注意的是,共享文件夹权限仅应用于该共享文件夹中的所有文件和子文件夹,并且仅当通过网络访问该文件夹或文件时才有效。当在本地打开该文件夹或文件时,共享文件夹权限不起保护作用。要保护本地计算机上的文件或文件夹,请使用 NTFS 权限(参见《电脑报》2001年第1期H4版《Windows 2000初级实用技巧》一文第九条)。
}
