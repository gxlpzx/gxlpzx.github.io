{手机也能发贺卡}

{ 春节将至,或许你想给朋友邮寄一份祝福。可是,紧张的工作、买贺卡、跑邮局……让你头痛不已！发电子贺卡,你的朋友又无法上网,无法感知那段温馨的时刻。别急,就让《神通助理》当你的“爱心大使”吧！你只须做的就是轻点几下手机按键,一份精致的贺卡、温馨的祝福、动听的音乐,瞬间就会传递给你的朋友。
下面请跟我一起一步一步做吧！
第一步：你的手机必须有接收中文短消息的功能(没有？那就换一个手机！或者登录http://www.any8.com﹖any8网站直接发送)。
第二步：开通手机服务
请发送短消息：“KTFW”(“开通服务”的拼音首字母),发送,发送号码：2010(2010为深圳地区号码,其他地区请看下表)。
你将收到反馈短消息：你登录www.any8.com的用户名为“13×××××(你的手机号码)”,密码为“×××××(系统自动生成)”。开通成功！
第三步：发送贺卡或音乐
在信息栏中编辑以下内容：
fshk＃朋友手机号＃数字(1～300任意一个数字)＃留言,其中fshk为“发送贺卡”的拼音首字母。
发送2010,OK！你的朋友将收到你发去的手机贺卡。
发送音乐同理：
fsyy＃朋友手机号＃数字(1～50任意一个数字)＃留言。
注：1.如果仅使用了“fshk＃朋友手机号”,朋友收到的是默认贺卡及祝福语。
2.朋友的手机必须是诺基亚手机。
例如：给我的朋友张三(手机号：139××××××)发张贺卡(“恭喜发财”,位置号3),发送格式为：
fshk＃139xxxxxx＃3＃恭喜发财
嘟……嘟,张三的手机收到画面！
嘟……嘟,张三的手机收到短信“神通助理用户小虫给您发来贺卡：恭喜发财”。
以上办法适用于无法上网的手机用户,你也可以直接上网(http://www.any8.com﹖any8)发送贺卡。
试试看,先给自己发一张吧！ }
