{提高主页排名的工具}

{ 一、TopDog Web Position Analyzer/Submitter
最新版本：5.8.0.115
软件大小：3623KB
授权方式：共享软件
应用平台： Windows 95/98/NT/2000/ME
下载网址：(1)http://newhua.infosail.com/down/TopDogSetup.exe；(2)http://www.cnvnet.com/download/d/TopDogSetup.exe
这个软件可以分析你的站点在顶级搜索引擎的排列情况 然后生成一份易懂的报告帮助你提高站点的排名。它的缺点是不实用功能过多(这只是笔者个人之见),以致体积太大。
二、排名奇兵
最新版本：V1.0
软件大小： 527KB
授权方式：试用软件
应用平台： Windows 95/98/NT/2000
下载网址：(1)http://download.tiandown.com/pub/software/internet/int-prom/toprank.exe；(2)ftp//ftp.tiandown.com/pub/software/internet/int-prom/toprank.exe
排名奇兵是由登录奇兵的作者岑绍辉编写的一个小工具,和登录奇兵一样,是笔者强烈向广大网页制作者推荐的好东东。有了排名奇兵,你就可以分析你所选关键字的使用频率,获得在各大搜索引擎中排名第一位的网站显示给各搜索引擎的蜘蛛程序隐藏部分的秘密 META 标记档案,从而提升你网站在各大搜索引擎中的排名。排名奇兵软件将提供各个关键字在 goto.com 网站每月的被搜索次数。根据关键字每月被搜索次数,你可以知道上网用户在搜索引擎中经常查询哪些关键字,从而得知怎样的网站会更加吸引用户访问。选择被搜索次数较多并且可以与你的网站内容相结合的关键字,输入到排名奇兵软件中,排名奇兵软件则会列出在 yahoo、InfoSeek、HotBot、AltaVista、Excite、Lycos、WebCrawler、Northern Light 等 8 个著名搜索引擎中含有这个关键字并且排名第一的网站,并自动提取排名第一的网站显示给搜索引擎的蜘蛛程序隐藏部分的秘密 META 标记档案。将以上所提取的秘密档案与你网站的内容相结合后插入到你的网页中,配合使用登录奇兵软件将你的网站登录到各大搜索引擎中,你的网站将会获得尽可能好的排名！}
