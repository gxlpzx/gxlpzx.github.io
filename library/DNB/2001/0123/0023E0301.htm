你来，我也来——近期Palm新品回味

从3月份开始 ，Palm OS阵营的新产品就一个接一个地发布，现在我就选其中主要的几种向大家一一作出介绍。
3月6日：m105
Palm要推出505的消息一直受到大家的关注，但在千呼万唤始不见时，Palm m105却率先在3月6日正式推出售价199美元。 m105其实相当于 m100的升级机种，外形与m100一样，操作系统也是Palm OS3．5．1，同样不可升级。内存则从 m100的2M增加到了8M，同步则把m100使用的电缆改成了串行底座。价格则比149美元的m100贵了50美元。
Palm m105 新增了许多功能，附送移动上网工具包(Mobile Internet Kit，MIK)可用手机作为Modem无线上网，实时浏览支持Palm网页剪存技术(web-clipping)的网站内容，还可以用该套件内附的MultiMail及SMS短讯编辑软件，实时收发E-mail(支持POP3、IMAP4)或编辑发送手机短讯。m105还随机附赠了AvantGo软件及之前m100没有的Hotsync Mail邮件同步软件。此次配合m105还推出了14色彩壳，再加之前m100的5款，一共有19款彩壳，使你可以自由选择，加强了流行感及个性化。
从产品定位上看，去年8月m100上市取代了Palm Ⅲe，而m105将取代Palm Ⅲxe。m105发布之后，Palm的网站上就开始停止了Palm Ⅲxe销售。
3月12日：Visor Edge
Palm m105是Palm公司针对低价位消费群体的一款掌上电脑。而在3月12日Handspring 正式推出了Visor Edge，售价是399美元，成为这股新品发布潮中高价位产品的领头军。Visor Edge是Handspring公司成立以来推出的第5款Visor产品和第1款可无限扩充的超薄型掌上电脑，是Handspring公司特别针对PalmⅤx所推出的挑战机种。机身上下盖都是采用铝合金外壳，厚度比PalmⅤx略薄一些，但由于机身比PalmⅤx略长、又带有铝合金上盖，因此重量仍达到136克，比PalmⅤx的117克略重一些。^23050501b^
Visor Edge为单色显示，16级灰阶，采用了Palm OS 3．5．2H2操作系统，不可升级，而处理器采用了33MHz Dragonball VZ处理器，性能明显比Palm Vx的20MHz处理器提升许多，机身上也多了一组隐藏式麦克风装置，这款机器最大的特色是它将Springboard扩展槽分离开来，利用Springboard转换卡连接扩展槽和机身，可以方便地从Visor Edge背面安装和取下。所以这款机器才能在特别薄的同时却仍然具备很强的扩充能力，这正是与现在的数码产品追求轻薄潮流相吻合的。Visor Edge是首款采用金属外壳的Visor掌上电脑，外壳有3种颜色：银色、蓝色、红色。Handsping还同时发布了一款墨黑色的Visor Platinum，售价299美元。
3月14日：索尼 Clie PEG-N700C
索尼在1999年11月宣布获得Palm操作系统授权时，这个电器巨头希望能成为掌上电脑市场上一股强大的力量。然而，当去年8月索尼在美国推出首款Clie掌上电脑后，销售情况却一直不很理想。3月14日，索尼公司的日文网站上发布了新款Clie PEG-N700C的详细信息，拥有分辨率为320320的TFT显示屏，其显示屏分辨率为Palm掌上电脑的4倍(m505或者Visor Prism的是160160)。不过颜色数方面就不是最高的了，为256色TFT屏幕，比起m505和visor prism64k色要差很多。它采用了33MHz的摩托罗拉Dragonball VZ处理器，操作系统为Palm OS 3．5。^23050501c^
最具“杀伤力”的是它独有的多媒体功能，这也是索尼力推的也是对消费者最有诱惑力的。PEG-N700C设有扩音器、一个外部音乐控制器以及一个Audio Player 1．0软件可以播放保存在Memory Stick(记忆棒)上的音乐。被称为Magic Gate Memory Stick的存储器可以提供音乐等内容的版权保护。PEG-N700C还内置gMovie 2．0和TV Scape 1．0。
此款机型是索尼公司的精心之作，它们希望能凭借这款机器奠定索尼公司在全球掌上电脑的地位。但其日文界面还是让日本之外的用户有点望而却步，不过英文界面的N710很快就要面世了，这对于全球的发烧友来说真的是一个无法抵挡的诱惑哟！(未完待续)
