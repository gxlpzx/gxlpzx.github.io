局域网通信软件LAN Conference2．1

某日在网上得到一局域网通信软件LAN Conference2．1，试用后觉得功能全，软件小巧，适用面广，不敢独享，现介绍如下。
该软件不需要服务器端软件，适用于有公共协议如TCP/IP，NetBeui，IPX/SPX的Novell或NT对等网，Win9x系统，它允许多人同时共享同一个讨论组(聊天室)，它也允许在两个不同的客户端传递私人信息(文本，文件，语音)，它也可以建立个人聊天室，压缩后的安装文件才900多K。
软件安装运行后提示输入昵称，输入后回车，软件在屏幕状态栏右下角生成缩略图标，双击该图标，出现如图所示的界面，此时出现的是默认的公共聊天室，在发言区输入所要发的话后回车，在该聊天室的网友都可以看到你的发言。
在图1中的程序图标上单击鼠标右键会出现如下菜单，在此可新建聊天室(New chat room)，也可进行该软件的参数设置(preferences)，出现图所示的参数设置界面。
在图2中，软件运行后的界面上有发言区，发言显示区，用户列表，状态栏。
在发言区输入要说的话，回车即可。单击用户列表中的某个网友，会出现一个功能菜单，在此可以给某人发送私人消息(send a private message to user)；可给某人发送一个信息框，引起他(她)的注意(send a messagebox to user)；发送一个文件(send a file to user)；发送一条语音信息send a voice message to user；与某人进行“对打游戏”(battle user)。
在某些情况下，如果想快速最小化，可点击“用户列表”右上角的图标(三个相连的计算机图标)
在图3中可对软件参数进行设置，如有新的信息到来时发出嘟嘟声(beep when new message is received)在信息前面显示时间标志display time stamp on messages允许发送，接受语音信息和允许发送和接受文件等功能。
