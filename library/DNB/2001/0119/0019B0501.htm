用ACDSee提取文件列表

现在的硬盘容量越来越大，恐怕很少有人能记得硬盘中到底装了哪些东西，如果能制作一份硬盘中的文件夹和文件的清单，一定会带给你不少方便。下面介绍用ACDSee提取文件列表的方法供大家参考：
先运行ACDSee3.0，选择要制作文件清单的文件夹，然后在“View(查看)”菜单中进行如下设置：选择除“Thumbnails(缩略图)”以外的任何一种浏览方式来浏览文件夹；如果只想制作图片文件清单，打开“Hide non-Images(隐藏非图片文件)”选项；如果想将隐藏文件也添加到文件清单中，请打开“Show hidden files(显示隐藏文件)”选项；用鼠标指向“Show Columns(显示栏)”，可以打开其中的所有选项，这样可以在文件列表中显示全部信息；在“Arrange Icons(排列图标)”选项中设置你想要的文件的排列顺序。设置好后，从“Tools(工具)”菜单中执行“Generate file listing(生成文件列表)”命令(或直接按“Ctrl+G”组合键)，ACDSee3.0会将当前文件夹中的所有子文件夹和文件保存到一个名为“folder-Contents.txt”文本文件中，并自动用“记事本”打开该文件。
用ACDSee3.0创建的文件清单，能够显示文件或文件夹的Name(名称)、Size(大小)、Type(类型)、Modified(修改日期和时间)，如果是图片，还可以显示Image Properties(图片属性)和相关Description(描述)信息，但无法对子文件夹产生文件列表。而且每次产生的列表文件都保存在相同的文本文件中(在Windows 98中一般位于“C\Windows\Temp”文件夹中，在Windows 2000中一般位于“C\Documents and Settings\用户名\Local Settings\Temp”文件夹中)，所以建议大家将这个文件另存到一个合适的文件夹中，以免下次生成文件列表时被覆盖。
