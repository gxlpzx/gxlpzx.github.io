升技蓝魔KT7E主板评测报告

虽然VIA已经发布了支持DDR内存的KT266芯片组，但是由于DDR SDRAM的价格一直居高不下，因此对于一般的家庭用户而言，KT133A芯片组仍然是AMD Athlon和Duron的最佳平台。然而，目前支持133MHz外频的新款Athlon CPU起始频率都在1GHz以上，其价格并不是所有人都能承受的。一般的家庭用户可能更愿意选择Duron，毕竟在低端处理器市场上，Duron的性价比非常之高。针对这种情况，近日VIA又推出了一款新的芯片组——VIA Apollo KT133E，以取代原来的KT133芯片组。各主板厂商也迅速动作，因此我们很快就收到了升技基于KT133E芯片组的主板——蓝魔KT7E，下面我们就来看看升技蓝魔KT7E在我们的主板评测中究竟有如何的表现。
芯片组简介
作为KT133芯片组的后继者，KT133E芯片组仍然采用了南北桥的架构，北桥芯片为VT8363E，南桥芯片则采用了我们熟悉的VT82C686B超级南桥。由于到目前为止我们还没得到关于新的VT8363E芯片的详细资料，因此还不清楚VT8363E与VT8363以及VT8363A之间的区别究竟在哪里，只知道KT133E芯片组的默认外频仍然为100MHz，与支持133MHz的KT133A芯片组比起来在性能上要略微低一些。不过，由于采用了686B超级南桥，KT133E芯片组在磁盘性能上又要好于KT133芯片组。因此总的看起来，KT133E的性能应该是介于这两者之间，对于VIA推出这样的一款芯片组，其目的何在我们现在还不得而知。
升技蓝魔KT7E
和升技以往的设计风格不同，升技蓝魔KT7E主板这次出人意料地采用了蓝色的PCB基板，看上去和技嘉的主板非常相似。升技蓝魔KT7E主板的板型显得比较宽大，即使是在为用户提供了6根PCI和1根ISA插槽的情况下，整个主板的布局也很规范。CPU插槽附近已经预留了足够的空间，而且预设了4个安装基孔，是为以后安装大体积的风扇准备的，毕竟Athlon和Duron的发热量还是非常惊人。主板上的大多数元器件都采用了贴片设计，看上去非常整洁。主板上的跳线非常少，只有3个，大多数的设置都可以在BIOS菜单中完成，对于家庭用户来说应该是非常方便了。主板CPU插槽和电源附近共安装了12个2200μF的大电容，这些电容可以有效滤除主板电源供应部分的杂波，对于整块主板的稳定运行至关重要。从整个主板的做工和用料来看，升技蓝魔KT7E主板做得不错，体现了一个大厂应有的风范，蓝色的主板，加上精良的做工，看上去更加漂亮了。
升技蓝魔KT7E主板支持AMD Durron 600MHz～850MHz或更高以及Athlon 700MHz～1.2GMHz或更高频率的处理器，并支持200MHz的Alpha EV6前端总线。主板上为用户提供了6×PCI、1×ISA、3×DIMM、1×AGP扩展槽。2×USB接口，另外2个USB接口可以通过主板上的跳线引出。3条DIMM插槽最大可以支持1.5G SDRAM内存。整块主板的扩展性非常好，而且还提供了现在其它很多主板已不再使用的ISA插槽，为那些手中还握有ISA声卡和必须使用其他特殊扩展卡的用户提供了一个比较好的解决方案。令人意外的是，升技蓝魔KT7E主板取消了目前较为流行的AC’97以及CNR的设计。长期以来，针对是否应该内置AC’97声卡的争论就一直存在。AC’97当然可以满足一部分用户的需要，而对一些对音效要求较高的用户来说，AC’97的存在反而成为一种鸡肋。不过取消AC’97声卡的设计有利也有弊，因为另外购置一块声卡，其成本可能要比内置的AC’97声卡要高。虽然CNR插槽的功能虽然远比AMR强大得多，不过其实用性却受到越来越多人的质疑。从目前的情况来看，可能使用过CNR插槽的人并不是很多，取消CNR也不会招致太多用户的反对。对于目前比较流行的集成磁盘阵列控制芯片，提供额外的Ultra ATA100和RAID功能，虽然升技蓝魔KT7E主板并没有集成IDE控制芯片，但主板上已留有芯片和另外两个IDE接口的位置，不排除升技会推出另外一个带有RAID功能的KT133E芯片组主板的可能性。
SoftMenu Ⅲ
自BH6开始，升技主板的超频性能就一直被众多DIY玩家所称道。而这次送测的升技蓝魔KT7E主板也依旧采用了方便的Jumperless设计。集成了升技专利的“SoftMenu Ⅲ”技术，使用户能够很方便地在BIOS中进行所有CPU和超频方面的设置。除了标准的100MHz、133MHz外频之外，SoftMenu Ⅲ中还提供了多达20个介于100MHz和155MHz之间的前端总线频率供用户选择。同时还提供了一个名为CPU FSB Plus的调节选项，允许用户在选定频率的基础上以1MHz为单位逐渐增加外频，最大可以增加28MHz。这个选项是非常实用的，它最大限度地增大了超频的成功率，相信一定会受到超频玩家的欢迎。除此之外，SoftMenu Ⅲ菜单中还提供了强大的CPU核心电压和I/O端口电压的调节选项。CPU核心电压的调节范围非常广，可以提供1.10V～1.85V的调节；I/O端口电压的调节选项也非常之多，除了标准的3.3V、3.4V外，还有3.5V、3.6V……3.9V可供用户选择。如此完善的超频功能，在其他主板上是不多见的。使用SoftMenu Ⅲ菜单的一个好处就是，所有的超频设置都可以在BIOS中完成，而不需要去另外设置跳线，因此显得非常方便。
性能测试
测试平台
CPU：AMD Athlon 850
内存：128MB KingMax SDRAM
硬盘：Maxtor 54610H6 43G ATA/100
显卡：GTS Pro/32M DDR SGRAM
光驱：SONY 52×
软件系统：Win98SE+DirectX8.0a
测试中，主板我们安装了VIA最新的2.9v版四合一驱动程序，并且安装了580-2150版IDE驱动补丁。安装这个补丁是非常必要的，它可以有效改善整个IDE子系统的性能。显卡驱动我们采用NVIDIA最新的12.0公版，同时也安装了英文版 DirectX8.0a。测试过程中，主板的CAS参数被设定为2，Windows的显示频率为1024×768×16bit色深，刷新频率为85MHz。
对于本次测试所使用的测试软件，我们依然采用了Ziff Davis Media今年刚发布的Business Winstone2001和Content Creation Winstone2001。虽然SYS Mark2000地测试成绩在大多数情况下只是作为参考，不过由于它包含了我们常用的一些应用软件的脚本程序，像Word2000、Excel 2000、PHOTOSHOP 5.5以及CorelDraw 9等，因此它的测试成绩也从一个侧面反映出整块主板在运行这些软件时究竟会有如何的表现。除此之外，我们也采用了SiSoft Sandra 2001 Perfessional，由于目前内存带宽已成为限制系统性能发挥的瓶颈之一，因此SiSoft Sandra 2001 Perfessional中的内存带宽测试也变得越来越重要了。另外，像3D Mark2001、Quake 3等针对系统图形性能的测试软件，当然也是本次测试中必不可少的。为了充分体现升技蓝魔KT7E主板在本次测试中真实表现，像Business Winstone2001和Content Creation Winstone2001的测试我们都进行了5次，然后取其中成绩最好的一次。WinBench99也运行了3次，也是取其中成绩最好的一次，以期对升技蓝魔KT7E主板有一个比较客观、真实的评价。
整体性能测试
从Business Winstone2001和Content Creation Winstone2001的测试成绩来看，升技蓝魔KT7E主板的表现还是不错的，分别取得了34.4和34.5的成绩。由于芯片组所支持的标准外频仍为100MHz，因此如果与KT133A芯片组主板相比较，在性能上会有一定的差距。其实这种差距是我们应该想到的，毕竟KT133A芯片组266MHz前端总线所提供的带宽要比KT133E芯片组高得多，在性能上领先也是必然的。不过，我们也可以换一个角度来看，毕竟新款133MHz外频的Athlon处理器不是每个的家庭用户都承受得起的，对于大多数使用Duron CPU的用户来说，也许KT133E芯片组主板更适合一些，至少不会造成资源浪费。在KT133A之后推出KT133E芯片组，据我们推测，VIA可能是想用KT133E芯片组替代原来的KT133芯片组，因此在性能上KT133E芯片组应该与KT133A芯片组性能更为接近。因此对于即将要买Duron CPU的用户来说，也许KT133E芯片组是一个不错的选择。在SYS Mark2000中，升技蓝魔KT7E主板在各项测试中表现不错，三项主要的得分分别为171、168和173，比较令人满意。
子系统测试
通过WinBench99的测试我们可以看到，升技蓝魔KT7E主板的各子系统的表现都比较好，与CPU相关的CPU Mark99和FPU WinMark测试成绩分别为77.1和4640。由于南桥芯片采用了686B超级南桥，因此IDE子系统的表现要明显好于KT133芯片组，得分分别为5830、19100。对于目前已经非常流行的ATA/100，KT133芯片组的686A南桥所支持的ATA/66确实显得有点落后了，单在IDE子系统性能上就要差KT133E芯片组一大截。从这一点上我们也可以看出，VIA确实想用KT133E替代KT133，为低端Athlon和Duron处理器提供一个性能比较好的平台。另外，升技蓝魔KT7E主板在Business Graphics WinMark和High-End Graphics WinMark中的表现也比较令人满意，得分分别为397和1150。
SiSoft 2001 Perfessional测试
SiSoft Sandra 2001 Perfessional的CPU Dhrystone和FPU Whetstone，以及Integer MMX和Floating-Point SSE测试项主要是针对处理器方面的测试。从测试成绩来看，Athlon 850 CPU在升技蓝魔KT7E主板有不错的发挥，各项分数都较高，说明升技蓝魔KT7E主板对处理器的支持还是非常稳定的。ALU/Memory Bandwidth和FPU/Memory Bandwidth是iSoft Sandra 2001 Perfessional的测试重点。从得分来看，升技蓝魔KT7E主板表现还不错，能够在外频为100MHz的情况下取得401和451得分，也还真是不容易了。
3D Mark20001及Quake 3
由于在测试中我们使用了Geforce2 GTS显卡，因此无论是3D Mark20001，还是Quake 3，升技蓝魔KT7E主板所反映出来的测试成绩都十分不错。Quake3 640×480×16bit色深下，Demo1和Demo2的测试成绩分别为121.6和120。即使是在1024×768×16bit色深的情况下，也取得了114.4和116.4的好成绩。
##1总 结
从整个测试情况来看，升技蓝魔KT7E主板无论是在性能上，还是在超频设置上都有着不俗的表现，其强大的SoftMenu Ⅲ BIOS菜单给我们的印象深刻。针对目前的市场状况，KT133芯片组最终将要被KT133E所取代，因此对于家庭用户而言，KT133E芯片组将是一个不错的选择。而且据升技的销售人员称，升技蓝魔KT7E主板已在上个星期就上市了，是最早上市的几款KT133E芯片组主板之一。不错的性能，再加上良好的超频能力，相信升技蓝魔KT7E主板一定会受到用户的欢迎。
