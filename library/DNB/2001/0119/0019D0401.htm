IE插件大观园

一、大小任你改——Resize IE
下载网址：http://www.dougworld.com
Resize IE可以让使用者直接将IE浏览器窗口调整为640×480及800×600的大小，或者让IE浏览器窗口对齐屏幕的右上方，是一个很不错的小工具。将Resize IE下载到本地，解压后执行其中的Install.bat，程序就会将其中的HTML文件复制到C\ResizeIE目录下。以后你只要在使用 IE 时在窗口中按下鼠标右键，在跳出的快捷选单中你会看到多了五个调整窗口大小的选项。这五个选项的功能如下：Resize640×480(640×480的窗口大小)、Resize800×600(800×600的窗口大小)、Resize800×Full(800×全屏幕的窗口大小)、Resize800×Full top-left(800×600全长的窗口大小及将窗口对齐画面右上方)及Move to top-left(将目前窗口对齐画面右上方)。
这里笔者还告诉大家一个小秘密，我们在C\ResizeIE中，会发现几个HTML文件。我们用记事本打开其中的一个Resize800×600.htm就会看到如下的语句：
<script LANGUAGE="JavaScript" defer>
<--var newX = 800 newY = 600
var xOffset = screen.width - newX/2 yOffset = screen.height - newY/2
external.menuArguments.top.resizeTonewX newY
external.menuArguments.top.moveToxOffset yOffset
//--></script>
只要是做过网页的，稍稍懂得JavaScript的朋友可能已经看出来，我们只需更改newX、newY这两个参数，比如newX=320、newY=480，我们就可定义320×480的屏幕大小了，现在读者可以定制自己喜欢的IE窗口大小了。
二、梦幻大变脸——IE 5.0 Toolbar Wallpaper
下载网址：http://joysoft.soim.net
这个插件在IE5的工具菜单上会增加一个Toolbar Wallpaper(工具栏壁纸)选项，通过它你可以任意更改IE5工具栏的背景。具体的使用方法如下：安装该程序，它会在浏览器IE5工具菜单中加入Toolbar Wallpaper项，选取该选项就会出现一个对话框。在Select an image下面列出了壁纸，你每点选一个对话框上面的工具栏在略图中就会显示出使用该壁纸的效果。笔者觉得程序所提供的壁纸都非常不错，色彩搭配合理，不但不影响使用，还会使原来灰暗的工具栏变得漂亮许多。另外，在对话框的最下面还有一个复选项：Show desktop wallpaper in list。如果选上它，就可以将Windows的壁纸都加进来，不过，笔者觉得它们的效果不是太好。选择好壁纸并按OK后，记住还要重新启动一下IE5。当你再次打开浏览器时，是不是有眼前一亮的感觉？笔者发现读者自己可以添加喜欢的背景作为IE5的工具栏背景：因为IE5的背景实际上是一个BMP位图。例如：我们用画笔或Photoshop制作一名为wall.bmp的位图并复制到C\program files\Internet Explorer\Bitmaps(IE5的默认安装路径)下，我们再运行IE5，打开这个插件，就可以在预览菜单中看到wall的选项了。当然，IE5的这个插件也支持把位图放在Windows目录下。只要选中Show desktop wallpaper in list(在清单中展示桌面墙纸)的选项就可以了。
三、 国货插件——网易EaseBand 1.0
下载网址：http://easeband.163.com/EaseBand-2000.exe
EaseBand作为国产的IE插件有着极为强大的功能，它可以随心所欲地改换IE浏览器的皮肤，使IE更加个性化和丰富多彩；它也能更加方便地在网上进行搜索，你可以在浏览任何网站的时候进行搜索，并备有多个热门的搜索引擎以满足你的需要。同时推荐和更新最优秀的十大网站，新版本还增加了快速查询股票信息的功能，在“搜索栏”中直接输入股票代码即可知道股票行情。值得一提的是它使IE浏览器工具栏的皮肤更换极其简单，只需要到easeband.163.com/skin/Skin.htm选中喜欢的图片，点击图片，然后重新启动就可以了。
四、皮肤随时换——IE Skiner 1.2
下载网址：ftp//ftp.simtel.net/pub/simtelnet/win95/webbrows/ie5skn12.zip
换IE浏览器工具栏的皮肤当然酷，不过更酷的是让皮肤随时自己换。下面推荐的是IE Skiner。这个软件严格地说不算是插件，不过功能和插件差不多，我们只需要把所有的Skin图片放在这个软件的BMP目录下然后运行IE Skiner，让它驻留后台，这样IE的浏览器的Skin就可以自动更新了。
五、微软插件——IE 5.0 Power Tweaks Web Accessories
下载网址http://www.microsoft.com/windows/ie/webaccess/pwrtwks.exe
Power Tweaks Web Accessories所提供的功能会使你在操作IE5时变得更加轻松。它在工具栏上增加了一个Offline/Online按钮，使用者可以很容易地通过它在离线和在线状态间进行切换，这个功能按钮对那些使用笔记本电脑的朋友尤为有用。而且它在IE5工具菜单中增加的Add to trusted Zone和Add to restricted Zone两个选项可以使你很容易地将当前的站点设置为可信站点或限制站点，不需要像以前那样到Internet选项中去逐一设置了。对于目前网上越来越多的不安全问题，设置站点的可信度与安全级别也就显得尤为重要，所以这个功能还是非常有用的。另外，该程序还为鼠标右键菜单增加了Copy an URL Location功能，这样用户可以方便地将当前页面的地址复制到E-mail或文件中。
六、Web Accessories for IE 5.0
下载网址：http://www.microsoft.com/windows/ie/webaccess/IE5wa.exe
这是一个IE5的工具包，它包含了八个提供给高级用户的功能：图像放大、缩小功能(Zoom In/Zoom Out)可以把网页中的图片任意地放大或缩小；快速在多个搜索引擎中进行查找(Quick Search)；图像显示开关(Image Toggler)可以让网页中所有的图片不下载，这样可以节省网页下载的时间；高亮文本显示(Text Highlighter)可以用黄底颜色显示选中的文本。另外在新窗口中打开页面中的框架(Open Frame in New Window)、列出页面中所有链接(Links List)、列出页面中所有图片(Image List)、使用默认搜索引擎查询网页中的文字(Web Search)等功能都非常好用。Web Accessories的使用方法很简单，只要在适当的地方按鼠标右键，就会出现相对应的功能。例如在图片上按鼠标右键，会多出“Zoom In”和“Zoom Out”的功能。
七、Microsoft Web Developer Accessories
下载网址：http://www.microsoft.com/windows/ie/webaccess/webdevaccess.exe
这个IE插件，主要功能是检查网页上某一个对象的HTML语法，或者了解整个网页所使用的全部对象及语法，对网页设计人员相当有用。安装后，在网页中按下鼠标右键并选择“Document Tree”选项，就会把目前观看的网页内容，以树状图的方式排列出该网页中对象内容及相关信息(包括网页内容、网页放置主机、通讯协定、式样表等)在一个新的窗口。我们只要点选左侧的“+”号就可以观看下拉式的内容了。另一个激动人心的功能(View Partial Source)是我们可以选择网页中任何一个对象，按下鼠标右键并选择“View Partial Source”，就可以看到整个HTML文件中与此相关的一小段HTML的源码。这对学习网页制作的人非常有帮助。
