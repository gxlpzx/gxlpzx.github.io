小巧易用的邮件服务器Xtrim nMail Server

Xtrim nMail Server是一款简单易用的邮件服务器软件，具有中小型局域网络邮件服务的基本功能。它的独特之处在于能够实现基于IP地址的SMTP Relay保护，局域网内用户可以免去设置发信服务认证的麻烦，邮件服务器又可以有效地避免转发外来信件。它还支持多域名，可设置邮箱大小以及拒收垃圾邮件之类的功能。
Xtrim nMail Server由Xtrim公司出品，现在的版本是V1.02，测试版没有时间的限制，该软件可应用在NT/WIN2000上，可以到http://www.xtrim.com/downloads.htm上下载该软件，大小为3729 KB。
Xtrim nMail Server设置界面非常简单明了，下面介绍一下软件的设置。
在Manager管理菜单下有两个选项config setting配置设置和user manage用户管理。
1.先单击 “config setting”项，进入服务管理设置窗口如图，填入下面几项：
● Domain你的主域名，如没有也可填入本机的IP地址。
● DNS Server域名服务器的IP地址。
● Mailbox Size邮箱默认的大小。
(单击“Save the change”存储设置，注意每次存储改变的设置后都须重启一下。)
● Make logfiles：如须记录日志文件，则在“Make logfiles”上打钩。
● Relay Protect：在“Relay Protect”上打钩，则可设置基于IP地址的转发保护，单击“Relay Allow”可以加入多个允许发送邮件的网段的IP地址和掩码。
● Multi Domain如果邮件服务器有多个域名可单击“Add Domain”加入你需要的域名。
● Spammer Protection 单击“Add Spammer”可以加入你想要拒收的邮件服务器的IP地址。
完成以上设置后，即可单击“Start Mail Service”启动邮件服务器。
2.单击“Manager”菜单中“user manage”项，进入用户设置窗口。单击“Add”键填入用户信息可以加入用户。如在某用户的“Manager”选项打钩，则该用户为管理员。要禁止某用户，可去掉“Use Allow” 选项上的钩。
设置完后是不是觉得很简单？更详细的应用还是留给大家自己去研究吧。
