慢工出细活？——VIA Apollo KT266芯片组初步测试

作为与Intel并驾齐驱的全球最大主板芯片组制造商威盛VIA，在2000年可谓大红大紫，从694X到KT133/ KT133A，几乎每一款产品都备受好评，取得极大的市场份额。特别是KT133/KT133A芯片组，一直被认为是支持AMD Athlon和Duron的最佳平台。再加上威盛与AMD之间的战略伙伴关系，以及威盛作为DDR内存规范的坚定倡导者，可谓占尽天时地利人和。因此，在133MHz外频的新款Athlon处理器发布后，大家都认为威盛会尽快推出基于Athlon和Duron 处理器的DDR芯片组。但威盛的动作却让人有些琢磨不透，支持Intel处理器的ApolloPro266年初就已经推出，而支持Athlon和Duron处理器的DDR芯片组——KT266最近才姗姗来迟，远远落后于AMD的AMD760以及Ali的MAGiK1，如果算上主板厂商设计和制造的时间，那么等到KT266芯片组主板大量上市，恐怕要到月初了。对于KT266，人们寄予了厚望，但它会不会像KT133/KT133A那样一鸣惊人，现在谁也说不清楚。最近，我们有幸拿到了微星、技嘉、硕泰克以及七喜DFI四家主板厂商的VIA KT266工程样板，也算是第一次向大家展示KT266 DDR芯片组的性能，也有机会详细了解第一批上市的KT266主板。
一、姗姗来迟的VIA Apollo KT266芯片组
从VIA的技术资料上看，VIA Apollo KT266芯片组将是AMD新款266MHz 前端总线Athlon处理器的最佳平台。在规格上把它与KT133A芯片组比较，VIA Apollo KT266芯片组确实有其过人之处，其中增加的对DDR内存模组的支持以及采用V-Link总线就是最大的特色。
V-Link
虽然从芯片组结构上看， VIA Apollo KT266芯片组仍然采用了传统的南北桥式的设计，北桥芯片为VT8366，南桥芯片为VT8233。不过由于V-Link的加入，使得这种架构具有了新的意义。在VIA以往的芯片组设计中，南北桥之间通常是通过PCI总线进行连接，PCI总线每秒只能提供133MB的带宽，而且还得和其它PCI设备共享，这在一定程度上制约了芯片组自身性能的发挥。因此在与Intel“Hub Architecture”架构芯片组内存带宽比较测试中，VIA芯片组一直处于下风。而现在，V-Link的加入就是VIA对Intel最好的回应。说得简单点，V-Link其实就是一条133MHz的8bit数据通道，它直接把南北桥芯片连接起来，每秒能够提供266MB的数据传输率，这对于提升整个芯片组的性能应该会有一定的帮助。
其实，稍微对Intel“Hub Architecture”架构熟悉的朋友就会发现，VIA的V-Link技术几乎就是Intel“Hub Architecture”的翻版，只是在称呼上有所不同而已，两个技术在本质上应该是完全相同的。但是，一般的电脑用户可能在使用中很难体会出PCI和V-Link之间的差别来，不过南北桥间266 MB/s的带宽对于整个系统都是必需的。
强大的DDR内存支持
内存带宽一直是限制系统性能发挥的瓶颈之一，为此Intel和VIA提出了不同的解决方案。实践证明，Intel的Rambus自推出至今一直是困难重重，至少在现阶段还是不被大多数人所接受，尽管如此Intel还是固执地坚持在它的i850 P4芯片组上使用Rambus，这样DDR就更可能成为下一代内存规范的标准。从理论上讲，DDR确实是解决系统内存带宽瓶颈的有效方法之一。就拿DDR266来说，它能够为系统提供2.1GB/s的内存带宽，与PC133的1.064GB/s比起来，整整提高了1倍。如此大的诱惑力，就难怪VIA和AMD坚定地站在了DDR一边。
作为VIA第一款Athlon平台的DDR芯片组，VIA Apollo KT266支持DDR200(DDR1600)和DDR266(DDR2100)SDRAM，最大能够支持多达4GB的DDR SDRAM。并且兼容普通的SDRAM内存，以及VCM(虚拟通道内存)。与前面发布的两款DDR芯片组比较，Ali MAGiK1 DDR芯片组所支持的内存和VIA Apollo KT266一样，而AMD760则是纯DDR的解决方案，自然也不用提供对普通内存的支持了。因此，我们可以说VIA Apollo KT266芯片组所支持的内存规范比较全面。不过这还要看主板厂商如何设计，如果主板上没有提供普通SDRAM插槽，那我们当然也不能在DDR主板上使用普通SDRAM内存了。
支持6个USB
除了北桥芯片不同外，VIA Apollo KT266芯片组还采用了新的VT8233南桥，它能提供目前一般南桥芯片所能提供的所有功能，像ATA/100、AC’97以及MC’97等。和原来的686B超级南桥不同的是，VT8233提供了6个USB接口，为使用多个USB设备提供了一套比较完备的硬件解决方案。除此之外，新的VT8233南桥终于提供了对AMD PowerNow技术的支持，因此可被用在像笔记本电脑等便携式设备上。
二、性能测试
测试平台
CPU：AMD Athlon 1.33GHz
内存：256MB HY DDR SDRAM(CL=2)、GL2000 PC133(CL=2)
硬盘：西部数据WD400 40G ATA/100
显卡：技嘉 GA-CF2000 Geforce 2 GTS/32M DDR SGRAM
软件系统：Win98SE+DirectX8.0a
作为对比测试的ApolloKT133A主板，我们选用了能够良好支持133MHz总线1.33GHz Athlon的硕泰克SL-75KAV-X主板，KT266和KT133A主板驱动采用VIA最新的4.29V四合一驱动，显卡驱动程序则采用的是nVIDIA 10.08公版，安装了英文版DirectX8.0a。
从各大网站对DDR芯片组的评测报告来看，以前发布的两款DDR芯片组，只有AMD760在性能上略微领先于KT133A，而Ali MaGiK1则表现平平，在多项测试成绩中都要落后于KT133A。KT133A取得这样的成绩，确实令VIA感到欣慰，不过反过来说却是对KT266的一种压力。KT266究竟取得怎样的成绩才能令人满意？为此我们设计了两个平台：KT266平台和KT133A平台，两者相比较，我们就会对KT266芯片组的性能有一个比较清楚的认识。
整体性能测试
从Business Winstone2001、Content Creation Winstone2001以及SYS Mark2000的测试成绩来看，并没有出现我们想象中那种KT266芯片组大幅度领先KT133A芯片组的情况，KT266只是在前两项测试中略微领先于KT133A，而SYS Mark2000竟然还落后于KT133A，所以在总体性能上，我们很难说KT266比KT133A好，毕竟在这些整体性能测试中，内存带宽对性能的影响并不大。
内存带宽测试
从SiSoft Sandra 2001 Perfessional的内存带宽测试来看，KT266在ALU/ Memory Bandwidth和FPU/ Memory Bandwidth中的测试成绩分别为522和723，要比KT133A的399、419高出一大截，尽管还没有达到提高1倍的理论值，但带宽提升的幅度已经非常明显。而且我们也发现，不同厂商设计的KT266主板，内存带宽性能还是有不小的差别，但都比KT133A高出不少。
WinBench99、SiSoft Sandra 2001 Perfessional子系统测试
由于两个平台基本上是基于同一套系统，只是所使用的主板有所不同。而且无论是686B超级南桥，还是VT8233，它们所提供的功能也非常近似，因此反映在测试成绩上非常接近，各有输赢，很难分辨出谁优谁劣。其实WinBench99并不是本次测试的重点，不过它可以反映出系统中各子系统的状况。从这一点来讲，WinBench99测试还是必不可少的。
3D Mark2001和Quake 3测试
内存带宽对3D Mark2001和Quake 3的测试成绩影响比较大，因此反映测试成绩上，KT266都要略好于KT133A，不过领先的幅度还不够大。
三、KT266主板首次曝光
1. 微星K7T266 Pro-R(MSI 6380)
有消息说，微星K7T266 Pro将是第一块正式上市的Athlon平台DDR主板。而且整块主板无论是做工，还是布局设计都非常好，体现了一个名牌大厂应有的风范。
微星K7T266 Pro-R仍然采用微星惯用的绿色PCB基板，ATX的架构，为用户提供了5×PCI、1×AGP Pro、2×USB以及一个CNR插槽。其它的4个USB接口可以通过附件包中USB扩展口从主板上引出来。3个DDR DIMM插槽，最大支持3GB的DDR SDRAM。整块主板不仅采用了AGP Pro插槽，而且还内建了Promise PDC20265R磁盘阵列控制芯片，除了可以提供另外两组IDE通道外，还支持RAID0、RAID1两种模式。因此，磁盘子系统无疑是微星K7T266 Pro最为强劲的部分。对于目前微星最具特色的USB PC2PC技术，也以内置网络控制芯片的形式集成在了主板上。USB PC2PC可以说是微星主板上最具创意的设计了，用户只须通过USB就可以实现PC间互连，用过的朋友一定对它印象深刻。除此之外，微星基于KT266芯片组的主板还有另外一个版本，编号为K7T266 Pro-RU，加入了对最新的USB2.0的支持。USB2.0支持480MB/s的数据传输率，与USB PC2PC结合在一起，相信会有更好的表现。
整块主板的做工和用料相当不错，主板CPU插槽和电源附近的5个4700μF的铝壳电容非常引人注目，这些电容可以有效滤除主板电源供应部分的杂波，提高主板的稳定性。微星K7T266 Pro-R主板电源部分的设计也非常特别，采用了8颗MOSFET组成 “Dual Phase”双相型两次滤波电路为CPU供电，而其它厂商如DFI、技嘉、硕泰克等都是采用6颗MOSFET组成三相回路来完成这项工作。据微星的工程师介绍，他们采用这种设计的理由主要有三点：一是目前的AMD CPU工作电压低，但要求电流大(瞬间最大41A)。采用多颗MOSFET分担的方式，可以保证满足CPU对电流的需求。而微星这种设计可以保证>46A的供电，是按照支持AMD以前预发布的1.5GHz的Athlon CPU标准设计的(瞬间最大46A)，可以支持1.7GHz 的Palmino核心CPU。二是这种设计有利于降低单颗MOSFET的发热量，降低系统散热负担。三是这种设计有利于提高供电系统的效率。看来微星的工程师们在这块K7T266 Pro-R主板上确实是费了一番心思的。
微星K7T266 Pro-R的频率调节完全是由BIOS来完成的，整个主板上只有一颗CMOS清除跳线。BIOS中则提供了分段线性超频技术，它可以根据CPU的不同而提供相应的线性超频段，最高可选外频为153MHz。而且在BIOS中也可以进行CPU核心电压的调节，调节范围在1.70V～1.85V之间。除此之外，BIOS中还提供了一个DDR电压调整选项，可选电压为2.6V和2.7V，用户可以根据自己的需要进行选择，以确保超频过程中DDR内存条稳定地工作。除了上面介绍的这些之外，像SMD LED诊断灯等微星独有的专利技术我们也可以在这块K7T266主板上找到。
2.技嘉GA-7VTX
技嘉的主板一向以做工精良、性能稳定而著称，这次参测的技嘉GA-7VTX当然不能例外，虽然仍只是一块工程样板，不过已显大将风范。技嘉GA-7VTX一如既往地采用了蔚蓝色的PCB基板，再加上精良的做工和合理的布局，因此整块主板看上去非常漂亮，是4块参测样板中做工最好的一块。
从规格上看，GA-7VTX为用户提供了5×PCI、1×AGP Pro、2×USB以及一个AMR插槽。3×DDR DIMM，最大支持3GB的DDR SDRAM。板子上除了集成了AC’97声卡外，还内置创新4声道CT5880音效芯片，可以为用户提供较好的音效。双BIOS设计一直是技嘉的专利，它的存在使用户多了一份安全感，而且使用起来也非常方便。在CPU插槽附近有4个基孔，是为以后安装大体积的风扇做准备的。从整块主板的规格看，技嘉GA-7VTX也可以算得上是技嘉KT266芯片组中比较高档的主板了。而且据技嘉的工程师透露，这块主板还可以支持200MHz外频或者说是400MHz的DDR外频，因此还有很大的升级空间。
在超频设置方面，GA-7VTX一改往日技嘉稳重的作风，不仅在主板上提供了4个DIP开关，可以分别对CPU外频、倍频、AGP电压以及RAM电压进行调节，BIOS中也采用了分段线性超频技术，最高可选外频达到了200MHz。如此强劲的超频实力，再加上技嘉主板一向稳定的品质，相信一定会为你带来惊喜。
3. 硕泰克SL-75DRV
硕泰克现在的主板制造技术已越来越成熟，尤其在VIA系列芯片组上颇具势力，而且主板设计本身也非常有特色。这次硕泰克推出VIA Apollo KT266芯片组主板的速度也相当块，有可能与微星、技嘉的主板产品同步上市。从最近的几次测试来，硕泰克主板最大的特点在于其超频性能上，无论是i815E，还是i815EP主板，硕泰克主板的超频性能在同类产品中都是出类拔萃的，不知道硕泰克SL-75DRV会不会继承它前辈的这种优良传统。
硕泰克SL-75DRV主板采用了黄色的PCB基板，ATX的架构，为用户提供了5×PCI、1×AGP Pro、2×USB以及一个CNR插槽。3×DDR DIMM，最大支持3GB的DDR SDRAM。板载VIA AC’97声卡，CODEC芯片为VT1611A。整体布局比较合理，主板上大大小小的电容非常之多，看上去有点凌乱，不像微星和技嘉在大多数地方都采用了贴片电容。
硕泰克SL-75DRV完全延续了硕泰克SL-65ME主板强劲的超频能力，仍然采用了分段线性超频，而且硕泰克SL-75DRV主板的BIOS中也提供了CPU核心电压调节功能，调节范围在1.10V～1.85V之间。遗憾的是BIOS中没有I/O端口电压的调节选项。另外，硕泰克SL-75DRV上还采用了硕泰克自行设计的一种所谓的RED STORM(红色风暴)的自动超频技术，它能够自动搜索最优化的频率设定，使用起来非常方便。除此之外，硕泰克SL-75DRV上还采用了一种Smart DOC的技术，其作用是实时监控CPU及风扇的运转，如果出现异常，它会发出警报通知用户，严重时它会在3秒钟内自动关机，以保护CPU不会被烧坏。
4. 七喜DFI AD72
七喜DFI这次也以较快的速度推出了KT266主板，这款AD72主板仍然采用了黄色的PCB基板，ATX架构。整块主板的布局比较好，走线也非常清晰，而且采用了宽板设计，因此在插拔各种设备的时候显得游刃有余。钻石AD72主板也为用户提供了5×PCI、1×AGP、2×USB以及一个CNR插槽。除此之外，钻石AD72还同时提供了2×DDR DIMM以及2 ×SDRAM DIMM，是4块参测主板中唯一一块同时支持DDR和普通SDRAM内存的主板。对于目前手中只有SDRAM的用户，也许钻石AD72会是一个比较好的选择。不过需要注意的是，主板上已经明确标明了不能在同一时间同时使用DDR和普通SDRAM内存，因此用户在使用过程中一定要注意，损坏了内存条，那就得不偿失了。
钻石AD72主板上有一个CPU倍频选择开关，不过对于目前已经锁了倍频的Athlon和Duron CPU来说，意义已不太大了。和其它三块主板一样，钻石AD72在BIOS中也采用了分段线性技术，最高可选外频高达250MHz，异常引人注目。而且CPU核心频率也可调，其调节范围在1.5V～1.85V之间。但是令人遗憾的是，主板上没有提供诸如对I/O端口电压以及DDR电压的调节选项，如果加上去，整个超频设置就显得更加完善了。
结论
从整个测试的情况来看，VIA Apollo KT266芯片组的表现的确离我们的期望还有不小的差距。似乎KT266芯片组并没有把DDR的高内存带宽完全发挥出来。而且在普通办公应用中DDR的高带宽也对性能没有影响。如果我们现在回过头来看看已经发布的三款基于Athlon 平台的DDR芯片组，其中Ali MAGiK1已经被认定是一款不能够成大气候的产品；AMD760虽然发布得最早，在性能上也是3款DDR芯片组中表现最好的一款。但AMD的主业毕竟在CPU，发布主板芯片组只是为了带动其他芯片组厂商开发支持自己CPU的芯片组。我们寄予厚望的VIA，至少在KT266上还有进一步完善的地方。我们期待着当VIA Apollo KT266芯片组主板正式上市后会带给我们真正的惊喜。
