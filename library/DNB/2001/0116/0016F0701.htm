专家坐堂

操作系统
如何打开未注册的程序？
问：当我打开不能被Windows识别的文件时，Windows都会要求我选择用某一个程序来打开。在不知道用什么程序来打开这个程序时，我一般会选用记事本。若下次不用记事本我该如何换成别的程序来打开它呢？而且“始终以该程序打开这种类型的文件”的选项，有时打钩，有时却没有，这是为什么？
答：出现打开方式菜单就说明没有在Windows中注册该文件的文件类型。Window 9X会记录哪些文件是指定用哪些程序来打开，对于那些没有注册文件类型的文件，Windows 9X就不知道要用哪个程序来打开它了，所以会出现一个菜单要你指定用某一个程序来打开这个文件。如果该文件类型是第一次被打开，并且没有指定打开程序，在出现的菜单上就会让你指定用某个程序来打开，同时还会有一个“始终以该程序打开这种类型的文件”的选项。只要你第一次指定了用某个程序来打开该文件后，这个选项就被勾选了，那么下次开启这种类型的文件时，就会以你第一次用来打开的那个程序来打开它。举例来说，如果你要开启Pdos.def这个文件，因为.def的文件类型是没有在Windows中注册的，所以当你打开该文件时，就会出现一个程序菜单，要你指定打开该文件的程序，如果你选择NOTEPAD(记事本)并且对“始终以该程序打开这种类型的文件”选项打了钩，那么以后你打开扩展名为.def的文件时，都会以记事本来打开。如果你想用别的程序来打开.def文件，你可以按住“Shift”键，再按鼠标右键，选“打开方式”来选择别的打开程序。
如何终止使用计划任务程序？
问：不知道为什么鼠标在开机不到一分钟后，就像搞恶作剧一样，在桌面上到处乱跑，而且乱打开了好多窗口，随意执行程序。由于鼠标的乱点，它打开了“计划任务程序”，并且在任务栏的右下角里出现。我想问问，如何把在任务栏里的“计划任务程序”去掉，让它不在任务栏里出现呢？
答：鼠标在开机的时候出现这种情况，多半是由于你使用的是劣质鼠标。如果不使用“计划任务程序”，可用鼠标左键双击任务栏中的“计划任务程序”图标，在弹出的“计划任务程序”界面中，点击“高级”菜单，再选择“终止使用计划任务程序”，最后关闭“计划任务程序”即可。
硬 件
金邦内存条的区分
问：最近内存价格大跌，本人也添加了一根内存条。听说金邦内存条的性能不错，但是它好像有好多个版本，应该怎样来区分这些条子呢？
答：金邦内存条是台湾金邦科技生产的，金邦科技主要的内存产品有GL2000千禧条和金邦金条两种。而金邦金条又分为金色金邦金条、红色金邦金条、绿色金邦金条、银色金邦金条和蓝色金邦金条。各种金邦金条的SPD均是确定的，用以适用于不同的主板。红色金条PC133内存适用于i810E、i815、i820、i840、VIA 693A、VIA 694X芯片组的主板；绿色金条PC100内存适用于i440BX、i440GX、i440LZ、i440ZX、VIA 691、VIA 692、SiS 530、SiS 620芯片的主板；蓝A色金条适用于AMD-750、AMD-760芯片的主板；蓝V色金条适用于VIA KX-133芯片的主板；蓝T色金条适用于VIA KT-133芯片的主板；金色金条PC133内存适用于服务器用的双处理器主板；银色金条PC133内存适用于笔记本电脑。其实要区别这些内存条，可以从内存电路板的颜色上进行识别，从内存条上贴的标签编号可以看出。例如“金64 MUAKT00009”，其中的金代表金色金邦金条；红代表红色金邦金条；绿代表绿色金邦金条；银代表银色金邦金条，蓝代表蓝色金邦金条；GL代表GL2000千禧条。
解决Sound Blaster AWE64
易死机现象
问：原来我使用的是花王ALS07声卡(非即插即用)，现在将声卡换成创新的Sound Blaster AWE64了。为什么在Windows 95下，我一玩游戏就死机？
答：这个问题可能是因为分配给AWE64的系统资源与此游戏要求的系统资源不符。也可能是由于在安装新声卡之前，为原先非即插即用的ALS声卡设置好的系统资源并未从设备管理器中删除掉。解决方法是把ALS07声卡从设备管理器中删除掉，并将Creative Labs AWE64 16 BitsAudioSB16 Compatible的系统资源设置为如下缺省值：
中断 05
Direct Memory Access 01
Direct Memory Access 05
I/O 基址范围 220——22F
I/O 基址范围 330——331
I/O 基址范围 388——38B
或者在游戏中运行Sound/music setup utility来重新设置声卡，用以保证该设置与Windows95下已经分配的系统资源相符。
网 络
利用300电话卡上网
问：不知能否利用300卡拨号上网？如果可以，应如何设置？还有，用300卡上网能否看到所用的电话号码？
答：如果你要利用300电话卡进行拨号上网，填写拨号的电话号码格式为“300，卡号#密码#ISP号码#”。但要注意的是，这样将会暴露300电话卡的卡号及密码。建议你每次上网拨号时，新建一个拨号连接。当然还有一个手动拨号的方法，但前提条件是你必须在MODEM中再接一部电话机，然后用300电话卡在电话机上进行拨号，输入卡号及密码，然后再按下拨号连接。在电话号码中，只要输入ISP号码就可以了。虽然这样做会比较麻烦，但与上一方法相比就安全多了。
多用户应如何配置？
问：因为我单位的电脑是多人使用的，所以我就想在计算机上设定多重用户菜单。请问我该怎样做？
答：要出现多重用户菜单，你可以试试使用“Microsoft友好登录”功能，到“控制面板/网络/配置”卷标，按下“添加”按钮，选择“客户”再按“添加”按钮。在弹出的对话框的厂商中选择“Microsoft”，在网络用户中选择“Microsoft友好登录”，再按下“确定”。这时返回到网络的配置对话框下，在“主网络登录”中，选择“Microsoft友好登录”之后，按下“确定”，重新开机。这样下次开机进入Windows时，应该出现多重用户菜单。如果你不想使用多重用户，可以把所有用户删除之后，并在“主网络登录”中，选择“Windows登录”，下次重新开机时，会再出现一次询问窗口，这时候你随便输入另一个使用者的名称，并且不要输入密码，这样在下次开机时就不会出现询问窗口了。
打印网页中的部分内容
问：我想用打印机打印网页中想要的那部分内容，可以吗？
答：当然可以，你只须先将要打印的部分用鼠标选中使它成反白，然后在选择的区域上按鼠标右键，再选择打印。这时，会弹出一个打印机打印的设置菜单，在“打印范围”选项中选择“所选内容”默认设置为全部。这样就可以只打印你选择的那部分内容了。
