走进Palm的世界——同步和安装

仅仅依靠Palm内建的几个基本程序，是不可能让Palm如此风靡的，Palm的魅力就在于它可以和桌面电脑很好地配合，安装大量的应用程序，从而无限地扩展Palm的功能。这次我们了解一下Palm和桌面电脑同步资料以及安装应用程序的方法。
同 步
找到电脑的COM口(或者是USB接口)，连接掌上电脑的底座cadle(图1)，然后安装随机光碟上的Palm桌面同步软件Palm Desktop(图2)，对照说明书和提示完成安装。
安装过程中会提示你起一个用户名，也是你的同步名。很多Palm软件在注册时也需要用户名，所以最好用英文。
把你的Palm放上底座，按下底座上的同步按钮，你的Palm和你的桌面电脑就开始第一次亲密接触，Palm中的设置数据被保存到了你的电脑中。
这里解释一下同步(HotSync)的概念。对于Palm的新用户来说，这是个新名词。HotSync是Palm的专用术语，是指Palm和桌面电脑资料互相更新达到一致。同步可以根据你的设置使Palm的内容覆盖桌面电脑的内容，或反之，也可以两者之间的资料取一个并集。
Palm Desktop的基本应用程序Date、Address、ToDo List、Memo以及Expence也会和Palm Desktop 进行同步(图2)，每次同步时，Palm上的资料和桌面电脑中的资料互换更新，如果有同一个记录，将按照时间先后以新资料覆盖旧资料。因为有这个功能，你可以在Palm Desktop上完成大量资料的输入，如输入几十个朋友的通讯地址，同步后这些资料就会立即出现在你的Palm上。
Palm Desktop支持多用户使用，也就是说可以在同一个Palm Desktop上保存多个Palm用户的资料，而且每个用户都可以使用独立的密码隐藏起私人资料。这样公司的电脑就没必要每台都安装Palm Desktop了。
安装Palm程序
下面以安装一个软件的步骤进一步说明同步的使用和设置。以安装程序CSpotRun为例。开启Palm Desktop，按左下角的“Instsll”键， 点击Add，选择要安装的软件(图3)。
确定你的Palm已经放在底座上，并且底座与电脑正确连接，按下底座上的HotSync键，随着“嘀”的一声提示音，电脑桌面和Palm上同时出现了提示信息，告诉你正在安装应用程序。同步完毕，你便会发现Palm屏幕上多了一个CSpotRun程序图标，恭喜你，你的第一个软件已经成功地安装了。
提示：如果你有多个程序文件要安装，大可不必这么辛苦，只要把这些软件复制到你安装Palm Desktop的目录——你的用户名目录安装就行了。
用第三方工具安装软件
如果你只是安装应用软件，那么没有必要每次用Palm Desktop 进行安装和同步，Pilot Install 这个桌面软件能使你的安装工作更加轻松快捷。该软件可以在www.palmgear.com下载，只有一个可执行文件，打开这个软件，立即出现文件浏览窗口，让你选择要安装的文件。不仅支持标准的Palm格式的文件，你还可以直接安装纯文本文件和Midi文件(当然这些文件将自动被转换成.PDB的Palm标准格式)。确定后出现图4的界面。
你可以继续增加要安装的软件，调整你的传输速度。你不需要运行HotSycn和Palm Desktop程序，只要直接确认你的Palm正确和电脑连接，直接按下同步按钮就可以了。
不仅如此，Pilot Install还支持以拖动方式直接安装网上的文件。有了这个软件，你完全可以放弃Palm Desktop中的Installer了。
根据以上的介绍，你可以在你的Palm和桌面电脑之间架起一座便捷的桥梁，从而充分享受Palm给你带来的方便和乐趣。而且，能提供类似功能的软件并不只限于此，寻找和熟悉这些软件本身也是一种乐趣。
