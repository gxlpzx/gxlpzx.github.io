{Office外挂软件齐亮相}

{ Microsoft Office是现今使用最广泛的办公软件工具包，它的每一次更新换代都会引起一次不小的办公软件大战，虽然在每次残酷的竞争中Office都能抓住用户的升级心理，但是在我们每次以很大的信心和希望打开刚升级的Office时都免不了有些失望，因为你盼望已久的功能在这次的升级中又被漏掉了，所以直到现在的Office 2000都还有很多不尽如人意的地方。比如Word查不出错字，Excel使用中的种种不便，Outlook甚至还会丢失信息，PowerPoint的千篇一律等。那么我们就只有等待吗？答案是令我们高兴的，因为这些不足与漏洞已被第三方公司相应的软件弥补。现在我们就一同来给Office添砖加瓦。
●增加Word的效率
虽然微软的Word文字处理软件已经改善了你的写作习惯，但是你还需要这些第三方的软件，来为你的文字编辑工作助一臂之力。在使用下面介绍的这两个工具时，会使你的写作更加轻松。尽管它不会使你成为作家，但它确实给你在处理大量邮件和校对工作时带来很大的便利。
ProWrite 1.56
发行公司：Nereosoft
下载地址：http://www.nereosoft.com/download＿trial/prowrite.exe
软件类型：免费软件
软件大小：17.6MB
Nereosoft的ProWrite，采用了微软工具栏的按钮操作模式，并可以同Microsoft Word和Outlook紧密结合，成为你的一位高效率的秘书。它可以通过你的Outlook通讯录内的联系资料，直接输入Word的标签（Label）、备忘（Memo）、传真封面和信件模板（Template）中，快捷地寄出或处理大量的邮件。至于使用其他办公软件的用户，ProWrite可以从Lotus Nots，Novell Group Wise或Corel Central中，找到相关的通讯录。例如：当你需要一次寄出大量邮件时，你可以选择合适的模板、发件人资料、日期，然后选择收件人的地址，即可完成。 这个17.6MB的大家伙，虽然在你下载的时候多费些时间，但当你完成这个下载任务之后，ProWrite给你带来的便捷，会让你惊呼它的超值。
DocProofReader 2.0.14
发行公司：Esady
下载地址：http://www.docproofreader.com/DPRsd200.exe
软件类型：免费软件
软件大小：2.38MB
术语、拼写错误、简称以及语法和标点的错误都会使一个好作品逊色。虽然Word有拼写和语法的检查但形同虚设，Word自身若要摆脱这些烦恼还有一定的距离，但是DocProofReader却完成了这项使命。这个校对分析工具，可以为作者提供比Word内置的校对工具强大得多的文档校对功能。它可以检查标准的信件和文本文件，找出简单的错误，如漏掉的标点符号、字间距离，用高对比的颜色显示未经注释的术语和简称。你还可以把评论加到你的原文中去，同Word中的Track Changes功能相似，或者制作一份已编辑了的文本附件，同时还可以保留原文进行比较。
●加强Excel的应用
Power Utility Pak 2000
发行公司：JWalk and Associates
下载地址：http://www.j-walk.com/ss/pup/pup2000/pup2000t.exe
软件类型：免费软件
软件大小：1.76MB
这套附加程序可以使Excel的操作变得更加直接，并且可以增加你喜欢的功能。例如，它可以在你的Excel工作薄内建一个永久的日历页面，紧贴着你的财政进度。还可以把工作表存储成单独的图像，只需点击程序中的图表工具（Chart tools），并以GIF，JPEG，或TIFF格式输出图表，使你发送电子邮件、打开和观看时更加方便。这个程序甚至还包括一个非常实用的信用卡编号检查工具。例如，在上网购物时，可以用它来检查输入的信用卡号是否有效。
Power Utility Pak 2000（下文简称PUP2000）在安装前需要将“宏”的安全级别定为中级，安装界面也是一目了然，安装完毕并启动Excel后你会看到一个新的工具栏（图1），如果PUP2000没有自动加入到Excel中，你也可以用手动加入“宏”的方法启动PUP2000。PUP2000中自带的工具，可以说是给Excel的操作带来了革命性的变化，其中的实用工具就详细地分为十大类，如：工作表工具、图表工具、印刷工具、公式工具、日期时间工具、审核工具等。你还可以同时打开所有的PUP2000快捷工具面板，只需点击PUP2000下的“Create a PUP 2000 Toolbar”你就看到一个拥有50个工具的快捷启动面板，这样PUP2000的全部秘密就尽收眼底了，刚开始看到这么多的快捷图标有点儿不知所措，不过在熟练之后所带来的快捷是不言而喻的。PUP2000还有个有趣的功能——“Time Wasters”，里面共有四个小游戏，虽然趣味性不是很强，但也能给你枯燥的工作带来些新鲜的感觉。50个实用工具想快速上手也非易事，所以PUP2000中完善的帮助文件是你首选良师，包括小游戏的玩法也囊括其中哟！
Edwin’s Power Tools 3.0
发行公司：Edwin Tam
下载地址：http://users.vol.net/edwintam/ept/downloads/ept30.exe
软件类型：共享软件
软件大小：1.96MB
Edwin’s Power Tools是一个集合的工具套装程序，它可以改变Excel 9x和Excel 2000以及Excel 5.0 for Mac等多个版本的外观和格式，使Excel更加个性化。你可以使用它来改变你报表中文字的位置和外形。可以缩放存储格资料和公式细节来方便阅读和打印。还可以在相同的工作本内复制你的工作表，这样就可以随意测试不同的数值，同时又不影响原来的数值。
EPT的安装也同样不需你劳神，安装完毕后Excel中会出现EPT3.0的工具栏，其中的“立体格式工具”可以快速使你的各种报表的不同区域产生不同的立体感，这样处理后的报表看上去更加条理清晰。“平面格式工具”方便给单元或区域加入阴影也可改变它们的颜色。“精确选择”可以对指定区域进行准确选择。“文字处理工具”是EPT3.0的核心工具。“恢复工具”可以恢复EPT3.0做出的所有改变。“特色工具”可以用JPEG、GIF、PNG中的任何一种图像格式输出报表。这便是EPT3.0中六个主干，使Excel使用更加个性、便捷。
●提升Outlook表格
Outlook虽然已经是目前最好的电子邮件程序之一，但它还有许多地方需要改进和完善。普遍存在的邮件损坏、日期显示错误和各种安全隐患，都会给你留下难忘的经历。于是便有了下面的这些工具来修复这些故障和其他困扰着Outlook的缺陷。
Outlook 2000 OST Integrity Check Tool
发行公司：Microsoft
下载地址：http://download.microsoft.com/download/outlook2000/scanost/2000/WIN98/EN-US/scanost.exe
软件类型：免费软件
软件大小：1.50MB
如果你的硬盘已坏或者你的电脑已经感染病毒，你离线Offline存储的Outlook文件和资料，可能无法与服务器同步更新。当Outlook出现同步错误或不能读取附件时，建议你使用该工具进行修复。这个工具安装在“Microsoft Office工具”中。当你运行它时，它可以把所有存储在硬盘里的文件，同Exchange服务器上的资料进行对比，修正所找到的错误。
你不需要怀疑Microsoft给自己的软件做的外挂工具包，赶快Down下来用就是啦！
Wireless Secretary3.37
发行公司：JTech Software
下载地址：http://www.jtechsoftware.com/ws20000.exe
软件类型：共享软件
软件大小：2.43MB
也许你的Outlook日历是你安排活动的唯一工具。但是当你离开办公室的时候又该如何使你不把这些安排抛到脑后呢？Wireless Secretary会把Outlook中设定的约会、任务和重要的事情，通过传呼或手机用字母和数字向你发出提示（图3）。你还可以在你传呼机上收到Outlook邮件，通过JTech Software’s Wireless Secretary网页建立新的约会，利用电子邮件将这些资料放置在你的Outlook文件夹中，可惜这个共享软件只能使用七天，不过七天的试用中你已经能够决定是否应该拥有它。
Wireless Secretary3.37（下文简称WS2000）简捷快速的安装给你带来一个良好的开始。在初次启动时会弹出“设置向导”，设置完毕后就可以看到WS2000的主界面，在“Outbox”选择需要发送的文件，这个面版内的信息都是Outlook“发件箱”中的待发文件，再在“Delivery”里选择发送方法，并在相对应的发送方法下方的资料栏中正确填写里面的资料，如呼机、手机号码、E-mail地址等。然后在“Options”中设置好定时发送的时间段，一切OK后就可以点击“Save”存储到WS2000自己的“发件箱”中，它会把这些信息准确准时发送到指定的设备中来提醒你约会、任务和重要的事情。
Outlook 2000 Personal Folders Backup
发行公司：Microsoft
下载地址：http://download.microsoft.com/download/outlook2000/pfbacku/2000/WIN98/EN-US/pfbackup.exe
软件类型：免费软件
软件大小：183KB
我们大家都感受过Outlook的脆弱，它是那么容易受到病毒的攻击。Personal Folders Backup的出现帮Outlook挽回了一点颜面，不过Personal Folders Backup并不是一个杀毒软件，它是专门备份邮件的工具，虽然比较被动，但还给Outlook穿上的一件不太厚防弹衣。当你进行备份的时候，你需要选择是用硬盘还是服务器来存储这些重要的邮件，还可以预设备份时间。以后，如果出现不测，可以将它们完全恢复，包括收件箱、日历和通讯录。
它的使用最为简单，在退出Outlook时Personal Folders Backup就会自动弹出备份窗口，点击“Save Backup”后就完成备份了。在弹出备份窗口中的“Options”选项中你可以设定系统提醒你每次备份相隔的天数和放置备份文件的文件夹。这样在不知不觉中你就给你的邮件上了一道保险锁。
●使Powerpoint更动人
你可能每天面对的都是相同的PowerPoint简报、同样的字体，同样的颜色，同样的……如果你想使你在会议室演说时有更多的人留意你的话，如果想赚到更多眼球的注视，那么你就是下面这些程序真正的“主人”。
PowerBacks Sampler 2000
发行公司：Blue Worx
下载地址：http://www.powerbacks.com/freepb48.exe
软件类型：共享软件
你是否想要把新的背景图像和主题加到你的PowerPoint中？PowerBacks的四个可下载的套装的程序，包括210种背景图案，而且CD-ROM版提供了980种独立的背景图案。还有不同的材质和纹理素材。你还可以找到专为会议、商务介绍和走势分析而特别设计的主题图案。就连艺术图片也逃不脱PowerBacks Sampler 2000的手掌，它特有的艺术图库，可以在沉闷的环境中，加入高分辨率的风景和活灵活现的图像。而这些图像样本全部为标准的 JPEG格式，1024×768像素，因此你可以在其他方面使用这些图像。
HiLighter 1.0
发行公司：Altiris
下载地址：http://www.altiris.com/hilighter/HL＿Eval.exe
软件类型：共享软件
软件大小：1.56MB
使用HiLighter你可以将简报中的特定部分擦去。该工具使用方便，可在PowerPoint中发挥出它的最佳性能，但是它可能会把你的窗口弄得一塌糊涂。使用HiLighter后，你可以将你的鼠标变成一个黄色的标记，圈起被强调PowerPoint幻灯片中的某些部分。或者你可以使整个屏幕变暗（鼠标箭头除外）的手电筒效果，还可以选择看起来像是激光笔做出的一个红色的小圆点。你甚至可以使用鼠标箭头和大小调整框来截取加入标记的屏幕、幻灯片和整个幻灯片的某些部分。
HiLighter是一个独立的桌面工具，它并没有和Office中的任何工具关联，所以在安装和使用过程中不会出现任何兼容性问题。启动HiLighter，会在桌面上出现一个工具面板。你可以点击面板中的“Properties”键，对不同工具的各种属性进行设置。我个人认为它更像一个桌面涂污工具，不过HiLighter确实能够吸引更多眼球的注视。（山西 靳景 贾恒）}
