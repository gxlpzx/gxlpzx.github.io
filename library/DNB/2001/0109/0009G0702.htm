{行星侧面}

{ 我记得两年前，Verant曾经向广大的游戏玩家们透露过一个独一无二的新游戏——一个拥有华丽的3D场景，允许成千上万的玩家在线持枪血拼的第一人称射击类网络游戏——《行星侧面》。这曾经在热爱类似游戏的玩家中引起了不小的轰动，因为它本身突破了如Quake3、《帝国时代》等游戏中人数受到限制的情况，而能像《网络创世纪》和《永恒的任务》那样支持众多玩家同时在枪林弹雨中享受游戏快感。同时，对熟悉Verant Interactive的人来说，“Verant Interactive”这个名字本身更是意味着高品质的网络游戏。现在，《行星侧面》已经基本制作完成。
《行星侧面》的游戏背景发生在遥远的未来世界，那是一个毫无规则的、到处硝烟弥漫的混乱世界。在那里，金钱代表了一切。生命是廉价的，而武器却昂贵无比，所以如果想在那样一个世界中生存下去，玩家唯一的出路就是和某个大集团签约，成为其所属的雇佣兵，完成他们分配给你的任务以获得酬金。一旦你和某个超级集团签约后，它就将提供一切你所需要的必备品，包括住房、军械库等。说到这里，就不得不提一下游戏中超级集团的概念了。在《行星侧面》中，一共涉及到四个超级集团，它们分别专政于军事、科学研究、工艺技术及矿产资源四大领域。加盟签约后，玩家就会居住在由集团控制的中心城区，在那里，一切武器将会被禁止使用，所以玩家根本就不用担心会被其他玩家恶意偷袭PK了。
这款游戏，值得一提的是独特的等级报酬系统。如果玩家在刚进入游戏的时候只拥有普通公民的身份，那相应的被委派到的任务也是简单的，当然报酬可想而知也是少得可怜。当然，这种情况并不会维持多久，因为系统在玩家游戏的时候还提供一种类似于在线辅导的功能，比如教你如何使用《行星侧面》中的各种交通工具和重型武器等。随着玩家顺利地完成这些简单的任务（比如快递、侦察、巡逻、护送等），经验和等级也就相应地增加和提高了。到那时候，玩家则可以得到更多高难度的任务和丰厚的报酬，甚至于在任务公告牌中发布自己的私人任务！(^09070601A^)
《行星侧面》中的游戏环境由各区域板块构造而成，它们行成了一个长宽都是8公里的庞大世界。而每种区域都具有自己的特色，各不相同。从丛林至荒漠，从森林到冻原，只要是你能想得到的地形，你都能在这个游戏中找到。区域之间都是用一些“控制点”来连接着，玩家可以在“控制点”选择前往相邻的区域。当然，虽然各区域毫无相同之处，但是散在各地的安全避难所却是统一的。或许你会对“安全避难所”这个词感到相当陌生，不清楚它在游戏中的功能和特色，我就此来做个简单的介绍吧＾＿＾
其实，游戏中的“安全避难所”有两大基本特点，对于玩家“调整修养”的功能，我就不用多说了吧。大家可以看其名猜其意。可想而知的是，一旦玩家进入“安全避难所”，所有的武器将会失去功效，因为暴力在这里是被禁止的。我想突出的是它的另一个类似于市场性质的附加功能，那就是你可以在这里尽情地与你的队友甚至敌人进行货物的交易。有趣的是，游戏允许玩家自行组成战队占领“安全避难所”周围领域，从而获得经济优势。
如同所有其他的第一人称射击游戏一样，玩家在《行星侧面》里将会拥有无数次的死亡。虽然如此，但是Verant Interactive却很自信的保证到，玩家绝对不会因为这个原因，而减少一丝对游戏的兴趣和参与性。因为玩家死亡后遭到的损失是有限的，游戏中根本就不会出现玩家因为死亡，导致损失昂贵并且稀有的物品而痛心疾首的情况。即使有所遗失，所有的一切也都会随着你在所属集团中心区域的重生而得到补偿。这将意味着，玩家始终是有足够的武器和弹药来进行战斗的。
无论如何，《行星侧面》毕竟是一种网络游戏，所以它本身的运行脱离不开服务器的支持。它的每台服务器都可以支持超过3000人同时在线，而每台服务器上又设置了8块不同的作战大陆。每次当你的团队正在准备完成一项任务的时候，总会出现敌方的部队来进行干扰阻碍的情况。比如说，当你接受集团A的任务和你的队友一起要将某样物品安全护送到目的地的途中，一定会碰到来自B集团委派来破坏你们任务的另一队人马，一旦双方正面遇见，就马上会陷入混战之中。实际上，不管物品有没有送达目的地，这种情况始终会出现的。因为服务器始终会对大陆上游戏的运作情况自动进行监督和分析，通过自动的命令来调整大陆上的游戏状况，引诱玩家能不期而遇，展开战斗。
无论从哪方面来看《行星侧面》这款游戏，我始终都觉得它属于系列FPS游戏类型中的代表作。据悉，到时候Verant Interactive会像当初发行《永恒的任务》那样，大量发行一些BETA版的光盘进行公开测试，以确保游戏的稳定性。“我相信我们的游戏是高质量的在线游戏，绝对不会让大家失望。”《行星侧面》的首席设计师Kevin如是说。希望如他所言，到时候我们能看到的是一个令人激动兴奋的全新游戏！不过希望归希望，《行星侧面》到底是否会有如《创世纪》或《永恒的任务》那般获得巨大的成功，它是否值得我们期待，还有待于我们玩家亲自体验了。到时候，我们再众说纷纭吧。
游戏类型：第一人称射击类网络游戏
制作公司：Verant Interactive
发行公司：Sony Computer Entertainment Inc.
上市日期：2001年3月
系统要求：P2-233、64 MB RAM 、3D加速卡、网络支持}
