{巫术VIII}

{ 巫术、魔法门与创世纪并称欧美三大RPG系列，《巫术》系列的生命力和后两者一样长，这点看一下《巫术》后面的序号就可以知道。一个游戏能够出到八代是非常不容易的，这证明了巫术系列有许多的支持者，此次的八代有怎样作新变化呢？
延续与开始
八代将延续七代的故事，Astral Dominae，一个含有生命秘密的神器。它诞生于一个位于宇宙尖端名为Dominae的行星上。由于位置的原因，Dominae成了各星际往来的必经之地，这就是此次游戏冒险的世界，在这里有无数的秘密等待玩家去挖掘。整个游戏世界相当庞大，且结局多样化，有很好的耐玩度。
游戏可以让玩家将六代和七代的角色带入游戏，如果你没有以前的存档或是第一次玩此游戏，那游戏会在一开始对整个系列的完整的介绍。新手可以立即上手。
声光色效
《巫术》系列的游戏画面一向出众，八代承袭此风格，并且画面表现又有很大提高。游戏采用全3D制作，因此画面非常之华丽。各种光影效果在强大的引擎作用下显得非常精彩，种种特效绝对让玩家大呼过瘾。从官方提供的截图来看，游戏的画面绝对登峰造极，细腻的画质将诸如光影、水流逼真地展现在玩家眼前。游戏提供了几种常用的分辨率，配上一块不错的3D卡玩家能够充分体会到高分辨率带来的享受。
战斗！战斗！
RPG免不了战斗，《巫术八》的战斗系统采用阶段性，比起其他同类使用回合制的游戏显得更加真实，并用到了主动权的概念，比如你在和一只怪物单挑，你决定攻击，怪物决定逃跑，那是打不到怪物的，因为在这一回合怪物取得了主动权。因此在战斗时逻辑性就显得异常重要了。你要判断敌人的下一步动向，采取相应对策，而不是向其他回合制RPG一样你打我一拳我再回你一拳，看谁血多。
战斗中的魔法有一百多种，攻击类、恢复类、应有尽有，各类属性的魔咒配上华丽的效果，把整个战场显得异常精彩，刀光剑影，电闪雷鸣。像看电影一样。
对于喜欢使用武器的玩家，游戏不会让玩家失望，武器也多达一百多种，这可是对付敌人的必备物品啊，在这剑与魔法的世界尽情享受吧！
移动中为了方便玩家随时观察周边情况，在屏幕的左右下角提供了两个面板，分别反映了靠近玩家的怪物和伙伴的情况。游戏采用的第一人称视角，呈现了相当好的真实感。
制作：Sir-Tech Software
发行：待定 游戏类型：RPG
类似游戏：巫术系列
配置：P2-233、64MB RAM、3D加速卡
上市日期：2001年5月}
