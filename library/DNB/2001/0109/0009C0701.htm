{巧用＠BIOS Writer升级非技嘉主板}

{ 由于刷新BIOS有一定的危险性，加上在纯DOS模式下全英文的操作界面，令不少初学者望而却步。笔者通过实践，发现技嘉针对自家主板开发的BIOS刷新软件——＠BIOS Writer非常不错，它能在Windows下实现对技嘉主板的升级，并具有在线升级功能。该软件操作简单，直观方便。说出来你也许不信，最近，笔者还用这款软件给硕泰克主板刷新了BIOS！具体过程是这样的：
我首先到http://www.pcshow.net网站的“驱动加油站”下载了＠BIOS Writer的可执行文件biosflash106e.exe，大小为435K。该文件下载后无须安装，直接运行即可。运行后界面如图1所示，它能自动侦测出主板的BIOS芯片类型、电压，容量和版本号。另外，在BIOS信息的左下方是默认的执行操作，共有四项，除第一项“Internet Update”（网络升级）外，其余均为不可更改。而且第一项仅针对技嘉自己的主板，所以不要选择。选项右边有四个按钮，从上到下依次为：“Update New BIOS”（升级新的BIOS），“Save Current BIOS”（保存现有的BIOS），“About this program”（关于这个程序），“Exit”（退出）。建议大家升级前先查看“About this program”中的信息。点击该按钮后，它会弹出一个消息框，其中显示目前该软件支持的芯片组及BIOS芯片的有关内容。
因为＠BIOS Writer不支持非技嘉主板在线升级，所以要刷新非技嘉主板的BIOS，还得先到主板厂商站点下载主板最新的BIOS文件（注意主板的型号，千万不能搞错）。此外，开始升级前一定要关闭诸如如实时监控病毒程序、内存管理软件等后台运行文件，同时把主板上防BIOS写入的跳线帽拔掉，以及在BIOS设置程序中将防BIOS写入的选项设为“Disable”，然后点击图1中的“Update New BIOS”按钮，并在弹出的窗口中选择要刷新的BIOS文件，然后在弹出的消息框上点击“OK”按钮，＠BIOS Writer便会自动为你更新BIOS。还有一点要注意的是，在更新完成前最好不要进行其他操作，防止出现意外。整个更新过程在Windows下进行，时间在10秒左右，更新完成后程序会弹出消息框，提示升级成功，并要求重新启动计算机。在机子重启后自检时，你就会发现BIOS已经更新成新的版本了。
怎么样？＠BIOS Writer很爽吧！如果你想升级BIOS，但又不习惯DOS模式下的操作方式，不妨试试它吧！}
