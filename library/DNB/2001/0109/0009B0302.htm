{电子书制作一点通}

{ 近来，网友们经常在网上可以见到几百K大小的电子书，这种电子书籍图文并茂，有的还有背景音乐，可按目录章节阅读。由于这种电子书本就是EXE可执行文件，因此读者阅读时无需像阅读PDF文档或WDL文档时那样安装专门的阅读工具。另外，这种电子书的制作也简单易学，只要你有基本的网页制作技术，再加上我们功能强大的E-book制作软件WebCompiler 2000，OK！现在跟我学吧。
在制作自己的电子书前，应当先准备好制作电子书的素材，即先将素材制作成Html文件，你可以像制作网页一样充分发挥创作才能，将文本、图像、背景音乐等制作到Html文件中，制作好电子书的封面页面和各下级链接页面，主页面上通过链接进入各个章节。在所有的页面制作完成后，将所有的Html文件和使用到的图片、音乐文件都拷贝到一个临时目录中，例如C\E-book。素材准备好的标志是你可以像浏览网页一样完整地浏览目录里每个网页。
然后我再向大家介绍制作电子书的主角—WebCompiler 2000。它是一款免费软件，大小是279K，目前是英文版的，可以到http://www.eshunet.com去下载。现在我就向大家介绍制作电子书的过程。
Step 1. 准备
首先在主菜单“Files”（文件）选择“Open...”（打开）打开主页面文件，出现一打开文件窗口后，到刚才存放电子书素材的目录中选择封面主页面文件，例如C\EBook\main.htm，然后按“确定”，你制作好的电子书的主页面就会被调入到程序中。
Step 2. 设置
在主菜单“Files”选择“Compiler Options...”（编译选项），出现选项窗口，它共有七个选项卡，在这里可以设置想要制作的电子书的名称、功能、外观、口令保护等。
1. General（一般）选项
在Output Executable File（输出的可执行文件）中，输入电子书编译完成后存放文件的路径及名称。设置发布的电子书的有效阅读期限，还可以输入电子书过期后显示的页面。注意：如果使用电子书阅读期限功能的话，此页面也应在制作素材阶段以Html文件制作好并和其它素材放在同一目录中。
2. Functionality（功能）选项
在这里可以设置所制作的电子书的一些辅助功能。例如，如果制作出的电子书有主菜单的话，可在此将Copy Enabled复选项选中，则生成的电子书的主菜单Edit下就会出现Select All菜单。下面的七个复选框与此类似。Help File Specification（帮助文件规格）里可以设定电子书使用的帮助文件。
3. Appearance（外观）选项
在这里设置所制作的电子书的外观。Windows Sizing（视窗形式）选项可以设置电子书窗口的大小，根据你的需要可以设为Windows缺省值（Windows Default）、最大化（Maximized）、指定启动大小（Specify Start Size）、指定固定大小（Specify Fixed Size）等。一般情况下，为了使屏幕阅读面积尽量大，选择最大化（Maximized）就可以了。另外还能设置通过选中各个复选项为电子书制作地址栏、菜单、状态栏、滚动条等。
4. Branding（商标）选项
想让你制作的电子书带上自己的标志吗？可以事先制作一幅45×45像素的位图（BitMap），将它存放在素材目录里，然后在Logo项下指定此文件。
如果要让电子书有一个有特色且反映电子书内容的图标，可以绘制一个16色的32×32像素的图标文件（Icon），并把它存在素材目录中，然后在下面的Icon项中输入此图标文件名称。
5. Page/Directory Protection（页面及目录保护）选项
在这里可以管理电子书页面和目录的保护功能。对页面和目录设置访问口令实际是在主菜单的Utilities（效用）中进行“Define Page Password Key”（设置对当前页面的访问口令）、“Define Directory Password Key”（设置对目录的访问口令）、“Page to show if Access Denied（阅读权限时弹出的页面）等设置。
6. Feedback（反馈）选项
在这里可以设置读者发送反馈电子邮件的SMTP服务器（SMTP Server to use for Feedback）、邮件地址（E-mail“From”Address）。如果反馈邮件仅使用文本而不使用附件，可以选中SMTP/MAPI Feedback Style（SMTP/MAPI反馈样式）选项。
7. Directories（地址录）选项
在缺省情况下，WebCompiler 2000在编译电子书时会自动把素材存放目录及其子目录下的文件都编译进去，如果制作的电子书要链接到其它目录中的文件，可以点击“Add”将其目录添加进去。
Step 3. 生成
讲了这么多，恐怕有人会说制作电子书那么复杂啊，实际上一般情况下，只需设置通用（General）选项和外观（Appearance）选项即可，其余的选项可以根据需要进行设置或者使用缺省值。
现在，各种设置都完成了，让我们进行最后一步——生成电子书。选择主菜单Files下的Compile（编辑），会弹出一个编译窗口，进度条指示编译的进度。编译窗口上有Test和Ok两个按钮，按“Test”可以测试生成的电子书的效果。OK！效果还不错吧。
（山东 高峰）}
