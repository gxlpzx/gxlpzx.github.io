{软件搬运三力士}

{ 现在的硬盘容量虽然在以近乎10G的速度增长着，可是软件的“扩容”速度更让人无法忍受，很多程序默认的安装路径总是“C\Program Files”，时间一久，C盘自然“人满为患”了。简单地移动文件夹虽然确实能节省空间，但你会发现被移走的Windows程序可能大部分不能使用了（绿色软件除外），因为现在的软件在安装时除了要建立自身文件夹外，还要向几个系统目录中拷贝文件，此外修改INI、BAT、SYS文件……更改注册表更是它们的拿手好戏。那么卸载后再安装？太麻烦了，还是请个“软件搬运工”吧，它能把各种软件（甚至Windows）在分区间移动，所有和该软件相关的文件夹、快捷方式、动态链接库以及注册表等都会被同时移动或更新。
小生经过精挑细选，为朋友们推荐这三款软件如附表：
一、功能齐全的MagicMover
MagicMover（可不能简称为MM啊＾＿＾）？好像没听过，但PartitionMagic5.0你一定听说过吧，MagicMover就是该软件的一个组件，功能十分强大，然而令人遗憾和不解的是PowerQuest公司在新近推出的PQMagic6.0版本中将MagicMover组件去除了，所以该组件的1.02版本成了“孤本”。
现在我们以移动ACDSee为例，了解该软件的使用方法。点击“开始”→“PQMagic5.0 Pro”→“MagicMover”，出现主界面，可以看出移动软件有两种方式：一是可以选择“开始”菜单或“桌面”上的应用程序，二是在硬盘分区中直接寻找文件和文件夹。界面上的“Option”按钮可以设置软件搜索的驱动器，一般A盘不予选中，左侧显示当前操作的进度。我们先用第一种方法移动ACDSee，从“Programs”中找到ACD Systems，或者下一层的ACDSee Browser，点击“Next”，软件开始搜寻相关文件和链接，之后，进入路径设置界面。输入目标文件夹的路径（这里以D\Acd为例），或点击“Browse”（浏览）选择，这时可以点击“Details”看看MagicMover要进行什么操作，呵呵，除了相关文件夹和注册表外，它还要对几个INI、BAT、SYS文件下手呢！然后点击“Move”按钮，如果目标文件夹不存在，软件会提示是否建立，点击“是”后，MagicMove就开始搬运软件了。从进度指示条可以看出更新注册表耗时较长，移动结束后的界面会显示一些信息，像此次移动ACDSee共移动了4.84M的容量，点击“Details”可以再次看看操作详细步骤。点击“Close”将退出MagicMover，点击“Continue”将返回主界面。
现在我们用第二种方法再将ACDSee搬回去。选择“Scanned Drives”下的“D\Acd”，点击“Next”，在目标路径中输入“C\Program Files\Acd”， 然后点击“Move”按钮，ACDSee又乖乖地回到原来的文件夹中了。
MagicMover还能够对同一文件夹中的所有软件进行移动操作，方法是在“Scanned Drives”直接选择文件夹（如含有多个软件的C\tools），点击“Next”后，软件会显示“MagicMover was unable to determine a source folder. Multiple application source folders selected.”（多个应用程序的源文件夹被选中），点击“Move”，该文件夹下的所有软件都会被顺利地移动到目标路径下。
MagicMover在分析一个可执行文件时，如果它所在文件夹含有其它组件和子文件夹，会将它们一同移走。这样做可以避免软件其它部分疏漏情况的发生，也说明了该软件以文件夹为主要操作对象的思路。例如要移动Excel，MagicMover会将Word等其它Office组件所在的“C\Program Files\Microsoft Office\Office”一同移走。所以为了使MagicMover搬运得更准确，我们在选择需移动程序时应尽量选择较高层的文件夹，例如要移动Office，应选择“C\Program Files\Microsoft Office”，而不是“C\Program Files\Microsoft Office\Office”，避免留下残余文件。另外，MagicMover不能移动Windows和它的组件。
二、小巧玲珑的Mover98
Mover98的名气实在无法与PQMagic和Norton相比，但是它几百K的容量，强大的功能与其它两位动辄几十M的大哥相比，实在让人不敢小觑。
启动Mover98后会自动弹出操作向导，界面上列出了选择移动程序的四种方法：一是从开始菜单选择；二是手动从“C\Program Files”中选择；三是从硬盘各个分区中任意选择；四是移动Windows文件夹（别的软件都禁止的事情，这里居然任你选择，建议还是少移为妙）。如果你上次曾经移动过某程序，这个界面上会有一个“Undo”选项，供你反悔时用＾＿＾。以移动金山毒霸为例，我们选择第一种方式，点击“Next”，选择“金山毒霸”，再点击“Next”进入“选择文件夹”界面，这个步骤设计得十分体贴用户。如果Mover98定位的文件夹层次太深或者太浅，你可以双击修改（一般默认即可）。继续点击“Next”选择目标驱动器和文件夹，如果未选择目标文件夹，Mover98会自动填上默认名称，最后点击“Finish”，就会退出向导，进入Mover98主界面，金山毒霸已经添加到列表中了。Mover98左侧为功能区，上方的“Folder”菜单对应“Add（添加文件夹到移动列表）”、“Remove（去除列表中的项目）”、“Move（开始移动）”、“Option（选项）”按钮，“Help”菜单对应几个帮助按钮。右侧上方为移动列表区，下方为驱动器变化对比区。我们还可以利用“Add”按钮向列表中添加任务，从而实现同时移动多个程序。添加完毕后，点击“Move”按钮，会出现一个“Option”选项，询问是否需要快速搜索、文件检查、生成报告，为了节省时间，“Integrity check”可以不选，点击“Start”， Mover98就开始搬运软件了，从进度中可以看出依次进行的是“文件搜集”、“注册表更新”、“调整关联”、“移动文件夹和文件”，最后生成的报告可供打印和保存。
“Add”的另一种方式是“Express”（快速），可以直接填入源文件夹和目标文件夹。如果想将某一路径下的所有程序一同移走，只需选中该文件夹即可。Mover98还可利用右键启动，从“我的电脑”中直接选择要移动的文件夹或文件，单击鼠标右键，选择“QuickMove to”，从弹出的面板上选择目标文件夹，点击“Move”即可。
Mover98目前最高版本为3.1版，未注册版只能使用5次，这真是个遗憾。
三、让人失望的Norton CleanSweep 2001
提起Norton CleanSweep，确实让人直竖大拇指，因为它的程序跟踪卸载功能太强了，但是它的程序移动功能却有些小毛病，在2001版中也未见太大改进。例如移动ACDSee后居然切断了和图形文件的关联，移动“金山毒霸”居然把右键菜单中的启动项弄丢了，当然大部分程序还是能成功搬运的。使用方法是点击主界面上的“Programs”菜单，选择右侧的“Move Wizard”，以后的操作类似MagicMover，不再赘述。需注意的是，它的搜索程序功能不是搜索文件夹，而是搜索可执行文件。
如果你比较喜欢用功能丰富的工具软件，那就选择PQMagic5.0 Pro，在享受软件移动之余还能享受无损分区等许多实用的功能；如果你喜欢简洁单一功能的软件，那就选择小巧玲珑的Mover98吧，就凭敢于移动Windows的勇气，也该推它一把；至于Norton CleanSweep，如果你需要卸载程序的功能，一定要选择它，但是移动程序嘛，偶尔试试也不反对。＾＿＾
