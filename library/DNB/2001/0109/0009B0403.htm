{“隐蔽”克隆Windows 98}

{ 编者按：Windows 98以其强大的功能、友好的界面、简单的操作深受广大用户青睐，但Windows 98系统的脆弱性也令用户痛苦不已，尤其给学校机房的系统维护工作带来了很大的麻烦，要么不能启动，要么出现致命错误，一般难以修复。超级保镖、硬盘还原卡等软硬手段由于兼容性及其他方面的因素，也往往难以奏效。重新安装在所难免，面对几十台机器令管理者苦不堪言。在此，山东的朱西讲朋友在此向广大机房管理者推荐一种一劳永逸的方法——“隐蔽”克隆Windows 98。此法通过克隆（Ghost）软件与魔术分区大师（Magic Partition 3.0版）相结合，将克隆Windows 98系统生成的映像文件保存到一个隐含分区，以达到保护数据的目的，必要时从隐含分区的映像文件自动恢复Windows 98系统。具体步骤如下：
1.划出部分硬盘空间用Fdisk程序建立基本活动分区，格式化后安装Windows 98系统及应用软件，使系统处于最佳状态。（为确保安全性，建议将Fdisk.exe改名或删除。）
2.将魔术分区大师中的Pqboot.exe文件拷贝到“C\windows\command”文件夹，并在此文件夹下建立包含命令行“Pqboot /P：2”的批处理程序Sys1.bat。
3.用魔术分区大师在剩余硬盘空间（或部分剩余空间）建立一个“特殊”基本分区，分区类型（Partition Type）为FAT，并将其改设为非隐含分区后重新启动计算机。
4.将新建立的“特殊”基本分区格式化为系统盘后，将文件Ghost.exe、Pqboot.exe拷贝到此盘，并在此盘根目录下建立自动批处理程序Autoexec.bat，内容为：
＠ECHO OFF
IF EXIST DISKC.GHO GHOST -CLONE，MODE＝PLOAD，SRC＝DISKC.GHO1，DST＝11 -SURE -FX
IF EXIST DISKC.GHO Pqboot / P1
5.更换活动分区。在DOS提示符状态下执行批处理程序Sys1.bat后，计算机会自动重启，此时安装有Windows 98系统的基本分区即变为隐含分区。
6.执行Ghost.exe，将Windows 98隐含分区克隆生成映像文件DISKC.GHO，并将其存放于C盘根目录下。
7.更换活动分区。在DOS提示符状态下执行命令“Pqboot/P1”，计算机自动重启后存放映像文件DISKC.GHO的分区即变为隐含分区。
至此，整个硬盘的自动恢复系统安装完成。机房管理者只需把此硬盘作为母盘，利用Ghost软件的整盘克隆功能，将其他机器的硬盘一一进行克隆即可。所有工作完成后，管理者就再也不用为维护Windows 98系统而苦恼了。以后，如Windows 98系统损坏，只需在DOS状态下运行批处理程序Sys1.bat ，不需其他任何人工干预，系统会自动将Windows 98系统快速恢复。欢迎大家来信讨论，我的Email是sf77＠china.com。}
