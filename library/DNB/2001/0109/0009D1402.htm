{OICQ黑客工具大曝光}

{ 使用OICQ的网友都不愿意看到自己的OICQ账号给黑掉的一幕，要防备盗窃OICQ密码的软件，就要从认识它开始。
●OICQ2000密码破解工具——OICQ2kpass
功能介绍：它只能破解本地电脑OICQ登录选项中曾选了“自动登录”功能的OICQ账号，如果是破解多号的OICQ目录，得出的密码是最后登录保存的密码。
运行平台：Win95/98/me/NT/2000
授权方式：免费软件
软件大小：230K
OICQ测试版本：OICQ2000b Build 0115
使用说明：无须安装，解压软件后，你只要运行OICQ2kpass.exe文件，然后选择好OICQ所装的目录名，再按“取得保存密码”这几个字，OICQ密码就全看见了。
下载网址：http://www.51299.com/OICQ/OICQ2kpass.zip
防御方法：不要使用OICQ的自动登录功能。
●OICQ密码间谍OICQLog920
功能介绍：OICQ密码间谍是一个具备自动隐藏功能，自动开启监听记录OICQ密码，OICQ聊天内容的软件，适用目前使OICQ各个版本。
运行平台：Win95/98/me
授权方式：免费软件
软件大小：253K
OICQ测试版本：OICQ2000b Build 0115
使用说明：首先要运行一次软件包的Sysmon.exe文件，它就会在每次开机时自动运行监听记录功能，密码会记录在c:\log.txt，聊天记录文件在c:\inst0391.tmp文件夹内。
下载网址：http://www.51299.com/OICQ/2001/0217/OICQLog920.zip
防御方法：OICQ小侦探2.0http://www.51299.com/OICQ/tty2.0.zip就是一个检查计算机里是否潜伏有OICQ密码间谍的软件点击“查找间谍”即开始为你工作，如果找到，点击“查看密码”，就能看到你被盗的密码，点击“清除间谍”即可将你的计算机里的OICQ密码间谍软件清除干净。}
