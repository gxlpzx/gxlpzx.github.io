{近距离作战5技巧指南}

{ 《诺曼底登陆》是《近距离作战》系列之五，它无论是在画面及人工智能方面都比以往有很大提高，是一款真正可以满足二战迷的即时策略游戏。
在选择PLAY GAME进入单人模式后，出现三个选项BATTLES、OPERATIONS、CAPTINGS分别代表真实的历史战斗、局部战斗和大规模战役，右边的图案是战斗双方的选择， 下有八个选项，先是真实选项，左面是我方，右面是敌人，可以设定自己的部队是久经考验的精锐还是没打过仗的新兵。然后有四个选项：
1.始终可以看到敌人，你可以始终观察敌人的部署和动向，不过这样就没意思了。
2.火力不会伤害到自己的部队。
3.始终可以看到敌人状态，这个选项打开后你在战斗中点击敌人就可以看到他的状态，弹药啦、死伤啦等等。
4.永远服从命令。士兵有抗命的可能，其抗命可能性与其军衔成反比（军衔越高越服从），这样你命令士兵迎着炮火前进，很可能听到就是一串拒绝的德语，当选择以后，你的作战单位就会完全听从你的命令，即使你让它自杀。
选取之后按NEXT进入下一屏，这一屏是你的兵力菜单，左边是预备队的数目，右边是成编制的部队。预备队的各项指标都要低一些，因为这款游戏最出色的地方在于可以表现出一个士兵在战火纷飞的环境下内心反映流露无遗，会害怕，会发呆，会发疯，会投降等等不可预料的行动。一个新兵很容易在猛烈的炮火下出现精神崩溃，这时就需要富有经验的老兵给予精神支持，军官在身边也能起到一定的稳定军心的作用，战斗中如果军官战死或重伤，对士兵的影响是很大的，战斗中如果陷入包围，部队大部伤亡后会放下武器自动投降。所以应尽量避免部队被全歼和技术兵器的损失。左下方是指挥官的肖像，点击EDIT OPPONET可以看对方的兵力，笔者就曾经看到过巴顿。玩家可以在这一屏中对兵力进行调整，也可以观看部队情况，用每一个士兵的各项属性，来衡量他们的身体及心理的状况，如士气（Morale）、经验（Experience）、紧张（Suppression）、疲劳（Fatigue）、健康（Health）、团结（Cohesion）以及所得勋章等等。再点击NEXT便进入游戏了。(^09070302A^)
游戏首先是布阵，在每场战斗的开始都给你足够的时间选择并布置好作战单位，把部队布置的范围限定在地图的高亮部分。一般来说电脑排出的阵型既保守又没用，所以你最好还是自己调整一下。在你想要调整的部队上按左键拖动至新位置既可。布阵尽量找一些进可攻退可守的地方。士兵最好放在屋子和丛林中，坦克放在道路上或者掩体后面。炮放在树林或者掩体后。布置完毕之后按BEGIN开始游戏，最开始一般都是平静，小鸟唧唧喳喳的，还有奶牛在叫。但是这种平静一般只能维持2分钟左右，在双方部队各自进入射程之后，炮火就开始了。在你的作战单位上点击右键，出现一个菜单，使用时先点选要下达之指令，然后把鼠标移动到画面上。指令从上到下分别为：
MOVE：移动，从一点到另一点。点选之后把鼠标移动到目标地点单击左键。特点是受攻击后可以迅速隐蔽。
MOVE FAST：冲锋，被击中几率增加。
SNEAK：匍匐前进，被击中几率最小。
FIRE：开火，点选之后把鼠标移动到射击对象身上按左键确定，如果连线变红，表示因阻碍而无法射击。
SMOKE：烟雾，点选之后把鼠标移动到释放烟雾方向按左键确定。
DEFEND：防御，点选之后卧倒，不主动进攻，画面上会出现一个蓝色弧线，它代表防御方向，单击该弧线可以更改防御方向，受攻击可能变小。
隐蔽：点选之后卧倒隐蔽。画面上出现绿色弧线。亦可以更改，受攻击可能更小。
游戏时要注意以下几点：
1.注意步兵坦克协同进攻。千万不要让坦克独自进攻，否则容易被敌方击毁，德军的反坦克能力一般要比美军强。一般来说，坦克作前导，步兵要随坦克推进，机枪班在两侧掩护。坦克在作战时要注意消灭敌方火力点和装甲车辆，起到进攻在前，后退在后的作用。
2.部队在前线移动时尽量用隐蔽移动。轻重机枪是防守阵地或进攻时压制敌人火力的利器，对付大股进攻步兵效果最好；如果距离近的话，还能摧毁装甲车。开战前就要将它们架设到制高点，射击面要开阔，不能有障碍物。
3.不可低估炮兵的力量。游戏中使用的炮分为三类：反坦克炮、迫击炮、反步兵炮。炮兵可以进行火力侦察以及烟雾掩护等，专门的反坦克炮及加农炮对打坦克有很好的效果，迫击炮如果直接命中也可以击伤摧毁敌方车辆。在射击中，要时刻注意使炮兵确定攻击目标，否则有时电脑会自做主张，停止攻击，显示要节省弹药什么的，很搞笑。在开阔地攻击多个装甲目标时，最好派一个迫击炮单位在后面向前方释放烟雾弹，这样被击中的几率会更小。反坦克炮对付步兵也很好使，尤其是对掩体或房屋里埋伏的敌人。但要注意别让敌步兵太靠近，否则会被敌步兵用手榴弹或冲锋枪搞掉。如果条件允许，最好布置一轻机枪班在附近掩护。
4.装甲部队分轻重装甲车及轻重坦克。装甲车及轻型坦克装甲薄弱，多配备重机枪及小口径炮，少数配有迫击炮及火焰喷射器。由于正面装甲是最厚的，而侧面和尾部装甲相对比较薄，因而要尽量把你坦克的正面装甲朝向敌人，而不要把侧面暴露于敌人的火力之下。除非遇到敌人的大口径反坦克炮，德军的重型坦克在战斗中几乎是无敌的，但要提防空袭。
5.士兵注意尽量依靠掩体。游戏中的掩体很多，比如屋子、战壕等等，士兵要尽量在这些物体的掩护下行动，这样伤亡会减少很多。
6.没武器和弹药时让自己没有武器的士兵在已牺牲的士兵身上停一下，就会发现他们已经捡起了武器。所捡的武器种类和子弹数量受原武器拥有者限制。另外，美军和德军的武器和子弹相互是兼容的所以到敌人的尸体捡东西也是可以的。
好了就写这么多，祝各位喜欢这个游戏。}
