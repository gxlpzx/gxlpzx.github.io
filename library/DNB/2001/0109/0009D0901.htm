{上网七件事之杂乱事}

{ 上网还能干什么？除了前几期介绍的六件事之外，这里还有最后一件事，就是杂乱事了。顾名思义，杂乱事自然要有点乱了，但它们又是和网络应用密不可分的。下面我们就先从网上视听开始，讲讲这杂乱事。
最近网络流媒体发展得如火如荼，一些比较著名的广播电台都在网上安家落户，并且提供在线视听服务。朋友们都是知道的，使用IE中提供的电台功能，就可以有选择地在线收听网络广播信息了。不过就目前国内的网络带宽而言，在线播放视频目前还难以达到令人满意的效果，但是在线收听广播的效果还是可以接受的。不过也不要为此叹息，随着我国国际出口带宽的增加，相信不久的将来，收听和收视流畅的广播和视频绝对不是奢望。
在众多的流媒体播放软件中，Realplayer是最受欢迎的网络流媒体播放器，是网上收听收看实时音频、视频和Flash的最常用的工具，在28.8kbps或更快的连接速度下，能够实现接近CD音频的效果。它的特点就是不必下载音频、视频内容，只要带宽允许，就能完全实现网络在线广播，在众多的播放器中，它几乎支持所有的多媒体文件格式。
另外，Microsoft Windows Media Player 7.0是一个全新的基于DirectShow 体系结构的多媒体播放器，提供了可扩张性、灵活以及方便的功能，支持目前流行的大多数多媒体格式，已经拉开挑战Real Player的序幕。Apple QuickTime 4是一款老牌的多媒体播放软件，应用也非常广阔，目前，网络上的数字视频几乎一半都是采用 QuickTime的影片格式，它已成为当前网络数字视频和流媒体传输的主导技术。 QuickTime可以处理目前流行的视频、音频、动画、图像以及文本文件等，支持200多种数字格式的文件。
随着互联网技术的发展，网络IP电话逐渐成为受网友欢迎的网络通讯工具之一，使用IP电话可以任意拨打长途，只需支付市话费，而无需支付高额的长途话费。网络电话主要分为三种，即PC-PC、PC-Phone和Phone-Phone，其中最后一种是需要特殊的电话设备来支持，适合普通用户使用。而大多数网民关心的还是前两种网络电话，目前比较常用的网络电话有很多种，比如MediaRing Talk、Net2phone、Internet Phone等，各种网络电话都有不同的特点。
Internet Phone是一个共享软件，它的通话方式为PC-PC和PC-Phone，对硬件的要求很低，由于采用了先进的算法，即使使用28.8K的调制器，而且只要对方也在使用IPhone，就可以在网上比较清晰地听到对方的话音了。MediaRing Talk是一个免费软件，可以实现的通话方式为PC-PC，它的界面非常友好，通话质量也有比较高的水准，而且噪音非常小，即使在网络使用高峰阶段，它的通话音质也同样会达到令人满意的效果。但是如果被叫方没有安装同样的软件，它就无计可施了。再就是Net2Phone了，它是一个免费软件，但是打电话需要额外“买”时间，通话方式为PC-Phone，但是话音质量较前两种要逊色了一些，有时会出现说话断断续续或变调现象，不过也不失为一款非常不错的网络电话软件。
对于经常出门在外的你，是否也打算千里之外遥控你的电脑呢，在网上，这已经不再是梦想了。通过远程控制就可以实现，所谓远程控制，是指管理人员在异地通过计算机网络、异地拨号或双方接入Internet等手段和方式，联通需被控制的目标计算机，将目标计算机的桌面环境显示到自己的计算机上，通过本地计算机对远方计算机进行配置、软件安装程序和修改等工作。远程控制可以实现远程遥控、文件传输、远程屏幕浏览、键盘交谈、Internet连接，有的还可以进行语音交谈、故障报警，并通过Web页面进行远程控制，甚至还可以进行远程重启计算机、远程格式化硬盘、远程系统安装、远程软件安装等。远程控制的实现过程可以通过远程控制软件来完成，远程控制软件一般分为一个客户端程序Client和一个服务器端程序Server。
PcAnywhere是远程控制软件中的佼佼者，可以提供全面、安全和可靠的远程控制能力，可以通过网络对远程服务器、用户台式机和便携式移动计算机进行控制，而不必直接接触计算机就可以对故障进行诊断和排除。支持TCP/IP、NetWare、IPX/SPX、NetBIOS和Banyan VINES，不必考虑网络的连接方式是直接LAN连接、Modem、ISDN、NetWare Connect NASI、Internet还是红外连接等。
由于远程控制软件允许对用户的被控端计算机进行远程访问，因此就有可能被控计算机暴露在大庭广众之下，而那些未经授权的用户也就可以进行访问了，并获得有关的网络资源。因而，远程控制技术的安全问题也不容忽视，此类软件如木马冰河、YAI、特洛伊木马等。
其实上网还不止以上提到的这些事，还有很多，比如寻医问药、虚拟旅游、网上读大学，还有的像什么网上办公、读书看报、网上购物，参加社区活动等，总之来自网上的世界无比精彩，为之感动的朋友们，赶紧驾“猫”冲浪去吧。}
