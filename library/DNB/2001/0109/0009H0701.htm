{“4C”的赛博}

{ 在进入21世纪的今天，计算机产业和通讯产业已逐渐成为中国的支柱产业，并将成为下世纪中国经济的希望。据相关信息统计，中国计算机市场在进入20世纪90年代后，多年保持超过40％的增长率。IT、通讯市场也涌入越来越多的商家和一些精英。媒体对IT、通讯业的报道也持续升温，人们的注意力越来越集中在这个领域了。
任何市场都有一个“巢”，“巢”的作用在于提供更大的利润空间和优良的管理，同时也可以更好地为客户和市场服务，并把握住市场的动态，及时反馈市场的要求。对于电脑买卖市场也需要这样的一个“巢”，所以，各个地方的电脑城在近几年如雨后春笋般一个一个快速地冒了出来。当然这些电脑城自身不仅是一个“巢”，也是一个商品，也是要经受市场经济的考验的，所以电脑城也像其他商品一样遵循则一个原则——优胜劣汰，一个倒下去了总有另外一个会站起来。赛博数码广场就是其中的一个佼佼者。赛博数码广场的英文名称是“CyberMart”，赛博是英文单词 “cyber”的音译，是网络、数码的意思，“CyberMart”代表的就是数码产品交易的场所，这也是赛博数码广场名称的来历。
说起赛博数码广场，可能很多人并不会感到陌生，在《电脑报》2001年的“市场与渠道”版面中，对全国各大城市的介绍就不止一次提到了赛博数码广场。1999年12月25日，当赛博数码广场上海旗舰店（赛博总店）在上海建立以后，赛博数码广场在全国各个大城市相继开业，并迅速地赢得了当地商家和顾客的支持。到目前为止，已经在重庆、武汉、南京、合肥等地建立了分店，并预计在5年以内在北京、广州、西安、沈阳等一二级城市再建立50家分店。
赛博数码广场的经营是成功的，这与它鲜明的口号和准确的定位分不开，它在全国首先提出了“4C”的经营理念。所谓“4C”就是计算机产品（Computer）、通讯产品（Communication）、数码家电（Consumer electronics）和网络产品（.com）。所以“4C”＝“电脑卖场＋手机世界＋数码家电＋网络产品”。赛博数码广场另外一个概念是“Shopping mall”，或者也可以说是“One stop shopping”，即一次性购足的概念，赛博将整合手机、电脑、数码家电和网络产品，整合它们之间的联系并加以应用。赛博数码广场不仅销售品种齐全的数码产品，如电脑整机及配件、电脑外设、耗材、软件、手机及其它通讯产品、数码家电和网络产品等等；赛博还是一个集购物、娱乐、学习、维修、服务为一体的人性化的购物广场。赛博数码广场于商场内还设有独具匠心的功能区，比如免费游乐区、免费上网区、手机医院、电脑医院等等。当然，非常重要的一点是，赛博同时为消费者提供优质的售前和售后服务，例如销售咨询、免费送货、免费上门安装、手机及电脑的免费检测和维修等，让消费者在舒适、亲切的环境中买得更放心、称心、开心。
另外，CyberMart也是一种产业渠道，是“4C”产品生产厂商与消费者直接交易的场所，厂商在赛博展示其公司形象和最新产品，消费者在赛博则可以了解和接触新产品。
走进赛博，我们可以看到明亮的大厅，人头攒动。赛博有它固定的商场布局，一般来说，门前的空地是厂家和商家做促销活动最好的场所，周末的时候，这里是最热闹的地方。一楼为品牌馆，包括品牌整机、ISP ＆ICP、形象店等，荟萃国际国内知名电脑数码产品品牌；二楼为装机区；三楼为多功能馆，包括电脑书店、网吧、免费电脑教室、电脑外设、耗材、配件及软件区等，这也是赛博所拥有的特色——功能区。功能区根据性质分为培训性、服务性、多元性三类。培训性功能区包含免费电脑教室、免费上网区、免费电玩区；服务性功能区包含电脑医院、手机医院、网吧、电脑书店，多元性功能区包括会员中心、礼品中心、商务中心、软件交换俱乐部、简餐、文具店等。功能区为消费者提供很高的附加价值，向消费者提供学、玩、修、沟通的机会。举例来讲，很多消费者没有条件接触到Internet，不能使用E-mail，但是他们有去了解和掌握的需求，这种矛盾是知识与财力的矛盾，CyberMart的免费上网区就将为他们提供一个很好的上网机会。免费教室是电脑入门者优异的学习场所，可以培训家庭电脑消费者正确操作和应用方法，为家庭电脑消费者提供后续服务等；电脑医院、手机医院为消费者提供深层配套服务，除给予消费者方便外，更大的意义是解决了消费者的检测和维修问题。在赛博，消费者可以尽享网上冲浪的乐趣，没有电脑也可以学电脑，还可以在游乐区感受电玩的紧张、刺激和欢快。功能区的作用是为消费者服务，排除他们购买时的各种难题。
“拥抱消费者”是赛博数码广场看待消费者的核心观念，也就是说赛博经营的中心思想是对消费者的服务。赛博数码广场在可能涉及的各个方面都为消费者进行了周全的考虑；比如说产品方面，赛博数码广场为消费者提供尽可能多和好的商品，经常举办的展示活动、优惠活动、特价活动等都是照顾消费者的切实利益。“4C”业界的最新动态和新产品的上市也将在赛博首先得到发布和展示。在服务方面，消费者在CyberMart首先可以得到免费的咨询和指导。为提高整体服务水平，赛博还针对经销商以及营业人员提供各种不同的培训，以提高经销商和营业人员的服务意识和服务水平，赛博的手机医院和电脑医院还向消费者提供免费的检测及维修，使消费者在赛博得到除厂家保证外的又一重服务保障，赛博数码广场实行的是“让顾客满意服务”——“CS”服务。消费者在赛博数码广场享有充分的自由，拥有多种多样选择，因为赛博是集购物、学习、娱乐、维修、服务为一体的数码广场。功能区的设置，联合厂商、经销商和消费者的活动令赛博成为一个丰富多彩的数码世界，赛博创造了真正的人性化的购物广场。
现代社会是数码的时代，赛博数码广场对业界厂商、经销商和消费者来说将成为一个渠道，发挥着媒介桥梁的作用。然而无论如何，我们可以看到，赛博全面规划和设计的一切，都没有脱离以服务消费者为核心的经营思想。可以预见，把握创意、实力和正确方向的赛博数码广场将可为中国“4C”行业的发展提供有益的经验。 }
