{联想电脑天禧二90X0登场}

{ 3月5日，联想1＋1家用电脑天禧二90X0世纪经典版电脑进入市场。天禧二90X0世纪经典版电脑可谓高新科技产品的典范。首先，采用的是P4 1.3GHz处理器，而配合P4 1.3GHz处理器的是850 PRO豪华主板，其稳定的性能可使P4的性能发挥最完美。其次，天禧二90X0世纪经典版采用48× CD-ROM＋COMBO（DVD＋CD-RW）双光驱组合，其中COMBO集播放DVD、VCD与可重复擦写光盘功能于一体，为用户带来的是超强的多媒体体验。另外，为更好配合游戏和上网功能，天禧二90X0世纪经典版特有的光电力反馈鼠标，光电定位准确，感应灵敏，并且可以将逼真的运动效果展现给使用者，从而带来非凡的上网和游戏新感受。值得一提的是，其标配的摄像头和视频捕捉卡，为用户带来一套完整的多媒体应用解决方案，可实现自己摄录、编辑、制作VCD、电子贺卡等视频节目。
据联想负责人介绍，现在，联想的研发重点着眼于技术创新，从天禧二90X0电脑中就可以看出，技术创新及其应用处处可见。然而，在把任何一项新技术运用到产品之前，联想都会进行反复检验与测试，即联想推出的产品不仅是客户需要的，而且在技术上也是非常成熟、最有把握成功的。目前，联想家用P4机型在技术上已经非常成熟。该负责人还充满信心地说，联想此次推出的主流价位的天禧二70X0 P4机型，不仅在性价比上极具竞争力，而且其强大的因特网功能，高水平的工业设计，必定会更好地满足高端用户需求，赢得市场主动权。
另外，在刚刚结束的寒假促销中，联想家用电脑销售取得了一天接单6万台的佳绩。为回馈广大用户厚爱，联想决定，联想1＋1家用电脑部分机型于3月5日大幅升级，同禧528、未来先锋500的CPU由PⅢ866升级到PⅢ933。更让人惊喜的是，凡购买天禧二90X0世纪经典版的用户，可免费获得一台市价1098元的联想Z32喷墨打印机，而购买天禧6XXX的用户，则可免费获得一个PC摄像头。 }
