全“芯”世界——初识主板芯片组

“芯片组”一词是从英语“Chipsets”翻译过来的，它们是组成电脑主板的一组非常重要的芯片。芯片组通常由一块至三块集成电路组成。电脑中的CPU、内存以及显示、存储和输入、输出设备都必需通过主板来进行系统连接，这样才能组成一台完整的电脑。电脑运行时主板芯片组将在CPU的控制下通过主板各种接口有机地协调系统中各输入、输出和存储设备的正常运行，以保证电脑顺利完成所运行的各种程序。因此电脑的功能强弱、性能优劣与主板有密切关系，而电脑主板性能优劣和功能强弱又主要取决于芯片组的性能和质量，所以，如果将CPU比喻为电脑的“心脏”，那么将芯片组比喻为电脑主板的灵魂一点也不过分。
技术结构
目前各种主板上所使用的芯片组根据技术结构特点一般由一至三片集成电路组成，其中Intel(英特尔)公司的中央控制型芯片组主要是由三片组成的，VIA(威盛)公司、ALi(扬智)公司生产的南、北桥型芯片组主要由两片芯片组成，SiS(矽统)公司所生产的单片型芯片组主要是由一片组成的，它们的技术结构和特点如下：(^29060301e^)
##2●中央控制型芯片组
Intel公司自i440BX芯片之后，开始研发i810系列“中央控制型”芯片组，这种芯片组由GMCH“Graphics & Memory Controller Hub/图形、内存控制中心”芯片、ICH(I/O Controller Hub/I/O控制中心)芯片以及FWH(Firmware Hub/固件中心)芯片共同组成(图(^29060301a^)1)，目前Intel所产各类i815以及支持Pentium 4的i850芯片组仍然采用这类结构。
##2●南、北桥结构型
南、北桥型结构芯片组(图(^29060301b^)2)由北桥(North Bridge)芯片和南桥(South Bridge)芯片共同组成。一般来说，靠近CPU的芯片是北桥芯片，靠近PCI槽的为南桥芯片。目前南桥芯片、北桥型芯片组分为传统型和改进型两种，两者之间的区别主要是南、北桥之间的数据交换总线类型和带宽。目前常用的KT133A主板采用的是传统型南、北桥芯片组，而最新的VIA PRO266/KT266主板使用的都是改进型的南、北桥芯片组。
北桥芯片一般为主板提供CPU、内存、显示处理系统的管理功能，南桥芯片与主板BIOS配合，在电脑加电时对系统中各种器件和设备进行检测。
##2●单片整合型
将南、北桥芯片合二为一就成为了单片型芯片组，目前这类芯片组主要由SiS公司生产，典型的有SiS630(如图(^29060301c^)3)、635、SiS730、735等。SIS公司虽然将南、北桥芯片整合在一起之后，但其内部电路结构仍然按南、北桥结构分工。同样为了提高南、北桥电路之间的数据交换率，SiS公司也专门开发了一种速率为1GB/s的南、北桥专用总线，因此提高了芯片内部的数据交换速率。
将南、北桥芯片整合在一起之后可以为厂家降低生产成本，也可以为主板生产商在设计主板时带来方便(减少了芯片组在主板上的占用面积)，因此单片型芯片组在主板市场上也占有一席之地。
分类和特性
目前主板芯片组中大部分由Intel公司和VIA公司生产，少量由SiS公司、AMD公司以及ALi公司生产。由于自Intel Pentium Ⅱ开始，Intel CPU和AMD CPU的安装接口开始不兼容，因此主板芯片组也开始按支持CPU类型进行分类，由于Intel Celeron、PentiumⅢ以及VIA C3、Cyrix MII都使用Socket 370插座，而AMD的雷鸟(Athlon)和毒龙(Duron) CPU使用Socket A(即Socket 462)插座，所以目前的主流芯片组也按此进行分类。
##21.Socket 370
在支持Socket 370接口CPU的芯片组中，Intel公司的i810、i815和i850三种芯片组是目前主板应用的主流，其中810E面对低端用户，i815系列中的i815E、i815EP则面对中、高端用户，而i850芯片组专用于Socket 423接口的Pentium 4，随着Pentium 4的不断降价，i850芯片组的应用也将越来越广。
在兼容级芯片组，VIA公司也分别以Apollo PLE133、Apollo Pro133A和Apollo Pro266三种芯片组面对低、中、高端用户，其中Apollo PLE133整合了显卡，但不提供AGP插槽，主要用于组装低端电脑，其竞争对象是Intel公司i810E芯片组。另外SiS公司的一种单芯片SiS633和SiS635芯片组也是支持Socket 370架构CPU的，但仍主要面对中、低端用户。
就以上各种芯片组而论，尽管它们的功能有所不同，但组装电脑之后的软、硬件兼容性、对AGP显卡和IDE存储设备的支持情况和系统性能稳定性都比较好的还应该算Intel的芯片组，这可能是由于CPU和芯片组都是由Intel自己设计和生产的缘故吧！
##22.Socket A
目前只有VIA公司、SiS公司和AMD公司自己为雷鸟和毒龙生产芯片组。目前支持Socket A接口CPU的芯片组中主要有VIA公司的Apollo KT133A、SiS公司的SIS730、AMD公司的AMD760和AMD761等，它们的具体技术特征见表(^29060301d^)2。由于AMD目前市场占有率最高的是VIA公司的KT133A和KT266芯片组。就以上芯片而言，根据一些媒体的评测和玩家的体会，功能较强、性能比较稳定的是AMD公司的AMD760，但由于芯片组生产不是AMD公司的主业，因此市场极少见到AMD760芯片组的主板。
