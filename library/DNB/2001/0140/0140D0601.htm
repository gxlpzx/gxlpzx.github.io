保护隐私的两款软件

编者按：信息时代，用户的个人资料成了网站的重要资源，同时也成为被买卖的对象。据说，在网上只要花费不多的钱就能买到大批用户的个人资料：如姓名、地址、电子邮件、电话号码甚至是信用卡号码等，所以，保护自己的个人信息非常重要、也非常必要。
一、 免费软件的代价
绝大多数的共享软件和免费软件中会有一些广告，其中多数广告都会在我们的磁盘上偷偷安装一些小程序，我们称之为间谍程序，它们会帮助广告商收集用户的私人信息，例如用户点击广告的情况、下载过什么软件、喜欢用什么软件、上网习惯、邮件账号等，一有机会就会发送到远程的某台计算机上。
即使是绝顶编程高手，也不会把下载的软件全部反编译一遍，剔除其中的可疑代码，这也不是笔者写这篇文章的目的，本文介绍一款能够搜索间谍程序的软件Ad-aware。
1.下载和安装
Ad-aware能够搜索我们的硬盘中有哪些间谍程序，能从内存、注册表和磁盘中把它们彻底清除。我们可以到软件的提供网站http://www.lavasoft.de下载，也可以到我的主页http://azhouonline.home.chinaren.com或者http://azhouonline.longcity.net下载。软件大小为641KB，免费软件。软件的安装非常简单，不再多说。
2.软件的设置
要想让软件全面、准确地找到隐藏在系统里的间谍软件，需要对软件进行合理的设置。在主界面中点击“configuration”按钮，在设置窗口中可以看到，“General Settings”下面有四个选项：“Mark all references found by default”，搜索之后把所有的记录全部选中，但并不是立即清除；“Display smart hints in List view”，当鼠标移动到某个记录时自动显示详细资料；“Play alarm sound”是设置提示声音；“Auto save log file”，自动保存，下面是自动保存的位置。在“Scan Option”下共有七个选项：“Scan on Windows startup”，每次系统启动时自动搜索；“Automatically remove all references found”，搜索结束后自动清除间谍程序，这个选项一定要慎重选择，因为某些软件离开了间谍程序后可能罢工；“Scan Memory”，搜索内存，如果找到内存中正在运行的间谍程序，立即给予中止；“Scan Registry”搜索注册表；“Extended Registry Scan”，扩展搜索，能够搜索出一些未知的间谍程序；“Scan Boot drive”，检测硬盘主分区，搜索磁盘上的所有间谍程序；“Scan all fixed drives”，搜索所有的硬盘(图1)。设置完成之后，按下“Proceed”按钮加以保存，如果设置错了，可以按下右上角的“Default Settings”恢复到默认状态。
3.软件的使用
在主界面上按下“Scan my system”按钮，系统即会按照设定的方式进行搜索，搜索的结果将以列表的形式提供给我们(图2)。如果设置了扩展搜索和搜索磁盘，Ad-aware不仅能够搜索出间谍程序，还能搜索出不安全的Cookies。对于搜索到的结果，我们可以选中前面的复选框，然后按下“Continue”按钮将之删除。
需要提醒大家注意的是，这款软件也不是万能的，有时它也会误报，如果真的把那些不该删除的程序删掉了，有可能造成部分软件不能使用。所以建议大家先备份注册表，然后只搜索内存和注册表，清除注册表中的间谍软件信息后，如果过一段时间后感觉系统没有什么变化，再全面检查，彻底清除，以防不测。
二、 网上冲浪的烦恼
为了对付不安全的Cookies和Web Bugs，IDcide公司专门制作了一款小软件IDcide Privacy Companion，下面介绍一下这款软件的详细使用方法。
1.下载和安装
我们可以到软件的提供网站http://www.idcide.com下载。软件大小为1.32MB，免费软件。安装方法在此不再多说，非常简单。
2.软件的使用方法
这是一款浏览器的插件，启动IE之后，会在IE标题栏右侧提供三个按钮(图3)，左面的是网页监视情况，中间的是网站跟踪情况，右面的是安全级别和下拉菜单。
当我们上网时，如果IDcide Privacy Companion发现当前网页有监视我们的Cookie或者Web Bug存在，左面的图标就会变成一眨一眨的大眼睛(图4)，如果发现有站点的Cookie在跟踪我们，中间的图标就会就成三只眼睛。如果我们设置得当，软件一旦发现我们被监视，会自动将其中止，眼睛就会被打上一个“×”号。
点击左侧或中间的图标，软件会把详细情况以列表的形式告诉我们(图5)，在“Sites”页内，告诉我们网页中有哪些内容企图监视我们的行踪，在“Tracking Networks”中告诉我们哪些站点企图跟踪我们。
3.软件的设置
这款软件提供了高级防护、基本防护和不防护三个级别，我们可以点击右面的图标从下拉菜单中选择，其中“High Privacy”为高级防护，在这种级别下，发现不安全的行为将自动中止；“Medium Privacy”为基本防护，它能警告我们网页不安全；“No Privacy”为不防护，这是为了防止在保护模式下某些网站无法访问。
点击下拉菜单中的“Settings”，能够对默认的防护级别和每个防护级别的防护内容进行设置(图6)。默认防护级别可以设置为高、基本、不防护和上次设定的级别，安全防护内容可以设置为防止网络追踪和站点监视，我们可以根据自己的需要来设置。
需要提醒大家的是，如果某些网站在屏蔽了Cookies就不能访问时，开启这款软件会对访问网页有影响，如果把安全级别设为了不防护也无法访问，那就暂时把它“Disable”好了。
编后：我们前面介绍了两款软件，分别用于防范间谍软件和网络追踪，其实这还远远不够，网上能够搜集个人资料的方法很多，我们的浏览器和操作系统也还存在很多的问题，仅用这两款小软件还不能完全彻底地保护自己，笔者写此文的目的正是想抛砖引玉，希望大家能够把保护隐私这个问题重视起来，也希望国内的编程高手们编制一些能够有力保护个人隐私的工具，另外，我们在上网时也应随时注意，不向网站和外人泄露自己的真实情况，不把重要的信息存放在电脑中，养成良好的浏览习惯，定期清除历史记录和Cookies，不随便使用网上下载的软件，彻底杜绝资料外泄。
