专家坐堂

光驱为何一读盘就死机？
问：我的光驱是ACER 40×，用了一年多，现在读盘能力大大下降，而且很容易出现读盘死机的现象(操作系统为Windows98)。经过测试，发现大部分读盘死机的情况是发生在读取具有自动运行功能的光盘上，其它光盘虽然难读，但不会死机。请问如何解决？
答：光驱在用久了之后比较容易出现读盘死机的现象，不仅仅是在Windows98中，在Windows2000中也会发生。出现这种原因，除了光驱本身读盘能力下降之外，其FIREWIRE也存在一些小问题。你可以通过升级光驱的固件(FIREWIRE)来修正原固件中的一些BUG(ACER 40×光驱发售后又先后两次发布了新的固件升级程序)。如果问题依然存在，建议你禁用操作系统的Autorun功能，可采用下面两种方法：1)临时禁用法。在插入光盘的同时按住“Shift”键，可以禁止光盘自动运行；2)永久禁用法。运行注册表编辑器(REGEDIT)，找到HKEY_CURRENT_US
ER\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer子键，将字符串NoDriveTypeAutoRun的值更改为“B5 00 00 00”即可(原为95 00 00 00)。
什么是RAID？
问：RAID是什么意思？具体解释一下好吗？
答：RAID的英文全称是Redundant Array of Inexpensive Disks——廉价磁盘冗余阵列。简单地讲，RAID可以把多块IDE硬盘或SCSI硬盘当做一块硬盘来管理和读存数据，这样做的好处是可提高磁盘系统的稳定性并且提速。除了这个功能外，RAID另外一个重要功能就是容错。当一个RAID系统中一块硬盘出现了故障后，并不会影响系统的正常使用，其它的硬盘会继续工作。常用的有下列几种工作方式：
RAID 0：没有为系统安全性考虑，不提供冗余功能，当有两个20GB硬盘时，提供40GB的总和容量，文件被分割成大小相同的两份分别保存在两块硬盘上，可提高硬盘I/O口数据交换速率。
RAID 1：磁盘镜像，即两块20GB硬盘组成后只可使用20GB，两块硬盘上数据一样，当一块损坏后，另一块仍可正常使用。
RAID 0+1：至少使用三块硬盘才能构成，提供奇偶校验冗余高安全性。比如3块硬盘组成RAID 0+1后，当数据写入时，会在随机分配的一块硬盘上产生校验冗余信息，另两块分别平均保存数据。
英文Windows2000如何支持中文？
问：因为工作需要，我使用的是英文Windows2000操作系统，所用软件95%为英文软件。考虑到要与其他人交流，最近准备装一些中文程序，有朋友推荐使用外挂中文系统来支持中文程序，我先后安装了《中文之星》、Richwin等多种中文平台，感觉使用后容易造成其它程序出错。请问Windows2000下有什么好的中文平台？除了中文平台，还有没有其它方法支持中文？
答：操作系统中文平台程序基本上由国内的软件开发商开发，在兼容性和稳定性方面还无法做到与操作系统完全吻合，因此会对一些程序产生不良影响。不过，就你的情况来看，你并不需使用中文平台，因为英文版的Windows2000本身就可以支持多种语言，你只需要按下面的方法进行设置就可以了。打开“Control Panel”，双击“Regional Options”组件，在“Language settings for the system”栏中选择“Simplified chinese”，单击“Set default”按钮，将简体中文设置为默认值，而后在弹出的对话窗中选择“ChinesePRC”，在“Your locale”栏中选择“ChinesePRC”，单击“OK”，重新启动计算机即可。
重现Windows98登录对话框
问：我在安装完Windows98后，系统第一次弹出登录对话框时没有输入任何口令就直接确认了，按照系统的提示这样以后就不会再出现登录对话框了。现在我想重新将登录对话框找回来，但找遍了所有的设置内容也没有找到。请问如何解决？
答：该问题可以通过修改注册表的方法来解决。运行注册表编辑器，找到HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Network\Real Mode Net子键，将字符串AutoLogon的值更改为01。如果没有该字符串，可以新键该字符串并赋值为01。重新启动计算机即会出现登录对话框。
如何恢复电脑的休眠功能？
问：我装的Windows2000 server ，其休眠功能常常莫名其妙地消失，而且我在重装Windows 2000 server时还是没有休眠功能，如何恢复休眠功能？
答：在控制面板中的“电源管理/休眠”项中启用休眠功能即可。笔者曾注意过这么一个现象，在Windows 2000 Server版中安装了终端服务之后，休眠功能即会消失，在安装过各种补丁后仍无法打开休眠功能，因为这些服务都要求服务器不能中途停止响应而取消了休眠功能，你所说“莫名其妙的消失”可能与你安装的服务程序有关。
浏览器为何有时不能登录网页？
问：我的电脑使用Windows2000 + SP2操作系统，网络浏览器为NetCaptor 6.20，最近经常突然出现问题，像输入www.sohu.com.cn等很普通的网址，电脑没有进行任何搜索操作，直接就显示出“找不到该页”的提示，遇到这种情况时关闭浏览器后再开启有时就能解决。我估计是中了黑客程序，用Norton Antivirus 2001检查却检查不出。于是使用手动查找的方法，发现在添加删除程序中，有一项“网络实名”不是我安装的，该项通过添加删除按钮还删除不了。如何清除？
答：导致你无法访问网页的原因应该不是“网络实名”程序，而可能是NetCaptor程序出错，该程序使用IE内核，也存在一些小问题(如脱机浏览标识不正确，有时访问网页出错等)，建议你安装最新的NetCaptor 6.50 RC1版。
网络实名也就是原来的3721中文网址录，开启该功能后，只要直接在“地址”栏输入中英文名字、拼音或拼音字头即可访问相应网站。如输入“新浪”或“Sina”就可以直接访问新浪网页，确实比较方便。当然，就像IE的Cookies一样，并不是每个用户都喜欢这种功能，你可以采用下面的方法关闭或卸载它。
打开控制面板，双击Internet选项，单击“高级”选项卡，去除“启用网络实名”项目前面复选框中的“√”号即可关闭网络实名功能。要想卸载网络实名，必须登录到3721的网站http://www.3721.com/setup-5.htm，而后按说明点击“完全卸载网络实名功能”按钮，重新启动计算机就可以了。
