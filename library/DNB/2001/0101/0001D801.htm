2000年网络工具软件大盘点

    随着2000年的走过，网络的应用已经越来越普及，上网人数继续高速增长，然而不可忽视的问题是中国网络的发展相对于发达国家而言，还是显得有些缓慢，硬件设施不够完善，网络带宽狭小，上网费用偏高，网络拥挤等。也正因为此，中国的上网一族对网络工具软件的挑选和使用上有着自己独特的见解。本文将对一些网络常用工具软件进行简介，希望能帮助各位网友从中挑出一两款中意的东东。

一、共享上网类

1．All Aboard!

软件大小：1179KB
目前版本：2.2 All Aboard!
    可以实现多用户共享一个网络连接，支持Modem、Cable Modem、DSL等多种上网方式。安装方便、功能齐全。

2．CProxy Server

软件大小：4358KB
目前版本：3.4.1 CProxy Server
    可以有效的阻隔局域网中的电脑直接连接到互联网上，提供局域网用户通过单点IP连结到互联网上的功能，也提供URL位置筛选等功能，支持的通讯协定以及Server种类更是多，包括了HTTP、Secure HTTP、NNTP、FTP、Telnet、SMTP、POP3、TCP/IP等，并且还具有mirror其他网站的功能。

3．Internet Gate

软件大小：809KB
目前版本：1.51
    让多台电脑共用一台Modem，ISDN，Cable Modem上网，自带防火墙，能有效避免不正当侵入。适用于中小型公司、企事业单位，在节约上网费用的同时又确保了网络的安全。

4．JProxyma

软件大小：1275KB
目前版本：3.0
    JProxyma可以让您家中的脑以及含有JINI技术的电器用品通过一条电话线上网。JProxyma的Proxy Server快取功能非常使用，它可以将局域网中的电脑所浏览过的网页储存在装有JProxyma的这台Server上，这样当其他电脑浏览到重复网页时便会先读取Cache中的网页资料，这样可以大大减小网络的负荷量以及连网的通讯费用。

5．SyGate

软件大小：3894KB
目前版本：4.0 Build
    套比WinGate设定更容易使用更方便的软件，SyGate是一套允许使用者在局域网中，通过一个国际互联网连接，分享给整个局域网的使用者，让两台以上的电脑，同时上网的软件，支持Modem、ISDN、Cable Modem上网方式。

6．WinGate

软件大小：3585KB
目前版本：4.1
    你只有一个Internet账号、一条电话线和一个调制解调器？你想让多个局域网用户都能用各自喜爱的WWW浏览器、Email、News、FTP、Telenet等等访Internet？WinGate能帮你办到。
