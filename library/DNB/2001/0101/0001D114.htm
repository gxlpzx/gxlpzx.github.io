Anfy JAVA ，使你成为JAVA制作高手

　　你想让你的主页具有与众不同的面貌吗？你想让你的主页具有不同凡响的视觉感受吗？那么就来使用Anfy java吧，让你梦想成真。 　　

    Anfy 1.4.3 －－ 包括 41个特殊效果的applet。可以在你的网页，桌面墙纸或屏幕保护上添加魔术般的效果，有菜单树、焰火、水面、湖面、横幅/幻灯片放映 、Anfy网络摄影机、文字卷动等效果。

    Anfy的安装很简单，完全傻瓜化。Anfy向导预设将此向导安装在"c:\Program File\AnfyTeam\Anfy"文件夹中。如果需要请更改安装路径，并完成安装。现在你已经在[开始]菜单中增加了Anfy向导的快捷方式并准备使用此程序了！ 　　

    当你首次执行Anfy向导，会显示一个类型菜单。向导将每一类图形效果展示为一系列菜单，你能够在Anfy向导的设定菜单中单击“下一步”或“上一步”按钮前进或后退。在每个菜单中，通过单击底部的"预览"按钮，你总可以在你所选择的浏览器上预览你正在创建的物件。反正她的操作极易上手。各种特效“分分钟钟搞定”。很容易上手，你所要做的就是点击鼠标而已。 　　

    笔者随便用Anfy java制作了一个特效，拿到友人处炫耀，友人那瞪大的眼睛着实让我得意了好几天。朋友，难到你不想试试吗？这么好的软件到哪里找去呢？下载地址是：http://www.anfyteam.com/ln/chisimp/ajdownl.html,在网站上作者为　Anfy提供了各种语言版本，有中文、繁体中文、英文、德文、日文等20多个版本，总会有一款适合你。她的中文版本大小为2917kb。与她提供的41个特效相比，其体积也算小了。朋友，还不赶快上网，download一个下来。品尝java咖啡的芬芳。 　徐国强
