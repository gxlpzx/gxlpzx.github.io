国产软件：叩击海外之门

<    br>仿佛是一夜之间，互联网公司风光尽失，就像当初争先恐后地找一顶.com的帽子戴上一样，现在又争先恐后地纷纷以一个技术公司自居。技术型公司趁势崛起，成为人们注意力的焦点。 软件公司作为技术型公司中的当然代表，在沉寂了很长时间之后，开始向世人展示自己的潜力与希望。从去年的市场情形来看，一款软件正版销量突破十万，已经不是什么特别了不起的事情，这在前两年却几乎是不敢想像的，由此我们可以判断，国内软件市场已经开始走向成熟。

<    br>真正令人兴奋的事情还不止于此——一向备受外国软件冲击与欺压的国产软件，终于可以昂首挺胸的走出国门，参与国际竞争了。“软件出口”成为去年软件业界的一大热门话题，金山、北大方正、中科院华建集团等国内著名软件厂商，纷纷在海外市场拓展方面创造佳绩。 这一天的到来似乎晚了些，但是我们毕竟迈出了宝贵的一步。

<    br>出口：中国软件人的夙愿 就我国的现实国情来看，软件产业的地位与作用尤其重要。道理很明显，软件产业主要以人才的脑力劳动作为基础，对资金与设备的要求相对较低，在高新技术领域中，软件行业是发展中国家最有可能在较短时间内获得跳跃式的飞速发展的突破口。

<    br>正是基于上述认识，早在1988年，北京软件行业协会就接受原国家科委的委托，进行专项课题研究，完成了“关于我国软件出口问题的研究报告”，在报告中得出了“大力发展软件出口、积极加入国际竞争与交换是加速发展我国计算机软件产业的根本出路”的结论，并提出了许多具体的政策建议。这一报告得到当时国家科委宋健主任的重视，并亲笔作了批示。

<    br>由于种种原因，这一正确的指导思想没有得到很好的贯彻，以致十几年后，当时与我国同处于一条起跑线上的印度，远远的把我们抛在了身后。以出口额来计算，相差将近四十倍。1999年，印度软件出口额达到40亿美元，是仅次于美国的世界第二大软件出口国，而刚刚过去的2000年，估计软件出口额将突破60亿美元。

<    br>往事难追，现实也令国人汗颜，但是，中国软件人始终也没有放弃进军海外的信念，也从没有停止过努力探索。

<    br>可喜的突破 中国软件界孜孜不倦的追求终于获得了回报。去年以来，各大软件公司纷纷在国际市场奏响了凯歌。请看： 中国最大的通用软件公司——金山公司去年在软件出口方面异军突起。先是WPS繁体版在香港、台湾上市，创造国产游戏销量记录的大型武侠RPG游戏《剑侠情缘Ⅱ》热销海峡两岸。如果说上述成功主要得益于大陆、香港、台湾三地在语言、文字、文化上的血脉同源。那么更令人振奋的则是金山词霸系列软件推出的第一个多语言版本——《金山词霸.net》中日英版成功登陆日本、美国，紧接着，《金山词霸》又将大规模出口美国、日本、加拿大、澳大利亚、新加坡、韩国、香港等十余个国家和地区。

<    br>北大方正不仅凭借出色的中文排版技术，占领了全球华文报业80％的市场份额，并自主研制开发出日文排版软件，成功进入日本印刷市场。 中国科学院华建集团向美国、澳大利亚、香港等地提供网络信息处理交流平台，仅去年上半年的软件出口合同总金额已达2．16亿元，出口额大幅飚升。

<    br>上述事例，都向我们传递着这样一条信息：中国软件出口，即将迎来春天，这对中国软件业来说，就像一声令人振奋的春雷。 两种成功模式：美国模式与印度模式 国产软件出口曙光初现，但这毕竟只是一线曙光，还远远不是一轮红日。在这个时候，更应该冷静的分析其他国家的成败得失，学习先进经验。

<    br>目前看来，国际软件产业的发展出现了两种比较成功的模式，一个是“印度模式”，另一个是“美国模式”。 印度模式的核心在于以市场需求来推动产业化的规模。印度是按照市场需求来组织企业生产经营活动，以承接项目的形式推动软件产业发展的。印度国内的软件市场狭小，所以它选择的是出口导向型的发展战略，几乎全部的软件开发、生产、经营活动都是围绕国际市场开展的，90％的销售额来自海外。 而美国模式的核心在于技术创新，以技术创新来推动市场的产生，从而带动产业的规模。美国软件业起步较早，像IBM、Microsoft、CA等一批世界顶尖级的大公司，依靠雄厚的技术实力和资金实力主导着世界软件产业的潮流，以自己的不断创新来创造新的市场需求。在美国，不仅软件巨头视创新为生命，即便是规模较小的软件公司也有一股“语不惊人死不休”的气概。 简单的评价这两种模式孰优孰劣是没有意义的，它们都是在特定的国情中形成的，但是，对于国内软件业来说，也都有值得借鉴的地方。因为，中国的整体发展水平要明显高于印度，国内市场空间大有潜力，在满足国内市场这方面来说，应该向美国学习，积极创新。而在开拓国际市场方面，毫无疑问，则应该努力学习印度的一些成功做法。 进军海外：一场持久而意义深远的战役 软件出口，对于国产软件的长远发展来说，意义非同寻常，甚至可以说是中国软件产业获得大发展的根本出路。对此，业界人士都有着非常清醒的认识。 金山公司的总经理雷军说过：“我们应该把拓展海外市场当做一项根本战略，这对于整个中国软件产业来说，意义是非常深远的。海外的软件市场空间非常广阔，拓展海外市场，能够为软件企业带来现实利润。更重要的是，通过出口，我们可以广泛吸取国外发展软件的经验，在技术上、运作上以及意识上实现与国际先进水平的接轨，这对于我们的长远发展无疑是至关重要的。此外，中国即将加入WTO，这对于软件业来说，既是挑战，也是机遇。留在国内打防御战，只能是被动挨打，所以，我们必须也要攻出去，在国际竞争中，走向成熟。”

<    br>谈到国产软件出口的前景，雷军充满了自信：“我对于中国软件产业也充满信心，软件的竞争说到底是人类智力的竞争，中国人的聪明才智是举世闻名的，我们拥有大量高素质的人才，中国理应成为软件大国。” 美国人在其《美国国家关键技术报告》中写道：“软件本身对大多数其它国家关键技术起着必不可少的支撑作用，先进软件的发展在新一代军事与商业系统的推广及可靠性方面越来越成为一项重要的制约因素。” 把软件摆在这样的战略高度，是丝毫都不夸张的，因此，国产软件的振兴愈发显得任重而道远。 在我们踏入新世纪之际，在中国软件业的将士们，向着国门之外大踏步的出征之际，让我们真诚的祝愿他们一路走好！ (陈啸锐)
