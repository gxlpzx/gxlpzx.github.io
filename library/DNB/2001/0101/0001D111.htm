让你的网页也JAVA一回

    静态的网页已经吸引不了网民的目光，有互动性以及动态的网页才能留住网民的脚步。因此一些能让网页产生动感以及互动性的技术就流行了起来。其中JAVA语法就是方便又具有跨平台性的热门技术。这次就让我们学习一点怎样在网页中插入一些JAVA部件，让我们的网页效果更丰富。

    目前最常见的有JAVA SCRIPT和JAVA APPLET，JAVA SCRIPT是内嵌于Hmtl文件中的代码，程序码可以在Html文件中看到，而JAVA APPLET是独立于Html文件之外的，只有参数可以在Html文件中看到，执行完后才将结果显示为整个网页的一部份。所以使用JAVA SCRIPT语言时只要复制整段语言然后贴入Html文件中就可以看到效果，而使用JAVA APPLET时，还要同时上传类别文件。以下我们就介绍如何利用现成的网络JAVA SCRIPT和JAVA APPLET来使你的网页更生动。

    双击该html文件就能直接用你的浏览器来打开，打开后你会发现内容都是空白的，因为我们还没有输入任何文字。 如果你想要继续编辑这个网页，只要再把原先的html文件类型改成txt，就又可以直接在文字编辑器中编辑了。这样把文件类型改来改去是有点麻烦，其实只要在浏览器中浏览该网页时，直接在画面中按鼠标右键点选“查看源文件”，就可以马上跳出一个记事本，你可以直接编辑其中的源代码，编辑完后按[Alt]+F+S，再按浏览器的“刷新”键，就会看到新修改的网页内容。

    使用JAVA SCRIPT 有了已经建好的网页后，你就可以找一些JAVA SCRIPT的程序范例，目前提供JAVA SCRIPT程序范例下载的网站非常多，有的网站会直接公开其程序码让你直接选取复制，有的则需要你先查看网页的原始码并自行把程序码复制到你自己的Html文件中。例如在以下网页中有一个简单的月历JAVA SCRIPT程序范例，我们可先到 http://nii.org/calendar-with-greeting.html，你会看到一个用Java Script写的月历。 依步骤把这个月历加入到自己的网页中：

1.    在网页中按鼠标右键点选“查看源文件”。

2.    在跳出的记事本中会给出这个网页的源码。把从 之间的语法全部选取并按下“编辑”“复制”。

3.    打开你刚做好的那个网页文件，将鼠标游标移到在和和标签之间按下Ctrl+V。

4.    将该网页文件存盘。 这样就完成了JAVA SCRIPT程序码的转移，可以双击你新改完的网页文件，用浏览器查看你的成果，你现在可以看到一个月历就在在你的网页中。JAVA SCRIPT程序语言的起始就是 ，所以一定要完整的拷贝从起始到结尾之间的文字，不然就可能产生程序执行错误。 使用JAVA APPLET JAVA APPLET范例在使用上要比JAVA SCRIPT麻烦一点，除了要在网页源码中贴入一些设定参数外，还要下载该范例本身的JAVA APPLET文件。我们以要使用一个同心圆文字的特效为例，先链到 http://www.netvigator.com.tw/~jeremys/animcer.htm，你可以看到一个同心圆文字的效果，再依以下步骤就可以把该范例用在你自己的网页中：

1.    按“下载这个JavaApplet”的链结以下载animcer.zip，这个文件中包含了这个范例所需要的class文件。

2.    选取从标签之间按鼠标右键选“粘贴”。

3 .    将该网页文件存盘。

4.    把animcer.zip解压缩，将其中的AnimCercles.class文件放在你网页文件所在的目录下，(例如，如果你的网页文件是在桌面上，你就要把这个AnimCercles.class也放在桌面上)，这样才能执行。 如果你要把网页上传到网站的主机上，记得一定要把这个AnimCercles.class也一起上传到同一目录下，这样才能保证网页的正常运行。要是下载的压缩文件中有另一个html文件，通常是这个JAVA APPLET的说明以及参数设定，这个文件仅供你参考，不需要一起上传到主机中。除了这个html文件外，如果压缩文件中包含其他文件(如图图形文件)，都要一起上传到同一目录下，这样当你双击你的网页文件后，就可以在浏览器中看到执行后所出现的同心圆文字效果。

    如果你要修改这个同心圆的展示效果，只要再度打开网页的源码文件，并依照下载网页中的说明，修改参数，就可以有不同的效果，例如你想要修改显示的文字内容，只要把其中的“Right NOW!”改成你想要的文字就可以了。 网上有很多像这样提供JAVA范例下载的网站，你现在就可以开始大显身手，把你的网页装点的更生动。 不过要注意的是，JAVA语言是在浏览者的电脑上执行的语言，所以如果你同时在一个网页中插入太多的JAVA效果，不但会大量消耗浏览者的系统资源，也会减慢浏览速度，所以一定要注意适量。

    另一点要注意的是，JAVA范例也是有著作权的，如果你下载的网页有特别说明不准任意复制或是要经过作者同意，就要特别注意，以免违反著作权。
