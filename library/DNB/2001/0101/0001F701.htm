专家坐堂

声卡的全双工与半双工

问：如何判断一声卡是全双工还是半双工？是不是在超级解霸下不能唱卡啦OK的声卡就是半双工的？还有intel信息需要时间倒退3－4年以上才能从主流市场上买到。Bonny Guo 810主板集成的声卡是不是都是半双工的？

答：全双工是指能双向工作，即输出的同时能够输入。如果你唱卡啦OK的同时可以录音，你的声卡就是全双工的。半双工声卡几年前就退出市场，目前买不到半双工声卡，intel 810主板集成的声卡一定是全双工的，不过我没试过。

学习3D MAX是否对CPU、显卡要求很高？

答：就学习3D MAX而言，不需要配置很高的电脑，笔者曾在配置为VX主板、K7-75 CPU、32MB内存、S3765显示卡的最低档586级电脑上使用过3D MAX 2.5，能够完成绘制3D图形、贴图、着色、配音、动作设计、设置灯光和设置摄像机等工作，并能较好地进行一般的渲染制作，例如渲染制作单幅面3D图形。但在观看3D连续动作场面时有声音延时及脱色现象，渲染制作3D动画速度很慢，例如在渲染制作一幅50帧画面的简单3D动画时，居然用了4个小时，你可以在低档的电脑上学习、制作3D动画，在高档电脑上进行渲染制作成品。

可否从硬盘安装Windows 问：我有一台486电脑，没有光驱，请问怎样往硬盘里装Windows 3.2或95？可否将所有的Windows 文件装到硬盘里，然后执行安装程序？

答：可以呀！我从来是将Windows 的所有安装文件COPY至硬盘的D分区(或其它分区)上，从硬盘上安装Windows 。将Windows 的所有安装文件COPY至硬盘上，从硬盘上安装Windows 好处多多，例如，添加Windows 应用程序、安装新硬件需要插入Windows 安装光盘时，系统能自动去硬盘上的Windows 安装目录中查找所需要的文件，省去了插入Windows 安装光盘的麻烦，以后重新安装Windows 也很方便。如果你的硬盘足够大，将你需要的所有硬件驱动程序(如显示卡、声卡、MODEM)全部COPY至硬盘的D分区，以后安装这些硬件就不再需要插入相应的驱动程序盘，即可以节省时间，又能延长驱动器的寿命。 怎样升级不知名主板的BIOS

问：很多读者来函问道，由于无法知道主板的具体型号，无法升级主板的BIOS。

升级BIOS能够提升主板的性能，改善主板的兼容性，使主板适应于各种新型硬件。但在升级BIOS时，需要弄清BIOS是哪个厂家提供的，使用了错误的升级文件，有可能导致电脑无法启动。在国内大多数硬件网站(如http://www.newhardware.com.cn/)上，有一个称为CTBIOS 的实用程序，它能识别出大多数主板,检测出主板使用的芯片组、BIOS ID、BIOS版本号、BIOS生产日期、主板生产商和BIOS下载网址，帮你解决升级主板BIOS这一难题。CTBIOS的使用很简单，只需在资源管理器中双击CTBIOS图标即可运行(也可在DOS下运行)。 无法启用Windows me的系统还原功能

问：我的机器是组装的，主板是MSI-6153，CPU是P3 600EB，原来使用的是Windows 98SE，前一星期升级为Windows me，但是无法使用其中的系统还原功能，我只要一启用系统还原，计算机就告诉我系统还原在重启计算机后才能使用，但重启后情况还是一样，同样的Windows me在我办公室的计算机中系统还原使用很正常，在别人的机器上安装同一Windows me也很正常。请问这是什么原因造成的，如何解决？

答：安装Windows me过程中，系统曾给出过提示界面，要求选择是否保留原有系统以备还原，如果你选择了不保留原有系统安装Windows me，则无法使用系统还原功能。 为何只能以29帧/秒播放视频文件

问：我的计算机配置为技嘉6VX7-4X主板、赛扬533 CPU、耕宇TNT2 M64 16MB显卡、希捷二代20.4GB硬盘， 128MB内存、索尼24倍速光驱。如此高配置的计算机在用XING 3.3版播放其自带的测试程序时，只有29帧/秒；用Windows 98 媒体播放器播放时也一样。请赐教。

答：NTSC制式规定播放速度为30帧/秒；PAL制式规定播放速度为25帧/秒；不管你用多好的设备和如何高档的播放程序，播放NTSC制式的文件或影碟速度均不会超过标准规定的播放速度30帧/秒。 总是出现“Windows 在造访注册表时出现错误”对话框

问：单位的“老机”换了一块CPU(6X86MX 300)，原来只能工作在233MHz，经设置跳线使工作于266 MHz。但开机进入Windows 98时，总是出现“Windows 在造访注册表时出现错误”的对话框，我反复好几次使用Windows SPEED或者SCANREG来恢复注册表才能让电脑正常工作。不知这是为什么，请尽快指点迷津。是不是因为跳线上266这个原因而使得电脑的稳定性下降呢?

答：6X86MX 300标称工作频率为233MHz(3.5×66 MHz)或225MHz(3X75 MHz)，使其工作于266 MHz属超频使用。而6X86系列CPU超频性能不好，会出现你所述的现象，建议还原设置，使其正常工作于233MHz。 何处有ALG2064显卡驱动程序

问：一台486电脑，Windows 32操作系统，显卡是ALG 2064.在Windows 32的情况下，在我手里的2064驱动程序中，没有SETUP这个文件，我从哪里才能得到该显卡的驱动程序？ 答：在电脑报配套光盘中，提供有大量的硬件驱动程序，你要的ALG 2064显卡驱动程序，在97年出版的电脑报第三期中就有。 如果设置主板外频跳线 问：技嘉586tx3主板，如何才能将外频跳至75MHz和83MHz。

答：不同主板跳线的设置方法不尽相同，要设置主板外频跳线，请参见主板说明书进行，如果你没有主板使用说明书，请去“http://www.gigabyte.com.cn/home/technology.htm”下载主板使用说明的文件，不过有些主板使用说明文件未提供非标准外频(如83MHz)跳线的说明，主板是否提供有83MHz的隐藏跳线只好由你自己根据主板上的外频跳线设置情况组合试试。 恢复光驱的自动读盘功能

问：我的电脑原先放光盘时会自动读盘，近几天安装了一个优化内存的软件，不知道为什么就不会自动读盘了，请问有什么方法可以解决？

答：估计是优化内存软件减少了光驱的缓存容量，请选择“控制面板/系统”打开系统属性窗口，在“性能”选项卡的“高级设置“栏目”中按“文件系统”按钮，打开“文件系统” 属性窗口，在“CD－ROM”选项卡中，将“追加的高速缓存大小”滑块移至最大，在“最佳访问方式”下拉选择框中选择“四倍速或更高”试试。 如果不能解决问题，请选择“控制面板/系统”打开系统属性窗口，在“设备管理器”选项卡的设备表中选择你的光驱，后按“属性”按钮，打开光驱属性窗口，在“设置”选项卡中的“自动插入通知”前打钩。 没有驱动程序如果重新安装Windows 98

问：我想重新安装Windows 98，可是我所有硬件的驱动程序都没有。这还能进行重装吗?如果能的话，我该怎么做呢?是不是重新安装Windows 98以后，原先已经安装的软件也都要重新安装?

答：复盖安装Windows 98不需要重新安装原有硬件的驱动程序，原先已经安装了的软件大多也不需要重新安装。格式化硬盘后安装Windows 98须重新安装所有的硬件驱动程序和应用程序软件。 为何CPU工作频率与CMOS设置不符

问：我的电脑是tcl718A，sis620的主板，64MB的内存(pc100)，最近用一根128MB PC 100的HY内存条替换原来的内存条后，开机自检,系统显示CPU工作于450MHz，但在BIOS设置界面中，BIOS&CPU FEATURES SETUP 一项中的CPU INTERNAL CORE SPEED 显示却是350MHz。CPU CORE:BUS FREQ.MULTIPLE 是3.5×。不知是什么原因，望在百忙中予以解答。

答：有些主板的BIOS发现系统更新了硬件以后，会自动将CMOS参数自动降低为最安全的设置。对支持CPU工作状态设置的BIOS程序，会自动将CPU工作频率降为主板支持的最低主频CPU。你的系统外频为100 MHz，而100 MHz外频CPU的最低主频为350MHz，因而BIOS设置界面中应当显示350MHz(3.5×100 MHz)，属正常现象。哪么为什么开机自检后系统显示CPU工作于450MHz(4.5×100 MHz)，因为你的CPU被锁定于4.5×，也属正常现象。 为何Windows 2000中无法安装内置猫的驱动程序

问：我的联想射雕56kII型内置猫在Windows 98下跑得很好。但在Windows 2000下只能找到它，无法安装驱动程序。请问怎么办？

答：在Windows 2000中不能使用Windows 98的驱动程序，请安装Windows 2000的驱动程序，也可以试试使用Windows NT的驱动程序。
