十款流行MP3播放器点评

　　MP3是当今最流行的一种数字音乐格式，它的压缩比高，而且音质又不错，因此，便携式MP3播放器就成了许多厂商必争之地。MP3播放器在市场上的品种变得非常丰富，下面就向大家推荐八款当今流行热销的MP3播放器。

一、帝盟Rio 600播放器

　　 Diamond(帝盟)在相继推出Rio-300和Rio-500的mp3播放器后，2000年7月份，又推出新款Rio-600 mp3播放器。Rio-600是第一台可随心换外壳的mp3播放器，有蓝、紫、橙、黑、白、绿等多种颜色外壳可以选择，使其非常有特色。Rio 600外形设计及材质选择也非常独特，采用了流畅的弧线形设计，机身采用塑料制成，外表做成磨砂效果，颇具时尚魅力。体积大小为：99mm x 63mm x 27mm，重量(无电池)： 85克。 　　

　　 Rio 600内建32MB内存，最高可支持320MB的存储容量，可以购买闪存Flash Memory或IBM Microdrive微型硬盘来扩展内存。但不能直接增加SM或CF存储卡。配备USB接口，传输速度非常快。一节1.5V AA 碱性电池最多可支持11.5小时连续播放。Rio 600支持的音乐格式非常丰富，除支持MP3及WMA音乐格式外，还支持AAC和InterTrust技术格式，这样可以支持未来更多的数字音乐格式。 总评：音质优美，外形独特，色彩多样，支持音乐格式多，性价比高，存储容量大，但不能直接支持SM或CF存储卡。 价格：￥1680 - 1890

二、三星yepp-E64播放器

　　 三星公司的MP3播放器是目前市场上是畅销的产品之一，三星yepp YP-E64系列是其主流产品。yepp YP-E64播放器采用铝质前壳，机身小巧，大小：65mm x 87mm x 17.2mm，重量：75克。YeppYP-E64除内置64MB闪存外，也备有存储卡插槽作扩充之用。中文LCD屏幕信息显示，通过USB与计算机连接，传输速度较快。 　　

　　 yepp YP-E64的音质效果也很不错，但是由于它无法调节高音和低音，用户只能接受它的默认设置。三星Yepp YP-E64除了播放MP3音乐外，也同样具有录音功能，效果不错，其声音的还原效果非常理想。此外，三星Yepp YP-E64还提供了非常实用的电话簿功能，为用户提供多方位的服务。此系列还有32M/96M内置闪存的产品供你选择。

　　 总评：体积小，提供电话簿和录音功能，音质好，无法调节高音和低音。价格偏高。

价格：
YP-E64：￥1850 ~ 2180； 　　　
YP-E32：￥1420 ~ 1680； 　　　
YP-E96：￥2500~2800

　　 三、创新NOMAD Ⅱ播放器

　　 NOMAD Ⅱ MP3 播放器专为追求时尚的消费者而设计。其外形充满“曲线美”，多数功能可以通过LCD屏幕上的菜单来选择。Nomad Ⅱ没有内置的闪存，而是采用SmartMedia卡作为存储介质，目前有两款储存卡，一款是32MB，另一款是64MB。NOMAD II通过USB与PC机连结，传输速度很快。NOMAD II配有一个可随身携带有线遥控器，即使是在运动中也可以轻松地控制播放，很具便携性。 　　

　　 Nomad II支持多种音乐格式，可通过对核心芯片写入新的程序，使之不断支持最新的流行数码音频格式，如：MP3、WMA等音乐格式。此外，NOMAD II还允许用户通过下载特别的软件，来支持未来的数码版权管理技术，从而使NOMAD II能够播放合法的数码音乐格式。　　

　　 同NOMAD I一样，NOMAD II具有FM收音机和语音录制功能。 总评：机身小巧(体积： 65 x 93 x 21mm，重量： 88克(不包括电池))，音质优美，支持音乐格式丰富，随身携带有线遥控器更具便携性。 价格：￥2350(参考价)

　　 四、最薄的MP3播放器JNC SSF-882 　　

　　 JNC SSF-882是目前世界上最薄的MP3播放器，它外观小巧，厚度仅8.7毫米,整机外形宛如一只名片夹。大小(cm)：(56×86×8.7)，重57克，非常便于携带。彩色金属外壳有红、蓝、银灰多种颜色可选。SSF-882内置64MB/96MB存储器，大LCD显示屏，采用USB接口方式,传输更快速。内置高容量锂电充电电池，只需要1小时左右便能充满，可使用时间却长达8小时以上。为了更省电，SSF-882设置了自动休眠功能，只要3分钟内没有使用便会显示SLEEP，如果再过3分钟仍没有使用此机，它会自动关机。JNC SSF-882软件管理功能也非常丰富：文件下载、上传、转移、删除、全部删除、快速格式化、完全格式化、更新等等。另，SSF-882 MP3具有2小时左右的数码录音功能。

　　 总评：外观小巧，音质好，设有自动休眠及自动关机功能，软件管理功能丰富，电池使用时间长。

价格：
SSF-882(64M)：￥2180 －2250

五、方正太空MP3播放器

　　 国产品牌的播放器在市场上也占有重要的一席之地。方正就是其中之一。尤其是方正太空系列随身听，人气值颇高。太空播放器机身呈银色，(大小：90mm x 66mm x 18mm，重量：75克)稍带清冷的风格，比较适合男士使用。有个大的LCD显示屏，不过只支持英文显示，按键丰富，设置方便。内置32M内存，配有MMC外置存储卡插槽，以便扩充之用。采用并口与计算机连接，传输速度较慢，这是它的不足之处。 　　

　　 太空随身听的音效不错，播放MP3时均衡模式有六种，其中有三种模式模拟3D音场效果。太空另一优点就是附带的耳塞，在所有播放器中太空所配的耳塞算是比较好的。太空播放器还具有一大特点是它有A-B重复播放模式，可在随意设置的两点之间反复的播放。太空除具有播放MP3音乐功能外，也具有录音功能。

　　 总评：具有A-B重复播放模式，音质好，并口传输使其传输速度较慢。

　　价格： ￥1,650.00－ 1,980.00

六、联想MR-2 MP3播放器

　作为国内知名产商的联想，自然也不会错过MP3播放器，在此向大家推荐联想的MR-2(大眼睛)播放器。大眼睛因其有一个超大型LCD屏幕而得名，它能实现菜单式显示，使你轻而易举的实现智能化的操作，在播放过程中还有"贪吃蛇"的动画效果。大眼睛身形小巧，大小：82mm x 60mm x 17mm，重量：70克 ，便于携带。内置32MB内存，并且预留了一个SmartMedia卡插槽，你可以方便的扩充内存。MR-2在工作过程中非常节能，一节五号AA电池可以连续工作十二小时以上。由于大眼睛采用的是并口传输，在传输速度比较慢，这一点很难令人满意。 　　

　大眼睛音质好，除了播放MP3音乐外，还具有录音功能，录音时间是长达130分钟，并且可以通过随身听的PC连接线把录下的声音文件上传到PC机。

　　 总评：轻便小巧，屏幕大，能实现菜单式显示，音质好，录音时间长，较节能，并口传输使其传输速度较慢。

　　 价格：￥2130

　　 八、高智能的Mpman MP-F35播放器

　　 作为世界上第一家研制出MP3机的老牌厂商世韩，具有得天独厚的创新优势，它为MP3机设计出丰富多彩的功能！MP-F35在外形上没有多大的特别，仍采用金属外壳，大小：57.5mm x 83.6mm x 17.2mm 重量：74克，LCD显示屏。MP-F35采用USB接口与计算机相连，传输速度快。内置64MB闪存，最高可扩充至128MB，扩充时可选用三种不同容量的Smart Media卡(16/32/64MB)。一节AA1.5V碱性电池可使用10小时。 　　

　　 MP-F35的特点主要表现在它的功能上，Mpman MP-F35算是一款高智能的播放器。它支持Windows95(OSR2.1)/98和2000的操作系统，使用线控耳机收听音乐，除了播放MP3音乐外，其录音功能很强大，录制声音文件可达4小时。还有调频收音机功能；电话簿与备忘录的功能；使用多种语言文字显示ID3-TAG信息功能；电话号码/备忘录功能以及删除按键。

　　 总评：高智能播放器，功能强大，机身小巧。

　　 价格：￥2400

　　 MP3播放器选购要点

1、外观：外形不用多说了，只要自己喜欢就好。由于随身携带，重量当然要轻，最好不要超过100克(电池除外)。体积在100mm*60mm*20mm以内为好。另外按钮的分布要合理，用起来顺手最重要。

2、容量：至少要32MB，16MB的机子最好别选择。因为一首四分钟歌曲的容量大概是3.8MB，32MB也只能存8首歌曲。当然以后也可以考虑买闪存来增加容量。

3、显示屏：显示屏当然越大越好，还要注意显示屏带不带夜光。还要看它支不支持中文显示，如果不支持，显示的中文歌名会变成乱码。

4、格式：MP3播放器除了可以播放MP3外，最好还支持其它的音频格式。如微软的WMA格式，(一首WMA格式的歌曲的容量大小只有MP3的一半，但音质相差并不大)。还有它如果支持文件格式的话，就可以当移动存储器用了。

5、功能：有些MP3随身听带有录音，调频收音机、记事薄等功能，你可以根据自己的需要选择合适的机型。

6、接口：MP3随身听最好是选择USB接口的，它的传输速度比并口快很多，而且可以支持即插即用。

7、其它：最后要注意一下随机附带的配件和软件，当然最重要的是看看格价能否接受。
