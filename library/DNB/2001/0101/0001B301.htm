WPS 2001抢鲜报道

    1999年金山公司发布的WPS 2000中文办公系统受到过众多用户的好评，经过一年多的努力， 在新世纪之初，WPS 2001 智能集成办公信息处理系统即将问世。我们以最快的方式获取了该产品的主要信息，在此以飨读者。

    在原有基础上，WPS 2001进行了全面的改造和扩展。作为一款运行可在中文Windows 98/2000环境下的纯 32 位软件，它功能丰富、使用简便、性能稳定，适应当今无纸化、网络化的电子办公模式，在中文编辑方面更适用于国内用户进行单位办公或家庭信息处理。与 WPS 2000 相比，它的改进主要体现如下：

一、全新的用户界面 WPS 2001 对操作界面作了较大改进，表现为单一界面下的集成办公环境。新增多个功能模块，并保留了 WPS 2000 的原有功能。

二、六大功能集成

●文字处理与图文排版 WPS 2001 在WPS 2000 已有文字处理功能基础上增加了格式刷、自动编号等技术，另外可在任意位置灌入WPS文件。它兼有多种办公软件的功能，解决了文件格式兼容性问题。
●电子表格 全新的电子表格面向需要进行大量数据计算和处理的企业用户。不但支持立体表格，还可进行90多种常用函数计算，以用于数学模拟和财务计算。它还有丰富的统计图表，能以控件方式提供ET表。
●电子邮件 WPS 2001编辑的文档可以直接通过电子邮件进行传送，所有收到的电子邮件都可以在WPS中进行编辑、存盘等操作。同时内置电子邮件接收、管理功能，可进行邮件加密发送、远程邮箱管理等操作。
●多媒体演示制作 专业化的多媒体演示文档制作系统支持演示模板和大纲模式，具有整页插入、删除、拖动以及动画效果即时预览的功能，背景和外观切换灵活。并且提供了丰富的素材仓库。
●网页浏览 提供基本的网页浏览功能。
●图片浏览 提供基本的图片浏览功能。支持常见的图像格式。

三、四大兼容

●兼容 DOC 文档：不用通过中间转换，直接读取 DOC 文件。包括 Word 95/97/2000。 ●兼容 HTML 文档：能读写 HTML 文件。
●兼容 Excel 文档：电子表格能直接读取 Excel 文件。
●兼容 PPT 文档：多媒体演示模块能直接读取 PowerPoint 文件。

四、公文、表单流转

    WPS 2001 提供电子邮件方式的公文流转和表单流转。这一机制可广泛应用于诸如答卷、填表、问卷表、申报表、月/日/年度报表等交互式文档表单的制作。

五、信息库

    利用 WPS 2001 的跳转、链接功能，可组织一个企业内部信息库。现在的WPS 2001 作为一个信息处理平台集成办公信息处理系统的特色， 多种工具条和丰富的图文符号是各类技术人员制作专业资料的强劲工具。随着网络的流行，WPS 2001 更是与 Internet 紧密相连，用户不但可从网上得到升级服务，还能随时了解WPS产品最新动态。
