迎接新生命 绘上生命的色彩

　　一、Win98的安装

　　 在硬盘分区和格式化处理之后就可以进行Win98的安装了，所需时间根据具体电脑性能而定，一般在25至50分钟左右，安装的具体步骤如下： 第一步，使用Win98启动软盘引导系统，选择“支持光驱方式”进入DOS状态。 第二步，将存储有中文Win98安装程序的光盘放入光驱(假设盘符为E：)，依次键入“e:”、“cd win98”和“setup”命令启动Win98安装程序，屏幕上显示出一段关于Win98的介绍和欢迎页面，按回车确认“继续”后安装程序开始对硬盘进行扫描，检查硬盘有无问题。 第三步，检测正常后，点击“继续”按钮，安装程序会开始调入安装向导程序(如图4-2)，然后提示用户确定安装路径和目录(安装目录一般为缺省值C:\Windows)。确定安装目录之后，安装程序在完成“准备目录”、“检查硬盘空间”等工作后，会提示创建“启动软盘”，准备一张空软盘，看见提示后放入软驱，电脑会制作一张启动盘用于系统的引导和修复。如果已有启动盘，可选择图中“取消”。 第四步，在完成启动盘制作后，此后屏幕提示用户选择安装方式。安装方式有“典型”、“便携式”、“袖珍”和“自定义”四种，其中“自定义”可由用户自己选择安装Win98中哪些内容，而“典型”方式则是会安装一般用户的常用程序，因此多数用户一般可选择“典型”方式进行安装。选择“典型”后点击“下一步”后，屏幕提示要求用户决定安装Win98中部分程序组件的方式，一般均选择“推荐”方式。 第五步，点击“下一步”屏幕提示用户输入“计算机名”和“工作组名”，这两个名称主要用于网络上计算机的身份识别。 第六步，单击“下一步”后屏幕相继显示计算机所在地选择，一般选择缺省值“中国”，单击“下一步”，在提示“开始复制文件”时再单击“下一步”，屏幕显示，安装程序开始从光盘中向硬盘拷贝文件(如图4-3)。 第七步，在安装程序在拷贝文件结束后安装程序会重新启动系统，并准备开始为本机硬件安装有关程序。此时安装程序会提示提示用户输入姓名和所在单位,输入后点击“下一步”后，安装程序显示有关软件使用协议(如图4-4)，提示用户是否同意该协议(如果不同意就不能继续Win98)，选择“同意”后继续。 第八步，安装程序提示输入产品密码，(如图4-5)在输入正确产品密码后(密码一般见Win98产品说明书封面)，点击“下一步”，Win98会验证密码的真确性。部分版本的Win98提示输入密码的时间上会有不同。 第九步，如果密码输入正确，安装程序将继续自动设置系统硬件，只有在提示用户选择时区时(见图4-6)需用户干预一下，然后又可以自动安装至全部结束为止,此后系统再次重新启动。 第十步，系统重新启动之后，屏幕提示用户输入登录密码，此时用户可自行决定是否输入密码，如果此时不输入密码，以后再启动时通常不再要求用户输入密码。点击确定，Win98安装初步完成。 一般情况Win98在安装过程中可以正确检测到各种硬件设备，并安装其对应的驱动程序。单对于一些Win998无法识别的较新设备，必须由人工添加驱动程序，否则硬件将不能正常工作。

　　 二、部分设备的驱动程序安装

　　 1．为VIA的主板安装4合1补丁 由于Windows98对VIA公司芯片组的支持不太好，不能为其安装正确的驱动程序，所以必须由用户自己安装VIA公司单独提供的驱动补丁，否则计算机不能正常使用。 第一步，启动电脑进入Win98，运行补丁安装程序。在主菜单中选择自己的主板芯片组类型后回车确认，屏幕显示准备安装的程序选项(见图4-7)。 第二步，在第二级菜单中选择自己准备进行的内容，如安装4IN1补丁还是安装其它的工具软件等，此处选择“VIA Service Pack 4 IN 1 Driver”后回车确认，开始安装并显示4IN1补丁程序的安装菜单(见图4-8中“3”，图4-8是4IN1安装程序的5个提示画面，为节约篇幅笔者将其合并在一幅图中)。 第三步，在图4-8中选择准备处理的程序内容，以及4IN1程序中具体项目的安装或卸除。 第四步，在项目选择确定之后开始安装，在安装结束时提示用户结束安装后是否需要立即重新启动系统使设置生效，此处点击提示中“Finish”选择立即重启。 第五步，在系统重启时Win98将不断检测到新硬件并为其安装相应的驱动程序，然后进入Win98桌面。

　　 2．显卡驱动程序安装 由于有些显卡Win98不能识别，因此要由用户自己安装驱动程序。此处介绍一下典型的安装步骤： 第一步，启动系统进入Win98桌面，将所用显卡的驱动程序光盘放入光驱，运行驱动安装程序，安装完毕后按提示重新启动系统。 第二步，如果Win98重新启动后能够检测到显卡，那么将会自动进行安装或者提示用户指引驱动程序所在的目录，选择正确地驱动目录，安装程序就会继续进行。 第三步，如果驱动程序安装正确，则显卡工作正常；如果显卡仍不能正常工作，那么可以打开“控制面板/系统”选择“设备管理”菜单作进一步进行检查，由于这部分稍微复杂，这里就不详细介绍。

　　 3．声卡驱动程序安装 第一步，当Win98基本安装成功并重新启动时，可能检测到声卡，此时屏幕上提示“发现新硬件”，继续选择“搜索设备的最新驱动程序”后单击“下一步”后将声卡的驱动程序光盘放入光驱，并为Win98指引驱动程序路径。 第二步，通过“浏览”指引到光盘上驱动程序所在的目录，然后系统找到驱动程序并开始安装(如图4-9)。 第三步，当Win98将声卡所有的驱动程序都安装完成后，会提示重新启动电脑。

　　4．打印机驱动和应用程序 打印机同样需要安装驱动程序。一般可通过“开始”、“设置”、“打印机”菜单(见图4-10)，然后单击其中的“添加新打印机”图标进行安装。 当我们依次单击显示的“安装向导”中“下一步”并选择安装“本地打印机”后，Win98列出可供用户选择的驱动程序(见图4-11)，如果用户有自备的驱动程序，那么选择图中的“从软盘安装”，如果没有则通过Win98提供的打印机清单选择，根据打印机型号选择驱动后，点击“下一步”，有时会提示插入Win98安装盘，拷贝部分文件后提示用户是否打印样张(可根据自己需要选择)，然后完成安装，在打印机菜单中增加了一个以打印机型号命名的打印机图标(见图4-12)。这样用户就可以使用打印机了。

　　 三、显示设置

　　 显示器连接无误、显示卡驱动安装正确以后，我们还必须进行另一项工作才能方便使用显示器。 我们在显示器上看到的每一幅图像、每一篇文字都是由无数个点(象素)构成的。简单点谈，分辨率就是代表这些点的密集程度；色深代表这些点的色彩逼真度；刷新率代表每秒中屏幕的更新速度。我们要正确使用显示器，就必须对在Windows下对显示分辨率、色深、刷新率进行设置才能真正完成显示的安装设置。

　　 第一步，将鼠标停在桌面上，按鼠标右键，选择“属性”，再点击“设置”(这时弹出的窗口我们简称“显示设置”窗口，如图5-1)。在这个窗口中有不少重要的显示信息和设置选项。如果你的显示器和显示卡的驱动都安装正确，那么你可以看到“显示：某某显示器(在某某显示卡上)”这条信息。如果显示器驱动未安装的话，则会看到“显示：无法识别的监视器(在某某显示卡上)”。有些显示器用户可能没有驱动程序，这时用户可以点击“显示设置”窗口的“高级”按钮，这是弹出显卡属性窗口，点选“监视器”，可以看到“无法识别的监视器”旁有一个“更改 (C)...”按钮。点击，出现“选择设备”窗口(如图5-2)。该窗口左面显示生产商名称，右边显示监视器型号。如果用户清楚自己的显示器型号可以尝试在这个窗口安装WIN98自带的驱动程序。但可能更多用户并不清楚，这时我们可以根据显示器的大小为显示器安装WIN98承认的通用标准显示器。在“生产商”下选择“标准监视器类型”，在“型号”下根据显示器大小选择合适的型号：一般14寸的显示器设置成“Super VGA800×600”；一般15寸可设置成“Super VGA1024×768”；17寸则一般可以设为“Super VGA1280×1024”。

　　 第二步，根据自己显示器的大小设置显示分辨率。显示器屏幕越大分辨率可以设置越高一些，一般14寸设置为640×480，15寸设为800×600，17寸可设1024×768。

　　 第三步，设置色深，色彩由低到高排列分为16色、256色、16位色、24位色和32位色。色深越高，代表色彩越接近真实世界的颜色(一般在24位色以上就可以称为真彩色了)，图像也就更逼真，同时显卡的负担也就越重。一般用户处理文本和上网，设置成16位色就已经是绰绰有余了。对配置较低的用户256色也是勉强可以接受的。

　　 第四步，设置显示刷新率。刷新率的高低由显卡和显示器的技术指标共同决定，所以用户在设置显示刷新率前要详细了解自己的显卡和显示器(特别是要要安装正确的显示器驱动)。点击“显示设置”窗口的“高级”按钮，进入显示高级设置窗口。点选“适配器”标签，在这里可以看到适配器的驱动程序的有关信息(如图5-3)。在该窗口中部可以看到“刷新速度”设置栏。点击黑色的三角型下拉符号，可以看到该显示器在当前分辨率下支持的各种刷新率和“默认的适配器”、“优化”。一般刷新率在75Hz以上时画面基本无明显闪烁感，有助于保护用户的视力(当然能设在85Hz以上会更好)。建议一般用户选择“优化”选项，这样系统会根据显示卡和显示器支持的最大刷新率自动进行设置。

　　 第五步，在完成以上四步后，屏幕一般会出现偏移，用户可以通过显示器的调节按钮等对显示屏幕的位置进行调整。当调节比较满意后，一般用户的显示设置就基本完成了。但对于经常切换显示分辨率的用户(多数是游戏玩家)，还应继续按前面讲解的方法，对显示器的不同分辨率分别进行设置。

　　 第五章 救护新生命——解决简单故障

　　 用户在使用电脑的过程中难免遇上这样或那样的问题。这些问题一般分为两大类，软故障和硬故障。软故障由于使用软件的不同，问题五花八门，限于篇幅我们只简单介绍一下简单硬件故障的解决。 对于初级用户而言最应该注意电脑各个部件的联接问题。主机和显示器的电源是否插好，各外设部件是否都与主机正确联接，需外接电源的外设电源开关是否已经打开。启动失败时仔细找找原因。有时在用了软盘后忘了从软驱中取出这样的小疏忽也会导致电脑无法启动，让不少初级用户束手无策。 对于主机部件出现的一些问题，机箱喇叭会通过“Di”“Di”的蜂鸣声进行报警，下面将各种不同蜂鸣声代表的意思“翻译”给各位读者。

　　 对于Award BIOS： 1短：系统正常启动。恭喜，你的机器没有任何问题。 2短：常规错误，请进入CMOS Setup，重新设置不正确的选项。 1长1短：内存或主板出错。 1长2短：显示器或显示卡错误。 1长3短：键盘控制器错误。请检查主板。 1长9短：主板 Flash RAM或EPROM错误，BIOS损坏。 不断长响：内存未插紧或损坏。 重复短响：电源有问题。 对于AMI BIOS 1短：内存刷新失败。检查条。 2短：内存ECC校验错误。 3短：系统基本内存检查失败。 4短：系统时钟出错。 5短：中央处理器(CPU)错误。 6短：键盘控制器错误。 7短：系统实模式错误。 8短：显示内存错误。 9短：ROM BIOS检验和错误。 1长3短：内存错误。 1长8短：显示测试错误。显示器数据线没插好或显示卡没插牢。 有时计算机出现故障后屏幕上还会出现一些提示信息： (1)CMOS Battery State Low(CMOS电池不足)； (2)Keyboard Interface Error(键盘接口错误)； (3)Hard disk Drive Failure(硬盘故障)； (4)Hard disk Not Present(硬盘参数错误)； (5)Missing Operating System(硬盘主引导区被破坏)； (6)Non System Disk Or Disk Error(启动系统文件错误)； (7)Replace Disk And Press A Key To Reboot(CMOS硬盘参数设置错误)； (8)Invalid Media Type Reading Drive C:(硬盘参数不匹配)； (9)Invalid Drive Specification(硬盘BOOT引导系统被破坏)； (10)Invalid BOOT Diskette，Diskette BOOT Failure(软盘引导系统错)； (11)FDD Controller Failure BIOS(软驱控制错误)； (12)HDD Controller Failure BIOS(硬盘控制错误)； (13)Drive Error (BIOS未收到硬盘响应信号)； (14)Cache Memory Bad, Do Not Enable Cache(主板Cache故障)。 常见的故障还可能是病毒引起的，用杀毒软件试试。
