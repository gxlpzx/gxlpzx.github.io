网页制作实用技巧

在网页中加入“添加到收藏夹”的功能

〈1〉普通实现方法

在网页〈body〉 〈/html〉语句之间加入以下代码即可

〈a href＝″javascriptwindow．external．Addfavorite ＇http//my．com＇＇收藏夹名字＇″〉加入收藏夹〈/a〉

〈2〉右键自动弹出式 实现方法：在网页〈head〉 〈/head〉 语句之间加入以下代码即可
〈 SCRIPT language＝JavaScript〉
function click 
if event．button＝＝2 
window．external．addFavorite ＇http//my．com＇＇收藏夹名字＇  document．onmousedown＝click 〈/SCRIPT〉
注：在if event．button＝＝2 中将 2 改为1 为点击左键弹出，改为3 为左右键同时点击时弹出。 网页中常用右键弹出，另外两种视特殊情况而用。 在网页中加入“设为首页”的功能 一些网页中“设为首页”的功能用到了注册表，既费事又不安全。其实用JavaScript语句可以轻松实现。

在网页〈body〉 〈/html〉语句之间加入以下代码即可
〈a href＝＃ onclick＝this．style．behavior＝＇url＃default＃homepage＇this．setHomePage ＇http//my．com＇〉设为首页 〈/a〉

注：将“http//my．com”换成你的主页地址，将“收藏夹名字”改为你希望显示在收藏夹中的名字，“加入收藏夹”“设为首页”为网页中的链接文字。注意区分“；” 和“：”。
