"解剖"品牌电脑广告

　　快过年了，小愚手中的票子早就攥得发热，这次拿到了一笔不小的年终奖金，怎么着也要捧一台电脑回家自己好生亲近亲近。作为一名“爱国人士”，当然要支持国产的品牌电脑啦，但看着IT媒体上铺天盖地的各家品牌电脑广告，这家说是率先跨进GHz时代，那家又以Windows Me为招牌，还有的干脆叫做“因特网电脑”或“潜龙”、“维也纳”什么的，弄得小愚晕头转向。 这不，小愚又甜言蜜语加威胁将我们那位乐于助人的老鸟“请”进了电脑城。 实际上，目前市场上所有的电脑都可以分为两大类：一类是品牌电脑，由专门的电脑厂商生产组装，采购时就像到家电商场购物一样，只要你整个儿搬回家就可以用了，而且一般都有较长时间的三包服务；另一类就是兼容机，它是由电脑销售公司根据用户提出的配置要求，帮你购买各种硬件，然后进行组装。由于品牌机在各配件的选用和搭配上一般都经过了严格的工序和反复的实验，出厂的时候还必须通过质量认证，因此品牌电脑的稳定性比那些自己动手组装的兼容机要好一些，故障率相对较低。尤其值得称赞的是品牌电脑的售后服务做得相对较好，不管是软件故障还是硬件故障，只要一个电话拨过去，一般就会有专门的技术人员上门服务。下面老鸟将同大家一起走进电脑城，给大家解释一下电脑广告。

　　 下面，我们和小愚一起来看看两份典型的品牌电脑配置广告：

　　 第一款(入门级配置)： Intel CeleronⅡ处理器566MHz 64M 15G 810 17寸数控 40× 集成显卡 1.44M 56K Modem 木质音箱 Windows SE操作系统 建议价：5988元

　　 第二款(豪华型配置)： AMD Athlon 800MHz 128M 30G硬盘 KT133 17寸纯平 48× TNT2 32M 1.44M 56K Modem 木质音箱+超重低音炮 互联网滚轮鼠标 Windows Me操作系统 建议价：11388元

　　 看了这两份品牌电脑的配置广告，不知道大家能看懂多少，下面听老鸟为大家一一道来。

　　 Intel CeleronⅡ 566MHz/AMD Athlon 800MHz：无论是选择品牌电脑还是兼容机，CPU这都是最重要的内容之一。“Intel(r) Celeron (r)Ⅱ 566MHz”这里的信息说明CPU是Intel公司的工作主频为566 MHz的赛扬二代处理器。“AMD Athlon 800MHz”则代表AMD公司的主频为800 MHz的速龙处理器。在有些INTEL的CPU介绍中往往还能看到的“EB”之类的标识，其中“B”说明CPU是133外频、“E”说明是铜矿(Coppermine)处理器。假如看到“866EB”，那么就说明它是外频133MHz主频为866 MHz的铜矿CPU。广告中常提到的“GHz时代”，意思就是说CPU主频达到1000MHz以上。目前市场上常见的CPU中从性能角度看，速龙和铜矿是一个档次，钻龙和赛扬二代基本接近，但从稳定性角度讲，Intel的CPU要略好一些(所以市场上品牌机的CPU还是Intel的居多)。 64M/128M：指的是计算机内存容量的多少。内存对计算机性能的影响也是非常的大，内存越多程序运行也就越快。建议一般家庭和单位配置128M以上的内存，这样不论是面对办公还是娱乐都可以做到得心应手。 15G/30G：这里是讲的硬盘容量。硬盘在电脑里扮演的是“库房”的角色。“库房”越大，装的东西自然更多，用起来当然就更方便。目前在市场的主流硬盘是在20G左右。硬盘还有其他一些性能指标，比如转速、缓存和接口速度等。转速越高、缓存越大、接口速度越快硬盘资料的读写也就更快。这些在广告上很少提及，用户可在购机时向商家咨询。 810/KT133：这是指主板所用的芯片组型号。“810”代表主板使用的是Intel公司的810芯片，“KT133”表示威盛公司的KT133芯片组。一台电脑能使用什么样的CPU，最多支持多大的内存，硬盘接口型号等功能都是由主板来决定。主板的型号直接制约了以后升级的途径，用户在购买时一定要注意，最好多向有关专家咨询。 17寸数控/17寸纯平：这是“17寸”指的是显示器的尺寸，跟彩电的尺寸是一回事，尺寸越大屏幕也就越大。一般家用电脑选择显示器最起码都要在15寸以上，17寸当然最好了。“数控”指的是显示器调节方法是数字控制方式，以前的显示器用的都是模拟控制，现在一般都是采用数控技术。“纯平”指的是纯平显示器，这种显示器的显示效果比一般的显示器效果要好得多。有时我们还会看到TCO 95/TCO 99的字样，这说明显示器通过了著名的瑞典专业雇员联盟制定的TCO认证(TCO是国际上关于显示器辐射要求的最严格标准)，它一共分TCO 92、TCO 95、TCO 99三种，其中TCO 99认证最为严格。一般来说只要通过了TCO 95认证的显示器对人体的辐射已经可以忽略不计。大家在购买显示器时最好把这些问清楚，这可是关系到大家自身健康的大问题。 40×/48×：这是指光驱的速度，一般也说成40倍速和48倍速。一般来讲当然速度越快越好，不过速度越高，发热量和噪音相应也就越大，其实40×也已经足够了。 集成显卡/nVIDIA RIVA TNT2 32M：这是告诉我们电脑所用的显卡情况，“集成显卡”就是说把电脑的显示卡功能集成到主板芯片中，这样做可以节约成本，同时对一般用户来说并不影响使用。“nVIDIA RIVA TNT2 32M” 中的“nVIDIA”是指世界上最著名的显示芯片生产厂家，使用相同芯片的显示卡性能差距一般来说非常小。“RIVA TNT2”是指显示芯片的型号，这里的“32M”是指显存的容量大小，当然也是越大越好。 1.44M是指计算机配备有3.1寸的软驱，可以使用容量为1.44M的软盘来传递数据。 木质音箱/木质音箱+超重低音炮：这里说是使用了何种的音箱，从理论上讲，木质音箱效果会好于塑料音箱。 56K Modem：这是在说“猫”呢，Modem又叫调制解调器，是上网的关键部件。按上网连接速度可以将“猫”分为14.4K、33.6K、56K几种，现在一般都是56K的。买“猫”的用户还要注意一下“猫”是安装在机箱里还是机箱外，机箱外的“猫”相对更加稳定。 Windows 98 SE操作系统/ Windows Me操作系统：操作系统是电脑使用的软件基础，一台电脑必须先安装操作系统后才能使用。“Windows SE操作系统/ Windows Me操作系统”都是目前市场占有率最高的微软公司的操作系统，品牌电脑一般都会预装正版的操作系统，可别小看这个操作系统，单独购买的费用肯定会吓你一跳。

　　 老鸟这边滔滔不绝讲了这么多，小愚开始有些吃不消了，赶紧掏出独门法宝“不耻上问”，提出几个菜鸟必问：

　　 1.声、显卡集成到底是怎么回事？ 现在一方面是各种配件越做越高档，另一方面是主板All In One的整合趋势也越来越明显，Intel和VIA这两家世界最大的主板芯片公司都分别推出了如i815E、PM133等整合芯片组。声、显卡集成的含义就是将声卡、显卡的功能整合到主板上，这些集成的声卡、显卡完全足以满足一般用户的需要。选择这些声、显卡集成的电脑，可以省下一笔不小的费用，等以后有了需要再升级也不迟。但大家必须注意并非所有的整合主板都提供良好升级能力，像使用810整合芯片组这类不提供AGP插槽的主板，升级时局限性就很大(目前绝大多数显卡使用AGP接口)。整合主板一般都提供PCI插槽，这样声卡的升级就没什么问题。

　　 2.互联网滚轮鼠标是一种新型的鼠标吗？ 所谓互联网滚轮鼠标，实质上是在过去的两键或三键鼠标上增加了一个滚轮以实现翻屏功能，一般鼠标需要整只手移动来完成的工作，滚轮鼠标只需轻轻动一下手指头就能实现，对用户来说非常实用。

　　 3.因特网键盘与上网有关吗？ 是的，为了迎合网络时代家用电脑的需求，与普通键盘相比，因特网键盘不仅更加美观，而且一般预设了十多个上网热键，只要轻轻一按，就可进入新闻、证券、学习教育等与日常生活密切相关的各类主要网站。有些还能通过键盘上的热键实现开机、关机、播放CD等功能。对于初学者和家庭用户而言，这种键盘是非常方便的。

　　 4.那些某某之家、某某工作室之类软件有什么用？ 这些一般是品牌电脑厂商为方便用户使用电脑而设计的工具集合软件，功能非常丰富，从日常办公、百科知识、家庭娱乐、上网指南等包罗万象应有尽有，对于初级用户非常实用。 一般来说，品牌电脑都会随机赠送一些正版软件如Windows 98安装光盘和其它一些实用软件，有的还会赠送防尘罩、鼠标垫等小礼品。有些品牌电脑厂商还会定期推出一些购机优惠措施，如加×××元送打印机等，一般用户可以结合自己的需要选择。另外，购机后一定不要忘记办理厂商注册、填写保修单等手续。只有在计算机生产商注册后，用户才能真正享受品牌电脑的优质服务。 (江苏 王志军)
