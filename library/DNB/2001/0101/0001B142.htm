在Director中制作动画光标

    Macroemdia Director是一款运行于Windows平台下的功能强大的多媒体制作工具，它的有些功能要远远强于AuthorWare(为什么许多人选择AuthorWare是由于它比较简单)， Director曾被誊为“多媒体巨匠”和“多媒体标准”。但是非常可惜的是，尽管在Dirctor之中，除了内定的的游标外，使用者也可以自行绘制游标图形，但是图形大小为必须为16x16大小，而且图形转换为1-bit的格式，也即单色，这样制作出来的产品就比较单调，如果我们能够制作出彩色的甚至是动画光标，那就一定会为我们的多媒体产品增色不少。

    通过笔者的摸索，发现了在Director中制作动画鼠标光标的方法，愿意写出来与大家一起共同交流。

    1、启动Director，建立新文件，并新建一组演员，并将它们全部选中，再选取Modify菜单下的Cast to time命令来创建动画；

    2、在Score窗口选中角色，将其墨水效果设置为Background Transparent，即背景透明。

    3、单击Insert菜单下Film Loop命令，在弹出的对话框中输入循环动画的名称，本例为mouse(见附件图图3.bmp)，单击OK，可以发现在Cast窗口中又多了一个演员mouse。

    4、回到Score窗口，并将Score窗口中原角色删除，再在Cast窗口中将演员mouse拖至舞台产生新的角色，同时设置Score窗口中角色的画面跨度拖为1帧。

    5、编写程序，在Score窗口中双击脚本通道的第一帧，输入如下代码： on enterframe cursor 200 set the loch of sprite 1 to the mouseh set the locv of sprite 1 to the mousev updatestge end on exitframe go to the frame end 7、关闭Script、Cast、Score窗口，点下“播放”钮来观看电影效果，可以发现，一只可爱的动画鼠标就出现在屏幕上了，而且它会随着鼠标的移动而移动，产生了动画光标的效果。
