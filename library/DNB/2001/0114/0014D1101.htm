{让你的网页更“小”}

{ 做过网页的朋友一定知道，用Dreamweaver或Frontpage做的主页有大量的垃圾代码，这样多的垃圾肯定阻碍你上传主页的时间，更重要的是你的客人会因为你的主页浏览的龟速对你的主页有个不好的印象。那怎么办？不用急，我这里有一个好东东，看看是不是你想要的呢？
这个好东东叫主页稀释剂，是一个十分不错的网页减肥软件，能够将你的网页文件压缩，去除一些不必要的字节。它和同类软件相比，具有安全、快捷、操作简便的特点，最高压缩率可达40%，因为它不会在压缩后会产生一些HTML的语法错误，导致一些Javascript不能运行；也不会改变你的主页的外观以及功能。
运行程序后能够看到，“主页稀释剂”的界面虽然简单了点，甚至可以说是简陋的。不过俗话说麻雀虽小，五脏俱全，各位可不要因此小瞧了它哦，相信大家用过它后就会被它强大而实用的功能所折服！找到你要压缩的网页，然后程序就会自动在你指定的文件夹中找到HTML files ( .htm  hmtl  .dhtml  .shtml )文件，并在左上方的窗口显示出来，如果你制作的是XML files ( .xml 文件，可以点击文件类型的下拉菜单选择。OK，在左上方的窗口中选择一个或者按住Shift键选取所有的HTML files文件，然后点击“添加”按钮，这时你选取的网页文件就会出在下方的窗口，看看有没有选错，如果有些网页不需要减肥，可以点击“移去”按钮将它移去。选好了所有需要减肥的网页文件(主页稀释剂一次可以压缩255个文件)，点击“压缩”按钮，主页稀释剂就会快速将网页文件压缩至原来的70%～80%。在所有的网页文件压缩完成后，主页稀释剂会弹出一个窗口告诉你原来文件大小和压缩后文件的大小，让你比较一下。
“主页稀释剂”是一款免费的国产软件，目前它的最新版本是1.4，适用于Win95/98/NT/2000，下载解压缩到一个文件夹中即可使用了，不用安装。这个软件有中英两个版本，有需要的朋友可到http://wwwljx.163.net/download/htc.zip下载中文版，http://wwwljx.163.net/download/hte.zip下载英文版，软件大小为249 KB。
如果你还在为你的网页个头太大，结果吓走了不少网友而发愁的话，不妨试试在做好网页后用主页稀释剂让你的网页“瘦”下来。}
