{硬盘“保健医生”——SIGuardian}

{ 现在，越来越多的用户倾向于购买性价比高的大容量硬盘，以便能够存储更多的数据。但是你想过没有，如果有一天硬盘出现了问题，而你又没有及时做好备份，那么存储在硬盘上的数据就有可能丢失。对付这个问题的一个办法就是经常关注硬盘的“健康状况”(当然，备份好硬盘上的数据是最佳的方法)，在硬盘出现问题之前就做到心中有数，然后采取相应的对策。
SIGuardian便是一款小巧但十分有用的硬盘“健康状况”监视软件，它能在硬盘“怠工”之前就发现潜在的问题。现在的硬盘几乎都使用了S.M.A.R.T.(Self-Monitoring Analysis and Reporting Technology，自我检测、分析和报告技术)来监测硬盘的稳定性，SIGuardian就是利用这种自测技术的结果来评估硬盘的使用寿命，它支持所有符合ATA标准的硬盘。SIGuardian为共享软件，大家可以到http://www.siGuardian.com去下载其Lite版的30天试用版本V1.4 Build #103。
运行SIGuardian之后，其主界面如图所示，如果你的系统中连接有两个以上的硬盘，必须先在“choose HDD Drive”后面的下拉框中选择要该软件检测的硬盘，然后单击下面的图标按钮查看对应的信息：
1.点击“General”(常规)按钮，可看到关于此硬盘的常规信息，包括容量、柱面、磁头、硬盘型号、工作模式等。
2. 点击“Disk health”(硬盘健康)按钮后，下面显示的是要检测的硬盘的健康状况报告，这里列出了SIGuardian开始监视硬盘的时间、最后检测的时间以及检测结果，该项下面的两个按钮的含义分别是：
1“Simulate death”(模拟“死亡”)用于模拟当检测到硬盘发生问题时要执行的操作(SIGuardian一旦检测到硬盘潜在的问题，就在窗口中显示警告信息，并提醒你赶快备份数据)。
2“Simulate change"(模拟状况改变)可模拟硬盘健康发生改变时的状况，每单击一次，则在窗口中显示一个“Nearest prognosed end day of disk lifetime”(预计最近的硬盘寿命终止时间)。
3.在“Options”(选项)下可以设置该软件对硬盘的监视方案，如果仅仅希望每次启动时监视硬盘状况，则应在“Monitor disk health only at system startup”前打上钩，如果不选该项，则表示让SIGuardian一直监视硬盘，然后在窗口下面设置让SIGuardian检测硬盘的间隔时间。设置好以后，SIGuardian到时会自动通知发现的错误。
建议：最好不要勾选“Options”中的“Monitor disk health only at system startup”(仅在系统启动时检测硬盘健康)项，这样可以让SIGuardian一直监视你的硬盘工作状况，另外检测时间间隔不要设置得过短，以免你在做其它工作时被它弹出的提示信息打断。
怎么样？SIGuardian确实不错吧！有了它，相信你的硬盘就能更加安心地工作了！}
