留住匆匆的脚步——WINDOWS的启动和关机大剖析

爸爸在小愚的“电脑热情”影响下，搭上电脑学习的“末班车”，小愚好不得意。有天，爸爸虔诚地向小愚请教Windows开关机的画面，小愚是只知其一不知其二，这个问题嘛还是得请教老鸟。 ##1 Windows 9x/Me篇 (一)多重启动菜单 用过Windows 95的朋友都会遇到过在进入开机画面前的选项提示，这就是多重启动菜单。 小愚：什么叫开机画面？ 老鸟：开机画面就是当你启动Windows后，出来的第一个画面。 而Window 98/Me不像前者那样默认进入多重启动菜单，而必须在启动过程中出现“Starting Windows 98……”的提示信息前快速按下F8键或Ctrl键，此时就可以看到Microsoft Windows Startup Menu，即多重启动菜单。下面我们就逐一介绍每个选项的意思。 1.Normal 这是启动时的正常模式，也是Windows 9x/Me的缺省启动模式，正常启动时会直接以该模式启动，此时将加载所有的系统文件和驱动程序。 小愚：我们一般用的都是这种模式。 2.Logged——注册模式 此时系统会自动创建一个启动注册文件BOOTLOG.TXT(位于C盘根目录下)，它记录了启动系统时的启动过程，主要是正常加载Windows初始化的组件和驱动程序，并在执行过程中给出提示信息。当系统出现错误时，你就可以根据错误的具体情况查看该文件。 小愚：作为初学者，这点只做了解就可以了。 3.Safe mode——安全模式 我们通常使用该模式来诊断和解决系统中出现的问题。此时，系统只使用最少的驱动程序和基本选项加载Windows，并且会在显示器的四角显示“安全模式”字样，在安全模式中，Windows 使用默认设置(VGA 显示器、Microsoft 鼠标驱动程序和启动 Windows 所需的最少设备驱动程序)，用户将无法连接到网络，且无法访问 CD-ROM 驱动器、打印机和其他设备。 小愚：什么是设备驱动程序？ 老鸟：允许特定的设备(如网卡、调制解调器、打印机等)与Windows通讯的程序。例如，如果没有串口驱动程序，“网络连接”就不能使用调制解调器与网络连接，虽然系统中可能已经安装该设备，但只有已经安装并配置了驱动程序以后Windows才能使用该设备。当系统启动时，设备驱动程序会自动加载(对于所有启用的设备)并运行。 4.Step-by-Step Confirmation——步进模式 这时用户可以逐步确认是否执行Windows 加载序列中的每个活动。Windows启动时的每一步都会显示出来，对于每条要运行的命令，请按Y键确认，如要跳过某个命令，可按N键，只有当前命令运行成功后，才会提示下一条命令；如果命令运行不成功，用户看到的将会是出错消息，这对我们查找系统启动错误很有用。 5.Command prompt only——命令行提示行模式 启动后将直接进入命令提示行方式你可以通过命令行的方式来启动Windows。进入该模式后，在C\WINDOWS>后运行WIN.COM/﹖命令可获取参数。每个参数的作用不同，如，M参数是以安全模式(Safe mode)启动系统；N参数同样是以安全(Safe mode)模式启动系统，但可以访问网络，等等。有兴趣的话读者可查阅一些资料。无论以哪一个参数启动Windows，参数D都是必不可少的，之后你才能使用其它参数。再如，运行WIN.COM/D/F命令则以安全模式的方式进入Windows系统。 小愚：参数好比开关，选择不同的开关，就有不同的进入Windows的方式。 6.Previous Version of MS-DOS 以前版本的MS-DOS模式，也就是使用安装Windows以前的DOS系统来启动，当然假如你在安装Windows时并未选择保存旧的DOS系统，该选项将不存在。 小愚提醒大家，由于Windows Me已摒弃了MS-DOS，因此第5、6项将不再存在。一般常用的是安全模式(Safe mode)，其他的有兴趣了解了解也可以。 (二)关机菜单 由于Windows 9x/Me的关机菜单比较简单，小愚这里就不多说。但Windows Me的关机菜单与Windows 9x有所不同，它以下拉框的方式代替了原来的单选框方式，其它并无区别。 ##1 Windows 2000篇 小愚对网络功能更完善的Windows 2000也产生了浓厚的兴趣，就顺带着向老鸟请教了一下。 (一)Windows 2000开机时的选择 出现多重启动菜单的条件和Windows 98/Me的一样，前面已经讲过，下面介绍Windows 2000的启动菜单。 1.安全模式 如果计算机不能启动，则可以从安全模式启动。 在安全模式中，Windows 2000 使用默认设置(VGA显示器、无网络连接、Microsoft 鼠标驱动程序和启动 Windows 所需的最低设备驱动程序)。如果计算机在安全模式下没有成功启动，则可能需要使用紧急修复磁盘 ERD 功能以修复你的系统。例如，如果安装新的软件后计算机无法启动，你可以在安全模式下以最少的服务启动，然后改变计算机的设置，或者删除引起问题的新安装的软件。如果以安全模式启动时没有再出现故障现象，你就可以将默认设置和最小设备驱动程序排除在可能的故障原因之外。 小愚：什么是ERD？ 老鸟：ERD也称为紧急修复磁盘，它是通过“备份实用程序”创建的磁盘，包含关于当前Windows系统设置的信息，如果计算机不能启动或者系统文件被破坏或删除，可以使用这个磁盘修复计算机。 2.带网络连接的安全模式 与“安全模式”相似，此时只使用基本文件和驱动程序以及网络连接来启动 Windows 2000。 3.带命令行提示符的安全模式 此时，仍然只使用基本的文件和驱动程序启动 Windows 2000。不过，与上述两项有所区别的是，登录后屏幕出现的是类似于DOS系统的命令提示符，而不是Windows桌面、开始菜单和任务栏。 4.启用启动日志 选择此方式后，启动Windows 2000的同时将系统的所有驱动程序和服务记录到一个名为ntbtlog.txt的文件中，它位于%windir%目录中。实际上，安全模式、网络安全模式和命令提示符的安全模式，都会将一个加载所有驱动程序和服务的列表添加到这个启动日志，该文件对于确定系统启动故障问题的准确原因很有用。 5.启用VGA模式 这种方式只使用基本VGA驱动程序来启动Windows 2000，当安装了使Windows 2000不能正常启动的新显示卡驱动程序时，这种模式特别有用。当你在安全模式(安全模式、网络安全模式或命令提示符安全模式)下启动Windows 2000时，其实都是使用基本的显示卡驱动程序。 6.最后一次的正确配置 小愚还是觉得该选项最为实用，它可以使用 Windows 上一次成功关闭时所保存的注册表信息来启动 Windows 2000。不过，请朋友们只在配置不正确时使用(如由于升级驱动程序或安装某软件导致Windows 2000产生错误时)，因为最后一次的正确配置并不能解决损坏或缺少驱动程序或文件所导致的问题。 小愚心中暗喜：这下就不怕Windows 2000崩溃了，反正可以通过这一项菜单来恢复。 7.目录服务恢复模式 请注意，该方式不适用于Windows 2000专业版的普通用户，这是针对Windows 2000服务器版操作系统的。 8.调试模式 启动Windows 2000，同时将调试信息通过串行电缆发送到其他计算机。如果正在或已经使用远程安装服务在你的计算机上安装 Windows 2000，你还可以看到与使用远程安装服务恢复系统相关的附加选项。 9.正常启动 这个问题小愚就不啰嗦了。 (二)Windows 2000关机时的选择 小愚和使用Windows 2000的朋友一样对这个画面很熟悉了。 1.关机 这个选项使用的频率最高，请菜鸟们注意，只有在通知你可以安全关闭计算机的消息出现以后，才能关闭计算机。因为系统运行时，Windows 2000 将重要的数据存储在内存中，并且在关闭计算机之前需要将数据写入到硬盘中。还有一种关机的方法是同时按下“Ctrl+Alt+Del”键，在出现的“任务管理器”对话框中单击“关机”按钮，然后在“希望计算机做什么？”列表中，单击“关机”即可。 2.注销 该选项是用来注销当前用户，你可以使用另一用户名重新登录系统。 3.重新启动 4.等待 选择该选项后，系统进入“等待”模式。不过你必须从“控制面板/电源选项”中进行设置(而且需要主板、显卡、显示器、电源等硬件的相应支持)才行。如果在这里选择了“启用休眠支持”还将在“关机”菜单中增加“休眠”一项。 看了这些内容，朋友们不再对Windows的启动和关机心中无底了吧？小愚也把这些菜单的含义弄清楚了，更重要的是还得到了爸爸的夸奖。江苏 王志军
