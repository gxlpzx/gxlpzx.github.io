{光盘刻进了病毒怎么办？}

{ 刚买来一台Acer CRW8432IA刻录机，给朋友刻了40张数据盘，每张20.4MB。可是，今天朋友找我来了，说刻进了CIH病毒。天哪！怎么办？这些盘都不要，损失可惨了，而且母盘已经还人了。有没有什么办法能消除刻进了光盘的病毒？经过一番实验，成功了。成功的喜悦不敢独享，特将该办法写出来和大家一起分享吧！
软环境：随机软件Adaptec Easy CD Creator 4中文版。
原理：利用Adaptec Easy CD Creator 4中文版提供的多段式刻录功能，先将数据复制到硬盘，再用杀毒软件杀之，导入片段后，将原来的文件删除，或者替换原来的数据，刻上新的数据即可。
具体操作：
1.反正系统已经感染了CIH，先将数据复制到硬盘上的一个临时文件夹，如Temp。(其实只需将.exe复制即可，因为其它文件未感染。但是光盘中的文件夹过多，这样操作就麻烦了。)
2.用一干净启动盘重启机器，用杀毒软件将系统杀干净。取出所有软盘及光盘。
3.重启系统，此时先将系统光驱的“自动插入通告”功能去除，参见下图。具体过程是在系统属性的CD-ROM属性中修改，将设置选项卡中，“自动插入通告”前单选框中的钩去掉，再重启系统。因为光盘中软件的自动运行会使你的系统重新感染上病毒，这样就前功尽弃了。
4.启动Adaptec Easy CD Creator 4中文版，插入刚才有毒的光盘。
5.导入片段。在CD版面中将含有病毒的文件删除(一般为.exe。
6.重新添加数据。
7.制作CD。
8.检验。用杀毒软件查毒，无毒。大功告成，挽回损失，其它剩下的39张只需重复第7步即可。
说明：用其它刻录软件也能达到此目的，如Adaptec DirectCD3.01中文版。}
