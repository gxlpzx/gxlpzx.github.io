{二手电脑存储外设介绍}

{ 由于电脑硬件发展迅猛使得大量电脑外设进入二手市场，使普通用户有机会以非常低的价格买到商业系统的外设，虽然这些存储产品都不是最新的型号，但其中不少产品对个人来说也不能算淘汰货。在众多常见的二手电脑设备中MO驱动器、磁带机、ZIP驱动器、外置盒比较有使用价值。
二手的MO驱动器主要有230MB和640MB两种，有外置和内置的产品，一般230MB的产品二手货价格在300～400元左右，640MB的MO驱动器还属于主流产品，价格在600～800元左右，在众多品牌的MO中，富士通产品口碑比较好，广泛应用于专业领域，二手货的数量也比较多。
如果用户只是用来存放大容量的数据，那么二手磁带机的成本更低。随着更大容量的磁带机的应用，很多4GB～8GB的产品进入二手市场，由于在很多人印象中磁带机都属于过时产品，个人应用很少，所以二手货价格较低，多在150～400元之间，驱动器价格和单位存储成本远较MO便宜。全新DDS1 4mm 4G磁带在30元左右，一些用过的磁带价格更为低廉。磁带机采用顺序存储的记录形式，加上磁带的成本比较低，所以主要用于大容量数据的存储。磁带机单位存储成本与CD-R相当，而且可以擦除。磁带机的种类很多，外置内置的产品都有，而且速度也不相同(存储64MB数据的时间从2分钟到3.5分钟不等)，选购时要注意一下指标。常见二手磁带机中HP、Seagate、FUJITSU品牌性能比较好。
ZIP驱动器有100MB和250MB两种，大部分二手货都是100MB的产品，ZIP在个人用户中有一定的用户比例，二手ZIP驱动器大部分是并口外置的产品，价格在250元左右。外置盒可以配合原来内置电脑设备独立在微机机箱外工作，有的外置盒还支持使用硬盘。外置盒有SCSI、并口和USB多种接口型号的产品，二手货中主要是SCSI和并口的产品。有的外置盒具备独立音频输出的端子，配合一台带有播放键的CD-ROM就可以成为一台CD机。限于端口速率，高速设备不建议使用并口外置盒，而SCSI接口的外置盒一般没有这个问题。很多二手MO驱动器、磁带机、ZIP驱动器都采用SCSI接口，虽然性能优于IDE或并口连接，但一般个人电脑中都没有SCSI接口，需要另外附加SCSI卡。高速SCSI卡价格比较贵，二手的SCSI卡相对比较便宜，Adaptec AHA-2940UW 40M PCI SCSI卡约450元左右，Adaptec AHA2940W 20MB PCI SCSI卡约350元。
此外，选购二手存储设备要注意产品的接口和驱动程序，对于SCSI设备最好带有连接线。
}
