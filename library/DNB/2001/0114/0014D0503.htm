{浏览器NetCaptor的新功能}

{ 本文向你介绍一款极佳的浏览器软件，虽然是英文版，但只要你不是太菜，看了我的介绍，应该很快就能上手。
NetCaptor(以下简称NC)是一款广告赞助软件，所以是免费的。你可以到http://www.netcaptor.com下载其最新版6.5.0 Beta 1只有824KB。不过你首先要有IE4.0以上版本，因为它采用了IE的内核。安装完后，启动NC，出现对话框，提示是否将NC设为默认浏览器，你可以先选择No。这时你可以看到，它的界面和IE酷似，不过它有很多IE所没有的功能，能极大地提高浏览的效率。下面介绍一下它的新功能。
1.用它可以同时浏览多个页面，在浏览器下方显示了每个打开页面的标签。而不像IE那样，点一个链接就弹出一个窗口，弄得我们手忙脚乱的，一会儿最大化一会儿最小化。而且每个页面都只在后台打开。这既不影响我们前台浏览又减少了系统资源的浪费，当然也提高了显示速度。在默认情况下，所有标签都显示在最后一行，你可以将其设置为多行显示。选择Tools菜单下的NetCaptor Options，NC的很多新功能都可以在这里设定。单击左栏中的Browser Tabs，把右栏中最后一行的Display browser tabs on multiple lines前面的钩选中。这样你以后打开的所有窗口就在浏览器下方以多行显示，一目了然。
2.你可以利用工具栏上的Load Images按钮快速设定是否加载图片，使你很方便地在效果和速度之间作出取舍。
3.在地址栏中可用Ctrl+Enter在键入部分前后分别加上http://www.和.com，用Shift+Enter加上http://www.和.net，用Ctrl+Shift+Enter加上http://www.和.org。这只是默认值，你也可以根据需要自己设定。
4.它是广告商的克星。想必新手上网也有这样的体验，只点了一下，突然唰唰唰，不知从哪儿又冒出几个窗口来，尽是些广告。用NC两三下就可以搞定，而且是根除。先把广告页面激活，然后按F8，回车，你可以试着重新打开这个链接，怎么样？
5.NC增添了一个叫做CaptorGroups的功能，用它可以把你自己喜欢的网址归类，然后只要双击这个类名，你就可以打开该类的所有网页。而且你在浏览网页的时候随时都可以按F6将当前页保存在你以前所定义的一个类里边，或者按F7将当前页保存在一个新的类别里。
6.NC提供了丰富的快捷键，是同类其它软件所望尘莫及的。当你熟练了这些快捷键，到时候一手按鼠标一手按键盘双管齐下，是不是很爽？详细情况自己去研究Help吧。}
