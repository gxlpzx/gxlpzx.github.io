{OICQ黑客工具大暴光}

{ 使用OICQ这段时间里，遇到过源源不断接收到一些存心发送过来的相同信息骚扰吗？这就是所谓的消息攻击，飘叶OICQ千夫指就是这类OICQ信息攻击软件。
飘叶OICQ千夫指
运行平台：Win95/98/me/NT/2000
授权方式：免费软件
软件大小：165K
OICQ测试版本：OICQ99C Build 0820
软件下载网址：http://www.51299.com/oicq/qfz.zip
使用说明：这是一个绿色软件，不需要安装，你解压出来就能直接使用，这个版本的飘叶OICQ千夫指只能对OICQ99C Build 0820以下版本有效，你只要使用OICQ查IP补丁(http://www.51299.com/oicq/2001/0330/oicq0310.zip)，先查找出对方IP和端口，然后在飘叶OICQ千夫指软件相应栏目内填上查到到的IP和端口，再选好假冒的号码数目和信息语句，按“发送”即可完成。
防御方法：安装OICQ2000版就能阻拦这个版本的飘叶OICQ千夫指OICQ垃圾信息攻击。
飘叶OICQ千夫指 2.1
运行平台：Win95/98/ME/NT/2000
授权方式：免费软件
软件大小：145K
OICQ测试版本：OICQ2000b Build 0325
软件下载网址：http://www.51299.com/oicq/2001/04/02/qfz21.zip
使用说明：飘叶OICQ千夫指 2.1需OICQ2000版本配合使用，你首先要在OICQ主菜单的系统参数设置中取消“自动弹出信息”，接着在OICQ中双击对方图标弹出“发送信息”窗口，把当前的“普通模式”转换为“对话模式”；接着选择飘叶OICQ千夫指2.1的“指责语句”，按下“发送”按钮即可，如果你觉得不需要再发送OICQ垃圾信息，就按下“停止”按钮终止攻击。你也可以修改或新加一些“指责语句”按下软件中的“编辑语句”，在编辑发送语句栏目里自由添改新内容，完成后选择确定即可。
防御方法：使用OICQ的功能——黑名单，不用理会他发来的OICQ垃圾信息，把他的QQ头像轻轻的拖到黑名单里去，这就把问题解决了，如果还需要找他，就把他的QQ头像拖出黑名单，这时他发来的OICQ垃圾信息已没有。}
