AutoCAD 2000标注公差技巧几则

AutoCAD 是目前国内外使用最广泛的机械设计软件，AutoCAD 2000丰富的绘图命令、强大的编辑功能、三维造型功能和良好的用户界面深受广大工程技术人员的欢迎。
《机械制图》国家标准(GB4458.5-84)对零件图线性尺寸公差的标注式样，规定有三种：
1.公差代号标注如；
2.极限偏差标注如；
3.同时标注公差代号和极限偏差如。
AutoCAD2000所提供的尺寸公差标注，是在尺寸管理命令【Dimension Style Manager】中建立一个“公差标注式样”，在【Tolerance】(公差)和【Primary Units】(主要单位)选项卡中进行上下偏差等有关参数的预先设置(具体设置略过)，每标注一个不同的尺寸公差都要返回【Dimension Style Manager】中进行设置，很麻烦，且只能完成上述标注式样的第2种——标注极限偏差。笔者在进行AutoCAD和《机械制图》教学实践中，对尺寸公差标注获得一些体验，现总结如下：
##1一、属性匹配法：(主要命令：【Properties】属性工具)
利用预先设置的一个尺寸公差式样，然后通过【Properties】属性工具进行修改。例如把的上下偏差改为。
1.单击【Properties】命令按钮，弹出【Properties】属性工具对话框。
2.单击已标尺寸，回到【Properties】对话框，在【Tolerances】选项中对Tolerances limit upper /Tolerances limit lower显示的上下偏差值进行分别修改即可。
##1二、尺寸修改法：(主要命令：【Dimension Edit】尺寸标注编辑命令)
利用“尺寸标注编辑命令”标注尺寸公差是对图中已经标注的尺寸进行修改，不论原尺寸是什么式样，都可以方便进行修改，如把图中已标尺寸58改为。
1.单击【Dimension Edit】命令按钮。
2.命令提示行显示Enter type of dimension editingHome/New/Rotate/Oblique<Home＞输入N确认，显示【Multiline Text Editor】多线文本编辑对话框。
3.在【Character】(字符)选项卡中输入%%C65+0.009^-0.021(注意插入符‘^’分开上下偏差)删除‘< ＞’(原尺寸)拖动鼠标选中+0.009^-0.021单击【】(堆叠/取消堆叠)按钮，单击【OK】按钮退出对话框。
4.鼠标变成捕捉框，捕捉已标尺寸58，确认就修改成(对图中任何尺寸都可以完成修改，有点像【Match Properties】命令)。
##1三、文本编辑法：(主要命令：【Linear Dimension】线性尺寸命令+【Multiline Text Editor】)
文本编辑法是在标注线性尺寸时调用【Multiline Text Editor】多线文本编辑对话框，按一定格式输入符号和数字，即可在鼠标已捕捉的两点之间标注尺寸公差。以标注为例说明。
1.单击线性尺寸【Linear Dimension】命令按钮(若要标注倾斜方向尺寸公差，单击【Aligend Dimension】)。
2.在图中捕捉要标注尺寸的两点，命令提示行显示Specify dimension line location orMtext/Text/Angle/Horizontal/Vertical/Rotated，输入M确认，显示【Multiline Text Editor】多线文本编辑对话
3.在【Character】(字符)选项卡中输入%%C65K7+0.009^-0.021删除‘< ＞’(原尺寸)拖动鼠标选中括号中+0.009^-0.021单击【】(堆叠/取消堆叠)按钮，单击OK按钮退出对话框。
4.返回图中鼠标左击确认。
##1四、命令输入法：(主要命令：【Linear Dimension】+【Text】)
命令输入法是在标注线性尺寸时，直接在命令提示行按一定格式输入符号和数字。以标注为例说明。
1.单击线性尺寸【Linear Dimension】命令按钮(若要标注倾斜方向尺寸公差，单击【Aligend Dimension】)。
2.在图中捕捉要标注尺寸的两点，命令提示行显示Specify dimension line location or Mtext/Text/Angle/Horizontal/Vertical/Rotated，输入T确认。
3.在命令行中直接输入%%C65K7\H0.6x\S+0.009^-0.021(注意符号、大小括号的格式和大小写，若要标注，省略K7和大小括号)。
4.回车确认。
##1五、文字说明中注写：(主要命令：【Multiline Text】)
若是在文字说明(如技术要求)中需注写尺寸公差，如注写。
1.单击段落文字命令按钮【A】(Multiline Text)
2.用鼠标在图中拖出“输入多行文字定位框”，显示【Multiline Text Editor】多线文本编辑对话框。
3.在【Character】(字符)选项卡中输入%%C65+0.009^-0.021拖动鼠标选中+0.009^-0.021单击【】(堆叠/取消堆叠)按钮，单击【OK】按钮即完成的标注。
