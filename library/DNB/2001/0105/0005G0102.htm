{宇峻新史诗}

{ 曾经有那么一段日子，无聊的中文角色扮演游戏让人感到有些恶心，整个中文游戏业几乎停滞不前时，《超时空英雄传说2：复活魔神》的推出让我唏嘘不已，尽管这只是款角色战棋策略类游戏，但它在角色的个性刻画，剧情的铺垫以及情感的描写上比当时任何一款角色扮演游戏都要强百倍。
当然，很多玩家都希望好游戏有续篇，所以其外传《超时空英雄传说2：北方密使》继承了前作的剧情，并加入了双角色的新颖方式来演绎一段新的传奇，不过它毕竟有着太浓的前篇的血统，而且剧情方面有所不足，所以感觉不如前者。
而对于《超时空英雄传说3：狂神降世》来说，我相信这会是一款跨越式的作品，对于宇峻作品的创意，我向来是敬佩的，但问题是宇峻这一步会走得多远。
从一开始，《超时空英雄传说3：狂神降世》就放弃了前作的剧情延续性，也许很多玩家会感到有些惋惜，但我个人倒觉得这种新生的生命力也许更能激动人心，一个大陆的统治被新力量所挑战，而战争使得生命闪光，当新旧力量在商讨和平之时，在最后瞬间来自暗黑的利刃斩断了橄榄枝，游戏就在这个史诗篇章的深深迷雾和忧虑中掀开序幕。
如果说看游戏介绍，我想《超时空英雄传说3：狂神降世》有一些特点还是颇为吸引人的，比如魔力源的设计，玩家所操控的角色将共享魔力；此外还有改进的半即时战斗系统，以及多战场同步战斗系统等等，这些设计无非就是要告诉玩家，这款游戏真的与以前的角色战棋策略游戏截然不同。
我也不怀疑宇峻的美工能力，其所刻画的角色造型都不错，但先前早期的作品画风显然深受日式画风影响，而之后虽然努力调整自己的画风，意图形成自己与众不同的风格，但显然还并没有完全把握神韵，所以《新绝代双骄贰》就显得精美有余，但缺乏画龙点睛之笔，这次《超时空英雄传说3：狂神降世》也是展示美工能力的尝试，但从宣传上来看，我个人觉得还是欠缺一些味道，希望作品推出后有所改观。
关于转职，对于策略战棋游戏来说也是颇为吸引人的，在《北方密使》中，游戏的转职系统非常自由，于是玩家可以轻易地练出超强的战士，那时感觉并非嫌敌人多和强，而是嫌敌人数量太少和太弱了，而新作中相信会作一定的调整，而且还有隐藏的职业存在，这又是个十分吸引人的卖点。
对于角色战棋策略游戏来说，由于任务关的设计方式使得其在剧情上应该远不如角色扮演游戏那么有内涵，所以如何将剧情设计得更吸引人就看宇峻的能力了，游戏的开头和背景都烘托得十分有气势，我想其中应该不乏神来之笔。至于游戏角色之间的情感线，由于剧情关不会太长，所以如何演绎男女主角的情感，也并非那么容易——相对来说，我一直非常喜欢《超时空英雄传说2：复活魔神》的情感线，我玩了两次，情感上都偏向于依娃，而这种情感的偏向是非常自然的。
至于《超时空英雄传说2》中所有好的因素，如多剧情多情感多选择多线发展，完整和自由的转职系统，以及炫丽的情侣合技都会出现，隐藏关自然也少不了，玩家随着一些隐藏的关系和剧情还能获得某些称号，我想这款游戏可能会让很多玩家投入精力的。
其实宇峻所制作的游戏在策划和策划细节上是不遗余力的，所以即使是游戏的攻略，可能就可以出厚厚的一本书，而想完全领略其中细节，你可能还得投入很多的时间，但这也正是一款游戏的成功标志，在我们已经无法在欧美游戏中领略角色的乐趣时，中文游戏正好在这个缺口上能大展拳脚。
在2000年，有两款中文游戏是我第一时间会购买的，一款是《轩辕剑叁外传：天之痕》，另一款就是《超时空英雄传说3：狂神降世》，不过很可惜的是，后者已跳票到2001年3月发行。中文角色战棋策略游戏的前一个高峰是其前作创下的，我想它应该会突破自己的，所以我相信这款游戏值得等。
原菲尼安领主的小儿子。在三年前菲尼安失陷，带领着部分贵族及难民逃出之后，就舍充了原来的姓名，使用克荷林这个名字，以隐藏自己的过去。在各种姻缘际会下，以高超的剑术领导着一群朋友，在中立岛米米卓瑞成立了血狮鹫佣兵团。
克荷林在山夫依德山林结识的少女，异族风格的服装与饰品烘托出她神秘的气质。她从未向克荷林提及自己的身份，但不知为何她似乎很早以前就认识克荷林。在结识克荷林之后，为了不为人知的理由，执意待在血狮鹫佣兵团中。
拥有强大魔力的小女孩，外表天真活泼，但经常说出许多超龄成熟的话，对穆伦教的教义十分执着，因此对教廷的许多做法十分不满。

}
