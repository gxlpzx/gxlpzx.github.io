{商务快车Info-Express}

{ 你想在互联网上发布个人求购信息吗？你想在互联网上发布公司的产品信息吗？你想迅速获取互联网上的商业信息吗？Info-Express能够轻松地帮你办到。
一、 软件简介
Info-Express是深圳商讯网信息有限公司（http://www.suminfo.com）出品的一款傻瓜型互联网营销软件，它不但能够帮你在8848、中华网、阿里巴巴等国内上百个商业站点和BBS上发布各类商业信息，而且还能够为你在国外的各大商业网站上发布商业信息。它强大的搜索功能可以迅速为你搜索到国内外上千个商业站点、BBS上指定的商业信息和相关公司的E-mail地址，并且能够把这些搜索到的信息下载到你的硬盘上供你仔细研究。另外，它还具有非常智能化的信息发布功能，能够自动区分简体、繁体、英文站点，为你在不同的站点上发送不同语言的商业信息。此外，Info-Express还允许用户通过自定义的方法把没有被Info-Express列入的站点加入搜索列表，以满足用户的特殊需求。
Info-Express是一款商业软件，最新版本V3.5，文件大小5.93MB，试用版本可以免费使用55次。
二、系统设置
软件启动后，首先弹出的是商品类别选择窗口，这里有近30个商品大类，不过每个用户只能够选择其中的两个类别，而且一旦选定后就不能更改了。把需要使用的类别添加进列表框中，然后点击“确定”按钮，程序的主界面就出现在我们的面前（如图），同时还会出现一个“系统设置”对话框，Info-Express只有在进行完系统设置之后才能开始工作。系统设置对话框一共有5个标签页面。
1.系统参数：在这里你需要设置Info-Express并行工作的线程、连接超时时间、错误重试次数，还需要设置发送邮件服务器的地址（SMTP）以及搜索信息的保存路径。如果你是通过代理服务器上网的，在这里也可以做相关的设置。
2.用户信息：此处需要填写以下的一些项目：姓名（中英文）、性别、出生年月、职业。
3.公司信息：填写准备发布的信息内容（中英文）以及公司的名称。
4.签名/主页：可填入自己的中英文签名，使用签名可以使发布的信息更具个性化色彩，更具吸引力。如果你有主页，也不妨把地址填在这里，这样可以使客户多一个渠道了解你或者贵公司。
5.联系方法：必须如实地填写有关的联系信息，使客户能够方便地跟你联系。需要填写的项目有：E-mail地址、电话、传真、邮编、详细联系地址。
三、 软件的使用
1.发布信息
系统设置完成之后，我们就可以把在“系统设置”中撰写好的商业信息发布到互联网上去了。点击工具栏“发布信息”按钮，在“发布分类”列表框中展开需要发布的信息类别，例如家用电器，选择其中的一个子项，又例如视听器材，这时在右边的列表框中就会出现该类别信息所允许发布的站点。如果你是该软件的正式用户，就可点击工具栏“全部选中”按钮，Info-Express会帮你向该类别中所有的站点发布信息（如果需要在国外的网站上发布信息，只须按下“转到英文栏目”按钮即可）。如果你不是正式用户，那就只允许一次选择5个站点（其中五星级的站点只能选一个）。当你选择某些站点的时候，可能会弹出一个注册信息窗口，这是因为有些大的站点只允许该站点的注册用户在站点上发布信息，因此，要想在这些站点上发布信息，首先必须成为这些站点的注册用户。在这里有两种注册方法：自动注册和人工注册。对于大部分站点，Info-Express能够替用户自动注册，你只须点击一下“自动注册”按钮，其余的一切Info-Express都会帮你办妥。如果有些站点自动注册没有成功，你就需要进行人工注册了。人工注册也非常简单，Info-Express会迅速地把网站中的注册页面下载下来供你填写。注册完毕之后，你就可以在这些站点上发布信息了，不过有个别的站点可能还需要用户通过E-mail确认身份。站点选择好之后，点击“下一步”按钮，出现信息提交界面，信息的正文就是你在“系统设置”里已经填写好的内容，在这里你可以对信息的内容以及信息的主题进行重新编辑。另外，你在这里还需要填写信息发布的说明。信息发布说明的内容包括：关键字、信息类别、有效期（所发布信息的有效期限）。一切准备就绪之后，点击“发送”按钮，下面就让Info-Express为我们忙了。信息发送完毕后，Info-Express会给出一个详细的发送结果报告，告诉我们哪些任务发送成功，哪些任务发送失败，这时候直接点击发送成功的项目，我们就可以在各大商业站点的网页上浏览自己刚刚发布的商业信息了，真正的所见即所得。
2.搜索信息
按下“搜索信息”按钮，在分类列表中选择希望搜索的类别，右边的列表框中会出现该类别下所有的网站，可以通过转换按钮切换中英文网站，选中需要搜索的站点，点击“下一步”按钮，出现搜索提交界面，在这里需要填写搜索的关键词、设置搜索网页的层数以及是否下载详细内容。另外，还需要填写搜索信息的说明，具体内容和发布信息说明基本相同，最后点击“搜索”按钮，Info-Express很快就会为我们把需要的信息从指定的网站上下载下来，同时还会为我们找出许多与这些信息相关的E-mail地址，以方便我们同这些公司进行业务联系。
另外，Info-Express还提供了邮件的群发功能，这样用户就可以把一封写有详细商品信息的邮件同时发往一些相关的公司。由于这些邮件是有的放矢（Info-Express帮我们搜索到的客户E-mail地址），效果自然不会差。不过，未注册的用户不能使用该功能。
}
