{电脑上用CD唱卡拉OK}

{ 如果你因为自己的“独特”歌喉而不敢在朋友面前高歌的话，那么Rave Karaoke这个软件就很适合你了。在CD-ROM上插入一张音频CD后利用这个软件的消原唱功能再配合电脑上的麦克风和音箱，你就可以在家里关上门自个儿练习唱卡拉OK了，当然如果你认为已经练到“绕梁三日”的境界的话，大可以把声卡的输出端口接到音响上，唱给全世界听。
下面我们先来看看这个软件对机器的要求，首当其冲的当然就是你的电脑要配有麦克风、音箱、和 CD-ROM。如果你想效果好，麦克风和音箱都不能马虎的，特别是麦克风千万别用那种几块钱的电脑麦克风，最好是那种音响用的，效果会好很多。音箱是低音炮和5.1音箱的最好，没有也只能将就了。其次 CD-ROM 最好高于 8 倍速度，DVD和CD-R都只要能放CD唱片就可以了。另外一个重要的就是声卡了，要使用这个软件，你的声卡必须支持全双工，否则就没戏了。什么是全双工？简单来说就是声卡可以一边录音一边放音。市场上比较常见的S90、创新128等型号都支持双工，但像 ALS007之类的老声卡就不支持了，大家要搞清楚哦。
Rave Karaoke 这个软件目前最新版本是1.5版本，可以到http：//www.raveanddodger.com下载，文件大小是 2.7M。软件的安装和一般软件没什么区别，按几次“Next”就可以顺利完成整个安装过程了。运行软件后，可以看到软件的界面。
常用的功能按钮都已经在图上加了注释，现在来详细介绍一下。首先在CD-ROM里放入CD唱片，这时软件会尝试读取这张CD的一些描述信息，如唱片名字、作者、歌曲名字等，如果找不到，你也可以按下上面那个Download CD Info键来让 Rave Karaoke 连上Internet到CDDB去找，就像Winamp一样。但一般我们还是自己动手写上去更省事。为此你可以在“Options”上面按一下鼠标，跟着在弹出的选单里面选“Edit CD Information”，接着软件会调用Windows的写字板来给你输入唱片名字、作者、歌曲名字等信息，编辑完后保存起来，那么以后你再用这张CD来唱卡拉OK的时候，Rave Karaoke 就可以自动调出并显示这些信息了。另外，在这个弹出的选单上你还可以选择很多功能，比如抓音轨、重唱、编辑歌词、调用调音台、和输入注册码等。
现在我们来看软件界面上的功能键用法。软件界面的左边圆圈里面有四个功能键，分别是播放、停止、后退和快进。然后是这个圆圈下面的三个按钮，左边的是录音键（Record Me），中间的是消原唱键（Reduce CD Voice），它可谓是这个软件最有用的功能了，按下这个键后，CD唱片里面歌手的声音将会大大的被削弱，只剩下伴唱和乐器的声音，这样你就可以无须为没有卡拉OK专用的伴奏CD而发愁了。接着右边这个是回放键（Play Me），配合左边的录音键实现录音的回放功能。在这消原唱键（Reduce CD Voice）的下面有两个音量控制滑杆，左边是 CD 的音量右边是麦克风的音量。利用这两个滑杆，你可以根据需要分别调整你的歌声伴奏音乐的音量大小。
接着我们来看看软件界面右边的这八个大按钮，软件把这八个大按钮称作“Mic FX”，我们姑且称之为音效滤镜。左边这四个音效滤镜是混响效果滤镜，分别是“Concert Hall”、“Shower”、“Sports Arena”和“Grand Canyon”，通过这四个滤镜你可以为你的声音加入回音、混响等效果。右边的四个音效滤镜非常有趣，它们可以把你的声音变成你自己也不认识的怪物声音，就像电影里面那些会说话的机器人、怪物发出的声音一样，非常滑稽。另外，这八个音效滤镜的中间也有一个音量滑杆，你可以通过它来调整滤镜效果的强弱。
利用Rave Karaoke，你可以不需要那些专门的卡拉 OK VCD 就可以在电脑上利用 CD 唱片来唱卡拉 OK 了，节目源从此丰富了很多，如果你也喜欢闲来唱上两首的话，那么Rave Karaoke最适合不过。但是，提醒一句如果该软件没注册，无论是消原唱还是音效滤镜都是无效的。
}
