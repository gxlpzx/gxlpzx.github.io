{光盘刻录应用汇粹}

{ 一、WinOnCD的应用
除了在前面介绍的Easy CD Creator以外，WinOnCD也是一个非常著名，而且功能强大的光盘刻录软件，支持多种资料格式，支持几乎所有的刻录机产品。WinOnCD可以从很多刻录机的配套软件中获取，安装程序非常简单，在完成安装之前，安装程序会提示插入一个Winoncd的驱动程序盘，可以从安装光盘的有关目录中找到。启动WinOnCD 3.6以后，系统会给出一个选择新项目窗口，非常方便用户选定相关的任务操作。下面就通过四个实例，来介绍一下它的使用方法。
1.快速操作入门
利用Winoncd可以实现光盘的复制，启动WinOnCD 3.6后会出现“New Project”对话窗口，选择“Copy”标签页面并双击“cd copy”图标，打开复制光盘的主接口窗口，如果需要刻录多张光盘，可以单击“DISC”按钮，在“copies”中设定复制的数量。完成设定以后，就可以单击“Record”按钮开始复制，这时系统会自动扫描并分析源光盘的内容格式，如果扫描无误，系统就会开始复制光盘了。
Winoncd制作资料光盘在同类软件中是最方便的了，操作性也非常好。下面就来体验一下Winoncd的无穷魅力吧。
在“New project”窗口中，选择“Data”标签页面中的“CD-ROM ISO 9660/Joliet ”图标后，按“确定”按钮打开制作资料光盘窗口。在窗口上方的源资料窗口中选择所要刻录的文件或活页夹，然后拖到下方的“Editor”窗口中，并放置在相应的目录（比如根目录）中。当选定的文件或活页夹包含有长文件名，WinOnCd会出现“Conflict”对话框，这时选中“Apply to all”项并按下“Ignore”按钮，就会使长文件名不会变成乱码了。
另外也可以通过鼠标右键菜单命令对当前目录及文件进行各种编辑操作。资料准备好以后，单击“Disc”打开刻录参数设定窗口，在“Fixation”中提供了两个单选项，其中“Create Multisession CDcurrent session is closed an additional sessions possible”选项表示使用Multisession方式刻录，这样做的好处是，可以保证日后还可以继续刻录资料，直到刻满为止，使光盘空间不至于浪费。如果选择“Close disc.sessions cannot be added lateer”选项，则表示关闭光盘，今后将不能再写入任何资料，一般当光盘的容量将要刻满的时候，可以设定该选项。
另外，在“Test write”中提供了三个选项，如果选择“Simulate write only”，那么只进行模拟资料刻录，而不真正向光盘中写入资料。选择“Simulate writing once then write if successful”则保证只有当模拟刻录成功以后，才能真正写入资料，建议选择此项，可以提高刻录成功率，缺点是刻录过程较慢。“Write immediately”表示不进行模拟刻录，而是立即写入资料，相对于第二项，刻录速度较快。
在“Output”选项中还可以对写入Write方式进行选择，可以直接写入刻录机或者制作成镜像文件Image File类型进行。此外，还可以在“Speed”中对当前刻录机的写入速度进行测试。除此以外，还建议选择“Write On fly”方式进行刻录，这样可以使刻录更加高速地进行。设置完成以后，就可以按下“Record”按钮进行刻录了。
如果需要向一张已经刻录有资料并且没有关闭的光盘追加资料，只需在“New project”窗口的“Data”标签页中选择“Append session ISO 9660/Joliet”图标，然后再按“确定”按钮，Winoncd就会自动读入已刻录有资料的光盘的内容，其余操作和上面的方法都是相同的。
2.一片双用
一片双用的光盘，在CD唱机中就是音乐光盘，在CD-ROM中则成为资料光盘，现在大多数能播放歌曲又有屏幕保护功能的光盘就是采用这种格式制作的，好处是可以充分发挥光盘的利用率。在Winoncd中，在制作一片双用的光盘方面有其独到之处，较Easy CD Creator等同类功能要强大得多。
在“New project”窗口的“Audio”页中选择“CD Extra”图标项目，打开制作接口窗口。在源文件管理窗口，将所要刻录的音轨文件和数据文件分别拖到“Tracks”窗口的“Track”和“ISO 9660”轨道中，Winoncd会自动将音轨文件排放在前面，以符合CD Extra格式的要求。
Winoncd还特别提供了对音轨文件的编辑功能，而且都非常实用，无需调用音轨编辑软件就可以实现一些特技和编辑功能。在“Track”窗口中，右键单击Track轨道，在弹出的快捷菜单中执行“Editor”命令即可切换到“Editor”窗口，在这里可以对刻录光盘的参数格式进行设置，还可以对音频轨道进行一些特技处理。系统提供了包括“General”、“Edit”、“Play”、“effects”四个大的项目设置功能，比如单击“Fade in”或“Fade out”按钮，可以实现对声音的淡入和淡出设置，单击“Volume”按钮可以设定淡入和淡出时音量变化的百分比等。除此以外，还可以单击“Play”按钮试听，单击“Export”按钮将当前选定的音轨导出为WAV等音频文件格式，单击“Record”按钮进行录音设置等。完成音轨的设置以后就可以切换到“Disc”窗口，进行刻录了，同时需要注意的是，Winoncd在刻录CD Extra光盘时，会强迫刻录方式设为Close cd，也就是说，日后将不能再向其中添加任何资料和音轨文件。
二、用Nero Burning Room制作Mac与PC资料共存的光盘
Nero又是一个功能非常强大的刻录软件，提供了包括光盘对拷、刻录资料光盘、制作音乐光盘等多种功能，而且还特别的支持Mac（苹果机）的资料格式，可以直接将Mac格式的硬盘资料刻录到光盘中，这种光盘被称为Hybrid CD。
首先启动Nero打开主接口选单，在这里我们发现，Nero提供了包括和其它同类刻录软件几乎相同的功能，而且还特别提供了刻录“Hybrid”格式光盘的功能，选中“CD-ROM Hybrid”图标，然后在列表区域选择HSF分区资料，然后单击“New”按钮进入制作主接口窗口，这时Nero已经装入了HSF格式的资料信息。将PC格式的资料信息拖放到刻录窗口中，然后按下刻录按钮就可以开始刻录Hybrid格式的资料信息了。
三、要把光盘变硬盘
听起来似乎有点神了吧，要把光盘变硬盘？嘿嘿，其实只要有了刻录机，实现起来也是很简单的，但需要Direct CD的支持。Direct CD 可以使用户在资源管理器中，向CD-R/RW光盘中直接写入和删除资料，就如同操作软盘和硬盘一样方便。
Direct CD用写资料包技术把资料写入CD-R或CD-RW盘，此文件系统提供对CD-R或CD-RW驱动器访问的驱动器代号。比如说吧，如果你在Word中编辑了一个文档，就可以直接将文件保存在光盘中，除此以外，还可以实现在光盘和硬盘之间的资料复制、移动等多种操作。Direct CD 2.5只能用于资料光盘的刻录，而不能识别Audio Track格式，所以不能用于刻录音乐光盘。和其它刻录软件不同的是，使用Direct CD刻录必须先将光盘格式化才能使用。
首先将空白碟片放入刻录机驱动器中，这时会出现 Adaptec Direct CD向导欢迎信息，单击“下一步”按钮出现“格式化光盘”窗口，这时Direct CD会自动分辨放入刻录机中的碟片类型是CD-R还是CD-RW，如果是对已格式化的 CD-RW 盘进行格式化，可以选择“快速”或“全部”。完成以后单击“下一步”出现“给光盘命名”窗口，光盘名最长可为 11 个字符，最后单击“完成”，光盘格式化开始，完成后，会出现“Direct CD 盘已准备好”信息提示窗口，在系统托盘区会出现一个小红点，表示光盘已准备好，可以写入资料了。
光盘被格式化为 Direct CD 盘后，添加资料的方法很简单，大致包括以下几种方法：
在资源管理器中把文件直接拖放到CD-R图标上；
在 Windows 应用程序的文件菜单中选择“另存为”，并选择CD-R或CD-RW驱动器盘符；
可以使用Window的“发送”命令将资料存放到碟片上；
使用Windows中的 MS-DOS 命令，实现拷贝、删除等。
无论使用上述的哪种方法来添加资料，都如同在硬盘上操作文件一样简单。完成刻录以后，绝不是简单地从CD-R或CD-RW驱动器中弹出光盘，你需要按一下刻录机的“Eject”退出键，或者右击系统工具栏的右边那个Direct CD的小图标，就会出现“弹出”窗口，Direct CD会根据驱动器中光盘的类型和使用目的，提供两种选项。
1.UDF格式退盘
Direct CD刻录软件采用包刻写方式和UDF文件格式进行光盘刻录，采用UDF格式退盘时，刻录机不会关闭数据段就会直接退盘，而不在用户资料区刻录导入区和导出区，所以这种光盘只能在运行Direct CD的刻录机上被读取，而无法在标准的CD-ROM驱动器上读取。
2.ISO9660格式退盘
如果打算将弹出的 Direct CD刻录碟片在其它计算机的标准驱动器上读取时，必须选择为ISO 9660格式退盘，这时刻录机会关闭数据段再行退盘，退出的光盘可被普通CD-ROM驱动器读取。如果同时选择“Protect discsoit can not be written to again”选项，那么刻录机会关闭光盘后才退盘。关闭数据段后的光盘可以被继续追加新的资料或进行删除操作等，而关闭光盘后则不行。另外，此格式的信息在光盘第一片段中占有 22 MB 空间，在每一附加片段中占有 13 MB 空间。只有 CD-R 盘才能以 ISO 9660 格式被关闭。CD-RW 盘不支持 ISO 格式。
如果再次写入更多资料到Direct CD盘片，需要将Direct CD 盘插入CD-R 或CD-RW 驱动器，等待出现“Direct CD盘准备好”窗口以后，单击“确定”就可以给光盘追加资料了。
删除Direct CD盘片中的内容，可以直接在Windows 资源管理器中删除。需要注意的是，只有 CD-RW 驱动器能读、写和删除CD-RW盘的资料，从CD-R 盘中“删除”文件，只能使这些文件在文件系统中不可见，并不释放磁盘空间。
