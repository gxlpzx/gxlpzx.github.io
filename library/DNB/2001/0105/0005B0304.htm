{图标大师E-Icons}

{ 大家是否厌倦了Windows自带的图标呢？是否想过换成自己喜欢的图标，使桌面更具有个性呢？但Windows有很多图标都不可以随意改变的，有时要修改注册表，但注册表可是危险的东西，一不小心可能使整个系统崩溃，但是只要在“图标大师”E-Icons的引导下，你就可以安全放心地把系统的图标来一个大修改。下面我们简单介绍一下它的使用方法。
1. 当打开软件后，程序会让你选择当前使用的系统，选好后点击“ok”就可以进入软件的主界面了，有一个菜单栏，它的下面是快捷工具栏，我们可以通过快捷工具进行一些有关文件打开、保存、更改之类的操作。在快捷工具栏下有三个标签，它们分别是“Change Icons”（更改图标）、“System”（系统）和“About E-Icons”（关于E-Icons）。
2. 在E-Icons的主界面下方我们可以看到两个选项框，左边的选项框列出了可以进行图标变化的选项，包括“Desktop”（桌面）、“Start”（开始菜单）、“Driver”（驱动器）、“Folders”（文件夹）、“File”（文件）“Others”（其他）。这里我们双击“Desktop”在弹出的菜单中找到一些不能直接改变图标，如“我的电脑”，“我的文档”，“回收站”等。
3. 我们以更改“我的电脑”的图标为例，单击“我的电脑”（My Computer），右边的选框改变为一个工作区，这里放在最上的是“我的电脑”本来的图标，下面就是可选择的图标，选好自己满意的图标后按下“Replace Icon”（替换图标）就可进行图标的更改了。
4. 最后，按下“Apply”键，再点击“ok”就可以了。候选图标中没有你喜欢的，只要按下“Browse”键，在系统中浏览并选择你所需要的图标，再按上面的步骤就可以了。程序还提供了一个主体档案（Theme），在那里有10种方案以供选择。
在“System”这个标签里，我们可以为回收站改改名字，还可选择是否显示大图标、快捷方式下是否显视小箭头、使用多少颜色数等等。“About E-Icons”是介绍这个软件的相关信息，这里不多讲了。希望大家在这位大师的带领下，让自己的Windows变得更有个性。该软件可以在Win95/98/NT/2000等操作系统下使用，它的下载地址是：http://download.sina.com.cn/scgi/down.pl﹖key＝E％2DIcons。}
