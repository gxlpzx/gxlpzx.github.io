{2001年DIY市场的冲击波}

{ DIY的爱好者也许近期都会发现，国内各大计算机硬件网站和一些IT专业媒体上频频出现i815EP主板只售人民币815元的宣传广告，捷波品牌给大家留下了深刻的印象。
i815EP芯片组是Intel公司在2001年推出的一款新品，据笔者所知基于该芯片组的主板，捷波的零售价是最低的，为此特地采访了捷波（中国）有限公司的总经理钟浩先生。
记者：恕我直言，在DIY市场捷波的品牌尚无知名度。作为一个新的品牌，进军DIY市场难度是相当大的，你是否认同这一点？
钟浩：首先我要说明的是捷波并不是一个新的品牌，我们在IT行业也是一名老兵了。捷波公司1986年成立于中国台湾，是一家专业的主板及界面卡制造商，产品主要销售往欧洲和北美。1995年，捷波资讯（中国）有限公司成立，六年来我们的业务主要是OEM。计算机市场主要由三个部分组成——国际（内）品牌机、地方自有品牌机和DIY机。三者的市场份额一直摇摆不定，捷波公司前几年的市场策略是集中进军品牌机市场，所以在DIY市场露脸的机会不多，知名度也就无从谈起了。2000年末，我们注意到品牌机和DIY机的市场占有份额已经基本稳定，据第三方调查资料来看，DIY机在中国市场占有率是35％，即市场每年需求约200万主板，这个市场对任何厂商来讲都相当有吸引力，考虑到捷波在OEM市场的业务已进入良性增长期（主板的年销量已达22万片），所以开拓DIY市场成为了我们今年业务成长的重点。
记者：此次捷波推出这款零售价相当低的i815EP主板，是否意味着捷波要牺牲利润或品质，以低价在DIY市场开展恶意竞争？
钟浩：你有这种想法很自然，如今在IT业界，推陈出新的速度相当快。大凡新品推出，期间无不经历上市时的高价到被其他新产品替代后的低价过程，所以多数DIY用户由于经济承受能力有限，往往只能望梅止渴。捷波深刻理解这一点，故推出高性价比的产品是我们切入DIY市场的开始。
当然，我们决不会以牺牲利润或以低质的产品去开拓市场。任何企业如果这样做的话，都无异于自杀。捷波成立以来，一直秉承“稳中求速度”的经营策略，一步一个脚印地发掘市场，发展客户。之所以捷波能以人民币815元震撼性的零售价格提供i815EP母板，这是我们多年来苦练内功的成果。捷波公司在产品营销上，已经建立了一套比较完善的管理制度，不仅能顺应趋势，及时研发出最新的产品，还能有效地控制产品成本和销售成本。举个例子，物流是营销活动中的重要一环。国际上一个好的销售公司能将物流成本控制在1％之内，而捷波目前能将物流成本控制在接近国际水平。我们将这些多出来的利润让利于最终用户，所以人民币815元的价格是十分合情合理的。
记者：DIY市场是群雄纷争之地，捷波公司以前擅长的是OEM，对于DIY市场的开拓，面对众多的竞争对手，捷波有何应对之策？
钟浩：DIY市场有着相当大的用户群，自然是各大主板厂商的必争之地。从实力上划分，捷波处于中上水平，所以我们的销售策略也是据此而制定的。捷波针对DIY市场提出的概念是向DIY用户提供“度身定做”的高性价比的产品。此次我们推出的i815EP主板只售人民币815元的活动，就是我们推广此概念的第一个有力明证。我们在2001年的目标是进入国内DIY主板市场的前六名，我们将会有计划有步骤地开拓我们的市场。
农历新年的钟声已经敲响，记者感觉到捷波在DIY市场擂起了战鼓，今年捷波是否能成为DIY市场的一匹黑马？让我们拭目以待。
}
