{跨入新世纪的南京电脑市场}

{ 珠江路科技街位于南京中心城区玄武区内，以珠江路为主线，西起中山路，东接黄埔路，并向南、北延伸，是南京市科技人才、高新技术企业、民营科技企业最为集中的区域。珠江路周边高校、科研院所云集，有东南大学、南京大学、南京航天航空大学、海军指挥学院、中科院南京分院、熊猫集团等，科技优势十分明显。珠江路科技街以经营电子电脑、系统网络、软件开发和通信产品著称，联想、方正、IBM、Intel、COMPAQ、NEC、东芝等国内外著名的电脑厂商纷纷加盟珠江路。目前沿街两侧聚集电脑公司近千家，拥有自主知识产权和品牌的电脑公司13家，16个电脑市场，5个通讯产品市场，经营总面积近20万平方米，珠江路科技街已成为华东地区最大的电子电脑产品集散地。全街电脑公司职工约为7400人，大专以上学历达90％，其中35岁以下约占70％。
由于珠江路属于市中心地区，交通便利，人气旺盛，因此汇集了数千家大大小小的电脑商家，密集程度非常高，以每百米20家企业和年产值50亿元的骄人业绩稳居华东各大电子市场之首，辐射范围也比较广，江苏本省和安徽省大部分地区的电脑配件货源都出自这里，而且货源种类齐全，新品上市和价格变动的反应速度也非常快，基本上能够做到与国际接轨。此外，各大电脑配件品牌的代理商非常齐全，但是从总的市场规模来说，比起北京、深圳等地的电脑市场还有一定的差距，而且电脑商之间的竞争已经到了白热化的状态，电脑配件的利润已低到了难以想像的地步。规模较大的公司凭借雄厚的实力，压制规模较小的公司，导致珠江路每年都有数十家公司倒闭，但是前仆后继的精神在这里体现得尤为明显，每年又会有数十家新公司成立。
长期以来，电子商城是珠江路经营的主要方式之一，其中首屈一指的就是位于珠江路中段的雄狮电子城。两年前这里还是一家长途汽车客运公司，自客运公司把它改建为电脑市场后，短短两年的时间，它就发展成为南京乃至整个华东最大的DIY市场了。雄狮商城为二层楼的结构。一楼分为前场和后场，前场的前半部全是柜台式结构，经营各种芯片、板卡散件、外设、显示器、机箱、耗材、网络产品、零配件、书籍等，前场的后半部为柜台和包厢混合式结构，主要经营一些国产非名牌散件、耗材、线缆、二手产品、坏件维修等。包厢主要以散件装机为主，兼卖散件、外设、网络产品，大多在前台设有柜台卖散件并吸引整机顾客。一楼的后场在另一幢楼内，大多数为包厢和办公室结构，多为厂家的办公室或库房，二楼基本为包厢式结构，辅以少量柜台。近年来，南京市玄武区政府大力开发珠江路，去年4月，在珠江路的最东端新开了太平洋和21世纪两家电子广场，装潢和布局都比雄狮上了一个台阶，但是半年来这两家商城一直人气不足，仍然不能和雄狮电子商城相比，这与人们长期的购买惯性有关，看来两家新商城想要人气东移，尚需假以时日。
另外，珠江路上的电脑商城正在经历着更新换代的过程：新秀21世纪广场现在已经成为赛博数码广场南京店，光阳电子城已经被强强联手的宏图三胞吃下，以宏图三胞电脑资讯广场的新面貌出现。位于珠江路和太平北路交汇处的联想科技商城更名为联想中关城，而且很快将被拆除，在原城兴建珠江路上最大的电脑商城——百脑汇，届时从现在的中关城到玄武区政府将都是它的“地盘”。此外，太平洋电子城也将逐渐淡出卖场型的经营方式，而向服务型方式转变，据说将划出一块营业面积作为珠江路上第一家麦当劳分店，楼上则以写字间租赁业务为主要发展方向，目的是避开雄狮电子城进行得很成功的传统经营模式。而当年为珠江路的发展立下过汗马功劳的几大商城已经相继“陨落”了，如玄武科贸、新世纪电子城等等，现在要么成为盗版碟集散地，要么成为二手交易市场，已经难以重现当年的辉煌了，也许这就是残酷的市场竞争法则所决定的吧！
当前，珠江路科技街的发展正面临前所未有的新机遇，在省、市的统一领导下，将进一步贯彻落实全国科技创新大会精神和科教兴市战略，充分发挥珠江路科技街周边地区人才、知识和智力优势，积极借鉴国内外科技园区成功经验，大力发展以信息产业为核心的高新技术产业，促进科工贸一体化发展。用5～10年的时间，把珠江路科技街建成为南京市电子信息产业的孵化基地、创新基地、产业化基地和高素质人才的培养基地。通过科学合理规划，精心组织建设，逐步建成基础设施完善、信息网络发达、企业富有活力、现代气息浓郁的高科技园区，使它更具有对科技创新人才的凝聚力、对国内外投资的吸引力、在国际市场上的竞争力。2000年10月，第二届珠江路电脑节暨玄武区金秋经贸洽谈会成功举行，与世界科技城市联盟大会（WTA）科技成果展示会结合，使珠江路在全国乃至世界范围内的知名度大大增加，其中最主要的一项成果就是广邀IT行业精英，对珠江路进行全面“扩容”，如投资1000万美元的蓝天科技城已经在“电脑节”期间开工，在珠江路西段将建46层的“新世界中心”以及珠江路“硅谷大厦”，在文德里地区将建2.26万平方米的珠江路“信息产业大厦”，并把浮桥-竺桥-逸仙桥一段沿河两边的8米控制带建成步行街式的“网络景观走廊”等等。
从珠江路目前的发展状况来看，跟北京中关村还有一定的差距，但是相信凭借政策和资金上的扶持，在未来的一至两年内实现“北有中关村，南有珠江路”将不再是遥远的梦想，而是每个南京人都触手可及的现实！
}
