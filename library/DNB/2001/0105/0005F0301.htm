{Adaptec Easy CD Creator轻松上手}

{ 当我们有了刻录机，并准备好了CD-R和CD-RW盘片，接下来就要进行实际的刻录操作了。不过，还不要着急，要知道，刻录光盘，还要选择一款合适的刻录软件，Adaptec Easy CD Creator4.01中文版不但功能完善、强大，而且对大多数刻录机都有良好的兼容性，并提供了刻录向导模式，操作起来也非常简单，刻录性能稳定，是备受大众看好的刻录软件。
另外，这里还要说明的一点就是，无论你是使用CD-R还是CD-RW，如果没有特别说明，在接下来的刻录操作中都是一样的，两者之间唯一的不同就是能否重复写入。
一、刻录前的注意事项
1.关闭后台程序
在正式刻录之前，为了保证刻录操作的顺利进行，需要先检查系统是否已经正常启动，是否设置了影响刻录过程的常驻程序，或者定时自动执行的程序，如果有，就先将它们关闭吧。因为刻录机在正常刻录过程中，需要系统不间断的提供刻录所需要的资料源信息，而驻留程序，都有可能在刻录过程中被执行，从而中断刻录资料的正确传输，造成缓冲区欠载Buffer Underrun错误，使刻录失败，导致盘片损坏、报废。
首先应该关闭光盘“自动插入通告”功能，禁用光驱自动运行功能。在“我的计算机”图标中单击鼠标右键，选择“属性”命令，切换到“设备管理”标签页，展开“CD-ROM设备”项，选定要作为资料源的光驱，按下“属性”按钮，然后选定“设置”标签，取消“自动插入通告”选项就可以了。
其次，还要关闭会影响刻录过程的一些应用程序，比如常驻内存的防毒软件在刻录过程中，会侦测传输的文件，启动预设的自动扫描功能，对内存、驱动器FAT表或磁盘文件检查，这就很容易造成刻录工作彻底失败。
另外，Windows的任务管理程序和计划任务程序会依设定时间启动磁盘扫描或磁盘碎片整理程序，这样，系统必须停顿相当长的时间来配合。
再就是屏幕保护程序了，如果启动以后，会从硬盘读取资料，造成中断、暂停刻录程序所需资料的传输。因此，以上一些程序务必先行关闭，才能保证刻录过程的正常进行。至于具体方法，就不一一赘述了。
2.准备足够硬盘空间
刻录光盘除了要关闭光驱的自动插入通告和一些常驻程序之外，还必须保留足够的硬盘空间。在实际刻录之前，光盘刻录软件会将所要刻录的内容转换成光盘的存储格式，并将一些资料暂存在硬盘上，如果硬盘的预留空间过小，会直接影响刻录的稳定性。此外，如果磁盘有错误或碎片过多，会造成资料无法正确读取，导致刻录失败，所以还需要对硬盘进行磁盘扫描和碎片整理，保证刻录工作能正常进行。
二、复制光盘
也许很多朋友购置刻录机的一个重要用途，就是复制一些光盘方便自己的使用了。一些好的软件光盘、CD、VCD、歌集等，从朋友那里借来，总是要还的，如果能够复制一张和原来一模一样的，岂不是自己也有得用了么？
Adaptec Easy CD Creator提供了一个复制CD的工具CD Copier，可以一次复制单张或多张完全相同的CD。它可以复制的CD类型包括资料光盘、音频光盘、DirectCD光盘、混合模式、CD Extra、照片CD和视频CD等。下面就来看看它是如何操作的吧。
首先在光驱中放入源光盘，并将一张CD-R放入刻录机中，然后单击“CD Copier”命令按钮，打开“CD Copier”对话窗口。
在“复制自”中选择资料源光驱，并设定好“刻录到”的刻录机类型。打开“高级”标签页面。
在高级页面标签中，设定“CD刻录速度”为当前刻录机的写入CD-R的最高速度，然后选择“测试并复制”单选项，可以使刻录过程先进行测试，如果因为测试时不能以高速刻录，程序会自动提示降低刻录速度，而不会使盘片报废了，虽然这么做会多耗费一些时间，但可以提高刻录成功率。如果要复制多张光盘，可以选择“只复制”单选项，这时就可以在“副本数量”中设定要复制的数量了。另外，如果复制多张光盘，最好选择“先将源CD复制到硬盘驱动器”选项，可以保证资料在传输上更加快速和稳定，当刻录结束以后，硬盘上的暂存文件会被自动删除。
完成设定以后，就可以单击“复制”按钮开始复制光盘了，这时程序会先进行模拟刻录，模拟成功以后会开始写入资料，并最终关闭光盘，这样以后就不能再向这张光盘写入资料了。
三、制作资料光盘
在前面，我们介绍了如何一对一地复制光盘，但是有的时候，我们需要刻录的资料并不一定是整张光盘的内容，有可能是硬盘上的资料，或者是从光盘中摘选的数据文件等，也就是说，由于待刻录资料的不确定性，决定了我们需要按照自己的要求来制作光盘资料，这时候，就需要用到Easy CD Creator的制作资料光盘的功能了。资料光盘的制作，也是我们日常刻录过程中，最常用到的操作。制作资料光盘，可以存储计算机资料，诸如硬盘上的文件和文件夹。与音频 CD 不同，资料 CD 只用于资料存储，无法在家庭或汽车立体声 CD 播放器上播放。
1.轻松刻录资料光盘
下面就来使用Easy CD Creator制作一张简单的资料 CD。在功能选择窗口中，选择“资料CD”选项，打开Easy CD Creator主接口窗口。
单击工具栏上“新建”按钮旁边的小箭头，并从下拉列表选择 “资料 CD”，这时资料 CD 版面就会出现了。将一张空白 CD-R碟片插入刻录机目标驱动器，在“资源管理器节”中，选择想刻录的数据文件或活页夹，然后单击 “添加”命令按钮，另外，还可以直接将需要刻录的资料直接拖放到“空的资料版面”中需要存放的目录中，这时在“CD版面”的右侧窗口就会出现添加的内容。重复此步骤，直到资料 CD 版面包含欲刻录的所有文件和活页夹，完成资料的添加以后，就可以单击“制作 CD”按钮，这时会弹出一个“CD 制作设置”对话框。
单击“高级”按钮展开高级设置卷帘窗口，在“目标设备”中，选择当前系统的刻录机类型，在“写入速度”中，可以调整刻录机刻录的速度，如果使用的是高速刻录机，将会有多种速度可供选择，需要注意的是，应该确保当前使用的CD-R光盘支持高速刻录，否则就要降低刻录速度，以提高刻录的成功率。如果需要刻录多张光盘，可以在“副本数量”中进行设置。
在“制作选项”中，共有三个单选项，“只测试”选项用来做传送要刻录的文件到 CD 刻录机的测试，而非真正写入CD。选择“测试并制作CD”选项，在刻录的过程中，首先对源资料进行测试，测试成功以后才能进行刻录，这样可以有效地提高刻录成功率，建议选择此项。“制作CD”选项，则是跳过所有的测试，直接进入CD的制作，这样可以提高制作流程的速度。
在“写入方式”中也提供了三个选项，选择“Track-at-Once”选项，则一次刻录一条轨道，以 Track-at-Once 模式刻录的轨道之间会有间隔，如果资料轨道之后是音轨，那么间隔则为两三秒，音轨间的间隔通常为2。在“Track-at-Once”选项下还提供了三个选项，选择“保持片段打开”选项，以后还可以将歌曲添加到CD中第一片段，需要注意的是，添加完所有的歌曲以后，如果要在音响设备中播放，必须选择“关闭片段”选项。如果选择“关闭片段并保持 CD 打开”选项，可以用来关闭片段，由于CD保持打开，所以日后还可以继续刻录追加资料片段，直到CD满为止。选择“关闭 CD”选项可以CD进行写保护，以后就不能再刻录资料到当前CD盘片了。
根据自己需要设定好各个选项以后，就可以单击“确定”按钮制作CD了。
2.编辑CD 版面
在以上的制作过程中，可以完成一个简单的资料CD制作，但是，很多时候，由于不同的要求，需要对目标盘的CD版面进行修改，下面，我们就来看看如何修改资料CD版面，比如创建、修改、删除文件及活页夹，达到所要刻录CD制作的要求。
在CD-R版面上，在根目录下创建一个新活页夹，可以在CD版面中，用鼠标右键单击根目录，在弹出的快捷菜单中，选择“新建/活页夹”命令，并为新活页夹命名为“我的资料”就可以了。
当创建了多个活页夹以后，在制作CD前可以更改CD版面中的文件和活页夹次序。比如可以将出现在资料CD版面窗口末尾处的一个活页夹移到列表的顶部。在资料 CD 版面中选择要排序或移动的文件或活页夹，只需将它拖到资料 CD 版面窗口中想让它出现的位置。需要注意的是，更改 CD 版面中的文件和活页夹的次序只影响它在 CD 上的物理位置，移动 CD 版面中的文件和活页夹不会更改它原来在计算机硬盘中的存放位置。
四、制作启动光盘
Easy CD Creator还提供了制作启动光盘的功能，首先应该准备好一张干净无毒的系统启动软盘，应该将必要的系统文件拷贝到这张磁盘中，比如Format.com、Fdisk.exe、Sys.com、Xcopy.exe、Mouse.exe、Mscdex.exe、Himen.sys、Emm386.exe、光驱驱动程序等，然后在磁盘中编写好可以顺利安装光驱驱动程序的Autoexec.bat以及Config.sys两个文件。
其中Autoexec.bat文件至少应该包括以下内容：
LH \MSCDEX.EXE /D:MSCD000 /M2/V。
另外，config.sys文件至少应该包括以下内容：
DEVICE＝\HIMEM.SYS/V
DEVICE＝\EMM386.EXE ram/v
DOS＝UMB HIGH
DEVICE＝\cdrom.sys /D:MSCD000
需要注意的是，其中的cdrom.sys最好设定成你使用的光驱驱动程序，这样就可以做出一张可以顺利开机的光盘了。
完成引导软盘的制作以后，就可以在Easy CD Creator中单击工具栏上“新建”按钮旁边的小箭头，从下拉列表选择“可引导CD”命令，这时系统会提示在软驱中插入启动盘并开始读取引导信息文件。完成以后，软盘的内容被复制到系统且存储在 CD 版面中，BOOTCAT.BIN 和 BOOTIMG.BIN两个文件被添加到CD版面的根目录中，接下来就和制作资料CD 的过程是一样的，在此就不赘述了。
