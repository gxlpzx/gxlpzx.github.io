{CPU的安全卫士——Hardware Sensors Monitor Pro}

{ 说起Hardware Sensors Monitor Pro，知道的人可能不多。它是一个功能极为强大的硬件传感监视工具，能帮助我们监视电脑中主要硬件（如CPU、主板等）的工作情况，让你随时了解系统是否处于正常状态。只要主板上有相应的传感器，Hardware Sensors Monitor Pro就会自动搜集有关信息，并且将它显示出来。该软件不仅可以监视CPU温度和电压、主板温度和电压、风扇转速等信息，而且它还内建了CPU温度、电压等参数的标准值，当你的处理器的相关参数出现与标准值较大差别的时候，Hardware Sensors Monitor Pro就会及时向我们报告。
一、了解Hardware Sensors Monitor Pro
Hardware Sensors Monitor Pro是Alexander Berezkin公司开发的产品，大小只有573.83 KB，适用于Windows 9x/Me/NT/2000平台。该软件的主要功能有：1.实现软件降温功能；2.CPU 抑制功能；3.可以显示三种不同的温度；4.可以自己定制警告声音文件和警告程序；5.程序启动后会在系统工具栏托盘内生成图标，而且该图标是可变的，随时反映硬件状况；6.可以将所监测的信息详细记录到日志文件中；7.主板支持极为广泛（当然该主板必须带传感器）。
Hardware Sensors Monitor Pro的正常工作需要传感器、主板和芯片组的支持，几乎所有带传感器的主板都得到了Hardware Sensors Monitor Pro的良好支持，朋友们可以参考“List of supported mainboards”中的有关说明；此外，该软件所支持的芯片组也相当多，有Intel的TX、LX、BX、GX、ZX、810、810E、820、840，SiS的5591/5595和Aladdin 芯片组，AMD的Irongate（751/756），VIA的VP3、MVP3（Apollo）；Apollo Pro133等。
二、使用Hardware Sensors Monitor Pro
当Hardware Sensors Monitor Pro主程序运行之后，我们会在系统托盘区看到一个类似于温度计的小图标，从温度计中汞柱的高低和颜色你可以了解系统硬件当前的信息。汞柱为绿色时表示基本正常，黄色表示不太正常，假如是红色则说明有危险了。
假如你想看到更多、更详细的信息，只要用鼠标右键单击这个温度计图标，从快捷菜单中选择“Restore”命令，然后就会弹出一个“Hardware sensor monitor”窗口，这里一共有三项内容：“Temperatures”（主板和CPU的温度）、“Cooling fans”（风扇转速）、“Volatares”（电压）。不过有时可能无法看到全部内容，因为它需要传感器的支持，如果某硬件本身不带传感器或者传感器的设置不正确，都有可能无法显示相应的参数。假如某项参数超过标准值一定范围，我们会看到红色的警告信息。
三、设置Hardware Sensors Monitor Pro
要想让Hardware Sensors Monitor ProHmonitor更好地为我们工作，必须对它进行设置才行。用鼠标右击温度计图标后选择“Settings”命令（或者点击图1控制菜单后选择“Settings”命令）可以打开如图^05030401b^所示的设置窗口。
1.Hardware Setup：这栏可以让我们手工添加传感器地址。可能不同的厂商会使用不同的地址，因此假如你的主板能够探测 CPU的温度，而Hardware Sensors Monitor Pro却无法显示的话，你可以在这里手工修改传感器的地址。
2.Temperatures：我们可以在该栏内修改当温度变化超过多少时，图标显示为黄色或者红色，“Warn”表示当参数不正常的时候向用户发出警告信息，“Execture”则允许让你选择一个应用程序，当参数不正常的时候，Hmonitor将自动运行此应用程序。这两个选项的默认选择都是空项，即表示不警告，也不运行警告程序。
3.Fans：该栏是风扇设置。由于 CPU电压是一个很关键的参数，太高了不行，太低了也不行，其中“Voltares”中设置的是上限值。在“Thresholds”中设置的是下限值，“Allowed deviation”一般为“10％”，表示只要 CPU 电压与标准值相差不超过 10％就可以忽略，否则就要发出警告，或者运行警告程序。
4.Advanced：该栏中的“Measurement”项是让我们选择使用摄氏温度（Celsius）或者华氏温度（Fahrenheit）来表示 CPU 和主板的温度；“Timeouts”项则是用来设置刷新频率。由于CPU的温度和风扇转速、电压等信息相对来说还是比较稳定的，一般情况下并不需要Hardware Sensors Monitor Pro经常向我们报告，因此笔者建议将这个刷新的时间设置得大一些。
5.Voltages：如果在此栏中勾选了“Warn”单选框，那么每当硬件参数与内置标准不符时，都会弹出一个警告窗口，点击窗口内的“OK”后，警告窗口会暂时消失，稍待片刻这个窗口还会出现，除非你在“Settings”中关闭这个“Warn”选项。
6.Log：是否将监测结果记录到一个日志文件中。选中该选项中“Lot type Active”标签下的“Event log”和“Data log”复选框就可以将监测信息分别记录到Eventlog.csv和Datalog.csv文件中。当然你也可点击右边的浏览图标另行设置日志文件。
7.CPU info：Hardware Sensors Monitor Pro除了可以向我们报告系统硬件的使用状况信息名，还特别提供CPU的相关信息，包括CPU的类型、主频、外频、倍频数等。
除此之外，点击图1窗口控制菜单，还有不少重要的设置：“Temperatures”、“Coolers”、“Voltages”三项可以设置是否显示如图1窗口中的显示内容，“Run with Windows”可以设置是否让Hardware Sensors Monitor Pro随系统启动而启动。
