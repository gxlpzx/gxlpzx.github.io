{FoxMail 3.1使用秘技}

{ 一、工具栏按钮的自定义
FoxMail 3.1允许用户增加或者删除系统提供的所有工具按钮，方法是：用鼠标右键单击工具栏或者单击“查看”菜单下的“工具条设置”命令，即可打开“工具条设置”对话框。选中“分类”下的名称，其中可供使用的按钮就会显示在“有效按钮”下。按下鼠标左键选中某个按钮，若显示手形光标，说明工具栏中没有这个按钮，可以将它直接拖放到工具栏；若显示环形光标，说明工具栏中已经有了这个按钮。删除某个工具栏按钮很简单，你｀只要把某个按钮拖出工具栏，然后关闭“工具条设置”对话框即可。
“工具条设置”对话框下方有六个复选项，可以设定工具栏按钮的显示效果，包括是否显示按钮名称、按钮与文字的排列方式、是否采用浮动按钮、是否采用彩色按钮、是否采用透明按钮（在Win 2000下有效）等。
二、邮件打印预览
FoxMail 3.1支持邮件的多内码预览和打印。选中邮件单击“文件”菜单下的“打印预览”命令，即可打开预览窗口。单击鼠标左键可以25％的比例递增显示比例，单击鼠标右键可以25％的比例递减显示比例。在预览窗口的“打印份数”栏内可以设置邮件打印份数，单击“打印”按钮即可开始打印。
三、程序整体转移
FoxMail 3.1新增了程序整体转移功能，用户可以把FoxMail目录整体转移至任何文件夹。转移后的程序仍可正常运行，非常便于用户合理配置系统，例如你可以在C分区容量不够时将FoxMail 3.1移到D分区。
四、POP3账号顺序调整
FoxMail 3.1在“其他POP3”框内增加了两个按钮，选中某个POP3账号，单击上下箭头按钮，可以调整该账号收取邮件时的顺序。
五、恢复误删邮件
如果用户将“废件箱”清空，而后发现误删了一封重要邮件，可以按以下方法恢复（前提是你没有对“废件箱”进行压缩）：用鼠标右键单击“废件箱”，选择快捷菜单中的“属性”命令，打开“邮箱属性”、“工具”选项卡，单击其中的“开始修复”按钮，被删除的邮件就会“飞”回来。邮件恢复以后，你最好将“废件箱”和邮件的原始邮箱压缩一下。
六、最小化设置
为了避免FoxMail 3.1最小化以后占用任务栏空间，你可以将“查看”菜单下的“最小化时在任务栏显示”选中，最小化的FoxMail 3.1成为系统托盘中的小图标。此时若想打开主窗口，只须用鼠标左键单击系统托盘中的FoxMail图标；也可以用鼠标右击托盘中的FoxMail图标，选择快捷菜单中的“显示主窗口”命令。
七、快速改变窗口布局
FoxMail 3.1状态条左侧新增了一个三角形按钮，单击它可以使窗口在水平分隔和垂直分隔方式之间快速切换。
八、快速显示/隐藏消息框
FoxMail 3.1新增了一个消息框，用于显示收到的邮件、网络连接等信息。它可以在接收/发送邮件前显示，你只要单击状态条左侧的黄色方块按钮，就可以显示或隐藏消息框。 }
