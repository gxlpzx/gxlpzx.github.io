{真正的“邮递员”——G-Lock EasyMail}

{ 我们通常收发电子邮件所使用的工具大多为Outlook Express或FoxMail等，然而若将其称为“邮递员”，则实在是过奖了，因为这些邮件工具不过是将我们的电子邮件发送到指定的SMTP 服务器，然后再经过 SMTP 服务器处理，将这些邮件加入到待发列表中等待发送……于是，我们发送的这些邮件须通过若干服务器的中转，才能发送到收信人的POP3服务器，直到此时，收件人才能收阅邮件。如果邮件发送的整个过程中所涉及的服务器不出现任何问题，那么因中转所耗费的时间可以忽略不计，在很短的时间之内就可以将邮件成功发送了；但假若途经的某个服务器出现了微小的故障或错误，那邮件就有可能“石沉大海”。况且在邮件的发送过程中要经过多次中转，邮件的内容难免不会被泄漏。由此看来，电子邮件的安全系数也不够理想。 G-Lock EasyMail正是一款克服了上述问题的邮件工具，成为真正的“邮递员”。
一、软件的下载与安装
G-Lock EasyMail 是一款ZDNet推荐的五星级软件，其最大特点就是可以将电子邮件直接发送到收件人的POP3服务器上，而无须经过其它中转，从而提高了邮件发送的速度以及保密性。最新版本为 2.0a 版，可以登录其主页http://www.GLockSoft.com 下载得到，其它如天下网（http://www.TianDown.com）、华军软件园（http://www.newhua.com）等网站也有收录。
下载后将得到一个 1.32MB 的 ZIP 文件，使用 Winzip 将其解压缩后，用鼠标左键双击运行其中的 Setup.exe 即可进入安装向导，按照提示连续单击 Next 按钮直至安装成功。
二、软件的设置
通过“开始”菜单或用鼠标左键双击软件在桌面生成的快捷方式图标即可启动程序，首先我们会见到注册信息，除非你愿意以将近30美元的价格进行注册，否则暂且不要理会它。点击“Continue（继续）”按钮进入程序主界面。界面左边共有六个工具按钮，由于是首次使用此软件，为了使其能够正常的工作，我们需要点击“Settings（设置）”按钮进行必要的参数设置。
在弹出的“Settings”窗口中，共有5个选项卡，其中包括：
1.General（普通设置）
首先，在“Primary DNS”和“Secondary DNS”分别填入主要和次要的域名服务器的IP地址，它们可以通过相应的工具软件或者向 ISP 查询得知。
然后，在“Timeout min”选定以分钟为单位的超时时间，并可以在“Domain name for handshaking”文本框中输入一个大型网站的网址作验证之用，同时建议选中“Find recipient’s address at all SMTP server it is registered（在所有已注册的 SMTP 服务器中查找连接速度最快的）”选项。
值得注意的是下面的“Internet Service Provider SMTP Settings”文本框，如在此输入我们通常所使用的由 ISP 或者各大网站免费提供的外发电子邮件服务器的地址，如“SMTP.163.net”，将来发送邮件的时候，我们可以点击程序主界面的“Mail（邮件）”菜单，在“Use ISP SMTP”的折叠菜单中指定通过此 SMTP 服务器，如“SMTP.163.net”发送电子邮件。
2.Sender Info（发件人信息）
与其它邮件软件无异，在“From”和“Reply To”下面的文本框中输入希望对方见到的身份名称及邮件地址，以作为发件人的标识，便于对方回复邮件。
3.Socks（代理服务器）
此软件支持 Socks 代理服务器，所以即便是在局域网中也能够使用此软件发送电子邮件。而直接登录互联网的用户可对此置之不理。
4.Drag and Drop（拖放操作）
通过适当的设置，可以有效地与 Outlook Express 整合使用，更增加了软件的易用性。
5.Other（其它）
（1）Check spelling before sending e-mail：在邮件发送之前进行邮件内容的拼写检查。不过由于是英文软件，这项拼写检查功能对于中文是无能为力的；
（2）Jump to Log when messages：不建立邮件发送日志；
（3）Save log to file：将日志另外保存为其它文件；
（4）Clear log every × days：每隔×天清理日志。
三、软件的使用
点击“Message”按钮，进入相当于 Outlook Express 或 FoxMail 等软件的邮件编辑窗口。“From”文本框中是发件人的电子邮箱，与刚才所设置邮件地址相符；在“To”文本框中输入收件人的地址；“Bcc”文本框中输入暗送的地址；“Group”用作群发邮件，可选择地址簿中的邮件地址，或单击文本框后的工具按钮选择已编辑好的邮件列表文件；“Subject”是邮件主题。值得注意的是邮件正文的编辑窗口左上方有一个“N”形的按钮，用鼠标左键单击可选择此邮件的发送优先权，包括“Highest（紧急优先）”、“High（高级优先）”、“Normal（普通）”和“Low（低级优先）”4个优先级别。此外，还可以点击“Attach”按钮添加附件。
待邮件编辑好后，单击“Send”工具按钮即可将其直接发送到收件人的 POP3 服务器上了，也就是说随着邮件的发送成功，对方立即就可以收取到邮件了——这就是我们使用这款软件的直接原因！
四、邮件的管理
点击“Sent Items”按钮，进入发送列表窗口，其中记录了所有已经发送的邮件；如果邮件数量太多，还可以使用“Quick Search”进行快速查找。
点击“Log”按钮进入日志窗口。其中保存了每一封邮件发送的详细记录，包括邮件的发送时间、收件人的服务器、回复的电子邮箱等内容。
五、导入地址簿
这款软件自带了地址簿（Address Book），并且可以将已有的地址簿直接导入到其中，免除了重复整理地址簿的繁琐工作。方法是：点击“Import（导入）”按钮，再点击“Browse”按钮选定由 FoxMail 等软件将地址簿导出的 CSV 文件，点击“Next”按钮，为该地址簿命名，然后再指定要导入的内容，如，在“First Name”栏指定导入 CSV 文件中的“姓名”，“Email”栏指定导入CSV文件中的“邮件地址”等内容，再点击“Next”按钮，直至确定导入工作完成。
另外，软件界面中还有一个按钮是“Stay on top”，用于使该软件总显示在屏幕最上方，可保证软件不被其他程序或窗口覆盖。}
