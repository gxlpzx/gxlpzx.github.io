{如何清除历史记录}

{ 有很多软件都会自作聪明地记下我们最近操作过的文件，这样虽然可以节省我们查找文件的时间，但有时我们并不想别人知道自已最近操作过的文件，特别是在别人的电脑或在公用电脑上就必须清除这些历史记录。下面列出几个常用软件的历史记录清除方法：
1.Office 97/2000中文版:运行Office 97/2000，单击“菜单→工具→选项”，选取“常规”选项卡，取消“列出最近所用文件数“、“最近使用的文件清单”，或将其个数（项数）改成“0”，然后单击“确定”即可。
2.WPS 97/2000:用记事本打开“C:\Windows\winwps.ini”初始化配置文件，查找字符串“recent file list\”，将下面包含“file﹖＝“（﹖表示数字1～4，下同）的几行删除掉，保存后退出。
3.Photoshop 5.0以上（英文版）:在“C:\Program Files\Adobe\Photoshop5.0\Adobe Photoshop Settings”（这是默认安装目录）下找到“Photos50.ini”配置文件并打开，删掉“File﹖＝”的那几行，保存后退出即可。
4.IE 4.0以上中文版本:右键单击桌面上的IE图标，选择“属性”在“常规”选项卡中，单击“清除历史记录”按扭，然后单击“确定”即可。
