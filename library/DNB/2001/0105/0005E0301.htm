{CE上的几款资源管理器}

{ 在Windows中，资源管理器是一个能直接、快速管理所有软硬件资源的系统软件，在Windows CE中也有一个资源管理器，但功能却要弱很多。不知是不是微软为了防止用户修改或误删某些CE中的重要文件，这个资源管理器竟然只能浏览My Documents中的文件，对于那些已惯于在系统中探寻“宝藏”的用户来说很不方便。幸好还有其他的CE软件商推出了自己的资源管理器软件，对CE用户来说可是必备的系统工具哦。
一、dTree
dTree是一款由Derago公司推出的短小、快捷的文件管理软件，主文件大小只有32K，但功能却很强大，甚至有红外线直接传输文件功能。
这个软件只有快捷按钮，没有下拉菜单，操作起来非常方便直观。各按钮对应的功能依次为：剪切(Cut)、复制(Copy)、粘贴(Paste)、创建链接(Creat Link)、删除(Delete) 、红外线传输(Beam)、帮助(Help)、版权(Info)。基本操作跟Windows中差不多，其它几个特色功能的使用方法是：
创建链接(Link)。即创建快捷方式。先在需要创建链接的文件或文件夹上单击，选定文件或文件夹，然后点“Copy”按钮，再在目标文件夹上点一下，最后点“创建链接”按钮就行了。
在“开始”菜单上创建快捷方式。先选择需要创建快捷方式的可执行文件.exe，然后直接点“创建链接”按钮就可以了。点击“开始”菜单，就会发现该文件的快捷方式已经列在最上面了，可以很方便地使用一些常用软件。
红外线传输文件。如果同事也有一台装有dTree软件的CE掌上电脑，那么就可以用这个红外线功能来方便地在两机间传输文件了。先点击选择你要传输的文件，然后点“Beam”按钮，同事在另一台掌上电脑的根目录\下点“Beam”按钮，就开始传输文件了。
另外，dTree还能直接查看Pwi便笺文件、Html文件、Wav声音文件和数据库文件.dbf，与Derago公司的另一软件dNote配合还能查看并编辑Txt文本文件。
dTree打开文件目录的速度很快，基本没有停顿感。但这也带来一定的缺点，文件没有属性显示，甚至连文件的大小都没有，而且文件的图标也很单调，有些原本有自己图标的程序也都是一样的“面孔”，可执行文件和数据文件不好识别。而最大的缺点就是它居然没有创建新文件夹的功能，真不知软件作者是怎么想的。
下载地址http://www.derago.com/，文件大小177K。
二、Palm Explorer
Grundle公司推出的资源管理器软件，该公司推出的还有一个RegEditCE注册表编辑器软件，也是常用的。
比起dTree来，Palm Explorer以下简称“PE”的界面要好得多，每种文件类型都有各自不同的图标，而且都有文件大小的显示，如果点“属性”按钮还能看到更详细的文件属性。在下拉菜单“View”中，还能改变文件列表的几种方式， 其中“大图标”方式就很像Windows中打开“我的电脑”中的效果，用起来还真有种亲切感。
PE最有用的特色功能，就在于它能够把RAM和ROM中的文件区分开来显示，对掌上电脑用户非常有用。因为，掌上电脑的RAM可是“一寸内存一寸金”呀，如果你要看CE中哪些文件是没有用想删除的，把ROM中的文件屏蔽掉就好办了可不要乱删呀，否则有些东东不能用了可别怪我:-P。怎么做？选择“View/Options”，出现文件显示设置窗口，其中“Wildcard”可以使用通配符来选定列出哪些文件，而“Type”项就是我们需要的了，不想列出哪种内存中的文件，去掉前面的方框中的钩就行了。至于下面两项内容的设置就不用多说了吧。
PE也有一定的缺点，就是每次切换文件夹时，其他文件夹就不能显出来，要切换只能点“回到上一级目录”按钮，感觉有点像Windows中查看文件夹时设置成“在同一窗口中显示”的样子，操作起来有点不方便。还有，要想使用上面介绍的特色功能，可是要注册的哦。
下载地址http://www.grundlesoftware.com/wince，文件大小118K。
三、QExplorerQE
一打开QE，你会发现跟以上两个软件不同的是，它有三个窗口。左面的大窗口用来显示目录树文件夹，右上方的小窗口用来显示正在查看的文件夹的内容，而右下方则是快速查看窗口，可以用与文件相关联的程序直接查看文件内容，可以直接查看文本文件.txt、声音文件.wav、图形文件.bmp等，还可以用来显示快捷方式.lnk 所指向的文件名，这可是其它软件所没有的功能。另外，这些窗口的大小都能任意调节，也可以只显示其中的两个窗口，只要点击窗口中间的小三角形或点拉细长条形标志就可以了。
QE的其它几个功能也很强大。 选择菜单“File/FileInfo”或直接点相应按钮，可以查看包括文件创建日期和时期在内的详细资料；“FindFile”可以使用通配符，快速查找任意文件夹中的文件；而“QE Setup”可以设置所有文件类型的关联方式，如把文本文件.txt 设置成以其它文本阅读工具打开； 就连“Version”选项也没有闲着，里面可以显示出程序内存和数据内存的使用情况以及显示屏和电源的一些信息。选择菜单“Edit/Mark”或直接点相应按钮，可以连续标记多个文件，就像在Windows中按住Ctrl键再点鼠标一样，可以很方便地对多个文件进行剪贴之类操作。在菜单“View”中，可以选择对文件进行按文件名、类型、大小、日期等方式升降序排列，方便文件的浏览和整理。
下载地址http://www.e-panda.com/download/wince，文件大小46K。
以上是笔者在网上找到的几款比较常见的资源管理器软件，功能各有侧重，至于哪一款更适合你，那就要看个人的喜好了，赶快下载一个装上吧。
}
