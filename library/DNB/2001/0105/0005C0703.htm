{驱动程序更新快报}

{ 朋友们好，新一期的驱动更新快报又和大家见面了。在春节这段假期中，各硬件厂商推出了不少新的驱动程序。下面就听我向大家一一道来。
首先NVIDIA为其TNT、TNT2、GeForce、GeForce2系列芯片显卡用户提供了两款新的6.67版驱动程序，它们分别适用于Win9x/Me与Win2000操作系统。
ATI Radeon系列显卡的用户可以松口气了，经过了数款Beta版本的测试，ATI正式推出了针对Win2000系统的5.13.01.3100版驱动以及适用于Win9x/Me 系统的4.13.7072版驱动。它们修正了Radeon在许多游戏中的显示问题。
丽台为其WinFast GeForce 256、GeForce2 GTS、GeForce2 Pro、Ultra显卡发布了新的驱动程序，分别为运行于Win9x/Me下的4.12.01.0648版与Win2000下的5.12.01.0666版，其中针对Win2000版本的驱动还将Leadtek独特的WinFox Hardware Monitor软件升级到了1.17版。
Matrox的G450、G400、G200显卡用户注意了，新的驱动程序来了！分别是For Win9x/Me的6.23.005版，For Win2000的5.33.006版与For WinNT4的4.73.019版。另外Matrox还为上述三款显卡推出了一个名叫MTS Tweak Utility的超频调试工具，目前版本是4.00.014版。
IBM推出了最新的IBM硬盘专用Disk Manager 2000程序3.10版。Disk Manager 2000可以一步一步地帮助你顺利完成IBM硬盘的安装和配置，它内置硬盘专用Disk Manager（DM）9.54版，可以对硬盘进行分区、格式化，并具有让硬盘突破主板容量支持瓶颈528MB、540MB、2.1GB、4.2GB、8.4GB、32GB的功能。下载后运行程序，它将自动为你创建一个能够启动的IBM DOS2000软盘，软盘内包括Disk Manager 2000的全部内容。
西部数据硬盘的Data Lifeguard Tools套装工具已经推出了2.60版，此程序下载后会自解压到一张空软盘上。其中包含可以让你的系统支持大容量西部数据硬盘的EZ-Install、全面检测并修复磁盘错误（并将硬盘数据清零）的Diagnostics工具、检测BIOS中硬盘数据的BIOS Check工具、切换Ultra ATA 33/66/100硬盘工作模式的Ultra ATA Manager等几个软件。
这几周华硕、精英、建邦、梅捷、硕泰克、磐英、建基等厂商也都为自己的多款主板发布了新的BIOS程序，篇幅所限，不能一一详细列出，详情可查看各厂商主页或驱动程序站点。}
