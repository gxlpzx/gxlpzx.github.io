{Windows2000注册表在局域网中的应用}

{ 一、解决启动时没有网络提示对话框的问题
我们在连网过程中，经常会遇到一些问题，如向网络中添置电脑或是重装电脑的操作系统后，此计算机开机后不出现登录网络的对话框，从而无法登录网络的情况。要出现这个对话框，只能在“开始”菜单中选“注销”并以用户登录系统才行。结合自己的一些使用经验，总结出如下原因和解决方法，供有类似问题的网络用户参考。
这类问题一般由以下几个原因造成：
（1）网络登录主域名不正确
（2）网卡配置不正确
（3）计算机的Cache里保留有Microsoft or Netware network passwrd
（4）第一次登录网络或Windows2000桌面时输入了空口令
解决方法：第一种和第二种情况可以通过重新进行网卡配置或登录域名来解决，第三种情况可以通过重新安装“Microsoft网络客户”或“Novell网络客户”来解决：在“控制面板”中单击“网络”，在出现的对话框中删除已安装的“Microsoft 网络客户”或“Novell网络客户”，放入Windows 2000安装盘安装即可。第四种情况我们可以修改Windows 2000注册表文件，方法为：
运行注册表编辑器（单击“开始→运行”，输入“regedit”），找到HKEY＿LOCAL＿MACHINE\ SOFTWARE\Micorsoft\Windows\currcnt version\network\red mode net 子键分支。将“Auto logon”项的值设为“01”（其值可能为00），如果没有Auto logon串，可以新建这个项Auto logon，并将其值赋为01
二、局域网络的自动断开控制
Windows 2000使用两种不同的自动断开连接参数。一种用于断开远程访问服务（RAS）连接，另一种用于断开局域网络连接，对于局域网络的自动断开控制可通过修改注册表实现：
启动注册表编辑器，打开注册表的“HKEY＿LOCAL＿MACHINE\SYSTEM\CURRENT CONTROLSET\SERVICES\LAN MAN SERVER\PARMETERS AUTODISCONNECT”子键分支。在此子键分支中可以找到局域网络的自动断开连接参数，它的作用是在等待一定的时间之后断开空闲的连接。其可用的范围为0～4294967295（0Xffffffff），用户也可以在命令提示符下，用“Net Config Server”命令设定时间。如可以用下面的命令将自动断开连接时间设定为30分钟：“Net Config server/auto disconbect：30”。“AUTODISCONNECT”这个REG＿WORD类型值的可用范围为：1～65535分钟，将它设为-1，则可以禁用自动断开连接，但是如果设为零，则RAS的自动断开连接参数将关闭。
三、设定在RAS响铃多于一声后使之应答
若要使RAS在“X”遍铃后应答，需要修改注册表“HKEY＿LOCAL＿MACHINE\SYSTEM\CURRENT CONTROLSET\CONTROL\CLASS”子键分支，确定调制解调器（Modem）子键及此键值数据在此分支中的位置（通常是第4个），双击此子键展开分支，确定出现问题的调制解调器，通常0000是第一个调制解调器，双击并突出显示Monitor键，双击第一个调制调器，双击并突出显示Monitor键，双击第一个键值项数据，它应当为“ATSO”，并将它改为响铃的次数，如3。停止并重新启动RAS服务，且重新引导Windows 2000系统（以上注册表修改可以在HEKY＿CVRRENT＿CONFIG根键下完成，这时需要在“HKEY＿CURRENT＿CONFIG\system\currtntcontrol set\control\class”子键中添加必要的键值项数据）。
四、远程访问Windows 2000注册表
在Windows 2000操作系统中，只有管理员组的成员才能访问注册表。用户可以编辑注册表中的“HKEY＿LOCAL＿MACHINE\SYSTEM\CURRENT CONTROLSET\CONTROL\secure pipeserver\winreg”子键分支来修改默认设定。操作步骤：启动32位注册表编辑器（单击“开始→运行”，输入“regedt32”），如果“securepipservers”键不存在，则可用REG＿SZ类型添加到此子键分支下面，选择Winreg键，并在RegEdt32菜单上单击“安全→权限”来选择需要的访问权限，如分配给系统管理员“Full Control”（全部权限）。
这些访问许可也可避开某些服务，如目录复制器和打印假脱机等，则需要远程访问注册表。可以将访问权限给予运行这些服务的账号。或者可以编辑注册表的“HKEY＿LOCAL＿MACHINE\SYSTEM\CURRENTCONTROLSET\CONTROL\securepipeserver\winreg”子键分支，在此分支下添加一个“Auowedpaths”键，它带有一个Null类型。}
