{软件一点通}

{ ●在DOS下提取Cab包中的文件
我们都知道在Windows安装盘上有很多Cab文件格式的压缩文件，当Windows的某个系统文件损坏或丢失，我们便可以从中提取以恢复系统。可是，如果重要的系统文件被损坏（如vmm32.vxd）无法进入Windows系统，那么我们必须在DOS下从Cab包中提取文件，具体方法如下：假设vmm32.vxd这个文件丢失，用启动盘进入DOS并切换到X:\WIN98\下（X\WIN98是光盘中Windows文件夹的路径），在光标提示符下键入：“extract/A Base4.cab/L C:\windows\system\vmm32.vxd”其中Base4.cab是第一个cab文件，/A参数表示在这个文件（Base4.cab）起依次往后搜索，/L参数表示将抽取出来的文件放在后面这个路径下，完成后重启即可。
●恢复OE标识密码
Outlook Express（简称OE）通过标识来管理多用户，以不同的标识来区别不同的使用者，在不同的标识之间切换要通过密码验证，阻止侵入者，达到安全目的，但忘记密码，OE就会拒绝用户进入。
OE把标识密码加密后存放在HKEY＿LOCAL＿MACHINE\Software\Microsoft\Protected Storage System Provider\\Data\89c39569-6841-11d2-9f59-0000f8085266键下（其中表示用户，如默认用户就是Defaul），有多少个标识就有多少个子键，不过不同的计算机那一长串数字可能会有所不同，把该键值导出备份，然后删掉，再打开OE，如果出现询问密码对话框，直接按“确定”即可。
●设置TE为非默认浏览器
打开腾讯浏览器（Tencent Explorer，简称TE），单击“工具→TE选项”，这时会出现一个对话框，你可以选择默认浏览器的设置，如Internet Explorer或者Tencent Explorer，还可以设置“启动时不检测默认浏览器”，免除了烦人的对话框。
●禁用Word“智能剪贴”功能
Word提供的“智能剪贴”功能可以自动调整删除或插入文字处的空格。打开该功能时，在删除或插入文字后，Word能够自动调整该文字前后的空格，例如Word能删除句号之前多余空格，或在两个单词之间添加空格。这对英文输入来说确实十分方便，但对汉字来说稍有画蛇添足之嫌。要取消Word插入“剪贴板”中的内容时添加空格的功能，步骤如下：单击“工具→选项→编辑”，清除“使用智能剪贴”复选框即可。
●找回Windows Me备份组件
目前Windows Me备份功能已经不出现在安装选项中了，同时“控制面板”中的“添加/删除程序”属性工具也隐藏了该功能，该功能已被隐藏到Windows Me安装光盘中。实现该功能的安装步骤如下：打开Windows Me安装光盘中的“Add-ons\MSBackup”文件夹并运行可执行文件“Msbexp.exe”。安装完毕后，重启Windows Me。而后单击“开始→程序→附件→系统工具”，就会看到“备份”工具了。
