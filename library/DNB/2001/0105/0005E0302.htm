{巧设Palm的背光}

{ 让Palm背光一划就亮
如果你要在晚上使用Palm那么一定要打开它的背光（Palm Ⅲc除外）。按着电源开关两秒钟，背光就会亮起这是Palm默认的背光开启方式。在这里我要给大家介绍开启Palm背光的另一种方法让Palm背光一划就亮。
1.执行PrefsPreferences/控制台，并切换到Button（按钮）；
2.到所示的画面，点选“pen……笔……）”；
3.到所示的画面，在下拉选项中选取“backlight”(背光),然后“确定”。这样就增加了一种开启Palm背光的方式。现在到暗一点的地方去试一试效果将触控笔从手写区划到屏幕顶部哇!好酷!背光真的亮了,再划一下,背光就关上了,操作起来真是既方便又快捷。
改变Palm的背光方式
有些Palm机型的背光方式是将黑色字体变成了发亮的字体看起来也许不太习惯希望黑色字体不变只要背景发亮就好。其实Palm的内建程序已经提供了这种转换背光的方式。
为了顺利完成转换我们先要了解一下Palm的shortcut快捷键的写法。
1.执行PrefsPreferences/控制台，并切换到owner（使用者）；
2.打开“参数设定”的下拉菜单选取最后一项“手写板说明” 。
3.找到Shortcut的输入方法。
掌握了Shortcut的写法接下来就要转入正题了。
1.在MemoPad记事本里，新增一个记事本。
2.在手写区画一个Shortcut符号。
3.写入一个英文的句点注意要点手写区两次
4.写入数字“8”。如果成功了话记事本上会出现“Inverting Backlight”代表背光方式已经改变了拿到暗处打开背光果然是亮的背景黑的字体蛮清楚的。
如果按照上述方法再做一次记事本上会出现“Normal Backlight”就回到了Palm默认的背光方式。 }
