{《突袭》中的兵工厂}

{ 《突袭》是一款不错的游戏，我已经爱不择手。前几期《电脑报》的几篇文章对《突袭》的评论是该游戏集各类游戏百家之长，但唯一的缺陷是“没有不停出兵的兵工厂”，只凭电脑给你的有限的兵力和装备，与强大而狡猾的敌人斗智斗勇。我对此观点仅同意一半，原因是《突袭》中不仅有兵工厂，而且兵工厂的掌握使用，与指挥官的战术运用、策略谋划决定着战争的胜败 ！
《突袭》在许多战争场景中，地图有一些工厂，外表来看，有的规模较大，不仅有各式的厂房、设备，还有各种管道、锅炉；有的则较简陋。各个工厂的生产方式不同，生产的武器装备也不同，有的有产量限制，有的无产量限制，与《星际争霸》、《红警》等不同之处是不需采集矿物或能源，人为无法制定其生产计划及下达生产或停止生产的指令，完全靠工厂自动生产。笔者在某战役中因己方的兵工厂不停地生产，敌我力量对比悬殊太大，又无法控制其停止生产，为避免落下战争狂人之嫌，只有下令炮击己方工厂。各工厂内各生产车间生产的品种也有分工，有的管生产各式坦克、自动火炮、装甲车。有的管生产各式火炮有的管生产各式车辆如弹药输送车等，但不能生产士兵。的确，《突袭》中的兵工厂是不能生产士兵的这与该游戏追求逼真的风格相对应。
现在谈谈如何使用兵工厂及其策略
己方的兵工厂只须派出士兵占领各厂房车间（这是能否生产的关键），厂外围派重兵把守。尤其要加强防空火器的配置，敌人会把你的兵工厂作为空袭的首要对象。过一会儿，武器装备会源源不断地生产出来。此时，你可将兵工厂作为轴心，将生产出来的武器，尤其是各种火炮在厂区外围逐渐向外布防。注意将重型榴弹炮在厂区内集结，密切注视其火力覆盖范围内敌方的动向，敌人是很会偷袭你的兵工厂的哟。
对敌方的兵工厂，有的战役将占领敌方的兵工厂作为取胜的条件，因为往往该次战役，己方的兵力十分薄弱，且基本无增援部队，敌方则实力强大，最讨厌的是敌方的重炮将各要道严密封锁，使你寸步难行，时不时还派部队袭击。占领敌方的兵工厂，要派出小股精干的部队，将自动移动的主控命令设为“关”，这样就可以指哪打哪，在肃清厂区内外的敌方守敌时，注意不能打坏厂房及附属设备，否则你占领的工厂则是一座无法生产的死厂。随后即派士兵占领各厂房及设备。以后你知道该怎么做了吧。提醒一句，敌方是不会白白让你占领他的工厂的，其反扑是十分凶猛的。有时，他们干脆将整个工厂摧毁，他得不到，你也别想得到好处！有的战役敌方有几处兵工厂，而且各工厂生产的武器种类也不同，有的工厂则专门生产火箭炮，够爽的吧！还不想法占领。不过敌方的兵工厂生产的数量好像都有限制。有时敌方的兵工厂的车间已被敌兵占领，派士兵去占领的同时也摧毁了工厂。对占领无望的敌兵工厂，为了不让敌方的实力坐大，可派重型榴弹炮远程炮击，将其摧毁。
}
