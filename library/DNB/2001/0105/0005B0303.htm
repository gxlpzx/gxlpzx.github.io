{文字保管工具——一拖即存}

{ 当你在网络上看到自己中意的文章却苦于找不到一个可以随手为你快速、准确分类保存这些内容的工具的时候，可以试试“SaveText”，它的中文名称叫做“一拖即存”。该软件属于剪贴类软件，无需特别安装，只要将压缩文件解压就可以使用了。运行后，它会在程序文件目录下生成“Savetext.ini”和“Skin.ini”两个文件作为程序的运行记录。它的操作类似网络蚂蚁，十分简单。
第一步：运行软件后，桌面的任务栏中将出现一个显示着“存”字的小图标。同时，桌面上还会出现一个醒目的“拖存窗口”（类似于网络蚂蚁）。点击窗口右边的按钮以激活软件的“自动拖存”功能。
第二步：用鼠标选中相应内容并拖曳至窗口中。
第三步：双击拖存窗口，或在窗口上点击右键，选择菜单上的“打开编辑与设置窗口”，打开程序的文本处理窗。在这里，你将看到刚才选择保存的内容。
第四步：程序第一次运行时，会自动建立一个名叫“非分类”的类别。假如你需要对自己下载的文章按照一定方式分类，可在左上角“类别”项的下拉菜单中选中“新建类别”，在左下方可找到“新建类别”栏，在旁边的输入框中输入名称并单击“浏览目录并确认”就可以了。更改类别的方法与之相似。
第五步：选中“每次自动保存”，程序将在每次保存时自动在后台将所选内容保存为一个文本文件（位于该软件安装的相同目录下），文件名为所选内容的前几个字。单击主菜单中的“程序设置”栏并选中“声音提示”项，可以设置系统发出声音，提示你任务已完成。
另外，本软件还具有诸如“文件合并”、“浏览文件”、“ 内码转换”、“段落重排”等实用功能，“断电重启自动恢复”功能可以在遭遇断电、死机时对文件进行自动保存。设计可谓十分周到。
本软件可用于在网上摘取文章，尤其适用于只需保存文章部分内容的时候。有了它，你就可以不再使用IE为你慢吞吞地保存网页内容了。喜欢它的朋友可以到http://charlieou.myrice.com/cool/search/soft2/cih/0158.htm去下载试用。这是一款完全免费的软件。 }
