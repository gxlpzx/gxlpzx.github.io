让孩子安全上网

有计算机的家庭都将面临一个令家长们头痛的问题，即如何教育子女更加安全、有效地使用互联网。众所周知，互联网实际上是一把双刃剑，它在使人从中获得大量极有价值的信息，学到最先进的科学文化知识，提高自身素质的同时，也因其自身的复杂性，使人们往往被各种信息垃圾包围，久而久之身受其害。怎样才能使孩子们轻松、安全、健康、快乐的上网呢？这里，笔者介绍几种方法，供家长朋友参考。 ##1 家庭上网注意事项 1.和孩子一起上网 和孩子一起上网时，一则可以帮助他们在网上增强安全意识和对自己的行为负责的观念，二则可以联络父(母)子间的感情，三则可以互相学习，共同提高。 2.不要轻易透露私人信息 要教会孩子不要把自己网上的私人信息随便透露给别人，尤其是在聊天室和BBS上，以免受到不必要的伤害和骚扰。 3.远离网上危险信息 告诉你的孩子不要理睬那些带有攻击性的、危险的电子邮件、交谈请求或其它交流方式，不要进入那些令他们感到不舒服的站点。 4.不要轻易与网友见面 5.有危险要及时报警 如果你的孩子感觉到有来自网上的危险，应该迅速报警，与公安部门取得联系。 ##1 使用合适的浏览器 这里为你介绍一款专为孩子们设计的浏览器——Kidsafe Explorer，它可以很好地保护孩子，使他们免受网络垃圾的侵害。这是一个共享软件，你可以从http://www.newhua.com.cn/下载。在不注册的情况下，可以免费使用30天，软件本身无任何功能限制，但它是结合于IE的，所以运行时系统必须安装有IE浏览器。 安装后，第一次启动会弹出一个对话框，要求你为Kidsafe Explorer设置一个密码，这个密码是对Kidsafe Explorer进行配置和管理的密码。设置好后，进入Kidsafe Explorer浏览器，它所有的操作都可以通过点击按钮来完成，十分方便快捷。 屏幕左边的地址列表框中是家长设定的允许孩子访问的站点，孩子们要访问这些站点只需单击网站的按钮即可，不需要输入URL，使用非常方便。软件安装后已经预装了一些适合孩子们访问的网站，你还可以自行设定，将你喜欢的站点添加到地址列表中。具体的设置方法是：单击工具栏上的Configure按钮，弹出一个要求输入密码的对话框，输入你在第一次启动Kidsafe Explorer时输入的密码，选择Configure或Supervisor Mode。 若选择Configure则进入家长配置屏幕(Configuration Screen)在这里，你可以自由地设定允许你的孩子访问的站点，分别输入这些站点的站点描述(Description)、站点地址(Site address)、图标文件名称(Image file name)。添加完毕后，点击保存按钮，就会发现这些站点的图标已经出现在地址列表框中。 若选择Supervisor Mode，则进入地址输入模式。输入地址后，点击GoTo，你就可以像普通的浏览器一样上网浏览各种站点，发现适合孩子的站点时，点击地址栏旁边的Add按钮，该站点即被加载到屏幕左边的站点列表框中。 再来看看另一个很重要的设定。选择Configure进入家长配置屏幕(Configuration Screen)，单击Option菜单项下的Advance Option选单，弹出Advance Option对话框，共有四个选项：1.Enable subdomain restriction，该项被选中，孩子就只能在站点列表框中指定的站点浏览。 2.Disable desktop while program is running 选中该项后，桌面将会被锁死。你会发现任务栏不见了，桌面上的其它任何程序也都不能运行。3.Disable context menus and…选中此项，鼠标右键菜单和快捷方式均不能使用。4.Must enter the supervisor password to exit the program 选中后，只有键入密码才能退出Kidsafe Explorer。 通过对孩子们所访问的站点进行限制，可以在保护孩子的同时，又让孩子从网上学到许多有用的知识。 ##1 酷站推荐 最后，为家长朋友推荐几个适合孩子们的站点： 1.中国少年雏鹰网 http://www.chinakids.net.cn/ 它是中国内地发展较早的简体中文儿童网站，曾于1999年杀入CNNIC“中国优秀网站”，是目前唯一获此殊荣的少儿网站。采用多媒体和动画效果，内容丰富，里面不但有百科知识、每日新闻、下载及游戏天地，还有旅游乐园及娃娃电脑。这些生动活泼的内容可以吸引孩子去接触电脑、接触信息化世界，全面提高孩子的整体素质。 2.红泥巴村 http://hongniba.com.cn 这是一个很有特色的儿童网站，首次引入了主题公园的概念，以一个虚拟的村落为主题，以堆泥巴为象征，设计了系列卡通人物作为主持人，以一大批简体汉字儿童网站作为基本村户，同时收录了部分经典英文和繁体汉字儿童酷站，搭建起了一个儿童娱乐与教育的平台。 3.中央电视台的大风车 http://www.cctv.com/life/kids/index.html 它是中央电视台大风车节目的相关网站，从这里你可以了解到大风车节目的各栏目内容、节目安排及相应的一些文字资料，通过小信鸽你还可以给主持人发邮件，还可以找到央视的动画城、东方儿童、芝麻开门等另外几个少儿节目的网页。 如果你还想知道更多适合孩子们的站点，不妨通过搜索引擎去查找一下。最后，希望您能在网上给孩子找到一片纯净的天空，让他在学中玩、玩中学，不断增长知识，健康成长。
