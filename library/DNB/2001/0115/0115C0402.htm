驱动程序更新快报

朋友们，两周一次的驱动快报又和大家见面了！ 按照惯例，首先向大家介绍来自NVIDIA的消息，11.01版的NVIDIA TNT、TNT2、GeForce、Quadro、GeForce2、Quadro2系列芯片显卡驱动已经发布了！它有两个版本，分别适用于Win9x/Me与Win2000操作系统。通过打开驱动的Inf文件，我们发现这款驱动已经加入了对Geforce3(即NV20)的支持。 这两个星期NVIDIA的老对手ATI也没闲着，它的Radeon显卡驱动有了快速的4.13.7099版和注重稳定的4.13.7093官方版(For Win9x/Me)；另外，针对Win2000系统的5.13.3124版驱动也同期发布了。颇受欢迎的ATI DVD Player的最新4.1.0.0版已经发布了，但仅能在Win2000下运行。建议拥有ATI显卡的朋友们使用。要注意的是，安装时必须使用ATI的原配光盘。 德国ELSA(艾尔莎)公司为它们的ERAZOR Ⅱ/Ⅲ/Ⅲ LT/Ⅲ Pro/X/X2以及GLADIAC GeForce2 GTS/MX/ULTRA等显卡，推出了适用于Win2000的最新驱动程序，它基于NVIDIA公版驱动——7.58版。 使用Intel i810、815、820、840、850芯片组主板的朋友们注意了！英特尔于4月8日发布了新的6.10.011多语言版Ultra ATA Storage驱动，它适用于Win9x/NT4/2000操作系统，安装后能让你的系统充分发挥ATA DMA/Ultra DMA技术带来的性能优势，希望大家不要错过机会！ Maxtor(迈拓)硬盘的MaxBlast Plus应用程序有了最新的1.27S版。迈拓MaxBlast Plus其实就是StorageSoft公司推出的著名Drive-Guide软件的Maxtor版。MaxBlast Plus是一款非常专业并且易于使用的硬盘工具，从最初的自动检测、分区到完成格式化，即便是新手也能轻松搞定所有的操作。该工具的EZ-Copy功能是在硬盘容量超出主板BIOS支持范围情况下的解决方案之一，它能够帮助主板支持528MB、2.1GB、4.2 GB、8.4GB、32GB、64GB等容量的硬盘。同时，它还能为你创造一个启动盘，这样，即使系统无法启动也一样能够利用这个强大的工具对硬盘作出管理。新的版本加入了POWERMAX.EXEMaxtor Diagnostics Program诊断程序并且新增了对ATA66/100的支持。总之，这是迈拓硬盘用户的必备工具。 IBM硬盘专用的Disk Manager 2000程序也推出了新的3.10.14版。Disk Manager 2000可以一步一步地帮助你完成IBM硬盘的顺利安装配置，它内置硬盘专用Disk ManagerDM9.55版，可以对硬盘进行分区、格式化，还能让主板支持大容量的硬盘。下载后运行程序，它也会自动创建一个能够启动的IBM DOS2000软盘，软盘之内包括Disk Manager 2000的全部内容。 这两个星期ECS、Epox、Gigabyte、Asus等厂商为他们的部分产品推出了最新版本的BIOS，大家可以留意一下厂商主页或到专业的驱动程序站点查询。(^15030402a^)
