开启美丽的新世界

阿果眼前出现的是一片美丽而奇怪的森林，他挠了挠头，实在搞不清怎么会到这个地方…… ##1 梦的启示 阿果一边走着一边在努力回忆自己刚才的经历，明明是在看着《最终幻想Ⅷ》的游戏电影，怎么就会无缘无故突然到了这里呢？ 好了！大家可能都有点模糊了，呵呵！先让我们介绍一下主人公阿果吧！ 怎么说呢？他是一个超级《最终幻想Ⅷ》Fans，说实话看这部动画电影已经不下1000次了(笔者注：哗啦！读者倒了一大片！)，可是每每听到王菲那仿佛从悠远太空传来的声音和看到那动人传说的画面，他就激动不已。那份狂热的劲，看看他的小卧室就不难了解了：墙上的海报、书桌玻璃下压的图画无一不是关于《最终幻想》的……他的电脑中呢？好家伙)，唯一装的游戏是《最终幻想Ⅷ》！哈哈！而且这一阵他正在研究如何制作出自己的《最终幻想Ⅷ》电脑壁纸和屏幕保护呢！那他怎么会突然到了森林呢？ 这个森林很奇怪，不是平常见到的那种，这里的一些树上挂着些矩形的图片，很艳丽很美的图片；而另外一些树上挂着会动的东东。 “欢迎你，我亲爱的果果！” “你是谁？” 眼前突然出现一位慈眉善目的白胡子爷爷，把阿果吓了一大跳。 “我就是‘图形大爷’，哈哈！孩子，你不是想要自己动手做心爱游戏的壁纸和屏保么？” “是呀！”他很奇怪这个爷爷怎么会知道。 “你不是说你很想了解一些关于图形格式的知识么？” “全中！爷爷” “好了，不要性急，慢慢听我说。我会为你把心中的结都解开的，呵呵！让我们开始吧！” 所谓的图像格式，是指我们利用计算机以数字方式存储图像的方法，不同的图形格式意味着不同的编码方法。图形格式大致可以分为矢量式与点阵式这两种。下面给你具体解说： ##2 一、矢量式图形 优点：图像可以很方便地无极放大、缩小、旋转，精确度较高的可以制作三维图形等，并且进行放缩旋等操作时不会影响图形的质量。 缺点：它不能制作出颜色丰富或者颜色变化较大的图像作品。同时也不易在不同的软件间交换文件。 我们常用来制作和处理矢量图形的软件大致有下以下几种：FreeHand、CorelDraw、Illustrator 、AutoCAD等。 常见的矢量图形格式有：WMF、AI、FH7等 1.WMF格式：全名为Windows Metafile，即Windows图元文件格式。是微软公司开发的一种矢量格式，被广泛运用于Office等软件中。 2.AI格式：全名为Adobe Illustrator，哈哈！就是制作这种图形的软件名称了。它是由Adobe开发的，一种被广泛支持的矢量格式。 3.FH7格式：又是一款以图形软件命名的格式，我想你不难猜出这就是由Freehand制作出的图形了。 还有很多其他的，如FT7、DXF、CRD等，都差不多，就不再多说了。 “我们继续说图形格式的另一类：点阵式。”图形大爷说。 ##2 二、点阵式图形 我们一般称之为位图，这类图形格式刚好弥补了矢量式图形格式的不足，它可以方便的制作出颜色丰富、变化纷呈的图像来，对于现实生活中的事物可以完美地表现，并且十分容易在不同的软件之间交换。 因为点阵式图形的以上优点，它成为了我们生活中最常用的图形格式类别。无论是广告设计、视频制作还是工程建设等都广泛采用了适当的点阵式图形。 说起具体的点阵式图形格式更是数不胜数，而常见的有以下几种：Bmp、Jpeg、GIF、PNG、TIFF、PSD等。 1.BMP格式：BMP就是Bit Map Picture(位图)的缩写，它是最普遍的点阵图格式之一，同时也是Windows及OS/2两种操作系统的标准格式。 BMP只能存储四种图像数据：单色(1位)、16色(4位)、256色(8位)和24位真彩色。BMP图像数据有压缩或不压缩的，我们常用的是不经压缩的。其中压缩方式只有RRLE416色和RLE8256色两种，而24位BMP格式的图像文件无法压缩。这种格式的优点是能够很好的保证图像的画面质量，缺点是图像文件一般比较大。 2.Jpeg格式：Jpeg的全称为Joint Photographic Experts Group(联合图像专家组)，这是一个在国际标准组织ISO下从事静态影像压缩标准制定的委员会。由它制定出的第一套国标静态影像压缩标准：ISO 10918-1 即我们现在在说的Jpeg了。 这种格式采用的是有损压缩的方法对图像进行编码，以期达到更小的体积和较完美质量。 “哈哈！到这里，果果你是不是有点听不懂了，什么叫‘有损压缩’呢？好的，先让我们把这个问题搞清楚。”图形大爷接着说。 所谓的压缩方式大体上可以分为无损和有损压缩两种。无损压缩是指在对图像的大小进行压缩编码时不会丢掉图像本身原有的颜色、位置等一些信息，因此保证了图形的质量不会变化。但这种方式往往得不到令人满意的压缩比，所以在颜色数较少的和颜色变化不大的图像压缩中应用得比较多，就好像待会我们要说的GIF格式。相反，有损压缩在减小图像体积的同时是以牺牲一部分图像的信息为代价的。但我们通常可以得到很好很高的压缩比，在压缩算法比较先进的前提下，我们同样可以得到同源图像差不多质量而体积小得多的图像。我们现在讨论的Jpeg就是这样一种格式。 Jpeg格式采用了非常高效的压缩算法，在保存时能够将人眼无法分辨的资料删除，使得在高压缩比(压缩比在21至401之间)的情况下，依旧可以获得十分不错的画质。 并且Jpeg还有一个优秀的性质就是可以调节压缩比，使我们在对图像压缩质量的把握上有了更大的灵活性。 3.GIF格式：一种由Compu Serve公司制定的格式。GIF是可交换图像文件格式英文Graphics Interchange Fotmat 的缩写。它可以简单分为两类：静态GIF(这其中分为87a和89a两种版本)和动态GIF(ANIMATED GIF)。 GIF格式采用的是无损压缩的方法。但是它的颜色数由于8位存储格式被限制在了256色中，因此对于颜色数较多或颜色变化较大的图像处理就不尽如人意，但处理颜色较为简单的图像的确是一名悍将。它同Jpeg格式一样有着令人心动的压缩比，并且同样可以灵活地调节压缩比。 GIF同时具有Jpeg无法比拟的优越性：可以制作透明背景(GIF的格式之一GIF89a)和动态的图像。也就是说我们看到的图像将没有那呆板的矩形边框，并且可以是动的。哈哈！你来的时候都看见了，那树上会动的东东就是GIF图像了。这种格式同时还有一个渐进传输显示(interlaced)的特性(就是指图像传输过程中可以先呈现一个较模糊的图像，然后渐渐清楚)，比其他图形格式的传输显示方式更具优越性。 4.PNG格式：PNG是可携式网络图像Portable Network Graphics的缩写。它几乎综合了Jpeg和GIF的优点，它不仅能储存256色以下的“index color”图像，还能储存24位真彩图像，甚至最高可储存至48位超强色彩图像。它采用无损压缩的方式，并且压缩比较之GIF格式更高，同样它和GIF一样具有渐进传输显示的下载方式。它具有透明性质，而比GIF的透明更为优越的是它可以显示不同层次级别的透明(就好像透明度一样，我们终于可以定义半透明了！)。 5.TIFF格式：为标记图像文件格式，是英文Taggen-Image File Format的缩写。它采用的是一种无损压缩的方式，不会破坏任何图像数据，更不会劣化图像质量，但压缩比非常小，只有2：1。 6.PSD格式：搞图形处理的人大概都很清楚这种格式的，对了，这就是大名鼎鼎的图形处理软件Photoshop所专用的图形格式。支持Windows和Mac平台。它完整的存储了制作者在Photoshop里处理的一切信息，包括图层、通道、透明度等。对于图像的大小有限制，最大图像像素为30000×30000。这种图形格式被广泛应用于商业艺术中。 当然了，还有很多很多的图形格式呢！比方TGA、PCD、PCX、CAL等等。 “哇赛！好精彩的图形世界啊！”阿果听得不禁入了神。 “呵呵！傻孩子，看你高兴的，这些基础知识还只是图像领域中一个小小部分呀！你要想实现自己的目标，做一个自己的壁纸和屏保可还要继续探索呦。”图形爷爷笑着说“我可以告诉你的就这么多了，接下来你就要自己去寻找利器来完成你的任务了！” “啊！爷爷……难道我要走了么？”阿果一睁开眼睛，却发现自己趴在电脑桌前。 原来……这只是一场梦……
