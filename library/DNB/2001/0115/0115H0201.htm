掌上电脑龙虎斗之Pocket PC篇

在掌上电脑市场的两大阵营中，Palm系列固然风行一时，市场占有率遥遥领先，但以微软为首的“传统”电脑公司一派势头也丝毫不弱，一直有和Palm系统分庭抗礼之势。当基于Windows CE 3.0操作系统的Pocket PC推出以后，它们的性能和功能都迈上了一个新的台阶。完善的多媒体功能、徇丽的彩色屏幕，基于Windows CE 3.0的Pocket PC绝对是掌上电脑中的“贵族”，而随着国内掌上电脑市场的高速成长，“贵族”也要“飞入寻常百姓家”，中文化的Pocket PC已经悄悄来到我们身边。 ##1 产品介绍 ●康柏 iPAQ H3630 iPAQ H3630是目前尺寸最小的Pocket PC，它的体积大小同Palm Ⅲc差不多。H3630配有16MB ROM和32MB RAM。H3630采用了Intel最新的206MHz的StrongARM SA1110微处理器。我们手上这台H3630使用的是英文版Windows CE 3.0操作系统加上CE Start和蒙恬手写输入，不过当H3630在国内正式上市销售时，将是完全中文操作界面并实现本地化的版本。(^15080201a^) H3630的造型又炫又酷：银灰色烤漆的金属机壳，符合人体工程学的流线型设计，除了一个录音按钮外，其余的都在机身正面。超大的具有“回车”确认功能的十字方向钮，特殊的手写笔存放装置，这一切使得它看上去感觉非常抢眼。机身正面液晶屏幕右上方是H3630的电源开关按钮，左上方的小孔是录音孔。录音按钮就隐藏在机身左侧上部向外凸出处。液晶屏幕下方是四个快捷按钮。机身上端正中是红外线传输装置，左面是麦克风插孔，右面是带按键控制的手写笔存放槽。电源接口在机身底端左侧，正中间是传输底座接口和外接扩充底座接口。有意思的是H3630的扬声器就藏在方向控制旋钮的下面。H3630的传输底座是一个银灰色和黑色、造型十分独特的蝶形底座托架，其底座的传输线是RS232串口线。不过。H3630也提供了USB接口的传输电缆，以适应用户的不同需要。 ●惠普 Jornada 545 惠普Jornada 545是全球第一款采用中文Windows CE3.0的Pocket PC。Jornada 545使用的是日立SH3 133MHz的32位处理器，ROM和RAM均为16MB。Jornada 545采用的是可充式的锂电池。并提供了USB和串口两种连接方式。 第一眼看到Jornada 545，都会被它高贵典雅的造型所吸引：大翻盖的保护外壳和宽大的黑色金属机身，机身两侧还套有带防滑花纹的橡胶软垫，看上去很简洁但充满质感，给人以雍容华贵的感觉。它可以更换成其它色彩的外凸弧形的保护外壳内有存放手写笔的卡槽，显得十分个性化。与其它Pocket PC的手写笔不同，HP的手写笔是扁平的，接触面积大，手感很好。机身正面约三分之二的面积被标准的240×320点阵CSTN液晶显示屏所占据，它能够支持4096色。屏幕下方类排列四个用户自定义快速启动按钮(预设为HP主菜单、联系人、日程表、任务)。(^15080201b^)机身左侧上还有一个上下方向的浏览旋钮，旁边红色的按钮就是录音按钮。机身正面最下方的位置是椭圆形的电源开关，喇叭在其右边的位置。麦克风接口和红外线端口位于机身顶端的两侧，中间是CompactFlash TypeⅠ扩充卡的插槽。在插槽上方正中的位置有一个很醒目的电源指示灯，当你在充电时，指示灯呈橘黄色，充电完毕后呈绿色。 Jornada 545能够横向显示！这是一个隐藏的内建功能。当用户紧紧压住机身左侧的上下旋钮，同时将手写笔点在屏幕显示时间的位置约三秒，这时屏幕就会跳出一个Run和Clock的菜单，接着点选Run，并在出现的文字栏中填入“Rotate r”的命令后，用笔点击机身的复位孔，重新启动Jornada 545。之后屏幕上就会出现横向的启动画面，然后进入设置菜单中的调整屏幕选项，将触控点击变为横向显示后，就可以横向使用Jornada 545了。特别适合于阅读文本和浏览网页。 ●联想 天玑5000 联想最新推出的天玑5000，是国内自行研发的第一款基于Windows CE3.0的Pocket PC，它即将于2001年4月18日上市。当我们收到这台天玑5000时，机身背面还贴有研发样机的字样，不过其性能同上市的正式产品相差无几。天玑5000也是采用Intel最新的206MHz的StrongARM SA1110微处理器，配有16MB ROM和16MB RAM。(^15080201c^) 在这三款Pocket PC中，天玑5000尺寸最大，银灰色的流线型机身呈倒梯形，上宽下窄。香槟色的面板上除了液晶屏幕以外最显眼的要数椭圆形的十字方向按钮。与H3630不同，天玑5000的十字按钮没有连在一块，而是四个方向按钮绕着中间的确定按钮。这样操作起来更加灵活方便，同时也减少误操作的发生。在十字按钮的左面是喇叭，右面则是录音孔。十字按钮的左下方是一键上网按钮，右下方是快捷菜单按钮。机身左则上部有两个按钮：录音和电源开关。红外线端口在其下方喇叭水平的位置。耳机插孔和电源插孔则位于机身右侧喇叭水平的位置。CF扩充槽在机身顶部右端，手写笔筒紧靠其右。复位孔在机身背后铭牌的下方。天玑5000整体造型中规中距，给人留下稳健厚重的印象。 关于Windows CE 3.0 Pocket PC采用的Windows CE 3.0最大的诱惑力在于内建的Pocket Word和Pocket Excel，它们基乎就是台式机中Word和Excel的缩小版本，基本的操作功能相差无几，而且文件完全与台式机上相容(文件若中包含图片，需要转换格式)，使得台式机的Word和Excel文件与Pocket PC上的文件无碍交流成为可能。为了方便Pocket PC用户离线浏览网页，Pocket PC上已安装了Pocket IE。 Windows CE 3.0还做了许多程序和界面的改进，统一将开始菜单置于屏幕上端，操作选项置于屏幕底端，以便在操作手写笔时减少移动笔尖的距离以及点击目标和点选的次数，节约操作上所花费的时间，使其更符合掌上操作的习惯。还增加了许多的随机软件，在Windows CE 3.0当中，其个人信息管理(PIM)与台式机的联系大大加强了。联系人、任务栏、收件箱和日程表，都可以与台式机中同项直接交换文件，甚至可以替代台式机中的同项。由于使用中文版的缘故，Jornada 545和天玑5000都提供了农历的日程安排。中文版的Windows CE 3.0还取消了一些不适合国内用户的软件。 性能比较 我们从以下几个方面考察各家Pocket PC的性能。 ●特色应用程序 Windows CE 3.0仍然存在无法关闭当前任务程序的毛病，好在各家的Pocket PC中，都考虑到这一点，利用它们所提供的快捷程序，都能轻松解决这一难题，并提供更多支持。 康柏电脑上常见的“Q”快捷按钮也出现在H3630上面：提供了可关闭当前任务的Q Menu快捷菜单和Q Start快速启动都已经分别指定在面板右边的按钮上。值得一提的是Q Start快速启动，它能对H3630上的程序指定类别分组显示。而Q Utility快速设定工具则可以调整Q Menu快捷菜单出现的程序快速链接。由于是英文版的Windows CE 3.0，H3630还提供了能够查看BMP、JPG等图形格式的工具Picture Viewer。真希望在H3630的中文版本上还能见到它。 Jornada 545则提供了丰富的专用程序，在程序名称前都标有HP字样：HP任务切换器(在打开的程序和文档间切换或关闭)、HP设置、HP安全性、HP备份、HP游戏按钮、HP主菜单(打开Pocket PC上的程序或文档的快捷链接，可以自定义使用)。 天玑5000的快捷任务同HP任务切换器极为类似，除了在打开的程序和文档间提供切换或关闭，它还提供灯光、对比度和音量的即时调整。并提供了袖珍版的金山词霸，方便查询生词。 ●显示 H3630的背光具有显示5级亮度设置。并安装有环境光感应器，可以察觉周围环境光线强弱的变化，能自动调节显示背光亮度。在强光下显示效果仍然不错。Jornada 545的显示屏背光设置有四种，不过背光灯的亮度稍嫌暗了一点。在强光下显示效果仍然很一般。不会是受到机身颜色的影响吧。天玑5000也是5级亮度设置，比较起H3630来说，天玑5000仍要显得灰暗一些。 ●手写输入 与Palm不一样，Pocket PC的屏幕没有专用的书写区域，整个屏幕都可以作为书写区域。虽然H3630使用的是英文版的Windows CE 3.0，但是通过CE Star软件和蒙恬手写提供了中文输入的功能，也提供了全屏幕输入、小窗口输入和连笔划输入。由于H3630的屏幕的保护膜质地非常柔软，因此在屏幕上书写十分轻松，误差也很小。而Jornada 545和天玑5000除使用汉王手写输入，支持小窗口输入外，也可以全屏幕输入，在直接输入繁体字时，Jornada 545也能较好地识别出来，转化为相对应的简体字。在连笔划输入时，Jornada 545也有较高的识别率，这也要归功于它的液晶显示屏幕的保护膜较软，使得书写毫不费力，降低书写时的误差。天玑5000同Jornada 545一样，在各种手写输入中拥有较高的识别率，识别率与Jornada 545不相上下。 ●录音和播音 在录音时，我们选择以PCM格式进行录制。测试表明H3630拥有最好的录音质量，录放时的杂音都很小，音量也是这三台PocketPC 中最大的，完全可以用“洪亮”一词来形容，在播放MP3时，声音清晰响亮，效果非常好。而Jornada 545除非在很安静的环境下录音，否则播放时将会有较明显的杂音相伴。不过在播放MP3音乐时，其效果还不错，声音清晰，同H3630相比，音量要小一些，但是比天玑5000的音量要大一些。不过天玑5000在录音时噪音很小，而播放时音量比起Jornada 545相对要弱一些。 ●影像的放映 由于Pocket PC的Windows Media Player不支持常见的RM、MPG和MPEG格式。因此必须通过安装其他播放软件才能观看。使用PocketTV播放影像文件的测试时，虽然这几款Pocket PC的机器使用的是只有4096色的液晶显示屏，不过以肉眼观看时，并没有感觉到色差出现。由于H3630拥有环境光感应器，可以根据周围环境光线强弱的变化来自动调节屏幕显示，因此在播放MPG中效果最佳。由于CPU的强劲有力，不但在播放MP3和MPG时完全没有停顿现象出现，甚至还可以同时再打开运行其他的如Pocket Word之类的程序。相比之下，Jornada 545的播放效果也算是不错，影像和声音都很流畅。但是由于CPU处理能力较弱的缘故，要是播放影音文件的同时再开启其他应用程序，就相形见绌，不能胜任了。天玑5000也虽然同H3630一样能够在播放影音文件时进行多任务操作，但是屏幕效果要暗一些。 ●扩充性能 为了节省重量和体积，H3630采用外接扩充底座来接入扩充卡设备，并提供了CF和PCMCIA卡两种扩充底座选择。这使得H3630的扩充能力得到大大提高。而Jornada 545和天玑5000只有内建的支持Compact Flash TypeⅠ制式扩充卡的扩充槽，而没有外接的扩充装置。 ●离线浏览 我们在使用Pocket IE时发现，一个纯文本的网页在H3630上的开启时间比在电脑上打开时间要多花三五秒。天玑5000上所花的时间也与H3630大致相同。但是在Jornada 545上打开时，由于受CPU处理速度的影响，打开一个相同的页面时间大致是H3630和天玑5000的3倍，约半分钟时间。这可能是Jornada 545硬件配置落后的原因。惠普公司还有一款32MB内存的Jornada 548，也许在Jornada 548上打开页面的感觉就很不相同了吧，希望我们能有机会一试。 结 论 毫无疑问，流畅的多任务运行、和优秀的多媒体播放优异的整体性能将使H3630成为众多Pocket PC迷向往的目标。如果你的荷包十分饱满，而且又喜欢时刻与潮流同步，那么H3630绝不容错过。 当Jornada 545插上耳机关上保护外壳播放MP3时，完全就是一台MP3随身听；在离线浏览网页或阅读电子书时，可以利用横向的超大屏幕和调整保护外壳的角度来获取最佳的视觉效果。这些都是成为商务人士选择Jornada 545的理由。 不可否认，天玑5000的出色性能是我们始料未及的，它让我们彻底改变了国产掌上电脑研发落后的错误看法。随着天玑5000的上市，会引起更多人的关注目光，并将中国的掌上电脑事业推向一个新的高峰。(^15080201d^)
