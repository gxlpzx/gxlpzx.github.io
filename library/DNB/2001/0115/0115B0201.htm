武林兵器谱——文件管理器

虽然Windows已经从上个世纪十几兆的窈窕身材发展成为今天数百兆的庞然大物，但是它的许多功能仍像鸡肋一样让用户无所适从，资源管理器就是其中之一。为了弥补资源管理器的不足，许多优秀的文件管理软件应运而生，这些软件一般都具有多窗口、快捷键、文件编辑(包括十六进制编辑)、文件过滤、批量改名、多媒体文件预览、支持各种压缩格式、强大的网络功能等特性，而这些特性都是资源管理器不具备的，本文以WinNavigator为例介绍一些主流文件管理软件。6款软件资料如附表所示。 这些软件除了Super Explorer1.22需要到http://www.galcott.com/se.htm下载外，其它软件均可在华军软件园http://www.newhua.com下载。 ##1 匠心独具的WinNavigator WinNavigator(以下简称WN)在文件管理软件中算是老资格了，从1998年至今，它一直以强大的功能、别致的界面设计吸引着越来越多的用户。 ##2 1.程序界面 WN的界面经过匠心独运的设计，为文件操作提供了最大的显示空间(见图1)(^15020201a^)。A.将驱动器栏放在窗口上方，左右两个窗口去掉了资源管理器中的目录树，既可显示目录又可显示文件，；B.将Windows中宽大的滚动条细小化，置于下方；C.局部工具栏的设计更方便文件操作；D.文件显示窗口采用DOS风格，省去了抢占显示空间的边边框框。WN采用不同颜色或字体来区别文件类型，如绿色代表可执行文件，红色为文本、压缩文件，斜体表示隐藏文件等。 ##2 2.设置 A.基本设置：WN默认设置为中文显示，但是在显示某些参数时会有乱码，建议改为英文界面，此外还需更改其它一些设置。点击“配置”→“主选项”，共有五个选项卡，在“Main”中将“右键弹出菜单”选中，在“面板”中将“显示隐藏文件”选中，注意左右文件显示窗口需分别设置，别忘了右边也要选中啊(见图2)(^15020201b^)。好了，现在可以在“Language”中将语言设为英文了(当然如果你不介意乱码，也可一直使用中文)，WN在最新版本中增加了BIG5内码，方便了其他中文用户。 B.颜色、字体设置：一是可以设置界面颜色，点击“配置”→“改变颜色”；二是可以设置文件颜色，点击“配置”→“文件颜色”；此外还可在内置的七种配色方案中进行选择，点击“配置”→“颜色组织”，你会发现有许多熟悉的软件风格在等着你。字体可以在“面板”→“改变字体”中更改。 C.热点文件夹：该功能可以将常用的路径或命令收藏在一起，并赋予快捷键便于调用。点击“配置”→“热点文件夹”，这里既可设置热点文件夹又可设置热点命令，各按钮功能见图注(见图3)(^15020201c^)，以后就可以通过快捷键或局部工具栏上的按钮进行调用了。 D.用户菜单：如果你认为WN菜单可调用的Windows程序太少，可以通过用户菜单自行添加，点击“配置”→“用户菜单”进行设置。 ##2 3.文件和文件夹的基本操作 A.定位和选择：WN中的文件定位方式很灵活，一是在驱动器栏中选择盘符，然后在文件显示窗口逐级点击文件夹，找到所需文件；二是可以通过局部工具栏中的驱动器、根目录、热点文件夹进行定位，点击局部工具栏上的Sort按钮还可对文件进行各种排序。文件和文件夹的选择是十分重要的，与Windows的操作不同，WN内置了一套选择方案，如“Ctrl”和“”为全选，“Ctrl”和“+”选择当前一类文件，“Ctrl”和“-”取消当前一类文件的选择，空格键可以逐个选择，用“命令”→“选择一组”可以使用通配符选择，鼠标右键也有选择命令。 B.基本操作：WN的操作和资源管理器略有不同，用鼠标在相同或不同驱动器间拖动文件都是复制，“Alt+鼠标左键拖动”为移动文件，而“Ctrl+鼠标左键拖动”为创建快捷方式，此外，利用快捷键也可实现以上操作，F5为复制，F6为移动(更多快捷键请查看软件帮助)。通过“文件”菜单、鼠标右键菜单或快捷键可以方便地进行新建、删除、改名等操作。 C.压缩包管理：WN支持ZIP，ARJ，RAR，LHA，ARC，CAB等多种压缩格式，无需任何外挂程序，对于压缩包可以像文件夹一样双击进入浏览，进而双击文件进行预览，也可右键单击压缩包进行解压缩，还可创建ZIP，ARJ，RAR三种压缩包，未注册版中无法创建压缩包。 D.文件预览：可以预览文本文件，HTML文件，BMP、GIF、JPG等十余种图形文件，WAV，MID，MP3等音频文件，AVI，MPG等视频文件，功能十分强大。预览方式有两种，一是点击“面板”→“快速查看”，当前文件显示窗口会变为快速查看窗口(见图4)(^15020201d^)，如果是音频文件，下方的多媒体播放面板可就派上用场了，从左往右依次是上一曲、播放、暂停、停止、下一曲、循环播放，“PL”按钮是播放清单，点击它可以组建你自己的播放列表，图形文件、文本都将在预览窗口中显示内容，HTML文件则显示源代码；二是按F3打开独立的查看窗口进行浏览，对于图形文件将进入类似ACDSee全屏看图的界面，上下翻页键进行切换，加减号进行放缩，运行流畅快捷。 E.批量修改属性：文件选定后，点击“文件”→“属性”，可以设置或清除文件的各种属性，还能修改文件的创建、修改访问时间。 ##2 4.其它重要功能 A.文件编辑和十六进制编辑：对于文本文件，按F3后进入WN自带的文本编辑器，在这里可以进行查找替换、大小写转换等许多编辑操作。对于HTML文档，在右键菜单中选择“编辑”，也可进入编辑窗口进行修改。WN还可以进行十六进制文件编辑，在“文件”中选择“十六进制编辑”就可进入相应的编辑窗口。 B.Internet功能：WN内置了基于IE内核的浏览器，叫WebBrowser，当选中HTML文件按F3时，这个浏览器将自动打开，在这里可以像使用IE一样，尽情地浏览Internet网页。WN还内置了拨号工具，点击工具栏上的“电话”按钮即可进行拨号。 C.其它：WN还内建有CD播放、系统信息、拨号网络、记事本等快捷图标，还能够进行MP3到WAV之间的格式转换，为当前文件夹下的文件制作文件列表，文件名称的大小写转换，比较文件夹等。 务实去华的Explor2000 Explor2000声称是没有菜单的文件管理软件，实际上该软件只是将菜单文字以图标代替，将菜单栏、工具栏融为一体而已。Explor2000借鉴了资源管理器的目录树和鼠标键盘的组合操作方式(如“Shift+鼠标左键拖动”为移动文件)，在此基础上新增了一个文件显示窗口，鲜明的色彩设计、实用的功能、简单的操作使得它成为一款优秀的文件管理软件(见图5)(^15020201e^)。该软件的特色功能有十六进制编辑、预览十余种图形文件、文件夹同步、支持十余种类型的压缩包、丰富的快捷键、快速进入DOS路径……等等。 追求功能的Turbo Browser 2001 如果按功能多少排序的话，Turbo Browser 2001(以下简称TB)是当之无愧的冠军，它集成了丰富的Internet功能，支持大多数文件格式，使用户随时随地都能轻松地处理文件，图6为用TB播放下载到硬盘中的ASF流媒体文件(见图6)(^15020201f^)。 TB的主要功能有：1.强大的网络功能，如FTP文件传输、HTTP上载、下载、路由追踪、Telnet、E-mail发送；2.HTML预览编辑功能，TB采用双窗口方式处理HTML文件，上方显示源代码，下方为预览效果，可贵的是TB提供的HTML编辑功能十分强大，具有突出显示语法，自动刷新预览，无限制Undo/Redo等功能；3.强大的预览功能，可以预览Office 2000、PDF、CAD、图片、E-mail正文附件、Zip等压缩包、MP3等音视频文件、甚至ASF等流媒体文件；4.图片预览工具，如放缩、捕捉颜色数值、旋转剪切等；5.采用压缩格式备份文件夹；6.良好的程序源代码编写环境，支持十余种语言语法，语法突出显示；7.其它，如十六进制编辑、文件批量改名、离线浏览Cache中的HTML文件等等。 口碑极佳的Windows Commander Windows Commander是口碑极佳的文件管理软件，它界面朴素、布局合理、操作方便灵活、功能丰富，许多特点被同类软件竞相模仿(见图7)(^15020201g^)。该软件内置两种操作方式，一种是标准的Windows风格，一种是NC风格，可在“配置”→“选项”中设置。 它的特色功能主要有：支持中文，支持文本、音频、视频等文件的预览，十六进制编辑，支持各种压缩包，FTP功能，支持中文双字节，支持Unicode编码，能够分割、合并文件，强大的批量改名功能，转换文件编码，文件夹同步等。以“文件分割合并”为例，介绍该软件的特色功能，选中文件后，点击“文件”→“分割文件”，在窗口中设置分割大小即可，合并时双击第一个分割包就能自动完成文件合并。 Windows Commander虽然十分优秀，但它仍有一些不足，如文档只能预览，不能编辑，图形仅支持BMP格式，不能浏览HTML文档等。 小巧玲珑的Super Explorer Super Explorer最大的特点在于界面直观、操作简捷，此外该软件容量小、功能强大，也是一款优秀的文件管理软件(见图8)(^15020201h^)。 特色功能有：可以同时打开四个文件显示窗口，并能轻松排列，能预览Office文档、MP3、图形等多媒体文件，批量改名，加/解密，彻底粉碎文件，显示System、User和GDI资源，支持压缩包，六种格式的图形转换等。 功能突出的EF Commander EF Commander的界面类似Windows Commander，它的容量是这几款文件管理软件中最小的，但是功能却十分强大，如：十六进制编辑，支持中文双字节，查看/编辑文本(编辑功能很强)、HTML文档，预览图形、音频等多媒体文件，支持压缩包，分割合并文件，FTP文件传输，PING命令，文件编码转换，目录同步等，值得一提的是MP3播放界面显示信息是最全的(见图9)(^15020201i^)。 ##1 小 结 介绍了这么多文件管理软件，朋友们是否有了“乱花渐欲迷人眼”的感觉，实际上优秀的同类软件还很多，朋友们在选用时只要本着好用够用的原则即可，不一定非要追求功能最多，因为功能复杂化势必产生界面复杂、性能下降的弊端，那你的工作效率就不升反降了，当然如果你的爱机性能够强劲，那就充分发挥它的潜力吧。
