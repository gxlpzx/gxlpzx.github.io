自制WebCam网页

WebCam 是将摄影机(Cam)与 Web 网页结合在一起的网络应用，它让参观网页的人可以看到远程摄影机实时所拍摄到的画面，就像看卫星转播一样真实。 CoffeeCup WebCam 就是一个专门用来制作 WebCam 网页的工具，它可以将你自己计算机上摄影机Cam拍摄到的画面，送到网站上，让参观网站的人能够实时看到你摄影机所拍摄的内容。 ##1 一、CoffeeCup WebCam的主要功能 CoffeeCup WebCam支持多重摄影机与多重网站，可以将任何摄影机拍摄的画面上传到任一个指定的网站，或是依照使用者事先安排好的行程传送。而且支持动作侦测的保安功能，若侦测到摄影机监视的画面有任何东西移动时，就会立即发送 E-mail、上传图片、或发出警报声音报警。 此外，CoffeeCup WebCam还有以下功能： ●Fast Cam支持快速影像更新，可以自动实时更新网页上的影像； ●可以同时连接数个摄影机，并且兼容绝大部分 USB 及并行端口摄影机； ●可以和网页的访客对谈聊天； ●可以协助使用者建立并上传非常酷的个人网页； ●可用多个摄影机轮流上传至不同网站； ●可以截取并储存成 .avi 影片； ●可以建立最爱影像目录供快速重新读取及上传； ●可使用 JavaScript 更新网页，不需要任何 Applets； ●可以自行设定标题和日期的字型、颜色、尺寸。 想要架设 WebCam 的人一定要试试 CoffeeCup WebCam，虽然受限于目前网络传输的速度限制，不可能有卫星转播的显示效果，但是依然有相当多的地方可以使用这种技术。比如，在家里装设一个摄影机，而在办公室便可以看到家里的状况(可用于监视家里有无小偷)。 ##1 二、CoffeeCup WebCom的使用 第一次打开CoffeeCup WebCam，将会在设置向导的提示下一步一步完成必要的设置： 首先是FTP服务器的名称以及给远程目录起个名字，注意不要在FTP服务器名称前加http://或者ftp//，文件夹的名称要注意你的服务器是否对大小写敏感。 接下来就是填写你在服务器上申请的用户名和密码。 然后你选择是通过局域网还是调制解调器连接互联网，如果是后者，还要进行拨号设置。 最后请选择你所使用的摄影设备，设置便大功告成。 进入CoffeeCup WebCam 的主界面(如图)。最上面是菜单、工具栏，中间是预览屏幕，下面是任务状态窗口，一看便知该怎么做。 点击工具栏上的第五个按钮进行完Image Settings(图像设置)后，即可开始制作影片(工具栏上的第四个按钮)。 CoffeeCup WebCam 还提供有一大堆的诸如浮雕、锐化、腐蚀、柔化、模糊、翻转、透视、轮廓、波纹、漩涡等等特效滤镜，你可以一个一个试一下，挑一种最酷最炫的特效馋死别人…… 做好后，可以点击工具栏上的第一个按钮预览做好的影片，如果满意，即可上传(按第三个按钮)。(^15040402a^) 一切顺利的话，很快你(当然还有所有网友)就会看到自己的杰作已经展现在互联网上了！ 软件性质：Trial(试用)；试用限制：21 天；购买金额：美金 30 元；更新日期：2000-09-21；最近版本：3.0；语言界面：英文；文件大小：3.2 MB；操作系统：Windows 95/98/2000/Me/NT。 下载地址：http://www.coffeecup.com/或者http://meyou.126.com
