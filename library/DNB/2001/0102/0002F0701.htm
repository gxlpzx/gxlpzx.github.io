专家坐堂

如何确定SDRAM的性能指标
问：这一段时间，内存价格“猛”降，我很想趁便宜的时候买一条128MB的内存条，可是关于内存条性能指标较为全面的介绍太少，报刊杂志上看到的最多也只是如何识别PC100和 PC133的文章，我想请问一下，什么样的内存条才是最好的？
答：目前使用的内存条一般都是SDRAM的，这种内存条为168线接口，带宽为64位，3．3V电压。在购买时主要要注意区分PC100和PC133两种规格，同时还应该注意一下有无ECC奇偶检验，SPD是否损坏以及芯片和电路板的做工等，以免买到假货。
安装Win2000使用那种格式？
问：我想请问一下，安装Win2000使用那种格式比较好？
答：Win2000本身支持三种格式：FAT16、FAT32以及NTFS格式，如果你经常打游戏或者你的机器还需要运行Win98的话，你最好使用FAT32格式的，如果你打算将你的机器做为服务器使用或者需要较好的安全性能、用户管理机制的话，最好使用NTFS格式。
MPEG4无法正常播放
问：我最近攒了一台电脑，配置是CPU赛扬－2/566；主板：硕泰克65JVB－X；声卡：创新SBLIVE数码版；内存：金条PC133 128MB；显卡：RIVE128。此机运行大型程序和播放其他类型光盘时都很稳定，可播放MPEG－4的光盘时，当音效和人物对白声音平稳的时候一切都很正常，可一旦出现高音或低音(如爆炸声、轰鸣声、交响乐等)音箱中就会出现非常嘈杂的爆音(音箱决无问题)。希望你能帮我解决这个大问题。
答：总的来说，MPEG－4对机器的要求并不算太高，你的机器配置已经足够了。如果你的音箱质量不差，且在播放MP3或者是VCD时都没有出现上述问题的话，你不妨从以下几个方面考虑一下：不要使用太高的采样频率，在Windows Media Player的“文件”菜单中选择“属性”，再选中“高级”，将其采样频率选择为“Half”或者更低；保证MPEG－4的光盘质量；使用Win98第二版或者是到微软的官方网站下载较高的Windows Media Player版本。
为何不能正常上网
问：我使用的是一个不知道品牌的Modem，连接速度也还不算太慢，因此也就一直凑合着，可是近来拨号时总是报告：“拨号网络适配器没有安装，请单击网络属性查看”。请问我该如何解决这个问题。
答：拨号网络适配器是拨号上网必需的，你可以将你的Modem驱动程序以及TCP/IP协议等全部删掉后重新安装，系统将会自动给你推荐一个拨号网络适配器。
实现单机双线上网
问：由于本地实在偏远，电信局只提供Modem上网，不支持ISDN服务，于是我们的上网速度始终只有到56K的“终极”速度。我想请问一下，如果在条件可以的情况下，是否可以实现单机双线上网，怎样做，会对电话打入有影响吗？
答：首先单机双线上网是可以实现的，但是当连接成功以后，这两部电话都将无法打入，这和一部电话上网的情况是差不多的。具体的实现方法如下：1．除了两个Modem和两根电话线以外，你还必须要向你的ISP申请一个账号，是一个允许多个用户同时拨入的账号(现在各地的163账号都应该支持该功能)；2．按照普通方法建立一个新的连接，然后选择该连接的“属性”中的“多重连接”标签，在其中添加第二个Modem；3．在拨号上网的连接过程中，当第一个连接成功实现后，系统将自动连接第二个，在断开时，先断开第二个。
制作自己的屏保图像
问：初学者怎样才能直接把自己的照片制作成屏保图像？另外，是否还可以配上自己喜欢的音乐？而不再使用系统自带的屏保程序。
答：系统支持以．SRC结尾的屏保程序，而你的照片显然不是这种格式，因而不能简单地直接把你的照片当成屏保图像。你可以安装一个屏保制作工具，比如：Screen Save Builder，使用它对你的照片(最好是JPG格式的)稍微编辑一下，并添加声音后，保存为SRC格式，放到c\Windows目录下，并将该文件选定为当前屏保程序就可以了。
为何Windows Me中
不能删除IE？
问：我的机器现在安装的是Windows Me，使用起来感觉不错，但由于没有上网，为了尽可能地节省空间，想把自己根本用不上的那些系统自带的东西给删掉，可是我发现Windows Me中的IE没地方删？请问IE怎么个删法？
答：我建议你不要删除Windows Me中的IE，这是因为Windows Me系统在运行时，经常要调用到IE的内核，如果没有它的话，系统极有可能无法正常工作，比如：你将可能无法使用Windows Me的帮助或者资源管理器属性等等，这些都必须使用到IE的内核程序。因此Windows Me不能像删除其他组件一样删除IE，如果你确实想尝试一下，你可以使用专门的IE分离工具实现删除目的。
降低Excel的计算精度
问：我经常使用Excel完成日常工作中的报表，可是Excel在计算时往往计算得过于精确，比如：10/3，得3．333333而我只需要3．33就可以了以往都是自己在抄数据时自己省略的请问有什么办法可以降低Excel的计算精度﹖
答：以Excel 97(Excel 2000使用方法相同)为例，你只须选中你需要规定数据精度的单元格，单击右键，在快捷菜单中选择“设置单元格格式”，在“数字”标签中选中“数值”一项，然后在窗口的右边选择你所需要的小数点位数即可。
如何使用Windows Me的
系统还原﹖
问：我安装的是Windows Me，其实我觉得它也没有想象中的那么好，因为我又把它弄死了，听朋友们说，Windows Me有一个很方便的“系统还原”工具，可是不知道它在什么地方？无须给出非常详细的说明，只要说明它在什么地方以及在使用过程中应该注意些什么就可以了，谢谢！
答：简单地说，Windows Me 的“系统还原”工具实际上是一个后台工具，它能够记录你在重大更改之前的系统信息，使用方法很简单，你只要打开“开始”菜单中的帮助，在打开的网页上面选择“使用系统还原”，然后根据系统提示进行操作就是了。在这里需要注意的是，恢复的只有应用程序和注册表设置，而对我们常见的文档文件并没有恢复作用。
区分RGB和CMYK彩色模式
问：我有一台HP Color LaserJet 5M打印机，在打印设置中发现有RGB和CMYK彩色模式，它们之间有什么区别呢？
答：虽然HP Color LaserJet 5M打印机使用CYMK墨粉，但不是一定必须要向它发送CYMK数据。大多数桌面彩色激光打印机接收RGB数据时，会得到最好的输出效果。CMYK主要应用于商用印刷行业。在图形软件包应用于商用打印机之前，必须由专业图像人员来配置。如果你的Photoshop或者类似的应用软件还没有正确地配置，不要转换到CMYK模式。配置过程相当复杂，通常需要一种彩色测量装置，比如分光光度计。还要记住转换彩色模式可能导致数据的丢失。每一种模式的彩色全范围不同，当你转换到较小的模式时，数据不能被恢复。因为CMYK是最小的彩色模式之一，转换到这种模式往往是在打印前做的最后一件事。这也是ColorSmart所要的工作，因此没有必要这样做。但是向HP Color LaserJet 5M打印机发送CYMK PostScript(R)数据时，ColorSmart就不工作了。在PCL(打印机控制语言)中，驱动程序保证打印机只接收RGB数据。
何为Tiny BGA封装方式
问：听说创胜的内存用了Tiny BGA封装方式，请问Tiny BGA封装方式与其他的封装方式有何区别？还有内存的规格中CL＝2与CL＝3有什么区别？内存是插单条128MB的好，还是插二条64MB的好？
答Tiny BGA封装方式是胜创公司研发的新一代内存颗粒封装方式。它与传统的TSOP内存颗粒封装方式相比，利用胜创的TinyBGA封装技术可以减少颗粒与母板之间的接脚距离，并降低内存的杂讯干扰，以及加强颗粒的散热效果等优点。CL(Cycle Lenght)是指SDRAM的周期延迟时间，一般其值为2ns或3ns，而数字越小越快。至于内存是插一条128MB还是插2条64MB为好，笔者认为还是插一条128MB比较好，因为这样可以节省一条内存插槽。
如何录制VCD中的音乐
问：我在Windows 98第二版中，想把VCD中的一段大概50秒的声音录下来，我该怎么做，需不需要什么特别的软件？
答：如果只要录制50秒的音乐，可以利用“开始/程序/附件/娱乐”里的“录音机”。首先你只要将声卡的Out端用端子线接至Mic端，然后用播放器播放你的VCD，到了你想要开始录的那段音乐时，打开录音机程序，并按下右边的红色按钮“录音键”，就可以开始录音了。如果Mic没有声音或录制出来的音量太小，要检查一下右下角任务栏的音量控制(在黄色小喇叭上按两下)，到“选项/属性”中，在“调整音量”中点选“录音”，然后在“显示下列的音量控制”中把“麦克风(Mic)”选项打钩再按确定。这样就可以出现控制录音音量的画面，你可以调整麦克风的音量大小。
解决电视卡无讯号输出问题
问：朋友送我一块皇朝KT1000TV的电视卡，但是其电视画面的部分图像出不来，就是没讯号，这是什么问题，应如何解决呢？
答：首先你先检查你的电视卡的线路是否安装正确，仔细查看电视卡说明书上的安装方式，再检查驱动程序是否安装正确。因为没有讯号，所以信号线的连接也要注意，通常电视卡都是用一条25pin的接脚接至原先的显卡，再用一条25pin的线从电视卡连接至显示器才会有讯号。有些显卡还需要支持DirectDraw才可以正常工作。另外，还要注意一下它的工作制式，即NTSC(美制)和PAL(欧制)制，要调整为正确的制式才能正常工作。
解决SCSI接口卡与声卡的冲突
问：我的主板是技嘉GA－6CXC，最近买了一台刻录机和SCSI接口卡。当我安装好SCSI的驱动程序之后，却发现IRQ有冲突，系统提示我删除或更改IRQ，但当我把内建的声卡给删除掉后就好了，但这样做就没有声音了，其中断地址是IRQ11，请问我该如何解决此问题？
答：这是由于声卡的IRQ中断与SCSI接口卡的IRQ中断冲突。建议你在“控制面板”里的“系统”里查看一下“设备管理器”，选择计算机的图标，再按“属性”按钮，就可以找到所有的“中断IRQ”。通常计算机的IRQ数值有16组(0－15)，你可以看看那一组IRQ是空的，将这个数值让给SCSI接口卡使用。如果全部的IRQ都被占用的话，建议你可以把一些平常没有用到的设备给移除掉，如COM2，可以让COM1接Modem，而鼠标接在主板上的PS/2接口上，这样就可以空出一个IRQ给多余的装置来使用了。
利用ATA－66扩展卡
可否设定开机顺序
问：本人电脑的IDE接口上已全部接满，三块硬盘再加上一个光驱，目前我想再加装一台刻录机，想请教专家以下问题。如果我买一块ATA－66的扩展卡，是否可以将原有的第一个硬盘(Seagate)接上去，并设定为启动盘。这些盘是C盘 (Seagate 13GB ST313032A Ultra DMA 66硬盘，接在ATA－66扩展卡上)；D盘(IBM 6．4GB 5DHEA硬盘，接在IDE1上，为Master)；E盘CONNER 850MB硬盘，接在IDE1上，为Slave)；F盘刻录机，接在IDE2上，为Master)；G盘(华硕40×光驱，接在IDE2上，为Slave)。其开机顺序是先由Ultra DMA 66先开机，还是可由用户自己设定？
答：Ultra DMA 66会接管开机程序，所以你可自行设定是利用ATA－66接口卡上的IDE设备或是系统的IDE设备来开机。当然用接口卡上的设备来开机时，Ultra DMA 66的硬盘便会成为第一顺序的设备。至于光驱与刻录机的顺序，可自行于Windows的“控制面板/系统/设备管理器”的CD－ROM项目中进行设定。
何为SCSI接口
问：请问SCSI接口是什么标准，它的性能如何？
答：SCSI(Small Computer System Interface)是将各种设备连接到计算机之扩展总线上的一种接口标准。使用SCSI接口卡除了传输速度较快之外，因为它独特的设计，使CPU的负担较小，传输也较为稳定。一块SCSI接口卡视不同的形式，可连接1至7、1至15或是1至31个内接或外接SCSI设备，包括磁带机、硬盘、扫描仪、MO光驱、CD－ROM、数字录音带(DAT)、刻录机(CD－R、CD－RW、DVD－ROM、DVD－RAM)等。依照版本演进的不同，SCSI又有SCSI－1、SCSI－2与 SCSI－3之分，其中SCSI－2又分为 SCSI－2、Fast SCSI－2(Ultra SCSI－2)、Wide SCSI－2(Wide Ultra SCSI－2)等几种，其中SCSI－1、SCSI－2具有8位的数据总线，传输速率分别是5MBps、10MBps，而Fast SCSI－2与Wide SCSI－2则都是16位的数据总线，传输速率分别为20 MBps与40MBps，较新的SCSI－3、Ultra SCSI－3则具有32位的数据总线，传输速率分别达到到80MBps、160MBps。要注意的是，外部设备的种类必须要和所使用的适配卡相匹配，这些SCSI设备才能正常地工作，这些规格都是向下兼容的，也就是说SCSI－2的接口卡可接所有SCSI－1、SCSI－2的设备，但是SCSI－1的接口卡只能接SCSI－1的设备。有些主板厂商，已在一些主板上内建SCSI接口芯片，但是如果使用的SCSI设备太多，则要添购一块SCSI接口卡。
去掉JavaScript脚本出错的提示
问：最近上网时，常常会出现类似JavaScript脚本出错的提示。我想将它关闭，应该怎么办？还有，JavaScript究竟是什么，它有什么作用？
答：JavaScript是由Netscapt与SUN公司合作开发的产品。JavaScript是一种跨平台的开放式脚本描述语言，它是Java和Livescript的衍生物，它生成的应用程序能够将客户和服务器上的对象和资源联系在一起，通过Internet动态地表示信息并与用户进行交互。JavaScript是用来改变网页外观的编辑命令和指令，它对网页设计有着重要的帮助。JavaScript脚本并不像真正的程序那样须经过编译，也不能单独运行，它只是创建一种指令告诉浏览器该做什么，网页开发者可以把它放在HTML文档的任意一个地方，当支持JavaScript的浏览器读到这些指令时即按要求完成这些动作。它可用来完成从动画制作到数据库连接等的所有事件。
如果你碰到JavaScript错误的提示，那么表示你正在浏览的网页的JavaScript语句有语法错误。如果你是使用IE 5．0的，那么建议你到“工具”/“Internet选项”里的“高级”卷标中，去掉下方的“显示每个脚本错误的提示”，就会不再出现上述提示了。
