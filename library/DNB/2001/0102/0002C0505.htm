让你的"视"界更精彩--轻松学用PowerStrip

　　PowerStrip是目前最优秀的显卡和显示器调试、超频工具，它不仅能很好地管理有关显示配置的方方面面，而且我们还可以通过它完成对屏幕大小、色彩、字体、屏保、节电模式、刷新频率等项目的设置。更绝的是它能够检测并优化你的显卡，甚至更改显卡工作频率，最大限度地发挥显卡的潜力，达到最理想的效果。
　　一、下载和安装
　　目前，PowerStrip的最新版本是3．0 Beta版(暂时未提供中文语言包支持)，同时支持Win9x/Me/NT/2000，朋友们可以到http//entechtaiwan．com或驱动之家(http//www．mydrivers．com)下载该软件。下载后得到一个名为ps3beta的可执行文件，大小约425KB。然后直接双击这个可执行文件就可以安装了，安装完毕该软件会提示你是否设置随系统启动而启动(依个人需要而定)。
　　二、PowerStrip功能详解
　　右击系统托盘区的PowerStrip图标，从快捷菜单中选择“Options”(选项)，在“Option”窗口中可以看到共有“Adapter information”、“Monitor information”、“Hot key manger”、“Resource manger”、“Desktop icons．．．”、“About”等六个命令：
　　1．命令说明
　　(1)Adapter information(显卡信息)：，从这里你可以看到非常详尽的显卡信息：共有PCI配置信息、AGP配置信息(当前显卡工作在AGP2×，新版本的PowerStrip支持AGP8×)、安装图形驱动程序、报告四部分。
　　(2)Monitor information(显示器信息)：该选项能清楚地显示出你机子的显示器信息：一共有Product identifiers(产品识别)、Displays parameters(显示参数)、Color Characteristics(颜色特征)、Power management(电源管理)、Timing descriptions五项内容。
　　(3)Hot key manger(热键设置)：通过对各种功能，如刷新桌面、优化内存等进行热键的设置，你可以很方便地启用这些功能。方法是：你先在“Functions and profiles”下拉列表框中选择某一功能，如“Refresh desktop”(刷新桌面)，然后在“Hot key”下面的“Global modifier”部分勾选组合键(“Ctrl+Alt，Ctrl+Shift，Win+Shift”中任选一种)，接着就可在按下这组组合键的同时按下另一个基本键，如“Ctrl+Shift+F7”，这就定义好了热键。要注意的是，必须将光标定位在“Hot key”文本框内，否则无法定义热键。另外也可以同时定义多个热键。
　　(4)Resource manger(资源管理器)：该命令以一个坐标图的形式向我们清楚地演示系统中的物理内存和当前可用内存及GDI资源等(图1)，当选中窗口右边的“Idle CPU”和“Automatically compact memory”这两个复选框之后，PowerStrip就会在CPU空闲时自动整理内存，当然你也可以随时点击“Compact memory now”按钮手工进行内存优化。拥有64MB以上内存的朋友不要忘记哟！
　　(5)Desktop icons．．．(桌面图标配置)：这里有“Save icon placement”、“Restore icon　placement”、“Auto－arrange icons”三种选择，它们分别是保存桌面图标、复原桌面图标、自动重排图标。
　　(6)About命令：一般来说软件中的“About”都是提供版本信息之类的东东，但这里的“About”还提供了一些有关该程序运行的设置功能，如是否随系统启动而启动等。
　　2．设置显示器
　　(1)右击系统托盘区PowerStrip图标，从快捷菜单中选择“Display profiles”下的“Configure”命令，打开如图2所示的窗口，左边可以对当前屏幕颜色进行设置(256色/16色/真彩32位)。
　　(2)我们还可以设置显示器的最大分辨率和最大刷新率，而且操作异常简单，只要分别拖动图2中“Resolution”和“Refresh rate”下方的滑杆到适当值即可。假如设置不当引起黑屏也不要紧，仍旧进入这里选择“Reset adapter defaults”恢复到默认设置就行了。
　　(3)点击“Advanced timing option”按钮还可打开调整显示器屏幕的窗口，在这里你可以调节显示窗口大小和位置(无论你的显示器是数控还是模拟的)。这个功能对于某些老式的不带调节键或是调节键失灵的显示器特别有用。 (4)从快捷菜单中选择“Colour”下的“Configure”命令，可以打开颜色调节窗口，其中有几何失真、亮度、对比度、温度四个部分，只要分别拖动它们下方的滑杆就可进行调节，而且右边的预览区会立即显示出调节后的效果。假如你觉得效果不佳，可以随时点击“Reset”恢复到默认状态。
　　3．显卡超频设置
　　右击系统托盘区图标，从快捷菜单中选择“Performance profiles”选项中的“Configure”命令，会看到如图3所示的显卡工作频率设置窗口，虽然同样只须简单地拖动滑杆到你需要的位置，不过危险性也相当大，朋友们最好先将显存频率和显示芯片的运行频率设置为一个较低的数值，然后逐步调高频率，调整后单击“Apply”即可。如果调整后显示正常的话，你可以单击“OK”，并保存调整结果。这样你就能将显卡逐步“超”到允许的极限。除此之外，你也可以从系统Windows文件夹中找到PSTRIP．CFG文件，打开该文件后搜索到你的显卡芯片，找到后把带有“MClk＝”字样的文件后面的数字改大些(这个数字其实就是显存的工作频率)。大家在调整时务必要仔细查看显示卡和显示器的说明书，尽量不要超过显卡、显示器所能支持的最大值，否则烧坏显卡或显示器就麻烦了。 假如朋友们对此软件的用法有什么问题，欢迎与我联系，笔者的E－mail是wzj9999＠yesky．com。 (江苏 王志军)
