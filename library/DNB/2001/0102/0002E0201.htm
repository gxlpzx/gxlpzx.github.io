两款代理服务器软件的使用

绝大多数的网吧都是使用一台Modem加一条电话线使整个局域网的计算机上网，此时的关键在于选择一个合适的代理服务器软件。代理服务器软件的选择应当主要基于两方面考虑：一是设置简单易于使用，二是功能强大方便管理。这里，给大家介绍两款代理服务器软件：Ccproxy和Wingate，基础差的朋友可以用Ccproxy——简单易用，基础好的朋友用Wingate——功能强大。
Ccproxy
Ccproxy也许是我见过的最简单的代理服务器软件了，它是一款免费软件，目前最新版本为3．0，其主页地址为http//microserver．top263．net。该软件的使用非常简单，下载后文件名为ccproxy 3．0．zip，该压缩文件不到120K，非常小巧。解压后有两个文件，文件名为：ccproxy．exe和ccproxy．ini。前者为软件主程序，后者为配置文件，可用记事本打开编辑以更改配置。
服务器端设置：
在“控制面板/网络/配置”中双击与网卡绑定的TCP/IP协议打开其属性对话框，为服务器指定IP地址192．168．0．1，子网掩码为255．255．0．1，单击“确定”(注：需要Windows安装软件)。重启后执行文件ccproxy．exe会在桌面任务栏上出现图标，软件主界面如图1所示。如希望Windows启动时ccproxy自动执行可选中“加入启动菜单”项。
客端设置：
同样，在“控制面板/网络/配置”中双击TCP/IP协议打开其属性对话框，单击“网关”选项，在新网关内输入刚才为服务器设置的IP地址192．168．0．1单击“添加”按钮，再单击“确定”。重新启动后，启动Internet Explorer选择“Internet选项”菜单项，在“连接”选项卡内选中“通过局域网连入Internet”和“通过代理服务器访问Internet”，再选择“代理服务器”内的“高级”按钮设置代理服务器，须设置http服务器地址为192．168．0．1，端口号为808，socks服务器地址为192．168．0．1，端口号为1080．
到此，只要在服务器上连通Internet，就可在客户机上使用WWW服务了。若要用OICQ聊天，须使用“注册向导”登录，并在“网络设置”这一步选中“使用PROXY SOCKET5防火墙”，并在“防火墙地址”栏输入服务器地址192．168．0．1，端口号为1080。
Wingate
如果你有50台以上的计算机要通过一个出口(如一条DDN专线)上网，也许希望对不同的用户给予不同的权限，并提供电子邮件、FTP等服务，那么用Ccproxy这样的代理服务器软件是不可能的。另外，如果你的服务器不得不成为二级代理服务器，那么Wingate将是最好的选择。
安装
正式安装前要进行与Ccproxy相同的工作，即设置作为服务器的计算机的IP地址和子网掩码分别为192．168．0．1及255．255．255．0，客户机的IP地址为192．165．0．1－192．168．0．255。
值得注意的是，在出现选择安装类型的界面时，若是安装客户机则应在“install type”一栏中选择“configure this machine as a wingate internet client”，若是安装服务器，应选择“configure this machine as a wingate server”。
设置与使用
启动Wingate的设置管理工具Gatekeeper(有人翻译为“看门人”)，它会先要求输入口令，其初始口令为空；然后WinGate提示输入新口令，确认后回车即可。然后就可以看到Gatekeeper的主界面(如图2)，Wingate的管理就通过它来完成。
由图可见，Gatekeeper主窗口分为左右两部分，如果你看不到左窗口，请选择“view/control panel”，左窗口有三个标签，分别列出了已预设的各种服务：System面板显示的是系统服务；Services面板显示的是用户服务；Users面板用于用户管理；右窗口列出的是Wingate的工作情况，如各客户机都访问了哪些站点等。

显然，我们需要的不仅仅是这些，下面看看如何来完成电子邮件服务：
Wingate预置了POP3服务，却没有SMTP，因此我们要做的第一件事就是添加SMTP服务。在Gatekeeper左窗口单击Services标签，再在左窗口中单击鼠标右键，在快捷菜单中选择“new services/TCP Mapping Services”，弹出如下对话框，Service栏与Description栏分别是服务名称与说明，你可按自己最容易识别的方式填写，再选中复选框“Enable default mapping″，在“Server”栏中填入ISP提供的SMTP服务器地址，若你连这也没有，也可填入免费邮箱服务商提供的SMTP地址，如“SMTP．163．NET”，然后在“on”栏中填入端口号25，然后单击“OK”，服务器端就设置好了。
在客户端以Outlook Express为例，只需对账号的服务器属性如图3设置即可。这里需要注意的是帐号名的设置，是在原账号名后加“＃”号，再跟上POP3服务器地址。现在单击“确定”后你就可以用Outlook收发邮件了。如果用其它的电子邮件软件如Foxmai其l设置方法类似于此。
Wingate的功能还远不止这一点，如二级代理功能。笔者所在单位使用一条专线进入163，代理就是用的Wingate。现在笔者办公室离单位较远，需拨号进入单位网络，而办公室中还有一个小局域网。为了少用几只Modem和电话线，只好做一个二级代理，其实也很简单，只需添加一个“UDP Mapping Services”服务。
其实，Wingate不是想象的那么复杂，但要用好Wingate也不容易。它的最新版本是4．1，其主页地址为http//wingate．geerfield．com，也可以到中国代理的网址http//www．CloudNet．com．cn/download去下载。Wingate本身又根据功能分三种，分别是：家庭版(home)、标准版(standard)、企业版(pro)。
到此，两款代理服务器软件的功能就介绍完了，是不是很简单？各位有什么想法可以email与在下交流，我的E－mail地址是：tutuempire＠163．com。
