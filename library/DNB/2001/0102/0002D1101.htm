Dreamweaver 4新功能实战

　　自定义快捷键
　　利用快捷键可以简化操作步骤，加快操作速度，因此深得一些高手的喜爱。许多软件都设计了快捷键，但不同软件的快捷键的设置方法不同，而且一成不变，你无法改变。Dreamweaver4的出现改变了这种状况，它独创了一个自定义快捷键的功能，有了它，你就可以随心所欲地定义各种操作命令的快捷键，再也不受软件设计者的约束了。
　　下面我们就来对Dreamweaver4的快捷键动个“手术”，让它按照我们设置的方式来操作。单击“edit→keyboard shortcuts”命令，出现快捷键定义窗口(图1)，首先我们来熟悉一下这个窗口界面。 current set下拉菜单中显示的是系统内置以及用户自己建立的快捷键方案，系统内置的有BBEdit、Dreamweaver3、homesit和macromedia standard四种。
　　commands下拉菜单显示的是与current中选择方案相对应的命令分类目录，以便大家选择不同的命令进行编辑。例如，你在current set中选择的是macromedia standard方案，那么在commands中就包括menu commands(菜单命令)、site menu commands(站点菜单命令)、code editing(代码编辑)、document editing(文档编辑)、site window(站点窗口)五类。
　　下面的一个窗口就是命令列表，单击“+”可以展开菜单，选择不同的命令进行编辑，单击“－”可以折叠菜单。
　　shortcuts(快捷键)窗口显示选择命令的快捷键，单击加号按钮添加新的快捷键，单击减号按钮删除快捷键。
　　press key中显示你键入的新的键名，单击右边的change可以使新输入的快捷键生效。
　　current set 右边的四个按钮分别用来完成对被选择方案的操作，它们的功能分别是：
　　duplicate set：复制方案， 由于系统内置的四种方案不能改变，如果你要对某种方案进行修改，首先要把该方案复制，然后在复制的方案上进行修改。 rename set：重命名方案。
　　export set as html：把快捷键方案输出为一个Html文件，以便于查看和打印。
　　delete set：删除方案。
　　熟悉了各部分的作用后，下面我们就来看如何进行操作了。
　　一、增加快捷键
　　1．在commands中选择命令分类目录， 例如选定menu commands，在下面的命令列表中就显示出了menu commands中包含的各种菜单命令。
　　2．在命令列表中单击File(文件)菜单左边的“+”号，展开File菜单选择new命令。 这时我们可以看到它的快捷键是“Ctrl+n”，下面我们就为它增加一个新的快捷键。注意，你可以为每个命令设置两个不同的快捷键。
　　3．单击shortcuts中的加号按钮，窗口中出现一个黑色显示条，同时光标移到press key框中。
　　4．用键盘输入你想定义的快捷键。
　　例如：“Ctrl+Alt+n”，立即就可以在press key 框中显示出来。如果你输入的快捷键已存在或有错误的话，系统会提示你重新输入。
　　5．单击Change 按钮， 新的快捷键就赋予了new命令，你可以在命令列表中看到new命令右边出现了“Ctrl+n”和“Ctrl+Alt+n”两个快捷键(图2)。
　　二、删除快捷键
　　假如我想把刚才给new命令设置的“Ctrl+Alt+n”快捷键给删除掉，可以进行以下操作： 1．在commands中选择menu comnands。 2．在菜单命令列表中单击file旁边的“+”号展开，选中new命令。 3．在shortcuts框中选中“Ctrl+Alt+n”然后单击减号按钮即可。
　　三、编辑快捷键
　　如果想给某个命令指定一个快捷键， 例如把new命令的快捷键由系统指定的“Ctrl+n”变成“Ctrl+Alt+n”，可以通过下面的操作来实现。 第1、2步操作同上。
　　3．把光标移到press key框中， 然后用键盘输入“Ctrl+Alt+n”。
　　4．单击change按钮，使设置生效。 利用上面介绍的方法，你就可以随心所欲地对Dreamweaver的快捷键进行改造， 让它完全按照你的意志运行，这种快乐恐怕是你操作其他软件体会不到的。
　　制作Flash文字和按钮 Flash动画是优美的， 然而它的制作却不容易，对于初学者来说，要想为了一个简单的按钮而让他下一番功夫去学Flash也是不可想像的。Dreamweaver4替用户着想，提供了Flash text和Flash button功能，利用它，你无须了解Flash 就可以轻松制作出漂亮的Flash文字和按钮来，令人遗憾的是，目前它不支持中文。
　　一、制作Flash文字
　　1．新建一个文档，在插入Flash文字之前首先把该页面保存一下，否则的话，在进行操作时，系统会提示你先保存文档。
　　2．单击insert →interactive images→flash text命令或objects面板中的flash text 按钮，出现插入Flash文字窗口(图3)。
　　3．在text框中输入文字， 输入前可以设置文字的字体font、大小size)、颜色color )以及字体样式(粗体B、斜体I)和排列方式(居左、居中、居右)，rollover color是用来设置字体的翻转颜色。
　　4．设置其他参数 link：超链接对象，单击browse按钮可以与其他页面建立链接，也可以直接在link框中输入网址(带http//)建立与其他网站的链接，要想建立电子邮件链接，可输入“mailto+电子邮件地址”。 target：设置链接页面的打开方式。单击下拉菜单进行选择，分blank(新窗口)、parent(父窗口)、self(自身窗口)、top(上面窗口)四种。 Bg color：设立文字的背景颜色。 save as：flash文字动画保存的文件名，你可以自己命名。
　　5．设置完毕，输入文字，然后单击ok即可。 想看一下Flash文字的效果吗，在编辑窗口中选定Flash文字，然后单击属性面板中绿色的Play按钮(图4)，把鼠标移到刚刚制作的Flash文字上，怎么样，翻转效果出来了吧，单击Stop按钮可停止播放，如果不满意的话，可以单击edit按钮重新进行修改。
　　二、制作Flash按钮
　　1．单击“insert →interactive images→flash button”命令或objects面板上的flash button按钮，出现插入flash按钮窗口(图5)。
　　2．设置按钮的各种参数： sample：显示按钮的预览样式。 style按钮的类型，有44种可供选择，单击get more styles按钮，可以到macromedia网站上下载更多的样式。 button text：按钮上的文字，不支持中文。 font：字体类型。 size：字体大小。 link：超链接，可建立与其他网站的绝对超链接(输入网址时带“http//”)和电子邮件链接(输入时带“mailto”)，也可以单击browse 建立与其他页面的链接。 target：链接页面的打开方式，有四种，含义同建立flash文字相同。 bg color：按钮的背景颜色。 save as：flash按钮的保存文件，可自己另命名。
　　3．输入完毕，单击apply按钮，可使设置立即生效，单击ok按钮，返回编辑窗口。
　　4．预览效果。单击属性面板上的play 按钮即可查看按钮效果，单击stop中止预览，单击edit铵钮可进行修改。 河南 张金贵
