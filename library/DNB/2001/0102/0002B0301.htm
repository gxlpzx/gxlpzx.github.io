文件保密工具--锁鼠标

　　“锁鼠标”是一款使用VB编写的纯32位软件，具有完善的保密效果，在锁定鼠标的同时，也锁定了Windows 操作系统的快捷键，它可以在用户离开电脑时完全锁住电脑，从而保证了不会被非法用户使用。
　　直接用鼠标左键双击EXE文件便可进入安装界面，然后依照安装向导的提示操作即可完成。“锁鼠标”的几个主要功能键有：“隐藏窗口”(将程序主界面隐藏至系统托盘区)、“锁定鼠标”(电脑进入锁定状态)、“更改密码”(更改解锁密码，缺省状态为无密码)、“系统设置”(系统功能的设置)、另外还有“关于”和“退出”按钮。
　　我们点击“系统设置”按钮后，可进行如下设置：(如图) ● 启动时自动锁定：“锁鼠标”程序启动时便自动进入锁定状态。 ● 启动时最小化：启动程序时自动将主界面隐藏至系统托盘区。注意：点击下此按钮后，“启动时自动锁定”功能将失效。 ● 总在最前：使程序主界面不被其他程序或窗口覆盖。 ● 按时间锁定：定时锁定电脑，锁定的时间在下面的下拉式菜单中进行选择设置。 ● 非法重启后自动锁定：非常重要的选项！若在程序锁定状态下使用复位键重新启动计算机，则下次系统启动后程序将自动锁定。 ● 在 Windows 启动时运行：随 Windows 操作系统同时启动。 ● 若连续三次密码输错，则……：点击该按钮后，如果在锁定状态下连续三次输入错误密码，程序将在指定的时间内不接受密码，或者干脆将计算机关闭。
　“锁鼠标”是一款由国人编写的免费软件，大小为216KB，程序小巧，功能全面，可运行于Windows 95/98/NT/2000平台。大家可以登录其主页http//www．LockMouse．shangdu．net下载得到其最新的1．3．0版本。 (天津 Emil Song)
