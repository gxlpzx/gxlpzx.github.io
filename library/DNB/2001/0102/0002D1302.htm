登陆软件大点兵

　　一、登录奇兵
　　最新版本：V3．0 软件大小：1106KB 授权方式：共享软件 应用平台： Windows 95/98/NT/2000/ME 直接下载网址： (1)http//download2．tiandown．com/pub/software/internet/int－prom/addurl3．exe (2)ftp//ftp．tiandown．com/pub/software/internet/int－prom/addurl3．exe 这是一个功能十分强大的国产软件，相对于其它洋软件来说，这个登录软件更适合于我们中国人使用！注册费也十分低廉，仅7．60 元。登录奇兵是一款可以在一小时内让你的网站同时登录到世界知名的1300个搜索引擎和Free For Au Links(FFA Links免费链接)的好东东，而且可以无限次登录，发送情况统计报告。它的使用方法很简单，只要依照提示在各个相关空格里填写你的基本资料和网站资料，填写好后点击“登录”按钮，即可大功告成。
　　二、AddWeb
　　最新版本：4．0．2．4 软件大小：3228KB 授权方式：共享软件 应用平台： Windows 95/98/NT/2000/ME 直接下载网址： (1)http//www．cnvnet．com/download/d/addweb．exe (2)ftp//alberta．cyberspacehq．com/addweb．exe 该软件可以自动将你的网站登录到上百个查询引擎中，主要特色包括：多站点简介、自动频繁引擎数据更新、允许一次提交多个网站、查询引擎检查、PageBuilder 产生更好的查询效果、HTML、TXT 和 E－Mail 报告、Proxy 支持等等。
　　三、SubmitWolf PRO
　　最新版本：4．02．003 软件大小：1640KB 授权方式：共享软件 应用平台： Windows 95/98/NT/ 直接下载网址： (1)http//www．newhua．com．cn/down/swolf402cn．exe (2)http//www．trellian．net/bin/swolf402cn．exe 一个比AddWeb更好的注册网页软件，而且有中文简体和繁体两种版本。能够把你网站的详细信息提交到300多个中文和1500多个英文的搜索引擎中。虽然SubmitWolf PRO比 AddWeb 更好，体积也更小，但由于推出时间短，SubmitWolf PRO还不是非常完善，比如功能不够多、稳定性还有待提高。
