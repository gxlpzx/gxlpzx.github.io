IRC之旅

　　IRC是英文单词“Internet Ralay Chat”的缩写，意思是互连网网际聊天，通过特殊的协议IRC协议，大家连到一台或者多台IRC服务器上进行聊天。它的特点是速度快几秒钟内就可以看到对方的“讲话”、功能多，所以通过IRC聊天是全世界网友的最佳选择。如果你厌倦了网页聊天(Web Chat)的蜗牛速度，那么就来感受一下IRC飞一般的速度吧…… IRC简介 IRC分为服务器端和客户端，服务器端开放一个或多个专门的端口供客户端接入，这个端口一般是6667、7000。也有用别的端口的，如1000、9000等等。服务器端的IRC系统是基于Unix类的操作系统，因此大部分的IRC系统也都运行在Unix类操作系统中。虽然在Windows NT下面也有服务器系统提供，然而由于Windows NT本身的原因，使得它不是十分的稳定。
　　一个完整的IRC服务端系统分为两部分：IRCD与SERVICES，IRCD是支撑IRC的基本系统，当IRCD运行起来后就可以提供基本的聊天功能，而SERVICES是IRCD的补充部分，提供注册名字、注册频道、留言服务等功能，完整的IRC服务器缺一不可。服务器软件大体分为两种，一种是JAVA的，也就是Web Chat，有些IRC站点给你提供WWW网址，这样你就可以通过浏览器聊天了，但是这不是你的最佳选择，因为这种办法速度慢启动速度慢、功能少，不过不用下载专门的客户端软件。还有一种当然是要下载客户端软件了，如果你想更多地享受聊天的乐趣，你最好下载聊天的专门软件。现在最流行的IRC聊天软件有三种
●MSCHAT，这是由微软公司发布的，它的特点是在聊天面板上每个人都变成了卡通人物，生动一些，可是MSCHAT不支持很多IRC命令。
●PIRCH，它拥有一流的图像界面，是比较流行的IRC聊天软件。
●mIRC，它有强大的命令功能，可以说是世界上最流行的IRC聊天软件，所以最好选择mIRC为你的聊天工具，你可以下载最新版MIRC5．8 在Windows下面，mIRC是比较流行的(当然也是比较好的)IRC客户端client软件。mIRC有着友好的操作界面，强大的私下交谈Query、密谈Dccchat、传送文件Dcc Send Files功能，你可以自己配置mIRC的设置，使得它符合你自己的喜好。而mIRC最有趣的地方也许就是可以编写脚本Scipt和机器人Bot。有人用脚本实现了收发E－Mail，播放MP3等各种各样的功能。特别是mIRC5 V0．71版本推出后，开始支持Msagent， 相信用不了多久，你也可以写出优秀而有趣的脚本或机器人……那么赶快下一个mIRC(E文不好的，呵呵……可赶紧到http//IRC．wz163．net下载中文版)，开始我们的IRC之旅吧。 mIRC的使用 mIRC的安装没有什么特别的地方，按照mIRC的安装向导安装，安装好了以后，会弹出一个窗口让你配置IRC服务器地址以及你的个人信息，选择“IRC Servers/Add”按钮弹出一个小窗口， 然后在“Descrption”栏中填入“温州IRC”，在“IRC Server”中填入“IRC．wz163．net”，默认端口6667，单击“Add”钮， 接下来配置你的个人信息，在“Full Name”后边填入你的全名，在“E－Mail Address”后边填入你的E－Mail地址，再在“Nick Name”后边填入你的昵称，我的昵称是孤帆一叶。注意在IRC上不允许昵称重名。在“Altnative”后边填入你的备用昵称，敲击“OK”钮。选择“File/Connect”开始连接。 恭喜你，你已经连到服务器上了。但怎么进入房间跟其他人聊天？这时你打入“/list”命令，就可以看到一个“Channel”的列表，你还可以加入参数“/list ＃channel －MIN ＃ －MAX ＃”，如果加了参数 －min和－max就会列出人数介于这两者之间的频道。也可以使用通配符 “”。
　　使用IRC，一般在“Status”窗口里输入命令，所有的命令都是以“/”开始的，下面是一些命令举例： /list 列出所有频道 /list ＃mIRC，列出频道名为 mIRC 的频道 /list －min 5 －max 50， 列出人数介于5到50之间的频道 /list boy， 列出频道名中含有boy的频道 /join ＃channel ，进入该频道。也可以双击某一频道进入 接下来当然就是注册你的昵称。使用/msg nickserv register 命令输入密码，密码是为你的昵称所选择的密码，电子邮件必须填写，但是系统不校验(可以随便填一个)。 你想更改你的密码，可以用“/msg nickserv set password”命令来输入新密码。 你不满意你的昵称或者不想在IRC聊天，你还可以“自杀”， 命令是“/msg nickserv drop”。 如果你只是不满意你的昵称，你可千万别轻易“自杀”，可以用“/nick nickname”命令修改昵称。 现在我们环顾一下四周，看看有没有mm哦！输入“/who”，哇，这么多人。 再看看谁在“＃将爱情进行到底”聊天室里“/who ＃”将爱情进行到底。注意，隐藏模式的人看不见。 IRC中的主持人另外，聊天室内有些Nickname(昵称)的前边有个“＠”，那是什么？＠这个记号很重要，它代表这个人在这个“Channel”(频道)中有管理员OP的权限，有了这个权限，你就可以执行其他很多功能，比如说踢别人出去、注册频道。网友们叫它“帽子”，戴了帽子的人叫“主持人”。比如看看“彩云”这个注册用户的信息“/WHOIS 彩云”，跟彩云句悄悄话“/msg 彩云”，跟彩云开个小窗口慢慢聊“/Query彩云”等。 不过一般情况下Channel OP的权限只有别的OP给你，或者你自己创建一个Channel，除此之外，基本没有其他办法自己获得Channel OP的权限。 要自己创建一个Channel，首先注册一个频道，命令格式是“/msg chanserv register ＃频道名密码简短说明”，比如“/msg chanserv register ＃IRC之旅”123 欢迎大家来到IRC之旅。注册频道后，主题当然总不能一直不变，可以用“/topicC ＃channel newtopic”来修改，使用“/Invite 彩云 ＃IRC之旅”，可以邀请彩云来坐坐。使用“/mode ＃IRC之旅 +I” 可以将频道设置为私人聊天室，这样别人想要加入“＃IRC之旅”频道，必须要接收到“＃IRC之旅”内的人的邀请“/invite”方可进入。另外还可以用/mode ＃IRC之旅 +V NICKENAME，把不喜欢的人赶走。讨好讨好彩云，给彩云戴个帽子“＠——/mode ＃IRC之旅 +O 彩云”(去帽子就是“/mode ＃IRC之旅 －O 彩云”)。哎，带帽子的人可以使用的命令太多，有兴趣的人可以参考软件的帮助。 下面给出一些著名的IRC地址及端口，以供大家参考： 温州热线服务器IRC．wz163．net6667 服务器EaseChat 重庆FreeChat服务器61．128．193．476667 深圳浪子服务器202．96．137．647000 北京263首都在线服务器IRC．263．net6667 中国教育热线服务器IRC．wode．net6667 (叶培春)
