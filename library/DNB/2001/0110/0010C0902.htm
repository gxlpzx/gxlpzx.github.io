{电磁干扰对Modem的影响}

{ 本人于去年12月买回金浪56K内置Modem一个。型号为PCI 56 CIRRUS LOGIC VOICE。一路小跑回家准备把它安装在“老机”上。把Modem插在第一个PCI插槽里；开机后，WIN98自检到新硬件。放入驱动光盘，接下来一路确认完成了安装。可是上网的时候问题来了，只听见小猫叫了一声就没反应了。屏幕显示“ 错误：630 拨号网络无法完成该连接，调制解调器没有正常响应。 请检测是否调制解调器已经连接计算机与电话线路”。难道是接触不良？将电脑关闭后把Modem重新插好，然后开机。第一次上网没问题，5分钟后切断连接重新上网，又出现 “错误：630”的提示。听说有些主板的第一个PCI插槽的兼容性有问题。于是在每个PCI插槽上都试了一遍，均不能正常工作。每次都显示“错误：630”的提示。
重装了一遍驱动程序，还是没有解决问题。一次偶然把手放在Modem上(其实是不小心的，很危险 )。调制解调器竟然工作了，又试了很多次，只要是把手放在Modem上，小猫就会欢快地飞奔上网。难道是电磁干扰吗？我将一块剪成Modem大小的铁片用塑料袋包好粘在Modem的有元件的一面(最好能在铁板上连一根地线接到机箱上)。问题解决了，至今已用一月有余，没有发现问题。
分析：因为机箱内的电磁波很强，干扰调制解调器。也许这就是内置猫不如外置猫稳定性好的一个主要原因之一吧！你是否也曾经遇到过类似的问题呢，不妨试试加以屏蔽的方法(有人用香烟中的锡箔做屏蔽的方法并不可取，学过电子技术的人都知道，只有导磁率高的金属才能做屏蔽材料，比如铁，镍等。锡箔虽然导电但不导磁，只能在一定程度上将静电屏蔽，对于磁场它无能为力)。}
