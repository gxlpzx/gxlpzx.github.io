{看型号识MD}

{ 常有朋友问：“索尼 MZ-E75可以录音吗？”，“夏普 ST77是录放机吗？”其实MD机的型号本身就很好地说明了这些问题，现在就让我们来全面认识MD机的型号。
索 尼
索尼 MD机的型号分别有MZ-R***、MZ-E***、MZ-G***，其中：
“R”系列代表可录放机型(如MZ-R900、R90/R91等)；
“E”系列则代表单放机型(如MZ-E900、E90/E75等)，
“G”系列较特别，它属于欧洲版本的型号，是带收音功能(其实是线控带AM/FM收音功能)的可录放型MD机(如MZ-G750等)。
至于数字方面的辨别，MZ-R900、MZ-R90/91、MZ-R55等是索尼主打的可录型MD机，而MZ-R700、MZ-R70则是普及机型，而MZ-R70PC的“PC”指搭配有PC-LINK套件，可通过USB口与电脑连接进行数码录音，而不需要通过声卡！
目前索尼最新的可录型MD机是MZ-R900、MZ-R700、MZ-G750，最新的单放型MD机是MZ-E900、MZ-E700、MZ-E600、MZ-E500，以上机型全都支持“MDLP”倍录功能！
夏 普
夏普 MD机的型号分别有MD-MT***、MT**、ST***、ST**，其中：
“MT”、“MS”、“SR”系列代表可录放机型(如MT77、MS722、SR70等)；
“ST”、“SS”系列则代表单放机型(如ST531、ST77、SS321等)；
至于数字方面的辨别，三位数的表示是日本制造的主打机(如MT831、ST531等)，而两位数的表示是马来西亚制造的(如MT77、ST77等)，夏普一样有与PC相连接的PC-LINK套件，不过不在MD机型号上表示，而需另购套件。
目前夏普最新的可录型MD机是MT77，最新的单放型MD机是ST77、ST70，以上机型全都支持“MDLP”倍录功能！
松 下
松下 MD机的型号分别有：SJ-MR***、SJ-MJ**，其中：
“MR”系列代表可录机型(如MR200、MR100等)；
“MJ”、“SW”系列代表单放机型(如MJ88、MJ77、SW90MD等)；
松下的MD机无论什么型号都是日本制造的。
目前松下最新的可录型MD机是SJ-MR200、最新的单放型MD机是SJ-MJ88、SJ-MJ80、SJ-MJ33、SJ-MW。
不过目前松下没有一款MD机支持“MDLP”倍录功能，可能是还没有得到索尼的授权。
爱 华
爱华 MD机的机型分别有：AM-F、AM-HX，其中：
“F”系列代表可录机型(如AM-F80、AM-F70等)；
“HX”、“CL”、“ST”系列代表单放机型(如AM-HX70、AM-CL33等)；
爱华的MD机无论什么型号都是日本制造的。
目前爱华最新的可录型MD机是AM-F80、最新的单放型MD机是HX77、CL33等，不过，2000年爱华并无新型号的可录型MD机推出！
VICTOR-JVC
VICTOR MD机的机型分别有：XM-R***、XM、P***，其中：
“R”系列代表可录机型(如XM-R70等)；
“P”系列代表单放机型(如XM-P2000、XM-PX70等)。
目前VICTOR最新的可录机型有：XM-R700、最新的单放机型有XM-P70、XM-P50、XM-P2000，以上几款都是支持“MDLP”倍录功能的！
健 伍
健伍 MD机的机型分别有：DMC-***R、DMC***，其中
结尾有“R”的代表可录机型(如DMC-M7R、DMC-L7R等)；
结尾没有“R”的代表单放机型(如DMC-M55、DMC-L5等)；
健伍的MD机其实就是夏普的OEM机型，如DMC-M7R相当于夏普 MT77、DMC-L7R相当于夏普 MT831。
目前健伍最新的可录型MD机有：DMC-M7R、最新的单放机型有DMC-M55，以上两款都支持“MDLP”倍录功能！
先 锋
先锋 MD机的机型分别有：PMD-R**、PMD-MK*、PMD-P*，其中：
“R”系列代表可录机型(如PMD-R55、PMD-R3等)
“MK”、“P”系列代表单放机型(如PMD-MK1/2、PMD-P5等)
先锋的MD机也是夏普的OEM版本，不过先锋经常在产品外壳作文章，如PMD-MK1/2米老鼠的纪念版MD机就是夏普 ST531的OEM版！
其它牌子
可录型：三洋 MDG-R3、DENON DMP-R70、DMP-50、DMP-R30等。
单放型：三洋 MDG-P3、MDG-P1、ONKYO MD-P30、MD-P20、MD-10等。
其它牌子的MD机并不是主流MD机，而且大多数是索尼、夏普、松下三大MD制作商的OEM产品，这里就不一一说明，希望以上型号的说明对各位朋友有所帮助。}
