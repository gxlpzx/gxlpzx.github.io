{注册表也要“瘦身”}

{ 很多人听到注册表，就会望“表”生畏，因为注册表一旦遭到破坏，将会造成系统崩溃。但是，注册表在使用过程中会越来越庞大，影响了系统启动和运行速度，只要你定期对注册表清除一些垃圾和不需要的内容，便可以实现注册表的精简。
一、精简不需要的主键或者键值
Windows Win95/98/Me/2000为了适应不同国家和地区的需要，可以设置多种“键盘布局”，针对不同地区进行“区域设置”，我们只要保留其中的一小部分，其余的都可以删除。
1.精简键盘布局
单击“开始→运行”，键入“regedit”，便进入了注册表编辑器。找到“HKEY_LOCAl_MACHINE\System\CurrentControlSet\Control\KeyboardLayouts”分支，单击“Keyboard Layouts”(注意：不是“Keyboard Layout”)下的子键，展开它们，同时在右窗口中可以看到它属于何种键盘布局，必须将“美国101”，“全拼输入法”，“五笔输入法”，“智能双拼法”等常用的输入方法保留，其它都可以删除，右击鼠标在快捷菜单中选择“删除”便可以了。
2.精简区域
打开“HKEY_LOCAL_ MACHINE\System\CurrentControlSet\Control\Nis\Locale”分支单击“Locale”子键，保留右窗口中的“中文”、“英语(美国)”、“英语(英国)”等自己需要的语种，其它的都可以删除了。
3.删除时区设置
打开“HKEY_LOCAL _MACHINE\Software\Microsoft\Windows\CurrentVersion\Timezons”分支，保留“北京”时间，其余的都可以删除。
4.删除失效的关联文件
打开“HKEY_CLASSES_ROOT\ Cgjfile\Shell”分支下面的“Command”子键，如果该键下无内容，便可以删除。
二、清除软件的垃圾
某些程序不带有“Uninstall”程序，删除后会在注册表中留下主键和键值，清除些类程序的主键的方法如下：
1.启动注册表编辑器，单击“编辑”菜单下的“查找”命令，出现查找对话框，将“主键”和“键值”项选中。
2.在“查找目标”框中输入可执行程序名，单击“查找下一个”按钮，则系统将在注册表中开始查找，在找到后选中主键，将其删除。
3.或者打开“HKEY_LOCAL_ MACHINE\Software\Microsoft\Windows\CurrentVersion\Uninstall”分支，删除一些无用的程序。
这样便成功地精简了你的注册表，赶快试试吧。}
