{老机加速四重奏}

{ 现在，许多朋友购机时都选择CeleronⅡ633或Duron750(甚至更高级别的PⅢ或雷鸟处理器)和其他相应的配置，速度自然不在话下，但对于仍在使用PⅡ，甚至586级别机器的用户来说，可就不那么乐观了，每次望眼欲穿地看着Windows慢慢地推开窗户，那种感觉……。今天，笔者在这里传授大家“几招”，让你不必花钱实现电脑“升级梦”，使老机也潇洒走一回。
一、优化CMOS设置
1.设置系统速度为高速：将“System Boot up Speed”设为High。
2.提高启动速度：将“Boot Sequcnce”设为C；A：，“Boot up Floopy Seek”设为Disable。
3.提高显示速度：将“System BIOS Cacheable”设为Enable；“Video BIOS Cacheable”设为Enable。
4.进行快速自检：将“Quick Power On Selftest”设为Enable。
5.不搜索、不测试软盘驱动器：将“Floopy Drive Seek at Boot”设为Disable。
6.不检测1MB以上内存：将“Above 1MB Memory Test”设为Disable。
二、优化Windows文件
1.删除系统目录下Font文件夹中不常用的英文字体(注意要保留“Times New Roman”、“Marlett”等字体，否则系统的基本字体会发生紊乱)，节约GDI资源。
2.对于非常熟悉Windows系统的用户建议删除Help文件夹，以及系统中的Welcome教程等。
3.用CleanSweep或SafeClean等软件进行注册表清理，去除多余的“脂肪”(注意做好注册表的备份，以防不测)。
4.用CleanSweep等清理软件，将屏幕保护程序、桌面图案和墙纸进行打包删除，不常使用的文本文件、Help文件、DOS文件也可选择性地打包删除。如果以后需要，可以从CleanSweep4.0程序中的Backup文件夹中选择恢复。
5.取消自动运行(Autorun)功能。运行注册表编辑器“Regdit.exe”，依次打开HKEY-CURRENT-USER\Software\Microsoft\
Windows\CurrentVersion\Policies\Explorer主键，找到右侧一个名为NoDriveTypeAutoRun的二进制数值，双击该数值后输入键值“b5 00 00 00”，并保存退出。
6.实现快速关机。有些计算机的Windows95/98关机特别慢，在选择“关闭计算机”后会等上好几分钟才关机。因此，有这种情况的用户不妨运行“Regdit.exe”，依次打开HKEY-LOCAL-MACHINE\System\CourrentcontrolSet\Control\Shutdown主键，然后在右侧区域单击鼠标右键选“新建→字符串值”，新建一个名为FastReboot的字符串，输入键值1，保存后就能实现快速关机了。
7.编辑系统文件system.ini中386enh段，加入“LocalLoadHigh=1”命令行，目的是在MS-DOS中获得最大的可用内存。
三、优化系统属性
如果你使用的是Windows 98系统，则可进入控制面板，双击“系统”图标，点击“性能”标签中的“文件系统”按钮，在“硬盘”标签中，打开“此计算机的主要用途”下拉框，选择“网络服务器”，强迫Windows把它看成一台网络服务器，加快Windows对硬盘的读写速度。同时将“预读式优化”值设为“全部”。然后点击“CD-ROM”标签，将“追加的高速缓存大小”滑块调到最大值位置，最后将“最佳访问方式”改为“四速或更高速”，以加快CD-ROM的访问速度。
四、优化硬盘设置
为了使硬盘获得较高的性能和效率，我们可在Windows 98中设定使用UItra DMA功能。操作过程如下：进入“控制面板”→“系统”→“设备管理器”→“磁盘驱动器”→“Generic IDE Disk Type46”→设置”(太老的电脑可能不支持DMA功能)，最后勾选“DMA”选项，按“确定”按钮退出并重新启动机器。
好了，现在试试你的电脑，感觉速度是不是比以前快多了}
