{春光无限的英特尔——2001年春季IDF一瞥}

{ 全球IT技术盛会——2001年春季IDF(Intel Developer Forum，英特尔开发者论坛)于美国时间2月27日～3月1日在美国加州的San Jose市拉开了帷幕。英特尔自1997年开始举办IDF大会，一年两次，与会者都是全球顶尖的技术高手，大家在为期三天的会议上就Intel公司公布的一些提案及业界最新的技术和产品发展趋势进行交流，所以，IDF可以看作是未来IT新技术、新产品的一次预览。本次IDF为第8届，参加人数超过了5000。大会期间，除东道主Intel做了6个主题演讲外，还召开了24个技术专题以及250个与各种技术相关的小型会议。让我们一起来领略此次IDF大会所带来的IT业界的春的气息！
新一代服务器I/O“InfiniBand”结构确立
下一代服务器I/O“InfiniBand”结构是一种在因特网数据中心内连接服务器和远程网络及存储设备的中央机构，目前Intel公司已经开始提供应用此项技术的工业样品，并计划在今年内将这些LSI产品投放市场。另外，预计支持InfiniBand的系统也将在今年内上市。作为面向下一代服务器的输入输出接口，InfiniBand应用于Gigabit以太网以及磁盘设备和服务器构成的串行(Serial)通信网络中，将能解决PCI总线速度太慢的问题。此项技术是Intel的“Next Generation I/O(NGIO)”和由美国IBM、美国康柏电脑公司以及美国惠普(HP)等主导的“Future I/O(FIO)”合并在一起的I/O架构，它在1999年8月份合并初期曾被称为“System I/O”。从技术方面分析，InfiniBand基于Switched Fabric架构，具有Link1组、4组以及12组的标准规格(Link1组时为2.5Gbps，4组时为10Gbps，12组时为30Gbps)。通常情况下，使用InfiniBand的系统是由多个子网(Subnet)构成的，子网之间通过路由器以及网桥连接，每个子网最多可以由64000个节点构成。
P4的新“衣裳”
在此次IDF大会上，Intel发布了新一代Pentium 4所采用的小型PGA封装结构，这种新型封装技术将用于预定在今年年底上市的采用0.13μm工艺制造的P4处理器(开发代号为Northwood)。该封装的尺寸约(^10030201a^)为目前P4封装的一半，但引脚数量由现行的423个增加了478个。之所以采用这种小型封装形式，是因为新一代P4的时钟频率将高达2GHz，而现在所采用的封装的引脚之间的间隔无法满足芯片对电气性能的要求。
McKinley正式亮相
Intel公司高级副总裁在此次IDF论坛上首次披露了配备新一代64位微处理器McKinley的样机。McKinley是Itanium处理器(研发代号为Merced)的后续产品，其专用芯片组为“870”，支持双CPU结构，内存采用DDR SDRAM。此外，McKinley的设计和Itanium有所不同，后者是将二级缓存单独设计在另外的芯片上，而前者却将一级缓存、二级缓存以及新增加的大容量三级缓存都封装到同一块芯片上，这样就增加了(^10030201b^)运算电路的数量，并通过提高总线时钟频率来加大数据总线带宽，如此一来，McKinley的数据带宽就达到了Itanium处理器的4倍。据悉，Intel已开始向OEM厂商提供McKinley样品，并计划在今年年底前推出测试版，2002年开始批量生产。
700MHz笔记本电脑用PⅢ面世
在本届IDF大会上，Intel公司正式发布了主频为700MHz的笔记本电脑专用PⅢ处理器。这是一款可在1.1V电压下工作的产品。此前，Intel公司已经发布了全尺寸笔记本电脑用的最高工作频率为850MHz的“mobile Pentium Ⅲ”和小型笔记本电脑专用的工作频率为500MHz的“Ultra Low Voltage mobile Pentium Ⅲ”、“Ultra Low Voltage mobile Celeron”等。这次发布的700MHz的“Low Voltage mobile Pentium Ⅲ”采用0.18μm工艺制造，具有英特尔独特的SpeedStep省电功能，可分别在交流电源和电池驱动时自动切换工作电压和工作频率。此外，该产品的FSB(前端总线)频率为100MHz，同时在处理器芯片上集成有256KB的二级缓存，可支持处理三维图片和动态图像/声音等矢量数据的扩展命令组“Streaming SIMD Extension(SSE)，并采用BGA(Ball Grid Array)封装形式，相信不久“Low Voltage mobile Pentium Ⅲ”就会成为新一代笔记本电脑的主流。
Web Tablet让网络更人性化
除了CPU外，Intel公司还公布了正在开发的家用无线因特网浏览手持终端“Web Tablet”的技术概要。Web Tablet是Intel在今年1月份举行的民用设备展览会“Consumer Electronics Show(CES)”上展出过样机的网络终端。利用此项技术，用户可以轻松共享家用个人电脑的因特网接驳处理能力以及打印机等资源，(^10030201c^)在家庭内任何地方都可以进行浏览WWW资源。Web Tablet配置有StrongARM SA-1110处理器，存储器采用的是StrataFlash闪存。Intel发布的Web Tablet技术概要如下：采用Wind River的因特网应用程序开发环境(包括WindStorm、VxWorks和Personal Jworks)；采用Espial的嵌入式浏览器“Escape”，为用户提供WWW浏览环境。另外通过Espial公司GUI组件的工具包“Espresso”简化Web Tablet应用程序的开发；Web Tablet音频环境采用BeComm的软件“Strings”。Strings是一种利用时钟对音频信息的路由、转送和发信方法进行管理的软件。Intel目前已经将Web Tablet定位为该公司所提倡的新一代个人电脑概念“Extended PC”的主打产品。
希捷、Intel和APT共推Serial ATA硬盘
本次IDF年会上，希捷公司(Seagate Technology)、Intel以及APT Technology等三家公司共同推出了符合硬盘新接口规格“Serial ATA修订版1.0(Revision 1.0)”的硬盘以及PCI主适配器(Host Adapter)。三家公司将通过支持Serial ATA的电缆线和接插件，将嵌入了APT的对应Serial ATA的桥接电路(Bridge)和物理层芯片的希捷的Serial ATA硬盘连接到Pentium 4电脑上去。Serial ATA是一种将硬盘、DVD驱动器以及CD-RW驱动器等存储设备连接到个人电脑上的串行接口规格，该规格目前分析将有望取代现有的并行接口，传输速度为1.5Gbit/s，而且可以进一步扩展到3Gbit/s以及6Gbit/s等。该接口与现有驱动软件确保互换性，与并行接口相比，具有布线容易、接口小以及工作电压低等特点，同时，使用该接口可以使电脑系统的设计变得更为容易。
IPMI标准新版本正式出台
美国时间3月1日，也就是此次春季IDF大会的最后一天，Intel、惠普、NEC以及戴尔电脑等公司共同发布了服务器管理接口标准IPMI(Intelligent Platform Management Interface)的最新版本——1.5版。IPMI是监视/管理温度、电压、冷却风扇、电源以及底盘(Chassis)等服务器物理状态的接口标准。新版IPMI 1.5可以很容易通过串行接口、Modem以及LAN等远程环境管理机构形成装配式(Rack Mount)因特网服务器系统。另外，新版本中还增加了在发生系统错误时自动发出警告的功能，它能帮助IT管理人员迅速修复系统。IPMI 1.5还支持Desktop Management Task Force的Alert Standard Forum、Compact PCI、SM Bus 2.0以及PCI Management Bus等标准。}
