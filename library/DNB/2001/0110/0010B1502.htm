{创作MIDI音乐的强力工具-Midisoft Studio LE}

{ 这月上网费又超限了，老婆去缴电话费了，回来准挨批。且慢，我有绝招，那边门一响，马上用一软件打开一古筝曲，在乐曲悠扬中，老婆的脸上挂满了惊奇：“你哪来的这个谱子”(她乃一音乐教师)，我一番介绍，嘿嘿，上网费不再提也。好东东不敢独享，特献给大家：创作MIDI音乐的强力工具-Midisoft Studio LE。
打开Midisoft Studio LE后，够酷吧，更酷是它的功能，诸君请看：
1.作曲
作曲时首先设置乐谱，单击菜单的“窗口”的乐谱窗口设置，出现设置窗口，这里默认是每小节16格， 如果谱中有三连音，就要在右边的一排中选择。然后单击菜单“乐曲”，选择“谱号”，出现的窗口既可以把所有音轨设为同一谱号，也可以把某一音轨单设，同时也可以在菜单乐曲中对节拍、音调、速度进行设置，非常方便。
　　下面开始作曲，用鼠标单击乐谱窗口左边工具栏的第二个音符工具，出现单步参数输入窗口，一般采取默认设置就可以了，要用到哪个音符，就用鼠标点击所示的工具。移到五线谱上，这时要注意用到前面设的每小节16格了，使音符工具对准网格线，点击在你所要的位置上就可以了。不对准网格线可是要出错的。同时音箱中传出乐音，如果要删除音符，用工具栏上带叉号的音符在要删除的音符上单击就可以删去。就这样做下去好了，一直到结束，然后单击左上方的播放按钮，就可以听你自己的大作了。什么，好难听，忘了让你设置各音轨演奏的乐器了，单击混音窗口中的音轨1，会出现窗口，你就可以选择各种乐器了，其余的音轨乐器选择方法也是这样。
曲作完了，可是我还想卖弄一下文采，填词，可以，单击菜单“乐曲”。然后点击“歌词”，在出现的窗口中选择需要填词的音轨，然后就可以填词了，不过与音符对准要试验几次才行。
2.插入声音文件
这是Midisoft Studio LE的一大特色，它可以分别在音轨A和音轨B中同时插入两个不同的声音文件。单击左边工具栏的最后一个波形工具，鼠标会变成波形，在音轨上对准要插入的网格线单击，出现“插入声音文件”窗口，可以选择要插入的声音文件，确定后就可以了。
3.MIDI转换成乐谱
这又是Midisoft Studio LE的一大特色，只要有现成的MIDI音乐，用本软件把它打开后它后自动转化为乐谱，现在想要哪个歌曲的谱，只在网上把MIDI下载后，用它来转换就行了，并还可以自己用各种乐器对它进行打磨，甚至我们的民族音乐它也可以转换呢。^_^
还有其它很多功能，不再一一列举了，快去http://jwjjt.yeah.net去下载吧。}
