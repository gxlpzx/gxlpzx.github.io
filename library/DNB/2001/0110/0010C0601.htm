{PC质检员——HWINFO}

{ 对于鲜有选购经验的初级用户来说，购买电脑时除了需要请一位参谋去帮你辨别硬件的真伪外，准备一个能够全面检验硬件的软件也是必不可少的。为此，笔者向你推荐一个优秀的硬件检测工具——HWINFO(该文件只有560K大小，你完全可以将其拷至一张软盘上使用)。
一、HWINFO的主要功能
先用一张干净启动软盘(内有HWINFO软件)启动电脑，进入纯DOS模式(注意：最好不要加载任何驻留程序)，输入“HWINFO”后回车，进入其主画面，然后执行“Info”菜单下的“MainBoard Info”命令，如图所示。图中所示的信息有：“Main Processor”(CPU的型号)、“Bus Type”(系统支持的总线类型)、“Mainboard Model”主板名称 、“Mainboard Chipset”(主板采用的芯片组类型)、“First Level Cache”(一级缓存)、“Second Level Cache”(二级缓存)、“System Memory Size”、“Machine Type”(机器类型)、“Computer Name”(电脑名称，该项通常是品牌机才有)、“BIOS Manufacture”(BIOS的类型及日期)。
二、HWINFO使用详解
1.如何检测CPU
(1)如果你仅仅是根据图1来判别CPU类型的话，那么你很可能上JS的当——他完全可以让CPU在高于其额定主频下工作(超频)，所以上面显示的CPU频率不可全信。要验证CPU是否被超频使用，还需结合来判断。以笔者的赛扬600为例，大家知道，该CPU外频为66MHz，倍频为9，这与图中显示的完全相同，可见笔者的CPU是真的。现在不少检测软件只能测出CPU的主频，而且往往是超频后的频率，并非其本来面目。HWINFO就不同了，它能测出CPU的外频和倍频，你只要根据CPU的相关知识，就很容易判断CPU的真假。此外，图2中“CPU Vendor Name”指的是CPU的生产厂商，而“Current Clock Ratio”指倍频，“Current FSB R Frequency”指外频(即系统前端总线频率或FSB)，如果这些数字与实际情况不符，那么你就要小心这个CPU是否被Remark过了！
(2)还有一个指标可以衡量CPU的性能：运行“Benchmark”菜单下的“CPU Benchmark”中“16-Bit Integer”(16位整数运算)命令，可见到所示的画面，其中用黄色指示条表示你的CPU与哪些CPU处在同一级水平上(笔者的CPU看起来得分比K7-600稍高一点)，此外，“CPU Benchmark”下的“Floating Point Index”浮点运算 、“32-Bit Integer Index”(32位整数运算)和“MMX Index”等项目也可测量CPU的相应得分，你可以比较一下，看你的CPU属于何种档次。
2.、内存的相关信息
在图1中将光标移到“Total”处并回车确认，即可出现所示的画面，这里显示了内存的相关信息，比如每根内存条插在哪个插槽上(Row)，每根的容量(Size)、类型(Type)，是否带有ECC纠错功能、内存条的速度Speed 等。注意：为保证所测出的内存条速度真实可信，请在BIOS中设置为让主板自动识别内存条的速度(前提是主板必须支持这项功能)。另外，将光标移到“Module”项目上，再回车确认，出现所示画面，这里显示了更详细的内存信息。如 “Memory Speed(内存速度)”、“Module manufacturer”(生产厂商)、“Serial Number”(序列号)、“SPD Version”(SPD版本号，这个数字也是区分内存条品质的一个因素)等等，在图5中，要仔细看“Supported CAS Latency”后面的数字，这个数字往往是区分PC100和PC133内存的标准之一，如果这项指标不符，那你可要小心了！
3.显卡信息
在“Info”菜单下执行“Video Info”命令可以查看显卡的相关信息。这里你可要看清楚图中“Video Chipset”(显卡芯片类型)、“Video Memory Size”(显存大小)、“Video Card Bus”显卡的接口类型 等各种关于显卡的信息，以防JS骗你没商量。
4.测试硬盘和光驱
(1)在“Info”菜单下执行“Drive Info”命令出现所示画面，图中显示了系统内硬盘和光驱的连接情况(比如是用同一根硬盘线连接还是单独用一根数据线连接)，为了提高光驱的读写性能，应该将它们分别连接在不同的数据线上，图中所示表明笔者的系统中就是采用的这种连接方法(硬盘在“Primary”的“Master”上，光驱在“Secondary”的 “Master”上)。
(2)在图6中，将光标移 “Primary”后的“Master”处按回车键，出现画面。这里会详细显示出硬盘的相关信息：“Fixed Disk Capacity”(硬盘容量)，“Cache Buffer Size”、“Disk Model”(硬盘型号，包括硬盘的生产厂商及硬盘型号)，特别是“Serial Number”中提供的信息可以帮助你日后与生产厂家联系看看是否为假货(目前很多产品都采用了电话防伪方式，当你拨通厂家的电话后输入产品序列号，即可查验是否为假货)，其它如“Max Transfer Modes”(硬盘最大传输模式，一般新硬盘至少应为“PIO 4”)、“DMA Transfer Modes”(硬盘支持的DMA模式)等信息也是你衡量硬盘品质的重要指标。红色的部分表示硬件本身不具备的性能。
(3)在主界面中运行“Benchmarks”菜单下的“Disk Benchmark”命令，可以检查硬盘读取数据的速度Read Transfer Rate ，其中“Buffered”表示有缓冲时的读取速度，Random表示不带缓冲时读取速度(这两个数字越大越好)，而“Average Seek”表示平均寻道时间，一般新硬盘都在9ms以下，这个数字越小越好。
(4)仍然在图6中将光标移至“Secondary”后的“Master”处按回车键，将出现光驱的详细信息。如果你要检查光驱是否真的达到了商家所说的速度，可以看看“Maxium Speed”一项中的数字。其它关于光驱的指标还有：“Drive Model”(光驱型号)、“Current Selected Speed”(当前速度，数字越大越好)、“Buffer Size”(缓存大小)、“Drive Supports”(光驱支持的光盘类型)等。
5.主要PCI设备信息
你还可通过HWINFO来检测其它外设，如各主板插槽上都安装了哪些设备。具体方法为：在图1 的“Bus Type”项目上回车，再在“Bus Info”的“PCI”上回车，可见到画面。这里主要显示PCI设备的相关信息，一般只须查看“DEVICE”一栏。比如，HWINFO检测到笔者系统的硬件信息为：主板芯片组为VIA VT82C686A 和VT82C596B，带有AGP控制器(即支持AGP显卡)，另外还有两个以前其它软件在Win9X下需要驱动程序才能检测到的设备：一个是MODEM，图中的信息正确显示了我的MODEM的芯片型号，另一个是声卡，我的声卡是集成在主板上的创新128“硬”声卡。
6.其它相关信息
HWINFO还可以检测系统硬件的其他信息。比如，在主界面“Test”菜单下执行“SensorStatus”命令，屏幕上将显示出系统内的温度、电压和风扇转速等情况，这里显示的温度主要是CPU和主板的当前温度值，而“Vcore Voltage”则是CPU当前的工作电压，“Fan1 speed”指的是CPU风扇的转速，而如果主板上另外安装了散热风扇，则会在“Fan2 Speed”一项中显示出来。
看了以上介绍，你还犹豫什么？赶快到http://www.hwinfo.com去下载其最新版本吧。有了它，你装机时就不必再担心会受到JS的欺骗了。}
