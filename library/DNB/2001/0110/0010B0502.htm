{巧用GHOST从光盘启动}

{ 能自启动的光盘是越来越多了，但有一些操作系统的安装光盘并没有引导功能，而你又不想在硬盘上安装DOS，有没有直接启动光盘的办法呢？
有一个简单巧妙的办法是：利用NORTON GHOST 2001 (版本号：6.5)制作一张启动盘即可，步骤如下
1.安装NORTON GHOST 2001；
2.启动NORTON GHOST 2001，点击“NORTON GHOST BOOT WIZARD(GHOST启动盘制作向导)”；
3.接着会出现一个对话框，有三个选项：
①：STANDARD BOOT DISK WITH LPT AND USB SUPPORT(标准启动盘，支持LPT和USB)；
②：PEER TO PEER NETWORK BOOT DISK(点对点网络启动盘)；
③：CD-ROM BOOT DISK ——GHOST BOOT DISK WITH GENERIC CD-ROM DRIVERS AND NO NETWORK SUPPORT(支持普通CD-ROM但不支持网络的启动盘)。
4.选择第三项：CD-ROM BOOT DISK，一路点击“下一步”或“确定”，不要更改任何选项，直到完成。
好了，一张具有CD-ROM支持功能的GHOST启动盘制作好了，你还要完成以下的几步，才能实现我们最初的目的：启动一台安装有Windows 95/98的干净机器，将我们刚才制作的软盘放进软驱，删除它里面的“GHOSTPE.EXE”文件(这是GHOST的核心文件)。将常用的COMMAND命令(如：FDISK、FORMAT等等)拷贝到这张软盘里，非常重要的一步：修改软盘里的AUTOEXEC.BAT文件，其方法如下：
@echo off
MOUSE.COM
LH \MSCDEX.EXE /Dcd1 /Dcd2 /Dcd3
CD GHOST
echo Loading...
GHOSTPE.EXE
将“CD GHOST”和“GHOSTPE.EXE”删除，在末尾加上“COMMAND.COM”修改后的AUTOEXEC.BAT文件如下：
@echo off
MOUSE.COM
LH \MSCDEX.EXE /Dcd1 /Dcd2 /Dcd3
echo Loading...
COMMAND.COM
(别忘了保存修改后的AUTOEXEC.BAT文件)
好了，将这张软盘放进需要从光盘安装操作系统的机器里，重启(别忘了在BIOS中修改引导顺序)，机器最终找到适合的光驱驱动程序，可以浏览光盘上的内容了吧。在空闲时，仔细研究一下这张软盘的内容，你会收获良多！(重庆 李辉)}
