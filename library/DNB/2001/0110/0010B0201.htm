{给注册表做做清洁——RegCleaner}

{ 清理注册表的工作向来都是一个令人头疼的事情，一不小心就可能对系统造成破坏，严重的还可能导致系统崩溃。不过好在有许多专门的注册表清理软件可以帮助我们，使我们完全摆脱上述烦恼，并且可以使你的 Windows 注册表文件大大地减肥，进而让系统执行得更顺畅。这里笔者向大家推荐的是RegCleaner( http://www.vtoy.fi/jv16/RegCleaner.exe，476KB)。在各大软件下载网站的注册表清理软件类下载排名中，RegCleaner都是名列前茅，由此可见RegCleaner目前已成为最受欢迎的注册表清理软件之一。它以其强大的功能、简便的操作逐渐成为了广大电脑爱好者的必备的系统维护工具。RegCleaner版本更新速度也相当快，目前又推出了最新的版本4.0。本文将以此版本为例，介绍一下RegCleaner的使用功能。
一、安装
首先运行RegCleaner安装文件，出现安装界面，由于RegCleaner具有对包括简体中文、繁体中文、英文、西班牙等18个国家和地区的语言支持，所以在安装的时候一定要选择安装Language Packs(语言包)，其他设置按默认即可。整个安装过程十分简单，一路点击回车即可。
二、运行和语言设置
安装结束后，双击桌面图标，运行并进入RegCleaner程序界面，在这里我们只需进行简单的设置，就可以将它转化为中文版，从而方便大家的使用：选择菜单命令“Options→Language→Select Language”，在打开的对话框中选择文件“Chinese GB.rlg”，最后单击“打开”回到主页面。你会发现界面文字已变成了大家熟悉的简体中文。汉化后的RegCleaner界面。
三、基本功能
RegCleaner主界面上方有7个选项卡，分别对应了7大基本功能，下面就对它们逐一进行介绍。
1.软件
以清单形式自动列出所有在Windows注册表中登录软件的详细资料，包括：作者、软件、年代等。由于种种原因，有些软件虽然已经被卸载，但是在注册表中仍然留有残留的垃圾键值，这是我们不希望看到的。首先在已删除的软件名称前打钩，然后单击“移去选中的项目”，RegCleaner就会将该软件的全部残留信息记录从注册表中彻底清除。另外，选择任意一个软件名称(以Foxmail为例)，单击鼠标右键，选择“高级信息”，RegCleaner会显示Foxmail在注册表中的全部键值。
2.启动列表
显示计算机启动时载入的程序列表，如果机器的启动时间太慢的话，你大可在这里删除不必要的启动程序，以加快机器的启动速度。
3.卸载菜单
Windows的卸载菜单，你可以使用程序本身的卸载程序或从此栏中删除程序。相当于控制面板中的添加/删除程序中的可删除软件列表。以ACDSee为例 ，先选中ACDSee，然后点击“卸载”可将其删除。如果某软件已经删除，但是还在添加/删除程序中留有名称信息，可以先选中它，然后点击“移去选中的项目”，即可删除垃圾信息。
4.文件类型
显示在注册表中登记的文件类型，左侧显示的是文件扩展名，右侧显示的是该文件类型的描述。
5.新建文件
显示鼠标右键菜单中能够新建的文件类型，如果你觉得新建菜单里面的一些东西实在是没有必要的话，可以在这里删除相应选项。
6.资源管理扩展
相当于“资源管理器”中菜单“工具→文件夹选项→文件类型”，显示已注册的文件类型，左侧显示文件的扩展名，中间显示对此类型文件可进行的操作如：Edit、Open，右侧显示文件操作程序的具体安装路径。
7.备份
显示你删除注册表键值的备份信息。包括备份类型、描述、删除日期，时间等信息。在误操作的时候可以帮助你及时恢复备份。这也是RegCleaner所具有的特色功能之一。
其中有4个选项，作用分别是：
结合：可以将两个或两个以上的备份结合成一个备份。
重命名：可以将备份类型名称改为更容易记住的名称。
恢复备份：还原已删除的注册表键值。
移去选中的项目：删除备份记录。
四、菜单功能
1.自动清理注册表
如果你嫌手工清理太麻烦的话，可以选择菜单“工具→清理注册表→自动清理注册表”让RegCleaner自动清理注册表中的垃圾信息。
2.查看DLL文件及共享DLL文件
在Windows的System子目录下存有大量的被系统或应用程序共享的DLL文件，但是由于我们经常要安装和卸载软件，就会在System目录下留有一些DLL垃圾文件。选择菜单“工具→查看DLL文件→查看共享DLL文件”可以列出所有的DLL文件，你可以删除你所知道的没有文件共享的DLL文件。
3.修改搜索/打开菜单
选择菜单“工具→修改搜索/打开菜单”，用于显示桌面“开始→搜索/打开”菜单中的项目。其中，“修改打开菜单”的功能仅能用于Windows2000。
4.查找功能
选择菜单“查找”，弹出“查找”窗口，可以帮助你快速查询想要的键值。
RegCleaner可以说是一个功能非常全面且强大的软件。说到这里，你是不是已经对RegCleaner的基本功能和使用方法有了大致的了解了吧。感兴趣的朋友还不把它down下来一试为快，它是绝对不会让你失望的！}
