{邮件加密利器ZixMail}

{ 由于E-mail邮件的安全性不高，对于一些重要的邮件我们就需要找一些专门的加密工具进行加密。A-Lock就是其中比较优秀的一款加密工具(去年第41期的《电脑报》曾介绍过)，但A-Lock只能对邮件的正文部分进行加密，如果邮件带有附件，它就无能为力了。现在给大家介绍一款能够对整个邮件进行加密的邮件加密工具ZixMail。经过它加密过的邮件，接收方接收的是一封带有附档的邮件，这个附档就是被加密过的邮件(包括附件)。要想打开这个附档，接收方必须有开启这个附档的密钥(该软件称之为签名)。与一般的邮件加密工具相比，ZixMail具有以下的特点：ZixMail对邮件的加密是通过专门的网络服务器来完成的；ZixMail提供了邮件的撰写、加密及发送一条龙服务。
一、软件的下载与安装
ZixMail是一款免费软件，最新版本V1.2.813，文件大小3.9MB，下载地址http://www.zixmail.com/。软件安装完毕后，在桌面上会出现两个快捷图标，通过它们可分别执行申请加密服务和撰写、加密及发送邮件的任务。
二、软件的使用
1.申请加密服务
ZixMail的加密方式比较特殊，它只对从申请了加密服务的邮件地址发出的邮件进行加密，所以我们首先必须为邮件地址申请加密服务。
点击“ZixIt Worldwide Signature Server” 快捷图标，启动申请加密服务程序(图1)(^10040401a^)。点击“Create New” 按钮，弹出“Create Signature”(创建签名)对话框。首先，填入需要申请加密服务的邮件地址。接着，填写邮件地址的签名。然后，设置签名的恢复功能，有了该功能，即使签名遗失了，签名服务器也能够轻松地帮你找回丢失的签名。签名恢复设置方法如下：在“Recovery Question”项后的正文框中填写一个问题，在“Recovery Answer”项后的正文框中填写一个答案，设置完毕后点击“OK”按钮，系统会提示已经把你的申请发给了ZixIt Worldwide签名服务器。大约过1分钟(最长时间不会超过10分钟)，你会收到一封发自ZixIt Worldwide签名服务器的确认信。点击邮件正文部分的链接地址 http://wwss.zixmail.com/a﹖BAbsdAvEupMeI193dJIw/JVA ，签名服务功能即被激活。最后，回到“ZixIt Worldwide Signature Server”窗口，这时候你会发现申请了签名服务的邮件地址已经在列表框中了，点击“Update Status” 按钮，列表框的“Status”项下即刻由“New” 变成了“Active.” 。到此为止，所有的前期准备工作全部完成。如果你还需要为其他的邮件地址申请签名服务，则可重复以上的步骤。
2.使用ZixMail
①发送加密邮件
点击“ZixMail” 快捷图标，弹出ZixMail窗口(图2)(^10040401b^)，ZixMail的界面有些像Outlook Express的新邮件撰写窗口。我们可以先给自己试着发一封签名邮件，看看ZixMail的加密功能到底如何。在缺省状态下，ZixMail发送邮件使用的是自己内置的SMTP服务器，你也可以设置为距自己比较近的SMTP服务器。点击“Options” 下拉菜单中的“SMTP Settings” 项，弹出“SMTP Setup”对话框，选中“Specify an SMTP Server”项，在“Server Name” 后的正文框中输入SMTP服务器的地址(如smtp.21cn.com)，在“Server Port Number”项后填入25，然后点击“OK”按钮即可。
撰写邮件这里就不再多述了，稍有不同的是它有两个插入附件的按钮，“Attach File9s ”按钮用来插入文件，“Attach File list”按钮用来插入文件夹。邮件撰写完毕后，点击“Send”按钮，弹出邮件发送界面，这里有许多无关紧要的选项，一般使用缺省的设置就可以了，不过在邮件地址下的正文框中一定要填上你的签名，然后点击“OK”按钮，接下来的事情就不需要你操心了，ZixMail会替你把一切打点好。邮件加密、发送完毕之后，ZixMail会给出详细的发送信息，告诉你邮件是否成功发送。
②阅读邮件
收到ZixMail发来的邮件后，在邮件正文部分你会发现有一个名为“ZixMail.zix” 的附档，该附档就是经过加密的邮件。双击该附档，弹出“文件下载”对话框，选中“在文件的当前位置打开”项，点击“确定” 按钮，弹出“Open ZixMail”窗口，在邮件地址下方的正文框中填入该邮件地址的签名，然后点击“OK” 按钮，邮件即被打开。同时会弹出“Add To Address Book”对话框，如果你想把该邮件地址添加进ZixMail地址簿，请点击“Add” 按钮；如果你不想把该地址加入地址簿，请点击“Cancel”按钮。要是该邮件地址已经在地址簿中，便不会弹出“Add To Address Book”对话框。如果你不希望在打开邮件的时候弹出该对话框，请清空对话框中“Always prompt to add when reading ZixMail from someone not in my address book”项前的复选框。
③修改签名
为了预防别人窃取签名，我们可以经常对邮件地址的签名进行修改。点击菜单栏“Actions”下拉菜单中的“Edit Signture”项，弹出编辑签名向导：首先，未注册的用户只能够选择第一个选项“Change Signature Phrase Only” (仅修改签名)；如果是注册的用户(注册费5美元)，可以选择第二项，该项能够同时修改加密编码和签名。我们下面介绍的是按照第一个选项进行的。其次，更改签名：这里有三个正文框，在第一个正文框中输入原先的签名，在第二个和第三个正文框中输入新的签名，连线上网后，点击“OK”按钮，签名便修改好了。
④恢复签名
要是你不慎忘记了签名，而你在申请签名的时候已经设置了签名的恢复功能，那么ZixMail能够很轻松地为你恢复遗失的签名。点击菜单栏“Action”下拉菜单中的“Recover Signature”项，弹出“Recover ZixIt Signature”对话框，在第一个正文框中输入需要恢复签名邮件地址，在第二个正文框中输入你在设置签名恢复功能时填写的答案，然后按“OK”按钮(这时需要在线)，遗失的签名将会很快送到你的桌面上。
三、结语
ZixMail以其对邮件的全面加密和对邮件撰写、加密及发送的一条龙服务，赢得了不少用户的青睐，但它仍有令人不满意之处：首先，由于ZixMail对邮件的加密是通过网络服务器来进行的，因此需要耗费一定的网络时间，这就增加了用户的上网费用。其次、接收邮件方必须安装有ZixMail才能够打开被它加密的邮件，这样就使ZixMail的应用范围受到一定的限制。不过从另一个角度来看，这也许是它的一个优点，因为这在无形中提高了ZixMail加密邮件的安全性。}
