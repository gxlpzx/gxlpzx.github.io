{用Dreamweaver轻松做网络相册}

{ 复杂的网络相册的制作，在Dreamweaver里只要简单地使用一个命令即可完成，极大地方便了需要制作网络相册，把生活中精彩的瞬间通过网络给朋友们分享的人，使大量的图片的定位、建立链接、规范图片网面等等诸如此类的重复动作简单化，减小工作量。下面就开始我们的网络相册的制作。
首先在硬盘站点的根目录下建立两个子目录：Image、Photo前面的一个目录用来保存要建立相册的源图像文件，后面一个目录用来保存将要建立的相册。然后把所有要使用的图像拷贝到image的目录里。接着运行Dreamweaver，然后选择系统菜单“commdnds/create web photo Album”则可以打开网络相册面板。如果你没有安装Macromedia Fireworks这个软件的话，就会弹出一个对话框提示你使用这个功能需要先安装Macromedia Fireworks。在这个相册面板里有许多的项目，在Photo Album Title这个项目里给相册起一个名字，在Source Images Folder里填上源相片文件夹的路径，在Destingtiom Folder里则填上用于存放相册的路径，在Thumbnail Size里设置略缩图像的大小，在Columns 里填上每页显示相片的列数，其它的各项保持缺省即可。用鼠标单击“OK”按钮后，就会自动打开Fireworks.exe图像处理软件，对源目录下的图像进行批处理。当所有的图像处理完后，会显示一个对话框，提示用户相册已建立完毕，单击“确定”按钮，则会关闭相册设置面板并显示出最终产生的相册。
在生成相册中，相册及其所有微缩图像通过超链接打开的文件，都以HTML文件格式的形式，保存在相册面板选项设置的相册目标目录中，即在本地站点的photo子目录下，而且在这个目录里还建立了其它的三个子目录及一个相册的主页文件Index.htm。打开Index.htm文件，你就可以预览到你所做成的网络相册了。当用鼠标单击相册中的一个微缩图片时，则会显示出该图片的原始大小，同时在上面会显示出相册标题、该图像文件的文件名、及源图像。如果你在网络相册的面板里把Create Navigation Page for Each Photo这一项选中的话，则会为相册中的每一个相片设置相同的导航条，即将该相片显示在一个单独的网页上，同时相片上面有用于浏览相册中所有相片的浏览条。通过浏览条可以显示上一幅图像、下一幅图像或显示相册的主页。
做网络相册就是这么轻松简单。}
