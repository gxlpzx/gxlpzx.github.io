{一步一步装网卡}

{ 当我们的计算机需要进入局域网使用网络资源的时候，我们需要在计算机里面安装网卡，本文将带领你一步一步了解在计算机中怎样安装和设置网卡。
在安装之前我们需要准备好网卡，螺丝刀和网卡的驱动程序磁盘。
首先我们关掉计算机的电源，对于现在使用ATX电源类型的计算机还需要把电源线拔掉，才能彻底切断计算机的电源。打开计算机的机壳，根据你所安装的网卡的总线类型，再把主板相应总线插槽所对机箱位置上的挡板取下来。现在取来网卡，注意手不要接触到金手指(网卡与插槽的接触面)，然后从网卡上部用力插入相应的总线插槽，用螺丝把网卡固定在机箱上。
下面就可以把连入局域网的网线连接在网卡上，根据所在局域网使用网线类型，把相应的BNC接头接在网卡的BNC接口上；RJ45插头插在网卡的RJ45插孔中。现在就可以盖上机箱外壳，接上电源，打开计算机，进行驱动程序的安装。
Windows 9×/Me下的安装
启动计算机，进入Windows，它将检测到安装了新的硬件，并提示“发现新的硬件……将安装新硬件的软件”。然后Windows启动“添加新硬件向导”来进行驱动程序的安装。单击下一步，进入安装向导搜索驱动程序窗口，这时我们插入网卡配套的驱动程序软盘，选中“指定位置”，输入相应操作系统网卡驱动程序的软盘文件夹(例如A\win98)，或者单击浏览并定位到相应操作系统网卡驱动程序的软盘文件夹。然后单击下一步，Windows就会提示找到相应的驱动程序并进行安装。由于网卡的生产厂家在驱动程序中都预先设定安装某些网络协议，安装的过程中Windows需要从Windows本身的安装文件中安装协议，所以你还要准备好Windows的安装盘。
如果你计算机的Windows安装过比较新的系统升级补丁，则在安装协议时，系统会提示，从Windows安装文件安装的协议文件比当前的还旧，你只需要按照Windows的建议选择保留就可以了。
当安装结束，系统将会要求重新启动计算机，以便使驱动程序开始工作，让网卡工作起来。
Windows NT 4下的安装
由于Windows NT对即插即用不支持，Windows启动以后并不能发现新硬件，所以我们需要手工安装。进入控制面板的网络，在适配器一栏，点击添加，然后Windows将建立适配器列表，选择“从磁盘安装”，然后系统提示插入磁盘，我们插入磁盘并且输入驱动程序在磁盘中的位置例如A\winnt4，然后点击确定按钮，就开始从磁盘中安装网卡的Windows NT驱动程序，安装完成以后也需要我们重新启动计算机。
Windows 2000下的安装
Windows 2000综合了Windows 9x 和 NT的优点，所以安装过程与Windows 9x类似，比较简单，这里就不赘述，只是Windows 2000安全性比较高，有时在安装过程中会提示驱动程序没有数字认证签名，由于大部分厂家都没有向微软交钱获取数字认证，所以不用理会继续安装就可以了。
当驱动程序安装完成并且重新启动以后，我们的网卡也就开始工作了。我们的网络有些是100M有些是10M，现在很多PCI接口的网卡是10M/100M自适应型，能够设置网卡的工作速度。通常这些网卡都能自动调整工作速度适应网络，某些情况下我们可能需要自己设定网卡的工作的速度，下面我们再看看怎样为网卡指定工作速度。
我们进入控制面板的网络设置，在配置中选中网卡然后点击属性，再进入“高级”面板，在这里我们就可以设置网卡的工作参数，其中的LinkSpeed参数就是用于设定网卡的工作速度，缺省值一般是AutoMode或者AutoNegotiation厂家不同而显示的名称略有不同即自动适应工作方式，我们可以选择自己需要的速度让网卡工作在该速度下。一般有以下这些速度可以选择：10 Full Mode10M全双工，10 Half Mode10M半双工，100 Full Mode100M全双工，100 Half Mode100M半双工。全双工和半双工的区别在于全双工可以同时进行该速度下的传送和接收，而半双工在该速度下就只有传送完成后才能接收。
按照上面一步一步进行操作，网卡的安装应该可以轻松完成。}
