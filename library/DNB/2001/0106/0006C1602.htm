{朋友？奸商！}

{ 稍有一些购物经验的人都知道，买东西最好不要到朋友那儿去买(亲戚除外)，更不能到通过别人介绍的朋友那儿去买，这不是说朋友的东西不好，而是万一出了什么问题再去讨还公道，就要考虑到情面问题，更有可能会损伤友谊。这条生活经验在购买电脑时也同样有效，下面我就把自己的遭遇讲述一下，希望能引以为戒。
去年12月初，我的小姨要为它的女儿买台电脑学习使用，不要求玩游戏，预算在6000元以内，并求助于我。于是我趁周末去了徐家汇的百脑汇和太平洋电脑城摸了一下价格，并且找了一家声誉较好的公司谈妥了价格，说好第二天去买。当时的配置是赛扬Ⅱ 600，升技BE6(815E芯片组，带显卡、声卡)，HY 128M，星钻20G(缓存2MB)，三星750S显示器，Acer 40×光驱和键盘、鼠标，还有机箱等共计6000元。
等到第二天，小姨开车来接我时，还带了另外一个陌生人，听说是姨父的同事，并且准备到姨夫的同事的朋友(关系好复杂哦)那儿去买。我一听马上坚决反对，并举出n条理由。可那位同事的一句话“我的机器就是他那儿买的，用了一年了没问题，而且因为是熟人，所以价格便宜，不用担心假货。”就使我的反对被置于了九霄云外。
经过半个多小时的颠簸，我们终于来到了那位同事的朋友的电脑公司。一开始，我就将那张配件清单拿给JS看。他看了之后马上说：“这个价格我不卖。”接着姨夫问他是否价格太低，并且提出可以加价。但JS却一反常态说：“不是说价格太低，如果我要卖给你们的话还可以再便宜100元，但这种类型的配件我没卖过……”并且在对我的电脑知识大加赞扬的同时却贬低了我的社会经验，让姨夫和他的同事认为我的能力还不足以帮助他们买电脑，并对我失去信任。之后的事就可想而知了，我的话没起一点作用，而JS拿出了他准备已久的配机方案。我一看，大吃一惊，赛扬Ⅱ 600，杂牌主板(VIA 693A)，HY 128M，西部数据15G(缓存512K)，杂牌TNT2－M64 32M(显存8ns)显卡，其余的和我差不多，但价格为62XX元。我粗粗一算，天哪，将近800元的暴利。但这时，姨夫他们已被搞得晕头转向，竟然同意，连我想看显卡真伪的行动也被横加指责，还说朋友的东西不会假。而在我刚进门的时候就看到了假货——雅马哈734声卡。
最后JS又在机箱的挡板上贴上了几张小纸条，这是为了防止你撕开看到机器里的真面目，如果撕开了他就不给保修了。
就这样，购机结束了，JS获得了大丰收，而我也总结了一些经验，这也是我所要告诫大家的：
1.买电脑的时候事先选好的配置千万不要因为JS的各种各样的理由而任凭JS改配，尽管可能在改了以后价钱要便宜一些，但是JS的利润可就更大了，而且买的东西性能不一定好。大家一定要坚持自己的意见，因为在电脑城里每个商家都是可以调货的，自己没有的货也可以从其他商家那里得到。
2.买电脑要去当地的那些较大的电脑商城和信誉较好的商家那里去买，由于竞争激烈，而且去的都是懂电脑的人，所以那儿的JS不是那么黑，同时还要找专业经营电脑的大公司去买。
希望这篇文章对你有所启发，祝大家能在购机热潮中买到价廉物美的电脑。 }
