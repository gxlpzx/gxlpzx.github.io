{ Realplayer截图三法}

{ Realplayer的出现，提供了一个低成本观看影视的可能。至少我是用Realplayer看了许多过去想看而花费不起的电影、电视剧。但是，和许多朋友一样，当我想把屏幕上迷人的画面截下来时，发现无法和VCD一样截图。用了几种常用截图工具截下来的都是黑黑的。难道Realplayer的画面只能“匆匆一瞥”，无法“长驻心间”？兄弟我网上网下拜师访友，千辛万苦，终于搜集到几种realplayer的截图大法。看到网上不断地有网友询问Realplayer如何截图，赶紧请小编给点版面，把方法告诉大家。
第一种方法是用《东方影都》立体版，它直接提供了相关功能。不过你要花40元钱，呵哈，支持国产正版软件吧！
如果你的爱机里装有大名鼎鼎的HyperSnap－DX的话，我就告诉你第二种方法(我用的是4.01.09版)。什么，截不下图来？忘了说了，首先要做些设置，follow me：进入“捕捉→启用特殊捕捉”，把两个有关DirectX的复选框都选上。好了，要截图时按下Scroll Lock键或自定义的快捷键。是不是截下图来了？
“舍近求远”可能是很多电脑迷的通病了，明明手头的软件就可以用，偏偏不断地下载、安装、试用新的软件。说到Realplayer截图也一样。其实用Windows自带工具就行了(你不能不佩服微软及Windows的功能强大吧)。这是第三种，最简单的方法：当你播放Realplayer时，按“Ctrl+P(暂停)”，再按Print Scree键。然后，打开“开始→程序→附件→画图”，再粘贴。请相信自己的眼睛，这是真的，这幅美丽的画面你已截下来了。}
