{保存网页有新招}

{ 当我们用IE浏览因特网时，如果看到漂亮的网页或有用的资料，总是想把它们保存下来，以便日后查看，你是怎样做的呢？相信绝大部分朋友都是通过“文件”…→“另存为…”，然后在对话窗口中选择保存为HTML文件。但大家也一定发现了，用这种方法保存网页，在默认情况下，还会产生一个存放网页图片的文件夹，这样整理起来很麻烦。有没有更好的办法呢？其实，你只要把“另存对话框”的“保存类型”拉下来就会发现，还有好几种保存类型供我们选择。
当我们看到想保存的漂亮的网页时，选择保存类型为“Web 电子邮件档案(.mht)”，这样IE就会把网页上的所以元素，包括文字和图片集成保存在一个mht类型的文件中，不管把这个文件带到哪里，双击以后，IE都将为我们打开一张图文并茂的网页，是不是方便多了？事实上，这种格式就是用来发送HTML邮件的文件格式，这就是为什么发送HTML电子邮件时只需要一个文件就够了的原因。
在对图片不太重视的情况下，则可以选择“Web页，仅HTML(.Html；.htm)”，这时就会甩掉图片文件夹的尾巴。如果你只是冲着文字资料而去，那干脆选择最后一种类型“文本文件(.Txt)”，这样，既简单明了又节省了磁盘空间。
}
