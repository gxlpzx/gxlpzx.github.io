{升级Windows Me应注意显示问题}

{ 现在很多人都将自己的Windows98升级到Windows Me，体验千禧版视窗系统带来的强大功能。但在升级过程中，大家往往会忽视显卡或显示器的设置是否正确这一问题，结果导致显示系统出现故障。为此，笔者就其中可能出现的问题进行了总结，望能给升级系统的朋友提供一些帮助。
一、与显卡有关的问题
一般而言，系统升级到Windows Me后，它会自动更新显卡的驱动程序和 DirectX(微软开发的用于加速视频性能的3D图形接口)，但在这一更新过程中可能会遇到以下问题：
1.Windows Me安装光盘中没有包含原显卡的驱动程序，那么系统因为不认识该显卡就会自动以标准VGA模式640×480×16色启动；
2.有的显卡自身或原系统内包含专门的显示配置程序，如“刷新率实用程序”、“颜色匹配实用程序”等，升级到 Windows Me后，由于兼容性的原因可能导致这些实用程序不能继续使用；
3.通常由厂家提供的显示驱动程序和配置工具程序是相互依存的，如果删除其中一项，就可能会破坏另一项。而新版Windows内含的显示驱动程序有意识地做成了“通用驱动程序”，以便为标准的 API(应用程序编程接口)和功能提供更稳定的支持。不过由于每个驱动程序可能包含不同的配置，因此不可能用一个驱动程序来支持所有的实用程序。
当出现上述三个问题时，你可以通过厂商提供的最新驱动程序和相关的配置程序进行显卡的升级来解决。
4.如果你的 PCI 显卡被 BIOS分配使用IRQ 15，而一个功能正常的IDE硬盘控制器在默认情况下也被配置成使用 IRQ 15，这时Windows会把 IRQ 15分配给 IDE 硬盘控制器。这样一来系统将强制显卡使用VGA模式。解决方法是：(1)如果BIOS允许，可以禁用显卡的IRQ或手工配置显卡使用其它IRQ；(2)从硬件供应商处获取 BIOS 的升级版本。
二、与显示器有关的问题
在升级到Windows Me之后，也要注意对显示器进行适当的调整，以免出现问题。
1.在将显卡的分辨率调到更高之前，应当指定准确的显示器类型。方法是：单击“开始”→“设置”→“控制面板”，打开“显示属性”对话框(当然也可以用鼠标右击桌面)，单击“设置”标签，再点击“高级”按钮，单击“监视器”图标，然后选择“更改”。这时系统会显示所有监视器驱动程序的列表，从中找到你的显示器并正确设置即可。如果列表内没有你的监视器类型，那么你最好到该监视器生产厂商的主页上去下载相应的驱动程序，或者在列表中找一个类型相近的显示器代替(通常这种选择不会对显示输出性能和质量产生负面影响)。
2.正确设置显示器的刷新率，以免损坏硬件！步骤可按上面所述，先进入“设置”选项卡，然后单击“高级”，点选“适配器”标签，然后从列表中选择一个刷新率。注意：必须先选定监视器后才能设置刷新率。如果将“监视器”设置为“未知监视器”，则无法自定义刷新率，也就是说刷新率受显卡和监视器两者性能的影响。虽然Windows 提供显卡和监视器组合能力内的所有刷新率，但也不可将刷新率设得过高！
3.如果升级后显示结果看得见，但图像显示有问题，则可能需要进行如下设置：用鼠标右键单击桌面，然后单击“属性”，点选“设置”选项卡，然后单击“高级”，单击“性能”选项卡，将“硬件加速”滑块向左移动一格。如果问题还没有解决，可重复上述过程，将滑块再向左移直至问题解决为止。
注：把“硬件加速”滑块向左移将禁止使用显卡的某些图形加速功能。这种方法往往能解决由于显卡不兼容造成的显示问题。
4.如果在机器启动时无法正常显示，可在启动时按住Ctrl 键不放，直到出现启动菜单，然后选择“安全模式”，让Windows在VGA 模式下启动。启动成功后再用右键单击桌面，选择“属性”，单击“设置”，然后单击“确定”。Windows会提示是否以 VGA 模式重新启动，单击“是”，计算机重新启动。当机器在640×480×16色模式下运行后，重新设置。方法是：右击桌面，选择“属性”→“设置”，然后从中选择一个合适的分辨率并重启 计算机，这样处理后问题一般都会解决，而且设置的分辨率、颜色数都是可用的。
5.虽然多数程序允许动态调整颜色数，但有些程序在颜色改变后不能正确显示颜色或其它信息。要避免这个问题，需要在运行程序前更改色深。如果在运行程序时更改色深，另外还可能需要重新启动程序，以确保更改设置有效。 }
