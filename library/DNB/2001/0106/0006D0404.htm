{图片采集工具：PicPluck}

{ 可能大多数人都有这样的经历，在看到某个网站上漂亮的图片时，恨不得把它们“一网打尽”，却又不得不捺着性子等待一个页面一个页面地打开，一张一张地把图片保存起来。不过有了PicPluck，你可以悠闲地采集图片了！PicPluck 能一次性下载电影和整个网站的图片。
一、软件安装
将下载的安装文件解压到某个临时文件夹，并运行其中的 SETUP.EXE。和其他大多数安装程序一样，它也使用了标准的 Windows 安装向导的界面，可以轻松完成整个安装过程。
二、使用简介
安装后，从“开始”菜单中选择 PicPluck ，程序便开始运行，程序的界面中包括了三个子窗口：Viewer、Browse和Plunker。其中“Viewer”为图形浏览器， 在此浏览器的地址栏中输入一个路径，指定图片文件保存的文件夹。然后点击“Go！”进入该文件夹。如果对保存路径不确定的话，可以用“Browse”来浏览整个文件系统。在左边的文件目录中选择一个图形文件，浏览器程序将为你显示出来。如果不想保留当前显示的文件，可以用“Delete Image”按钮删除。
1.查找网站图片
选“因特网浏览器Browse”，在此窗口中浏览网站，寻找你需要下载图片的页面。在 URL 输入栏中填写你要浏览的站点的地址，并点击“Go”按钮，PicPluck 便开始为你打开所要的地址。如果遇到想下载图片的页面，点击“Pluck this page”按钮进入“图片下载窗口Plunker”下载该页面链接到的图片(如^06040404a^1)。
注：查找网站图片也可以使用网上搜图能手Internet Graphics Finder。其下载地址是：http://www.newhua.com.cn/down/Grif31setup.exe，只要你在Internet Graphics Finder的搜寻方格中填入你想要搜寻图片的名称，比如某个歌星的名字，它就会为你搜寻网络上所有包含有该歌星的图片并让你马上就看到。
2.设置下载参数和进行下载操作
选“图片下载窗口Plunker”， URL：填写开始下载图片的地址。点击“Go”按钮开始下载图片。Incoming：设置用来临时存放下载的图片文件；Transfer：设置用来存放下载完成的图片文件(如^06040404b^2)。
(1)设置下载的层次：选“Get all links/images on just this page ending in .JPG/.GIF”：指定 PicPluck 下载当前页面上直接可见的或者仅有一层链接的图片。想下载更多的图片的话，可以设置下载的层次数量，选择“Up to × clicks awax”选项并输入相应的数字。如果层数足够多的话，甚至可以下载整个网站的图片！不过需要注意的是：层数的增加带来的除了更多的图片，还有更长的下载时间。而且时间的增加通常是成倍数增加的，如果网络速度不快的话，最好限制层数或者限制文件的大小。
(2)设置下载文件格式：PicPluck 可以下载 JPG、GIF、AVI、MPG 等四种文件。缺省的是只下载 JPG 和 GIF 格式的图片，如果浏览的网站包括了视频文件，而网络速度又足够快的话，可以选择相应的视频文件选项。
(3)其他设置：“Don＇t walk offsite”：一个网站通常拥有到别的站点的超链接，如果不限制 PicPluck 只在同一个站点下载的话，它很可能在几次跳转以后就不知道链接到什么地方去了。使用这个选项可以限制 PicPluck 只在当前站点下载。“Don＇t download anx pics if name is alreadx in incoming dir”：防止 PicPluck 重复下载已经下载过的图片。“Min/Max Picture size”：设置下载图片的大小的上限和下限。以千字节作为单位。“Use Maximum File Name Collision Avoidarx”：PicPluck 的最有特点的功能之一。它将通过 CRC 32 校验方法来检查文件是否重复，并且不保留下载的文件的原来文件名，而是使用一个每个文件都唯一的代码来作为文件名。这个功能将有效地防止来自不同站点的同名文件的重复问题。PicPluck 还有一个重要的功能就是例外地址的设置，凡是被设置为“例外地址”的站点都不会被 PicPluck 访问。在“View”菜单中选择“Edit Exclusion List”或者使用快捷键 “Ctrl＋E”，便可以打开例外地址编辑窗口，你可以在这个窗口中设置“例外地址”的关键字，此后，地址中凡是有这些关键字的站点都将被禁止访问。在“View”菜单中选择“Options”命令打开设置窗口，可以对代理服务器、缺省的图片保存文件夹以及一些自动功能进行设置。
下载完成以后，图片文件都保存在临时文件夹中。可以在 Plunker 窗口中直接浏览下载来的图片。使用“Delete”删除选中的文件，用“Move”将文件发送到图片保存文件夹中。
