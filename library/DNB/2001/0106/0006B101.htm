{ 收藏夹清理能手
——IE收藏夹网址检测2001
对于经常上网的人来说，IE浏览器中的收藏夹是不知不觉地“长大”。网址虽然越来越多，但并不是每个都天天访问。时间一长，有些网址的链接已经失效，自己却不知道。怎样才能快速地知道哪些网站地址已经失效，并删除这些无效的网址呢？使用《IE收藏夹网址检测2001》就可以轻松地解决这些问题。这是一个国人自己开发的全中文软件，使用相当方便。安装启动后，程序会自动导入IE收藏夹中的全部地址，你只须点击“开始”，程序就会使用二十个线程快速地检测列表中的所有地址。这是个注册软件，未注册的有次数限制，只能使用30次。
下载地址：http://heqingsheng.org/program/urlscank.zip 2.4MB
上网浏览新利器
——Fast Browser
IE浏览器的诸多毛病经常让人抱怨，有没有其它的选择？Fast Browser就是一个非常棒的多线程浏览器。它在浏览器的基础上扩充不少功能，让你上网冲浪变得更加轻松和简单。它能同时开启180个页面；其独特强大的在线翻译功能，能提供英汉、日汉等多达36种语言翻译或简繁转码，并提供多个在线翻译引擎；自带的小记事本，让你迅速查看当前页面的文本、源代码、链接；拥有强大的搜索功能，可使用500个搜索引擎进行搜索，将网上资源一网打尽(中文版另有数十个各种中文分类引擎)；语音版本提供语音功能，可以自动朗读英文网页。
下载地址：http://www.fastbrowser.net/download/fbInst.zip 1.3MB
系统优化工具
Sxstem Optimize
电脑用久了，就会出现一些问题，感觉系统的性能不如以前，一般都会想到用优化工具来优化自己的系统。 Sxstem Optimize就是其中一种。
Sxstem Optimize是一个免费软件，支持Windows 95/98/ME。它提供了一个优化向导帮助你优化你的Windows操作系统，可优化系统的项目有：DirectX、internet、BIOS、CD－ROM、Mouse Speed、VMM Map Cache、Minimum Time Slice、连线速度、显卡和CPU的性能。它是以更改你的注册表文件资料来调整到最佳的效果。最新版本为3.8版。
下载地址：http://johnsmith50.tripod.com/cgi－bin/Sxstem＿Optimize.EXE 134KB
系统安全管理员
——Securitx Administrator
Securitx Administrator是一款不错的系统安全保护工具，可以让你完全掌握计算机的访问控制权；根据需要定制有访问权限的人员及其权限的等级；保护许多的系统设置选项、隐藏驱动器、禁用DOS窗口、禁用注册表编辑器、禁止右键菜单、禁止网络访问等。Securitx Administrator还可以隐藏桌面图标、开始菜单选项及任务栏等。另外，可以设置密码限制用户运行指定的应用程序，并且可以查看计算机的使用记录。
Securitx Administrator的最新版为5.2
下载地址：http://www.softheap.com/download/secagent.zip 857KB
超级文件管理器
——Windows Commander
既然Windows Commander以下简称Wincmd是一个超级文件管理器，那么就必有它过人之处。与Windows资源管理器不同，Wincmd有两个并列的显示窗口，可以用于显示两个不同的目录，当需要在文件夹或驱动器之间复制、移动文件时显得十分方便。Wincmd的可定制工具栏上可以加入你需要经常运行的程序、还要在底部的编辑框内直接键入文件名运行程序和Windows命令。
Wincmd除了可以管理本地和网络中的文件，还可以管理WEB站点的文件，它内置的FTP客户端程序可以上传、下载文件，具有一般FTP工具软件的功能。
Windows Commander的最新版为4.52版，这个版本可以用Wincmd自带的Lister显示Unicode和HTML格式的文件；搜索文件时可以根据包含的Unicode码和16进制内容来进行查找以及打印目录树等。
下载地址：http://tucows.tierranet.com/files4/wc32v452.exe 1.29MB
不仅仅是拨号
——HiDialer 2000
也许朋友们早已习惯了Windows自带的简易拨号器，其实类似的工具软件还很多，HiDialer 2000就是其中的佼佼者。它除了具备拨号、显示连接速度、连接时间等基本功能外还有一些非常实用的功能，比如你可以让HiDialer 2000在网络连接成功后自动启动你指定的程序，比如：IE浏览器或Outlook等，并且可以在断开连接后自动关闭它们；你还可以设定定时拨号连接、调整系统时钟、自动重拨等；它还可以保持网络连接，以防止因空闲而断线，另外HiDialer 2000还是个理财好助手，它可以帮助你计算网络连接费用。
目前HiDialer 2000的最新版本为V2.6 build 777。
下载地址：http://www.kgsoft.com/ftp/hd2000.zip 1.02MB}
