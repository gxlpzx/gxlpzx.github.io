{ 软件名称：方文Data2000
运行平台：Win9x/Nt/2000
软件授权：免费软件
软件作者：方文设计室
软件版本：1.0 Beta3 Build 245
软件主页： http://funstudio.xeah.net
软件简介：Data2000是一款资料管理软件。 它界面华丽，是各位很熟悉的OutLook Express布局，符合使用习惯。该软件的一大特色是管理方式采用无限层次的分级管理体系，而且可以建立无数个文档集合，(使用本软件的STR格式，只要你的硬盘够大)。换句话说，你可以把你的文档集用软盘带着走，只要有Data2000的地方，就可以打开。本软件另一特点是它的编辑功能，除了普通的文书处理，如字体，大小，颜色等，还可以设置上标，下标，而且支持图片和OLE对象的插入，支持文本中的超级连接。另外，其它功能如全屏浏览、导入、输出、发送电子邮件等一应俱全。新版更增添了回收站和收藏夹，以及一个向导。用户还可以使用非常丰富的图标。整个编辑界面类似于Word，而且用户可以从主页上下载许多该软件的资料库。
作者自述：经过几个月的“磨练”，Data2000这款软件终于有了全新的面貌。虽然花了一年多的时间来开发，但我们并不觉得辛苦。相反，软件做好后，如何让大众接受，才是最为让人焦虑的一件事情。说心里话，不高的下载量让人觉得这一年多的时间没有回报。不过，我们并不灰心，因为我们的团队还很年轻。此软件的主要开发者也不过刚20岁。作为Delphi业余爱好者，本来就不想靠它赚钱，但最想得到的，是大众的认可。所以希望电脑爱好者能够试试这款软件，并提出中肯的意见，也能促进我们今后拿出更有水准的作品出来。}
