{让 VirusMD Personal Firewall守住你的家门}

{ VirusMD Personal Fire－
wall是一款功能强大且非常容易上手的个人网络安全防火墙，它无需任何设置，用户只需按下几个简单的功能按钮，它就能够肩负起“保家卫国”的重任。VirusMD Personal Firewall还有一个非常有趣的功能，可以让你和那些来去无踪的黑客们正面交锋，比如你可以给入侵的黑客发去如下的警告信息：你的IP地址已经被记录在案，你将被移交公安部门处理！嘿嘿，要是碰上了胆子小一点儿的黑客，裤腿里恐怕会流出不少的xx。
软件的基本信息
VirusMD Personal Firewall是一款共享软件，最新版本v2.0，文件大小2.65MB，未注册的版本可以免费试用30天，软件的下载地址是http://www.virusmd.com/。
软件的界面^06040701a^
软件安装完毕，在桌面上和开始菜单的程序组中会出现VirusMD Personal Firewall的快捷项。点击该快捷项，VirusMD Personal Firewall即被启动，不过这时候它并没有出现在桌面上，你在任务栏的系统托盘中可以看到一个钥匙形状的小图标，双击该图标，弹出程序主界面。程序界面分为四个部分：第一部分为“Actions”区域，该区域中有两排功能按钮，可以实现VirusMD Personal Firewall的所有功能；第二部分为“Blocked Ports”区域，在该区域中有12个端口输入框，在这里可以填入一些易被黑客攻击的端口号，这样VirusMD Personal Firewall就可以对这些端口实行特别的保护；第三部分为入侵黑客动态记录区域，当有黑客企图入侵时，在这里能够显示出入侵黑客的IP地址和企图入侵的端口号；第四部分为信息发送区域，你可以在这里撰写需要发给黑客的警告信息。
软件的使用
●保护易受攻击的端口：VirusMD Personal Firewall一次能够为你同时保卫12个计算机端口的安全。在“Blocked Ports” 区域中键入12个易被黑客攻击的端口号，然后点击“Protect”按钮，这些存在着很大安全隐患的端口就会被VirusMD Personal Firewall牢牢地锁住。在这里，VirusMD Personal Firewall已经为你预先设置好了12个易被攻击的端口，你也可以根据目前黑客程序使用端口的最新情况，进行必要的更改。
●获得最新的黑客程序信息：要想更好地保护你的计算机端口不被侵犯，你必须掌握最新的黑客程序信息，了解他们使用端口的情况。各类计算机媒体上经常会发布一些有关的信息，你也可以发信到以下的地址contact＠virusmd.com去获取有关的信息。另外，点击“Commonn Ports”按钮，可以调出一些常见的黑客程序所使用端口的列表，比如端口“25” ，就有13种黑客程序需要通过该端口入侵。
●警报系统：当有黑客企图闯过被VirusMD Personal Firewall保护的端口时，VirusMD Personal Firewall马上会给出警报信息，你会听到一阵警报声，紧接着会有一个女声向你报告：Intruder Detected(发现入侵者)，随之你的桌面上会弹出一个信息通报窗口，窗口中会显示如下的一些信息：The hacker has been stopped(黑客已经被拦截)！如果你有兴趣的话，你这时候可以给企图闯入的黑客发个警告，让他知难而退。当然如果这时候你想收拾他也是易如反掌，因为他的IP地址和入侵的端口号都在你的掌握之中。
●黑客入侵记录：当VirusMD Personal Firewall阻挡了一个黑客的进攻之后，在“Intruder IP”和“Ports”下的正文框中会显示出入侵者的IP地址和他企图进攻的端口，并且这些信息会自动生成一个日志文件保存在程序的安装路径下，这个日志文件的重要性可是非同小可，因为它是入侵者留下的罪证，你可以把这些罪证交给黑客所在的ISP或者有关的执法部门，让他们受到应有的制裁。
●端口扫描：如果你想了解当前计算机所有端口的状态，可以执行端口扫描功能。点击“Scan Ports”按钮，弹出“Scan Ports”窗口，点击“Scan All Possible Ports”按钮，VirusMD Personal Firewall便会迅速对计算机上万个端口进行扫描，并向你报告当前计算机的哪些端口正在被使用，扫描结束后，如果想保存扫描记录，点击“Save to log”按钮即可。当整个扫描工作结束之后，你可能会发现有许多端口已经被打开，其实这并不说明你已经遭到了黑客的攻击，因为许多正常的程序都需要使用一些端口，比如你的浏览器使用的端口是“80” ；这时候假如你已经启用了VirusMD Personal Firewall保护端口的功能，那么这些被保护的端口也会显示正在被使用。
●程序进程管理：VirusMD Personal Firewall 提供了程序进程管理功能，通过该功能，你可以查看到计算机当前所有在后台运行的程序，并且你可以随时终止其中任何程序的运行。VirusMD Personal Firewall提供该功能的目的是为了帮助你随时终止在后台悄悄运行的黑客程序。点击“kill applications” 按钮，弹出“Application Killer”窗口，点击“Show Running Tasks”按钮，在列表框中会出现当前在后台运行的所有程序，如果你想终止某个程序的运行，请选中该程序，然后点击“End Selected Task”按钮即可。
结语
尽管VirusMD Personal Firewall在功能上很有特色，也非常容易上手，但多少还是有些不尽如人意的地方：第一，由于它一次只能同时保护12个端口不被侵犯，会给人一种寡不敌众的感觉；第二，再就是选择哪些端口作为保护对象，如何最有效地利用这些宝贵的端口资源对一般用户来说并不是一件容易做到的事情。但瑕不掩瑜，VirusMD Personal Firewall还是值得大家一试的。}
