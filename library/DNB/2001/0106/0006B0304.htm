{磁盘清扫机Clean Disk Securitx}

{ 当我们要从电脑上删除文件和资料时，通常的方法就是将它投入回收站并清空。但有许多反删除工具可以让已被删除的文件“起死回生”，这样，一些机密的资料就将全盘“曝光”。所以并不是清空回收站就可以高枕无忧了。这里笔者介绍一款好工具─Clean Disk Securitx，它能将不需要的机密资料(包括历史记录等)彻底删除、不留痕迹！软件使用方法如下：
第一步：在菜单的“Drive to secure”(欲保护的硬盘)中使用浏览方式选择相应的盘符(如C、D、E、F……)。
第二步：在对象栏中选择需删除的对象类型，这里共有八项：从上到下分别表示“清扫标准空闲空间”、“ 清扫文件不连续空间“、“清扫最近文件列表”、“清空回收站”、“清扫Windows临时文件夹”、“清扫互联网浏览器临时文件”、“ 清扫历史记录”、“清扫Windows交换文件”。用户可根据要操作的对象进行选择。
第三步：对于“Method”(方法)选项，建议保持默认项“Simple”(简单)就可以了。
第四步：点击“Clean”(清除)，软件会提示你是否先运行磁盘检查程序。完成这一步后，清扫工作就开始了。程序可清理硬盘中未使用的空间，让被删除的文件不会有“undeleted”(恢复被删除文件)的机会，这个清理的动作并不会对存在的文件有任何影响；另外可以彻底删除存在的文件，这个功能就是结合了删除及清理文件的动作，直接将文件“人间蒸发”。有了它再也不怕重要文件被别人盗走了，当然，使用时请一定要小心谨慎，不要对重要的资料进行误删除。
Clean Disk Securitx是一款共享软件，大小为626KB，可使用在Win95/98/NT下，感兴趣的朋友可以到http://www.inhua.com/CleanDiskSecuritx.htm去下载最新的4.8版。
}
