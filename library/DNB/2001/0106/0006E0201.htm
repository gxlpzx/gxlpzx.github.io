{MD：带我回家}

{ 技术篇
　　目前，电脑及互联网的日益普及，带动了新的音乐革命。磁带已经风光不再，人们对它那“劣质”的音质和易损的结构已经不能容忍了。数码随身听日渐普及，Discman、 MDWalkman、MP3播放器占尽风头。但由于Discman个头过大，虽然音质最好，但仍不是很受欢迎。反观MD和MP3具有体积小的优点，便于携带，符合未来发展趋势，所以很受青睐。不过，从长远看来，MP3可不是MD的对手。
MD，全称MiniDisc(中文意思即迷你光盘)，它是由索尼于1992年正式量产上市的一种音乐储存媒体。MD随身听又分可录型MD(Recordable，有磁头和激光头两个头)和单放型MD(Pre－recorded，只有激光头)。MD是集磁、光、电、机于一体的高科技产品。它既具有极为接近CD的音质和长期保存性，又具有卡带的可录性，一张好的MD碟可以反复录制100万次以上，字幕总容量为1700个字(可显示的文字有英文、日文和特殊符号)。一张普通MD碟容量约为140M，装在一个大小为70mm×70mm盒子里。它采用ATARCAdaptive Transform Acoustic Coding，直译为“适应地改变听觉编码”系统进行压缩。目前最多可以存储320分钟的音乐信息。
MD碟片的尺寸比CD小得多，但可以存储比CD时间更长的音乐，原因就在于ATRAC技术。这是一种类似于MP3的压缩技术，最佳音质的压缩比约为1∶5，与MP3相比要大得多。不过MD的音质远非MP3所能比拟。究其原因，MP3的压缩率虽高，但对音乐的破坏程度更高，这一点，在听古典音乐的时候表现尤为突出。不信，你可以去找一张原版雨果发烧碟，再用MP3格式压缩一遍，找个好一点的耳机对比着仔细听一下，效果绝对是差异甚大，而ATRAC技术是索尼公司潜心研究，根据人耳能感知的听觉范围建立了心理模型声音解压缩技术，把一些人耳听不见的音域范围的声音去掉后再压缩和还原，而且历史比较久远，有一定的技术基础，且较成熟，虽说压缩率小了一点，但与CD音质的差别，则是“需要一双金耳加上一副索尼 E888SP之类的耳机才辨得出”(＾＿＾)。所以在音质上，MD是远胜过刚出道不久且已无法发展的MP3。
现在ATRAC的最新版本是3.0，即MDLP(Mini Disc Long Plax)，录音时间达到了160分钟(LP2模式)到320分钟(LP4模式)毋庸置疑，时间是由质量换来的，但好于MP3仍是肯定的，已经在索尼的MD机MZ－R900、MZ－E900，夏普的MT77中应用了可惜目前价格不菲。
另一方面，MD也有着极为精密的结构。MD碟片的核心，就是一片类似于MO的存储介质。MD碟是在上下两层透明的聚合物薄片中间夹入磁性材料层组成的。MD录音技术建立在磁场调制抄录系统上，只须放入MD碟片，它将在激光束及磁力装置的运作中旋转，以在MDWalkman上进行录音。在录音时，两面的光束闸均会打开，以露出CD。一方面，激光束将CD上的一个小点加热至居里温度即摄氏180度，在此温度下所有的磁性品质均被中和。与此同时，磁头发放出与从录音源所接收的讯号互相对应的电磁场。随着CD的转动及冷却，全新的一组磁化资料便由磁头制造而成。这组新的磁化资料相当于所提供的声音讯号，并且可被激光拾声器所读取。再录音的次数可多达一百万次，而基本无损音质。
除此以外，MD还有一个很重要方面，那就是它的防震系统。不要认为MD是机械结构，所以它的防震性能不好。其实MD通常有一个大小为4M左右的DRAM，作为MD的暂存空间。由于ATRAC数据读入加上处理的速度是0.3M/s而激光拾取器从碟片上读入数据的速度是1.4M/s，因此加在ATRAC系统和激光拾取系统之间的DRAM可将激光拾取器读取的内容暂存，供ATRAC系统读取处理。在磁头由于震动而无法读取信号时，DRAM中的信息仍可以保持10sSP到160sLP4的时间不间断。所以防震是绝对够了的。不仅如此，有的MDWalkman的碟片时转时停，也就是由于DRAM的存在，这样一来，理论上还可以节电。
除了这些，目前还有索尼的省电芯片，可以把单放时间延长到100个小时。索尼 MZ－R70PC、索尼 MZ－R35SPPC可以通过一个“音频to USB”的MD－PORT与PC或MAC连接，很方便，无须任何软件，甚至无须声卡，把MD－PORT的USB端插入PC的USB接口，然后让PC或MAC播放音乐就可以了。道理在于这个MD－PORT就像是一个USB声卡，可以接收处理PC或MAC的信号，将其转换为音频信号输入MD中。夏普 MT77则是将PC或MAC的信号转入光纤，更保证了音质的纯正。
眼下，MD和MP3之争闹得沸沸扬扬。就我自己而言，我比较偏爱MD，一是由于它的技术含量远高于MP3，而且是公认的“高级装备”；二是它的存储媒体价格便宜极了。要是MP3的存储媒体也是这个价，更多的人就不见得要选MD了；三是它的技术成熟，故障率低。
但很遗憾，目前MD无法存储PC的文件。因为虽然MD是数字格式，但输入的数据无法跳过有损压缩部分，不像MP3是专为PC设计的(1992年的PC还处于286时代)，即使连上PC也只能够通过“光纤转USB”或“音频转USB”，不过这也有个好处，不管有没有声卡一样用。我想，如果MD加入了一般文件存储功能后，一定能全面打败MP3。因为目前MP3的实用价值很多时候体现在可以用作外部存储设备。
从长远发展来看，MP3和CD显然不敌MD，但CD作为MD的音源还是会存在的，MP3可就难说了。如果不考虑存储文件问题的话，MD无疑是数码随声听的最佳选择。
选购篇
现在MD的产品也是五花八门，种类齐全。从不到1000元的索尼 MZ－E60低价单放机，到2600元的索尼 MZ－R900顶级录放机，型号约有几十种之多。但从ATRAC压缩解码芯片的所有权情况来看，市场上的内核只有3种，即索尼、松下松下的技术并不算落后，但在市场上索尼、夏普基本是各占半壁江山，其余的MD，都是用这三家公司的机芯或直接OEM的，如建伍是夏普的OEM、胜利是松下的OEM，爱华、JVC、先锋等也都是这样。据一些发烧友说，索尼和夏普的ATRAC算法各有不同(目前索尼用的是ATRAC 4.5版本的，夏普用的是 ATRAC 6.0版本)，而夏普更以24bit解码超越索尼的20bit解码，因此理论上音质要比索尼的好。
MD主机的选购，需要注意的问题可不少：
1.行货和水货
MD是有行货和水货之分的。所谓行货是指由正规渠道入境的MD产品，水货则一般是未缴纳关税入境的。无论从产品的性能、附加值，或是从遵纪守法的角度来说，行货都是最佳选择。不过对于水货要纠正一点，水货在这里不是指假冒伪劣产品，因为除了那么几家以外，没有厂家会生产如此复杂的产品。水货比行货上市早，对于一等一的发烧友，为了提早了解产品性能而选购水货，也是无奈之举。行货的配件齐全，还附赠光纤等。不过有个特例，夏普 MT－831的水货比行货多一个漂亮的充电座(还是少一条光纤)。如何取舍自己看着办。不过我觉得，一般来说，不到资金绝对缺乏，没有必要选购水货。说起区分方法，行货有中文说明书(也许是几国文字中有中文)，一般是220V电源，而水货是在日本本土销售，所以说明书是日文的，电源110V，只附带音频模拟线，没有光纤。
2.单放机和录放机
由于MD原版碟片极少，所以自己录制MD碟成了一个无法回避的问题。MD空白碟片可以在MD台式机或MD录放机上录制。如果你的朋友或亲戚有可录MD机，那么买一个单放机就很值得了，目前最便宜的单放机索尼 MZ－E60武汉的报价不超过1000元。
3.录音时间
目前市场上绝大多数的MD机都支持在80分钟的MD碟上录160分钟的音乐。就MDLP目前而言只有索尼 MZ－R900、MZ－E900、夏普 MT－77等少数一些机型支持。而且，用LP4格式压缩的音乐只可以在支持LP4的MD机上播放不过自己录自己听还是可以的，虽说音质不佳，但课堂教学、现场录音效果远比MP3强。目前支持MDLP的录放机型多在2000元以上，你可以根据自己的需要选择。
4. 线控
目前我所见的MD机都带有线控，(夏普 MT77的线控)。爱华很多机型的线控都带有背光，晚上使用很方便，现在索尼与夏普的部分高档机型如夏普 MT77也有背光。选择线控还要注意一些细节，例如带不带皮带扣，按键设计符不符合你的手型等等。
5.颜色与风格
MD属于极为个性化的一种产品。不要忘了大多数MD都有颜色的选择，比如夏普ST66有4种颜色可选。另外有些MD有一些特殊的设计，例如索尼 MZ－R70有双耳机设计，一个带线控，一个不带等等。
总之，一切选择都以自己的个性与实际需要为出发点。下面我就一些主流机型进行简单的介绍。
一、录放机型
1.索尼 MZ－R70
索尼2000年新推出的机型，样式比较新颖。体形还可以，前部面板的设计有点像Yepp E系列，但背后的隆起实在不太好看。这是一款中低价产品，所以索尼省了线控上的LCD，也省了“口香糖”充电电池，导致播放及录音时间下降。不过，作为入门机型还是不错的。
2.索尼 MZ－R90
，索尼1999年的机型，中规中矩，各项性能比较平均，不过操作键饱受批评，是中档机型。
3.索尼 MZ－R900
索尼2000年的主打产品，性能绝对一流。加上多彩的个性化支持，以及MDLP的长时间录放和节电设计，MZ－R900无疑是顶级的录放机。不过价格偏高了。而且耳机效果一般，换一个耳机还差不多。
4.夏普 MT77
夏普和索尼协议的产物，MT66和MT831的换代产品。MT77也支持MDLP。它的显示灯比较新颖，操作性良好。LP4的音质与索尼 MZ－R900差不多，但调节功能不及MZ－R900，不过比MZ－R900便宜不少，还是很超值的。顺便提一下，夏普三位数型号的MD是本土生产，两位数的是马来西亚生产的。一般来说三位数机型是主打产品，不过好像MT77是个例外。
5. 松下 SJ－MR200
曾经是最薄、最轻、最小的录放机。性价比很不错，不过听说低音部分不怎么样。
二、单放机型
1.索尼 MZ－E60
历史上最便宜的单放机，武汉的报价是999元。不过除了这点以外，它就基本没有优点了。
2. 索尼 MZ－E800
性能和MZ－E900差不多，除了不支持MDLP外，基本没有缺点。价格也不错。
3. 索尼 MZ－E900
没得说。就是太贵了。
4. 夏普 ST－66
样子不错，仍见图2音质也不错，相比爱华 HX－70，贵了点。
5. 爱华 HX－70
最超值的单放机。打开节电功能后，播放时间超过了ST66和MZ－E800。DIP音效调节可以充分发挥出机器的性能。
一般来说，买了MD之后，就大功告成了。不过，还有几个小细节需要注意：
1. MD原配耳机一般不是最好的。所以建议买中高档机时把耳机换一下。好点的耳机有索尼 MDR－888SP(极品，460元)、爱华 HP－V533180元，知名度不亚于MDR－888SP等。自己在买机子的时候可以多试几副，挑选出最适合自己MD机的耳机。
2. MD机一般都是用自己或别人录的碟片。碟片的挑选也有讲究。索尼的最高级产品是MD2000。最高系列的型号是ES系列，74分钟/30元，其次是PRISM系列，COLOR COLLECTIONS系列等。其中只有PRISM系列见得到80分钟的碟，价格是74分钟/25元、80分钟/35元。COLOR COLLECTIONS系列就是彩碟，其颜色有红、蓝、绿、黄、黑五种，价格约是74分钟/20元。除了索尼，还有一家大供应商TDK也做MD碟，毫不逊于索尼。它的XA系列在索尼的MD2000出来以前是顶级MD碟，价格是XA74分钟/35元，XA PRO74分钟/80元(仅次于MD2000)。它的MUSIC JACK系列比较常见，也是彩碟，比COLOR COLLECTIONS多了一种金色。索尼和TDK也都有一个平价系列，索尼的是NEIGE系列，TDK的是XG系列，价格分别是74分钟/18元和74分钟/15元。质量都是有保证的，一般来说出现TOC读错误的几率很小。一般而言，一旦出现了TOC读错误，那么这张碟就报废了。
