{超级文件工具Turbo Browser2001}

{ Turbo Browser2001提供了非常丰富的文件管理功能和基本的互联网功能，而且界面简单明了，只要熟悉视窗系统界面就很容易掌握它的操作。从http://www.softseek.com/cgi－bin/download﹖08473－ftp//ftp.zdnet.com/softseek/trbwnt.exe下载后(大小为1.7M) ，直接运行安装程序即可。然后运行“开始/程序/ Turbo Browser2001”即可打开程序，界面如图1所示。下面让我来一步一步地介绍它的各种功能吧。
文件管理功能
这里的文件管理功能包括了我们常用的所有操作，比如打开文件、保存、重命名、新建文件或文件夹等等，其中打开文件命令还可以应用到打开远程站点上的文件(当然你要先建立远程站点的连接，这在后面的FTP工具中将会介绍到)。
1.文件移动与拷贝
Turbo Browser2001支持多窗口操作，可以同时显示几个文件夹下的文件，你可以直接用拖动的方式来移动文件，如果按住“Ctrl”键后再拖动即可复制文件。要把文件拷贝或复制到另一个没有打开的文件夹里，你也可以点击“Copx to”或“Move to”按钮来完成。
2.文件夹比较
运行“Tools / Compare Folders”，打开如所示的面板，选择好源文件夹、比较的文件夹和参数后，点击“Compare”(比较)按钮就会列出一份比较的结果，报告这两个目录下相同的或不同的文件有多少。
3.压缩与解压缩
Turbo Browser2001也提供了方便的压缩与解压缩功能。首先选择好要处理的文件然后运行“Tools/Zip...”即出现压缩面板，这里有一些参数你可以设置一下，比如压缩的比例、压缩的行为、压缩的参数等，也可以用默认的设置，然后点击“Zip”(压缩)按钮就行了。打开压缩包就更简单，只要单击就会在右边的预览窗口中显示压缩文件的内容。
4.文件备份与恢复
执行“Tools / Backup……” 命令，出现如所示的面板，选择新建一个Script文件。Script文件用来保存备份所作的参数设置，使你下次可用相同的参数来备份(但你要选择打开已存在的Script文件)。下一步填写好需要备份的文件夹和备份文件所在的目录；设置好参数后就可以备份了。
5.改变文件属性
从文件属性你可以看出该文件创建的时间、访问的时间和文档属性，若需要对这些进行更改，执行“Tools/Change file properties”，把需要更改的属性更正后按下“Applx”键即可。
6.文件过滤
Turbo Browser2001的文件过滤功能也很实用，当一个文件夹下包含多种类型的文件时，要找某一个文件有时会相当麻烦，这时你就可以使用过滤按钮了，选中要找的文件类型后，在文件列表框中就会只显示该类型的文件。
图片功能
Turbo Browser2001也可以作为图像工具使用，在处理图像时要先执行“View/Editor Toolbar/Image Toolbar”打开图像工具栏。你可以预览也可以编辑图像，如翻转、缩小或放大、选择一定区域拷贝等等。
1.预览图像
只要打开图像所在的文件夹，在文件列表中显示图像文件，单击图像文件就可以在工作区域显示图像。图像工具栏上有缩放按钮，可以对图像进行放大与缩小。
2.察看图片信息
当你用鼠标在图像上移动时，在图像左侧的显示区你可以看到颜色的信息(RGB值)、该处的坐标和颜色的十六进制的代码值。
3.编辑图像
Turbo Browser2001的图像功能也允许你对图像进行简单的编辑，图像工具栏上有矩形区域选择按钮，还有翻转工具按钮，也可以把选择好的区域拷贝到粘贴板中。
因特网工具
如果你懒得用CuteFTP这类工具，可以用Turbo Browser2001来上传、下载网站上的文件，还可以通过E－mail来发送文件，甚至可以使用Ping和Trace命令来查看网络的情况。
1.HTTP上传功能
执行“Internet/ HTTP Upload File...”命令出现面板，填写好上传的地址，选择好要上传的文件以及登录用的用户名和密码后，点击“Upload”按钮即可开始上传了。
2.HTTP下载功能
如果你知道要下载的文件所在的地址即可使用该命令，执行“Internet/ HTTP Download File……”命令，填写完后即可下载了。
3.下载整个网站
该功能非常实用、简单。执行“Internet /Download Web Site...”后出现如所示的面板，只须填写网站的地址“Enter the address of download site”和保存的路径即可，你也可以点击“Option”选项卡，对下载的层数、最大的下载页数等等进行设置。一切完成后按下“Start”按钮就开始下载了。
4.通过电子邮件发送文件
执行“Internet/Send Files via E－mail”命令，Turbo Browser2001就会调用默认的电子邮件程序把选中的文件作为附件，在选择好发送地址后就可以把文件发送出去了。
5.FTP工具
执行“Ftp/New Site”，设置好FTP地址名称、登录时的用户名、登录用的密码后，就建立了一个FTP连接。再执行“FTP / Dial－up”命令，如果设置没什么错误就会连上网站，并显示出站点上的文件目录，然后你就可以用拖放的方式来操作文件传输。
Html编辑功能
Turbo Browser2001的超文本编辑功能也很实用，单击所要修改的Html文件在右边的编辑窗口中就会分栏显示出文件代码和显示效果。代码编辑窗口中用不同色彩对不同的元素作标记，编辑起来非常方便。
与Windows Commander的比较
与Windows Commander相比，Turbo Browser2001已经不仅仅是一款文件管理工具，它还增加了多种实用的工具部件，不只提供对文件和文件夹的操作，还拥有对某些类型文件的简单编辑功能，它是一款多功能的工具软件，尤其是其中的图像功能和下载网站的功能非常实用。
在文件管理方面，Turbo Browser2001可以同时打开多个窗口，对文件的操作也省事不少。而Windows Commander只有两个窗口，操作起来有时会感到不便。
但Turbo Browser2001也有不足。与Windows Commander相比，它的文件管理有点不太体贴用户。在Windows Commander中用户可以自己编辑界面按钮栏，把常用的命令做成按钮，操作起来更加方便。而Turbo Browser2001虽然提供了大量的按钮，但是用户却不可以根据自己的情况来编辑。
总体上说Turbo Browser2001不失为一款非常优秀的文件管理软件。 }
