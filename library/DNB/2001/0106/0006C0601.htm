{轻松“指点”——MouseTool使用详解}

{ 编者：从本周开始，《硬件周刊》版块的某些版面作如下调整：原C11版的“品牌电脑”调整到《评测与市场》版块的H3版，C6版和C7版均为“软硬兼施”，望广大《电脑报》读者朋友注意。
一、MouseTool简介
大家都知道，使用鼠标最基本的操作就是先将指针指向相应的对象，而后辅之以单击、双击以及拖放等操作，这样需要的任务就能完成。MouseTool是一款十分有趣和实用的小工具，只要你的手指停止动作，它便能为你完成通常的点击操作，以缓解你的疲劳。和大多数软件一样，MouseTool的安装十分简单：下载该软件(下载地址为http://download.softhouse.com.cn/static/displax.html﹖categorx＝/tools/mouse＿kexboard/＆name＝41321mtool31)后只须双击安装文件，并按照提示进行操作即可。
二、MouseTool功能设置
运行MouseTool，出现所示的窗口工具条，上面有几个快捷图标按钮，其作用是用于切换 MouseTool 的单击状态，进行功能设置和查看帮助文档，大家可以根据自己的需要在不同类型的窗口中灵活控制 MouseTool 的缺省操作。先点击“Options”按钮弹出设置窗口，里面包括三个设置选项：常规设置(General)、热键设置(Hotkexs)和监视设置(Context Monitoring)。
1.在常规设置中，你可以设置一些常用操作。首先设置鼠标指针停留在一个单击目标上十分之几秒后MouseTool发出单击指令。例如你可在矩形框中输入数字5表示0.2 秒后发送单击指令，数字越小MouseTool 反应就越灵敏，但这样也常常导致误击操作，你可以设置一个适合自己习惯的延迟时间。其次，你还可设置在发送单击指令前鼠标指针可以移动的范围，它用屏幕上的像素数目表示(如果设置为2，那么如果在发送单击指令前鼠标指针的移动超过了两个像素的距离，就将取消发送指令)。这个选项主要是为了防止误击其他目标。另外在常规设置窗口你还可以根据需要选择是否保持MouseTool 窗口在最外层、是否在发送单击指令时给出一个声音指示、是否随 Windows 的启动而启动。如果启用“Manuallx Changing modes Overrides next click” 项目，则当使用状态改变按钮(位于 “Hotkexs ”设置表中)改变单击指令状态时，新的指令状态仅对下一个单击指令有效，这个单击指令发送结束后，就返回原来的单击指令状态。如果选择设置窗口下面的“ Enable Stretch Break Timer”(激活休息计时器)项，并适当设置两次休息的间隔时间或者设置到下次休息时的时间，MouseTool 就能够在设定的时间里显示一个提醒窗口来提醒用户休息一下，这时用户只有单击窗口中的“确定”按钮，才能够继续工作。这个功能一旦启用就是相对独立的，关闭 MouseTool 并不影响它的功能发挥。如果要取消这个功能，你可以打开此项设置将相应的项目禁止。
2.点击窗口中的“Hotkexs”标签，即可打开选项设置中的设置窗口。如果你想使用热键，先选中“Enable Hotkexs”项，然后设置鼠标双击、左击和右击的热键以及鼠标拖动的热键。通常程序会给出缺省的设置，如果需要改变，就单击每个热键旁边的 “Change”(改变)按钮，弹出热键设置对话框，输入新的热键，然后点击“OK”即可。另外按“Delete”按钮可删除已有的热键设置。
3.若点击窗口中的“Context Monitoring”标签，可以打开窗口类型监视设置窗口。在缺省情况下，对于窗口类型列表中没有的窗口，MouseTool 完成的是左击操作。单击一个要设置类型的窗口，而后单击上述窗口中的“Add”(添加)按钮，输入的窗口类型就被添加到窗口类型列表中，单击“Delete”按钮可删除窗口类型。单击类型列表中的某个类型，接着选定一个“Action”区域中的复选项目，即可指定 MouseTool 在这种类型的窗口中，缺省完成的是何种操作。在“Action”(行为)区域选定“Use default”(使用缺省值)以后，MouseTool 将根据用户是否启用“Smart Drag”(智能拖放)功能而决定是进行左击操作还是拖放操作。如果在某些情况下智能拖放功能引起了操作混乱，例如无意之中改变了文件的位置，就可以选定其中的“Disable Smart Drag” 项目来禁止智能拖放功能。设置好的操作类型可以在任何时候通过各种相应的热键或者单击 MouseTool 本身的状态改变按钮来覆盖。通常用户不会同时打开很多窗口，因此可以使用缺省的设置。所有设置完成以后，点击设置窗口中的“确定”按钮保存设置即可。
三、MouseTool的使用
1.启动 MouseTool 后，它会以最小化状态出现在系统托盘中。用鼠标右击图标，从弹出的菜单中你可以切换启用和禁止 MouseTool、执行左击、右击和双击操作，以及进行各项设置。如果选择常规设置中的“Enable Smart Drag” 项，则先将鼠标指针停住不动，MouseTool 就发送 Down(按住)指令，这时如果用户在一定时间内不进行其他任何操作，MouseTool 就接着发送Up 指令，而如果用户在延迟时间内移动了鼠标指针，MouseTool 就等到用户停止移动鼠标指针时再发送 Up 指令，这样就能实现拖放操作。因此，当你不工作时，最好将鼠标指针停留在屏幕上的“安全”区域，例如窗口的标题栏以及大部分灰色的空白区域，这样可以防止当无意中移动了一下鼠标指针时 MouseTool 发送操作指令。
2.利用其窗口类型监视功能，取消在不希望发生意外拖放的窗口中的智能拖放功能。在不同的窗口中，用户希望MouseTool完成不同的缺省操作。例如在一个文件打开或保存的对话框中，就可能需要 MouseTool 总是完成双击文件的操作；在桌面上，你可能希望 MouseTool 完成的缺省操作是双击快捷方式图标。
而对于一个按钮而言，可能不需要完成拖放操作因而希望禁止施加拖放功能等。为此，你可以启用窗口类型监视中的“Enable Context Monitoring” 项，这时位于窗口监视设置框底部的方框中显示的是最后单击窗口的 Windows 类型，将鼠标指针移到 MouseTool 窗口外部，你可发现方框中的内容有所变化。MouseTool 能够监视 Windows 窗口的改变，这样它始终明白鼠标指针所在窗口的类型，用户就可以告诉它在何种窗口中采用何种缺省的鼠标操作，如左击、右击、双击和拖放等。选择“Close”命令，退出此程序。
如果大家想换换鼠标操作的口味，MouseTool将是你不错的选择。关于该软件的使用有什么问题请与我联系，笔者的E－mail是wmmjcw＠china.com。}
