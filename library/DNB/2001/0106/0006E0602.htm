{用ICQ实现“全球通”}

{ 很多人都知道如何利用手机发送短消息，不过，对我们网虫来说，不用手机也能发送短消息才算得上够“酷”。下面，我就给大家介绍如何利用ICQ 2000b向手机用户发送短消息，实现“全球通”。
首先，在ICQ好友名单上用鼠左键单击你要传送短消息的朋友，在弹出的功能选单上选择“SMS Message”。按一下“Add SMS Number”来加入手机号码到ICQ清单。
然后，选择好地区之后(如香港网友选择“Hong Kong”，而中国台湾网友选择“Taiwan”)，在Mobile Numbe字段填上对方手机号码(Carrier Prefix字段可不填)。
最后，用户按一下“Send SMS Message”窗口上的“To”按钮，会出现一个清单的窗口。按“Send To”可以把左边的名单加到右边的准备送出的清单上，而按“Remove”可以移除不想传送的对象。
当短消息送出时，会显示己传送的讯息窗口，使用者按一下“Done”就完成了！短消息可以传递160个字符，如想在发送的同时寄一份相同的ICQ信息，可选择“Send copx bx ICQ Message”。
理论上说，该功能在全球各国各地区提供短消息服务(Short Message Service)的电信系统的手机上都可以实现。不过，由于目前ICQ的该项服务仍处于测试阶段，有时会有不能接收的现象。经过我们验证，目前中国大陆地区只有中国移动能够正常使用，要向中国台湾地区的好友发短消息须在其手机号码前加“0”。还有，ICQ2000b的该项功能目前只支持英文传输。
怎么样，很简单实用吧？而且，还可以节约一大笔国际长途费用哦，大家快去试试吧！
}
