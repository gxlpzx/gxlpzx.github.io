{万象幻境真能保万无一失吗？——对付万象幻境怪计两则}

{ 看了上期《网管类软件真的不安全吗？》一文，我的心情难以平静。看来像我这些以对付网管软件漏洞为生的菜鸟们，今后将以领取救济而维持生计。出于生计，小弟特意精心制作了对付包括万象幻境在内的网管软件的怪计两则，让大家一起来探讨探讨。
怪计一：金蝉脱壳
万象幻境有一招防卫手段，就是利用程序ClientSetup将F4、F5、F8键屏蔽掉从而不能进入MS－DOS模式，安全模式等。而且能够禁止运行DOS模式，但没有禁止运行安全模式，所以我们就进入安全模式好了。虽然F4、F5、F8键被屏蔽掉了，但别忘了进入安全模式另有它法。相信大家也很清楚，那就是利用Shift键。方法很简单：开机后按Shift键不放，在没有任何提示语情况下，系统直接切换到安全模式，剩下的事那就自己看着办吧。
刚才所说的还不算，作为开胃菜，吊吊胃口还可以。下面的这招怪计则是本文的中心。
怪计二：瞒天过海
记得渝中高人有话曰：只要把你的系统所在分区前的小钩去掉，这样无论用什么方法都看不到C区，以达到直路走不通，弯路也走不通的目的。也记得鲁迅先生一句名言：天下本没有路，走的人多了，也便成了路。还记得：有条件要上，没有条件创造条件也要上。看来我们今后在某些时候不能去走别人走过的路，所以我们要自己创造条件，另辟蹊径，来达到我们的目的。在这里我们要利用一下我们都非常熟悉的压缩软件——Winzip。
1.前期准备
首先要新建立一个空白的ZIP文件也可以加载任意文件容量越小越好然后通过E－mail的方式将它发送到自己的邮箱中邮箱最好是在别的网站申请的备用。这时我们自己铺的路就好了。
2.实战演练
使用时先将其从邮箱中找到，由于该ZIP文件所占的空间小，仅1K大小，所以在使用的时候，不管你用的电脑速度有多慢，都能很迅速地下载。在下载完毕后，任意找一该机的ZIP文件，再进行解压缩。待解压完毕后，这时就自动弹出一……哈哈哈，我不说你也该知道了吧？现在所在的位置是C:\unzipped文件夹下，然后就可以……
从另一方面，我们高兴地看到网管类软件是越来越完善，再也不是以前的鸡肋形象了。这也正是我们所希望看到的。毕竟我们都希望有一个安定的环境。老实说，万象幻境这款软件的确是不错，是我所见过的最好的。以上小弟的愚见，不论对错，一样恭候各位高人的指点。
}
