{手机电池的选购与使用}

{ 1.手机电池的类型
移动电话电池主要有三种类型：镍镉电池、镍氢电池和锂电池。镍镉电池因为缺点太多，已经逐渐淘汰不再使用。镍氢电池因为不含有镉金属，不会污染环境，又被称为“环保电池”。镍氢电池的电量储备比镍镉电池多30％～50％，价格较便宜，几乎没有记忆效应，应用较广泛。
锂电池是一种高能量的智能电池，是目前性能最好的常用电池，具有电量储备大、重量轻、寿命长、充电时间短及无记忆效应等许多优点。锂电池的缺点是价格较贵，对充电器也有选择性。目前锂电池已经成为手机电池的主流产品。
2.手机电池的选购
人们在购买电池时最常问的是“待机时间多少”、“通话时间多少”。其实，目前厂家给出的待机时间和通话时间基本上是一个估算值，仅能提供参考。正确评价电池的标准是“毫安时”，即电容量mhA。常见的电池容量有570、650、900和1100毫安时，现在随着手机型号的丰富，非标准的容量也很多。一般来说，电池容量越高，电池越重，使用时间越长。不过各厂家的技术标准不同，同样容量的电池实际使用时间并不一样。
电池的使用寿命是由电池的循环周期决定的，电池充放电一次为一个循环周期。优质的电池循环周期一般为500～1000次。电池每充放一次，电量都会有所衰减。一般新电池使用时需要3～5次的长时间充放循环，才能达到预期的效果。
在选购电池时，应该到有信誉保证的商家，选择原装的电池。如果你觉得原装电池太贵，可以留意消委会和技术监督局的公告，选择有ISO、CE认证、长城电工认证或消费者协会推荐品牌的产品，而不要相信太便宜的东西。
3.充电器的类型与选购
目前移动电话充电器主要有旅行充电器和座式充电器，分别简称“旅充”和“座充”。旅充一般采用快速充电方式，充电时间为2～3小时。座充一般设计成双槽，一次能充两节电池，一般采取涓流充电方式充电，充电时间较旅充要长。旅充和座充对电池充电的效果是一样的，旅充对手机不会有任何不良影响。
和电池一样，充电器应该购买原装或有UL、CE或TUV认证标志的充电器，否则会由于充电器的充电效果不佳而不能充足电池，甚至充坏电池和烧坏手机主板。
4.如何延长手机电池使用时间
怎样才能保证手机电池的寿命和充电后的使用时间呢？
首先，由于电池特性各异，应当依照说明书指示的方法充电。一般新电池最初两三次应至少充电12～14小时。日后电池充电时间依类型和容量不同，大约为1.5～6小时，切勿超过24小时。智能型快速充电器在充满90％电量的时候，就会转变指示信号灯，然后改用慢速充电将电池充满。你最好耐心将电池完全充满后使用，否则会缩短电池的使用时间。
要延长手机电池的使用时间，首先应使用GSM标准具有的“DTX”非连续性发射省电模式(即现在很多新手机提供的“省电模式”)。实验证明，DTX省电模式最多可以延长通话时间的30％～50％。其次，当网络信号不存在或极其微弱时，应关闭手机，并尽量选择在网络信号强的地方再通话。第三，手机显示屏是耗电的大户，因此应多在明亮处使用手机，条件允许的话可以关闭显示屏或按键的照明。最后，尽量在宁静的场所选择使用短电话铃声，这样既省电又减少了对环境的干扰。 }
