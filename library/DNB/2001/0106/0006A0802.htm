{网络时代的“大卫”和“巨人”}

{ 网络世界总在不断地上演着大公司吞并小企业的活剧，我们有理由担心：长此以往，网络会不会变成一个贫富更不公平的世界？其实古人早已为我们讲过了这方面的故事。
圣经中就有个很出名的故事，说有个可怕的巨人，不但力大无穷而且贪得无厌，严重威胁着人们的生命安全。后来，有个叫大卫的孩子用智慧和小小的石头打败了巨人，这个聪明的孩子不但成了英雄，而且他长大后，还成了在圣经中流传千古的大卫王。
资本主义兴起后，许多企业都发展成拥有数十万员工的庞大帝国，这些企业就像是自成体系的王国，有自己的企业文化、经营模式和价值观念，深刻地影响并决定了一时一地百姓的工作和生活。这甚至引起了社会结构的转型和变动，例如：学校有了统一的上课时间和课程标准，大家也不知不觉地共同过起了同样规律的朝九晚五的上班式生活。
庞大的企业就如同圣经中的巨人。在过去的工业时代，我们很少听说有人能用自己的力量去抵抗巨人，因此，近代发生过许多劳资双方因利益冲突而产生的严重对抗和流血事件。而保护工人的组织如工会等也逐渐出现，并成为社会不可或缺的重要组织。
随着时代的进步，现在的企业版图不再局限于一定的范围，现代化企业的经营模式在网络科技的帮助下走出了国家和经济体的界限，向全世界的每一个角落发展。无论从企业自身的规模还是经营范围来看，进入网络时代后的企业，更像是圣经中那个可怕的巨人，营业额动辄上百亿美元，员工随便就有几千上万人。这样的企业发展，往往令人担心将来的市场化世界，会比工业时代的弱肉强食更可怕。
然而，我们不要忘了，在以数字科技为基础的世界里，还有很多的“大卫”，这些“大卫”如开创时期的微软、网络开始流行时的Netscape、Yahoo，如MP3，如PDA中的palm，如操作系统中的Linux)都是在极小的规模中，用一个简单而实用的技术，轻易就占据了巨人企业所占领的版图，这样的实例比比皆是。这些“大卫”成功的故事，使我们更应该相信，网络时代绝对是更能发挥个人才能的理想化时代，在这样的时代中，没有任何人和任何企业可以压制或阻碍每一个有智慧的人新颖的创意。从人类整体发展的角度来说，这样的网络时代当然是一个最好的发展时代。
}
