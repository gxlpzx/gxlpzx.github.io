{让你更靓丽_图片处理一般技术(上)}

{ 在一般家庭的图片处理中，最常用的莫过于将图片进行扫描，然后校正色彩、修剪、做一些特技效果，最后打印或发送到网上或用邮件发给亲戚朋友等步骤。看似一件很简单的事但实际上图片处理要有很专业的技巧，是一件非常细致的工作，每一步都要小心行事。如果是重要的图片一定要请教专业的人员，因为如果哪步处理不当而没有及时调整过来，可能导致整张图片报废。要把这一流水线的作业过程搞清楚，或者要掌握高明的技巧，还真不是一件容易的事呢。下面我们用一个实例和图片处理基本常识两部分来阐述这一过程。如果对实例中有不明白的或对自己有更高的要求，不妨对图片处理的基本常识和技巧多做研究。
一、认识扫描仪
一般图片的来源多是从扫描仪输入来的，因此扫描质量的好坏是决定图片后期处理的关键，若是扫描时丢失了大量的色彩信息，后期补正是很难再将图片完整地再现。
扫描就是将模拟的图像信息(如照片、反转片)转换为计算机能够识别的数字信息(电子图像)，它是图像处理的最重要的一步。扫描仪分平板扫描仪和滚筒扫描仪。滚筒扫描仪是高档的扫描仪，不是一般用户能掌握的。由于平板扫描仪有操作简便、快捷，为一般用户所采用。故我们就以平板扫描仪为例。平板扫描仪分很多种，有高、中、低档的。我们就以Acer明基电脑的PRISA 620P扫描仪为例，对其使用进行讲解关于扫描仪的选择可参见第5期的电脑报E8版。
二.开始扫描
由于我们讲的是图片处理，所以当然要用到Photoshop了。
1.选取要扫描的图片(我们选用一张漂亮MM的照片)，放入扫描仪中。然后打开扫描仪的开关。
2.启动Photoshop 6.0，进入到Photoshop的画面。点击(File文件/Import导入/MiraScan V3.20扫描仪)，启动画面之后就进入扫描仪的控制面板。
在命令菜单栏中PRISA 620P只有简单的三项，文件、设置及帮助。这部分一般少做变动，只作简单介绍。
文件：提供你处理配置文件的各种方式，例如“装入配置文件、储存配置文件、另存配置文件、重设配置文件以及Auto Save Config(自动储备设置文件等)。
设置：可让你选择扫描尺寸、显示器Gamma值、底片类型以及自动预览等功能。
帮助：内容包含了扫描仪的在线信息、内容及有关扫描仪的相关信息。
在调整扫描设置选区和扫描仪参数设置这两部分可要注意了，需要一些专业知识。这要视我们图片的要求而定，如做印刷用，那质量要求就高一些，如做电子图片那要求就低一点。这里我们以印刷为例，因为印刷的质量远远高于电子图像的要求。
3.点击预览，(预览是指扫描仪会以较低的分辨率快速扫描原稿)然后会在预览视窗中显示预览图像。
4.将虚线选取框选取我们要扫描的范围
调整扫描设置选区的设置
如图1左边画圈的部分，我们选用彩色模式，由于是印刷，故选择高质量。分辨率设置为350dpi，一般印刷上分辨率设在300～400dpi较为合适，电子图片分辨率设在72～150dpi就够了。图片的缩放设置为100％(原大)。滤镜选择无滤镜。去网纹选择艺术杂志(175 lpi)
扫描仪参数设置
如图1右边画黑圈的部分，这部分除了对扫描图片进行选取、移动，放大显示、镜像、旋转等功能外，更重要的是对图片本身效果设置。这要看我们扫描图片的本身质量如何，如果图片变暗，那么我们就把明亮度、对比度调亮一点。如果图片的网纹大，我们就减去些网纹。
色阶：显示图像中的色阶
曲线：调整此曲线可以改变图像效果
色彩平衡：用以改变图像中颜色的混合、色调或减少某种色彩。
色彩调整：调整扫描图像的色相、饱和度以及亮度。
明亮度/对比：点击此按钮来调整图像的明亮度以及对比度。
5.当一切都调整好后点击扫描，然后就开始扫描，十几秒后扫描就完成了。
好，现在扫描结束了，但不是说万事大吉了。一般家庭和个人的扫描仪都是中低档以下的。扫描仪即使全力以赴尽了犬马之劳，图片还是免不了损失很大，所以在Photoshop中还要进一步对图片进行后期处理。
三、照片后期处理
以扫进来MM的图片为例。在扫描时我们虽然作了最大的努力，但由于扫描仪的限制，扫进来的图片还是有许多的不足。
(1)图片的网点较大(2)色彩的饱和度欠佳(3)由于照片的原因面部五官显得模糊(4)头发偏白显得发质不是很好等。今天我们主要是以五官为准来进行调节。
6.确定图片为CMYK模式(打印模式)，(点击Image图像/Mode模式/CMYK Color)检查尺寸正确与否。
7.去网点：一般在中等档次以下的扫描仪扫描出来的图片都有这个缺点。扫描出来的图片都要经过处理。不过这一步是有限的，不可能完全、彻底地去掉。这一步要慎重。
点击Filter滤镜/Noise杂色/Despeckle去网点，如这一步的作用不大，可以继续执行Filter滤镜/Noise杂色/Dust＆Scratches粉尘和涂抹。
Radius半径为1，Threshoid阈值15。Radius半径值越大图像越模糊，故不能太大。Threshoid阈值使图像尖锐，也不宜设置过大。我们这儿的设置如图。然后点击OK。
8.调整照片的饱和度：这需要一定的色彩知识(在下期将对图片调整的基本常识作一定的介绍)，例如黄种人肤色以黄、红色成分为主，一般黄色成分高于品红，但也不完全是这样。就这张图片而言，由于是少女的皮肤偏白，所以红色不要太多。打开通道面板查看M、Y通道。黄和红的饱和度明显的不够饱和，接下的工作就是需要提高黄版和红版的饱和度。
在上期我们主要讲了扫描及图片处理的一些基本常识，在这一期我们将重点讲对图片的调整及Photoshop中一些常用的调节工具，在此我们还是将实例和理论分开讲，如果对实例看起来有点困难的，则可以先看看右边的几种调节工具的使用方法。
9.回到图像面板中，点击Image图像/Adjust调整/Ｈｕｅ/Saturation 色像饱和度。
首先我们选取黄版，然后将当中的滑杆)Saturation饱和度)调到２６左右，如图６画圈的部分。我们可以看到图片上的黄色饱和。用同样的方法选取红版，根据需要提高红色的饱和度。要注意的是，黄色人种肤色当中的红色一般比黄少些，故红版的饱和度调到１８左右就行了。当然这要视不同的图片而定。
10.通过第９步的调节，有可能会使整个面部的肤色偏黄，一般少女的皮肤是偏白的。接下来我们在上一步的基础上做进一步的调整。
点击Image图像/Adjust调整/Curves曲线调节工具。我们还是先调整黄版，在Channel通道中，选Yellow黄色，调节的曲线如图７。
通过９和１０两步的调节，使颜色看上去好多了。在Photoshop中的Image图像/Adjust调整下面有很多种调整方法，而且每一种方法都有一定的长处。在右边的“Photoshop中几种常用调整功能面板介绍”中作了一定的介绍。这里要说的一点是，图片的情况是多样的，调节的方式也不同。除了在一定专业知识上多作了解外，还要对图片的情况多作分析。

四．图片的打印
当图片处理完，最后到了出成果的一步了。这一步主要是看打印机的档次了。当然是要选好一点的相片纸了。在打印之前要检查几点：)１)图片的尺寸。)２)图片的模式，)３)纸张是否正面放置。确定好后就可打印了。点击Ｆｉｌｅ文件/Ｐｒｉｎｔ打印，接着就出现打印设置，由于打印机不一样，设置也就不同了。不过你要明白，前面我们做了这么大的努力，当然是要打印精细啦。一般打印机分辨率设置在７２０ｄｐｉ以上就够了。
编后：此实例只代表图片处理中的很小的一方面。专业的图片处理不在此范围之内。一般图片的情况是多样的，处理的时候，要根据当时图片的情况来考虑用什么样的调整方法。有关这方面的知识还有很多，是一个长期的过程。此例目的是给大家引一个路。有句话：“师傅领进门，修行在各人。”好！后面就看自己的了。
Photoshop中几种常用调整功能面板介绍
在Photoshop中“Image图像/Adjust调整”选项下有多种颜色调节工具)如图１４)，下面介绍几种主要的调节工具。这些工具都具有自己的特色，不能说谁强谁弱，谁好谁坏。如是图像处理专业人员需要对调节工具作全面深入的了解，如是一般的用户，也应了解其中一些常用的调节工具。
Ｌｅｖｅｌｓ色阶调节
该工具的最大特点是带有自动分析阶调分布的功能)如图１５)。
Ｉｎｐｕｔ Ｌｅｖｅｌｓ)输入色阶)：它是指在当前通道下图像的最暗处、最亮处及中间调所对应的值，方框中的三个数据与阶调分布图下的三个滑块相对应，同时变化。例如我们改变当中的值，变化主要在图片的中间调。
Ｏｕｔｐｕｔ Ｌｅｖｅｌｓ)输出色阶)：用于改变图像的最暗值与最亮值，与下面的滑块相对应。
这部分根据需要可以滑动滑块来改变图像，拿不稳的可以点击Ａｕｔｏ自动，电脑将自动用于图像的调节。右下角的三个吸管按顺序作用是定义白场、中间调和暗场的数据并设定图像的白场、中间调和暗场。也就是你用三个吸管吸取图像中最亮的，中间调和最暗的颜色，你根据图片的需要作一定的调整后，数据将自动用于图像阶调的调节。
Curves曲线工具
曲线工具有极强的调节功能，对原稿阶调的修正、校正偏色等均有较完善的调节能力)如图１６)。
方格中的斜线为曲线调节控制线，当鼠标点住斜线后拖动就会产生一个节点，也就是调节曲线的控制点。在曲线工具图上按住Ａｌｔ键，同时在网格底纹中点击，数据分割线会由原来的３条增加到９条，每一格表示１０％的网点，这样调节图像会更精细准确。
方框左边的渐变条表示调节后图像输出值的明暗轴线，下面的渐变条表示亮暗调切换按钮，和方框左边的渐变条对应。
Ｉｎｐｕｔ输入：调节前的图像数据。Ｏｕｔｐｕｔ输出调节后所得的数据。例如我们在输入填入２４％、输出填入３４％，这样将表示原图亮度为２４％的颜色值适深至３４％。它和方框的斜线对应变化，这种调节实用于亮调偏浅，需要加深处理的原稿。Ｉｎｐｕｔ输入和Ｏｕｔｐｕｔ输出对应右边的曲线和铅笔是平滑曲线和任意曲线的切换按键，平滑曲线使阶调变化连续，任意曲线可使阶调变化突然。
值得一提的是曲线调节无论是在单色通道的调节，还是整体明暗色调和提高饱和度、降低曝光过量等方面都有突出的能力，通过任意改变平滑曲线和任意曲线，改变图像阶调分布情况，会出现颜色数据的跳跃变化，往往会产生出光怪陆离的奇妙景象，常常用于特技效果之中。建议大家尽量采用曲线工具进行调节。
Ｈｕｅ/Saturation色相/饱和度
该工具可以用来改变图像的色彩组成、颜色的饱和度及图像的亮度值)如图１７)。
Ｅｄｉｔ编辑：要求你选取读者操作的范围。你可选取全图，也可选取里面的某种颜色，如红色、绿色或者是蓝色……当选中某一颜色时只对当前颜色有效，在最底层的彩色标杆中会出现限制区域的三角形，表示变化均在此范围之中。
三个滑杆分别表示Ｈｕｅ色相、Saturation饱和度、Ｌｉｇｈｔｎｅｓｓ明度。
Ｈｕｅ色相：色相值在－１８０到１８０之间变化。当移动三角形滑块时，方框中的数值随之变动，同时图中最底下的彩色滑杆的颜色也在翻滚变化，上下两根滑杆的颜色形成一组对应值，表示改变色相后，将上一根的颜色改变为下一根同一位置上的颜色，当色相值为＋１８０或－１８０时，图像中颜色全用反色取代。
Saturation饱和度：用来改变图像颜色的饱和度。
Ｌｉｇｈｔｎｅｓｓ明度：用来改变图像的明暗关系。
右下角的Ｃｏｌｏｒｉｚｅ着色：作用为整个图片定义一个基本色调，下面的调节都是在此基色的基础上进行。最下面的两根彩色条表示颜色的变化情况。
Ｆｕｚｚｉｎｅｓｓ颜色容差：用于将当前色周围的多大区域设定为选区，值越大，选中的区域就越大。下面的方框为预显示框。
Ｓｅｌｅｃｔｉｏｎ选区、Image图像：用来设定预显示是以选区出现，还是以图像出现。
三个滑杆分别为Ｈｕｅ色相、Saturation饱和度、Ｌｉｇｈｔｎｅｓｓ明度：它们的作用类似Ｈｕｅ/Saturation色相/饱和度的调节。
Ｓａｍｐｌｅ取样：作用为于当前采样的颜色值，位于色相滑杆的以０表示。别的颜色以此为基准，顺序变化)这点与色相/饱和度工具不同)
三个吸管作用为加减选取用的颜色吸管，可用带加号的吸管追加选区；用带减号的吸管减去不需要的颜色区域。
Ｓｅｌｅｃｔｉｖｅ Ｃｏｌｏｒ可选颜色工具
该工具常用于调整图像中专色块的饱和度)如图１８)。
Ｃｏｌｏｒｓ颜色：要求读者选择对象颜色进行调节，可供选择的有红色、黄色、蓝色、洋红色；另外还有白色、中性色及黑色三种色块。
四根滑杆分别代表Ｃｙａｎ青色、Ｍａｇｅｎｔａ洋红、Yellow黄色和Ｂｌａｃｋ黑色，用于调节当前所选颜色的四色组成。
Ｍｅｔｈｏｄ方法：有Ｒｅｌａｔｉｖｅ 相对、Ａｂｓｏｌｕｔｅ绝对两种。相对是在原基础上加或减；绝对是直接往上添加，而不考虑原先数据。

图片调整的基本常识
在对图片进行调整时我们采用印刷品为标准，因为印刷是指以忠实原稿或高于原稿为原则。印刷一般是采用CMYK值(青、品红、黄、黑)。或者我们可以说是四种油墨，用一定的方法涂抹于纸张等承载体上，形成千差万别的图文信息。四色油墨各自的颜色表现、颜色的纯度等，都将直接关系到印刷品图像的色彩。特别是青、品红、黄三色油墨各偏向何种颜色，定量是多少更为重要。要掌握好这几种油墨的偏向首先要掌握好“灰平衡”的原理。灰平衡是印刷图像中一切颜色存在的基础，没有灰平衡，就没有图像中五颜六色。无论何种颜色都是在灰平衡数据的基准上变化而来的。我们在进行扫描时首先要考虑到印刷油墨的灰平衡。这是取得高质量印刷品的保证。
灰平衡的基本概念
灰平衡的概念来源于四色印刷油墨的偏色现象，其本质也在于此。灰平衡就是指在一定的印刷、打样条件下，将青、品红、黄三色油墨按一定比例叠印，得到视觉上中性灰的颜色，这时就称为实现了灰平衡。将该点的青、品红、黄的网点百分数据，称为该点的灰平衡数据。可见灰平衡数据不是一个，而是一组、一系列数据。
一组灰平衡数据有很多，而且影响它的因素也很多，如油墨的品牌、编号、纸张、印刷机等打样印刷的条件。要完全弄明白，读者也许会很头痛。其实只要了解灰平衡数据一般规律就不难了，比如在中间调附近，青的数据比品红、黄大7％—9％了，亮调处则大2％—4％等。一个简单的方法，记住C40、M33、Y34为标准的灰色。若有人问你C42、M25、Y33为什么颜色？若不了解灰阶数据，除非是非常熟练的老手能够说出相近的颜色，否则你是无法判断出这个颜色。用上灰阶平衡，问题就变得简单了。和灰阶数据一比较，C、Y的值相差不多，M的值比灰阶数据少8，那我们知道M为品红，品红的补色为绿色，缺品红就等于加大绿的比例，所以该色块为绿加灰色，则为灰绿色。其他的道理一样，只要有一定的色彩常识，再对灰阶平衡有一定的了解，只要知道CMY值颜色是不难判断的。在我们调整图片的颜色时心里就有底了。
处理照片的注意事项
我们就拿人物图片来说。人像原稿往往要求准确自然地还原真实情况。它的种类很多，不同肤色、不同年龄和性别都有很大的不同。我们应该大致掌握一些规律。比如黄种人肤色以黄红色成分为主，一般黄色成分高于品红，青为品红的一半以下，青版为层次版，黑版为暗调的骨架版，而黑种人肤色中的黑色与青色较重，黑版与青版的阶调较长较全。白种人肤色中的品红成分偏重，青版起到了层次作用。 }
