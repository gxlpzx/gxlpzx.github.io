{Dreamweaver时间线例解}

{ 打开一个网页，一个动画卡通(*.GIF)在网页上到处游动，很有点意思。这是在Dreamweaver中用时间线来制作的。具体的制作过程如下：
1.在网页一个适当的地方，比如在左上角拉一个30×30的层(可以更小)，插入一个你喜欢的小小的卡通动画，这样的动画，到处都可以得到，当然也可以自己制作一个。
2.选中这个层，注意不是选中这个小动画。按“Ctrl＋F9”，弹出时间线设置框，在第1帧处按右键，在弹出的对话框里选“Add Object”，这个动画的层便被添加到动画条上，动画条上显示Laxer x 字样，很滑稽的是如果此前你没有插入图片，是一个空的层，结果也一样，时间线接纳了这个空层，动画条上同样显示Laxer x 字样。换言之，时间线只移动“层”，不管层里有什么东西。
3.我们可以任意设总帧数，但总帧数多了文件就太大，实在也没有必要。这里我们设总帧数为45，点按动画条的45帧，把动画层拉到结束的位置，我们看见在起始和结束位置间多了一根位移线，于是我们设置了起始帧和结束帧。
4.按住“Ctrl”同时点按第15帧，15帧时间条上多了一个小圆圈，也就是说，我们在这里设了一个关键帧，我们把小动画拉动到我们在网页中想要的位置，比如向右挪动到最右边，看见了吧，位移线也跟着弯了一个弯，同样的方法，我们在第30帧处也设一个关键帧，拖到最左边。同样，我们也不要把路径设得太复杂，一两个弯就可以了，否则，你打开源代码看一看就知道了。然后在时间线设置框中点选Autoplax和Loop，这样我们指定页面载入后自动播放动画和不断循环地播放动画。好了，基本制作就完成了，按F12浏览一下，嘿，还真成功了。我们的卡通“小美人”，带着我们的友谊和祝福，从网页的左上角，走到最右面，辗转到最左边，然后来到最后结束的位置。
5.如果想让动画跑得慢点，可以增加总帧数，只要按住时间线最右边的关键帧，向右拖到适当的位置，比如90帧。不要去变动“Fps”后面的速度设置，默认是15 Fps，降低这个设置，动画的移动会产生明显的不连贯。同样，你想使各个区间的速度不相同，那么就移动相关的关键帧。两个关键帧间的距离被拉长，速度就变慢。比如我们用鼠标按住15帧向右拖，那么从开始到第一个点之间，卡通会变慢。
}
