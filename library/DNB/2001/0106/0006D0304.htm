{让网页自动弹出下载任务栏}

{ 经常下载软件的朋友，可能会注意到我们有时到一些网站下载软件时，在下载软件的页面中会提示你“等待x秒钟后自动下载”。果然，在x秒后，如果你的电脑中装有诸如“网络蚂蚁”或“FlashGet”等下载工具的话，它会自动弹出下载任务栏，并在下载URL地址栏中填入相应的下载网址。你只须点击“确定”按钮，就可以下载软件了。如果将其运用到我们的网页上，可以使浏览者操作更为方便。
实现这一功能其实很简单，你只需在网页中的<head>与</head>之间加入以下代码：
<META HTTP－EQUIV＝″REFRESH″ CONTENT＝″5url＝http://go.163.com/supersite/index.zip″>
在代码行中，“5”表示延迟的秒数，你可根据需要修改。“http://go.163.com/supersite/index.zip”表示下载软件的网址。
它的原理其实也很简单，即在打开网页等待几秒钟后再转至其他URL网址，如果URL网址后缀名为.zip、.exe、.mp3等属于下载工具所监视的文件类型，下载工具会自动弹出下载任务栏开始下载，这与你在网页上直接点击下载超链接所产生的的效果是一样的。
}
