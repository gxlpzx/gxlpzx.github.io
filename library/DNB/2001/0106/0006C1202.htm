{ 昂达10× DVD－ROM}

{ 昂达10倍速DVD－ROM操作面板较为简洁，面板各种插孔和按键与普通的CD－ROM相近，中间明确地刻着DVD标志，在DVD－ROM后部还带有模拟和数字音源输出接口，适用于不同音源信号的输出。它采用E－IDE接口，支持Ultra DMA33传输模式，支持8cm或12cm等多种规格的盘片。还内建512KB的高速缓存，保证高速数据传输率。
在技术特性上，DVD－ROM内部使用全钢机芯，并使用了DDSS平衡抗震系统，可保证维持DVD高速运转时的平衡、顺畅，进而达到避震减噪的效果，有效地克服了读盘时的震荡现象。同时该DVD－ROM还内设无电自动退盘系统，能够确保DVD－ROM在意外断电情况下仍然可安全退出盘片，保护光驱及盘片不受损坏。昂达10× DVD－ROM还特别设计为不锁区域码，可与各种格式的光盘兼容，支持全区码影碟的回放。
为了测试昂达10× DVD－ROM性能，我们使用了CD Winbench 99对其进行测试。从测试结果来看，这款10× DVD－ROM表现正常，它的DVD－ROM传输速率内圈速度达到6300KB/s，外圈最高传输率为11300KB/s，相当9.4×(1× DVD－ROM数据传输率为1200KB/s)，基本达到标称的10×，而CD－ROM传输的表现似乎更强，外圈最高数据传输率达到4870KB/s，超过标称的32× CD－ROM标准。在测试过程中，这款DVD－ROM比较安静，与一些DVD－ROM相比，盘片运转产生的噪声抑制处理较好。我们还测试了这款DVD－ROM的影碟回放功能，使用PowerDVD3.0实现的软解压播放，影碟画面快进快退播放平稳，对不同区码的影碟进行的识别测试，也非常成功。
昂达10× DVD－ROM采用的众多领先技术，具有较快的数据传输率和较强的读盘能力，而且还有不锁区码的优势，整体性能表现非常不错，加上昂达公司提供一年保换的可靠服务保障，相信它会让用户觉得物有所值。
产品资料
产品名称：昂达10×DVD－ROM
接口类型： ATAPI/E－IDE
持续数据传输率：10×DVD－ROM/32× CD－ROM
突发数据传输率：16.6M/秒(PIO模式4)、16.6M/秒(DMA模式2)
缓存：512KB
重量：1.1Kg
价格：766元}
