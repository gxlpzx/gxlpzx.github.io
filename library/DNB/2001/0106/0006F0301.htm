{日常维护——数据备份与自动维护篇}

{ 备份硬盘分区表、引导区和主引导扇区，防患于未然
大家都知道，分区表是硬盘的“命根子”，它受到破坏之后整个硬盘就会瘫痪，而分区表恰恰又是非常脆弱的，除刚才讲到的病毒以外，停电、死机以及误操作等都有可能破坏硬盘分区表，这就要求我们未雨绸缪，事先将硬盘分区表、引导区和主引导扇区等信息备份下来，当硬盘出现故障时可以轻松恢复。不过令人遗憾的是，Windows 2000并没有提供备份硬盘分区表、引导区和主引导扇区的功能，我们只有使用第三方的应用软件(如Nuts ＆ Bolts、Norton等)来保护数据的安全。
Nuts ＆ Bolts是一个与Norton齐名的磁盘工具软件，它提供了强大的应急盘保护功能，可对CMOS、分区表、引导区和主引导扇区等众多系统信息进行备份，并在应急盘上提供了许多对磁盘进行检测、整理的工具软件(这些软件都是DOS版的，我们可在使用应急盘启动系统之后直接加以使用，这就比不提供DOS版磁盘维护工具的Norton应急盘要好得多)，实为广大用户备份系统信息时的最佳选择。
要使用Nuts ＆ Bolts对硬盘信息进行备份，我们只需启动Nuts ＆ Bolts，然后在它的综合调度窗口中单击“Rescue Disk”按钮，启动系统的备份功能，然后再按照备份盘制作向导的提示进行操作即可。操作完毕之后，我们将会得到一张包含硬盘引导信息的Nuts ＆ Bolts应急盘，此后当硬盘的分区表、引导区或主引导扇区等信息被破坏之后，我们就可以使用该软盘启动计算机，并执行“RECOVER”命令，利用备份信息对系统进行恢复即可。另外，我们也可执行应急盘中的“RDDOS”命令，让Nuts ＆ Bolts自动对硬盘分区表进行恢复，并进行磁盘扫描、病毒检测等全套维护，使用非常方便。
假如我们还没有来得及备份硬盘信息，硬盘分区表已经被破坏，这时我们也不要坐以待毙，FDISK、KV3000等软件都提供了测试、修复和重建分区表的功能，我们只需执行“FDISK/MBR”命令，或启动KV3000并进入其图形界面之后单击F10按钮即可达到重建分区表，恢复硬盘数据的目的。
提示：由于不同硬盘可能会采用多种不同的分区方式，因此分区表非常复杂，任何软件的测试、修复和重建分区表功能都不是万无一失的，对诸如设置了多个引导分区之类的特殊硬盘往往无效，因此采用适当方法事先对硬盘信息进行备份才是我们的最佳选择。
对硬盘数据进行备份，保护数据安全
尽管我们可以采用前述的方法对硬盘进行适当维护，尽可能减少数据的丢失，但谁也无法避免意外的发生，及时对硬盘上的数据进行备份也就成为了我们必备措施之一；另一方面来说，我们在重装系统等操作之前也需要将用户自己的信息备份下来，这些都要求我们及时对系统进行备份。至于如何备份，不同用户会采用不同的方法，如：有的用户可能会使用Ghost将整个硬盘全部备份下来，有的用户则会使用WinZip对自己的数据文件进行压缩备份，还有的用户可能会采用拷贝的方法直接将自己的数据文件备份到软盘上，可谓“八仙过海，各显神通”！不过对于一个初学者来说，使用Windows 2000自带的“备份”程序应该算是我们最简单而可靠的选择。
利用Windows 2000的备份工具，我们可以备份硬盘上选定的文件和文件夹并可在需要的时候快速进行恢复，同时它还提供了详尽的向导功能，可一步步指导用户对数据进行备份，因此非常适合初学者使用。
使用Windows 2000备份程序的向导功能备份数据的步骤为：
1.单击“开始”按钮，然后依次选择“程序”→“附件”→“系统工具”→“备份”，启动Windows 2000的备份程序。
2.此时系统将会给出一个欢迎窗口，询问用户准备从事何种操作(主要包括“备份向导”、“还原向导”和“紧急修复磁盘”三个选项)，我们应从中选择“备份向导”按钮，启动系统的备份向导。
3.单击“下一步”按钮，从弹出的“要备份的内容”列表框中选择需要备份的对象。主要包括“备份整个Win2000系统”、“备份选定的文件、驱动器或网络数据”、“只备份系统状态数据”三种对象，我们可根据自己的需要加以选择。
4.单击“下一步”按钮，打开备份位置选择窗口，我们可根据自己的实际情况选择需要备份的目标磁盘及备份文件名。
5.单击“下一步”按钮，系统将打开“完成备份向导”对话框，将用户选择的备份项目列表显示出来，我们只需在检查无误的情况下单击“完成”按钮，系统即会开始进行备份。
使用Windows 2000备份程序的向导功能还原数据的步骤为：
1.启动Windows 2000的备份程序。
2.在系统给出的欢迎窗口中单击“还原向导”按钮，启动系统的还原向导。
3.在弹出的还原项目列表中选择需要还原的类型、驱动器及文件等内容。
4.单击“下一步”按钮，打开还原位置选择窗口，我们只需选择适当的还原位置即可。
5.单击“下一步”按钮，系统将打开“完成还原向导”对话框，将用户选择的还原项目列表显示出来，我们只需在检查无误的情况下单击“完成”按钮，系统即会开始进行还原。
利用任务计划程序自动对系统进行维护
前面所介绍的那些维护方法都不是一次性的，我们只有经常重复上述操作才能保证硬盘的安全运行，如果完全依靠手工进行当然也可以实现，但操作上似乎过于繁琐了一点！别着急，利用系统的任务计划程序(Windows 98称之为计划任务)可轻易解决这一问题。
所谓任务计划就是由计算机自动调用用户事先设置好的应用程序，从而达到简化用户操作的目的。利用Windows 2000的任务计划程序，我们可以安排任何脚本、程序或文档在最恰当的时候运行，从而满足自己的需要。
具体来说，我们若需利用任务计划程序自动对磁盘进行维护则应执行如下步骤：
1.单击“开始”按钮，然后依次选择“程序”→“附件”→“系统工具”→“任务计划”，启动Windows 2000的任务计划管理程序。
2.在“任务计划”窗口中双击“添加任务计划”图标，启动系统的“任务计划向导”。
3.在给出的程序列表中选择需要自动运行的应用程序，然后单击“下一步”按钮。
设置适当的任务计划名称并选择自动执行这个任务的时间频率(如每天、每星期、每月、一次性、每次启动计算机时、每次登录时等)，然后单击“下一步”按钮。
此时系统将会要求用户对程序运行的具体时间进行设置，如几号、几点钟、哪几个时间段才能运行等，我们只需根据自己的需要加以设置即可。
4.接下来系统将会要求用户设置适当的用户名及密码，以便系统今后能自动加以运行。
5.最后，我们只需单击“完成”按钮即可将相应任务添加到Windows 2000的任务计划程序中，此后它就会自动“记住”这个任务，一旦系统时间及相关条件与用户设置的计划相符，它就会自动调用用户所指定的应用程序，十分方便(每次启动Windows 2000的时候，任务计划程序都会自动启动，并在后台运行，确保用户的计划能够按时执行)。我们完全可以将磁盘扫描、碎片整理、病毒防治等软件都添加到计划任务中，然后由任务计划管理程序来自动激活(运行)它们，从而实现硬盘维护的自动化。
磁盘在计算机之间的转移
Windows 2000的磁盘管理工具特意提供了磁盘转移功能，专门用以解决硬盘在不同计算机之间进行转移的问题，从而极大方便了我们的使用。
在Windows 2000中将硬盘从一台计算机移到另一台计算机的具体步骤为：
1.在第一台计算机上对硬盘状态进行检查，保证硬盘的状态良好，防止硬盘转移出现问题。
2.关闭第一台计算机，取下需要移动的硬盘，然后将它安装到第二台计算机上。
3.启动第二台计算机并打开磁盘管理工具。
4.执行“操作”菜单的“重新扫描磁盘”命令。
5.此时磁盘管理工具就会对系统磁盘状况重新进行扫描，并将新添加的硬盘标记为“外部”状态。
6.右击标记为“外部”状态的硬盘，然后从弹出的快捷菜单中执行“导入外部磁盘”命令并按照屏幕提示操作即可。}
