{两个网络少年}

{ 周星海：(《剑搜索浏览器》和《网际快帆》的作者，14岁)
我接触电脑是1997年10月6日，那正好是我的生日，我家买了一台“Pentium 120＋16M＋2.1GB”配置的机器，当时可需要RMB8000多哦！买这台机器主要为了我老爸炒股用，操作系统还只是“Windows 3.2＋Msdos 6.22”。虽然这台机器不是专门供我使用的，但是我也经常玩玩这台计算机。我曾经无意中Format了计算机的硬盘，装上了Windows 95，使我爸的股票数据全都付之东流并挨了一顿骂。后来拥有了自己的计算机，就开始学习写程序，在我生活的小镇上，几位计算机“师傅”都是会点FOX的，于是我的硬盘里很快就有了Microsoft Visual Foxpro 5.0，不过我没有书籍，因为懒惰所以也没有去向那些师傅请教。直到有一天，我爸从单位的图书馆带来了一本书——《Visual Basic 4.0傻瓜书》，我开始对VB感兴趣，从此便开始了踏上学习VB的漫漫征途。1999年初，我上了网，那时候我就写了《小小Web浏览器》(很幼稚的名字吧＾o＾)，这个浏览器可真是“小”啊，功能少得只有最基础的浏览。在同时，我还建了个主页(实际上，上网之前我就在玩FrontPage了)。之后我还写了《小小Web浏览器》的2.0和3.0版本。后来创立了“剑”工作室(不过到目前为止，这个工作室还只有我一个人孤军奋战哦)，把原定为《小小Web浏览器》4.0的东西改名为《剑搜索浏览器》，并获得了好评。2000年的寒假，我又写了《剑搜索浏览器》2.0，这个版本主要改进了界面，还支持了6窗口浏览。同年的暑假，《剑搜索浏览器》摇身一变成了《网际快帆》，支持多窗口浏览，搜索引擎定义等新功能，还有一套脱离IE的书签。至于现在嘛，我在写《网际快帆Ⅱ》和《网际快帆网页版》(HTML格式，VBS代码)。想学的东西其实也很多，比如Flash、3DS MAX、ASP或CGI、VC等，最想学的当然是VC啦！
记得我刚开始对编程感兴趣的时候选择了VB，那时候到处去找不同的控件，找到了“剑”工作室，一个很有想像力的名字，而且周星海的名字听起来很像香港影星周星驰，于是我在E－mail中还动用了“老兄”这个称呼，后来无意中看到了控件中说明文件的介绍，“啊，在上小学六年级！只有12岁！”惊讶之极的背后让我深深地感叹时代的变迁，除了动画片、游戏，软件编程也进入了他们的生活，虽然他们的水平不高，虽然他们没有软件工程的思维，但他们是用心在学习，在编程。
蒋三强：(“VB木屋”的站长)
我与电脑结缘是在小学三年级，今年我14岁了，接触计算机已经有很长时间了，那时候在北京师范大学方正新天地电脑培训中心做那种“贵族式”培训。学的是Windows 3.2、Windows 95基本操作、Windows 95网络管理、VB、硬件维护，多次被评为“学习标兵”、“优秀俱乐部会员”。虽然结业考试成绩不算最优秀，但辅导老师说我很有灵气，很有前途。就借着辅导老师的吉言，我和电脑整整“厮守”了六年，在这六年的摸爬滚打中，我总结出了一整套学习电脑的经验理论和实践技巧，虽然不是很精悍，但对只有十几岁的我来说，还是很有价值的。
逐渐地，计算机已经成为我生命中不可或缺的一部分，如果没有了计算机，我真无法想象我的生活会是什么样子，就算有高级音响、游戏机、VCD，但那都无法代替计算机给我带来的乐趣，计算机到底有什么？计算机是一个无穷尽的世界，以前入门的时候我就觉得计算机会改变我的一生，开始的时候只是学操作，自从学了计算机编程，我便总想出新的开发点子，开拓这前人未曾开拓过的世界。虽然我至今还没有什么惊世大作，但我相信，不久的将来，这样的作品一定会出现的。在中国，软件业如果在这样的一个体制下，是很难成为支柱产业的，但在如此的逆境之中，我们如果又能站起来，那又是何等的伟大？所以我就下定决心，好好学习，一定要为中国软件业做出自己的一点儿贡献。
接触网络是从两年前开始的，一个偶然的机会，由于一开始对E－mail颇感兴趣，索性就给Safe Clean的作者赵羽珩发了个E－mail，并很高兴地收到了回复，这次网络成功通信让我更有信心对网络进行最深一层的探索。我发觉计算机网络和计算机程序语言一样有趣，她是另外一个无穷尽的世界，你可以尽情地创造，在这里，你可以结交到更多的朋友，学到更好的技术，更棒的编程技巧，当然，还有网络上最基本的通讯手段。
我的网站作品第一个是VB木屋，第二个是聆峰网，因为觉得前者规模不够大，所以就和几个伙伴寻思着做一个大一点儿的站，还在筹备之中，也希望大家多提意见，多多捧场。
认识蒋三强也是一个很偶然的因素，当时他曾经在国内某网站发表了一篇很独特的抨击中国教育状况的文章，当时也曾经激起了一个系列的讨论，很难想像这出自于一个孩子的手笔，我们十四五岁上学的时候，根本无法想像能够说出这样的话来，然而网络的出现给这些孩子带来了更加激进的思想，让他们的眼光更加的开放，说的话更加的大胆、超前。其实我在和蒋三强聊天的时候，虽然有时感觉到他有些自大、放纵，而且从上面的语句中大家可能也能够感受一二，但我却认为，这也从另外一个方面展示了他的自信心，网络和软件带给他们的决不仅仅是一个交流的工具，而是将深深的烙印刻在了他的学习和生活中，他们才是真正的网络一代。}
