{用Windows Me实现共享Internet}

{ 现在办公室或家庭有两台或两台以上电脑的情况已很常见。有时为了共享硬件和软件资源，常常通过采用某种形式将两台或两台以上单机连接成对等网。笔者所在单位办公室的三台微机就连成了一个简单的对等网，其中一台微机使用Windows Me操作系统，另外两台微机使用Windows 98操作系统，不仅可以共享激光打印机、光盘刻录机、扫描仪等硬件设备以及三台微机上的重要程序及数据，而且还可以共享一台微机上的调制解调器访问Internet。
关于如何组建对等网及在对等网中通过一套上网设备使多台微机共享Internet 的介绍文章《电脑报》已刊登过很多，笔者在这里仅将如何利用Windows Me新增的家庭网络功能在对等网内使多台微机共享Internet的方法做个简单介绍，为想在这方面进行尝试的朋友提供参考。
一、 硬件环境
两台或两台以上单机利用RTL8029兼容网卡组成对等网；作为主机的微机具备一套上网设备，如调制解调器、上网电话线等。
二、 软件环境
具备上网设备的主机必须安装Windows Me并具有一个ISP上网账号；其他微机可安装Windows 95 或更高版本，最好是Windows 98。
三、 主机设置
第一步：安装“Internet 连接共享”
1.打开“控制面板”中的“添加/删除程序属性”。
2.单击“Windows 安装”选项卡，单击“通讯”，然后单击“详细内容”。
3.选中“Internet 连接共享”复选框，然后单击“确定”。
4.单击“下一步”，然后按“家庭网络”向导中的指示操作。安装完成后将重新启动计算机。
注意：
1.如果最初是从光盘安装的 Windows Me，则系统会提示你插入光盘。
2.如果网卡未安装好，系统将给出提示，并拒绝安装“Internet 连接共享”。
第二步：配置“Internet 连接共享”
1.在桌面“网络邻居”图标下单击鼠标右键并选择“属性”选项，在出现的“网络属性”窗口中应包含“Internet 连接共享”、“Internet 连接共享(协议)→Internet 连接共享”、 “Internet 连接共享(协议)→拨号适配器”、 “Internet 连接共享(协议)→Realtek RTL8029AS PCI Ethernet NIC”、 “TCP/IP(共享连接)→拨号适配器”、 “TCP/IP(家庭网络)→Realtek RTL8029AS PCI Ethernet NIC”、“ TCP/IP→Internet 连接共享”等共享相关协议。如果没有，那么是前面第一步没做好，需要重做第一步。
2.选择“网络属性”窗口中的“TCP/IP(家庭网络)→Realtek RTL8029AS PCI Ethernet NIC”点击“属性”选项卡，当TCP/IP属性窗口出现后选择“指定IP地址”，在IP地址栏中填入“192.168.0.1”，在“子网掩码”栏中填入“255.255.255.0”，然后重新启动计算机。
3.重启计算机后，点击“开始” →“程序” →“附件” →“通讯” →“家庭网络向导”， 按照屏幕上的提示进行操作，制作一张“家庭网络Internet 共享连接用户配置软磁盘”，该软盘上共有5个文件：SETUP.EXE、NConn32.dll、NConn16.dll、ICSRMT.DLL、ICSAPI32.DLL。
4.在Windows Me主机中，通过“拨号网络”正确建立Internet拨号连接并设置好默认连接。
四、配置其他微机
配置其他客户计算机可以采用两种方法：自动配置和手工配置。
第一种方法：自动配置
1.将“家庭网络Internet 共享连接用户配置软磁盘”插入A盘。
2.在Windows中打开并运行A盘中的“SETUP.EXE” 。
3.按照屏幕上的提示进行操作。
4.重启计算机后，客户机的设置就完成了。
第二种方法：手工配置
1.在桌面“网络邻居”图标下单击鼠标右键并选择“属性”选项，在出现的“网络属性”窗口中选择“TCP/IP→Realtek RTL8029AS PCI Ethernet NIC”点击“属性”选项卡，当TCP/IP属性窗口出现后为每台客户机分配唯一的、范围在“192.168.0.2”到“192.168.0.254”之间的IP地址。
2.指定每台客户机的子网掩码为“255.255.255.0”。
启用每台客户机上的“DNS”，然后输入一个唯一的主机名。建议使用“网络属性”中“识别”选项卡上显示的客户机名称作为各自的主机名称。在“ DNS 服务器搜索顺序”下添加“192.168.0.1”。
3.在每台客户机上输入值为“192.168.0.1”的“默认网关”。
4.重启计算机后，客户机的设置就完成了。
五、使用Internet连接共享
当以上对等网中主机和客户机都分别正确安装设置好后，多个网络成员就可以分别或同时连接到 Internet 上网浏览、收发E－mail、上BBS聊天、网上炒股、上传下载程序或数据等。在主机桌面右下方的任务栏中有一个共享任务图标，通过单击鼠标右键我们可以调出快捷菜单，查看有几台微机在共享上网，同时也可以随时禁止其他微机共享上网。
六、Internet连接共享工作原理及特点
当网络中的某台计算机从 Internet 请求信息时，该请求首先送到“Internet 连接共享”的宿主计算机。然后，宿主计算机将该请求发送到 ISP。数据从 Internet 或 ISP 返回时，宿主计算机将数据传送回最初发出请求的客户计算机。由于对 ISP 或 Internet 来说，唯一可见的是“Internet 连接共享” 宿主计算机，因此只需要一个 ISP账户。
此外，“Internet 连接共享” 加强了网络的安全性。只有“Internet 连接共享”宿主计算机对 Internet 可见的特性，使得你的网络上所有其他的计算机对非法访问不再是“不堪一击”。
