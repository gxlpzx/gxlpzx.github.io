安能辩我是雌雄——两款权威CPU检测软件

　　暑假已经来临，DIY市场又将迎来新一轮的购机热潮。在组装电脑时，朋友们最怕的莫过于买到被Remark的假货了。现在市场上Remark的PⅢ和赛扬还不少，而且JS的造假手段越来越高明，真让人头疼。不过，笔者今天向各位介绍两款绝对权威的CPU真假检测软件，有了它们，你就相当于炼就了一双“火眼金睛”，可以放心大胆地挑选产品了。
Intel Processor Frequency ID Utility
Intel Processor Frequency ID Utility是Intel公司自己开发的检测Intel系列CPU的专用工具，它可以帮助最终用户识别CPU是否超出Intel的额定运行频率，其结果当然权威了。
##2 一、工具简介
Intel Processor Frequency ID Utility是通过一种特殊的频率确定算法（速度检测）来检测CPU正在以何种内部速率运行，然后再读取CPU中的内部数据与检测到的操作频率进行比较，最后将比较结果告诉用户。该软件支持目前市场上几乎所有的Intel处理器（包括各种主流移动CPU在内）。此外，Intel Processor Frequency ID Utility还可以识别多处理器系统中每个处理器的频率（最多支持32个处理器），但前提是操作系统必须是Windows NT或Windows 2000，否则我们无法使用该软件中的多处理器菜单和工具栏选项。(^28030402a^)
Intel Processor Frequency ID Utility是一款免费工具，具有多种语言版本，支持简体中文，英文、法文、意大利文、德文、日文等11种文字，大家可以从驱动之家（下载地址为http//www.mydrivers.com/tools/dir2/d1136.htm）找到它，文件大小为1.01MB。软件下载后需要进行安装，这里就不多说了。
二、如何看CPU的测试信息
运行Intel Processor Frequency ID Utility后，很快便可以看到CPU的相关信息（见图1），这里一共有“频率测试”和“CPUID数据”两个标签页，里面显示的就是CPU的测试结果。
1.在“频率测试”标签页中，软件分别从“处理器”和“系统总线”两方面显示“报告的频率”和“预期的频率”。其中“报告的频率”显示的是处理器和系统总线以该软件测得的实际操作频率。而“预期的频率”是Intel期望处理器和系统总线运行的速度，也就是包装上标明的速度。如果在“频率测试”标签页中显示有“通过”的字样，那么恭喜你！这颗CPU是货真价实的产品。
2.“CPUID数据”标签页的测试内容有“处理器分类”、“处理器细节”、“系统配置”、“处理器特性”四个部分，可查看的CPU信息十分详尽，其中前面两项信息最重要。
（1）处理器分类:这里提供了处理器的类型、系列、型号、步进四项信息。“类型”主要说明CPU是由最终用户还是由专业个人计算机系统集成商、服务公司或制造商安装，这里的“0”即说明所测试的CPU是用于由专业个人计算机系统集成商、服务公司或制造商安装的；“系列”主要用于说明Intel处理器的品牌以及属于第几代产品，这里的“6”是最常见的代号，它代表赛扬、奔腾Ⅱ、奔腾Ⅱ Xeon、奔腾Ⅲ和奔腾Ⅲ Xeon处理器，“5”则代表奔腾处理器和采用 MMXTM技术的奔腾处理器，最新的奔腾4处理器的系列值为“F”（十六进制）；“型号”与系列通常是相互配合使用的，用以确定计算机中所安装的处理器是属于处理器系列中的哪一种特定类型；“步进”用以标识生产Intel处理器的设计或制造版本数据（例如这里的“3”），用于标识一次“修订”。
（2）处理器细节:这里所报告的高速缓存信息包含二级高速缓存大小（如果二级高速缓存已经存在并已启用）以及一级数据和指令高速缓存大小。另外，这里通常还会报告CPU的封装信息，像这里的“S.E.C.C./S.E.C.C.2”是单边接触插盒的意思，它是一种矩形塑料的封装形式。现在最常见的当然是FC-PGA封装方式了，它的外观是一个绿色材料制成的薄方片。
三、检测CPU是否被超频
通过Intel Processor Frequency ID Utility，你还可以迅速判断你的CPU是否被JS超频。由于Intel处理器产品性能较佳，因此在一定限度内超频后仍然可以稳定运行，一般用户根本无法察觉。
如果你在图1所示的主界面中能看到“所测试的Intel（R）处理器在其频率规范内操作”这行信息，那么说明你的CPU是未经Remark过的，否则可就要去找JS算账啦。
最后，笔者要说明的是，由于Intel不断推出新的处理器产品，Intel Processor Frequency ID Utility也需要经常更新才能识别新一代的处理器，因此，你可以通过“文件/转到下载网站”命令进行在线更新。美中不足的是，这款工具只能用于检测Intel系列的CPU，无法识别非Intel公司制造的处理器！
AMD CPU Information Display Utility
作为Intel的死对头，AMD公司也推出了一款检测CPU的专用工具AMD CPU Information Display Utility，不仅可以检测AMD系列CPU，而且还提供对Intel、Cyrix、IDT的支持。AMD CPU Information Display Utility可以在http//file2.mydrivers.com/others/cpuinfo.exe下载。
该软件适用于Windows 9x/Me/NT/2000/DOS等各种操作平台，AMD CPU Information Display Utility的个头很小（才68KB），而且还是个绿色软件，下载后无须安装，直接运行即可使用。
AMD CPU Information Display Utility的检测信息很简单，主界面看起来十分简洁（见图2），(^28030402b^)一共只有8行数据:第一行显示了CPU的类型和厂牌，如果是AMD系列，则显示如“AMD Durontm Processor”的信息；如果是Intel系列，则显示“GenuineIntel”的字样。　第二行显示检测到的CPU主频（Speed:XXX MHz）。第三、四行显示的是CPU内部一级数据缓存和指令缓存的容量大小（雷鸟和Duron处理器的一级缓存均为64KB+64KB）。第五行显示CPU二级缓存的容量大小（如果是256KB，则表示是雷鸟处理器；如果是64KB，则表示是Duron处理器）。第六行显示CPU的系列（Family Mode）、步进信息（Step:x.x.x）。第七、八行显示是否支持Intel MMX、3DNow等多媒体指令集（如果支持则显示“YES”，否则为“NO”）。
