网上听广播

　　网络广播技术从诞生到今天也就短短的几年时间,但现在因特网上已经有非常多的网络广播电台。此时此刻,网络上有数不清的电台正在播放高质量节目。这些电台的类型多种多样,从传统的广播电台的因特网版本,到某个爱好者自己主持的业余电台。它们播送的节目五花八门,如HI-FI音乐、新闻、天气预报、交通预报。我们只要使用正确的接收软件,就能免费收听来自世界各地的广播节目。
一、网上广播的概念
　　网络广播Net Radio,Net Broadcast,Internet Broadcast,顾名思义,就是通过网络进行广播,即借助网络进行广播。这里讲的网络,不仅指因特网,也包括局域网,但主要是指因特网;而广播的形式,不仅有声音,也有图像和文字。
简单来说,网络广播就是通过在因特网站点上建立广播服务器,在服务器上运行节目播送软件,将节目广播出去,访问者在自己的计算机上运行节目接收软件,访问该站点,收听、收看、阅读广播信息。
　　网络广播有一个非常专业的名字"流媒体"Streaming Media,Stream Media,也叫流式媒体,流式广播。所谓流Streaming,也叫"流式"或"串流",是指一种传输数据/信息的方式。
二、网上广播的接收
　　在互联网上,用于声音广播的文件格式有很多种,除了RealAudio,还有MS Audio、MP3等。接收网络广播的软件的种类也是多如牛毛。接收软件使用流技术,可实时播放来自于因特网上任何电台的节目,而不用在播放之前等待下载巨型文件。有些接收软件可帮助你搜索感兴趣的电台,而有些提供了预置功能,让你迅速连接到指定的电台。下面就给大家介绍两款最常用的网络广播接收软件。
##2 1.用RealPlayer听广播
　　RealPlayer是网上收听收看实时音频、视频和Flash的最佳工具。它可以让你享受更丰富的多媒体体验,即使你的带宽很窄。使用RealPlayer不必下载音频/视频内容,只要线路允许,就能完全实现网络在线播放,极为方便地在网上查找和收听、收看自己感兴趣的广播、电视节目。(^28040601a^)
　　去http//www.real.com/或者在http//www.onlinedown.net/REALPLAYER.HTM下载最新的RealPlayer,目前最新的版本是8.0。然后就是安装,安装过程和大多数软件安装几乎相同,安装与配置非常简单。
　　提示:安装过程中你需要选择网络连接状态。根据不同的网络连接速度,RealPlayer会自动选择配置声音或视频的播放效果。当然,你也可以手动调配这些播放效果设定,不过如果这些效果设定所要求的网络传输速度超过了你的实际网络连接速度,那么就会出现播放不畅的现象。
　　然后你可以用三种方法来加载文件收听广播:
(1)单击广播网站主页上的播放广播的文件链接,RealPlayer将自动启动,并加载文件,或者拖动文件或链接到RealPlayer窗口或图标上,RealPlayer将启动并加载文件;
(2)直接在图1所标示的地址输入栏里,输入播放文件的地址后,即可播放。
(3)在图1的频道区直接点击已经加入的频道,RealPlayer会自动连接到频道地址,然后播放。
2.用Media Player收听网络广播
　　去微软的站点或者到软件屋网站http//www.softhouse.com.cn都可以,目前最新版本的Windows Media Player 7.0,软件大小为9.1MB,然后进行安装,如果你装的Winodws Me操作系统,Windows Media Player 7.0会在安装系统的时候,自动安装到你机器上。
　　Windows Media Player提供了一个网络收音机,你能通过它收听到全世界各国的广播电台。单击窗口左边的"收音机调谐器"按钮,会在右边的窗口出现网络广播的页面。Windows Media Player 已经内置了多个网络广播电台(图2)。(^28040601b^)
点击需要添加的电台,在窗口左边的"Presets"下拉列表选择文件夹你也可以单击"Edit"按钮来添加新的文件夹,然后单击"Add"按钮添加到文件夹中。当收听网络广播时,只需要在已经保存的文件里双击电台名称即可。
　　如果你知道要播放的流式媒体文件或已存储的多媒体文件的URL或路径,请单击"文件"菜单上的"打开",然后键入URL或路径便可以马上连接上地址,收听网上广播(图3)。 (^28040601c^)
