Video VoxPhone Gold使用详解

Video VoxPhone Gold是一款功能全面的网络通话工具，能够轻松地实现PC to PC、PC to Phone的网络通话。和其他的网络通话工具相比，它具有以下的一些特点：第一，便捷的E-mail呼叫方式：大多数的网络通话工具都是通过IP地址呼叫对方的，这样在通话前必须先查到自己的IP地址并发送给对方，操作起来不太方便，而Video VoxPhone Gold不但能够通过IP地址呼叫对方，而且还可以通过E-mail地址呼叫对方，使网络通话变得更加方便；第二，良好的兼容性：大多数的网络通话工具只能实现本软件用户之间的通话，而Video VoxPhone Gold则能够实现与其他网络通话工具之间的通话；第三，强大的辅助沟通功能：Video VoxPhone Gold能够像ICQ类网络寻呼工具那样进行文本方式的交流、互传文件；第四，Video VoxPhone Gold具有语音邮件的功能，当你一时无法与对方联系上的时候，可以通过语音邮件给对方捎个口信。
一、 软件简介
Video VoxPhone Gold为共享软件，最新版本V2.0，文件大小3.79MB，未注册的版本可以免费试用30天，软件的下载地址是http://www.voxphone.com或者http:// www.healthphone.com。该软件安装完毕后，在开始菜单/程序的“VoxPhone Gold”程序组中可以看到两个子程序：“PC to PC”和“PC to Phone”。原来这是两个独立的程序，点击“PC to PC” 项，弹出的是Video VoxPhone Gold；点击“PC to Phone”项，弹出的是著名的Net2Phone，一个专门拨打普通电话的网络通话工具。由于Net2Phone在《电脑报》上已有详尽的介绍(请见1998年《电脑报合订本》下册P115页、1999年《电脑报合订本》下册P377页)，这里不再赘述。
二、 软件的设置
1.基本设置
第一次运行Video VoxPhone Gold，首先会弹出一个很小的对话框，几乎看不见对话框中有什么东西，但你必须在其中两个微型的正文框(First Name和Last Name)中填上内容，填写完毕并确认后即弹出设置向导。
(1)填写用户信息：必填的项目包括：First Name、Last Name、E-mail地址、Mail Server(SMTP)；
(2)设置声卡驱动程序：一般Video VoxPhone Gold能够自动探测到你使用的声卡，如果你使用的是全双工声卡，请选中“Use Full Duplex”项；
(3)测试扬声器音量：点击“Play the Message”按钮，播放录音文件，如果觉得音量的大小不合适，可以拖动滑动块进行调节；
(4)测试麦克风音量：点击“Record”按钮，录上几句话，然后点击“Playback” 按钮播放刚才的录音，如果录音音量大小不合适，可以通过滑动块进行调节；
(5)环境设置：这里有三个选项，如果通话的环境比较安静，请选择第一项，如果通话的环境比较嘈杂，请选择第二项，如果你选择第三项，可以通过“Test”按钮让Video VoxPhone Gold自动测试环境噪音的等级；
(6)设置网络连接类型：根据你的上网方式或Modem的连接速率选择合适的网络连接类型；
(7)执行测试：点击“Start Test”按钮，Video VoxPhone Gold会对你的系统进行测试，主要测试系统对语音的压缩和解压能力；
(8)选择电话目录：该目录是用户个人信息在Video VoxPhone Gold服务器中发布的目录，在线用户可以根据该目录方便地进行网络通话。设置完毕之后，就会弹出程序主界面。
2.选项设置
点击程序主界面菜单栏“Tools”下拉菜单中的“Options”项，弹出选项对话框，在该对话框中一共有五个选项页面。
(1)“User”页面：该页面中的内容和设置向导中“用户信息”的内容完全一样。
(2)“Directories”页面：如果你想把自己的信息发布到Video VoxPhone Gold服务器的某个目录下，请选中“VoxPhone Online Users List”项(这时你在该项后的下拉列表中看不到任何可选的目录，这是正常的，只要你在设置向导(8)的“电话目录”中已经选择了一个目录，当连上网以后，Video VoxPhone Gold就会将你的信息发布到指定的目录下)。如果你希望程序启动后自动将你的信息发布到服务器指定的目录中，请选中“Automatically publish name on start”项。另外，在“Retrieve User From”项下的列表框中此时也是空的，当你连上网以后，服务器中的目录列表便会出现在该列表框中。
(3)“Blocking”页面：可以把一些不希望接入的电话加入过滤列表，这样凡是符合过滤规则的，Video VoxPhone Gold都会自动将它们过滤掉。过滤的方法有两种：直接过滤法和间接过滤法。如果你准备使用直接过滤法，请选中“Block the following users”项，然后把需要过滤的用户添加进列表框中，这样凡是列表框中用户打来的电话，Video VoxPhone Gold都会将它们过滤掉；如果你准备使用间接过滤法，请选中“Accept the following users only”项，然后把愿意接受的用户添加进列表框中，这样凡不是列表框中用户打来的电话，Video VoxPhone Gold都会将它们过滤掉。
(4)“Audio”页面：可以选择不同的语音数据压缩率，在下拉列表中，有六种不同的压缩率可供选择。如果你的网络速度比较快，你可以把压缩率设置得小一些，这样通话的质量会好一些；如果你的网络速度不够理想，你可以把压缩率设置大一些，以保证通话能够正常进行。
(5)“General”页面：可以设置来电拒接Video VoxPhone Gold将其转为自动拒接的时间，单位为秒。
三、 软件的使用
1.启动 Video VoxPhone Gold：当你启动Video VoxPhone Gold后(连线状态)，Video VoxPhone Gold会自动将你的个人信息发布到服务器指定的目录下并把服务器中其他在线用户的个人信息下载到在线用户列表(Online Users List)中，这样在线用户之间就能够很方便地通话了。
2.拨打电话：拨打电话有三种方式：给在线用户列表中的用户拨打电话，给本地地址簿中的用户拨打电话，直接输入E-mail地址或IP地址拨打电话。
(1)给在线用户列表中的用户拨打电话：点击“Call”按钮，弹出“Call”窗口，打开“Online Users List”页面，选中列表框中的某个用户，然后点击“Dial” 按钮，Video VoxPhone Gold 即开始拨号，如果对方接听了电话，你在程序主界面的列表框中能够看到对方已被添加进来，现在你们就可以通话了；如果电话拨不通，或者对方拒接，就会弹出一个信息提示窗口，询问你是否想给对方发送语音E-mail，点击“Yes”按钮，就会调出语音邮件录制窗口，你可以通过语音邮件与对方取得联系。另外，在这里你可以查看到在线用户的详细信息，还可以把自己感兴趣的用户添加进本地地址簿中。
(2)给地址簿中的用户拨打电话：打开“Local Address Book”页面，选中列表框中的某个用户，点击“Dial” 按钮即可。给地址簿中的用户拨打电话，不像给在线用户拨打电话那样每拨必通，因为对方不一定在线。在这里你还可以对地址簿中的内容进行编辑、添加或者删除用户。
(3)直接拨叫：打开“Direct Call”页面，在正文框中输入对方的E-mail地址或者IP地址(注意，给其他网络电话的用户拨打电话只能填入IP地址)。如果你是给Video VoxPhone Gold的用户拨打电话，请选中“Voxphone to Voxphone”项；如果你是给其他网络电话的用户拨打电话，请选中“Standard Based”项，然后点击“Dial”按钮即可。这里需要说明一点：使用E-mail拨叫对方，其前提条件是被叫方已经把自己的个人信息发布到服务器的某个目录下。E-mail拨叫的原理有些类似于域名，域名是通过域名服务器转换成IP地址的，而这里的E-mail地址是通过Video VoxPhone Gold服务器找到与之对应的IP地址的。
3.接听电话：当有电话打入的时候，如果此时你已经设置为自动接听电话(选中“Phone”下拉菜单中的“Auto Answer”项)，在程序主界面已连接列表框中就会出现一个用户的条目，你们就可以直接通话了；如果你没有设置为自动接听电话，当有电话打入时，会弹出一个对话框，告诉你是×××打入的电话，点击“Accept”按钮，即可接听电话；点击“Reject”按钮，拒接电话；如果超过一定的时间没有接听，Video VoxPhone Gold会自动拒接。
4.文本聊天：如果在通话的时候，双方的网络状态不够理想，无法进行正常的通话，可以通过文本方式进行交流(此时双方必须处于连接状态)。点击“Text Chat”按钮，弹出文本聊天窗口，该窗口分为上下两个部分，上面为文本输入窗口，下面为文本显示窗口(包括双方所发送的信息)，在文本输入窗口中输入需要发送的短讯息，点击“Send”按钮即可。
5.语音邮件：如果对方此时没有在线，你可以给他发送一封语音邮件。点击“Voice Mail”按钮，弹出“Voice Mail”窗口，在“To”后的正文框中输入收信人的地址，在“Cc”后的正文框中输入抄送人的地址，在“Subject”后的正文框中填入邮件的主题，点击“Text Message”按钮，会打开一个文本输入框，在这里可以写上几句话，最后，我们就可以开始进行语音邮件的录制了。点击红色的录音按钮，就可以开始录音。录音的长度没有限制，不过太长了，发送和接收邮件都会有困难。录制完毕后，点击左端的按钮就可以试听刚才的录音，如果效果满意，点击“Send”按钮，Video VoxPhone Gold就会把语音邮件发送出去。需要说明的是：对方接收到的是一封带有附件的邮件，附件名为voxmail.vox，要想播放该文件，必须使用一个专门的播放软件Voxware’s ToolVox Player(该软件为免费软件，可以从http://www.voxphone.com. 处下载)。
6.电话会议：通过网络电话开电话会议是一种方便而省钱的方式，下面介绍如何召开一个电话会议。
(1)建立一个电话会议：首先拨通所有电话会议参加者的电话，然后点击主窗口中的“Conference” 按钮，弹出“Conference Manager” 窗口，选中“Create New”项，在“Available Users” 项下的列表框中可以看到所有已经与你建立了连接的用户，选中列表框中准备邀请参加电话会议的用户，点击“Add” 按钮，该用户即被添加进“Users In Conference” 列表框中。依次把多个用户添加进该列表框中，一个会议最多可以有五个人参加(包括会议主持者本人在内)，最后点击“OK”按钮，这时你会发现主窗口用户列表中所有参加会议的用户项都被选中(为黑色)，这时候网络电话会议就可以开始了。
(2)把其他用户添加进正在召开的电话会议：点击“Conference”按钮，弹出“Conference Manager” 窗口，选中“Add to Existing”项，在该项后的下拉列表中选中当前会议项，然后选中“Available Users”列表框中需要添加的用户，点击“Add”按钮，然后点击“OK”按钮即可。
7.传输文件：Video VoxPhone Gold允许通话双方互传文件。点击“Tool”下拉菜单中的“File Transfer”项，弹出“File Transfer Send”窗口，在“Local File Name”项下的正文框中指定文件所在的路径，也可以通过正文框右端的浏览按钮指定文件所在的路径，然后点击“Send” 按钮即可。在文件接收方的桌面上马上会弹出一个“File Transfer Receive”窗口，在该窗口中可以看到文件来源、文件名、文件大小等信息，文件默认的保存路径是软件安装路径下的Download文件夹，也可以更改文件的保存路径。如果决定接受该文件，请点击“Accept”按钮；如果不想接受该文件，请点击“Reject”按钮。
另外，在程序主界面的下方可以看到一个“AGC”按钮，AGC(Automatic Gain Control )按钮的功能是自动控制麦克风的音量 ，使其处于最佳的状态。Video VoxPhone Gold还有传输视频文件的功能，但用户必须装有摄像头。由于国内目前的网络传输速度并不理想，因此该功能对国内网民而言用处并不是太大，这里不再详述。
四、 结语
从以上介绍，可以看出Video VoxPhone Gold是一款功能多样且易于上手的网络通话工具。笔者在测试过程中感觉它的通话质量相当不错，询问了几位不同地区的在线网友，他们也感觉通话质量很不错。不过笔者在试用过程中也发现了令人遗憾的地方：Video VoxPhone Gold提供的服务器好像不很稳定，有几次测试时根本无法下载在线用户列表(不知是服务器没有开启还是其他的原因)，虽然服务器是否正常工作并不影响用户之间的通话，但如果服务器不能正常工作，该软件所提供的E-mail呼叫功能就无法使用了。
