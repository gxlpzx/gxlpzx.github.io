充公挖掘网络的潜能——网络加速软件大放送

SpeedyNet
软件类别：共享软件30天试用版
加速原理 优化系统配置方案、本地保存IP
注册费用：$20.00
运行环境：Win95/98
软件大小：1.3 MB
下载网址：http://www.quiklinks.com/
用浏览器进入某个网站之前，IE都必须要先找到该网站的IP。其实这段时间是可以缩短的，SpeedyNet就是可以缩短这一段时间的好工具。安装SpeedyNet之后，将域名“翻译”成IP地位址的工作，就从IE落到SpeedyNet身上。因为SpeedyNet有一个事先建立好的资料库。另外，很重要的一点是：SpeedyNet的资料库可以随时从网络上更新，它会连接到SpeedyNet的服务器去加上最新的资料库，当然如果你有自己得到的新地址，也可以自己手动输入。还有一点，SpeedyNet最好用的功能应该是它还可以直接将IE浏览器的书签或Netscape的书签，直接加入所有的域名。加入书签时，你需要确定已经连接网络，因为在将书签加入资料库时，SpeedyNet需要顺便确定和域名相对应的IP。另外，SpeedyNet可以根据操作系统和连接方式对传输的数据包大小进行调整达到最大的吞吐量，从而提高网络的传输速度。而且，SpeedyNet还有个安全设置的功能，它能够自动扫描超过75个普通的容易被攻击的端口，如果有人试图连接你的计算机，它会向你发出警告；而且你也能够通过手动来检测出入侵你计算机的人，如果你选择了手动扫描，在选择了端口后，你要有一点耐心，它在一时半刻是停不下来的。
Net Optimizer
软件类别：共享软件
加速原理 优化系统配置方案
注册费用：$15.00
运行环境：Win2000
软件大小：597 KB
下载网址：http://customizer.tripod.com/net.html
Net Optimizer是一个专门针对Windows 2000操作系统所设计的网络最佳化工具。以容易使用的图形界面让使用者更改Windows 2000网络隐藏设定和选项，将Windows 2000操作系统的性能发挥到最佳。而Net Optimizer的优化效果是非常的明显的，由于它在优化的项目上是很全面的，它不仅可以对常见的MTU、MSS、RWIN、TTL、NDI、QueryTimeOut、MaxSockets、Namecache、PachCache进行优化，还能够对SlownetBootmodemTimenet等其他同类网络加速软件没有的项目进行优化！从使用中我们发现Net Optimizer的功能是非常强大的，而它的设置却是非常的简单，你可以按照它的各项和自己的实际情况进行正确的设置！
DUN Manager
软件类别：共享软件30天试用版
加速原理 优化系统配置方案
注册费用：$42.00
运行环境：Win95/NT
软件大小：971 KB
下载网址：http://www.magsys.co.uk/dunman/
在上网的过程中，拨号是大多数人必经之路。有没有软件能够实现集拨号和网络加速功能于一身呢？ DUN Manager是个不错的选择！在使用的过程中，DUN Manager能够简化和提高拨号网络和远程访问服务的效率。DUN Manager运行的界面相当的简洁和清晰。DUN Manager可以以图标的形式运行，只需单击便可开始/停止拨号连网，它能够一直保持“活跃”状态以免因为静止ISP而中断你的连接，当然还有很重要的一点是它还可以提供性能图表让你查看速度。DUN Manager还有几个诊断工具，包括Ping、Trace Route、Finger和地址查找 (Address Look up)，为你的使用提供了很大的方便。由于DUN Manager采用了客户压缩代理的加速原理，当上网的时候，SERVER上的数据是先传给代理服务器而不是直接传给客户，代理服务器对数据进行压缩后把压缩的数据传给客户，通过客户的解压软件将所压缩的数据还原然后进行浏览的。而采用这样的方法可以在一定的程度上解决网络数据的传输量小的问题而提高客户浏览的速度。DUN Manager的一些其他功能还有：单击连接/挂起、自动断线、详细的连接记录、运行程序、与PC时钟同步等，如果你对此感兴趣，你可以根据自己的情况进行设置。
Simple DNS Plus
软件类别：共享软件10天试用版
加速原理 实时网络加速、优化系统配置方案、本地保存IP
注册费用：$30.00
运行环境：Win95/NT
软件大小：2290 KB
下载网址：http://www.jhsoft.com/
Simple DNS Plus的运行界面非常的简洁明了，但是这一点也不影响它的强大的功能。它可以作为DNS 服务器使用，如果你还没有自己的域名，那么它将可以帮助你缓存所有的 DNS 记录从而加快上网速度；如果你有自己的域名，那么这个软件是不可缺少的，它使用友好的图形用户界面来管理域名和 IP 地址。但是不知道为什么这个软件的争议比较大，有的人说它的用处不是很大而有的人却说它还是很有效的。但在使用后发现，Simple DNS Plus能够将你的因特网速度、浏览速度、文件传输速度、下载速度以及邮件转移的速度发挥到极点，而且它还能够提高你的局域网的速度。另外它对域名和 IP 地址的管理非常方便和有效也是其他的软件望尘莫及的。
Netsonic
软件类别：自由软件
加速原理 实时网络加速、优化系统配置方案
注册费用：免费
运行环境：Win95/97/98/NT
软件大小：2229KB
下载网址：http://www.web3000.com/
Netsonic是个比较好的一个网络加速软件它的主要工作原理比较简单：就是通过智慧型缓存记忆模式将喜欢的网页存入在硬盘的缓冲区中使网页的浏览更加方便和迅速但是它的功能还是很丰富的，你可以在“Settings”进行详实的设置，Netsonic所支持的范围在“MTU Optimizer”里面有清楚的说明，它支持28.8K/33.6K/56KModem、ISDN、T1及局域网等各种上网方式。可能是受现在流行换肤的影响，Netsonic还有换肤的功能，这在网络加速软件中还是不常见的，起码可以满足一些人喜新厌旧的心理吧，你可以在“Skins”进行换肤操作。但是令人感到非常遗憾的是Netsonic的操作说明却比较差，有不少的操作说明必须到它的网站上才能够看到。
DNTools
软件类别：共享软件30天试用版
加速原理 优化系统配置方案、本地保存IP
注册费用：与版本有关
运行环境：Win95/98/NT/2000
软件大小：942 KB
下载网址：http://www.atmosoft.com
能够集拨号和网络优化功能于一身的软件不多，而 DNTools是其中一个很好的代表。在使用的过程中，我发现DNTools是一个极好的拨号连网优化工具，能够帮助你加快浏览网页、下载文件、传送邮件等速度。是通过将上网的 Modem 和提供上网的 ISP 系统做最佳化处理，也就是当你安装好这个工具，通过精灵引导设定好提供你上网的 ISP、拨接账号、密码、忙线时的重拨次数等，当你要上网时只要执行 ModemBooster，利用它的拨号功能来拨接上网，让你的 Modem 与提供你上网的 ISP 系统达到最佳的配合，平均能提升 10.50% 的传输效率，号称能够提升到 300% 的传输效率。它主要是通过优化注册表的相关设置包括MaxMTU、PMTU、TTL，提高传输速率，另外它还可以监视连线状态，如果断线或速度过低则用声音提示并自动重新连线。在DNTools中可以显示当前接收/发送的字节数及速率、最大值，因为具有“auto-ping”功能，可以防止下载超时，而且用HOSTS文件管理，对于经常访问的站点可以跳过DNS查找，加速网页浏览速度。
Termianl Overdrive 2000
软件类别：共享软件30天试用版
加速原理 实时网络加速、优化系统配置方案、本地保存IP
注册费用：与版本相关
运行环境：Win95/98/NT/2000
软件大小：942 KB
下载网址：http://www.atmosoft.com
Termianl Overdrive 2000 是我最近所发现的一个很好的网络加速软件，不但带来的效果好而且它的操作非常简单。如果你将它配合“小蚂蚁”和IE用，可以让你充分享受到网络加速的快感。另外该软件的功能还是很齐全的，它主要有下面的几个功能：(1)Auto Accelerate (2)Sericous Net Search (3)Netmarks(4)Smart Screen Saver(5)System Lock (6)Business Calculator 7Capture the Screen 8Virtus Scan。无论如何，从上面所列出的8项功能和在使用的过程中的感觉来看，Termianl Overdrive 2000都是个功能十分强大的软件，在网络上的热门下载软件推荐说明中也证明了这一点。Termianl Overdrive 2000的效果奇佳，如果你是拨号上网的，你可以选择“Opatimize for Dialup”；如果你想提高局域网的速度，那么你就点击“Opatimize for LAN ”；如果是通过局域网来拨号上网的，那么你只要先点击“Opatimize for Dialup ”，等到拨号后再点击“Opatimize for LAN”即可。至于Termianl Overdrive 2000的其他的功能，由于各人的需求不同，是否使用就由你自己来决定了！而我个人认为Termianl Overdrive 2000其中大部分的功能还是相当的好的，如果你喜欢你可以认真的在里面挖掘，说不定会有比较大的收获呢！
而且Termianl Overdrive 2000它非常的体贴我们中国的用户，你并不需要注册就可以永久使用它，唯一不足的地方是在你使用的时候它会提示你需要注册。不过如果你要升级到Termianl Overdrive 2000 pro 你就要花点钱了！
总的来说，现在的国产的网络加速软件在加速原理方面好像都是选择了“优化系统配置”，这可能是和大多的电脑迷的计算机的设备配置有关，但是如果当我们计算机的设备配置比较的合理或我们的网络条件有了很大的改善的时候，“优化系统配置”这样的网络加速软件所能够起到的作用就相当有限了。另外，和国外的同类软件相比(例如Termianl Overdrive 2000强大的功能)，国产的软件在功能上就显得比较单一了，我觉得我们还是应该放宽眼光，多开发其他类型的网络加速软件而下载软件多线程的前车之鉴是值得我们的软件开发者警醒的。
