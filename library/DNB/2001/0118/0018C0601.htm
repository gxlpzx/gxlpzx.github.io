磁盘缓存优化高手——Cacheman

电脑在运行一段时间后速度有时会很慢，这除了和内存的容量有关外，还可能是系统对磁盘高速缓存管理不善造成的。因为电脑在运行较大的程序之后，Windows为硬盘配置的磁盘高速缓存就会占据大量的物理内存(有时高达60%～80%)，并很难全部释放出来，这时一款磁盘缓存优化软件就显得非常重要了，笔者今天就向大家推荐这样一款工具——Cacheman。
Cacheman介绍
Cacheman是一款优秀的磁盘缓存优化软件，它的大小只有624KB， 最新版本是4.1版，适用于Win95/98/Me操作系统。该软件的下载地址是：http://www.esoftware.com.cn/file/system/sysimpro/2001031301.shtml，它可以根据用户的不同要求对系统的磁盘缓存进行优化。
Cacheman用法
一、查看系统资源
启动Cacheman后，首先可以看到它的Information(系统信息)标签页(如图)，Cacheman通过五个曲线图和数据来反映用户电脑系统当前的基本情况：1.Free Physical Memory：系统当前空闲的物理内存容量。2.Disk Cache：磁盘高速缓存的大小。3.Virtual Memory Usage：虚拟内存的使用情况。4.Free GDI Resources：可用图形资源的百分比(低于10%，系统可能变得不稳定)。5.Free User Resources：可用系统资源的百分比。(^18030401a^)
二、缓存优化设置
对当前的系统情况有了了解之后，你就可以点击“Settings”(磁盘高速缓存设置)标签页，进入缓存优化设置窗口。Cacheman采用了智能优化设置方法，即它会自动替用户对磁盘高速缓存进行优化设置。前提条件是：用户必须先根据自己的实际情况，在“Preset”(预先设置)菜单中先选择操作系统(Win95/98/Me)，再选择用户的使用类型，Cacheman一共提供五种用户类型：
1.Average System：适用于一般用户。
2.3D Games：适合于游戏玩家，特别是内存容量小于128MB的玩家，它可以帮助你避免在进行3D游戏时出现停顿现象。
3.CD Writer：适用于拥有CD刻录机的用户，Cacheman会将磁盘高速缓存中的最大值和最小值的设置得相同。
4.Lower Memory System ：适用于系统内存较少的用户。
5.Multimedia：适用于经常使用多媒体软件的用户。使用多媒体软件需要大量的内存空间，当然也需要更大的磁盘高速缓存。
6.Power User：如果你需要同时运行多个大型程序，可选用这种用户类型。经过Cacheman的优化，你会感到系统的反应速度更快。
7.Sound Recording：如果你经常使用电脑进行录音和音乐制作，请选择此用户类型。Cacheman会设置比其他用户类型都要大的磁盘高速缓存，使硬盘的运转稳定快速。
三、Cacheman基本设置
完成高速缓存的优化设置工作，点击“Options”(选项)标签页，进入Cacheman的基本设置窗口。在这里，你可以设置“Refresh Intervals”(更新时间间隔)，“Load Cacheman with Windows”(开机时是否自动启动Cacheman)等选项，或者选择软件的默认选项。
相信有了Cacheman这一磁盘缓存优化工具，你的系统一定会“健步如飞”！
