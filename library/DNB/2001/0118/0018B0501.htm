恢复双启动一法

如果我们同时在不同的分区安装了Windows 98和 Windows 2000，但Windows 98崩溃了或需要重装Windows 98时，双启动的界面没有了怎么办？
当然这时可用Windows 2000的启动盘来修复，但毕竟有些复杂，这里介绍用Windows 2000 Osloder来恢复双启动的方法。其实用Windows 2000 Osloder来修复和用Windows 2000启动盘原理差不多，但是过程简单多了。
Windows 2000 Osloder包含Windows 2000中的WINNT.EXE、DOSNET.INF、NTLDR等文件。另外如果有Bootfont.bin就可以使双启动的界面变为与Windows 2000操作系统版本相应的语言。还有Boot.ini文件，可以备份以前的，也可以按下面的简介自己制作。操作步骤如下：
1.将NTLDR拷贝到C盘根目录下。
2.将WINNT.EXE、DOSNET.INF拷入到同一目录下。
3.重启动计算机进入DOS6.22或DOS7.x(Safe Mode Command Prompt模式)
4.执行Winnt /a一路回车，至无法继续时按“Ctrl-Alt-Del”热启动，即可看到Windows 2000 OS Loader界面。
5.进入DOS或Windows 98，删除Swin_ntS.～bt文件夹。
6.修改Boot.ini文件，大功告成。
这里给出几点说明：
1.如C盘是FAT32格式，就要用DOS7.x启动，不然DOS6.22无法认出C盘。DOS7.x是Windows 98自带的，启动时按F8就可以选择了。
2.Boot.ini可以在Windows 98下用记事本写好保存。一般情况下，只有一个硬盘，C盘装Windows 98，D盘装Windows 2000，可以参考下例。
Boot.ini示例：
boot loader
timeout=10
default=multi0disk0rdisk0partition2\WINNT
operating systems
multi0disk0rdisk0partition2\WINNT="Microsoft Windows 2000 Professional" /fastdetect
C:\="Microsoft Windows 98 SE"
3.如果需要显示中文双启动界面，要在格式化C盘前备份C盘根目录下的Bootfont.bin文件。
