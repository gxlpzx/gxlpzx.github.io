ACDSee也能网上传情

ACDSee是大名鼎鼎的图像浏览工具，怎么把它同网上传情联系到一块了呢？原来ACDSee3．1(SR-1)新增了很多新的插件，其中就包括用电子邮件发送图片的插件，该项功能能够自动识别系统中Outlook Express设置的邮件账号，并提供了地址簿管理功能，将图像作为附件直接发送到指定的地址。其具体的操作方法如下：
启动ACDSee(SR-1)，借助于Ctrl或Shift键，在ACDSee的文件浏览窗口中选中一个或多个希望发送的图片文件，然后点击菜单“Plug-ins/Send image by e-mail”命令，打开邮件发送窗口(图1)(^30040403a^)，在From中选择已有的OE账号，然后在to输入框中直接输入收件人地址，或者通过右边的按钮选择相应的地址；在CC输入框中输入或选择抄送的地址，最后在Subject输入框中输入邮件主题和在Message输入框中输入相关邮件内容后，点击“Send”按钮图像即作为附件开始发送到指定地址。
点击“Options”按钮则可以打开选项设置窗口(图2)(^30040403b^)，该窗口共有三个选项：Basic Options用于添加、编辑和删除邮件发送的账号；Advanced Options选项卡共有两栏，上面一栏(Mail Server Options)用于设置邮件的最大容量和服务器的相应时间，下面一栏用于设置图片的相关属性，比如在发送前将图片自动转换为JPG格式和限定图片大小，以减小图片体积，减少发送时间；最后一项Address Book用于添加、编辑和删除邮件地址。
ACDSee的邮件发送并不使用第三方的邮件程序，而是自身具有直接的邮件发送功能，对于经常发送照片给亲朋好友的电脑爱好者来说，也不失为一个比较好的选择。
