十八般武艺样样会——磁盘镜像多面手WinImage

大家知道，HdCopy不但可制作软盘镜像，还具备了强劲的软盘复制、格式化、修复功能；Ghost是制作硬盘镜像的利器，用它制作的硬盘镜像可快速恢复操作系统；WinISO是使用、制作ISO文件(光盘镜像)的高手，而WinImage以其407KB的身材集上述十八般武艺于一身，作者的构思实在是令人折服。　　
软件详细信息：
最新版本：6．0
文件大小：407KB
软件授权：共享软件
使用平台：Win9X/Win2000/WinMe
下载地址：http://www．winimage．com/download．htm　　
一、 安装与工作模式转换
WinImage是绿色软件，下载后解压缩至指定目录，运行WinImage．exe弹出如图1(^30020202a^)窗口，默认工作模式为标准(Standard)，仅支持软盘、光盘镜像，如果要制作硬盘镜像，需要将“Options”下的“WinImage mode selection”设置为“WinImage professional mode”。　
二、应用举例
1．制作软盘镜像
①点击“Disk”菜单在弹出的驱动器列表中选择“Use Drive A”；
②然后选择同一菜单下的“Read Drive A”导入A盘文件，如图1。程序主窗口中显示了A盘中的文件名、大小、类型、创建时间等信息；
③依次进入“File→Save”菜单，在“另存为”对话框中输入生成镜像的文件名；
④再选择保存类型，如压缩的IMZ镜像格式、普通的IMA格式及PowerQuest Image file的VFD格式，在Comment一栏下添加注释。
提示：如果文件内容敏感，点击“Password”即可为其加上一把大锁。按“保存”结束操作，生成镜像文件(如图2)。(^30020202b^)　　
拯救软盘数据技巧：当磁盘有坏道时，Windows常常无法成功读取，WinImage是按磁道复制生成镜像文件的，我们可以尝试为坏盘制作镜像，然后写入新的软盘或使用Extract功能恢复数据。　　
2．编辑镜像文件
①打开一镜像文件；
②分别选择Image菜单下的Inject(添加)、Extract(提取)、Delete File(删除)、Creat Directory(新建目录)可直接对打开的镜像文件进行操作，修改结束后点击“保存”刷新镜像文件。
提示：从镜像中提取文件，选中列表中文件后点击 Extract，在弹出的窗口中选择保存路径后按“OK”按钮。提取文件操作类似于压缩文件的解压缩操作。
3．复制镜像文件
打开镜像文件后选择“Option”下的“Write Disk”或“Format and Write Disk”，前者直接向A盘写入镜像中全部文件，后者首先格式化A盘再进行写入操作。
操作技巧：如果需要制作多张相同内容的软盘，在“File”菜单下选择“Batch Assistant wizard”，在相应窗口中设定源盘与复制目标属性即可进行批量制作。
4．制作EXE文件
如果需要发布镜像文件，对方没有WinImage软件怎么办？使用File下的“Creat Self-Extracting File”命令将软盘镜像生成脱离WinImage的可执行文件，直接双击该文件就可将文件释放至硬盘或将镜像文件写入软盘。
5．ISO文件的制作与还原
在Disk菜单中还有一项“Creat CDRom ISO Image”，在光驱中插入光盘，点击它，在相应的窗口设定ISO文件的保存路径及文件名，点击“OK”便可以将光盘上的内容以ISO压缩包的形式保存下来。
提示：WinImage对ISO文件的管理功能稍稍逊色于WinISO，不能对光盘镜像中的文件进行删除、改名、添加等操作，只能使用Extract命令将ISO文件还原至硬盘中使用。　　
三、其它功能：
1．格式化软盘：类似HdCopy的格式化功能，WinImage也提供了多种标准(360KB、720KB、1．44MB等)与非标准(820KB、1．72MB、1．68MB、DMF等)的格式化选项。
2．制作硬盘镜像，选择“Disk”菜单下硬盘分区名称，操作与制作软盘镜像文件类似，只是不能加密，只生成IMA格式。还原时选择同一菜单下的“Write Disk”命令。(^30020202c^)
3．整理镜像碎片：经常进行读写操作后存在不同程度的文件碎片，影响读写性能，可用“Image”下的“Defrag current image”命令进行碎片整理。　　
WinImage常用的热键定义：Ctrl+O：打开镜像；Ctrl+S：保存；Ctrl+A：选择A盘；Ctrl+R：读取目标盘信息；Ctrl+W：写入文件；Ctrl+F：格式化&写入；Ctrl+X：提取文件。　　
WinImage操作简单，功能全面，表现亦出类拔萃，堪称磁盘镜像多面手。如同WinZip，实为各位菜鸟至大虾人手必备之工具。
