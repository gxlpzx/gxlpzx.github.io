深入了解Temporary Internet Files

如果你在家中上网了，Temporary Internet Files对你来说，可是一笔不小的财富。就让我们马上一起去探询它吧。
了解Temporary Internet Files目录
Temporary Internet Files默认的路径在Windows的目录下，是Windows用来临时存储我们最近浏览过网页文件的地方。也就是说，我们在浏览网页的同时，电脑在硬盘上开辟了一个缓冲区，把浏览的信息保存起来，浏览器在以后的工作中，将首先检查硬盘缓冲区中是否存在相应的数据，如果有，则直接从硬盘上读出，不再从网上下载(这一点我们在浏览器的“后退”、“前进”中就能体会到)。Temporary Internet Files正是缓存文件夹。通过上面的简单说明，就能得到以下结论：Temporary Internet Files这个缓存文件夹，不仅大大提高了我们的浏览效率，而且它保存着我们浏览过的大部分信息——网页HTM、GIF动画、JPEG图像、FLASH、MID、MOV等。
现在，你是不是已迫不及待地要打开Temporary Internet Files里边的文件了吧。别心急，文件在Temporary Internet Files下是不能直接被打开的，而且会出现一个警告语“在该项目中运行系统命令可能不安全”。但你可以把文件复制或剪贴到别的文件夹中就可以用了。不过，我还是推荐大家用ACDsee看图软件。(ACDsee的强大功能，我就不多说。如果你还没有就下载一个吧！)它可直接浏览Temporary Internet Files下的文件，不会出现讨厌的警告。
探索Temporary Internet Files中的宝藏
在Internet Explorer脱机状态下，打开Temporary Internet Files中的HTM文件看看，网页中的图片和插件(如FLASH、MOV、WAV)没有了。看看源代码，其实图片和插件虽然也下载到硬盘上，但网页的源代码中的图片、插件的地址不会改变。这不同于我们在浏览时用另存的方式保存网页，它同时会把源代码中的地址改过来，在离线浏览时，就会看到图片了。但FLASH、MOV、MID等是无法用另存下载的，这也是我们更关注Temporary Internet Files的地方。
说起FLASH大家都不陌生，作为当前最为流行的网页制作软件之一，不仅被广泛地运用到网页的广告中，而且更为广泛地被一大批称为闪客的网友和一些制作小组用于制作个人作品上，如FLASH MOVIE、FLASH MTV、FLASH GAME等，这可能就是大家喜欢FLASH的地方。
在这我向大家推荐几个不错的FLASH站点：网易动漫(http://www．cartoon．163．com)、七巧网(http://www．7qiao．com)、闪客帝国(http://www．flashempire．com)、Showgood(http://www．showgood．com)。大家有空不妨去看看。
我们了解了Temporary Internet Files，就再也不用频频地去光顾FLASH网站了。当我们浏览了带FLASH的网站后，FLASH会以扩展名是．swf的文件形式保存在Temporary Internet Files中，你可以用Internet Explorer播放，当然之前你还得下载播放FLASH的插件。值得一提的是，在线浏览FLASH时，不要一出现“play”时，就以为它下载到硬盘上了，最好能完整地播放一遍，这样就能确保下载到Temporary Internet Files中的完整性。这样喜欢收集FLASH作品的朋友，就可以把Temporary Internet Files中的SWF文件拷贝出来收藏了。
最后再说说MID和MOV，MID作为一种音频文件格式，常被添加到网页中。MOV作为一种视频文件格式，也常被作为插件添加到网页中。用QuickTime Player软件欣赏完网上提供的MOV后，它们也下载到你的Temporary Internet Files中了。MID文件也是同样的道理。我们也可以把它们从Temporary Internet Files文件里拷贝出来单独收藏。
对于一些文本文件，大家可别忽视了，它们一般都是些精美图片，ACDsee能很容易的区分，用ACDsee预览一下吧。
设置Temporary Internet Files
如果你有充足的硬盘空间，合理地设置好Temporary Internet Files目录，不但可以加快你的浏览网页的速度，还可以帮你储存更多的网页资源。
第一步：打开IE进入“Internet选项”，如图1。(^30040601a^)　　
第二步：在设置Temporary Internet Files区，点击“设置”按钮。
第三步：进入设置区后，如图2所示(^30040601b^)。在设置窗口中，可以用滑动条或者直接输入数字大小来调整Temporary Internet Files目录空间大小。“移动文件夹”来 改变Temporary Internet Files的默认目录。如果你想快速地进入Temporary Internet Files查看文件，用“查看文件”就可以直接进入，默认或者你自己设置的Temporary Internet Files目录里。
