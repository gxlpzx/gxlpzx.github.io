驱动程序更新快报

读者朋友们，大家好！新一期驱动快报又和大家见面了，让我们一起来看看这两周厂商们都提供了什么好东东！
显卡驱动方面：NVIDIA显卡的公版雷管驱动14．20版面世了，在基于雷鸟1．2G、DDR内存、GeForce3及Win 2000的测试环境下，这款驱动让显卡发挥了前所未有的效能，测试分数提高了将近一倍！令人遗憾的是，这款驱动还没有通过WHQL认证，稳定性还有待于提高。
ATI和Matrox显卡的视频播放效果是同类显卡中最好的，这次两家厂商又不约而同地发布了新版的视频工具包。其中ATI发布了ATI Multimedia Center 7．1版，这是为ATI-TV WONDER和TV WONDER VE专门打造的，但它需要Direct× 8．x及其以上版本的支持。Matrox 推出的是Marvel G400-TV、Marvel G200、Marvel G200-TV、Rainbow Runner G系列显卡最新Video Tools 2．04．015多语言版工具包，同时还附上了这几款显卡的最新驱动。
由于被收购的关系，3dfx已停止开发Voodoo系列显卡的驱动程序，但一些Voodoo爱好者和程序小组仍然在为Voodoo努力。这次他们发布了Voodoo3/4/5系列显卡的Win XP驱动，使用Voodoo系列显卡的朋友不用再为升级操作系统而担心了。
另外，很久都没有更新驱动的Creative(创新)发布了3D Blaster显卡的12．90版驱动，这款驱动基于NVIDIA公版12．90。
主板方面：Intel(英特尔)芯片组Chipset Software Installation Utility 3．10．1011版新鲜出炉了，它能够使你的Windows操作系统“认识”所有的的英特尔芯片组，并能够自动安装相应的．Inf文件，以实现芯片组的功能特征。这个版本除了支持Win XP外，还支持i830M(代号为Almador)、i845(代号为Brookdale)等芯片组。
技嘉发布了GA-8TX-C主板最新BIOS Fb版，这个版本修正了从Win 98SE系统中删除驱动导致系统死机的问题。要注意的是技嘉GA-8TX-C主板有两种型号，分别对应不同的BIOS容量，不能刷错！具体判断方法是：系统启动时如果屏幕显示的BIOS版本是F1/2/3，那么你的GA-8TX-C就是4M Bits flash的型号；如果屏幕显示的BIOS版本是FA/B/C，那么你的GA-8TX-C就是3M Bits flash的型号。同样GA-8TM主板也有这种情况。
升技也为自己的KT7、KT7-RAID、KT7A、KT7A-RAID、KT7E等主板推出了最新BIOS 3R版。该版本升级了BIOS编码，增强了ISA板卡即插即用的兼容性，增强了SCSI适配器的兼容性。(^30030403a^)
