Actiontec 56k USB来电等候modem
　　大多数拨号上网的用户会为只有一条电话线发愁，因为上网的时候外来的电话就无法接入。也许来自朋友的抱怨常常令你尴尬："昨晚本想打电话找你，可是你又在上网..."，可见上网已经对你的正常社交活动造成了负面影响。怎么办？是去申请第二条电话线，还是掏钱装条"叫响不叫座"的ISDN一线通呢？如果你尚未上网或者考虑更换掉慢吞吞的33.6k modem，那么本文介绍的一种富于创意的modem将让你远离上述的窘境！这就是Actiontec推出的新式56k USB来电等候modem(Call Waiting Modem)。其实，Actiontec公司生产了三种不同版本的来电等候modem，其中包括内置56k PCI、外置56k串口和外置56k USB三种接口不同的型号。到底这款modem有什么过人之处，请耐心读下去便知！

　　Actiontec 56k USB modem是我所测试的第一款USB modem，所以我也不知会发生什么情况。整个包装盒中包括了任何外置modem都配备有的部件：安装手册、通讯软件、一条电话线和外置modem本身。安装过程非常简单，只需将modem接到USB端口并安装驱动程序。使用USB modem的好处是无需附加AC适配器，因为该modem通过USB端口获得电源。该语音/数据modem同时支持56k Flex和V.90协议，采用朗讯(Lucent)的56k modem芯片组。

　　被测硬件信息： modem名称：Actiontec 56k USB来电等候modem 公司：Actiontec电子产品公司 类型：k56 Flex/V.90外置USB modem 接口：USB（通用串行总线） 零售价：104.95美元

　　最低系统要求： 带USB端口的486或以上IBM兼容机 16MB内存 CD-ROM驱动器 视窗95、98或MacOS 要求本地电话公司提供的来电等候服务。

　　Actiontec 56k USB来电等候modem的主要特点当然在于其来电等候功能。该modem允许在上网期间接入外来电话，此时用户可选择接听电话，或者任由电话铃响而保持上网。首先，你要在当地的电话公司登记使用来电等候服务，然后USB来电等候modem就象一台普通的电话那样在外电打入时提醒你。在7秒钟的时间内你必须决定，是接听电话还是不管它，继续上网。如果你接电话，此时modem挂起，但是如果要保持上网的话，短暂的线路交涉之后，就可继续冲浪。

　　有些时候，比如下载过程较长时，我不想有任何的外来电话打断下载，这时我希望56k来电等候modem能象一台标准modem那样工作。这里有两种方法可以实现：一是修改拨号帐号的Internet设置项，禁止使用来电等候；另一种更简单的方法是使用配套的来电等候工具软件(Call Waiting Utility)，选择下列选项之一：启用来电等候响铃；接听来电后自动断开网络；禁用来电等候提示；

　　来电等候功能正如产品广告所宣传的那样行之有效，尽管我真的希望有超过7秒更长的时间来决定是否接听。如果你正在等待一个非常重要的电话，工具软件中的“自动断开”选项也许你会用得着。另外，有不少的应用软件都支持来电提示，但在你决定是否接听电话之前你仍旧无法与对方交谈。对于那些希望在一条电话线上网冲浪，但又不想错过外面打入的电话的用户来说，该56k USB来电等候modem的确是一个非常实用的好产品。

　　该modem的指示灯可谓精简得不能在精简了。在前面板上只有三个指示灯，它们分别是Off Hook(摘机,OH)、Data Transfer(数据传送,DT)和Modem Ready(待机,MR)。当你正在拨号或启动modem时，OH灯着亮。启动PC时MR灯着亮，而modem在发送或接收数据时DT灯会不停地闪动。这些指示灯对一般用户而言已经足够了。

　　Actiontec 56k USB来电等候modem性能可圈可点。测试的连接速度总是低于同类的Rockwell modem，并且处于28.8-41.3kbps之间。与之作对比的其他modem，如USR Sportster，连接速度可稳定在45.6kbps或更高。另外，新款modem的连接时间也落后于其他的modem。不过一旦Actiontec modem连上网后，其性能就表现得非常稳定，测试中没有出现过其他56k modem偶尔的断线现象。总之，该modem的下载速度和稳定性是十分优秀的，没有发现掉线或其他异常情况。美中不足之处在于其整体的连接速率要比其他同类的modem要慢得多，比如USR Sportster 或者Zoom 56K等。

　　Actiontec 56K USB来电等候Modem虽然不是目前市场上性能最高的modem，但它有一把“杀手锏”－来电等候功能，它既帮你节省了第二条电话线的费用，又让你在尽情上网时不会错过打入的电话，真可谓一箭双雕！

五款56K“大猫”介绍


　　随着电信降价，电话费用的降低，上网的人也就越来越多，上网的必备工具“猫”也是必然之选。在此，我结合自己曾经亲自使用过的五款56K“大猫”的经历，为还没有“猫”的朋友们做一引荐。

　　1．联强国际“蝙蝠车”外置式

　　联强国际？？这个名字怎么好像没听过嘛（你听过？啊，那算我孤陋寡闻了）！的确，如果不是我亲自见到并使用之，还真不知道有这款产品！

　　这款产品的型号是 LEMEL MD-56KVR PLUS-黑（PRC），采用了 ROCKWELL 芯片，支持 V.90+K56FLEX、FAX、语音等基本功能。打开包装盒除了些 MODEM 必备的软硬件外，就是这只通身漆黑、造型奇特的“猫”了，整只“猫”打造的很精致，模仿蝙蝠的样子，不是绝对像，却也很“酷”了。

　　联强国际的服务不错，说明书纯内容有56页，从买猫安装到上网设置直至成功，都有比较详细的说明（从某些“老鸟”的角度来看这绝对是“骗钱”的垃圾，但是没上网的朋友还是多数啊？！）。

　　“蝙蝠车”的声音非常小，上线和下线时的那一声“嗒”几乎没有。

　　但是联强国际给了我一个非常不好的感觉，那就是“太小气”了，附带驱动盘中只有 Bitware、Acrobat Reader3.0、Cheyenne AntiVirus For Win95（30天试用版）、Cheyenne Backup For Win95（60天试用版）几个“阿土伯”级的软件。并且该“蝙蝠车”还要价580元。

　　另外，在实际使用过程中，我发现这款“猫”并不是专门为中国大陆生产的！为什么这么肯定，当然有根据。当我安装好“蝙蝠车”后，在 Win98的“控制面板”里多了一个设置项，其中可以设置该猫是在哪个地区使用，而其中只有“中国台湾、中国香港、日本、韩国、马来西亚、美国等等”唯独没有中国大陆，我也不知该不该推荐大家买了，好在如果纯粹是使用，各地区都差不多，没有什么太大的不同。

　　具体测试数据请看文末的表格，笔者将把几款猫进行全面的对比。

　　如果您是个比较富裕、追求造型新颖，以及不太在乎哪个地区的人，那您选择他吧。

　　2.帝盟“速霸”SUPRA 56K 外置式

　　“速霸”棗朋友们应该是相当熟了，铺天盖地的广告早已把我们的耳朵塞满。这款“猫”采用的也是 ROCKWELL 公司的芯片，支持 V.90和K56FLEX。

　　但是它不带语音功能，不要被我们在广告上所见到的“具有语音邮件功能”所迷惑；它不带电源开关，我所见过的外“猫”一般都有电源开关，而独其无；它不分 PHONE和LINE 接口，电话线可以随便插入哪个接口，这也是帝盟公司“SHOTGUN”技术所必需的吧。

　　“速霸”的声音也很小，附带软件一般。但是“速霸”的连接速度的确没给“速霸”这个名字丢脸，几乎在任何时候“速霸”的连接速度都是 52000，只有极少次跑过 50666 的，真的很快。在我用过的56K“大猫”中，只有后文的创新ISA56K内“猫”，可以与之相提并论。

　　可能是由于追求了速度，在连续上网5小时里，曾经有过1次断线现象，而当时是在下午5点左右，笔者正在联网杀敌，所以记得比较清楚。

　　如果您追求品牌、追求高速，“速霸”的确不错，但您也要为此多付出些血汗钱了，自个儿斟酌斟酌。

　　3.创新Modem Blaster Flash 56 V.90 内置式ISA接口

　　前文就说了，这款创新的内置式“猫”跑得很快，一点不错，最高时也是52000，在大部分时刻都能达到这样的速度，已经很不错了，偶尔也在50666附近转悠。

　　此“猫” 有着很不错的电话功能，支持G3标准高速传真速率，具有视频会议功能（接上视频头吧，对面的MM看过来！）

　　同样的毛病，附带的软件确实很一般，不多而且还很老：Creative Inspire、Microsoft Internet Explorer是创新产品的必备。除此之外，仅仅加了Microsoft NetMeeting、Cheyenne Bitware 而已。

　　该内“猫”带有FLASH ROM，支持软件升级，另外与众不同的是多了个“TAD”（电话设备应答接口），反正我是用不上，垃圾！:)

　　但是该“猫”就笔者使用来说，并不是像某些作者所说的“抗干扰能力怎么怎么”，不知朋友们有没有试过同时几千人上网？？我就遇到过，我们南京实行包月上网，每月21日开始，你看吧，21、22几天许多网民争先恐后的直奔 INTERNET，网络拥挤度可以用100%来形容。这时，该创新的“小猫”可就惨了，最低一次曾经在 40000 上鏖战，就差没冲过39999大关了！（^_^）当然，这只是很少出现的，要怪也怪网络带宽太窄了。

　　对于既注意钞票也追求品牌的朋友来说，创新的这块内置式ISA“大猫”当仁不让，300元左右。

　　4．实达“网上之星” 5600PB 内置式 PCI接口

　　实达作为国内品牌，也有不错的声誉，这款5600PB，是采用PCI总线方式，ROCKWELL芯片，当然支持V.90/FAX/VOICE了。可以收发传真、做语音信箱，利用麦克风作电话机使用。

　　该“猫”附带的软件很不错，东方快车和金山词霸，我想这可是不少中国人的电脑里都有的吧，让人觉得很超值喔。说明书也有47页之多，较详细。

　　作为56K的“猫”来说，5600PB的最大缺点可能就是连接速度了，笔者使用时最高只能上到49333，一般来说都是48000的样子，反倒不如一些国外品牌，看来“专为国内线路定制”的广告词还要等一等啊。

　　作为中国人支持支持国货还是应当的，并且随着价格的降低，5600PB还是值得考虑！

　　5．金网霸 CIRRUS LOGIC 芯片 内置式 PCI 接口

　　这款“猫”由于试用较早，具体型号记不得了。

　　它采用的是 CIRRUS LOGIC 芯片，支持 V.90和X2标准，以及语音、FAX功能。虽然号称支持V.90，但是却实实在在的让我大失所望，最快连接速度仅仅为41000，而一般都是在33600，这和33.6的“猫”有何区别嘛！

　　它可以在 Windows 环境下改变协议的，也就是说可以随意改成V.90或X2，这也算得上是它的优点了。

　　虽然价格很低，但由于性能不好，就不多说了，也不向朋友们推荐了。

USB猫+USB HUB=？


　　看见今天的图片了？有什么特别？别告诉我就看出是个猫。看看，再仔细看看它上面写着什么。看见了吧，是“USB+HUB”呦！

　　明白今天的新鲜玩艺是什么了吧，单纯从MODEM上说，没什么特别，USB的MODEM也是最近一半年才出现的，USB HUB也差不多，但是在将它们集成到一起，应该这是第一个在市场上露面的了。记得去年北京COMDEX的时，那时候USB MODEM还只是展览品，我就同欧洲一个使用ST芯片生产出第一只USB MODEM的厂家（不好意思，说了半天，可是我也忘了它具体叫什么）的工程师说，你们这老用USB口，而又不在提供给其他设备USB口，那势必造成用户使用起来的麻烦，至少也要用一个在提供一个吧。如今，我很高兴的见到了这样的产品，再也不用为缺少USB口而发愁了。

　　作为一个猫，它提供了猫的所有基本功能，如果仔细观察它上面的字迹，就会知道它还具有FAX和VOICE的功能。现在就一只带VOICE的猫最便宜的也要3XX，这一点大家打个平手。从外形上看，这个咪咪可真够小的，虽然在它的身上有一大堆要插的东西，可看它后背板一个接一个的插座，就知道它有多大了，肯定比一般的猫小，同DIAMOND SUPRA差不多，在个头上它胜出了。在看它的另一个卖点，当然就是USB HUB功能了。注意了后背板的朋友一定奇怪，这咪咪怎么只有3个下行的USB口？而不是通常的2、4、8个。据我估计，它里面所使用的USB芯片，也应该是4个下行的，但是其中一个被MODEM使用了，于是乎就那么少了一个，这样成本不会上升嘛。在附加功能上，它也胜出了。再看看价格，以前我们也报道过USB，那个漂亮的IMAC风格的4口HUB要3XX，最便宜的其它外形的也要250以上；USB MODEM最便宜的也差不多400以上，合在一起呢，也就是差不多700了，而这个小家伙只要540，是不是相比很便宜？所以价格上的优势还是不少的。

　　最后还要提醒大家两句，别说我没说在前面，其一是这个猫是软的，现在市场上硬的USB猫几乎没有，即便是有，其价格也会远远高于普通串口的外置硬猫；其二是它需要电源才能工作，因为除了供给MODEM的工作外，按照USB的标准，还要为每个端口提供不少于500MA的电流。这恐怕是唯一不及其它USB猫的地方，虽然它所附带的电源质量不错，可是分量是够大的，不宜随身携带，所以就失去了USB产品的移动优势。

　　总体上说，产品功能的集成有利于生产者降低成本，使用者降低花销，如果定位准确，对双方都是有利的。像这样的USB HUB+MODEM的集成，我个人认为相当的不错。至少对我来说，毕竟猫是不能不用的，USB口也是不够用的，而集成后使我少花了钱不说，还节约了桌面面积和电源插座，关键是后面这个我高兴，hehe，连换插座的钱都省了。

两种适用于掌上电脑的CF Modem卡


　　目前有两种CF Modem卡可以工作在你的掌上电脑中，它们分别是Casio JK-711MC56 Data/Fax Modem卡及Pretec 56k CompactModem卡。每种的工作方式都是将其插入掌上电脑的CF卡槽中，然后将它通过模拟电话线连接到ISP处以访问因特网。

　　这两种CF Modem卡的突出特点是能与56k模拟Modem卡实现同样效果的功能；简便插接式配置；插入CF槽工作；节省电池。

　　每种卡配置时无需在掌上电脑中装入驱动程序，只要将这种Modem卡插入CF槽（可在掌上电脑工作时插入），然后再将它与模拟电话线连接上就配置完毕，再建立一个拨号帐户，就可以访问因特网了。

　　所不同的是Casio Modem卡的大小与CF存储卡的大小相同，能完全插入CF槽中，但需要一根连接电缆将它与RJ-11电话线连接起来；而Pretec Modem卡插入CF槽后，上部要有一部分伸出槽外，这时因为其内置了与RJ-11电话线的连接部件，因而体积较大。（见下图） 但二者的功能完全相同。
