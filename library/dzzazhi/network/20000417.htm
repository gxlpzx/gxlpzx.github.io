【自由点评】
1......娱乐网站：切莫自娱自乐

　　与各个打着电子商务旗号的网站风风火火的势头相比，娱乐类网
站的出现就颇有几分润物细无声的味道。尽管在声势上，它不如商务
类网站，但是在数量上却绝对不让于任何类别的网站。尽管娱乐网站
的发展前景颇为人看好，但是这些网站在质量上良莠不齐，定位上模
糊不清却是明白存在的问题。
　　这里说的定位并非单指网站的内容与经营，而是包括其在整个网
络中的位置而言。有很多网站把娱乐这一块与生活类的网站放在一起，
内容上更是五花八门，什么美容服装，乱七八糟一大堆，要不就和体
育等类网站揉在一起，倒也动静分明。不过随着网络的发展，现在娱
乐网站自立门户的越来越多，最近一个例子就是兴网的出现,此之前
还有ChinaByte的Joday，在去年的早些时候就已经分开了，当然还有
更多。
　　在身份明了之后，要做的就是如何把网站经营得更好。在我们的
日常生活当中，娱乐一直就是不可或缺的组成部分，所以相当一部分
娱乐类网站开门伊始便打出旗帜要“好玩”，要成为“提供多媒体发
展的源动力与推动者”。最近刚刚在业内冒头的兴网就娱乐类网站的
发展前景，提出了要做一些与现实（或说传统）的传播行业相结合的
路子。这或许是眼下所有的娱乐类网站所共有的目标，但是做起来可
就不一样了。
　　现在的网站大多以提供一些过时应景的娱乐信息和数百上千甚至
上万的流行歌曲的下载为主要的服务项目，但是在内容更新上的漫不
经心和全无目的，使这些网站看上去更象是一些爱好者的个人主页。
因为过多依靠流行歌曲的下载来吸引点击率，有的网友甚至发出“娱
乐网站没有ＭＰ３就活不下去”的评论，活不活得下去我们不知道，
但现在的大多数娱乐网站在内容提供上确实存在缺乏互动与沟通，而
网络最大的好处就是提供了一个彼此更好的沟通方式，在结果上往往
也更直接，就像在某个ＢＢＳ上贴了一个发言，是否能引起别人的共
鸣，看一看跟随就知道，但是现在的这此网站没有考虑到受众的反馈，
只是简单地重复内容与信息的“更新更新再更新”，完全是一种传统
媒体自说自话的感觉，这就有点无限付出的味道了。
　　最近北京地区对春节期间的电视收视率作了一个调查，结果是强
调普通百姓参与的“超级模仿秀”成为娱乐类节目的收视率老大，而
炒作十足，星味十足的其它节目则退居其后，这从另一方面说明了未
来的娱乐传播发展的又一个趋势，即更注重小人物的参与。与电视相
比，网络的灵活与互动性更有利于大众的参与。
　　都说网络时代是一个“小人物的时代”，从某种意义上讲的就是
“实现小人物话语权的时代”，网站，或者网络，应该是一个平台，
它所承载的不仅仅是提供一些信息的服务，更包含了提供表现空间，
鼓励参与的内涵与实质。
　　发挥网络的长处，应该是娱乐网发展的关键。这个道理应该很浅
显，但真正在这样做的人少，兴网最近在做一个有关高校原创音乐的
网上大奖赛，是一个很明显的张开双手，欢迎加入的姿态，但是最终
结果会如何，是矫情炒作还是真心经营，有待我们走着瞧。

【热点追踪】
1......九个热键卖出631万元

　　美国地区法院法官托马斯．彭菲尔德．杰克逊(Thomas Penfield
Jackson)在本周一（4月3日）作出的裁决中，认定微软公司(Microsoft
Corp,MFST)违反了反垄断法。杰克逊法官的裁决包括，微软通过反
竞争行为来非法保护视窗软件的垄断地位和企图垄断浏览器市场。面
对法官的判决，微软立即声明将提出上诉。
　　尽管这个判决结果也是大多数人所期望的，但是，对那斯达克(Nasdaq)
市场来说，判决来得不是时候：因为投资者在过去一周已争先恐后地
抛售科技类股。本周一，那斯达克综合指数(Nasdaq Composite Index)
下挫349.15点，创下单日下跌点数之最，跌幅为7.6%,以4223.68点收
盘。微软股票周一下午4点在那斯达克股市的收盘价为90.875美元，
下跌了15.375美元，跌幅为14%。
　　在未来几个月中，杰克逊法官所面临的挑战是，拿出制止微软进
一步违法行为的修正措施，同时，还不能对这家在新经济中颇具实力
的公司的创新能力造成影响。具体地说，法官必须设法制止微软非法
使用垄断性的视窗操作系统来壮大在新软件市场上实力的行为。
　　但是，高科技行业发展迅速，微软的垄断还有多大价值？这个问
题在矽谷(Silicon Valley)和华盛顿引起激烈的争论。互联网的飞速
崛起使得微软及其控制十多年之久的电脑桌面系统的威力渐失。微软
已没有能力设定所有软件开发人员必须遵守的标准，大量投资和革新
都发生在互联网业，与微软控制的世界无关。RealNetworks公司是微
软网上传送音像产品的竞争对手。公司主席兼首席执行长罗布．格拉
塞尔(Rob Glaser)说："微软常常能取得一些惊人的成功，而这些成
功正好是市场的中心。目前，市场的中心已经转移。"他说得也许不
错。但是，对微软提起诉讼的美国司法部(Justice Department)和19
个州认为，微软在电脑操作系统上的垄断地位仍然是个问题。事实上，
政府及公司的批评认为，微软使用对付网景公司(Netscape)的同一手
法来压制格拉塞尔的RealNetworks公司。
　　法庭拿出制止违法行为的办法预计也要几个月，随后，微软还会
提出上诉。这意味着这场官司还会再打两年，甚至更长时间，留给微
软足够的时间在互联网软件方面积蓄力量。
　　期间，微软可能继续向新的软件市场进军。今年春季晚些时候，
盖茨将推出公司参与互联网革命第二阶段的5年计划。同往常一样，
这份计划再次引发了问题：软件巨人会不会使用其市场力量控制电信
和电子商务世界。
　　尽管在反垄断战场上巧舌如簧，但微软利用个人电脑领域的垄断
地位进入新兴市场方面的纪录却并非一片辉煌。盖茨及其将士们面对
的互联网是一种任何公司都无法主宰的媒体，就连微软的批评者们也
承认这一点。国际数据公司(International Data Corp.)对99类软件
进行了跟踪调查，结果发现微软只涉足了21类，市场份额超过36%的
只有4类。
　　微软的未来计划，特别是成为占支配地位的互联网伺服器软件提
供商的计划，是以PC桌面系统的垄断地位为基础。公司管理人员已公
开宣称，微软打算开发以Windows 2000伺服器操作系统(Windows 2000
server operating system)为基础的"全方位服务"互联网平台，包
括数据库、网络伺服器和“中间设备”。
　　微软把这项计划称之为Windows DNA 2000创新。这项计划背后的
意图是：提供一整套能够在低价PC伺服器上运行的应用软件和操作系
统软件，以便同Sun电子计算机公司(Sun Microsystems,SUNW,又译升
阳微电脑)等生产Unix伺服器软件的主要厂家展开竞争。
　　在微软垄断案的审理过程中，微软的管理人员一再强调公司的计
划没有违法行为。但执法人员的看法却大相径庭。事实上，微软的互
联网战略同其先前桌面系统建立垄断地位是异曲同工。微软的SQL伺
服器数据库或网络伺服器应用软件只能在微软的操作系统中运行。此
外，微软最新桌面应用软件中的许多高级的网络功能也要求伺服器运
行微软的软件。
　　国际数据公司的分析师丹．库斯尼茨(Dan Kusnetzky)说：“如
果你研究一下他们的整个战略，就会明白，微软提供的是连环扣式的
产品，如果你买了一个，你还得买下一个、再下一个。10年来，该战
术令微软的产品铺天盖地，随处可见。”

【数字生活】
1......“数字间谍”在行动
　

　　你有一个国外电子邮件笔友或者密友吗？你经常使用移动电话吗？
如果是的，那么很可能有个“数字间谍”正在监视你。
　　据美国和欧洲的情报专家说，一个叫做“梯形计划”的大规模电
子拦截计划对所有的Internet通信、移动电话谈话、传真和长途电话
进行搜索，寻找恐怖活动、军事威胁和跨国犯罪的证据。电子间谍活
动正在由秘密工作的美国国家安全局以及澳大利亚、加拿大和英国的
安全机构进行。美国科学家联合会的安全和情报专家John Pike表示，
这一系统正在协助警方抓捕并破获毒品案。但从另外一个角度上来看，
大部分的追查工作放在了无辜者的身上，这值得吗？
　　电子间谍的工作方式
　　“梯形计划”使用一项过滤程序来标出那些带有炸弹、枪和民兵
等关键字的信息。但由于有关“梯形计划”的内容透露得极少，所以
仍不清楚该系统是否能够把犯罪分子发出的信息同遵纪守法的公民发
出的信息区分开来。例如，芝加哥的一个人在发给日本的一个朋友的
邮件中，可能在介绍的一部枪战小说、或者在讨论最新的电影《第一
追击令》，甚至在谈论有关最近的一起恐怖事件的新闻报道时，使用
了两个或两个以上的这种关键字。当“梯形计划”查出这样的信息时
会发生什么事情呢？谁也不知道。
　　如果你是个一般的用户，你本应不会引起国家安全局的工作人员
的注意，但是，据欧洲议会的一个研究分支机构——科学技术选择评
估机构（STOA）委托起草的一系列报告称，“梯形计划”的设备每30
分钟能够处理100万份邮件。这也就是说，美国国家安全局已经布下
了一个天罗地网来捕获几条可疑的“金鱼”。这时，你一定会感到你
的隐私权受到了侵犯。
　　虽然STOA研究宣称，该系统对拦截的材料过滤得非常仔细，在100
万份信息中只有10条信息被拿去接受详细分析，并进行第二级软件过
滤，到达分析人员手中的信息就更少了。但是你是否担心最后这几条
信息中有一个与你这样一个奉公守法的人有关？此外，据报道，这个
系统还使用声波纹技术来搜索电话通信中的目标讲话人。
　　谁来对监视者进行监视？
　　令保护隐私的活动分子感到烦恼的是，一些无辜的人可能会成为
“梯形计划”的目标，或者该计划的间谍活动可能会超出法律界限。
　　“梯形计划”保密得很严，国家安全局甚至不承认这项计划的存
在，更不用说讨论它的目标确定标准或公民自由保障了。虽然有两个
支离破碎的文件根据联邦信息自由法被公布出来了，但这两份文件只
包含经过严格审查的部分内容。因此，在STOA的报告公布之后，引起
了人们的强烈不满。美国公民自由联盟（ACLU）还就此向众议院政府
改革委员会请愿，要求对“梯形计划”进行调查，以确保“梯形计划”
的工作符合法律。
　　而且，当社会进入Internet时代，本来我们就对我们的网上隐私
相当担心。前不久，Internet广告公司DoubleClick宣布计划将其用
户数据并入一个大数据库之后，一些用户组织和电子保密团体竭力反
对。此事引起的骚动使得DoubleClick搁置了这项计划。与此同时，ACLU、
移动通信协会、电子保密信息中心和其它一些机构还正在对联邦通信
委员会的一项法令提出质疑：即联邦通信委员会要求电信业支持安全
部门的监视行为。
　　如果我们知道老板可以监视我们的电子邮件，你会有何想法？虽
然给电子邮件加密可能会起到一些保护作用，而且一些ISP还可以提
供一个谁也跟踪不到的电子邮件名称，这样也可以提供一些保护。但
这样的解决办法对人们来说是不切实际的，而且它肯定不是一项可行
的长期选择。
　　现在，你讲什么话，往哪儿发信，可得注意了。
