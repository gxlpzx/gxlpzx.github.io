李志伟童话选(李大奇系列)
预订分数
    这天大奇回家，发现铁门的把手上又插了广告单。不知从什么时候起，花花绿绿的
广告单出现在他们和邻居的家门上。广告的内容五花八门：有增高的，有减肥的，还有
提高智力延长寿命的。由于吹得太神了，大家谁都不信。
    大奇扯下广告单，象往常一样揉成一团。但是犹豫一下，他又把广告单展开了。广
告上醒目的大标题引起了他的注意——

    预订分数！

    大奇忍不住好奇心，往下看起来——
    亲爱的同学，你是否为学习不好而苦恼？你是否为考试分数的“贫困”而惭愧？现
在好了，我们公司郑重推出“分数预订”业务！方法很简单，比如说你估计这次考试只
能考90分，但你想考个满分给爸爸妈妈一个惊喜，怎么办呢？给我们写信！你写信告诉
我们你所在的学校、班级，以及你想另外预订的10分，接下来，你就会如愿以偿！
    当然，我们为你做这一切不是免费的，每一分需要一块钱。千万不要小气，这几块
钱与爸爸妈妈得到的快乐相比，又算得了什么呢？
    心动不如行动，同学，我们期待着你的来信！

    　　　　　　　　　　　　　　　　　　　幻想镇分数预订公司

    “骗人的，”大奇一边说着，一边神差鬼使地把广告单叠好，放进了裤兜。
    爸爸妈妈还没有下班，大奇就先写作业。可是他今天老是走神。
    “如果真能预订分数……”大奇望着窗外出神，“唉，爸爸妈妈为我费了不少心，
可是我……我再也不愿看见爸爸妈妈在家长会上抬不起头的样子了，我对不起他们。真
的，如果能预订分数……”
    大奇满脑子转来转去的就是“预订分数”四个字。自己跟自己斗争了半个小时，他
终于在作业本上写起来——不是写作业，是写信。

    分数预订公司的叔叔阿姨：你们好！
    我是童话镇大猫路小学五·（2）班的李大奇，我看到了你们的广告。
    不满（瞒）你说，我的成绩不好。我想主要的原因嘛……嗯，不是我不冲（聪）明，
可能是我太贫（贪）玩了吧。我真的很想让爸爸妈妈高兴，所以一看到你们的广告，我
就动心了。
    过几天就要考语文了，我的语文成绩一直不好，特别是作文，一看到那些方格格我
就头皮发马（麻）。我估计自己这次只能考个60分，所以我向你们预订30分，这样加起
来不就是90分了吗？爸爸妈妈得知这个成绩，应该高兴得睡不着觉了吧？
    这是我剩（省）吃捡（俭）用存下来的30块钱，全部给你们寄去。你们可不要扁
（骗）我，因为扁（骗）人会变成小狗！

    　　　　　　　　　　　　　　　　　　　　　　　李大奇 今天写

    大奇封好信，按照广告上的说明填地址。填的时候他突然发现这个地址太怪了，就
去问邮局的阿姨：“阿姨，我这样写地址对吗？”
    阿姨接过信一看，地址是这样写的：

    幻想镇没有街负38号　分数预订公司 收
    邮政编码是：0 0 0 0 0 0

    “我从没见过这么奇怪的地址！”邮局的阿姨扬起眉毛，“我不知道对方能不能收
到。不过你可以投进邮筒试试，如果地址有错，你的信会盖上‘查无此人’的邮戳，完
好无损地退回来。”
    “那我就放心了，谢谢阿姨！”大奇把信投入邮筒。
    接下来的几天，大奇在焦急的等待中度过。他的信没有退回，他半是高兴半是担心
地考完了语文。老师的动作真快，第二天就把卷子改好了。
    “这次考试，大家都发挥出了自己的水平，”王老师捧着卷子说，“特别是李大奇
同学，他平时语文不好，但经过努力，这次考了95分！”
    “哇，95分耶！”大家都象盯明星一样盯着大奇。
    大奇得意极了，心想：嘿，预订分数还真灵！
    王老师换了一种批评的口气，又说：“但是有的同学可能是骄傲了，这次退步非常
大。比如说刘佳佳同学，她平时都考90多，这次却考了62分！”
    大家都看佳佳。大奇也望过去，只见佳佳低着头，脸蛋胀得通红。
    “怪了，”大奇想，“佳佳这次是怎么了？”
    卷子发下来了，同学们小声议论着，有的高兴有的泄气。大奇急不可耐地看自己的
卷子：哇，好多地方的答案都换了，特别是作文！考试时怎么写的作文他都忘得差不多
了，就记得脑子里想到什么句子，就稀里糊涂地往卷子上写。可是你再看看现在这篇作
文：条理清晰，感情细腻，简直跟女孩子写的似的，怪不得得了高分！
    “呜——！”刘佳佳突然哭起来，“王老师，王老师！”
    “怎么了？”王老师快步走过去。
    “这篇作文不是我写的！”佳佳指着自己的卷子说。
    王老师怀疑地看了她一眼，仔细查看卷子。只见作文是这样写的——

    　　　　 记一件好人好事
    那天，我正在借（街）心公园里早恋（练），突然看见马跑（路）上飞来一两（辆）
大卡车……

    “这么多错别字，怎么会是我写的嘛！”佳佳委屈地说。
    “确实奇怪，”王老师说，“可是，这的确是你的笔迹呀！”
    佳佳看了一眼卷子，没办法地哭了。下课后大奇跑过去一看，失去的记忆马上回来，
他知道佳佳的作文是谁写的了；同时，他也明白了，自己卷子上那篇清秀的作文，一定
是佳佳写的！
    放学回家后，大奇没有把卷子给爸爸妈妈看，而是飞快地冲进自己的小屋，摊开纸
开始写信——

    “分数预订公司”的叔叔阿姨，您好！
    你们按照我的要求改变了我的考试分数，非常敢射（感谢）！
    但是，拿到这个分数，我一点也高兴不起来。因为我并不知道，我加上去的分数，
居然要从别人那里抠（扣）除！佳佳学习那么用功，才考62分；而我那么贫（贪）玩，
却考了95分，这不公平！真的，我李大奇虽然贫（贪）玩，却不是个讨厌的自死（私）
鬼！
    现在，我心里非常的不安。过几天就是数学考试，我请求你们给佳佳加上30分！如
果一分一块钱，那我还得付你们30元，可是我没钱了。怎么办呢？就请你们在我的成绩
上抠（扣）去30分吧。这样，我就应该付你们负的30元，正好和佳佳的正30抵消，我就
不用付钱了。
    真的盼望你们能够满足我的原（愿）望。如果不能还给佳佳30分，这份愧疚的心情，
也许会揪（纠）缠我一辈子！

    我先射射（谢谢）您们了！
    　　　　　　　　　　　　　　　　　　　　　　　李大奇 今天写

    信发出去之后，没有退回。数学考试时，大奇答题特别认真，因为他知道，这里面
有30分要还给可怜的佳佳呀！过一天，数学分数出来了：大奇考了51分，佳佳考了全班
最高分：120分！
    “太奇怪了！”班主任王老师说，“100分的卷子，佳佳居然能考120分！”
    好多老师都跑来帮忙查卷子，可是查来查去、算来算去，佳佳就是120分！
    佳佳激动得哭了。大奇躲在人群后看着，悄悄地松了一口气。
    以后的日子里，大奇终于慢慢地收住了玩心，把精力用在了学习上。他的成绩缓慢
但稳步地上升着，王老师还特意表扬了他。
    大奇再也没有动过给“分数预订公司”写信的念头，“预订分数”的广告来了，大
奇就把它扔进垃圾桶。如果这份广告出现在你的面前，你会怎么样呢？
