清水塘，浊水塘
谢连波

    编者的话——
    本是一篇小议散文，但作者却写的津津有味，颇具特色。
    作者从自己亲身体验的一次养鱼的得失中，体会到事物的特定性质和数量界限，继
而上升到“度”的认识，针对现实生活中的种种“过度”的行为和观念提出自己的看法。
小小一事便起到针贬时弊，扬长避短的效应，实见作者观摩生活的能见度。此文值得一
读，读者不容错过。

    我曾看到过一口非常美丽的清水塘，那是采石灰石后遗下的，数亩方圆，清澈见底，
凉气森森的，人见人爱。三伏天扑进清水塘里畅游一番，那股舒服劲可别说了，真是用
什么词语都难以形容。
    那时我在公社当驻队干部，正为生产队发展副业增加收入发愁，想起那口清水塘不
禁大喜，就建议生产队养鱼，来一个物尽其用。可队长却说：“想得倒美，可你没听说
过‘清水塘里不养鱼’么？”
    我说：“说是那样说，但不见得是真的，试一试吧！”
    队长拗不过我，就真的弄来数百尾鳙鱼、凌鱼和蔓鱼的鱼苗放到清水塘里。因为是
我的主意，就特别地关注着塘鱼的生长情况，可不知怎的，半年过去了，尽管又是投料
又是护理，那鱼不但不见长大，还愈来愈少了，最后竟不知所终。
    我纳闷，就请教队长，他有点嘲讽地说：“我早就说不行嘛，古人说下的，能有错
吗？”后来，他还与我去细看了清水塘。原来，是有地下活水源。故使水永远保持清寒，
那么微生物就无法繁衍，鱼无食或少食，水温也很不适宜鱼的生长；水太清了，鱼们无
可隐形，易于受害。如此这般，鱼就消受不了清水塘之福了。
    鱼是生存于混水塘的。
    鱼为何喜欢混水塘呢？那是塘里有油水，有养料，活的畅快；同时混水还可以让其
潜行，得到了保护，遇害的比率少些。
    但水塘是愈混愈好吗？非也。
    80年代初期，有不少乡村的农民在屋内或院子内挖池养塘虱。
    因为在室内院内，池塘里的水只能是死水，主人又投料又勤快，剩饭剩菜泔水经常
往那里倒，池水弄得又脏又混，塘虱倒是长得快，但那肉却是带泥臭味的。而其它如鲫
鱼、鲤鱼等是绝对不能在此等池塘里放养的，放一条死一条。
    我还见过乡村一些过混而臭的鱼塘，数亩方圆的塘，有三分之二水面是死鸡死鸭或
女人例假用过的卫生纸等垃圾，水呈墨绿色，散发着阵阵臭气。这样的塘也不宜养鱼，
天气一热或一阵白雨（晴天突然下雨），鱼们就翻肚而死。这样的塘里平日总会浮着一
条条发腐发臭的死鱼。
    过混浊的臭水塘是不能养鱼的，那样的塘缺清静的水更缺新鲜充足的氧气，腐败环
境里多死鱼。鱼即使不死，那本质也是变臭了的，绝不能吃的。为此，我想到了事情应
有个度。
    度是什么呢？一是表明物质有关性质所达到的程度；二指哲学上一定事物保持自己
的数量界限。
    一般的混水塘最宜养鱼，是因为适度。
    适度的环境，有利于一个地方一个单位的人的团结。凝聚力的增加，积极性的提高，
有赖于适度的环境。
    适度，要我们时时认真把握。有时侯因业务往来，就会有限度地请客送礼，这是我
们的纪律允许的。如若因此而持手中有权，无论是公还是私甚至是请相好亦一律用公款
上酒楼，那就是无度了。所谓物极必反或过犹不及都是这个道理。
    虽然，人至察则无徒，但做师傅或领导，不能不察。不察怎知徒儿或下属的优劣和
功过是非？只有察，不时地考察，才能掌握情况，进行约束，指导和监督。做人有度，
管理者更应有气度，有容人之度，但是，又要有原则。若对原则性的错误也不批评指正，
那不是藏污纳垢吗？
