软件新天地

  …………………………………………………蔡旋
   

先和大家说说微软的新情况：Windows 2000的RC2已经推出了，但现在还是英文的，中文的只有Beta3。RC2的错误已经比较少了，如果你不用到某些特别的功能，不会碰到什么麻烦，看来Windows 2000离推出的时间不远了。另外，微软还在今年10月初发布了DirectX 7.0，但我对此没有什么感觉，版本升级得太快，只是多支持一些新的硬件而已。对游戏开发者来说，DirectX 7.0将3D图形与声音的编程都简化了，能让游戏制作者更加集中精力开发高品质的产品。此次的DirectX还有一个新的VB库件，难道用VB也能做出好游戏？

晴窗中文大侠2000 Ⅱ
版本：V2.3演示版
类型：商业
网址：http://www.sunwin.com.cn
评价：专一翻译很好，但不是所有软件都能译

不清楚大家现在都用些什么翻译软件，也有很久没有见到翻译软件的竞争了，刚刚推出的永久汉化2000和晴窗中文大侠会不会挑起新的大战呢？

以前只知道晴窗是某个图形软件的专用翻译软件，但最新的晴窗中文大侠2000已经能同时支持多套软件的即时翻译，而且继承其一贯专业的特点，针对每套软件进行单独的翻译。

晴窗中文大侠2000把软件分为工具、编程、图形及其它四类，每一类均包含当前最流行的英文软件。它本身是个外挂式的翻译软件，执行后会在Windows任务栏的系统托盘区添加一个中英文切换图标，碰到需要翻译的软件时，只需点击这个图标，软件就会被汉化，而且翻译后不会出现其它翻译软件的字符移位现象，与真正的汉化软件一样。

虽然晴窗中文大侠2000宣称具有智能翻译判别功能，能自动识别当前运行软件界面中该翻译与不该翻译的地方，但我在用WinRAR95时，还是发现它把不该翻译的地方翻译了。使用晴窗中文大侠2000时，最好关闭其它字典类软件，否则容易引起系统冲突。

晴窗中文大侠2000还支持Windows 98和Windows NT，用Windows NT的朋友再也不愁没有翻译软件用了。它会自动侦测运行的系统平台，并调用相应的启动部分。晴窗中文大侠2000采用开放性的翻译扩展功能，附带一个用户字典编辑器，当发现还有遗漏的词汇或认为有翻译不准确的地方时，用户可以自行添加或编辑。

看看电子小说——Read Book
版本：V1.2测试版
类型：共享
网址：http://readbook.126.com
评价：尽显电子小说的魅力

曾几何时，看电子小说也成为一种时尚。我家中小小的硬盘，就能装满几个书箱的书，原来拥有这么多藏书也不是什么难事。偶尔翻翻看看，还能对来访的客人炫耀。看电子书也得有个好软件才行，国内我首推Read Book。

Read Book支持TXT、HTML、WPS等多种文件格式，为了缩小存放空间，你也可以把书本压缩成zip格式，Read Book一样能够读取。我最喜欢的是它的智能分段功能，可以将文章中的段落重新排列，而且还支持Big5码的转换，许多时候它都会自动识别出来，不需要手工切换，繁体中文的文章也能照看不误。

你不可能每本书一次就能看完，所以Read Book提供了书签，你可在任意位置插入，并加上你的描述。如果在关机时忘记放书签也没有关系，Read Book会记住最后一次阅读的地方。
用Read Book看书是不用按键的，只需调整好你的阅读速度，按下Enter键，屏幕会自动上卷，而且它还会根据文章字数的多少自动调整。另外还可选用“鼠标控制”，以鼠标的屏幕位置决定滚屏的速度。

在电脑前看书应该说是比较伤眼睛的，但可以通过调整显示的颜色来减少对眼睛的伤害。Read Book预定有很多屏幕方案，都是些常用的配色方案和字体设置，不满意还可自定。当然要保护眼睛，还是不能在电脑前坐太久。

Read Book还支持插件，在作者的主页上提供MIDI播放插件及文字校正插件。MIDI播放可以边读书边听音乐，而文字校正则有点象替换功能，即把错误的文字改为正确的。
最新的Read Book 1.2测试版比起1.1又多了以下改进：
1.支持滚轮鼠标；
2.改进了文件列表窗口；
3.滚屏时隐藏鼠标。

离线浏览——Cache X
版本：V2.03
类型：共享
网址：http://www.mwso.com
评价：网上的资料可能就在你的硬盘上

众所周知，中国的上网费用高昂，我每次上网前总得先想好该干些什么，才不至于浪费辛辛苦苦赚来的钱。所有浏览过的网页其实都是下载到你的硬盘中，然后再打开的。因此当你没有上网的时候，还可以重温刚才的内容：把IE从“文件”菜单中设置为“脱机方式”，然后输入网页的地址就可以了。但如果忘记了网址呢？

更好的脱机浏览方法就是用Cache X。你能轻松浏览到放在Cache中的网页，它支持IE3、IE4、IE5 及Netscape，而且能比直接用浏览器看到更多的东西。

Cache X缺省是根据网页的名字分类，你也可设置为根据网页的类型、最后的访问顺序等来排列。除了一般的浏览外，你还可以对单独的网页进行删除、拷贝等操作，并能在庞大的Cache中
查找某些文字。

快猫加鞭
版本：V2.0正式版
类型：免费
网址：http://onekey.yeah.net
评价：操作简单的网络加速软件

快猫加鞭是一个网络加速软件，它能在不增加硬件投资的前提下，最大限度地优化网络配置，在有限的时间内获得更多的数据。

它的原理和某些软件相同，都是通过修改注册表的网络设置数值来提高速度。相对其它软件来说，快猫加鞭的操作更简单，轻轻一点即可完成所有优化项目，特别适合于英文不好、对计算机和网络又不熟悉、没有能力动手改注册表的初学者。同时，它也提供了直观的高级界面，以供详细设置各种参数。

它的优化项目较为全面，除了传统的MaxMTU、TTL和DefaultRcvWIN外，还具备优化QueryTimeOut、MaxSockets、磁盘的FileCache、PathCache及设置Modem初始化字符串的功能，并可自动寻找Modem的COM端口进行优化。

高级用户可能想自己定义各个项目，快猫加鞭保存了多种用户设置，只需轻松选择就能得出最佳设置。

值得一提的是作者还精心撰写了详细的说明文件，具体说明了网络优化的原理、实现和操作，对新手来说是网络进阶的极好教材。从中不仅可以学到这方面的知识，甚至可以据此辨别形形色色的网络加速软件的真假与实际效果。
