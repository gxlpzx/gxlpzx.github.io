明明白白我的“芯”



在个人电脑中，CPU就好比是人体中的心脏，可想而知，它的作用会有多大。所以对于计算机用户来说，能够彻底了解一下自己“心脏”的好坏，无疑对“健康”是相当有好处的。对于诸如“先天不足”或“早搏”之类的病，我们还是把它们扼杀在摇篮中为好！因此，在本文中笔者将为大家介绍怎样用软件来测试你的CPU，了解CPU的工作状态及详细的资讯。
识别篇
在中国大陆的计算机市场上，由于价格的因素，个人用户购买计算机时大多是以兼容机和DIY拼装为主。但是由于我国的计算机市场还不够规范，所以常常有些不法商家以次充好，蒙骗消费者。而在这些欺骗消费者的手段中，又以Remark CPU最为常见。我们所说的Remark CPU就是通过超频技术将原本工作频率较低的CPU跳高，当作更高一级的CPU来卖，从而获取更多的利润。下面，笔者将为大家介绍4款CPU测试软件，帮助大家更好地了解自己的芯片，以防受骗上当。

WCPUID
WCPUID是一款由日本人编写的测试个人电脑CPU频率的免费软件，在《大众软件》以前的杂志上曾有过介绍。你可以去http://www.h-oda.com/下载获得，其最新版本为2.6c，软件大小为66k。它可以直接在DOS下运行，解压缩后直接测试就可以了！
运行WCPUID后，主界面如图1。在这里你可以看出芯片制造商（vendor ID），本例中笔者的芯片为Intel的赛扬366，编号为0665（查看TYPE ID、FAMILY ID、MODEL ID、STEPPING ID，然后你可以在网上查询相关资料，看看究竟是何种频率的芯片）。同时，在“时钟频率”（clock frequency）这一项中，你还可以看出CPU的内频、外频和倍频，以及L1和L2的Cache值、是否支持MMX/SSE/3DNow！等信息。
在Feature Flg中，你可以了解到CPU支持哪些功能；在Cache Info和Cache Reg中，分别列出了L1和L2 Cache的详细资料；在AGP Info中显示了有关AGP bus的资料，如AGP的版本以及传输速率等。
WhatCpuIs
WhatCpuIs也是一款免费软件，它的最新版本为v1.0 beta1。虽说是测试版，但在使用过程中也没显示出有什么大的bug。WhatCpuIs所支持的CPU芯片有
Intel：Pentium/Pentium-MMX/Pentium PRO/Pentium Ⅱ/Celeron/Celeron A/Pentium Ⅱ Xeon/Pentium Ⅲ/Pentium Ⅲ Xeon
AMD：K5/K6/K6-2/K6-2(CXT)/K6-Ⅲ/Athlon
Cyrix：6x86MX/MⅡ
IDT：WinChip C6/WinChip 2/WinChip 2A
Rise：mP6
从http://www.gene6.com/你可以获得这个大小为497k的软件。在安装过程中，它只允许安装在你的主盘上，并在“控制面板”中直接生成一个小图标，如图2。
它能精确显示出CPU的各项情报，包括工作频率、型号、内建功能、L1和L2 Cache的资料以及是否具备Intel MMX/AMD 3DNow！或Pentium Ⅲ的SSE指令集等。
WhatCpuIs的功能主要分为5大类，如图3。
Processor：显示CPU的主要资讯，包括工作频率和产品型号。通过这两部分的资料你就可以知道CPU是否被 Remark过了；
Advanced：进一步显示CPU所支持的各项功能，里面的选项相当多，也相当细致。同时，如果你的CPU是PⅢ的话，还可以直接显示出它的序列号；
System：显示操作系统的版本号、使用者及注册号。如果你的计算机已联网的话，还可以显示出该计算机在网络上的名称和使用者；
Setup：该内容尚未开发完成，希望作者能在下一版本中完善它；
Link：通过Link连接，你可以直接访问主要的CPU制造商的网页，从而获得最新的资讯。

Intel R Processor Frequency ID Utility
由于Intel公司出品的CPU占据了大部分的个人电脑市场，所以在此专门介绍一款由Intel自己出品的CPU识别软件——Intel（R） Processor Frequency ID Utility。它可以使客户识别Intel处理器是否已超出额定的运行频率。同时，它也可以用来识别系统中是否已安装Intel的处理器。通过运行一种频率确定算法，它可以确定处理器以何种内部速率运行。然后，该实用程序即检查处理器中的内部数据，并将此数据与检测到的操作频率进行比较。该程序会将系统总体状态列表通知用户，其详细信息主要有：

Intel处理器名称
预期的微处理器操作频率
当前的微处理器操作频率
已测试的处理器（仅限于Windows NT中使用的双处理器/多处理器）
系统配置
处理器特性
处理器分类
处理器高速缓存

由于该软件有很多语言版本，其中也包括中文版，如图4，所以你可以很方便地了解测试的内容和结果。当然，Intel(R) Processor Frequency ID Utility只是针对Intel的CPU才有效，对AMD和Cyrix的CPU就毫无用处了。你可以到Intel的站点下载这个软件，大小为978k，也是免费的哦！

CPU Info
这个小软件是由我的一位同事编制的，只是在企业内部流传过。相对于上面介绍的几款软件来说，它显得比较简陋，所显示的内容也比较单一。但通过它，你还是能够了解一些主要的CPU参数的。
该软件大小为124k，全部文件也就是一个可执行程序，无需安装。运行该软件后，出现的主界面如图5。
在CPU Info里，你同样可以看出CPU的系列号、型号、stepping ID及它的L2 Cache值。在CPU Features里，你可以了解CPU都支持哪些功能，如MMX、I/O breakpoints、Page Size Ext等等。
由于该软件以前不曾在任何网站上发表过，所以如果你想要获得它的话，可以到我的网站http://libin.guomai.sh.cn/去看看，绝对是独家专有哦！

有了以上介绍的4款软件，相信各位读者以后购机时再也不怕自己的CPU是被Remark过的啦！在此，我首先推荐的软件为WCPUID，因为它体积小，又可在纯DOS的环境下运行，所以测试起来更加方便。
工作篇
好了！现在我相信你已经购回了正宗的芯片，并在使用它们了。那么此时此刻，你是否知道它们的工作状态呢？例如CPU的占用率为多少、CPU的工作是否稳定等等。而这些信息，你又可以通过哪些软件来了解呢？

系统监视器
系统监视器是Win98自带的一个工具，它位于开始菜单“程序”—“附件”—“系统工具”中。如果找不到的话也不要紧，你可以在“控制面板”中的“添加/删除程序”里选择“Windows安装程序”再添加上去就可以了。
在系统监视器里，实际上我们可以监视很多项不同的内容，如磁盘高速缓存、Microsoft网络服务器、文件系统等等。但是，它的缺省设置就是显示核心处理器的使用情况，因此一进入它的主界面，如图6，你就可以看到以折线图来显示的CPU的状态。如果你想了解计算机在最近一段时间内的工作情况，也可以通过“开始记录”来把这些数据储存起来，以备后用。

CPU Load
CPU Load是由Idyle Software公司出品的一个小软件。它的大小为407k，最新版本2.0.1.61，下载网址是http://www.idyle.com/cpuload/download/CPULoadSetup.EXE，它也是个免费的软件。
CPU Load的界面主要分为两部分，如图7，左边的Usage中显示出当前CPU的占用率，而右边的History则显示了近一段时间内的CPU使用情况。由于作者采用了图形化的显示方式，绿色的图案、方格再加上黑色的背景，整个界面给人以舒适悦目的感觉。
同时，你还可以选择在计算机启动时就加载该软件。当然，你也可以让它长驻在SysTray里，CPU Load会分别以绿色、黄色和红色来显示CPU处于空闲、忙和紧张3种不同的状态。

CPU Stability Test
CPU Stability Test是由Jouni Vuorio编写的一个自由软件，如图8。
它的最新版本为V1.5，软件大小为569k，你可以到http://www.saunalahti.fi/jv16下载获得。这也是个免安装软件，直接解压缩后，运行test.exe就可以了。
该软件主要的功能就是测试CPU以及系统的稳定性，它有点类似于“拷机”的性质，通过让CPU不间断地运行高负荷程序来看看它是否稳定。首次使用它时，你需要在Advanced中运行一下calibrate（校正）程序，如图9，这样它会自动替你建立一个数据文件。接着你需要做的事情就是按下“start”键，进行长时间的测试。在测试时，它让CPU一直进行大量的复杂计算，长时间且一直循环，看看CPU是否会因不胜负荷而导致错误或者死机。在整个检测中，第一步测试一般会花费30分钟，然后每隔5分钟再检查一遍。所以，建议你在空闲的时候再进行测试，最好是一个晚上都不要关机，以此来考验电脑的稳定性。
CPU Stability Test将稳定性分为7级，也就是0-6级。0当然是最不稳定的，而6则说明计算机在稳定性方面是相当出色的。看看笔者拷机下来的结果吧：6级！你的结果呢？什么？不及格？是不是超频超得太厉害了？还是降下来吧。
